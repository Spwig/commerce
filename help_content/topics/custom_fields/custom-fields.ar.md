---
title: حقول مخصصة
---

تتيح لك الحقول المخصصة إضافة بيانات إضافية إلى المنتجات، والتصنيفات، والطلبات، وملفات تعريف العملاء دون تعديل أي كود. استخدمها لتخزين معلومات محددة بالعمل مثل معرفات واجهات برمجة تطبيقات خارجية، مواقع المستودعات، بيانات الامتثال، أو أي سمة يحتاجها متجرك.

## الوصول إلى الحقول المخصصة

اذهب إلى **الإعدادات > الحقول المخصصة** في شريط التنقل الإداري.

![صفحة الحقول المخصصة](/static/core/admin/img/help/custom-fields/custom-fields-page.webp)

## المفاهيم الأساسية

### مجموعات الحقول

تُنظم الحقول إلى **مجموعات** — مجموعات منطقية تظهر معًا كقسم. على سبيل المثال، قد تحتوي مجموعة "معلومات الشحن" على حقول لمواقع المستودعات، أبعاد الحزم، والتصنيف المتعلق بالمواد الخطرة.

### تعريفات الحقول

تحدد كل تعريف للحقل:
- **الاسم**: الملصق الذي يظهر في النماذج
- **الاسم المختصر**: المفتاح القابل للقراءة بواسطة الآلة المستخدم في التخزين بالتنسيق JSON واستجابات واجهة برمجة التطبيقات
- **نوع الحقل**: نوع المدخل الذي يتم عرضه (نص، عدد، قائمة منسدلة، إلخ)
- **التحقق من الصحة**: قواعد مثل الحد الأدنى / الأقصى، طول الأقصى، نمط التعبير العادي، أو الخيارات المسموح بها
- **الرؤية**: ما إذا ظهر هذا الحقل على متجر البيع

### أنواع الحقول المدعومة

| النوع | الوصف | الاستخدام الموصى به |
|------|-------------|-------------|
| **نص** | مدخل نصي على سطر واحد | معرف واجهة برمجة تطبيقات خارجية، رمز العلامة التجارية |
| **مربع نص** | نص متعدد الأسطر | ملاحظات معالجة خاصة |
| **عدد** | قيم عددية | الحد الأدنى لعدد الطلبات |
| **عشري** | قيم عشريّة | تجاوز الوزن، بُعد مخصص |
| **نعم / لا** | مفتاح تبديل مربع اختر | هل هو هش، يتطلب توقيع |
| **تاريخ** | مُحدد للتاريخ | تاريخ الإطلاق، تاريخ الانتهاء |
| **تاريخ ووقت** | محدد للتاريخ والوقت | توفر مجدول |
| **عنوان ويب** | عنوان ويب | رابط المورد، عنوان ورقة المواصفات |
| **بريد إلكتروني** | عنوان بريد إلكتروني | عنوان اتصال المصنّع |
| **قائمة منسدلة** | قائمة اختيار واحدة | نوع المادة، الدولة الأصلية |
| **قائمة اختيار متعددة** | قائمة اختيار متعددة | الشهادات، الوسوم |
| **لون** | مُحدد للألوان | لون العلامة التجارية، لون الملصق |

## إدارة الحقول المخصصة

### إنشاء مجموعة حقول

1. افتح **الإعدادات > الحقول المخصصة**
2. اختر علامة التبويب النموذج (المنتجات، التصنيفات، الطلبات، أو ملفات تعريف العملاء)
3. اضغط على **إضافة مجموعة**
4. أدخل **اسم المجموعة** (مثلاً، "التكاملات الخارجية")
5. قم بتفعيل **عرضها على متجر البيع** إذا كان العملاء يجب أن يروا هذه الحقول (اختياري)
6. اضغط على **حفظ المجموعة**

### إضافة حقل إلى مجموعة

1. على بطاقة المجموعة، اضغط على **إضافة حقل**
2. أدخل **اسم الحقل** — يتم إنشاء الاسم المختصر تلقائياً
3. اختر **نوع الحقل**
4. قم بتعيين **نص مساعدة** و**قيمة افتراضية** إذا أردت
5. اضبط خيارات التحقق من الصحة (تختلف حسب نوع الحقل):
   - النص: الحد الأقصى لطول النص، نمط التعبير العادي
   - العدد / العشري: القيمة الدنيا والقصوى
   - القائمة المنسدلة: تحديد قائمة الخيارات
6. اضبط خيارات الحقل:
   - **الحقل المطلوب**: يجب على البائعين ملء هذا الحقل عند الحفظ
   - **عرضه على متجر البيع**: عرض القيمة على الصفحة الموجهة للعملاء
   - **قابل للترجمة**: السماح بترجمة القيمة (النص / مربع النص فقط)
7. اضغط على **حفظ الحقل**

### التعديل والترتيب

- اضغط على أيقونة **القلم** على أي مجموعة أو حقل لتحريره
- اسحب **مقبض التحكم** لترتيب المجموعات أو الحقول داخل المجموعة
- تؤخذ التغييرات في الاعتبار فوراً على جميع النماذج ذات الصلة

### حذف المجموعات والحقول

- اضغط على أيقونة **النفايات** على مجموعة أو حقل لحذفه
- الحذف هو **حذف ناعم** — تظل البيانات محفوظة في قاعدة البيانات ولكنها مخفية عن النماذج
- هذا يحمي البيانات الموجودة من فقدانها عن طريق الخطأ

## استخدام الحقول المخصصة في النماذج

بمجرد تعريف الحقول المخصصة لنمط، تظهر علامة تبويب **الحقول المخصصة** تلقائياً على النموذج المقابل لتحرير السجل.

### المنتجات والتصنيفات

1. افتح أي منتج أو تصنيف للتحرير
2. اضغط على علامة التبويب **الحقول المخصصة**
3. أملأ الحقول كما هو مطلوب
4. اضغط على **حفظ** — يتم تخزين القيم مع السجل

### الطلبات

تظهر قيم الحقول المخصصة للطلبات كقسم **فقط للقراءة** على صفحة تفاصيل الطلب. يتم تعيين الحقول المخصصة للطلبات عادةً عبر واجهة برمجة التطبيقات أو عند الدفع.

### ملفات تعريف العملاء

1. افتح ملف تعريف العميل
2. اضغط على علامة التبويب **الحقول المخصصة**
3. أملأ الحقول واحفظها

## الوصول عبر واجهة برمجة التطبيقات

### قائمة تعريفات الحقول

احصل على جميع تعريفات الحقول المخصصة لنمط:

```
GET /api/custom-fields/definitions/?model=product&app=catalog
```

**الاستجابة:*

```json
[
  {
    "id": 1,
    "name": "External API ID",
    "slug": "external_api_id",
    "field_type": "text",
    "is_required": false,
    "group": { "name": "External Integrations" }
  }
]
```

### قراءة قيم الحقول المخصصة

تُضمن قيم الحقول المخصصة في كائن JSON `custom_fields` في استجابات واجهة برمجة التطبيقات لنمط:

```json
{
  "id": 42,
  "name": "Blue Widget",
  "custom_fields": {
    "external_api_id": "API-12345",
    "is_fragile": true
  }
}
```

### كتابة قيم الحقول المخصصة

أضف `custom_fields` عند إنشاء أو تحديث سجل عبر واجهة برمجة التطبيقات:

```json
{
  "custom_fields": {
    "external_api_id": "API-67890",
    "warehouse_location": "WH-A3"
  }
}
```

تتم مراجعة القيم ضد تعريفات الحقول. تُرجع القيم غير الصالحة خطأ `400` مع التفاصيل.

### الاستعلام باستخدام الحقول المخصصة

تُصنف الحقول المخصصة لاستعلامات قاعدة البيانات السريعة. استخدم فلاتر استعلامات قاعدة البيانات لتصفية السجلات:

```
GET /api/products/?custom_fields__warehouse_location=WH-A3
```

## عرض متجر البيع

### للمطورين المعنيين بالمواضيع

استخدم وسم القوالب `render_custom_fields` لعرض الحقول المخصصة على متجر البيع:

```python
{% load custom_fields_tags %}

{# عرض جميع الحقول المرئية على متجر البيع #}
{% render_custom_fields product %}

{# الحصول على قيمة حقل معين #}
{% get_custom_field product "warehouse_location" as location %}
<p>يتم شحنها من: {{ location }}</p>
```

سيتم عرض الحقول فقط التي تم تمكين خيار **عرضها على متجر البيع** على مستوى المجموعة والحقل.

## أفضل الممارسات

- **استخدم أسماء وصفية** — تظهر أسماء الحقول في النماذج و على متجر البيع
- **حدد نص المساعدة** — ارشد البائعين إلى ما يجب إدخاله في كل حقل
- **نظم الحقول ذات الصلة** — حافظ على تنظيم النماذج وجعلها بديهية
- **استخدم القيم الافتراضية** — ضع قيمًا منطقية لتقليل إدخال البيانات
- **كن انتقائيًا في عرض متجر البيع** — أظهر فقط الحقول ذات المعنى للعملاء
- **استخدم الاسم المختصر في التكاملات** — الاسم المختصر هو معرف ثابت؛ يمكن تغيير أسماء الحقول

## استكشاف الأخطاء وإصلاحها

**لا يظهر علامة تبويب الحقول المخصصة:*
- تأكد من وجود مجموعة حقول نشطة على الأقل لنمط هذا
- تحقق من أن فئة الإداري تحتوي على `CustomFieldsAdminMixin`
- أعد تحميل الصفحة بعد تنظيف ذاكرة التخزين المؤقت

**لا يتم حفظ قيم الحقول:*
- تأكد من ملء الحقول المطلوبة
- تحقق من قواعد التحقق من الصحة (الحد الأدنى / الأقصى، أنماط التعبير العادي، الخيارات المسموح بها)
- تأكد من أن الحقل نشط وغير مُحذف ناعمًا

**تُرجع واجهة برمجة التطبيقات كائن `custom_fields` فارغ:*
- تأكد من أن النموذج يحتوي على `CustomFieldsMixin`
- تحقق من وجود تعريفات الحقول لنوع المحتوى الصحيح
- تأكد من أن المُسلسل يحتوي على `CustomFieldsSerializerMixin`
