title: المنتجات الرقمية
---

تتيح لك المنتجات الرقمية بيع الملفات القابلة للتنزيل وتراخيص البرامج والسلع غير المادية الأخرى. يدعم Spwig المنتجات الرقمية المستقلة، وكذلك المنتجات الهجينة التي تجمع بين التسليم المادي والرقمي.

![مزودو التراخيص](/static/core/admin/img/help/digital-products/license-providers.webp)

## أنواع المنتجات الرقمية

### المنتج الرقمي المستقل

عيّن **نوع المنتج** إلى **منتج رقمي** للعناصر الرقمية بالكامل:
- التطبيقات البرمجية
- الكتب الإلكترونية وملفات PDF
- الموسيقى والملفات الصوتية
- الأعمال الفنية الرقمية والقوالب

### المنتجات الهجينة

يمكن لأي نوع منتج أن يتضمن التسليم الرقمي عن طريق تحديد **هو منتج رقمي** في علامة تبويب المعلومات الأساسية. هذا مفيد لـ:
- **المنتجات الرقمية المتغيرة** — برامج بإصدارات أساسية/احترافية/مؤسسية
- **المنتجات الرقمية القابلة للتخصيص** — أصول رقمية مصممة حسب الطلب
- **حزم مادية + رقمية** — كتاب يتضمن تنزيلاً رقمياً

## إعداد منتج رقمي

### الخطوة 1: إنشاء المنتج

1. انتقل إلى **المنتجات > جميع المنتجات** وانقر على **+ إضافة منتج**
2. عيّن **نوع المنتج** إلى **منتج رقمي** (أو حدد **هو منتج رقمي** على نوع منتج آخر)
3. املأ تفاصيل المنتج (الاسم، الوصف، السعر)
4. احفظ المنتج

### الخطوة 2: إضافة الملفات القابلة للتنزيل

1. انتقل إلى علامة تبويب **المخزون** الخاصة بالمنتج
2. في قسم **الملفات الرقمية**، ارفع الملفات التي سيتلقاها العملاء بعد الشراء
3. لكل ملف، يمكنك تعيين:
   - **اسم الملف** — الاسم المعروض للعملاء
   - **حد التنزيل** — الحد الأقصى لعدد مرات تنزيل الملف (0 = غير محدود)
   - **أيام الصلاحية** — عدد الأيام التي يظل فيها رابط التنزيل نشطاً

### الخطوة 3: تكوين تسليم التراخيص (اختياري)

إذا كان منتجك الرقمي يتطلب مفاتيح ترخيص:

1. انتقل إلى **الإعدادات > إدارة التراخيص**
2. اربط مزود تراخيص (انظر أدناه)
3. في نموذج تحرير المنتج، عيّن مزود التراخيص

## مزودو التراخيص

مزودو التراخيص هم خدمات خارجية تقوم بإنشاء وإدارة مفاتيح تراخيص البرامج تلقائياً عندما يشتري العميل منتجك.

### أنواع المزودين المتاحة

| المزود | الوصف |
|--------|-------|
| **خادم تراخيص Spwig المدمج** | إنشاء مفاتيح ترخيص بسيط مدمج في المنصة |
| **Keygen.sh** | واجهة API متكاملة لإدارة التراخيص |
| **LicenseSpring** | إدارة تراخيص على مستوى المؤسسات |
| **Cryptlex** | ترخيص برمجي مع دعم العمل دون اتصال |
| **API مخصص** | ربط أي نظام تراخيص عبر REST API |

### ربط مزود تراخيص

1. انتقل إلى **الإعدادات > إدارة التراخيص**
2. انقر على **ربط مزود**
3. اتبع معالج الإعداد:
   - **الخطوة 1** — اختر نوع المزود
   - **الخطوة 2** — قم بتكوين الإعدادات العامة
   - **الخطوة 3** — أدخل بيانات اعتماد API
4. اختبر الاتصال للتحقق من عمله
5. احفظ التكوين

### بطاقة المزود

يعرض كل مزود متصل:
- **شارات الحالة** — نشط/غير نشط وحالة الاتصال
- **نقطة نهاية API** — عنوان URL للخادم المكوّن
- **إمكانيات المزامنة** — دعم مزامنة الطلبات والتفعيل وإلغاء التفعيل
- **أزرار الإجراءات** — تكوين، اختبار، ومزامنة الآن

### إمكانيات المزامنة

يمكن لمزودي التراخيص المزامنة على ثلاثة أحداث:

- **الطلب** — إنشاء مفتاح ترخيص تلقائياً عندما يُكمل العميل عملية الشراء
- **التفعيل** — تتبع وقت تفعيل العميل لترخيصه
- **إلغاء التفعيل** — التعامل مع إلغاء تفعيل الترخيص للاسترداد أو النقل

## تجربة العميل

### بعد الشراء

عندما يشتري عميل منتجاً رقمياً:

1. **تأكيد الطلب** — يُظهر أن التسليم الرقمي مشمول
2. **التسليم عبر البريد الإلكتروني** — يتم إرسال روابط التنزيل و/أو مفاتيح الترخيص تلقائياً
3. **صفحة الحساب** — يمكن للعملاء الوصول إلى تنزيلاتهم من لوحة معلومات حسابهم
4. **صفحة التنزيل** — روابط تنزيل آمنة ومحدودة الوقت

### أمان التنزيل

تنزيلات الملفات الرقمية محمية بـ:
- رموز تنزيل فريدة ومحدودة الوقت
- حدود اختيارية لعدد التنزيلات
- تواريخ انتهاء صلاحية تصبح بعدها الروابط غير نشطة
- متطلب تسجيل الدخول (للعملاء المسجلين)

## نصائح

- عيّن حدود تنزيل معقولة (3-5 تنزيلات) لمنع إساءة الاستخدام مع السماح بإعادة التنزيل.
- استخدم أيام صلاحية تتوافق مع فترة الدعم الخاصة بك (مثل 365 يوماً لسنة من الوصول).
- اختبر تدفق الشراء الكامل بطلب تجريبي لضمان تسليم روابط التنزيل ومفاتيح الترخيص بشكل صحيح.
- بالنسبة لمنتجات البرامج، اربط مزود تراخيص لأتمتة إنشاء المفاتيح بدلاً من إدارتها يدوياً.
- استخدم ميزة المنتجات الهجينة عند بيع سلع مادية تتضمن إضافات رقمية (مثل كتاب مطبوع + PDF).
