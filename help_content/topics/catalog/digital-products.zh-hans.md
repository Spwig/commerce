---
title: 数字产品
---

数字产品让您可以销售可下载文件、软件许可证和其他非实物商品。Spwig 支持独立数字产品，以及结合实物和数字交付的混合产品。

![许可证提供商](/static/core/admin/img/help/digital-products/license-providers.webp)

## 数字产品类型

### 独立数字产品

将**产品类型**设置为**数字产品**，用于纯数字商品：
- 软件应用程序
- 电子书和 PDF
- 音乐和音频文件
- 数字艺术作品和模板

### 混合产品

任何产品类型都可以通过在基本信息标签页中勾选**是数字产品**来包含数字交付。这适用于：
- **可变数字产品** — 具有基础版/专业版/企业版的软件
- **可定制数字产品** — 定制设计的数字资产
- **实物 + 数字捆绑** — 包含数字下载的书籍

## 设置数字产品

### 步骤 1：创建产品

1. 导航到 **产品 > 所有产品** 并点击 **+ 添加产品**
2. 将**产品类型**设置为**数字产品**（或在其他产品类型上勾选**是数字产品**）
3. 填写产品详情（名称、描述、价格）
4. 保存产品

### 步骤 2：添加可下载文件

1. 转到产品的**库存**标签页
2. 在**数字文件**部分，上传客户购买后将收到的文件
3. 对于每个文件，您可以设置：
   - **文件名** — 向客户显示的名称
   - **下载限制** — 文件可下载的最大次数（0 = 无限制）
   - **有效天数** — 下载链接保持有效的天数

### 步骤 3：配置许可证交付（可选）

如果您的数字产品需要许可证密钥：

1. 导航到 **设置 > 许可证管理**
2. 连接许可证提供商（见下文）
3. 在产品编辑表单中，分配许可证提供商

## 许可证提供商

许可证提供商是外部服务，在客户购买您的产品时自动生成和管理软件许可证密钥。

### 可用的提供商类型

| 提供商 | 描述 |
|--------|------|
| **Spwig 内置许可证服务器** | 平台内置的简单许可证密钥生成 |
| **Keygen.sh** | 功能完善的许可证管理 API |
| **LicenseSpring** | 企业级许可证管理 |
| **Cryptlex** | 支持离线的软件许可 |
| **自定义 API** | 通过 REST API 连接任何许可证系统 |

### 连接许可证提供商

1. 导航到 **设置 > 许可证管理**
2. 点击**连接提供商**
3. 按照设置向导操作：
   - **步骤 1** — 选择提供商类型
   - **步骤 2** — 配置常规设置
   - **步骤 3** — 输入 API 凭据
4. 测试连接以验证其是否正常工作
5. 保存配置

### 提供商卡片

每个已连接的提供商显示：
- **状态徽章** — 活跃/非活跃和连接状态
- **API 端点** — 已配置的服务器 URL
- **同步功能** — 订单、激活和停用同步支持
- **操作按钮** — 配置、测试和立即同步

### 同步功能

许可证提供商可以在三个事件上同步：

- **订单** — 客户完成购买时自动生成许可证密钥
- **激活** — 跟踪客户激活许可证的时间
- **停用** — 处理退款或转让时的许可证停用

## 客户体验

### 购买后

当客户购买数字产品时：

1. **订单确认** — 显示包含数字交付
2. **邮件交付** — 下载链接和/或许可证密钥自动发送
3. **账户页面** — 客户可以从其账户仪表板访问下载
4. **下载页面** — 安全的、有时限的下载链接

### 下载安全

数字文件下载受以下保护：
- 唯一的、有时限的下载令牌
- 可选的下载次数限制
- 过期日期，过期后链接失效
- 登录要求（针对注册客户）

## 提示

- 设置合理的下载限制（3-5 次下载）以防止滥用，同时允许重新下载。
- 使用与您的支持期限匹配的有效天数（例如 365 天代表一年的访问权限）。
- 使用测试订单测试完整的购买流程，确保下载链接和许可证密钥正确交付。
- 对于软件产品，连接许可证提供商以自动化密钥生成，而不是手动管理密钥。
- 在销售包含数字附加内容的实物商品时（例如印刷书籍 + PDF），使用混合产品功能。
