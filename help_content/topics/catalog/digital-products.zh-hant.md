---
title: 數位產品
---

數位產品讓您可以銷售可下載檔案、軟體授權和其他非實體商品。Spwig 支援獨立數位產品，以及結合實體和數位交付的混合產品。

![授權提供商](/static/core/admin/img/help/digital-products/license-providers.webp)

## 數位產品類型

### 獨立數位產品

將**產品類型**設定為**數位產品**，用於純數位商品：
- 軟體應用程式
- 電子書和 PDF
- 音樂和音訊檔案
- 數位藝術作品和範本

### 混合產品

任何產品類型都可以透過在基本資訊標籤頁中勾選**是數位產品**來包含數位交付。這適用於：
- **可變數位產品** — 具有基礎版/專業版/企業版的軟體
- **可自訂數位產品** — 客製化設計的數位資產
- **實體 + 數位組合** — 包含數位下載的書籍

## 設定數位產品

### 步驟 1：建立產品

1. 導航到 **產品 > 所有產品** 並點擊 **+ 新增產品**
2. 將**產品類型**設定為**數位產品**（或在其他產品類型上勾選**是數位產品**）
3. 填寫產品詳情（名稱、描述、價格）
4. 儲存產品

### 步驟 2：新增可下載檔案

1. 前往產品的**庫存**標籤頁
2. 在**數位檔案**區段，上傳客戶購買後將收到的檔案
3. 對於每個檔案，您可以設定：
   - **檔案名稱** — 向客戶顯示的名稱
   - **下載限制** — 檔案可下載的最大次數（0 = 無限制）
   - **有效天數** — 下載連結保持有效的天數

### 步驟 3：設定授權交付（選擇性）

如果您的數位產品需要授權金鑰：

1. 導航到 **設定 > 授權管理**
2. 連接授權提供商（見下文）
3. 在產品編輯表單中，指派授權提供商

## 授權提供商

授權提供商是外部服務，在客戶購買您的產品時自動產生和管理軟體授權金鑰。

### 可用的提供商類型

| 提供商 | 描述 |
|--------|------|
| **Spwig 內建授權伺服器** | 平台內建的簡單授權金鑰產生 |
| **Keygen.sh** | 功能完善的授權管理 API |
| **LicenseSpring** | 企業級授權管理 |
| **Cryptlex** | 支援離線的軟體授權 |
| **自訂 API** | 透過 REST API 連接任何授權系統 |

### 連接授權提供商

1. 導航到 **設定 > 授權管理**
2. 點擊**連接提供商**
3. 按照設定精靈操作：
   - **步驟 1** — 選擇提供商類型
   - **步驟 2** — 設定一般選項
   - **步驟 3** — 輸入 API 憑證
4. 測試連線以驗證是否正常運作
5. 儲存設定

### 提供商卡片

每個已連接的提供商顯示：
- **狀態徽章** — 啟用/停用和連線狀態
- **API 端點** — 已設定的伺服器 URL
- **同步功能** — 訂單、啟用和停用同步支援
- **操作按鈕** — 設定、測試和立即同步

### 同步功能

授權提供商可以在三個事件上同步：

- **訂單** — 客戶完成購買時自動產生授權金鑰
- **啟用** — 追蹤客戶啟用授權的時間
- **停用** — 處理退款或轉讓時的授權停用

## 客戶體驗

### 購買後

當客戶購買數位產品時：

1. **訂單確認** — 顯示包含數位交付
2. **電子郵件交付** — 下載連結和/或授權金鑰自動傳送
3. **帳戶頁面** — 客戶可以從其帳戶儀表板存取下載
4. **下載頁面** — 安全的、有時限的下載連結

### 下載安全

數位檔案下載受以下保護：
- 唯一的、有時限的下載權杖
- 選擇性的下載次數限制
- 到期日期，過期後連結失效
- 登入要求（針對已註冊客戶）

## 提示

- 設定合理的下載限制（3-5 次下載）以防止濫用，同時允許重新下載。
- 使用與您的支援期限相符的有效天數（例如 365 天代表一年的存取權限）。
- 使用測試訂單測試完整的購買流程，確保下載連結和授權金鑰正確交付。
- 對於軟體產品，連接授權提供商以自動化金鑰產生，而不是手動管理金鑰。
- 在銷售包含數位附加內容的實體商品時（例如印刷書籍 + PDF），使用混合產品功能。
