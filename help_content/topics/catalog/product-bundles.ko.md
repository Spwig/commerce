---
title: 제품 번들
---

제품 번들은 제품의 사전 조립된 패키지를 묶음 가격으로 판매할 수 있도록 해줍니다. 이는 선물 세트, 스타터 키트, 또는 할인을 통해 함께 제공하고 싶은 제품의 조합에 이상적입니다.

![번들 구성 요소 관리자](/static/core/admin/img/help/product-bundles/bundle-components.webp)

## 가격 전략

번들 가격이 어떻게 계산되는지 선택하세요:

| 전략 | 설명 |
|----------|-------------|
| **고정 가격** | 구성 요소 가격과 관계없이 전체 번들의 가격을 하나로 설정합니다. |
| **비율 할인** | 구성 요소 가격의 합계에 비율을 적용하여 가격을 자동으로 계산합니다. |
| **구성 요소 합계** | 번들 가격은 모든 구성 요소 가격의 총합과 같습니다(할인 없이 그룹화된 표시에 유용합니다). |

## 번들 생성

### 단계 1: 제품 생성

1. **제품 > 모든 제품**으로 이동하고 **+ 제품 추가**를 클릭하세요
2. **제품 유형**을 **제품 번들**로 설정하세요
3. 번들의 이름, 설명, 이미지를 입력하세요
4. 제품을 저장하세요

### 단계 2: 구성 요소 추가

**번들 항목** 탭으로 전환하여 번들에 제품을 추가하세요:

1. **+ 구성 요소 추가**를 클릭하세요
2. 드롭다운에서 제품을 검색하고 선택하세요
3. 각 구성 요소의 **수량**을 설정하세요 (예: 스킨케어 세트에 2개의 피부 마스크)
4. **정렬 순서**를 설정하여 표시 순서를 조정하세요
5. 선택적으로 구성 요소를 **선택 사항**으로 표시하세요 (고객이 이를 제외할 수 있음)
6. 구성 요소가 가변 제품인 경우 다음 중 하나를 선택하세요:
   - **고정 변형** — 모든 고객이 동일한 변형을 받습니다
   - **변형 선택 허용** — 고객이 체크아웃 시 선호하는 변형을 선택할 수 있습니다

하단의 요약은 **총 구성 요소 수**와 **번들 가치**(구성 요소 가격의 합계)를 보여줍니다.

### 단계 3: 가격 설정

**가격** 탭으로 전환하세요:

1. **번들 가격 전략**을 선택하세요
2. **고정 가격**인 경우 번들 가격을 직접 입력하세요
3. **비율 할인**인 경우 할인 비율을 설정하세요 (예: 15% 할인)
4. **구성 요소 합계**인 경우 가격이 자동으로 계산됩니다

## 번들로 묶을 수 있는 항목

| 제품 유형 | 구성 요소로 사용 가능? |
|-------------|-------------------|
| 단순 제품 | 예 |
| 가변 제품 | 예 (고정 변형 또는 고객 선택)
| 디지털 제품 | 예 |
| 커스터마이징 제품 | 아니오 |
| 구성 가능한 제품 | 아니오 |
| 제품 번들 | 아니오 (번들은 중첩될 수 없음)
| 선물 카드 | 아니오 |

## 재고 관리

번들의 재고는 구성 요소를 통해 관리됩니다:

- **모든 구성 요소가 재고에 있어야** 번들이 구매 가능합니다
- 번들이 주문되면 각 구성 요소 제품의 재고가 개별적으로 차감됩니다
- **어떤 구성 요소라도 재고가 없어지면** 번들이 사용 불가능해집니다
- 구성 요소의 재고 수준은 체크아웃 시 실시간으로 확인됩니다

## 선택 사항 구성 요소

구성 요소를 **선택 사항**으로 표시하여 고객이 번들을 커스터마이징하도록 허용하세요:

- 선택 사항 구성 요소는 기본적으로 포함되어 있지만 고객이 제거할 수 있습니다
- 선택 사항 구성 요소가 제외되면 번들 가격이 자동으로 조정됩니다
- 최소한 하나의 구성 요소는 **필수**(필수 항목)여야 합니다

## 고객 경험

고객이 스토어프론트에서 번들을 보고 있을 때:

1. **구성 요소 목록** — 모든 포함된 제품이 이미지와 수량과 함께 표시됩니다
2. **번들 할인** — 개별 구매에 비해 할인 금액이 표시됩니다
3. **변형 선택** — 변형 선택이 활성화된 구성 요소에 대해 고객은 선호하는 옵션을 선택할 수 있습니다
4. **선택 사항 항목** — 고객은 선택 사항 구성 요소를 켜거나 끌 수 있습니다
5. **한 번에 장바구니 추가** — 전체 번들이 하나의 항목으로 추가됩니다

## 팁

- 구성 요소 가격이 변경될 때 자동으로 조정되는 **비율 할인** 전략을 사용하세요.
- 제품 설명에서 **할인 금액**을 두드러지게 표시하여 번들 구매를 장려하세요.
- 고객 경험을 위해 번들을 3~5개의 구성 요소로 유지하세요. 너무 많은 항목은 고객에게 혼란스러울 수 있습니다.
- 선택 사항 구성 요소를 사용하여 동일한 번들의 **기본** 및 **프리미엄** 버전을 제공하세요.
- 모든 구성 제품이 여전히 활성화되고 재고가 있는지 정기적으로 확인하세요.
