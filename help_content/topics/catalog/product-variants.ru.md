---
title: Варианты товаров
---

Варианты товаров позволяют предлагать один товар в нескольких вариациях — например, разных размеров, цветов или материалов — каждый со своим SKU, ценой и уровнем запасов. Перейдите к любому **Вариативному товару** и нажмите вкладку **Вариации**.

![Варианты товаров](/static/core/admin/img/help/product-variants/product-variants.webp)

## Понимание вариантов

**Вариативный товар** — это тип товара, поддерживающий несколько вариаций. Например, футболка может быть доступна в:
- **Цветах**: Синий, Красный, Зелёный
- **Размерах**: S, M, L, XL

Каждая комбинация (например, «Синий / L») становится отдельным вариантом со своим запасом и ценой.

## Настройка вариативного товара

### Шаг 1: Установка типа товара

1. Откройте форму редактирования товара (или создайте новый товар)
2. На вкладке **Основная информация** установите **Тип товара** как **Вариативный товар**
3. Сохраните товар

### Шаг 2: Определение атрибутов

Атрибуты — это опции, которые отличают ваши варианты (например, Размер, Цвет).

1. Перейдите на вкладку **Вариации**
2. В разделе **Атрибуты товара** нажмите **+ Добавить атрибут**, чтобы назначить существующий атрибут, или **Создать новый**, чтобы определить новый
3. Для каждого атрибута укажите доступные значения (например, Маленький, Средний, Большой)

### Шаг 3: Создание вариантов

1. В разделе **Варианты товара** нажмите **+ Добавить новый вариант**
2. Настройте каждый вариант:
   - **Название** — Описательная метка (например, «Синий», «Большой / Красный»)
   - **SKU** — Уникальный код складского учёта
   - **Цена** — Цена, специфичная для варианта (может отличаться от базового товара)
   - **Запас** — Текущий уровень складских запасов
3. Повторите для каждого необходимого варианта

## Управление вариантами

### Детали варианта

Каждая карточка варианта отображает:
- **Название** и **SKU** — Идентификационная информация
- **Цена** — Текущая цена продажи
- **Уровень запасов** — Доступное количество с индикатором статуса (В наличии / Мало на складе / Нет в наличии)

Нажмите на карточку варианта, чтобы развернуть и отредактировать все его данные.

### Настройки, специфичные для варианта

Каждый вариант может иметь собственные настройки:

| Настройка | Описание |
|-----------|----------|
| **Цена** | Переопределение цены базового товара |
| **Цена сравнения** | Показать цену со скидкой с зачёркиванием |
| **SKU** | Уникальный идентификатор для инвентаризации |
| **Уровень запасов** | Независимое отслеживание запасов |
| **Вес** | Для расчёта стоимости доставки |
| **Изображение** | Изображение товара для конкретного варианта |

### Редактирование варианта

1. Нажмите на **значок редактирования** на карточке варианта
2. Измените нужные поля
3. Нажмите **Сохранить** для обновления

### Удаление варианта

1. Нажмите на **значок удаления** на карточке варианта
2. Подтвердите удаление

**Примечание:** Удаление варианта удаляет его запись о запасах. Это действие нельзя отменить.

## Атрибуты

### Что такое атрибуты?

Атрибуты — это многоразовые определения опций. После создания атрибута, например «Размер» со значениями «S, M, L, XL», вы можете назначить его любому вариативному товару.

### Создание атрибутов

1. На вкладке «Вариации» нажмите **Создать новый** в разделе «Атрибуты товара»
2. Введите название атрибута (например, «Цвет»)
3. Добавьте значения (например, «Красный», «Синий», «Зелёный»)
4. Сохраните атрибут

### Назначение атрибутов

Атрибуты можно назначать нескольким товарам. Один и тот же атрибут «Размер» можно использовать для футболок, брюк и обуви.

## Отображение в витрине магазина

В витрине магазина вариативные товары отображают:
- Селекторы опций (выпадающие списки или образцы) для каждого атрибута
- Автоматическое обновление цены при выборе варианта
- Наличие на складе для каждого варианта
- Изображения, специфичные для варианта

## Советы

- Используйте единообразные названия атрибутов для всех товаров, чтобы обеспечить единый опыт покупок.
- Настройте все атрибуты до создания вариантов, чтобы упростить процесс.
- Загружайте изображения для каждого варианта, чтобы покупатели видели именно то, что заказывают.
- Используйте систематические SKU (например, «FUTBOLKA-SINIY-L») для удобного управления запасами.
- Используйте цену сравнения на вариантах для проведения акций по конкретным размерам или цветам.
