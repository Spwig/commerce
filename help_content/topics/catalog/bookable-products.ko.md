---
title: 예약 가능한 상품
---

예약 가능한 상품은 고객이 구매 시 특정 날짜와 시간을 예약할 수 있도록 합니다. 이는 예약, 대여, 수업, 이벤트, 숙박 예약 등 모든 항목을 Spwig 관리자에서 직접 관리할 수 있도록 지원합니다.

## 예약 유형

| 유형 | 최적의 사용처 |
|------|----------|
| **예약** | 서비스: 상담, 헤어컷, 개인 트레이닝 |
| **대여** | 장비 대여, 차량 대여, 방 대여 |
| **수업 / 워크숍** | 정원이 정해진 그룹 세션 |
| **숙박** | 체크인/체크아웃 시간이 있는 멀티 나이트 숙박 |
| **이벤트** | 티켓이 필요한 일회성 또는 반복 이벤트 |

## 예약 가능한 상품 설정

### 단계 1: 상품 생성

1. **상품 > 모든 상품**으로 이동하고 **+ 상품 추가**를 클릭합니다.
2. **상품 유형**을 **예약 상품**으로 설정합니다.
3. 표준 상품 필드(이름, 설명, 가격)를 완료합니다.
4. 상품을 저장합니다.

### 단계 2: 예약 설정 구성

저장한 후, 상품 편집 폼에 **예약 구성** 섹션이 나타납니다. 예약 설정을 입력합니다:

#### 예약 유형 및 기간

- **예약 유형** — 서비스에 가장 적합한 유형(예약, 대여, 수업 등)을 선택합니다.
- **기간 유형** — 고정 기간의 세션을 위해 **고정 기간**을 선택하거나, 고객이 필요한 시간을 선택할 수 있도록 **고객이 기간 선택**을 선택합니다.
- **기간** 및 **기간 단위** — 기간을 설정합니다(예: `60` 분, `1` 시간, `2` 일).
- **최소/최대 기간** — 고객이 기간을 선택할 수 있는 경우 허용 범위를 설정합니다.

#### 버퍼 시간

예약 사이에 자동으로 추가되어 준비 또는 청소 시간을 제공하는 버퍼 시간:
- **예약 전** — 예약 시작 전에 예약된 분량
- **예약 후** — 예약 종료 후에 예약된 분량

예를 들어, 60분 마사지 예약에 예약 후 15분 버퍼가 있으면 다음 고객을 위한 15분의 준비 시간이 제공됩니다.

#### 사전 예약 윈도우


- **최소 예약 사전 통보** — 고객이 예약해야 하는 최소 기간 (예: `24시간`으로 설정하면 당일 예약은 허용되지 않음)
- **최대 예약 기간** — 고객이 미래로 예약할 수 있는 최대 기간 (예: `365일`)

#### 용량

- **슬롯당 최대 예약 수** — 클래스 및 이벤트에 대해 동일한 시간 슬롯에 예약할 수 있는 고객 수를 설정합니다. 프라이빗 약속에는 `1`로 설정합니다.

#### 확인

- **수동 확인 필요** — 체크되면 예약이 자동으로 확인되지 않습니다. 예약 목록에서 각 예약을 수동으로 승인해야 합니다. 예약을 확인하기 전에 고객을 검토하고 싶을 때 유용합니다.

#### 취소 정책

- **예약 취소 허용** — 고객이 예약을 취소할 수 있는지 여부
- **취소 마감 시간** — 고객이 예약을 취소할 수 있는 시간/일 수 (예: `24시간`)

#### 달력 표시

제품 페이지에서 고객이 날짜와 시간을 선택하는 방식:

| 표시 모드 | 최적 사용 사례 |
|-------------|----------|
| **달력 보기** | 일반 사용 — 전체 월간 달력 |
| **날짜 선택기** | 간단한 단일 날짜 선택 |
| **가능한 날짜 드롭다운** | 제한된 예약 슬롯이 있는 제품 |
| **날짜 범위 선택기** | 숙소 및 멀티데이 대여 |

#### 예치금

체크아웃 시 전체 결제 대신 예치금을 요구하려면:
1. **예치금 활성화**를 체크합니다
2. **예치금 유형**을 **고정 금액** 또는 **총 금액의 비율**로 설정합니다
3. **예치금 금액**을 입력합니다 (예: `50`은 $50, `25`은 25%)

#### 숙소 관련 설정

숙소 예약의 경우, 추가 필드가 나타납니다:
- **체크인 시간** 및 **체크아웃 시간** — 해당 숙소의 표준 시간
- **표준 수용 인원** — 기본 요금에 포함된 기본 고객 수

### 단계 3: 예약 자원 추가 (선택 사항)

자원은 예약에 할당되는 물리적 항목 또는 직원 — 예를 들어, "1번 방", "A 코트", 또는 "샘 강사"입니다.

1.

제품 편집 폼에서 **예약 자원** 섹션으로 이동합니다
2.

**자원 추가**를 클릭합니다
3.


자원에 **이름**을 지정하고, 동시에 처리할 수 있는 **예약 수량**(Capacity)을 설정합니다.
4.

선택적으로 자원 이미지를 추가할 수 있습니다.

자원은 시간대별로만 아니라 개별 자산이나 직원별로도 예약 가능 여부를 추적할 수 있습니다.

### 단계 4: 가용성 규칙 설정

가용성 규칙은 예약이 언제 가능하게 되는지를 정의합니다:

1. 제품의 **가용성**(Availability) 섹션에서 **가용성 규칙 추가**(Add Availability Rule)를 클릭합니다.
2. 이 규칙이 적용될 **자원**(Resource)을 선택합니다.
3. 예약이 가능한 **요일**(Days of Week)을 설정합니다.
4. 가용 시간대의 **시작 시간**(Start Time)과 **종료 시간**(End Time)을 설정합니다.
5. 계절별 가용성을 위해 선택적으로 **유효 시작일**(Valid From) / **유효 종료일**(Valid Until)을 설정합니다.
6. 저장합니다.

## 예약 보기 및 관리

### 예약 목록

**카탈로그 > 예약**(Catalog > Bookings)으로 이동하여 모든 예약을 볼 수 있습니다. 다음으로 필터링할 수 있습니다:
- 상태 (예약 대기, 확정, 취소, 완료, 미참석)
- 제품
- 날짜 범위

### 예약 상태

| 상태 | 의미 |
|------|------|
| **예약 대기**(Pending Confirmation) | 수동 승인 대기 (승인 필요 시) |
| **확정**(Confirmed) | 예약이 확정되어 활성화됨 |
| **취소**(Cancelled) | 고객 또는 당신에 의해 예약이 취소됨 |
| **완료**(Completed) | 예약 날짜가 지나고 예약이 완료됨 |
| **미참석**(No Show) | 고객이 참석하지 않음 |

### 예약 대기 상태의 예약 확정

1. **카탈로그 > 예약**(Catalog > Bookings)에서 예약을 엽니다.
2. **상태**(Status)를 **확정**(Confirmed)으로 변경합니다.
3. 저장 — 고객은 자동으로 확인 이메일을 받습니다.

### 예약 취소

1. 예약을 엽니다.
2. **상태**(Status)를 **취소**(Cancelled)로 변경합니다.
3. **취소 사유**(Cancellation Reason)를 입력합니다 (고객 이메일에 표시됨).
4. 저장합니다.

## 대기 목록 관리

시간대가 완전히 예약이 채워졌을 때 고객은 대기 목록에 추가할 수 있습니다. Spwig는 취소로 인해 예약이 생기면 대기 목록에 있는 고객에게 자동으로 알립니다.

### 대기 목록 보기

**카탈로그 > 예약 대기 목록**(Catalog > Booking Waitlist)으로 이동하여 모든 대기 목록 항목을 볼 수 있습니다.

각 항목은 다음과 같이 표시됩니다:
- 고객 이름 및 이메일
- 제품 및 원하는 날짜
- 상태: **대기 중**, **알림됨**, **예약으로 전환됨**, 또는 **만료됨**

### 대기 목록 상태

| 상태 | 의미 |
|--------|---------|
| **대기 중** | 고객이 대기 중이며 슬롯이 아직 확보되지 않았음 |
| **알림됨** | 고객에게 슬롯이 확보된 것을 이메일로 알림이 전송됨 |
| **예약으로 전환됨** | 고객이 슬롯을 확보하고 예약을 완료함 |
| **만료됨** | 원하는 날짜가 지나면서 슬롯이 확보되지 못함 |

### 대기 목록 고객에게 수동으로 알림 보내기

자동 알림 전에 특정 대기 목록 고객에게 연락하고 싶다면:
1. 대기 목록 항목을 열어주세요
2. 그들의 이메일 주소를 복사하여 직접 연락하세요
3. 그들이 예약을 완료하면 대기 목록 항목 상태가 **예약으로 전환됨**으로 업데이트됩니다

## 팁

- 고가의 예약(예: 사진 촬영, 사적인 이벤트)에 대해 수동 확인을 활성화하여 확보 가능한 슬롯과 요구 사항을 확인하고 확정 전에 일치시킬 수 있습니다.
- 초기에는 여유 있게 버퍼 시간을 설정하세요 — 실제 작업 흐름에 대한 이해가 깊어지면 언제든지 줄일 수 있습니다.
- 그룹 수업의 경우, **슬롯당 최대 예약 수**를 수업 용량으로 설정하고 대기 목록을 활성화하여 인기 있는 세션에 자동으로 대기 목록이 생성되도록 설정하세요.
- 숙소 제품의 경우 날짜 범위 선택기 표시 모드를 사용하세요 — 고객들은 도착 및 퇴소 날짜를 함께 선택하는 것을 기대합니다.
- 준비 시간이 필요한 경우(예: 맞춤형 케이터링 주문의 경우 최소 48시간), 최소 사전 통보 시간을 설정하여 마지막 시각 예약을 방지하세요.
- 바쁜 시즌 동안 정기적으로 대기 목록을 검토하세요 — 대기 목록 고객에게 수동으로 연락하는 것이 자동 알림보다 취소를 더 빠르게 채울 수 있습니다.
