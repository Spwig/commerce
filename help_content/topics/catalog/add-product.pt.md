---
title: Adicionar um Produto
---

Este guia orienta você na criação de um novo produto na sua loja. Os produtos são organizados em várias abas — Informações Básicas, Mídia, Preços, Estoque e SEO — para que você possa preencher tudo de uma vez ou voltar para completar as seções depois.

## Primeiros Passos

Na barra lateral, navegue até **Produtos > Todos os Produtos** para ver o catálogo de produtos. Clique no botão **+ Adicionar Produto** no canto superior direito para abrir o formulário de criação de produto.

![Página de lista de produtos](/static/core/admin/img/help/add-product/product-list-page.webp)

## Aba Informações Básicas

A aba **Informações Básicas** é onde você define os detalhes principais do seu produto.

![Formulário de adicionar produto](/static/core/admin/img/help/add-product/add-product-form.webp)

### Campos Obrigatórios

- **Nome** — O nome do produto exibido para os clientes. Clique no ícone do globo para adicionar traduções em outros idiomas.
- **Slug** — Versão do nome adaptada para URL (gerada automaticamente). Desative o "Auto" para personalizá-lo.
- **SKU** — Seu código interno de unidade de manutenção de estoque.
- **Tipo de Produto** — Escolha entre: Simples, Variável, Digital, Pacote, Cartão Presente, Personalizável ou Configurável.
- **Status** — Defina como Rascunho enquanto trabalha e altere para Publicado quando estiver pronto.

### Campos Opcionais

- **Categoria** — Atribua o produto a uma categoria para organização e navegação na vitrine.
- **Marca** — Associe a uma marca, se aplicável.
- **É Destaque** — Marque para destacar este produto na sua vitrine.
- **É Produto Digital** — Marque se este produto inclui downloads digitais (arquivos, licenças).
- **Ocultar da Vitrine** — Oculta o produto das listagens do catálogo, mantendo-o disponível como opção de configurador ou componente de pacote.

### Descrições do Produto

- **Descrição Curta** — Aparece nas listagens e cartões de produtos. Mantenha-a breve e atrativa.
- **Descrição Completa** — Descrição detalhada do produto exibida na página de detalhes do produto. Use o editor de texto rico para adicionar formatação, imagens, vídeos e tabelas.

Ambos os campos de descrição suportam o recurso de tradução — clique no ícone do globo para fornecer conteúdo em outros idiomas.

## Aba Mídia

A aba **Mídia** permite gerenciar as imagens do produto usando a Biblioteca de Mídia integrada.

![Aba Mídia](/static/core/admin/img/help/add-product/media-tab.webp)

1. Clique em **+ Adicionar Imagens da Biblioteca de Mídia** para abrir o seletor de mídia.
2. Selecione imagens existentes ou faça upload de novas diretamente.
3. Arraste as imagens para reordená-las — a **primeira imagem** se torna a imagem principal do produto exibida nas listagens e cartões.
4. Escolha um **Tipo de Galeria** para controlar como as imagens são exibidas na página do produto: Galeria Padrão, Carrossel, Layout em Grade, Galeria com Zoom ou Visualização 360°.

## Aba Preços

Defina os preços do seu produto e configure promoções.

![Aba Preços](/static/core/admin/img/help/add-product/pricing-tab.webp)

### Preço Regular

- **Preço Regular** — O preço de varejo padrão que os clientes verão.
- **Moeda** — Selecione a moeda (a moeda padrão da sua loja é pré-selecionada).
- **Custo** — Seu custo de mercadoria, usado para cálculos de lucro. Este valor nunca é exibido para os clientes.

### Configurações de Promoção

Configure descontos temporários:

- **Tipo de Promoção** — Escolha entre: Sem Promoção, Preço Fixo de Promoção, Desconto por Valor ou Desconto por Porcentagem.
- **Valor da Promoção** — O valor ou porcentagem do desconto.
- **Datas de Início/Término** — Agende quando a promoção é ativada e expira. Deixe em branco para início imediato ou sem data de término.

## Aba Estoque

Gerencie os níveis de estoque e os atributos físicos do produto.

![Aba Estoque](/static/core/admin/img/help/add-product/inventory-tab.webp)

### Gestão de Estoque

- **Rastrear Estoque** — Ative para acompanhar as quantidades em estoque (ativado por padrão).
- **Limite de Estoque Baixo** — Receba alertas quando o estoque cair abaixo desse número (padrão: 5).
- **Quantidade em Estoque** — Total de unidades disponíveis.
- **Permitir Encomendas** — Ative para aceitar pedidos mesmo quando o produto estiver fora de estoque.

### Atributos Físicos

Insira o peso do produto (kg) e as dimensões (comprimento, largura, altura em cm) para cálculos de frete precisos.

### Identificadores do Produto

Códigos padrão de produto para listagens em marketplaces e sistemas de inventário:

- **GTIN** — Número Global do Item Comercial
- **EAN** — Número de Artigo Europeu
- **UPC** — Código Universal de Produto (EUA)
- **ISBN** — Para livros
- **ASIN** — Identificador Amazon
- **MPN** — Número de Peça do Fabricante

### Envio Internacional / Alfândega

Necessário para envios internacionais:

- **Código HS** — Código de classificação do Sistema Harmonizado
- **País de Origem** — Onde o produto é fabricado
- **Preço Unitário Alfandegário** — Valor declarado por unidade para a alfândega

## Aba SEO

Otimize a visibilidade do seu produto nos mecanismos de busca.

![Aba SEO](/static/core/admin/img/help/add-product/seo-tab.webp)

- **Título Meta** — O título exibido nos resultados dos mecanismos de busca. Clique no ícone do globo para traduzir.
- **Descrição Meta** — Uma breve descrição para os resultados de busca (máximo de 160 caracteres). Clique no ícone do globo para traduzir.
- **Gerar SEO Automaticamente** — Marque para gerar automaticamente o conteúdo SEO quando o produto for salvo.

Uma **Pré-visualização do Resultado de Busca** em tempo real mostra exatamente como seu produto aparecerá nos resultados de busca do Google.

## Salvando Seu Produto

Quando estiver pronto, use os botões de salvar no canto superior direito:

- **Salvar** (marca de verificação) — Salva e permanece na página do produto.
- **Salvar e continuar editando** — Salva e permanece no formulário para continuar trabalhando.

Seu produto ficará visível na vitrine assim que o status for definido como **Publicado**.

## Dicas

- Comece com o status **Rascunho** para que você possa aperfeiçoar o produto antes que os clientes o vejam.
- Faça upload de várias imagens — produtos com diversas fotos convertem melhor.
- Preencha os campos de **SEO** para melhorar a descoberta nos mecanismos de busca.
- Use **Categorias** e **Marcas** para ajudar os clientes a navegar pelo seu catálogo.
- Para produtos variáveis (por exemplo, diferentes tamanhos ou cores), escolha o tipo **Produto Variável** e adicione as variantes após salvar.
