---
title: Tags de Produtos
---

Tags de produtos são rótulos curtos e livres que você pode anexar a produtos para organização flexível e filtragem. As tags são leves em comparação com categorias — elas não têm hierarquia ou páginas dedicadas, mas são rápidas de criar e fáceis de aplicar a muitos produtos de uma só vez.

Usos comuns para tags:
- Agrupar produtos por tema (`summer`, `gift-idea`, `clearance`)
- Marcar produtos para propósitos internos (`staff-pick`, `new-arrival`, `bundle-candidate`)
- Criar coleções filtráveis em promoções e campanhas
- Localizar rapidamente produtos relacionados em diferentes categorias

## Criando uma tag

As tags são criadas automaticamente quando você as adiciona a um produto, mas você também pode gerenciá-las diretamente:
1. Navegue até **Catalogo > Tags de Produtos**
2. Clique em **+ Adicionar Tag de Produto**
3. Insira o **Nome** da tag (ex.: `New Arrival`)
4. O **Slug** é gerado automaticamente a partir do nome (`new-arrival`) — você pode editá-lo se necessário
5. Clique em **Salvar**

Slugs de tags devem ser únicos. O slug é usado internamente e em URLs quando as tags são usadas para filtragem no site de vendas.

## Adicionando tags a um produto

1. Navegue até **Produtos > Todos os Produtos**
2. Abra o produto que deseja etiquetar
3. Localize o campo **Tags** no formulário de edição do produto
4. Digite para pesquisar uma tag existente ou digite o nome de uma nova tag para criá-la no momento
5. Selecione ou crie tantas tags quantas forem necessárias
6. Salve o produto

## Gerenciando tags existentes

### Visualizando todas as tags

A lista de tags em **Catalogo > Tags de Produtos** mostra todas as tags em ordem alfabética. Clique em qualquer tag para editar seu nome ou slug.

### Renomeando uma tag

1. Navegue até **Catalogo > Tags de Produtos**
2. Clique na tag que deseja renomear
3. Atualize o campo **Nome**
4. O slug não é atualizado automaticamente ao renomear — atualize-o manualmente se necessário para mantê-lo consistente
5. Clique em **Salvar**

Nota: renomear uma tag atualiza-a em todos os lugares em que ela é usada, pois os produtos referenciam o mesmo objeto de tag.

### Excluindo uma tag

1. Navegue até **Catalogo > Tags de Produtos**
2. Marque a caixa ao lado da(s) tag(s) que deseja excluir
3. Selecione **Excluir tags de produto selecionadas** no menu suspenso **Ação**
4. Confirme a exclusão

Excluir uma tag remove-a de todos os produtos aos quais foi atribuída. Os produtos não são excluídos — apenas a etiqueta da tag é removida.

## Dicas

- Mantenha os nomes das tags curtos e consistentes — use minúsculas ou Title Case e escolha uma convenção e siga-a em todo o seu catálogo.
- Evite duplicar a lógica de categorias nas tags. As tags funcionam melhor para preocupações transversais (como `new-arrival` ou `staff-pick`) que não se encaixam bem na sua estrutura de categorias.
- Use uma tag `clearance` ou `sale` para marcar produtos para promoções limitadas no tempo — isso facilita encontrar e atualizar esses produtos depois.
- As tags não são visíveis para os clientes por padrão. Seu propósito principal é ajudá-lo a organizar e filtrar produtos no painel de administração.
- Se você encontrar-se criando muitas tags que se sobrepõem às suas categorias, pode valer a pena revisar sua estrutura de categorias em vez disso.
