---
title: 製品レビュー
---

製品レビューにより、顧客は製品に対する体験を評価し、コメントを投稿できます。あなたが承認したレビューは、ショッピングストアの製品ページに表示され、他の顧客が購入決定を下す際の参考になります。Spwigは、どのレビューが表示されるかについて完全な制御権をあなたに与えます — 承認されるまで何も公開されません。

**カタログ > 製品レビュー**にナビゲートして、すべてのレビューを管理してください。

## レビュー一覧

レビュー一覧は、ストア全体で提出されたすべてのレビューの概要を提供します。

| 列 | 説明 |
|---|---|
| **製品** | レビューされている製品 |
| **ユーザー** | レビューを提出した顧客 |
| **評価** | 1〜5の星評価 |
| **承認済み** | レビューが現在ショッピングストアに表示されているかどうか |
| **購入済み確認** | レビュアーが実際に製品を購入したかどうか |
| **作成日時** | レビューが提出された日時 |

### レビューのフィルタリング

右側のフィルタを使用して、特定のレビュー状態に絞り込むことができます。

- **評価** — 特定の星評価のみのレビューを表示（1つ星の苦情を調査するのに役立ちます）
- **承認済み** — 承認済みと保留中のレビューを迅速に区別
- **購入済み確認** — 確認済み購入者からのみレビューをフィルタリング

検索フィールドを使用して、製品名、顧客のユーザー名、またはレビューのタイトルでレビューを検索してください。

## レビューの承認と拒否

レビューは承認されるまでショッピングストアに表示されません。個別にまたは一括でレビューを承認または拒否できます。

### 一括アクション

1. レビュー一覧で、アクションを実行したいレビューの横にあるチェックボックスをクリックします
2. **アクション**ドロップダウンから **選択したレビューを承認** または **選択したレビューを拒否** を選択します
3. **実行** をクリックします

これは、新しいレビューのバッチを処理する最も効率的な方法です。

### 個別レビュー

1. レビュー行をクリックして詳細ビューを開きます
2. **承認済み** フィールドをチェックまたはアンチェックします
3. **保存** をクリックします

## レビューの内容

個別のレビューを開くと、顧客が提出したすべての詳細を確認できます。

Preserve all markdown formatting, image paths, code blocks, and technical terms.

- **タイトル** — 顧客がレビューにつけた見出し
- **コメント** — レビューの全文
- **評価** — スターレーティング（1〜5）
- **購入済みの確認済みレビュー** — レビューをした顧客がその商品の注文を完了している場合、自動的に設定されます
- **役に立った数** — 他の顧客がこのレビューを役に立ったとマークした回数
- **画像** — 顧客がレビューに添付した写真

レビューの内容は編集できません — 承認または拒否するだけです。

## レビューの評価を一目で確認

レビュー一覧は最新のレビューから並べ替えられており、新しく投稿されたフィードバックを簡単に把握できます。パターンを迅速に把握したい場合は:

- **評価 = 1 でフィルター** — 全ての批判的なレビューを1つのリストで確認できます。これらは、製品の問題や発送の問題を特定する価値があることが多いです
- **承認済み = いいえ でフィルター** — あなたのモデレーションキューを確認できます。このキューを定期的に確認して、顧客がレビューの掲載を長期間待たされることを防ぎましょう
- **購入済みの確認済み = はい でフィルター** — 実際に購入した顧客からのレビューだけを表示します。これらは他の顧客にとって最も信頼性が高い傾向があります

## ヒント

- 承認キューを定期的に確認するスケジュールを設定しましょう — 顧客はレビューが数週間掲載されないと気がつくことがあります。これにより、将来的なフィードバックが減る可能性があります
- 承認を迅速に行うことで、顧客との信頼関係を築くことができます。目に見える否定的なレビューに応答がないと、対応した苦情よりも最悪に見える場合があります
- 境界線にあるレビューを処理する際には、**購入済みの確認済み**フィルターを使用しましょう — 確認済みの購入者からのフィードバックは、あらゆる紛争においてより重みを持ちます
- 同じ問題を指摘する1つ星のレビューが多数ある製品は、梱包、製品の品質、または製品の掲載内容を調査する明確なサインです
- レビューは編集できないため、不適切な内容が含まれている場合、唯一の選択肢は拒否することです — モデレーション基準を一貫して適用してください
