---
title: プロモーションの例
---

このガイドでは、さまざまなプロモーションタイプを構成する具体的な例を示しています。各例には、プロモーションウィザードで入力する正確なフィールド値が含まれており、それに従ってください、または自分のストアに合わせて調整してください。

![プロモーションカード](/static/core/admin/img/help/promotion-examples/promotion-card.webp)

## 例: カテゴリに対する割引率

**シナリオ:** 冬のセールで、すべての靴に30％割引。

**マーケティング > 売上とプロモーション**に移動し、**+ プロモーションの作成**をクリックします。ウィザードの各ステップで次の値を入力してください:

| ステップ | フィールド | 値 |
|---------|---------|-----|
| ベーシック | 名前 | 冬のセール — 靴30％割引 |
| ベーシック | 説明 | 季節終わりのセール、すべての靴用 |
| ベーシック | 有効 | チェック済み |
| 割引 | タイプ | 割引率 |
| 割引 | 値 | 30 |
| 日程 | 開始日 | 2026年1月15日 |
| 日程 | 終了日 | 2026年2月28日 |
| 商品 | 適用対象 | カテゴリ |
| 商品 | 選択 | 靴、ブーツ、サンダル |

これにより、選択したカテゴリ内のすべての商品に自動的に割引が適用される、期限付きセールが作成されます。120ドルのブーツは84ドル、60ドルのサンダルは42ドルになります。

## 例: コレクションに対する固定金額割引

**シナリオ:** サマーエッセンシャルズコレクションの商品に15ドル割引。

| ステップ | フィールド | 値 |
|---------|---------|-----|
| ベーシック | 名前 | サマーエッセンシャルズ — 15ドル割引 |
| ベーシック | 有効 | チェック済み |
| 割引 | タイプ | 固定金額割引 |
| 割引 | 値 | 15.00 |
| 日程 | 開始日 | 2026年6月1日 |
| 日程 | 終了日 | (空 — 終了なし) |
| 商品 | 適用対象 | コレクション |
| 商品 | 選択 | サマーエッセンシャルズ |

> **注意:** 15ドルの割引は、対象商品ごとに適用されます。50ドルの商品は35ドル、30ドルの商品は15ドルになります。終了日を空欄にすると、プロモーションは無期限に続き、手動で無効化するまで有効です。

## 例: クリアランス用の固定販売価格

**シナリオ:** すべてのクリアランス商品を9.99ドルに設定。

| ステップ | フィールド | 値 |
|---------|---------|-----|
| ベーシック | 名前 | 最終クリアランス — すべての商品9.99ドル |
| ベーシック | 有効 | チェック済み |
| 割引 | タイプ | 固定販売価格 |
| 割引 | 値 | 9.99 |
| 日程 | 開始日 | (今日は) |
| 商品 | 適用対象 | コレクション |
| 商品 | 選択 | 最終クリアランス |

> **注意:** 固定販売価格は、元の価格に関係なく正確な販売価格を設定します。75ドルの商品と25ドルの商品はすべて9.99ドルになります。クリアランスラックや、すべての商品を同じ価格に設定したい場合に使用してください。

## 適切な割引タイプの選択

| タイプ | 仕組み | 最適な用途 | 例 |
|-------|--------|------------|----|
| **割引率** | 価格をパーセンテージで減らす | 商品の価格が異なる広範なセール | 20％割引 — 100ドルは80ドル、50ドルは40ドル |
| **固定金額割引** | 固定ドル額を引く | 特定のドル額の節約を伝えるプロモーション | 15ドル割引 — 100ドルは85ドル、50ドルは35ドル |
| **固定販売価格** | 正確な販売価格を設定 | クリアランス、均一価格、"すべてXドル" | 9.99ドル — 元の価格に関係なくすべての商品が9.99ドル |

## 適切な対象の選択

| 対象 | 仕組み | 最適な用途 |
|------|--------|------------|
| **すべての商品** | ストア内のすべての商品に適用 | 店舗全体のセール、店舗全体のイベント |
| **カテゴリ** | 選択したカテゴリ内のすべての商品に適用 | 部門セール、季節ごとのクリアランス |
| **ブランド** | 選択したブランドのすべての商品に適用 | ブランド提携、ブランド固有のイベント |
| **コレクション** | 選択したコレクション内のすべての商品に適用 | 精選されたプロモーション、テーマ別セール |
| **商品** | 個別に選択した商品に適用 | 手選別された特典、限定セレクション |

## 調度パターン

プロモーションのスケジュールを設定するための3つの一般的なパターン:

| パターン | 開始日 | 終了日 | 用途 |
|--------|--------|--------|------|
| **即時、継続的** | 今日 | (空) | 永続的な価格引き下げ、長期的なセール |
| **日程範囲** | 未来の日 | 未来の日 | 季節イベント、祝日セール |
| **未来の開始、終了なし** | 未来の日 | (空) | 特定の日から始まる永続的な価格設定 |

開始日を未来に設定すると、スケジュールされたプロモーションが作成されます。これは、プロモーションダッシュボードの**スケジュール済み**タブに表示され、日付が来るときに自動的に有効になります。終了日を空欄にすると、プロモーションは手動で無効化するまで有効のままです。

## ヒント

- **説明的な名前を使用する** — 名前に割引額と対象を含めます（例: "サマーセール — 靴20％割引"）ことで、ダッシュボードでプロモーションをすぐに識別できます。
- **影響を受ける商品数を確認する** — レビュー手順では、どのくらいの商品が割引されるかが表示されます。数が間違っている場合は、ターゲティングを確認してください。
- **少なめに始めましょう** — 割引について不確実な場合は、小さなパーセンテージから始め、必要に応じて増やしてください。
- **マーケティングには固定金額割引を使用する** — "15ドル割引"は、広告やメールキャンペーンで簡単に伝えることができる具体的な節約額です。
- **公平性のために割引率を使用する** — パーセンテージ割引は価格に応じてスケールし、さまざまな価格ポイントで比例した節約を提供します。
