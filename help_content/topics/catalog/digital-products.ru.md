---
title: Цифровые товары
---

Цифровые товары позволяют продавать загружаемые файлы, лицензии на программное обеспечение и другие нефизические товары. Spwig поддерживает как самостоятельные цифровые товары, так и гибридные товары, сочетающие физическую и цифровую доставку.

![Поставщики лицензий](/static/core/admin/img/help/digital-products/license-providers.webp)

## Типы цифровых товаров

### Самостоятельный цифровой товар

Установите **Тип товара** как **Цифровой товар** для чисто цифровых позиций:
- Программные приложения
- Электронные книги и PDF
- Музыка и аудиофайлы
- Цифровое искусство и шаблоны

### Гибридные товары

Любой тип товара может включать цифровую доставку, если отметить **Является цифровым товаром** на вкладке «Основная информация». Это полезно для:
- **Вариативных цифровых товаров** — Программное обеспечение с редакциями Базовая/Про/Корпоративная
- **Настраиваемых цифровых товаров** — Цифровые активы по индивидуальному дизайну
- **Физических + цифровых комплектов** — Книга, включающая цифровую загрузку

## Настройка цифрового товара

### Шаг 1: Создание товара

1. Перейдите в **Товары > Все товары** и нажмите **+ Добавить товар**
2. Установите **Тип товара** как **Цифровой товар** (или отметьте **Является цифровым товаром** для другого типа товара)
3. Заполните данные товара (название, описание, цена)
4. Сохраните товар

### Шаг 2: Добавление файлов для скачивания

1. Перейдите на вкладку **Запасы** товара
2. В разделе **Цифровые файлы** загрузите файлы, которые покупатели получат после покупки
3. Для каждого файла вы можете установить:
   - **Имя файла** — Отображаемое имя, показываемое покупателям
   - **Лимит скачиваний** — Максимальное количество загрузок файла (0 = неограниченно)
   - **Дней до истечения** — Количество дней, в течение которых ссылка для скачивания остаётся активной

### Шаг 3: Настройка доставки лицензий (необязательно)

Если ваш цифровой товар требует лицензионных ключей:

1. Перейдите в **Настройки > Управление лицензиями**
2. Подключите поставщика лицензий (см. ниже)
3. В форме редактирования товара назначьте поставщика лицензий

## Поставщики лицензий

Поставщики лицензий — это внешние сервисы, которые автоматически генерируют и управляют лицензионными ключами программного обеспечения при покупке вашего товара покупателем.

### Доступные типы поставщиков

| Поставщик | Описание |
|-----------|----------|
| **Встроенный сервер лицензий Spwig** | Простая генерация лицензионных ключей, встроенная в платформу |
| **Keygen.sh** | Полнофункциональный API управления лицензиями |
| **LicenseSpring** | Корпоративное управление лицензиями |
| **Cryptlex** | Лицензирование ПО с поддержкой офлайн-режима |
| **Пользовательский API** | Подключение любой системы лицензирования через REST API |

### Подключение поставщика лицензий

1. Перейдите в **Настройки > Управление лицензиями**
2. Нажмите **Подключить поставщика**
3. Следуйте мастеру настройки:
   - **Шаг 1** — Выберите тип поставщика
   - **Шаг 2** — Настройте общие параметры
   - **Шаг 3** — Введите учётные данные API
4. Протестируйте подключение, чтобы убедиться в его работоспособности
5. Сохраните конфигурацию

### Карточка поставщика

Каждый подключённый поставщик отображает:
- **Индикаторы статуса** — Активен/Неактивен и состояние подключения
- **Конечная точка API** — Настроенный URL сервера
- **Возможности синхронизации** — Поддержка синхронизации заказов, активации и деактивации
- **Кнопки действий** — Настроить, Тестировать и Синхронизировать сейчас

### Возможности синхронизации

Поставщики лицензий могут синхронизироваться при трёх событиях:

- **Заказ** — Автоматическая генерация лицензионного ключа при завершении покупки покупателем
- **Активация** — Отслеживание активации лицензии покупателем
- **Деактивация** — Обработка деактивации лицензии для возвратов или передачи

## Опыт покупателя

### После покупки

Когда покупатель приобретает цифровой товар:

1. **Подтверждение заказа** — Показывает, что включена цифровая доставка
2. **Доставка по электронной почте** — Ссылки для скачивания и/или лицензионные ключи отправляются автоматически
3. **Страница аккаунта** — Покупатели могут получить доступ к своим загрузкам из панели управления аккаунтом
4. **Страница загрузки** — Защищённые ссылки для скачивания с ограниченным сроком действия

### Безопасность загрузок

Загрузки цифровых файлов защищены:
- Уникальными токенами загрузки с ограниченным сроком действия
- Необязательными ограничениями на количество скачиваний
- Сроками истечения, после которых ссылки становятся неактивными
- Требованием авторизации (для зарегистрированных покупателей)

## Советы

- Установите разумные лимиты скачиваний (3-5 загрузок), чтобы предотвратить злоупотребления, но при этом разрешить повторные загрузки.
- Используйте дни истечения, соответствующие вашему периоду поддержки (например, 365 дней для годового доступа).
- Протестируйте полный процесс покупки с тестовым заказом, чтобы убедиться, что ссылки для скачивания и лицензионные ключи доставляются корректно.
- Для программных продуктов подключите поставщика лицензий для автоматизации генерации ключей, вместо ручного управления ключами.
- Используйте функцию гибридного товара при продаже физических товаров с цифровыми дополнениями (например, печатная книга + PDF).
