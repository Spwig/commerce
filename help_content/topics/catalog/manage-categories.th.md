หมวดหมู่ช่วยให้คุณจัดระเบียนแคตตาล็อกสินค้าของคุณ ทำให้ลูกค้าสามารถเรียกดูและค้นหาสินค้าได้ง่ายขึ้น ไปที่ **Products > Categories** ในแถบด้านข้างของ admin

![รายการหมวดหมู่](/static/core/admin/img/help/manage-categories/category-list.webp)

## รายการหมวดหมู่

หน้าจัดการหมวดหมู่แสดงหมวดหมู่ทั้งหมดของคุณในรูปแบบการ์ดที่มี:

- **รูปภาพตัวย่อ** — ตัวระบุทางภาพสำหรับหมวดหมู่
- **ชื่อและ Slug** — ชื่อที่แสดงและตัวระบุที่เป็นมิตรกับ URL
- **จำนวนสินค้า** — จำนวนสินค้าที่ถูกกำหนดให้กับหมวดหมู่นี้
- **สถานะ** — ได้รับการเผยแพร่หรือร่าง

ใช้ **แท็บกรอง** ที่ด้านบนเพื่อดูรายการทั้งหมด ได้รับการเผยแพร่ หรือร่างได้อย่างรวดเร็ว ช่องค้นหาช่วยให้คุณค้นหาหมวดหมู่ตามชื่อ

## การสร้างหมวดหมู่

1. คลิกที่ **+ Add Category** ที่มุมขวาบน
2. กรอกข้อมูลหมวดหมู่:
   - **ชื่อ** — ชื่อที่แสดงให้ลูกค้าเห็น
   - **Slug** — สร้างอัตโนมัติจากชื่อ ใช้ใน URL
   - **Parent Category** — ปล่อยว่างสำหรับหมวดหมู่ระดับบนสุด หรือเลือกหมวดหมู่พ่อแม่เพื่อสร้างหมวดหมู่ย่อย
   - **Description** — คำอธิบายแบบ Rich text ที่แสดงในหน้าหมวดหมู่
3. อัปโหลด **รูปภาพหมวดหมู่** — แสดงในเมนูนำทางและรายการหมวดหมู่
4. ตั้งค่า **ฟิลด์ SEO** (ชื่อ meta, คำอธิบาย) บนแท็บ SEO
5. คลิก **Save**

## โครงสร้างลำดับชั้นของหมวดหมู่

หมวดหมู่รองรับการฝังที่ไม่มีขีดจำกัดเพื่อสร้างโครงสร้างต้นไม้:

- **หมวดหมู่ระดับบนสุด** — รายการนำทางหลัก (เช่น "Clothing", "Electronics")
- **หมวดหมู่ย่อย** — ฝังอยู่ภายใต้หมวดหมู่พ่อ (เช่น "Clothing > Men's > T-Shirts")

กล่องลงรายการหมวดหมู่พ่อจะแสดงเส้นทางลำดับชั้นทั้งหมดเพื่อช่วยให้คุณเลือกระดับที่เหมาะสม

## การตั้งค่าหมวดหมู่

### ความเป็นไปได้ในการแสดงผล

- **เผยแพร่** — หมวดหมู่จะปรากฏในร้านค้าและเมนูนำทาง
- **ร่าง** — หมวดหมู่จะซ่อนจากลูกค้า แต่สามารถเข้าถึงได้ใน admin

### หมวดหมู่ที่แนะนำ

เลือกหมวดหมู่เป็น **หมวดหมู่ที่แนะนำ** เพื่อเน้นแสดงในหน้าแรกของคุณหรือในส่วนนำทางพิเศษ หมวดหมู่ที่แนะนำสามารถแสดงได้โดยใช้ตัวอย่างตารางหมวดหมู่ใน Page Builder

### ลำดับการจัดเรียง

ควบคุมว่าหมวดหมู่จะปรากฏในเมนูนำทางอย่างไรโดยตั้งค่า **ลำดับการจัดเรียง** ค่าที่ต่ำกว่าจะปรากฏก่อน

## การกำหนดค่าสินค้าให้กับหมวดหมู่

มีสองวิธีในการกำหนดค่าสินค้า:

1. **จากฟอร์มแก้ไขสินค้า** — เลือกหมวดหมู่ในกล่องลงรายการหมวดหมู่ในแท็บ Basic Info
2. **การกำหนดค่าแบบกลุ่ม** — เลือกสินค้าหลายรายการจากรายการสินค้าและใช้การกระทำแบบกลุ่มเพื่อกำหนดให้อยู่ในหมวดหมู่

แต่ละสินค้าสามารถอยู่ในหมวดหมู่หลักได้เพียงหนึ่งเดียว ใช้แท็กหรือคอลเลกชันเพื่อจัดกลุ่มเพิ่มเติม

## หน้าหมวดหมู่ในร้านค้า

แต่ละหมวดหมู่ที่เผยแพร่มีหน้าเฉพาะที่แสดง:
- ชื่อหมวดหมู่และคำอธิบาย
- รูปภาพแบนเนอร์ (ถ้าตั้งค่าแล้ว)
- ตารางสินค้าที่มีสินค้าทั้งหมดที่ถูกกำหนดให้
- ตัวเลือกการกรองและการจัดเรียง

URL ของหน้าหมวดหมู่จะมีรูปแบบตาม: `yourstore.com/category/category-slug/`

## เคล็ดลับ

- ให้โครงสร้างต้นไม้ของหมวดหมู่มีความลึกไม่มาก — ความลึก 2-3 ระดับเป็นที่เหมาะสมสำหรับการใช้งานการนำทาง
- ใช้ชื่อหมวดหมู่ที่อธิบายได้ชัดเจนที่ตรงกับสิ่งที่ลูกค้าค้นหา
- เพิ่มรูปภาพหมวดหมู่เพื่อประสบการณ์การเรียกดูที่มีภาพมากขึ้น
- ตั้งค่าโครงสร้างหมวดหมู่ของคุณก่อนเพิ่มสินค้าเพื่อรักษาความเป็นระเบียบ
- ใช้คำอธิบายหมวดหมู่เพื่อ SEO — รวมคำหลักที่เกี่ยวข้องอย่างเป็นธรรมชาติ
