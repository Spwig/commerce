# 상품 추가

이 가이드는 스토어에서 새로운 상품을 생성하는 방법을 안내합니다. 상품은 여러 탭 — 기본 정보, 미디어, 가격, 재고, SEO —에 걸쳐 구성되어 있으며, 한 번에 모든 정보를 입력하거나 나중에 다시 돌아와서 섹션을 완료할 수 있습니다.

## 시작하기

사이드바에서 **상품 > 모든 상품**으로 이동하여 상품 카탈로그를 확인하세요. 상단 오른쪽 모서리의 **+ 상품 추가** 버튼을 클릭하여 상품 생성 폼을 열 수 있습니다.

![상품 목록 페이지](/static/core/admin/img/help/add-product/product-list-page.webp)

## 기본 정보 탭

**기본 정보** 탭은 상품의 핵심 정보를 정의하는 곳입니다.

![상품 추가 폼](/static/core/admin/img/help/add-product/add-product-form.webp)

### 필수 필드

- **이름** — 고객에게 표시되는 상품 이름입니다. 전 세계 언어로 번역하려면 지구仪 아이콘을 클릭하세요.
- **Slug** — 이름의 URL 친화적 버전(자동 생성됨). "Auto"를 해제하여 사용자 지정할 수 있습니다.
- **SKU** — 내부 재고 관리 단위 코드입니다.
- **상품 유형** — 선택 가능한 유형: 단순, 가변, 디지털, 묶음, 선물 카드, 맞춤형, 구성 가능.
- **상태** — 작업 중에는 초안으로 설정하고 완료되면 "발행"으로 변경하세요.

### 선택 필드

- **카테고리** — 상품을 카테고리에 할당하여 조직화하고 쇼핑몰 내 탐색을 도와줍니다.
- **브랜드** — 해당하는 경우 브랜드와 연결하세요.
- **특징 상품** — 이 상품을 쇼핑몰에서 강조 표시하려면 체크하세요.
- **디지털 상품** — 이 상품이 디지털 다운로드(파일, 라이선스)를 포함하는 경우 체크하세요.
- **쇼핑몰에서 숨기기** — 상품을 카탈로그 목록에서 숨기되, 구성 옵션 또는 묶음 구성 요소로 사용 가능한 상태를 유지합니다.

### 상품 설명

- **짧은 설명** — 상품 목록 및 카드에 표시됩니다. 간결하고 매력적인 설명을 유지하세요.
- **상세 설명** — 상품 상세 페이지에서 표시되는 상세 설명입니다. 풍부한 텍스트 편집기로 서식, 이미지, 영상, 표를 추가할 수 있습니다.

두 설명 필드 모두 번역 기능을 지원합니다 — 지구仪 아이콘을 클릭하여 다른 언어의 내용을 제공하세요.

## 미디어 탭

**미디어** 탭은 통합된 미디어 라이브러리를 사용하여 상품 이미지를 관리할 수 있습니다.

![미디어 탭](/static/core/admin/img/help/add-product/media-tab.webp)

1. **+ 미디어 라이브러리에서 이미지 추가**를 클릭하여 미디어 선택기를 엽니다.
2. 기존 이미지 또는 직접 새 이미지를 업로드합니다.
3. 이미지를 드래그하여 순서를 재정렬합니다 — **첫 번째 이미지**는 목록 및 카드에 표시되는 기본 상품 이미지가 됩니다.
4. **갤러리 유형**을 선택하여 상품 페이지에서 이미지가 어떻게 표시되는지 제어합니다: 표준 갤러리, 캐러셀, 격자 레이아웃, 줌 갤러리, 또는 360° 뷰.

## 가격 탭

상품의 가격을 설정하고 할인을 구성합니다.

![가격 탭](/static/core/admin/img/help/add-product/pricing-tab.webp)

### 일반 가격

- **일반 가격** — 고객이 보게 될 표준 소매 가격입니다.
- **통화** — 통화를 선택하세요(스토어의 기본 통화는 미리 선택되어 있습니다).
- **비용** — 상품의 원가, 이익 계산에 사용됩니다. 이 값은 고객에게 절대 표시되지 않습니다.

### 할인 설정

임시 할인을 구성합니다:

- **할인 유형** — 선택 가능한 유형: 할인 없음, 고정 판매 가격, 금액 할인, 비율 할인.
- **할인 금액** — 할인 금액 또는 비율입니다.
- **시작/종료 날짜** — 할인이 활성화되고 만료되는 시점을 설정합니다. 즉시 시작하거나 종료 날짜가 없는 경우 비워두세요.

## 재고 탭

재고 수량 및 물리적 상품 속성을 관리합니다.

![재고 탭](/static/core/admin/img/help/add-product/inventory-tab.webp)

### 재고 관리

- **재고 추적** — 재고 수량을 추적하려면 활성화하세요(기본값: 활성화됨).
- **저 재고 임계값** — 재고가 이 수치 이하가 되면 알림을 받습니다(기본값: 5).
- **재고 수량** — 제공 가능한 총 단위 수.
- **백오더 허용** — 재고가 없을 때도 주문을 허용하려면 활성화하세요.

### 물리적 속성

정확한 배송 계산을 위해 상품의 무게(kg) 및 차원(길이, 너비, 높이 cm)을 입력하세요.

### 상품 식별자

마켓플레이스 목록 및 재고 시스템에 사용되는 표준 상품 코드:

- **GTIN** — 글로벌 무역 항목 번호
- **EAN** — 유럽 제품 번호
- **UPC** — 유니버설 제품 코드 (미국)
- **ISBN** — 책용
- **ASIN** — 아마존 식별자
- **MPN** — 제조사 부품 번호

### 국제 배송 / 관세

국제 배송에 필요한 필수 정보:

- **HS 코드** — 하모니즈 시스템 분류 코드
- **제조 국가** — 상품이 제조된 국가
- **관세 단위 가격** — 단위당 선언된 가격입니다.

## SEO 탭

상품의 검색 엔진 가시성을 최적화합니다.

![SEO 탭](/static/core/admin/img/help/add-product/seo-tab.webp)

- **메타 제목** — 검색 엔진 결과에 표시되는 제목입니다. 지구仪 아이콘을 클릭하여 번역하세요.
- **메타 설명** — 검색 결과용 간단한 설명(최대 160자). 지구仪 아이콘을 클릭하여 번역하세요.
- **SEO 자동 생성** — 상품을 저장할 때 SEO 콘텐츠를 자동으로 생성하도록 체크하세요.

실시간 **검색 결과 미리보기**는 상품이 구글 검색 결과에 어떻게 표시되는지 정확히 보여줍니다.

## 상품 저장

준비가 되면 상단 오른쪽의 저장 버튼을 사용하세요:

- **저장** (체크마크) — 저장하고 상품 페이지에 머무릅니다.
- **저장 및 편집 계속** — 저장하고 폼에 머무라며 계속 작업합니다.

상품의 상태가 **발행**으로 설정되면 쇼핑몰에서 보입니다.

## 팁

- **초안** 상태로 시작하여 고객이 상품을 볼 때까지 완성도를 높이세요.
- 여러 이미지를 업로드하세요 — 여러 사진이 있는 상품이 더 잘 전환됩니다.
- **SEO** 필드를 채우면 검색 엔진에서의 발견성을 높일 수 있습니다.
- **카테고리** 및 **브랜드**를 사용하여 고객이 쇼핑몰 카탈로그를 탐색하는 데 도움을 주세요.
- 가변 상품(예: 다른 사이즈나 색상)은 **가변 상품** 유형을 선택하고 저장 후 변형을 추가하세요.
