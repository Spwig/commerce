---
title: "Наборы товаров"
---

Наборы товаров позволяют продавать заранее собранные пакеты товаров по специальной цене. Это идеально подходит для подарочных наборов, стартовых комплектов или любой комбинации товаров, которые вы хотите предложить вместе со скидкой.

![Bundle components admin](/static/core/admin/img/help/product-bundles/bundle-components.webp)

## Стратегии ценообразования

Выберите, как рассчитывается цена набора:

| Стратегия | Описание |
|-----------|----------|
| **Фиксированная цена** | Установите единую цену для всего набора, независимо от цен компонентов. |
| **Процентная скидка** | Автоматический расчёт цены как процент скидки от суммарной стоимости компонентов. |
| **Сумма компонентов** | Цена набора равна сумме цен всех компонентов (полезно для группового отображения без скидки). |

## Создание набора

### Шаг 1: Создание товара

1. Перейдите в **Товары > Все товары** и нажмите **+ Добавить товар**
2. Установите **Тип товара** как **Набор товаров**
3. Заполните название набора, описание и изображения
4. Сохраните товар

### Шаг 2: Добавление компонентов

Переключитесь на вкладку **Элементы набора** для добавления товаров в набор:

1. Нажмите **+ Добавить компонент**
2. Найдите и выберите товар из выпадающего списка
3. Установите **Количество** для каждого компонента (например, 2 маски для лица в наборе по уходу за кожей)
4. Установите **Порядок сортировки** для управления порядком отображения
5. При необходимости отметьте компонент как **Необязательный** (покупатели могут его исключить)
6. Если компонент является товаром с вариантами, выберите:
   - **Фиксированный вариант** — все покупатели получают один и тот же вариант
   - **Разрешить выбор варианта** — покупатели выбирают предпочтительный вариант при оформлении заказа

Сводка внизу показывает **Общее количество компонентов** и **Стоимость набора** (сумма цен компонентов).

### Шаг 3: Настройка ценообразования

Переключитесь на вкладку **Ценообразование**:

1. Выберите **Стратегию ценообразования набора**
2. Для **Фиксированной цены** — введите цену набора напрямую
3. Для **Процентной скидки** — установите процент скидки (например, 15% скидка)
4. Для **Суммы компонентов** — цена рассчитывается автоматически

## Что можно включить в набор

| Тип товара | Может быть компонентом? |
|------------|------------------------|
| Простой товар | Да |
| Товар с вариантами | Да (фиксированный вариант или выбор покупателя) |
| Цифровой товар | Да |
| Настраиваемый товар | Нет |
| Конфигурируемый товар | Нет |
| Набор товаров | Нет (наборы не могут быть вложенными) |
| Подарочная карта | Нет |

## Управление складскими запасами

Запасы набора управляются через его компоненты:

- **Все компоненты должны быть в наличии**, чтобы набор можно было приобрести
- При заказе набора запасы списываются с каждого компонента по отдельности
- Если любой компонент заканчивается, набор становится недоступным
- Уровни запасов компонентов проверяются в реальном времени при оформлении заказа

## Необязательные компоненты

Отметьте компонент как **Необязательный**, чтобы позволить покупателям настраивать свой набор:

- Необязательные компоненты включены по умолчанию, но могут быть удалены покупателем
- Цена набора корректируется при исключении необязательных компонентов
- Как минимум один компонент должен быть обязательным

## Опыт покупателя

Когда покупатель просматривает набор в вашей витрине:

1. **Список компонентов** — Все включённые товары отображаются с изображениями и количеством
2. **Экономия на наборе** — Показывается скидка по сравнению с покупкой товаров по отдельности
3. **Выбор варианта** — Для компонентов с включённым выбором варианта покупатели выбирают предпочтительный вариант
4. **Необязательные элементы** — Покупатели могут включать или отключать необязательные компоненты
5. **Единое добавление в корзину** — Весь набор добавляется как один товар

## Советы

- Используйте стратегию Процентной скидки для наиболее гибкого ценообразования — она автоматически корректируется при изменении цен компонентов.
- Покажите сумму экономии заметно в описании товара, чтобы стимулировать покупку наборов.
- Ограничьте наборы 3-5 компонентами для лучшего покупательского опыта. Слишком много товаров может показаться перегруженным.
- Используйте необязательные компоненты, чтобы предложить «базовую» и «премиум» версию одного и того же набора.
- Регулярно проверяйте, что все компоненты по-прежнему активны и в наличии.
