---
title: 價格魅力規則
---

價格魅力（也稱為心理定價）會自動調整您的產品價格，使其以特定數字結尾，這些數字對客戶來說更具吸引力。例如，與顯示 $20.00 的價格相比，價格魅力可以顯示 $19.99 — 這是一種廣泛使用的技巧，讓價格看起來更低。

Spwig 會根據您商店接受的每種貨幣自動應用價格魅力規則，因此您只需為每個規則設置一次。

## 價格魅力如何運作

當產品價格被計算（包括促銷或折扣後的價格）時，Spwig 會檢查是否存在適用於該貨幣的活動價格魅力規則。如果存在，價格會在顯示給客戶之前進行調整。此調整適用於高於您設定的最低閾值的價格。

您可以為您商店接受的每種貨幣配置不同的規則。例如，您可能對 USD 使用 `.99` 結尾，但對 JPY 四捨五入到最近的 `¥10`。

## 建立價格魅力規則

1. 進入 **目錄 > 價格魅力規則**
2. 點擊 **+ 添加價格魅力規則**
3. 選擇此規則適用的 **貨幣**（例如，`USD`、`EUR`、`NZD`）
4. 選擇 **規則類型**（請參閱下表）
5. 可選地設置 **最低價格閾值** 以排除非常低的價格
6. 如果您還希望在商品促銷時應用價格魅力，請勾選 **應用於促銷價格**
7. 確保 **啟用** 已勾選
8. 點擊 **保存**

每種貨幣只能存在一個規則。如果您需要更改規則，請編輯現有的規則。

## 規則類型

| 規則類型 | 範例 | 最適合用於 |
|-----------|---------|----------|
| **魅力 .99 結尾** | $20.50 → $19.99 | 大多數零售產品 — 经典的心理價格 |
| **魅力 .95 結尾** | $20.50 → $19.95 | .99 的稍微柔和替代方案 |
| **魅力 .90 結尾** | $20.50 → $19.90 | 感覺圓整但仍低於下一美元 |
| **向下取整** | $19.50 → $19.00 | 偏好整數的商店 |
| **向上取整** | $19.50 → $20.00 | 為了清潔顯示稍微向上取整 |
| **四捨五入到最接近的 5** | $23.00 → $25.00 | 高流量零售和市場 |
| **四捨五入到最接近的 10** | $23.00 → $20.00 | 高價商品，如家用電器 |
| **四捨五入到最接近的 100** | $1,234 → $1,200 | 高價值商品，如家具或電子產品 |
| **自定義結尾** | 任何 — 在下方指定 | 當您的品牌使用特定結尾，如 `.88` 時 |

### 自定義結尾

如果您選擇 **自定義結尾**，請在 **自定義結尾** 欄位中輸入結尾值。例如，輸入 `0.88` 以使所有價格以 `.88` 結尾（在某些亞洲市場中常見）。

## 最低價格門檻

使用 **最低價格門檻** 欄位來跳過對非常低價商品的價格魅力調整，因為調整後看起來會很奇怪。例如，設定門檻為 `5.00` 表示價格低於 $5 的商品將顯示其實際計算的價格，不應用價格魅力。

將其設為 `0` 以對所有價格應用價格魅力。

## 優惠價格

預設情況下，價格魅力會應用於常規價格和優惠價格。如果您希望優惠價格顯示其精確計算的值（在有限時間的促銷價格中，精確數字很重要時很有用），請取消勾選 **應用於優惠價格**。

## 停用規則

要臨時停止價格魅力而不刪除規則，請取消勾選 **啟用** 並保存。規則會被保留，隨時可以重新啟用。

## 小提示

保留所有 markdown 格式、圖片路徑、程式碼塊和技術術語。

- 如果不确定，可以從 `.99` 結尾開始設定 — 這是最廣泛被認可的心理定價技巧，對大多數產品類型都有效。
- 如果你販售低價商品（低於 $5），請設定一個最低門檻，以避免 $3.50 的商品被降至 $2.99。
- 在啟用新規則後，請透過商店前端查看產品，以確認價格是否即時顯示為優惠價格。
- 日元和類似的整數貨幣與 **Round to nearest 10** 或 **Round to nearest 100** 配合使用效果最佳，因為小數結尾看起來不太自然。
- 優惠價格會在所有折扣和促銷之後應用，因此你的促銷價格也會顯示為優惠價格，除非你取消勾選 **Apply to Sale Prices**。
- 你可以為不同貨幣設定不同的規則類型，這在你向不同市場銷售且有不同定價慣例時會很有幫助。
