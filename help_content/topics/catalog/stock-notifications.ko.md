---
title: 재고 알림
---

재고 알림은 제품이 다시 입고될 때 고객이 이메일로 알림을 받을 수 있도록 등록할 수 있도록 합니다. 재고 표시 설정은 제품 페이지에서 고객이 보는 내용을 제어합니다. 예를 들어 재고 상태 라벨, 재고 부족 경고, 제품이 품절되었을 때의 처리 방법 등입니다.

## 재고 표시 설정

재고 표시 설정은 전체 스토어의 기본 설정으로, 카테고리 또는 제품 수준에서 덮어쓸 수 있습니다.

**카탈로그 > 재고 표시 설정**으로 이동하여 이러한 옵션을 구성할 수 있습니다. 스토어에 하나의 설정 기록이 있으며, 이를 클릭하여 편집할 수 있습니다.

### 재고 상태 표시

| 설정 | 설명 |
|---------|-------------|
| **재고 상태 표시** | 제품 페이지에 "재고 있음" 또는 "재고 없음" 라벨을 표시합니다 |
| **재고 부족 경고 표시** | 재고가 부족할 때 "남은 수량 X개" 메시지를 표시합니다 |
| **재고 부족 임계값** | 재고 부족 경고가 표시되는 수량 (기본값: 5) |
| **정확한 수량 표시** | 일반적인 경고 대신 "남은 수량 3개!"와 같이 정확한 남은 수량을 표시합니다 |

### 품절 처리

**품절 처리** 설정은 제품이 재고가 없을 때 고객이 보는 내용을 결정합니다:

| 행동 | 고객이 보는 내용 |
|--------|-------------------|
| **목록에서 숨기기** | 제품은 카테고리 페이지 및 검색 결과에서 제거됩니다 |
| **사용 불가능 표시** | 제품은 보이지만 장바구니에 추가할 수 없습니다 |
| **"알림 받기" 버튼 표시** | 고객은 재고가 다시 입고될 때 알림을 받기 위해 이메일을 등록할 수 있습니다 |
| **백오더 허용** | 재고가 없어도 고객은 제품을 구매할 수 있습니다 |

**품절 메시지**를 설정하여 제품이 사용 불가능할 때 표시되는 텍스트를 커스터마이즈할 수 있습니다 (기본값: `재고 없음`).

**백오더 메시지**를 설정하여 백오더 가능한 제품에 표시되는 텍스트를 커스터마이즈할 수 있습니다 (기본값: `백오더 가능`).

### 배송 및 배송 표시


| 설정 | 설명 |
|---------|-------------|
| **"배송 출발지" 위치 표시** | 상품 페이지에 창고 이름을 표시 |
| **예상 배송일 표시** | 창고 위치를 기반으로 계산된 예상 배송일을 표시 |

### 전체적으로 백오더 허용

**백오더 허용**을 선택하여 기본적으로 고객이 재고가 없는 상품을 구매할 수 있도록 허용합니다. 개별 상품 및 카테고리는 이 설정을 재정의할 수 있습니다.

## 재입고 알림

상품이 품절 상태일 때 **"알림 받기" 버튼 표시**로 설정하면, 고객은 상품 페이지에서 이메일 주소를 입력하여 재입고 시 이메일을 받을 수 있습니다.

### 알림 요청 보기

**카탈로그 > 재고 알림**으로 이동하여 모든 고객의 알림 요청을 볼 수 있습니다. 각 기록은 다음과 같은 정보를 보여줍니다:
- 고객 이메일 주소
- 상품 및 변형(적용 가능한 경우)
- 선호 창고(고객이 지역 선호도를 선택한 경우)
- 요청이 생성된 시간
- 알림이 전송된 시간(아직 전송되지 않은 경우 빈칸)

### 알림이 전송되는 시점

Spwig은 상품의 재고 수준이 0보다 증가할 때 자동으로 재입고 이메일을 전송합니다. **알림 시간** 필드는 이메일이 전송된 시간을 기록합니다.

고객은 한 번의 알림 이메일을 받습니다. 제품이 다시 품절 상태가 되면 알림을 받은 후 다시 등록해야 합니다.

### 알림 요청 필터링

관리자 필터를 사용하여 다음을 찾을 수 있습니다:
- 특정 상품에 대한 요청
- 이미 알림이 전송된 요청(이미 연락된 고객을 확인하려면)
- 여전히 대기 중인 요청(재입고를 기다리는 고객)

## 상품 수준 재정의

전체적인 재고 표시 설정은 상품 또는 카테고리별로 재정의할 수 있습니다. 상품 편집 폼에서 **재고** 섹션을 찾아, 전역 기본값과 다른 상품별 **재고 부족 시 조치**를 설정할 수 있습니다.

이 기능은 대부분의 제품을 예약 주문을 허용하면서도 몇 가지 제품만 '알림 받기'로 설정하고 싶을 때, 또는 특정 제품이 재고가 없을 때 숨겨지도록 하고 싶을 때 유용합니다.

## 팁

- **저 재고 임계값**을 일반적으로 사용하는 재주문 지점으로 설정하여 재고가 완전히 소진되기 전에 고객에게 한정된 가용성에 대해 경고하도록 합니다.
- 재고가 없는 제품을 숨기는 대신 **'알림 받기' 버튼 표시** 옵션을 사용하세요. 등록한 고객은 재고 보충 주문을 정당화할 수 있는 실제 수요를 나타냅니다.
- **정확한 수량 표시** 기능은 신중하게 사용하세요. 대부분의 경우 '남은 수량 3개!'와 같은 메시지가 전체 재고 상황을 드러내지 않으면서도 긴급감을 유도하는 데 더 효과적입니다.
- 새로운 주문을 배치하기 전에 재고 알림 목록을 확인하세요. 대기 중인 알림 요청 수는 해당 제품에 대한 수요량을 알려줍니다.
- 예약 주문을 사용하는 경우 **예약 주문 메시지**를 업데이트하여 정확한 기대감을 설정하세요. 예: '2~3주 후 발송 — 지금 주문하여 자리 확보하세요'.
- 재고가 없는 알림과 이메일 마케팅을 결합하세요: 인기 있는 제품을 재고 보충할 때, 자동 알림 이메일만 보내는 대신 등록한 모든 사람에게 캠페인을 보내세요.
