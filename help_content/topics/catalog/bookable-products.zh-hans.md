---
title: 可预订商品
---

可预订商品允许客户在购买时预订特定的日期和时间。这支持预约、租赁、课程、活动和住宿预订——所有内容均可直接在您的 Spwig 管理后台中管理。

## 预订类型

| 类型 | 最适合 | 
|------|----------| 
| **预约** | 服务：咨询、理发、个人训练 | 
| **租赁** | 设备租赁、车辆租赁、房间租赁 | 
| **课程 / 工作坊** | 有固定容量的小组活动 | 
| **住宿** | 有入住/退房时间的多晚住宿 | 
| **活动** | 有门票的一次性或重复性活动 | 

## 设置可预订商品

### 第 1 步：创建商品

1. 导航到 **产品 > 所有产品**，然后点击 **+ 添加产品**
2. 将 **产品类型** 设置为 **预订商品**
3. 填写标准产品字段（名称、描述、价格）
4. 保存产品

### 第 2 步：配置预订设置

保存后，产品编辑表单上会出现一个 **预订配置** 部分。填写预订设置：

#### 预订类型和时长

- **预订类型** — 选择最适合您服务的类型（预约、租赁、课程等）
- **时长类型** — 选择 **固定时长** 用于固定长度的会话，或选择 **客户选择时长** 以允许客户选择他们需要的时间长度
- **时长** 和 **时长单位** — 设置长度（例如，`60` 分钟，`1` 小时，`2` 天）
- **最短/最长时长** — 如果客户可以选择时长，请设置允许的范围

#### 缓冲时间

预订之间会自动添加缓冲时间，以便进行准备或清洁：
- **预订前缓冲** — 预订开始前预留的分钟数
- **预订后缓冲** — 预订结束后预留的分钟数

例如，一个 60 分钟的按摩预约，加上 15 分钟的后续缓冲时间，可以为下一位客户提供 15 分钟的准备时间。

#### 提前预订窗口


- **最低提前通知时间** —— 客户必须提前预订的时间（例如，`24 小时`，因此不允许当天预订）
- **最大提前预订窗口** —— 客户可以预订的未来最远时间（例如，`365 天`）

#### 容量

- **每个时段最大预订数** —— 对于课程和活动，设置同一时间段内可以预订的客户数量。设为 `1` 可用于私人预约。

#### 确认

- **需要手动确认** —— 勾选后，预订不会自动确认。您必须从预订列表中手动批准每个预订。当您希望在确认前审核客户时很有用。

#### 取消政策

- **允许取消** —— 客户是否可以取消预订
- **取消截止时间** —— 客户可以在预订开始前多少小时/天取消（例如，`24 小时`）

#### 日历显示

客户在产品页面上如何选择日期和时间：

| 显示模式 | 最佳用途 |
|---------|--------|
| **日历视图** | 一般用途 —— 完整的月历 |
| **日期选择器** | 简单的单日选择 |
| **可用日期下拉菜单** | 可用时段有限的产品 |
| **日期范围选择器** | 住宿和多日租赁 |

#### 押金

若要在结账时要求押金而非全额付款：
1. 勾选 **启用押金**
2. 将 **押金类型** 设置为 **固定金额** 或 **总金额的百分比**
3. 输入 **押金金额**（例如，`50` 表示 50 美元，或 `25` 表示 25%）

#### 住宿相关设置

对于住宿预订，会显示额外字段：
- **入住时间** 和 **退房时间** —— 财产的标准时间
- **标准入住人数** —— 基础价格中包含的默认客人数量

### 第 3 步：添加预订资源（可选）

资源是分配给预订的实物物品或工作人员 —— 例如，"房间 1"、"A 球场" 或 "教练 Sam"。

1.

在产品编辑表单中，进入 **预订资源** 部分
2.

点击 **添加资源**
3.


为资源指定 **名称** 并设置其 **容量**（可同时处理的预订数量）
4.

可选地添加资源图片

资源使您能够按单个资产或员工跟踪可用性，而不仅仅是按时间段。

### 第4步：设置可用性规则

可用性规则定义了可以进行预订的时间：

1. 在产品 **可用性** 部分下，点击 **添加可用性规则**
2. 选择此规则适用的 **资源**
3. 设置预订可用的 **一周中的天数**
4. 设置可用时间段的 **开始时间** 和 **结束时间**
5. 可选地设置一个日期范围（**生效日期** / **截止日期**）用于季节性可用性
6. 保存

## 查看和管理预订

### 预订列表

导航到 **目录 > 预订** 以查看所有预订。您可以按以下条件进行筛选：
- 状态（待确认、已确认、已取消、已完成、未到场）
- 产品
- 日期范围

### 预订状态

| 状态 | 含义 |
|------|------|
| **待确认** | 等待手动批准（如果需要确认） |
| **已确认** | 预订已确认并处于活动状态 |
| **已取消** | 预订已被客户或您取消 |
| **已完成** | 预订日期已过且已履行 |
| **未到场** | 客户未到场 |

### 确认待确认的预订

1. 从 **目录 > 预订** 打开预订
2. 将 **状态** 更改为 **已确认**
3. 保存 — 客户会自动收到确认电子邮件

### 取消预订

1. 打开预订
2. 将 **状态** 更改为 **已取消**
3. 输入 **取消原因**（显示在客户的电子邮件中）
4. 保存

## 管理候补名单

当时间段已满时，客户可以将自己添加到候补名单中。Spwig 会在取消导致空位时自动通知候补名单上的客户。

### 查看候补名单

导航到 **目录 > 预订候补名单** 以查看所有候补名单条目。

每个条目显示：
- 客户姓名和电子邮件
- 产品和期望日期
- 状态：**等待中**，**已通知**，**已转为预订** 或 **已过期**

### 等待列表状态

| 状态 | 含义 |
|--------|---------|
| **等待中** | 客户已排队，尚未有可用时段 |
| **已通知** | 已通过电子邮件通知客户有可用时段 |
| **已转为预订** | 客户已占用时段并完成预订 |
| **已过期** | 期望日期已过，没有可用时段 |

### 手动通知等待列表中的客户

如果要在自动通知之前联系特定的等待列表客户：
1. 打开等待列表条目
2. 复制他们的电子邮件地址并直接联系他们
3. 一旦他们完成预订，其等待列表条目状态将更新为 **已转为预订**

## 小贴士

- 对于高价值预订（例如摄影会话、私人活动），启用手动确认，以便在承诺之前检查可用性并匹配需求。
- 初始设置时请慷慨设置缓冲时间 — 一旦了解实际的周转需求，您可以随时减少缓冲时间。
- 对于团体课程，请将 **每个时段最大预订数** 设置为课程容量，并启用等待列表，这样热门课程会自动建立队列。
- 对于住宿产品，请使用日期范围选择器显示模式 — 客户期望一起选择到达和离开日期。
- 设置最低提前通知时间，以防止最后一刻的预订，如果您需要准备时间（例如定制餐饮订单的48小时最低提前通知）。
- 在旺季期间定期查看您的等待列表 — 手动联系等待列表中的客户可以比自动通知更快地填补取消的预订。
