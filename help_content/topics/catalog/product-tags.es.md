---
title: Etiquetas de productos
---

Las etiquetas de productos son etiquetas cortas y de libre disposición que puedes adjuntar a productos para una organización y filtrado flexibles. En comparación con las categorías, las etiquetas son ligeras — no tienen una jerarquía ni páginas dedicadas, pero son rápidas de crear y fáciles de aplicar a muchos productos a la vez.

Usos comunes de etiquetas:
- Agrupar productos por tema (`summer`, `gift-idea`, `clearance`)
- Marcar productos para propósitos internos (`staff-pick`, `new-arrival`, `bundle-candidate`)
- Crear colecciones filtrables en promociones y campañas
- Encontrar rápidamente productos relacionados en diferentes categorías

## Crear una etiqueta

Las etiquetas se crean automáticamente cuando las agregas a un producto, pero también puedes gestionarlas directamente:

1. Navega a **Catalogo > Etiquetas de productos**
2. Haz clic en **+ Añadir etiqueta de producto**
3. Ingresa el **Nombre** de la etiqueta (por ejemplo, `New Arrival`)
4. El **Slug** se genera automáticamente a partir del nombre (`new-arrival`) — puedes editarlo si es necesario
5. Haz clic en **Guardar**

Los slugs de las etiquetas deben ser únicos. El slug se usa internamente y en URLs cuando se usan etiquetas para el filtrado en el catálogo.

## Añadir etiquetas a un producto

1. Navega a **Productos > Todos los productos**
2. Abre el producto que deseas etiquetar
3. Encuentra el campo **Etiquetas** en el formulario de edición del producto
4. Escribe para buscar una etiqueta existente o escribe el nombre de una nueva etiqueta para crearla en el acto
5. Selecciona o crea tantas etiquetas como necesites
6. Guarda el producto

## Gestionar etiquetas existentes

### Ver todas las etiquetas

La lista de etiquetas en **Catalogo > Etiquetas de productos** muestra todas las etiquetas en orden alfabético. Haz clic en cualquier etiqueta para editar su nombre o slug.

### Renombrar una etiqueta

1. Navega a **Catalogo > Etiquetas de productos**
2. Haz clic en la etiqueta que deseas renombrar
3. Actualiza el campo **Nombre**
4. El slug no se actualiza automáticamente al renombrar — actualízalo manualmente si es necesario para mantener la coherencia
5. Haz clic en **Guardar**

Nota: al renombrar una etiqueta, se actualiza en todos los lugares donde se use, ya que los productos hacen referencia al mismo objeto de etiqueta.

### Eliminar una etiqueta

1. Navega a **Catalogo > Etiquetas de productos**
2. Marca la casilla junto a la(s) etiqueta(s) que deseas eliminar
3. Selecciona **Eliminar etiquetas de productos seleccionadas** del menú desplegable **Acción**
4. Confirma la eliminación

Eliminar una etiqueta la elimina de todos los productos a los que se le asignó. Los productos no se eliminan — solo se elimina la etiqueta.

## Consejos

- Mantén los nombres de las etiquetas cortos y consistentes — usa minúsculas o Mayúsculas en el título y elige una convención y manténla en todo tu catálogo.
- Evita duplicar la lógica de categorías en etiquetas. Las etiquetas funcionan mejor para preocupaciones transversales (como `new-arrival` o `staff-pick`) que no encajan bien en tu estructura de categorías.
- Usa una etiqueta `clearance` o `sale` para marcar productos para promociones con plazo limitado — facilita encontrar y actualizar esos productos más tarde.
- Por defecto, las etiquetas no son visibles para los clientes. Su propósito principal es ayudarte a organizar y filtrar productos en el administrador.
- Si te encuentras creando muchas etiquetas que se superponen con tus categorías, podría valer la pena revisar tu estructura de categorías en lugar de eso.
