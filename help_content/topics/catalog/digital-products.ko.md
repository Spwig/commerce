title: 디지털 제품
---

디지털 제품은 다운로드 가능한 파일, 소프트웨어 라이선스 및 기타 물리적이지 않은 상품을 판매하는 데 사용됩니다. Spwig은 독립형 디지털 제품은 물론 물리적 및 디지털 배송을 결합한 하이브리드 제품도 지원합니다.

![라이선스 제공업체](/static/core/admin/img/help/digital-products/license-providers.webp)

## 디지털 제품 유형

### 독립형 디지털 제품

**제품 유형**을 **디지털 제품**으로 설정하여 순수히 디지털인 항목을 만드세요:
- 소프트웨어 애플리케이션
- 전자책 및 PDF
- 음악 및 오디오 파일
- 디지털 예술 및 템플릿

### 하이브리드 제품

**기본 정보** 탭에서 **디지털 제품** 체크박스를 선택함으로써 모든 제품 유형에 디지털 배송을 추가할 수 있습니다. 이는 다음과 같은 경우에 유용합니다:
- **변동형 디지털 제품** — 기본/프로/엔터프라이즈 버전을 포함한 소프트웨어
- **사용자 정의 디지털 제품** — 사용자 정의 디자인의 디지털 자산
- **물리적 + 디지털 번들** — 디지털 다운로드를 포함한 책

## 디지털 제품 설정

### 단계 1: 제품 생성

1. **제품 > 모든 제품**으로 이동하고 **+ 제품 추가**를 클릭하세요
2. **제품 유형**을 **디지털 제품**으로 설정하거나 다른 제품 유형에서 **디지털 제품** 체크박스를 선택하세요
3. 제품 세부 정보(이름, 설명, 가격)를 입력하세요
4. 제품을 저장하세요

### 단계 2: 다운로드 가능한 파일 추가

1. 제품의 **재고** 탭으로 이동하세요
2. **디지털 파일** 섹션에서 고객이 구매 후 받을 파일을 업로드하세요
3. 각 파일에 대해 다음을 설정할 수 있습니다:
   - **파일 이름** — 고객에게 표시되는 이름
   - **다운로드 제한** — 파일이 다운로드될 수 있는 최대 횟수 (0 = 무제한)
   - **유효 기간** — 다운로드 링크가 활성화되는 일수

### 단계 3: 라이선스 배송 구성 (선택 사항)

디지털 제품이 라이선스 키를 필요로 한다면:

1. **설정 > 라이선스 관리**로 이동하세요
2. 라이선스 제공업체를 연결하세요 (아래 참조)
3. 제품 편집 폼에서 라이선스 제공업체를 할당하세요

## 라이선스 제공업체

라이선스 제공업체는 고객이 제품을 구매할 때 자동으로 소프트웨어 라이선스 키를 생성하고 관리하는 외부 서비스입니다.

### 사용 가능한 제공업체 유형

| 제공업체 | 설명 |
|----------|-------------|
| **Spwig 내장 라이선스 서버** | 플랫폼 내에 구축된 간단한 라이선스 키 생성 |
| **Keygen.sh** | 기능이 완비된 라이선스 관리 API |
| **LicenseSpring** | 기업용 라이선스 관리 |
| **Cryptlex** | 오프라인 지원을 포함한 소프트웨어 라이선스 |
| **사용자 정의 API** | REST API를 통해 어떤 라이선스 시스템도 연결할 수 있습니다 |

### 라이선스 제공업체 연결

1. **설정 > 라이선스 관리**로 이동하세요
2. **제공업체 연결**을 클릭하세요
3. 설정 마법사에 따라 다음을 수행하세요:
   - **단계 1** — 제공업체 유형 선택
   - **단계 2** — 일반 설정 구성
   - **단계 3** — API 인증 정보 입력
4. 연결을 테스트하여 작동 여부를 확인하세요
5. 구성 내용을 저장하세요

### 제공업체 카드

각 연결된 제공업체는 다음을 표시합니다:
- **상태 배지** — 활성/비활성 및 연결 상태
- **API 엔드포인트** — 구성된 서버 URL
- **동기화 기능** — 주문, 활성화 및 비활성화 동기화 지원
- **액션 버튼** — 구성, 테스트 및 즉시 동기화

### 동기화 기능

라이선스 제공업체는 세 가지 이벤트에 대해 동기화할 수 있습니다:

- **주문** — 고객이 구매를 완료할 때 자동으로 라이선스 키 생성
- **활성화** — 고객이 라이선스를 활성화할 때 추적
- **비활성화** — 환불 또는 이전을 위한 라이선스 비활성화 처리

## 고객 경험

### 구매 후

고객이 디지털 제품을 구매할 때:

1. **주문 확인** — 디지털 배송이 포함됨을 표시
2. **이메일 배송** — 다운로드 링크 및/또는 라이선스 키가 자동으로 전송됨
3. **계정 페이지** — 고객은 계정 대시보드에서 다운로드를 액세스할 수 있음
4. **다운로드 페이지** — 보안 및 시간 제한이 있는 다운로드 링크

### 다운로드 보안

디지털 파일 다운로드는 다음으로 보호됩니다:
- 고유하고 시간 제한이 있는 다운로드 토큰
- 선택적 다운로드 횟수 제한
- 링크가 비활성화되는 만료일
- 로그인 요구사항 (등록된 고객에게)

## 팁

- 합리적인 다운로드 제한(3~5회)을 설정하여 악용을 방지하면서도 재다운로드를 허용하세요.
- 지원 기간과 일치하는 만료일 수를 사용하세요 (예: 1년의 액세스를 위해 365일).
- 테스트 주문으로 전체 구매 흐름을 테스트하여 다운로드 링크 및 라이선스 키가 올바르게 전송되는지 확인하세요.
- 소프트웨어 제품의 경우, 라이선스 제공업체를 연결하여 키 생성을 자동화하고 수동으로 관리하는 대신 사용하세요.
- 물리적 상품에 디지털 추가 기능(예: 인쇄된 책 + PDF)을 포함할 때 하이브리드 제품 기능을 사용하세요.
