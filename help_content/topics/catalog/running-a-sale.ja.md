---
title: セールの実施
---

このガイドでは、プロモーションシステムを使用して、最も一般的なセールの種類を設定する方法を説明します。サイト全体の割引、特定のカテゴリセール、または時間限定のフラッシュセールが必要な場合、これらのステップバイステップの例に従って、プロモーションを迅速に実施できます。

![サイト全体のセール設定](/static/core/admin/img/help/running-a-sale/sitewide-sale-setup.webp)

## サイト全体の割引セール

サイト全体の割引セールは、最もシンプルで人気のあるプロモーションタイプです。たとえば、「すべての商品20％OFF」です。これは5ステップのプロモーションウィザードを使用して設定できます。

1. **マーケティング > セール＆プロモーション**に移動し、**プロモーションの作成**をクリックします。
2. **ステップ1 — 基本情報**: 明確な名前を入力してください。たとえば、「サマーセール — すべての商品20％OFF」。内部参照用のオプションの説明を追加します。**有効**のチェックボックスをONのままにしておくと、プロモーションは開始日から自動的に有効になります。
3. **ステップ2 — 割引**: 割引タイプとして**百分率割引**を選択し、値として**20**を入力します。
4. **ステップ3 — 日程**: セール期間の**開始日**と**終了日**を設定します。プロモーションはこれらの日付に自動的に有効化および無効化されます。
5. **ステップ4 — 商品**: **すべての商品**（デフォルト）を選択します。これにより、カタログ内のすべての商品が割引を適用されます。
6. **ステップ5 — 確認**: 名前、割引タイプ、割引率、日程、商品範囲のすべての設定を確認します。すべてが正しいように見える場合は、**プロモーションの作成**をクリックします。

サイト全体のセールは今すぐスケジュールされ、開始日が到達すると自動的に適用されます。

## カテゴリセール

カテゴリセールは、特定の部門または商品グループを対象にします。たとえば、「アウター30％OFF」です。これは、季節限定のセールや部門限定のプロモーションに最適です。

サイト全体のセールと同じウィザード手順に従いますが、ステップ4に1つの重要な違いがあります。

1. **ステップ4 — 商品**で、**すべての商品**ではなく**カテゴリ**を選択します。
2. **利用可能なカテゴリ**パネルが表示され、検索フィールドが含まれています。セールに含めたいカテゴリ（例: 「アウター」、「ジャケット」）を検索して追加します。
3. これらのカテゴリに含まれる商品のみが割引を適用されます。

このアプローチは、季節終わりのセール、部門限定のプロモーション、または特定の商品ラインの強調表示に適しています。

### 適切なターゲティングの選択

プロモーションシステムは、いくつかのターゲティングオプションを提供しています。この表を使用して、どのオプションがセールに適しているかを判断してください。

| ターゲティング | 最適な用途 |
|----------------|------------|
| **カテゴリ** | 部門全体のセール（例: すべての靴、すべての電子機器） |
| **ブランド** | ブランド限定のプロモーション（例: 「ナイキウィーク」） |
| **コレクション** | カスタムグループ（例: 「サマーエッセンシャルズ」、「ギフトアイデア」） |
| **商品** | 手選された個々の商品 |

## フラッシュセール

フラッシュセールは、短時間の高割引プロモーションで、緊急性を生み出すことを目的としています。通常、24〜48時間で40〜50％OFFです。

1. 標準のウィザード手順に従って新しいプロモーションを作成します。
2. **ステップ2 — 割引**で、魅力的な割引（例: 40％または50％OFF）を設定します。
3. **ステップ3 — 日程**で、きつい日付の範囲を設定します。たとえば、金曜日の00:00から土曜日の23:59まで。
4. **高度な設定**で、高い**優先度**値（例: 10）を設定します。これにより、フラッシュセールは、同じ商品に適用される可能性のある他の進行中の低優先度プロモーションを上書きします。
5. フラッシュセールを**アニュアンスバー**（マーケティング > アニュアンスの下）と組み合わせて、ストアフロント全体にカウントダウンまたは緊急性のメッセージを表示します。

### フラッシュセールのヒント

- **短時間に保つ** — 最大24〜48時間。希少性は行動を促します。
- **意味のある割引を提供する** — フラッシュセールは、通常のプロモーションと比べて特別であるべきです。
- **チャネルを横断して宣伝する** — メール、ソーシャルメディア、ストアフロントのアニュアンスバーを同時に使用して宣伝してください。
- **在庫を監視する** — 高割引のフラッシュセールは在庫を素早く動かす可能性があるため、在庫レベルを常に確認してください。

## 季節および祝日セール

ブラックフライデー、クリスマス、サマーセールなどの主要なショッピングイベントは、事前の計画と、場合によっては複数のプロモーションが必要です。

### 事前計画

イベントの数日前または数週間前から、季節限定のプロモーションの開始日を設定して、事前に計画してください。これにより、プロモーションのマーケティング資料の準備、テスト、調整に時間を確保できます。

### プロモーションの重ね合わせ

最大の影響を与えるために、複数のプロモーションを重ね合わせることを検討してください。例えば:

- **サイト全体の20％OFF**プロモーションを、**優先度5**で作成します。
- **40％OFFクリアランス**カテゴリプロモーションを、**優先度10**で作成します。

優先度の高いカテゴリプロモーションはクリアランス商品に適用され、サイト全体のプロモーションはそれ以外の商品に適用されます。これにより、特定の商品に深い割引を提供できますが、広範なセールから手動で除外する必要はありません。

### 一般的な季節セールカレンダー

年間のセールカレンダーを計画する際の出発点として、この表を使用してください。

| イベント | 一般的なタイミング | 一般的な割引 |
|----------|------------------|------------|
| **バレンタインデー** | 2月1日〜14日 | 15〜25％ |
| **春のセール** | 3月〜4月 | 20〜30％ |
| **夏のセール** | 6月〜7月 | 20〜40％ |
| **学校用品の再開** | 8月〜9月 | 15〜25％ |
| **ブラックフライデー** | 11月の最終金曜日 | 30〜50％ |
| **サイバーモンデー** | ブラックフライデーの次の月曜日 | 25〜40％ |
| **クリスマス** | 12月1日〜24日 | 20〜35％ |
| **新年** | 12月26日〜1月2日 | 30〜50％ |

## 個別商品のセール

1つの商品を迅速に割引したい場合は、プロモーションを作成する代わりに、商品レベルのセールを使用できます。これは、商品編集フォームに直接組み込まれています。

1. **商品 > すべての商品**に移動し、セールをかけたい商品をクリックします。
2. **価格設定**タブを開きます。
3. **セール**セクションで、以下の設定を行います:
   - **セールタイプ** — パーセンテージ割引、金額割引、または固定価格から選択します。
   - **セール値** — 割引額または割引率を入力します。
   - **開始日 / 終了日** — セールの開始と終了日を任意でスケジュールします。これらを空にして、即時で無期限のセールにします。
4. 商品を保存します。

商品レベルのセールは、設定が簡単で迅速ですが、1つの商品にのみ適用されます。

### 各アプローチの使用場面

| アプローチ | 最適な用途 |
|----------|------------|
| **プロモーション** | 複数の商品、カテゴリ、またはブランドを一度に |
| **商品セール** | 1つの商品の迅速な割引 |

複数の商品を割引したい場合は、プロモーションが常に最適な選択肢です。これにより、割引を1か所にまとめたままにし、管理が容易になります。

## ヒント

- **事前に計画する** — プロモーションを事前にスケジュールし、直前に有効にするのではなく、すべてが正しく動作することを確認する時間を持ちましょう。
- **まずテストする** — ウィザードの確認ステップで表示される影響を受ける商品数を確認して、プロモーションが正しい商品を対象としていることを確認した後、有効にしてください。
- **明確な名前を使用する** — 「サマーセール20％OFF」は、「プロモーション1」や「テスト割引」よりも後で簡単に見つけて管理できます。
- **完了したら削除する** — 期限切れのプロモーションを無効または削除して、プロモーションダッシュボードを清潔に保ち、混乱を避けてください。
- **宣伝する** — アニュアンスバーとメールキャンペーンと組み合わせて、プロモーションを最大限に見せましょう。世界最高の割引でも、顧客がそれを知らない場合、何の効果もありません。
