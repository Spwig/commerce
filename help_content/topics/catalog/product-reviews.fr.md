---
title: Avis sur les produits
---

Les avis sur les produits permettent aux clients de noter et de commenter leur expérience avec un produit. Les avis approuvés par vous apparaissent sur la page du produit dans votre boutique en ligne, où ils aident d'autres clients à prendre des décisions d'achat. Spwig vous donne un contrôle total sur les avis qui sont visibles — rien n'est publié avant que vous ne l'approuviez.

Accédez à **Catalogue > Avis sur les produits** pour gérer tous les avis.

## La liste des avis

La liste des avis vous donne un aperçu de tous les avis soumis dans votre boutique :

| Colonne | Description |
|---|---|
| **Produit** | Le produit qui est évalué |
| **Utilisateur** | Le client qui a soumis l'avis |
| **Note** | Note en étoiles de 1 à 5 |
| **Approuvé** | Indique si l'avis est actuellement visible sur la boutique en ligne |
| **Achat vérifié** | Indique si le client a réellement acheté le produit |
| **Créé le** | Lorsque l'avis a été soumis |

### Filtre des avis

Utilisez les filtres à droite pour vous concentrer sur des états d'avis spécifiques :

- **Note** — affichez uniquement les avis avec une note d'étoile spécifique (utile pour enquêter sur les plaintes à 1 étoile)
- **Approuvé** — séparez rapidement les avis approuvés des avis en attente
- **Achat vérifié** — filtrez pour voir uniquement les avis provenant d'acheteurs confirmés

Utilisez le champ de recherche pour trouver des avis par nom de produit, nom d'utilisateur du client ou titre de l'avis.

## Approuver et rejeter des avis

Les avis ne sont pas visibles sur votre boutique en ligne avant d'être approuvés. Vous pouvez approuver ou rejeter des avis individuellement ou en masse.

### Actions en masse

1. Dans la liste des avis, cochez les cases à côté des avis sur lesquels vous souhaitez agir
2. Sélectionnez **Approuver les avis sélectionnés** ou **Rejeter les avis sélectionnés** dans le menu déroulant **Action**
3. Cliquez sur **Aller**

C'est la méthode la plus efficace pour traiter un lot d'avis nouveaux.

### Avis individuel

1. Cliquez sur une ligne d'avis pour ouvrir la vue détaillée
2. Cochez ou décochez le champ **Approuvé**
3. Cliquez sur **Enregistrer**

## Ce qu'un avis contient

Lorsque vous ouvrez un avis individuel, vous pouvez voir tous les détails soumis par le client :

- **Titre** — le titre que le client a donné à son avis
- **Commentaire** — le texte complet de l'avis
- **Note** — la note en étoiles (1–5)
- **Achat vérifié** — défini automatiquement lorsque le client qui évalue a un ordre terminé pour ce produit
- **Nombre d'utilitaires** — le nombre de fois où d'autres clients ont marqué cet avis comme utile
- **Images** — toute photo que le client a attachée à son avis

Vous ne pouvez pas modifier le contenu d'un avis — vous ne pouvez qu'approuver ou rejeter l'avis.

## Aperçu des notes

La liste des avis est triée par ordre chronologique, avec les avis les plus récents en premier, ce qui facilite le suivi des retours récents. Si vous souhaitez repérer des tendances rapidement :

- **Filtrez par Note = 1** pour voir tous les avis critiques dans une seule liste — ceux-ci révèlent souvent des problèmes de produit ou de livraison dignes d'être résolus
- **Filtrez par Approuvé = Non** pour voir votre file d'attente de modération — nettoyez régulièrement cette file pour éviter que les clients n'attendent trop longtemps avant que leurs avis soient publiés
- **Filtrez par Achat vérifié = Oui** pour voir uniquement les avis provenant de vrais acheteurs, qui tendent à être les plus crédibles pour d'autres clients

## Conseils

- Fixez un créneau régulier pour vérifier la file d'attente d'approbation — les clients remarquent quand leur avis met des semaines à apparaître, ce qui peut décourager les retours futurs
- Approuver rapidement les avis, y compris les critiques, renforce la confiance avec vos clients ; des avis négatifs visibles sans réponse peuvent sembler pires qu'une plainte traitée
- Utilisez le filtre **Achat vérifié** lors de la prise de décision sur les avis ambigus — les retours des acheteurs confirmés ont plus de poids dans tout litige
- Un produit avec de nombreux avis à 1 étoile mentionnant tous le même problème est un signal clair pour enquêter sur l'emballage, la qualité du produit ou votre fiche produit
- Les avis ne peuvent pas être modifiés, donc si un avis contient du contenu inapproprié, votre seule option est de le rejeter — soyez cohérent dans l'application de vos standards de modération
