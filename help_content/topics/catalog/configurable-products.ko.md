---
title: 구성 가능한 제품
---

구성 가능한 제품은 고객이 다양한 구성 슬롯에서 옵션을 선택하여 자신만의 제품을 만들 수 있도록 합니다. 이는 커스텀 PC, 맞춤형 선물 상자, 주문 제작 가구와 같은 주문 제작 항목에 이상적입니다. 각 구성 요소는 고객의 카탈로그에 있는 실제 제품입니다.

![제품 구성 관리자](/static/core/admin/img/help/configurable-products/product-configurator.webp)

## 작동 방식

구성 가능한 제품은 **슬롯**(선택 범주)과 **옵션**(고객이 선택할 수 있는 실제 제품)으로 구성됩니다. 예를 들어, 커스텀 PC는 프로세서, 그래픽 카드, RAM, 저장소와 같은 슬롯을 가질 수 있으며, 각 슬롯에는 선택 가능한 여러 제품 옵션이 포함됩니다.

## 가격 전략

최종 가격의 계산 방식을 선택하세요:

| 전략 | 설명 |
|----------|-------------|
| **성분의 합** | 최종 가격 = 선택된 모든 옵션 가격의 총합. 기본 가격이 필요하지 않습니다. |
| **기본 가격 + 조정** | 제품의 기본 가격에서 시작한 후 각 옵션에 따라 가격을 추가하거나 뺍니다. |
| **고정 가격** | 고객이 어떤 옵션을 선택하든 항상 동일한 고정 가격입니다. |

## 구성 가능한 제품 설정

### 단계 1: 제품 생성

1. **제품 > 모든 제품**으로 이동하고 **+ 제품 추가**를 클릭합니다.
2. **제품 유형**을 **구성 가능한 제품**으로 설정합니다.
3. **가격 전략**을 선택합니다(성분의 합이 가장 일반적입니다).
4. 제품 이름, 설명 및 기타 기본 정보를 입력합니다.
5. 제품을 저장합니다.

### 단계 2: 구성 슬롯 추가

제품을 저장한 후 **구성** 탭으로 이동하여 슬롯을 설정합니다.

1. **+ 슬롯 추가**를 클릭하여 새로운 구성 범주를 생성합니다.
2. 각 슬롯에 대해 구성합니다:
   - **이름** - 고객이 보는 이름(예: "프로세서", "색상")
   - **아이콘** - 시각적 식별을 위한 Font Awesome 아이콘 클래스
   - **필수** - 고객이 선택을 해야 하는지 여부
   - **최소/최대 선택 수** - 고객이 선택할 수 있는 옵션 수(기본값: 정확히 1개)
   - **정렬 순서** - 구성 마법사에서 슬롯이 나타나는 순서를 제어합니다.

### 단계 3: 각 슬롯에 옵션 추가

각 슬롯에는 고객이 선택할 수 있는 제품 옵션이 필요합니다:

1. 슬롯의 **옵션 관리**를 클릭합니다.
2. 카탈로그에서 기존 제품을 검색하고 추가합니다.
3. 각 옵션에 대해 구성합니다:
   - **가격 조정** - 기본 가격 + 조정 전략과 함께 추가하거나 뺄 금액
   - **기본** - 구성 관리자가 로드될 때 이 옵션을 미리 선택합니다.
   - **인기** - 고객이 결정을 도와주는 "인기" 배지 표시
   - **수량** - 이 구성 요소가 포함된 단위 수
   - **호환성 태그** - 일괄 호환성 규칙 생성에 사용되는 태그

**팁:** 구성 요소 제품은 구성 관리자에서만 사용할 수 있도록 **상점에서 숨기기**를 체크하여 상점에서 숨길 수 있습니다. 이렇게 하면 제품 카탈로그가 복잡해지지 않으면서도 구성 옵션으로 사용할 수 있습니다.

### 단계 4: 호환성 규칙 정의

호환성 규칙은 고객이 호환되지 않는 조합을 선택하는 것을 방지합니다:

| 규칙 유형 | 설명 |
|-----------|-------------|
| **필요** | 옵션 A가 선택되면 목표 슬롯에서 나열된 옵션만 사용 가능합니다. |
| **제외** | 옵션 A가 선택되면 목표 슬롯에서 나열된 옵션을 숨깁니다. |

규칙을 추가하려면:

1. **구성** 탭의 **호환성 규칙** 섹션으로 스크롤합니다.
2. **+ 규칙 추가**를 클릭합니다.
3. **소스 옵션**(트리거)을 선택합니다.
4. **규칙 유형**(필요 또는 제외)을 선택합니다.
5. **목표 슬롯**과 **영향을 받는 옵션**을 선택합니다.

옵션에 할당된 호환성 태그에서 규칙을 자동으로 생성할 수도 있습니다. 이는 많은 조합을 관리할 때 더 빠르게 작업할 수 있습니다.

### 단계 5: 사전 설정(선택 사항)

사전 설정은 고객에게 빠른 시작점을 제공하는 사전 구성된 설정입니다:

1. **구성 사전 설정** 섹션으로 스크롤합니다.
2. **+ 사전 설정 추가**를 클릭합니다.
3. 사전 설정에 이름과 설명을 입력합니다(예: "게임 빌드", "예산 기초")
4. 각 슬롯의 옵션을 선택합니다.
5. 미리보기 이미지를 업로드하거나 **대표**로 표시할 수 있습니다.

고객은 사전 설정에서 시작한 후 개인적인 슬롯을 선호에 맞게 커스터마이징할 수 있습니다.

## 고객 경험

고객이 상점에서 구성 가능한 제품을 보는 경우:

1. **마법사 인터페이스** - 슬롯을 단계로 제시하여 고객이 각 선택을 안내합니다.
2. **필터링** - 호환성 규칙에 따라 호환되지 않는 옵션이 자동으로 숨겨집니다.
3. **인기 배지** - 인기로 표시된 옵션은 결정을 도와주는 배지가 표시됩니다.
4. **사전 설정** - 대표 사전 설정은 빠른 시작 옵션으로 나타납니다.
5. **가격 업데이트** - 옵션이 선택될 때마다 총 가격이 실시간으로 업데이트됩니다.
6. **요약** - 장바구니에 추가하기 전에 선택된 모든 옵션을 보여주는 검토 단계입니다.

## 팁

- "성분의 합" 가격 전략부터 시작하세요. 이는 고객에게 가장 직관적이고 유지가 가장 쉽습니다.
- 호환성 규칙을 사용하여 무효한 구성이 발생하지 않도록 하세요. 고객의 지식에 의존하지 마세요.
- 가장 인기 있는 구성에 대해 2-3개의 사전 설정을 생성하여 결정 피로를 줄이세요.
- 구성 관리자에서만 사용되어야 한다면 구성 요소 제품을 상점에서 숨기세요.
- 설정 후 프론트엔드에서 전체 구성 흐름을 테스트하여 모든 규칙이 예상대로 작동하는지 확인하세요.
