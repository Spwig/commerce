# การเพิ่มสินค้า

คู่มือนี้จะพาคุณผ่านขั้นตอนการสร้างสินค้าใหม่ในร้านของคุณ สินค้าถูกจัดระเบียบผ่านแท็บหลายแท็บ — ข้อมูลพื้นฐาน, สื่อ, การกำหนดราคา, สต็อกสินค้า, และ SEO — เพื่อให้คุณสามารถกรอกข้อมูลทั้งหมดในครั้งเดียว หรือกลับมาเพิ่มเติมในภายหลังได้

## เริ่มต้น

จากแถบข้างขวามือ ให้ไปที่ **Products > All Products** เพื่อดูแคตตาล็อกสินค้าของคุณ คลิกปุ่ม **+ Add Product** ที่มุมขวาบนเพื่อเปิดแบบฟอร์มสร้างสินค้า

![หน้ารายการสินค้า](/static/core/admin/img/help/add-product/product-list-page.webp)

## แท็บข้อมูลพื้นฐาน

แท็บ **ข้อมูลพื้นฐาน** เป็นสถานที่ที่คุณกำหนดรายละเอียดหลักของสินค้า

![แบบฟอร์มเพิ่มสินค้า](/static/core/admin/img/help/add-product/add-product-form.webp)

### ฟิลด์ที่จำเป็น

- **ชื่อ** — ชื่อสินค้าที่แสดงให้ลูกค้าเห็น คลิกไอคอนโลกเพื่อเพิ่มการแปลสำหรับภาษาอื่น
- **Slug** — รูปแบบชื่อที่เป็นมิตรกับ URL (สร้างอัตโนมัติ) ปิดการใช้งาน "Auto" เพื่อปรับแต่งเอง
- **SKU** — รหัสหน่วยสต็อกภายในของคุณ
- **ประเภทสินค้า** — เลือกจาก: Simple, Variable, Digital, Bundle, Gift Card, Customizable, หรือ Configurable
- **สถานะ** — ตั้งเป็น Draft ขณะทำงาน แล้วเปลี่ยนเป็น Published เมื่อพร้อม

### ฟิลด์ที่ไม่จำเป็น

- **หมวดหมู่** — จัดประเภทสินค้าเพื่อการจัดระเบียบและนำทางในร้านค้า
- **แบรนด์** — ผูกกับแบรนด์หากจำเป็น
- **แสดงเป็นสินค้าแนะนำ** — ตรวจสอบเพื่อเน้นสินค้าในร้านค้า
- **เป็นสินค้าดิจิทัล** — ตรวจสอบหากสินค้ามีการดาวน์โหลดดิจิทัล (ไฟล์, ใบอนุญาต)
- **ซ่อนจากหน้าร้าน** — ซ่อนสินค้าจากรายการแคตตาล็อก แต่ยังคงใช้งานได้เป็นตัวเลือกในการกำหนดค่าหรือส่วนประกอบของแพ็กเกจ

### การบรรยายสินค้า

- **การบรรยายสั้น** — ปรากฏในรายการสินค้าและบัตร รักษาให้กระชับและน่าสนใจ
- **การบรรยายแบบเต็ม** — การบรรยายรายละเอียดที่แสดงในหน้ารายละเอียดสินค้า ใช้ตัวแก้ไขข้อความแบบอัดแน่นเพื่อเพิ่มการจัดรูปแบบ รูปภาพ วิดีโอ และตาราง

ทั้งสองฟิลด์การบรรยายสนับสนุนฟีเจอร์การแปล — คลิกไอคอนโลกเพื่อเพิ่มเนื้อหาในภาษาอื่น

## แท็บสื่อ

แท็บ **สื่อ** ช่วยให้คุณจัดการรูปภาพสินค้าโดยใช้ไลบรารีสื่อที่บูรณาการไว้

![แท็บสื่อ](/static/core/admin/img/help/add-product/media-tab.webp)

1. คลิก **+ Add Images from Media Library** เพื่อเปิดตัวเลือกสื่อ
2. เลือกภาพที่มีอยู่หรืออัปโหลดภาพใหม่โดยตรง
3. ลากภาพเพื่อจัดเรียงใหม่ — **ภาพแรก** จะกลายเป็นภาพหลักที่แสดงในรายการและบัตร
4. เลือก **ประเภทแกลเลอรี** เพื่อควบคุมวิธีแสดงภาพบนหน้าสินค้า: Standard Gallery, Carousel, Grid Layout, Zoom Gallery, หรือ 360° View

## แท็บการกำหนดราคา

ตั้งราคาสินค้าและกำหนดการขาย

![แท็บการกำหนดราคา](/static/core/admin/img/help/add-product/pricing-tab.webp)

### การกำหนดราคาทั่วไป

- **ราคาปกติ** — ราคาขายปลีกมาตรฐานที่ลูกค้าจะเห็น
- **สกุลเงิน** — เลือกสกุลเงิน (สกุลเงินเริ่มต้นของร้านคุณถูกเลือกไว้ล่วงหน้า)
- **ต้นทุน** — ต้นทุนของสินค้า ใช้สำหรับคำนวณกำไร ไม่เคยแสดงให้ลูกค้าเห็น

### การตั้งค่าการขาย

กำหนดส่วนลดชั่วคราว:

- **ประเภทการขาย** — เลือกจาก: No Sale, Fixed Sale Price, Amount Off, หรือ Percentage Off
- **ค่าการขาย** — จำนวนเงินหรือเปอร์เซ็นต์ของส่วนลด
- **วันที่เริ่มต้น/สิ้นสุด** — กำหนดเวลาที่การขายเริ่มและสิ้นสุด ปล่อยว่างเพื่อเริ่มทันทีหรือไม่มีวันสิ้นสุด

## แท็บสต็อกสินค้า

จัดการระดับสต็อกและคุณสมบัติของสินค้าทางกายภาพ

![แท็บสต็อกสินค้า](/static/core/admin/img/help/add-product/inventory-tab.webp)

### การจัดการสต็อก

- **ติดตามสต็อก** — เปิดใช้งานเพื่อติดตามปริมาณสต็อก (เปิดใช้งานเป็นค่าเริ่มต้น)
- **ค่าล่างสต็อก** — ได้รับการแจ้งเตือนเมื่อสต็อกลดลงต่ำกว่าจำนวนนี้ (ค่าเริ่มต้น: 5)
- **ปริมาณสต็อก** — จำนวนหน่วยที่มีอยู่ทั้งหมด
- **อนุญาตให้สั่งซื้อเมื่อสินค้าหมด** — เปิดใช้งานเพื่อรับคำสั่งซื้อแม้สินค้าจะหมด

### คุณสมบัติทางกายภาพ

กรอกน้ำหนัก (กิโลกรัม) และขนาด (ความยาว ความกว้าง ความสูง ในเซนติเมตร) ของสินค้าเพื่อคำนวณการจัดส่งที่แม่นยำ

### รหัสตัวตนของสินค้า

รหัสสินค้ามาตรฐานสำหรับการอัปโหลดในตลาดและระบบสต็อก:

- **GTIN** — Global Trade Item Number
- **EAN** — European Article Number
- **UPC** — Universal Product Code (US)
- **ISBN** — สำหรับหนังสือ
- **ASIN** — รหัสอ้างอิงของ Amazon
- **MPN** — รหัสชิ้นส่วนของผู้ผลิต

### การจัดส่งระหว่างประเทศ / ศุลกากร

จำเป็นสำหรับการจัดส่งระหว่างประเทศ:

- **HS Code** — รหัสการจัดประเภทระบบ Harmonized
- **ประเทศแหล่งกำเนิด** — ประเทศที่สินค้าถูกผลิต
- **ราคาต่อหน่วยสำหรับศุลกากร** — ค่าที่ประกาศต่อหน่วยสำหรับศุลกากร

## แท็บ SEO

ปรับปรุงการมองเห็นของสินค้าในเครื่องมือค้นหา

![แท็บ SEO](/static/core/admin/img/help/add-product/seo-tab.webp)

- **ชื่อเมตา** — ชื่อที่แสดงในผลการค้นหาของเครื่องมือค้นหา คลิกไอคอนโลกเพื่อแปล
- **คำอธิบายเมตา** — คำอธิบายสั้นสำหรับผลการค้นหา (สูงสุด 160 ตัวอักษร) คลิกไอคอนโลกเพื่อแปล
- **สร้างเนื้อหา SEO อัตโนมัติ** — ตรวจสอบเพื่อสร้างเนื้อหา SEO อัตโนมัติเมื่อสินค้าถูกบันทึก

**ตัวอย่างผลการค้นหาแบบเรียลไทม์** แสดงให้เห็นว่าสินค้าของคุณจะปรากฏในผลการค้นหาของ Google อย่างไร

## การบันทึกสินค้าของคุณ

เมื่อคุณพร้อมแล้ว ให้ใช้ปุ่มบันทึกที่มุมขวาบน:

- **บันทึก** (เครื่องหมายถูก) — บันทึกและอยู่ในหน้าสินค้า
- **บันทึกและดำเนินการต่อ** — บันทึกและยังคงอยู่ในแบบฟอร์มเพื่อทำงานต่อ

สินค้าของคุณจะปรากฏบนหน้าร้านเมื่อสถานะถูกตั้งเป็น **Published**

## เคล็ดลับ

- เริ่มต้นด้วยสถานะ **ร่าง** เพื่อให้คุณปรับปรุงสินค้าก่อนที่ลูกค้าจะเห็น
- อัปโหลดรูปภาพหลายภาพ — สินค้าที่มีรูปภาพหลายภาพมีประสิทธิภาพในการแปลงดีกว่า
- กรอกฟิลด์ **SEO** เพื่อเพิ่มโอกาสในการค้นพบผ่านเครื่องมือค้นหา
- ใช้ **หมวดหมู่** และ **แบรนด์** เพื่อช่วยให้ลูกค้านำทางแคตตาล็อกของคุณได้ง่ายขึ้น
- สำหรับสินค้าแบบ Variable (เช่น ขนาดหรือสีต่างกัน) เลือกประเภท **สินค้าแบบ Variable** และเพิ่มตัวแปรหลังจากบันทึก
