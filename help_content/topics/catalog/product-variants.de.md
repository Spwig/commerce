title: Produktvarianten
---

Produktvarianten ermöglichen es Ihnen, ein einzelnes Produkt in mehreren Optionen anzubieten — wie verschiedene Groessen, Farben oder Materialien — jeweils mit eigenem SKU, Preis und Lagerbestand. Navigieren Sie zu einem beliebigen **Variablen Produkt** und klicken Sie auf den Tab **Variationen**.

![Produktvarianten](/static/core/admin/img/help/product-variants/product-variants.webp)

## Varianten verstehen

Ein **Variables Produkt** ist ein Produkttyp, der mehrere Variationen unterstuetzt. Zum Beispiel koennte ein T-Shirt verfuegbar sein in:
- **Farben**: Blau, Rot, Gruen
- **Groessen**: S, M, L, XL

Jede Kombination (z. B. "Blau / Gross") wird zu einer separaten Variante mit eigenem Inventar und Preis.

## Ein Variables Produkt einrichten

### Schritt 1: Produkttyp festlegen

1. Oeffnen Sie das Produktbearbeitungsformular (oder erstellen Sie ein neues Produkt)
2. Setzen Sie im Tab **Basisinformationen** den **Produkttyp** auf **Variables Produkt**
3. Speichern Sie das Produkt

### Schritt 2: Attribute definieren

Attribute sind die Optionen, die Ihre Varianten unterscheiden (z. B. Groesse, Farbe).

1. Gehen Sie zum Tab **Variationen**
2. Klicken Sie im Bereich **Produktattribute** auf **+ Attribut hinzufuegen**, um ein vorhandenes Attribut zuzuweisen, oder auf **Neu erstellen**, um ein neues zu definieren
3. Geben Sie fuer jedes Attribut die verfuegbaren Werte an (z. B. Klein, Mittel, Gross)

### Schritt 3: Varianten erstellen

1. Klicken Sie im Bereich **Produktvarianten** auf **+ Neue Variante hinzufuegen**
2. Konfigurieren Sie jede Variante:
   - **Name** — Beschreibende Bezeichnung (z. B. "Blau", "Gross / Rot")
   - **SKU** — Eindeutiger Lagerhaltungscode
   - **Preis** — Variantenspezifischer Preis (kann vom Basisprodukt abweichen)
   - **Lagerbestand** — Aktueller Inventarstand
3. Wiederholen Sie den Vorgang fuer jede benoetigte Variante

## Varianten verwalten

### Variantendetails

Jede Variantenkarte zeigt:
- **Name** und **SKU** — Identifikationsinformationen
- **Preis** — Aktueller Verkaufspreis
- **Lagerbestand** — Verfuegbare Menge mit Statusanzeige (Auf Lager / Niedriger Bestand / Nicht vorraetig)

Klicken Sie auf eine Variantenkarte, um alle Details zu erweitern und zu bearbeiten.

### Variantenspezifische Einstellungen

Jede Variante kann eigene Einstellungen haben:

| Einstellung | Beschreibung |
|-------------|-------------|
| **Preis** | Den Preis des Basisprodukts ueberschreiben |
| **Vergleichspreis** | Einen Angebotspreis mit Durchstreichung anzeigen |
| **SKU** | Eindeutiger Identifikator fuer das Inventar |
| **Lagerbestand** | Unabhaengige Bestandsverfolgung |
| **Gewicht** | Fuer Versandberechnungen |
| **Bild** | Variantenspezifisches Produktbild |

### Eine Variante bearbeiten

1. Klicken Sie auf das **Bearbeitungssymbol** auf der Variantenkarte
2. Aendern Sie die gewuenschten Felder
3. Klicken Sie auf **Speichern**, um zu aktualisieren

### Eine Variante loeschen

1. Klicken Sie auf das **Loeschsymbol** auf der Variantenkarte
2. Bestaetigen Sie die Loeschung

**Hinweis:** Das Loeschen einer Variante entfernt ihren Inventareintrag. Diese Aktion kann nicht rueckgaengig gemacht werden.

## Attribute

### Was sind Attribute?

Attribute sind wiederverwendbare Optionsdefinitionen. Sobald Sie ein Attribut wie "Groesse" mit den Werten "S, M, L, XL" erstellt haben, koennen Sie es jedem variablen Produkt zuweisen.

### Attribute erstellen

1. Klicken Sie im Tab Variationen auf **Neu erstellen** im Bereich Produktattribute
2. Geben Sie den Attributnamen ein (z. B. "Farbe")
3. Fuegen Sie Werte hinzu (z. B. "Rot", "Blau", "Gruen")
4. Speichern Sie das Attribut

### Attribute zuweisen

Attribute koennen mehreren Produkten zugewiesen werden. Das gleiche Attribut "Groesse" kann fuer T-Shirts, Hosen und Schuhe verwendet werden.

## Anzeige im Shop

Im Shop zeigen variable Produkte:
- Optionsauswahl (Dropdowns oder Farbfelder) fuer jedes Attribut
- Automatische Preisaktualisierungen bei Auswahl einer Variante
- Lagerverfuegbarkeit pro Variante
- Variantenspezifische Bilder

## Tipps

- Verwenden Sie einheitliche Attributnamen ueber alle Produkte hinweg fuer ein konsistentes Einkaufserlebnis.
- Richten Sie alle Attribute ein, bevor Sie Varianten erstellen, um den Prozess zu optimieren.
- Laden Sie variantenspezifische Bilder hoch, damit Kunden genau sehen koennen, was sie bestellen.
- Halten Sie SKUs systematisch (z. B. "TSHIRT-BLAU-G") fuer eine einfache Bestandsverwaltung.
- Nutzen Sie den Vergleichspreis bei Varianten, um groessen- oder farbspezifische Angebote durchzufuehren.
