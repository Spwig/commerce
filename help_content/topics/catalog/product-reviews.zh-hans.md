---
title: 产品评论
---

产品评论允许客户对产品进行评分并撰写使用体验。您批准的评论将显示在商店前端的产品页面上，帮助其他客户做出购买决策。Spwig 允许您完全控制哪些评论可见 — 在您批准之前，没有任何评论会被发布。

导航到 **目录 > 产品评论** 来管理所有评论。

## 评论列表

评论列表为您提供商店中所有提交评论的概览：

| 列 | 描述 |
|---|---|
| **产品** | 正在被评论的产品 |
| **用户** | 提交评论的客户 |
| **评分** | 1 到 5 星的评分 |
| **已批准** | 评论当前是否在商店前端可见 |
| **已验证购买** | 评论者是否实际购买了该产品 |
| **创建时间** | 评论提交的时间 |

### 过滤评论

使用右侧的筛选器来聚焦特定的评论状态：

- **评分** — 仅显示具有特定星级评分的评论（用于调查 1 星投诉很有用）
- **是否已批准** — 快速区分已批准和待处理的评论
- **是否已验证购买** — 筛选仅查看已确认购买者的评论

使用搜索字段通过产品名称、客户用户名或评论标题查找评论。

## 批准和拒绝评论

在您批准之前，评论不会显示在商店前端。您可以单独或批量批准或拒绝评论。

### 批量操作

1. 在评论列表中，勾选您要操作的评论旁边的复选框
2. 从 **操作** 下拉菜单中选择 **批准所选评论** 或 **拒绝所选评论**
3. 点击 **执行**

这是处理一批新评论的最有效方法。

### 单个评论

1. 点击评论行以打开详细视图
2. 勾选或取消勾选 **是否已批准** 字段
3. 点击 **保存**

## 评论包含的内容

当您打开单个评论时，您可以查看客户提交的所有详细信息：

Preserve all markdown formatting, image paths, code blocks, and technical terms.

- **标题** — 客户为其评论提供的标题
- **评论** — 完整的书面评论内容
- **评分** — 星级评分（1–5）
- **已验证购买** — 当评论该产品的客户已完成订单时，系统会自动设置
- **有用次数** — 其他客户将此评论标记为有用的次数
- **图片** — 客户附加到评论中的任何照片

你不能编辑评论内容 — 你只能批准或拒绝它。

## 一目了然地查看评分

评论列表按最新评论排序，便于你及时掌握新提交的反馈。如果你想快速发现模式：

- **按评分 = 1 过滤**，以查看所有负面评论 — 这些评论通常会揭示产品问题或履约问题，值得解决
- **按已批准 = 否 过滤**，以查看你的审核队列 — 定期清理此队列，以免客户等待太久才能看到他们的评论
- **按已验证购买 = 是 过滤**，以仅查看实际买家的评论 — 这些评论对其他客户来说通常最具可信度

## 小贴士

- 设置定期检查审核队列的时间表 — 客户会注意到他们的评论需要数周时间才能显示，这可能会减少未来的反馈
- 及时批准评论，包括负面评论，可以与客户建立信任；可见的负面评论如果没有回应，可能比处理过的投诉看起来更糟糕
- 在决定如何处理边缘评论时，使用 **已验证购买** 过滤器 — 来自已确认买家的反馈在任何争议中都更有分量
- 如果一个产品有很多 1 星评论，且都提到了同样的问题，这显然是一个信号，表明需要调查包装、产品质量或产品列表
- 评论无法编辑，因此如果评论包含不当内容，你唯一的选择是拒绝它 — 在应用审核标准时要保持一致
