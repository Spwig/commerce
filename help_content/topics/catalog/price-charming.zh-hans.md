---
title: 价格魅力规则
---

价格魅力（也称为心理定价）会自动调整您的产品价格，使其以特定的数字结尾，这些数字对客户来说更具吸引力。例如，价格不再是 $20.00，而是显示为 $19.99 —— 这是一种广泛使用的技术，可以让价格在视觉上显得更低。

Spwig 会根据您店铺接受的每种货币自动应用价格魅力规则，因此您只需为每条规则设置一次即可。

## 价格魅力的工作原理

当计算产品价格（包括促销或折扣后）时，Spwig 会检查该货币是否有活动的价格魅力规则。如果有的话，价格会在显示给客户之前进行调整。此调整适用于高于您选择的最低阈值的价格。

您可以为店铺接受的每种货币配置单独的规则。例如，您可能对 USD 使用 `.99` 结尾，但对 JPY 则四舍五入到最近的 `¥10`。

## 创建价格魅力规则

1. 导航到 **目录 > 价格魅力规则**
2. 点击 **+ 添加价格魅力规则**
3. 选择此规则适用的 **货币**（例如，`USD`、`EUR`、`NZD`）
4. 选择一个 **规则类型**（请参阅下表）
5. 可选地设置 **最低价格阈值**，以排除非常低的价格
6. 如果您还想在商品促销时应用价格魅力，请勾选 **应用于促销价格**
7. 确保勾选 **启用**
8. 点击 **保存**

每种货币只能存在一条规则。如果您需要更改规则，请编辑现有的规则。

## 规则类型


| 规则类型 | 示例 | 最适合 |
  |-----------|---------|----------|
  | **魅力 .99 结尾** | $20.50 → $19.99 | 大多数零售产品 — 经典的心理定价 |
  | **魅力 .95 结尾** | $20.50 → $19.95 | .99 的稍微柔和替代方案 |
  | **魅力 .90 结尾** | $20.50 → $19.90 | 感觉圆整但仍低于下一美元 |
  | **向下取整** | $19.50 → $19.00 | 偏好整数的商店 |
  | **向上取整** | $19.50 → $20.00 | 为了整洁显示稍微向上取整 |
  | **四舍五入到最接近的 5** | $23.00 → $25.00 | 高流量零售和市场 |
  | **四舍五入到最接近的 10** | $23.00 → $20.00 | 价格较高的商品，如家电 |
  | **四舍五入到最接近的 100** | $1,234 → $1,200 | 高价值商品，如家具或电子产品 |
  | **自定义结尾** | 任意 — 在下方指定 | 当您的品牌使用特定结尾，如 `.88` 时 |

  ### 自定义结尾

  如果您选择 **自定义结尾**，在 **自定义结尾** 字段中输入结尾值。例如，输入 `0.88` 使所有价格以 `.88` 结尾（在某些亚洲市场中常见）。

  ## 最小价格阈值

  使用 **最小价格阈值** 字段跳过对非常低价商品的美化处理，因为在这些商品上调整会显得奇怪。例如，将阈值设置为 `5.00` 表示价格低于 $5 的商品将显示其实际计算价格，不进行美化处理。

  保留为 `0` 以对所有价格应用美化处理。

  ## 促销价格

  默认情况下，美化处理会同时应用于常规价格和促销价格。如果您希望促销价格显示其精确计算值（在限时促销中精确数字很重要时有用），请取消勾选 **应用于促销价格**。

  ## 禁用规则

  要临时停止美化处理而不删除规则，请取消勾选 **启用** 并保存。规则将被保留，并可在任何时候重新启用。

  ## 小贴士

  保留所有 markdown 格式、图片路径、代码块和专业术语。

- 如果不确定，可以以 `.99` 结尾 —— 这是最广泛认可的心理定价技巧，适用于大多数产品类型。
- 如果你销售的是低成本商品（低于5美元），请设置一个最低阈值，以防止3.50美元的商品被降至2.99美元。
- 在启用新规则后，通过查看商店前台的产品来检查价格 —— 精灵价格会实时显示。
- 日元和类似以整数为单位的货币最适合使用 **四舍五入到最近的10** 或 **四舍五入到最近的100**，因为小数结尾看起来不太寻常。
- 价格精灵会在所有折扣和促销之后应用，因此你的促销价格也会显示为精灵价格，除非你取消勾选 **应用于促销价格**。
- 你可以为不同的货币设置不同类型的规则，这对于向具有不同定价惯例的多个市场销售时非常有用。
