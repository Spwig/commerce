title: ตัวอย่างการส่งเสริมการขาย

คู่มือนี้แสดงตัวอย่างที่ชัดเจนว่าการตั้งค่าประเภทการส่งเสริมการขายต่าง ๆ ทำได้อย่างไร ตัวอย่างแต่ละตัวประกอบด้วยค่าฟิลด์ที่ถูกต้องที่ต้องกรอกในเครื่องมือช่วยสร้างการส่งเสริมการขาย เพื่อให้คุณสามารถทำตามหรือปรับใช้กับร้านค้าของคุณได้

![Promotion Card](/static/core/admin/img/help/promotion-examples/promotion-card.webp)

## ตัวอย่าง: ลดราคาเป็นเปอร์เซ็นต์ในหมวดหมู่

**สถานการณ์:** ลด 30% สำหรับรองเท้าทั้งหมดในช่วงลดราคาฤดูหนาว

ไปที่ **การตลาด > การขายและการส่งเสริมการขาย** และคลิก **+ สร้างการส่งเสริมการขาย** ใส่ค่าต่อไปนี้ในแต่ละขั้นตอนของเครื่องมือช่วยสร้าง:

| Step | Field | Value |
|------|-------|-------|
| Basics | Name | Winter Clearance — 30% Off Shoes |
| Basics | Description | End-of-season clearance for all footwear |
| Basics | Active | Checked |
| Discount | Type | Percentage Off |
| Discount | Value | 30 |
| Schedule | Start Date | Jan 15, 2026 |
| Schedule | End Date | Feb 28, 2026 |
| Products | Apply To | Categories |
| Products | Selected | Shoes, Boots, Sandals |

นี่จะสร้างการขายที่มีระยะเวลาจำกัด ซึ่งลดราคาอัตโนมัติสำหรับสินค้าทุกชิ้นในหมวดหมู่ที่เลือก คู่รองเท้าราคา 120 ดอลลาร์จะกลายเป็น 84 ดอลลาร์ และคู่รองเท้าสันสูงราคา 60 ดอลลาร์จะกลายเป็น 42 ดอลลาร์

## ตัวอย่าง: ลดราคาเป็นจำนวนคงที่ในคอลเลกชัน

**สถานการณ์:** ลด 15 ดอลลาร์สำหรับสินค้าในคอลเลกชัน Summer Essentials

| Step | Field | Value |
|------|-------|-------|
| Basics | Name | Summer Essentials — $15 Off |
| Basics | Active | Checked |
| Discount | Type | Amount Off |
| Discount | Value | 15.00 |
| Schedule | Start Date | Jun 1, 2026 |
| Schedule | End Date | (empty — no expiration) |
| Products | Apply To | Collections |
| Products | Selected | Summer Essentials |

> **หมายเหตุ:** การลดราคา 15 ดอลลาร์จะใช้กับสินค้าที่เข้าเงื่อนไขแต่ละชิ้นแยกกัน ผลิตภัณฑ์ราคา 50 ดอลลาร์จะกลายเป็น 35 ดอลลาร์ ผลิตภัณฑ์ราคา 30 ดอลลาร์จะกลายเป็น 15 ดอลลาร์ การทิ้งวันหมดอายุว่างไว้หมายความว่าการส่งเสริมการขายจะดำเนินต่อไปอย่างไม่มีที่สิ้นสุดจนกว่าคุณจะปิดใช้งานมันด้วยตนเอง

## ตัวอย่าง: ราคาขายคงที่สำหรับการลดราคา

**สถานการณ์:** ตั้งค่าสินค้าทั้งหมดที่ลดราคาให้เป็น 9.99 ดอลลาร์

| Step | Field | Value |
|------|-------|-------|
| Basics | Name | Final Clearance — Everything $9.99 |
| Basics | Active | Checked |
| Discount | Type | Fixed Sale Price |
| Discount | Value | 9.99 |
| Schedule | Start Date | (today) |
| Products | Apply To | Collections |
| Products | Selected | Final Clearance |

> **หมายเหตุ:** ราคาขายคงที่กำหนดราคาขายที่แน่นอนไม่ว่าราคาเดิมจะเป็นเท่าไร สินค้าราคา 75 ดอลลาร์และสินค้าราคา 25 ดอลลาร์จะกลายเป็น 9.99 ดอลลาร์ทั้งคู่ ใช้การตั้งค่านี้สำหรับชั้นลดราคาหรือราคาที่เท่ากันทุกชิ้นที่คุณต้องการให้สินค้าทุกชิ้นอยู่ในจุดราคาเดียวกัน

![Category Promotion](/static/core/admin/img/help/promotion-examples/category-promotion.webp)

## การเลือกประเภทการลดราคาที่เหมาะสม

| Type | How It Works | Best For | Example |
|------|-------------|----------|---------|
| **Percentage Off** | ลดราคาตามเปอร์เซ็นต์ | การขายที่กว้างขวางเมื่อสินค้ามีราคาแตกต่างกัน | ลด 20% — 100 ดอลลาร์กลายเป็น 80 ดอลลาร์, 50 ดอลลาร์กลายเป็น 40 ดอลลาร์ |
| **Amount Off** | ลบจำนวนเงินคงที่ | การส่งเสริมการขายที่มีข้อความลดราคาเป็นเงินจำนวนเฉพาะ | ลด 15 ดอลลาร์ — 100 ดอลลาร์กลายเป็น 85 ดอลลาร์, 50 ดอลลาร์กลายเป็น 35 ดอลลาร์ |
| **Fixed Sale Price** | ตั้งราคาขายที่แน่นอน | การลดราคา, ราคาคงที่, "ทุกอย่างราคา X" | 9.99 ดอลลาร์ — สินค้าทั้งหมดกลายเป็น 9.99 ดอลลาร์ไม่ว่าราคาเดิมจะเป็นเท่าไร |

## การเลือกเป้าหมายที่เหมาะสม

| Target | How It Works | Best For |
|--------|-------------|----------|
| **All Products** | ใช้กับสินค้าทั้งหมดในร้านค้าของคุณ | การขายทั่วเว็บไซต์, เหตุการณ์ทั่วร้านค้า |
| **Categories** | ใช้กับสินค้าทั้งหมดในหมวดหมู่ที่เลือก | การขายตามแผนก, การลดราคาตามฤดูกาลตามประเภท |
| **Brands** | ใช้กับสินค้าทั้งหมดจากแบรนด์ที่เลือก | การร่วมมือกับแบรนด์, เหตุการณ์เฉพาะแบรนด์ |
| **Collections** | ใช้กับสินค้าทั้งหมดในคอลเลกชันที่เลือก | การส่งเสริมการขายที่คัดสรร, การขายตามธีม |
| **Products** | ใช้กับสินค้าที่เลือกโดยตรง | การขายที่คัดสรร, การเลือกจำกัด |

## รูปแบบการกำหนดเวลา

รูปแบบทั่วไปสามแบบสำหรับการตั้งค่าตารางเวลาการส่งเสริมการขาย:

| Pattern | Start Date | End Date | Use Case |
|---------|-----------|----------|----------|
| **Immediate, ongoing** | Today | (empty) | การลดราคาที่ถาวร, การขายระยะยาว |
| **Date range** | Future date | Future date | เหตุการณ์ตามฤดูกาล, การขายช่วงเทศกาล |
| **Future start, no end** | Future date | (empty) | การกำหนดราคาถาวรใหม่ที่เริ่มต้นในวันที่เฉพาะเจาะจง |

การตั้งค่าวันเริ่มต้นในอนาคตจะสร้างการส่งเสริมการขายที่กำหนดเวลาไว้ มันจะปรากฏในแท็บ **Scheduled** บนแดชบอร์ดการส่งเสริมการขาย และจะเปิดใช้งานอัตโนมัติเมื่อวันที่ถึง ความว่างของวันหมดอายุหมายความว่าการส่งเสริมการขายยังคงเปิดใช้งานจนกว่าคุณจะปิดใช้งานมันด้วยตนเอง

## เคล็ดลับ

- **ใช้ชื่อที่อธิบายได้ชัดเจน** — ใส่ค่าการลดราคาและเป้าหมายในชื่อ (เช่น "Summer 20% Off Shoes") เพื่อให้คุณสามารถระบุการส่งเสริมการขายได้อย่างรวดเร็วบนแดชบอร์ด
- **ตรวจสอบจำนวนสินค้าที่ได้รับผลกระทบ** — ขั้นตอนการตรวจสอบแสดงจำนวนสินค้าที่จะถูกลดราคา หากจำนวนดูผิดปกติ ให้ย้อนกลับไปตรวจสอบการกำหนดเป้าหมายของคุณ
- **เริ่มจากขนาดเล็ก** — หากคุณไม่แน่ใจเกี่ยวกับการลดราคา ให้เริ่มด้วยเปอร์เซ็นต์ที่เล็กกว่าและเพิ่มขึ้นหากจำเป็น
- **ใช้การลดราคาเป็นจำนวนเงินสำหรับการตลาด** — การลดราคา 15 ดอลลาร์เป็นการประหยัดที่ชัดเจนซึ่งง่ายต่อการสื่อสารในโฆษณาและแคมเปญอีเมล
- **ใช้การลดราคาเป็นเปอร์เซ็นต์เพื่อความยุติธรรม** — การลดราคาเปอร์เซ็นต์จะปรับตามราคา ให้การประหยัดที่เหมาะสมกับจุดราคาต่าง ๆ
