---
title: 상품 태그
---

상품 태그는 상품에 붙일 수 있는 짧고 자유로운 형식의 라벨로, 유연한 조직화 및 필터링에 사용됩니다. 카테고리에 비해 태그는 계층 구조나 전용 페이지가 없지만, 생성과 적용이 빠르고 여러 상품에 한 번에 적용할 수 있습니다.

태그의 일반적인 사용 사례:
- 테마별로 상품 그룹화 (`summer`, `gift-idea`, `clearance`)
- 내부 용도로 상품 표시 (`staff-pick`, `new-arrival`, `bundle-candidate`)
- 프로모션 및 캠페인에서 필터 가능한 컬렉션 생성
- 다른 카테고리에 걸쳐 관련 상품을 빠르게 찾기

## 태그 생성

상품에 태그를 추가할 때 자동으로 생성되지만, 직접 관리할 수도 있습니다:
1. **Catalog > Product Tags**로 이동
2. **+ Add Product Tag**를 클릭
3. 태그 **이름**을 입력합니다 (예: `New Arrival`)
4. **Slug**는 이름에서 자동으로 생성됩니다 (`new-arrival`) — 필요 시 편집할 수 있습니다
5. **Save**를 클릭

태그 슬러그는 고유해야 합니다. 슬러그는 내부적으로 사용되며, 상품 필터링을 위해 스토어프론트에서 태그가 사용될 때 URL에도 사용됩니다.

## 상품에 태그 추가

1. **Products > All Products**로 이동
2. 태그를 추가할 상품을 열어주세요
3. 상품 편집 폼에서 **Tags** 필드를 찾습니다
4. 기존 태그를 검색하거나 새 태그 이름을 입력하여 즉시 생성할 수 있습니다
5. 필요한 만큼 태그를 선택하거나 생성합니다
6. 상품을 저장합니다

## 기존 태그 관리

### 모든 태그 보기

**Catalog > Product Tags**에 있는 태그 목록은 알파벳 순으로 표시됩니다. 태그를 클릭하여 이름 또는 슬러그를 편집할 수 있습니다.

### 태그 이름 변경

1. **Catalog > Product Tags**로 이동
2. 이름을 변경할 태그를 클릭합니다
3. **Name** 필드를 업데이트합니다
4. 이름을 변경할 때 슬러그는 자동으로 업데이트되지 않습니다. 일관성을 유지하려면 필요 시 수동으로 업데이트합니다
5. **Save**를 클릭합니다

참고: 태그 이름을 변경하면 모든 사용처에서 업데이트됩니다. 이는 상품이 동일한 태그 객체를 참조하기 때문입니다.

### 태그 삭제

1.

**Catalog > Product Tags**로 이동
2.

삭제하고자 하는 태그 옆의 확인란을 선택합니다.
3.

**Action** 드롭다운에서 **선택한 제품 태그 삭제**를 선택합니다.
4.

삭제를 확인합니다.

태그를 삭제하면 해당 태그가 할당된 모든 제품에서 제거됩니다. 제품은 삭제되지 않으며, 오직 태그 라벨만이 제거됩니다.

## 팁

- 태그 이름은 짧고 일관되게 유지하세요 — 소문자 또는 Title Case를 사용하고, 카탈로그 전반에 걸쳐 하나의 규칙을 선택하여 그대로 사용하세요.
- 카테고리 논리를 태그에 중복하지 않도록 하세요. 태그는 카테고리 구조에 잘 맞지 않는 교차적 관심사(예: `new-arrival` 또는 `staff-pick`)에 가장 효과적으로 사용됩니다.
- 시간 제한된 프로모션을 위해 제품을 표시하려면 `clearance` 또는 `sale` 태그를 사용하세요 — 나중에 해당 제품을 쉽게 찾고 업데이트할 수 있습니다.
- 태그는 기본적으로 고객에게 보이지 않습니다. 주요 목적은 관리자에서 제품을 정리하고 필터링하는 데 도와주는 것입니다.
- 자신이 만드는 태그가 카테고리와 중복되는 경우가 많다면, 대신 카테고리 구조를 다시 검토하는 것이 좋습니다.
