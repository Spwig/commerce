---
title: Коллекции товаров
---

Коллекции позволяют группировать товары для отображения на вашем интернет-магазине. В отличие от категорий — которые организуют ваш каталог в постоянную иерархию — коллекции являются гибкими, курированными группами, которые вы создаете для конкретной цели. Коллекция может выделять новые поступления, демонстрировать товары для сезонной акции или представлять отобранный набор хитов продаж.

Перейдите в **Каталог > Коллекции**, чтобы управлять своими коллекциями.

## Коллекции vs категории

Обе категории и коллекции группируют товары, но они служат разным целям:

|  | Категории | Коллекции |
|---|---|---|
| **Цель** | Постоянная структура каталога | Гибкие, курированные группы |
| **Иерархия** | Да — вложенная структура родитель/дочерний | Нет — плоские группы |
| **Количество товаров в группе** | Каждый товар принадлежит к одной категории | Товар может появляться в нескольких коллекциях |
| **Типичное применение** | Навигационное меню магазина, поиск по отделам | Страницы привлечения, акции, выделенные наборы |

Используйте категории для "того, как организован ваш магазин", а коллекции для "того, что вы хотите выделить прямо сейчас".

## Типы коллекций

При создании коллекции выберите тип, который соответствует тому, как вы хотите управлять списком товаров:

| Тип | Как добавляются товары |
|---|---|
| **Ручной выбор** | Вы выбираете конкретные товары, по одному |
| **Автоматические правила** | Товары добавляются автоматически на основе критериев, которые вы определяете |
| **Рекомендуемые товары** | Курированный редакционный выбор, управляемый вручную |
| **Сезонные** | Выбор, основанный на времени, обычно управляемый вручную для акций |

Типы "Ручной выбор" и "Рекомендуемые товары" дают вам точный контроль. Автоматические коллекции могут расти вместе с вашим каталогом без постоянного обслуживания.

## Создание коллекции

1. Перейдите в **Каталог > Коллекции**
2. Нажмите **+ Добавить коллекцию**
3. Заполните раздел **Базовая информация**:
   - **Название** — имя коллекции, которое будет отображаться на вашем интернет-магазине
   - **Slug** — путь URL для страницы коллекции (автоматически заполняется из названия; вы можете настроить его)
   - **Описание** — описание, отображаемое на странице коллекции интернет-магазина
4. Выберите **Тип коллекции**
5. Добавьте товары:
   - Для типов **Ручной выбор** и **Рекомендуемые товары**: используйте поле **Товары**, чтобы искать и добавлять товары
   - Для типа **Автоматические**: определите критерии в поле **Автоматические критерии**
6. Загрузите изображения:
   - **Изображение** — главное изображение коллекции, используемое на страницах списка и миниатюрных изображениях
   - **Изображение баннера** — более широкое изображение баннера, отображаемое в верхней части страницы коллекции
7. Настройте поля **SEO** (необязательно, но рекомендуется):
   - **Мета-заголовок** — заголовок страницы, отображаемый в результатах поиска
   - **Мета-описание** — описание, отображаемое под заголовком в результатах поиска
8. Установите **Параметры отображения**:
   - **Активна** — управляет тем, отображается ли коллекция на вашем интернет-магазине
   - **Выделенная** — помечает коллекцию для выделенного размещения в вашей теме
   - **Порядок сортировки** — управляет порядком, в котором отображаются коллекции на страницах списка (меньшие числа отображаются первыми)
9. Нажмите **Сохранить**

## Добавление товаров в коллекцию

Для ручных коллекций используйте поле автозаполнения **Товары**, чтобы искать в каталоге и выбирать товары. Вы можете добавить столько товаров, сколько нужно — нет ограничения.

Товары могут принадлежать к нескольким коллекциям одновременно. Например, товар может быть в вашей коллекции "Летняя распродажа" и в коллекции "Хиты продаж" без каких-либо конфликтов.

## Отображение коллекций на вашем интернет-магазине

Каждая коллекция автоматически получает свою собственную страницу по адресу `/collection/{slug}/`. Вы можете связать страницы коллекций с вашим меню навигации, редактором страниц или рекламными баннерами.

Флаг **Выделенная** используется вашей темой для определения, какие коллекции отображаются в выделенных местах — например, сетка выделенных коллекций на главной странице. Проверьте документацию вашей темы, чтобы понять, как именно отображаются выделенные коллекции.

## Управление видимостью коллекций

- **Активен** определяет, доступна ли страница коллекции для общественности.

Неактивная коллекция скрыта от клиентов, но сохраняется в админ-панели, чтобы вы могли включить её снова позже.
- **Порядок сортировки** определяет порядок, в котором коллекции отображаются на страницах списка.

Назначайте меньшие числа коллекциям, которые вы хотите отобразить первыми.

## SEO для коллекций

Каждая коллекция имеет свои собственные поля **Мета-заголовок** и **Мета-описание**. Эти поля определяют, что отображается в результатах поисковых систем, когда кто-то находит вашу страницу коллекции. Если вы оставите эти поля пустыми, ваша тема обычно будет использовать название и описание коллекции по умолчанию.

Хорошие заголовки SEO для коллекций должны быть описательными и конкретными:
- "Летние платья 2026 — цветочные и легкие стили" работает лучше, чем "Летняя коллекция"
- "Спортивные туфли для мужчин — легкие и вентилируемые" работает лучше, чем "Спортивные туфли"

## Советы

- Держите названия коллекций короткими и понятными — они отображаются как заголовки страниц и текстом ссылок в навигации вашего интернет-магазина
- Используйте сезонные или акционные коллекции с планом начала и окончания: создайте коллекцию, включите её, когда кампания начинается, и отключите (вместо удаления), когда она заканчивается, чтобы вы могли использовать её позже
- Поля **Порядок сортировки** стоит настраивать специально — по умолчанию для всех коллекций это 0, что означает сортировку по алфавиту. Назначьте конкретные числа, чтобы контролировать, какие коллекции будут отображаться наиболее заметно
- Коллекция без продуктов покажет пустую страницу клиентам — либо добавьте продукты до активации, либо оставьте коллекцию неактивной, пока она не будет готова
- Проверяйте флаг **Выделить** только для коллекций, которые вы действительно хотите выделить; большинство тем зарезервировали места для выделения для небольшого количества коллекций, и отображение может выглядеть перегруженным, если слишком много коллекций помечены как выделенные
