# 제품 변형

제품 변형을 사용하면 하나의 제품을 다양한 옵션(예: 다른 크기, 색상, 재질)으로 제공할 수 있습니다. 각 변형은 자체 SKU, 가격 및 재고 수준을 가집니다. **가변 상품**으로 이동하여 **변형** 탭을 클릭하세요.

![제품 변형](/static/core/admin/img/help/product-variants/product-variants.webp)

## 변형 이해

**가변 상품**은 여러 변형을 지원하는 제품 유형입니다. 예를 들어, 티셔츠는 다음과 같이 제공될 수 있습니다:
- **색상**: 파랑, 빨강, 녹색
- **크기**: S, M, L, XL

각 조합(예: "파랑 / 대형")은 자체 재고와 가격을 가진 별도의 변형이 됩니다.

## 가변 상품 설정

### 단계 1: 제품 유형 설정

1. 제품 편집 폼을 열거나 새 제품을 생성합니다.
2. **기본 정보** 탭에서 **제품 유형**을 **가변 상품**으로 설정합니다.
3. 제품을 저장합니다.

### 단계 2: 속성 정의

속성은 변형을 구분하는 옵션입니다(예: 크기, 색상).

1. **변형** 탭으로 이동합니다.
2. **제품 속성** 섹션에서 기존 속성을 할당하려면 **+ 속성 추가**를 클릭하거나 새 속성을 정의하려면 **신규 생성**을 클릭합니다.
3. 각 속성에 사용 가능한 값(예: 소, 중, 대)을 지정합니다.

### 단계 3: 변형 생성

1. **제품 변형** 섹션에서 **+ 새 변형 추가**를 클릭합니다.
2. 각 변형을 구성합니다:
   - **이름** — 설명적인 라벨(예: "파랑", "대형 / 빨강")
   - **SKU** — 고유 재고 관리 코드
   - **가격** — 변형별 가격(기본 제품과 다를 수 있음)
   - **재고** — 현재 재고 수준
3. 필요한 모든 변형에 대해 반복합니다.

## 변형 관리

### 변형 세부 정보

각 변형 카드는 다음을 보여줍니다:
- **이름** 및 **SKU** — 식별 정보
- **가격** — 현재 판매 가격
- **재고 수준** — 재고 수량과 상태 지시자(재고 있음 / 재고 부족 / 재고 없음)

변형 카드를 클릭하여 모든 세부 정보를 확장하고 편집합니다.

### 변형별 설정

각 변형은 다음과 같은 자체 설정을 가질 수 있습니다:

| 설정 | 설명 |
|---------|-------------|
| **가격** | 기본 제품 가격을 재정의합니다 |
| **비교 가격** | 취소선으로 표시된 할인 가격을 보여줍니다 |
| **SKU** | 재고 관리용 고유 식별자 |
| **재고 수준** | 독립적인 재고 추적 |
| **무게** | 배송 계산용 |
| **이미지** | 변형별 제품 이미지 |

### 변형 편집

1. 변형 카드의 **편집 아이콘**을 클릭합니다.
2. 원하는 필드를 수정합니다.
3. **저장**을 클릭하여 업데이트합니다.

### 변형 삭제

1. 변형 카드의 **삭제 아이콘**을 클릭합니다.
2. 삭제를 확인합니다.

**참고:** 변형을 삭제하면 해당 재고 기록도 함께 삭제됩니다. 이 작업은 되돌릴 수 없습니다.

## 속성

### 속성이란 무엇인가요?

속성은 재사용 가능한 옵션 정의입니다. "크기"와 같은 속성을 "S, M, L, XL" 값으로 생성한 후, 이 속성을 어떤 가변 제품에든 할당할 수 있습니다.

### 속성 생성

1. 변형 탭에서 **제품 속성** 섹션의 **신규 생성**을 클릭합니다.
2. 속성 이름을 입력합니다(예: "색상")
3. 값(예: "빨강", "파랑", "초록")을 추가합니다.
4. 속성을 저장합니다.

### 속성 할당

속성은 여러 제품에 할당될 수 있습니다. "크기" 속성은 티셔츠, 바지, 신발 등 여러 제품에 사용될 수 있습니다.

## 판매점 표시

판매점에서는 가변 제품이 다음을 보여줍니다:
- 각 속성에 대한 옵션 선택기(드롭다운 또는 샘플)
- 변형이 선택될 때 자동으로 가격이 업데이트됨
- 변형별 재고 가용성
- 변형별 이미지

## 팁

- 제품별로 일관된 속성 이름을 사용하여 일관된 쇼핑 경험을 제공하세요.
- 변형을 생성하기 전에 모든 속성을 설정하여 프로세스를 간소화하세요.
- 고객이 정확히 주문하는 제품을 확인할 수 있도록 변형별 이미지를 업로드하세요.
- 재고 관리를 위해 SKU를 체계적으로 설정하세요(예: "TSHIRT-BLUE-L")
- 변형에 **비교 가격**을 사용하여 크기별 또는 색상별 특별 판매를 진행하세요.
