# 카테고리 관리

카테고리는 제품 카탈로그를 정리하여 고객이 쉽게 둘러보고 제품을 찾을 수 있도록 도와줍니다. 관리자 사이드바에서 **제품 > 카테고리**로 이동하세요.

![카테고리 목록](/static/core/admin/img/help/manage-categories/category-list.webp)

## 카테고리 목록

카테고리 관리 페이지는 다음을 포함한 카드 형식으로 모든 카테고리를 표시합니다:

- **미리보기 이미지** — 카테고리의 시각적 식별자
- **이름 및 슬러그** — 표시 이름과 URL에 사용되는 식별자
- **제품 수** — 이 카테고리에 할당된 제품 수
- **상태** — 게시됨 또는 초안

상단의 **필터 탭**을 사용하여 모든 카테고리, 게시된 카테고리 또는 초안 카테고리를 빠르게 볼 수 있습니다. **검색 바**를 사용하여 이름으로 카테고리를 검색할 수 있습니다.

## 카테고리 생성

1. 상단 오른쪽의 **+ 카테고리 추가**를 클릭합니다
2. 카테고리 세부 정보를 입력합니다:
   - **이름** — 고객이 보는 표시 이름
   - **슬러그** — 이름에서 자동 생성되며 URL에 사용됨
   - **상위 카테고리** — 상위 카테고리로 만들고 싶다면 빈칸을 그대로 두거나, 하위 카테고리를 만들고 싶다면 부모 카테고리를 선택합니다
   - **설명** — 카테고리 페이지에 표시되는 풍부한 텍스트 설명
3. **카테고리 이미지** 업로드 — 네비게이션 메뉴 및 카테고리 목록에서 표시됨
4. SEO 탭에서 **SEO 필드**(메타 제목, 설명)를 구성합니다
5. **저장**을 클릭합니다

## 카테고리 계층 구조

카테고리는 무한히 중첩할 수 있어 트리 구조를 만들 수 있습니다:

- **상위 카테고리** — 메인 네비게이션 항목 (예: "의류", "전자제품")
- **하위 카테고리** — 상위 카테고리 아래에 중첩됨 (예: "의류 > 남성용 > 티셔츠")

상위 카테고리 드롭다운은 전체 계층 경로를 표시하여 적절한 수준을 선택하는 데 도움을 줍니다.

## 카테고리 설정

### 가시성

- **게시됨** — 카테고리가 가맹점과 네비게이션에 표시됨
- **초안** — 고객에게 숨겨졌지만 관리자에서는 접근 가능함

### 추천 카테고리

카테고리를 **추천**으로 표시하여 메인 페이지나 특별한 네비게이션 섹션에서 강조할 수 있습니다. 추천 카테고리는 Page Builder의 카테고리 그리드 요소를 사용하여 표시할 수 있습니다.

### 정렬 순서

**정렬 순서** 값을 설정하여 네비게이션 메뉴에서 카테고리가 어떻게 표시되는지를 제어합니다. 숫자가 낮을수록 먼저 표시됩니다.

## 제품의 카테고리 할당

제품을 할당하는 두 가지 방법이 있습니다:

1. **제품 편집 폼에서** — 기본 정보 탭의 카테고리 드롭다운에서 카테고리를 선택합니다
2. **대량 할당** — 제품 목록에서 여러 제품을 선택하고 대량 작업을 사용하여 특정 카테고리에 할당합니다

각 제품은 하나의 주 카테고리에 속할 수 있습니다. 추가적인 그룹화를 위해 태그 또는 컬렉션을 사용합니다.

## 가맹점의 카테고리 페이지

각 게시된 카테고리는 다음을 표시하는 전용 페이지를 자동으로 받습니다:
- 카테고리 이름 및 설명
- 배너 이미지 (설정된 경우)
- 할당된 모든 제품을 포함한 제품 그리드
- 필터링 및 정렬 옵션

카테고리 페이지 URL은 다음과 같은 패턴을 따릅니다: `yourstore.com/category/category-slug/`

## 팁

- 카테고리 트리를 얕게 유지하세요 — 네비게이션 사용성에 이상적인 깊이는 2-3레벨입니다.
- 고객이 검색하는 것과 일치하는 설명적인 카테고리 이름을 사용하세요.
- 더 시각적인 브라우징 경험을 위해 카테고리 이미지를 추가하세요.
- 제품을 추가하기 전에 카테고리 구조를 설정하여 모든 것을 정리해 두세요.
- 카테고리 설명을 SEO에 활용하세요 — 관련 키워드를 자연스럽게 포함하세요.
