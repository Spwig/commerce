---
title: Добавление товара
---

Это руководство описывает процесс создания нового товара в вашем магазине. Информация о товаре распределена по нескольким вкладкам — Основная информация, Медиа, Цены, Склад и SEO — поэтому вы можете заполнить все сразу или вернуться к незавершённым разделам позже.

## Начало работы

В боковом меню перейдите в раздел **Товары > Все товары**, чтобы открыть каталог товаров. Нажмите кнопку **+ Добавить товар** в правом верхнем углу, чтобы открыть форму создания товара.

![Страница списка товаров](/static/core/admin/img/help/add-product/product-list-page.webp)

## Вкладка «Основная информация»

На вкладке **Основная информация** задаются ключевые характеристики товара.

![Форма добавления товара](/static/core/admin/img/help/add-product/add-product-form.webp)

### Обязательные поля

- **Название** — Название товара, отображаемое покупателям. Нажмите на значок глобуса, чтобы добавить переводы на другие языки.
- **Slug** — URL-совместимая версия названия (генерируется автоматически). Отключите режим «Авто», чтобы задать значение вручную.
- **Артикул (SKU)** — Внутренний код складского учёта.
- **Тип товара** — Выберите один из вариантов: Простой, Вариативный, Цифровой, Набор, Подарочная карта, Настраиваемый или Конфигурируемый.
- **Статус** — Установите «Черновик» на время работы, затем измените на «Опубликован», когда товар будет готов.

### Необязательные поля

- **Категория** — Присвойте товару категорию для удобной организации и навигации по витрине.
- **Бренд** — Укажите бренд, если применимо.
- **Рекомендуемый** — Отметьте, чтобы выделить этот товар на витрине магазина.
- **Цифровой товар** — Отметьте, если товар включает цифровые загрузки (файлы, лицензии).
- **Скрыть с витрины** — Скрывает товар из каталога, но оставляет его доступным в качестве опции конфигуратора или компонента набора.

### Описания товара

- **Краткое описание** — Отображается в карточках и списках товаров. Должно быть лаконичным и привлекательным.
- **Полное описание** — Подробное описание товара, показываемое на странице товара. Используйте визуальный редактор для добавления форматирования, изображений, видео и таблиц.

Оба поля описания поддерживают функцию перевода — нажмите на значок глобуса, чтобы добавить контент на других языках.

## Вкладка «Медиа»

Вкладка **Медиа** позволяет управлять изображениями товара через встроенную медиабиблиотеку.

![Вкладка «Медиа»](/static/core/admin/img/help/add-product/media-tab.webp)

1. Нажмите **+ Добавить изображения из медиабиблиотеки**, чтобы открыть выбор медиафайлов.
2. Выберите существующие изображения или загрузите новые прямо из этого окна.
3. Перетаскивайте изображения для изменения порядка — **первое изображение** становится основным и отображается в списках и карточках товаров.
4. Выберите **тип галереи**, чтобы задать способ отображения изображений на странице товара: Стандартная галерея, Карусель, Сетка, Галерея с увеличением или Обзор 360°.

## Вкладка «Цены»

Настройте цену товара и параметры распродаж.

![Вкладка «Цены»](/static/core/admin/img/help/add-product/pricing-tab.webp)

### Обычная цена

- **Обычная цена** — Стандартная розничная цена, которую увидят покупатели.
- **Валюта** — Выберите валюту (валюта вашего магазина по умолчанию уже выбрана).
- **Себестоимость** — Ваша стоимость закупки, используется для расчёта прибыли. Покупателям не отображается.

### Параметры распродажи

Настройка временных скидок:

- **Тип скидки** — Выберите один из вариантов: Без скидки, Фиксированная цена со скидкой, Скидка на сумму или Процентная скидка.
- **Размер скидки** — Сумма или процент скидки.
- **Дата начала / окончания** — Запланируйте, когда скидка начнёт действовать и когда истечёт. Оставьте пустым для немедленного начала или бессрочного действия.

## Вкладка «Склад»

Управление остатками и физическими характеристиками товара.

![Вкладка «Склад»](/static/core/admin/img/help/add-product/inventory-tab.webp)

### Управление запасами

- **Учёт остатков** — Включите для отслеживания количества на складе (включено по умолчанию).
- **Порог низкого остатка** — Получайте уведомления, когда количество на складе опускается ниже указанного значения (по умолчанию: 5).
- **Количество на складе** — Общее количество доступных единиц.
- **Разрешить предзаказы** — Включите для приёма заказов даже при отсутствии товара на складе.

### Физические характеристики

Укажите вес товара (кг) и габариты (длина, ширина, высота в см) для точного расчёта стоимости доставки.

### Идентификаторы товара

Стандартные коды для маркетплейсов и систем складского учёта:

- **GTIN** — Глобальный номер торговой единицы
- **EAN** — Европейский артикульный номер
- **UPC** — Универсальный код товара (США)
- **ISBN** — Для книг
- **ASIN** — Идентификатор Amazon
- **MPN** — Код производителя

### Международная доставка / Таможня

Обязательно для международных отправлений:

- **Код HS** — Код классификации по Гармонизированной системе
- **Страна происхождения** — Где произведён товар
- **Таможенная стоимость за единицу** — Декларируемая стоимость за единицу для таможни

## Вкладка «SEO»

Оптимизация видимости товара в поисковых системах.

![Вкладка «SEO»](/static/core/admin/img/help/add-product/seo-tab.webp)

- **Мета-заголовок** — Заголовок, отображаемый в результатах поиска. Нажмите на значок глобуса для перевода.
- **Мета-описание** — Краткое описание для результатов поиска (не более 160 символов). Нажмите на значок глобуса для перевода.
- **Автогенерация SEO** — Отметьте, чтобы SEO-контент создавался автоматически при сохранении товара.

Функция **Предпросмотр результатов поиска** показывает, как именно ваш товар будет выглядеть в результатах поиска Google.

## Сохранение товара

Когда всё готово, используйте кнопки сохранения в правом верхнем углу:

- **Сохранить** (галочка) — Сохранить и остаться на странице товара.
- **Сохранить и продолжить редактирование** — Сохранить и продолжить работу с формой.

Товар станет видимым на витрине магазина после установки статуса **Опубликован**.

## Советы

- Начните со статуса **Черновик**, чтобы довести товар до совершенства прежде, чем его увидят покупатели.
- Загружайте несколько изображений — товары с большим количеством фотографий продаются лучше.
- Заполните поля **SEO**, чтобы улучшить обнаружимость товара в поисковых системах.
- Используйте **Категории** и **Бренды**, чтобы помочь покупателям ориентироваться в каталоге.
- Для вариативных товаров (например, разные размеры или цвета) выберите тип **Вариативный товар** и добавьте варианты после сохранения.
