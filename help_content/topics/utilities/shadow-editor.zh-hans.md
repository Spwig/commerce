---
title: 阴影编辑器
---

阴影编辑器允许您通过可配置的框阴影和文字阴影为元素添加深度和维度。阴影可以创建视觉层次结构，吸引人们对重要元素的注意，并为您的商店带来精致、现代的感觉。打开任何元素的 **样式选项卡**，查找 **效果** 组以访问阴影编辑器。

![阴影编辑器](/static/core/admin/img/help/shadow-editor/shadow-editor.webp)

## 阴影类型

编辑器在顶部提供两个选项卡：

- **框阴影** — 在整个元素的边界框周围添加阴影。适用于卡片、按钮、容器、图片和部分。
- **文字阴影** — 仅在文字字符后面添加阴影。适用于标题或叠加在图片上的文字以提高可读性。

每个选项卡都有其独立的配置。如果需要，可以同时为同一元素应用框阴影和文字阴影。

## 阴影属性

每个阴影层由以下属性定义：

| 属性 | 描述 | 范围 |
|----------|-------------|-------|
| **X 偏移量** | 阴影与元素之间的水平距离 | -50px 到 50px |
| **Y 偏移量** | 阴影与元素之间的垂直距离 | -50px 到 50px |
| **模糊半径** | 阴影边缘的柔和或扩散程度。数值越高，阴影越柔和。 | 0px 到 100px |
| **扩散半径** | 相对于元素扩大或缩小阴影的大小（仅限框阴影） | -50px 到 50px |
| **颜色** | 阴影颜色，可通过颜色选择器配置，支持完全透明度 | 任何带有 alpha 的颜色 |
| **内阴影** | 切换以在元素内部而不是外部渲染阴影（仅限框阴影） | 开/关 |

使用滑块调整值，或直接在输入字段中输入精确的数值。

## 多个阴影

您可以在单个元素上叠加多个阴影层，以创建复杂且逼真的深度效果：

- 点击 **+** 按钮添加新的阴影层
- 每个层在阴影列表中显示为一行，并带有自己的控件
- 拖动层以重新排序 — 阴影按照列表顺序渲染，第一个层在最上面
- 点击任何层的 **眼睛图标** 可临时隐藏它，而无需删除配置
- 点击 **垃圾桶图标** 删除一个层

将一个紧密、深色的阴影与一个宽广、柔和的阴影结合，可以创建一种自然的“提升”效果，模仿物理深度。

## 阴影预设

快速应用预设可让您通过单击添加常见的阴影样式：

| 预设 | 描述 |
|--------|-------------|
| **小** | 用于轻微提升的细微、靠近的阴影（卡片、输入框） |
| **中** | 用于交互元素的中等深度（按钮、下拉菜单） |
| **大** | 用于浮动元素的显著阴影（模态框、弹出框） |
| **柔和** | 宽广的模糊和低透明度，以产生柔和、扩散的光晕 |
| **清晰** | 最小的模糊和较高的透明度，以产生清晰、明确的边缘 |
| **内阴影** | 用于创建按下或凹陷外观的内部阴影 |

应用预设后，您可以调整单个属性以微调结果。

## 当前与新预览

在编辑器底部，两个比较框显示 **当前** 阴影（已保存）和 **新** 阴影（您的待定更改）。这种并排视图使您在提交更改之前轻松评估差异。点击 **应用** 以接受，或点击其他地方以丢弃更改。

## 出现位置

阴影编辑器可在以下位置找到：

- **页面构建器** — 在部分、容器、列和单个元素的样式选项卡的 **效果** 组中
- **页眉/页脚构建器** — 在小部件级别的阴影设置中，用于元素如标志、搜索栏和导航项

任何支持 **效果** 样式组的元素都会显示阴影编辑器控件。

## 小贴士

- 对于大多数元素，使用细微的阴影（小或柔和预设） — 过重的阴影会使设计显得杂乱。
- 将一个靠近的深色阴影与一个远处的浅色阴影结合，可以产生最自然的提升效果。
- 内阴影在输入框和容器上效果很好，可以创建“凹陷”面板效果。
- 文字阴影应保持最小 — 1px 偏移和轻微模糊可以提高图像背景上的可读性，而不会显得过时。
- 如果您的主题支持深色模式切换，请在亮色和深色背景上测试您的阴影。
