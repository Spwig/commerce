---
title: Редактор границ
---

Редактор границ предоставляет точный контроль над границами элементов, включая стиль, цвет, ширину по сторонам и радиус закругления углов. Он открывается в виде плавающей панели с прямым превью и двумя вкладками для базовых и расширенных настроек.

![Редактор границ](/static/core/admin/img/help/border-editor/border-editor.webp)

## Прямое превью

Коробка превью в верхней части редактора показывает изменения границ в реальном времени. Внутри прямоугольника с границей отображается слово "Preview", которое мгновенно обновляется при изменении стиля, цвета, ширины и радиуса.

## Базовый и расширенный режимы

Редактор организован в две вкладки:

| Вкладка | Что она содержит |
|--------|------------------|
| **Базовый** | Стиль границы, цвет, ширина (с контролем по сторонам), радиус границы (с контролем по углам) |
| **Расширенный** | Тонкая настройка радиуса отдельных углов и экспериментальное свойство Форма угла |

Большинство работ с границами выполняется полностью в вкладке "Базовый". Вкладка "Расширенный" полезна, когда требуется точный контроль над отдельными углами или когда вы хотите экспериментировать с новыми функциями CSS.

## Стиль границы

Раскрывающийся список с девятью вариантами, которые управляют внешним видом линии границы:

| Стиль | Описание |
|-------|----------|
| **Нет** | Нет границы (удаляет любую существующую границу) |
| **Твердый** | Одиночная непрерывная линия (по умолчанию) |
| **Пунктирный** | Серия коротких тире |
| **Пунктирный** | Серия круглых точек |
| **Двойной** | Две параллельные твердые линии |
| **Глубокий** | Вырезанная граница с эффектом 3D, которая выглядит как вдавленная в поверхность |
| **Выступающий** | Поднятая граница с эффектом 3D (противоположность глубокой) |
| **Внутренний** | Сделает элемент выглядящим встроенным или вдавленным |
| **Внешний** | Сделает элемент выглядящим поднятым или выдвинутым |

Установка стиля "Нет" полностью удаляет границу, независимо от настроек ширины или цвета.

## Цвет границы

Поле ввода текста с кнопкой выбора цвета. Введите шестнадцатеричное значение напрямую (например, `#3b82f6`) или нажмите на цветную панель, чтобы открыть полный выбор цвета с режимами ввода HEX, RGB и HSL, а также визуальной областью цвета. Цвет по умолчанию — черный (`#000000`).

## Ширина границы

Управляет толщиной границы в пикселях. Вкладка "Базовый" показывает четыре отдельных поля ввода для сторон:

| Сторона | Ввод |
|--------|------|
| **Верх** | Числовой ввод, минимальное значение 0 |
| **Право** | Числовой ввод, минимальное значение 0 |
| **Низ** | Числовой ввод, минимальное значение 0 |
| **Лево** | Числовой ввод, минимальное значение 0 |

**Кнопка переключения связей** (иконка цепочки) рядом с меткой управляет тем, связаны ли все четыре стороны:

- **Связано** (по умолчанию) — изменение любого значения обновляет все четыре стороны одновременно
- **Не связано** — каждая сторона может иметь разную ширину, что полезно для эффектов вроде границы только снизу или акцентной границы слева

## Радиус границы

Управляет закруглением каждого угла. Вкладка "Базовый" показывает четыре ввода углов:

| Угол | Метка |
|-----|------|
| **Верхний левый** | TL |
| **Верхний правый** | TR |
| **Нижний левый** | BL |
| **Нижний правый** | BR |

**Кнопка переключения связей** работает так же, как ширина границы:

- **Связано** (по умолчанию) — все четыре угла используют одно и то же значение радиуса
- **Не связано** — каждый угол может иметь разный радиус

Часто используемые значения радиуса:

| Значение | Эффект |
|--------|--------|
| 0px | Острые квадратные углы |
| 4-8px | Незаметное закругление, хорошо подходит для карточек и кнопок |
| 12-16px | Заметное закругление, современный мягкий вид |
| 50% | Полный круг или форма таблетки (в зависимости от размеров элемента) |

Выборник единиц поддерживает px, em, rem и % для значений ширины и радиуса.

## Форма угла (Расширенный)

Вкладка "Расширенный" включает экспериментальное свойство **Форма угла**. Это CSS-функция управляет тем, используют ли закругленные углы стандартную круглую форму или более угловатую форму "сcoop". Поддержка браузеров ограничена, и редактор отображает предупреждение о совместимости, когда текущий браузер не поддерживает это свойство.

## Действия в нижней части

| Кнопка | Действие |
|--------|--------|
| **Сбросить** | Возвращает все значения к состоянию, в котором редактор был открыт |
| **Отменить** | Закрывает редактор без применения изменений |
| **Применить** | Сохраняет настройки границы и закрывает редактор |

## Где он появляется

Редактор границ доступен в нескольких редакторах:

- **Редактор страниц** — выберите любой элемент, откройте вкладку Стиль и нажмите раздел Граница
- **Редактор заголовка/подвал** — добавьте границы к разделам заголовка, контейнерам навигации и областям подвала
- **Редактор меню** — стиль границ для элементов меню и контейнеров выпадающих меню

Редактор считывает текущие вычисленные стили границы из живого элемента на холсте, поэтому он всегда открывается с правильными существующими значениями.

## Советы

- **Используйте границы с умом** — тонкие 1px границы светлого серого цвета создают чистое разделение между разделами без добавления визуального веса.
- **Сочетайте радиус с тенью** — закругленные углы, сочетающиеся с мягким теневым эффектом (через редактор теней), создают эффект аккуратной карточки.
- **Попробуйте односторонние границы** — разорвите связь между сторонами и установите границу только снизу или слева для акцентных линий, разделителей разделов или индикаторов боковой панели.
- **Используйте процентный радиус для таблеток** — установите все углы на 50% для кнопки или значка, чтобы создать форму таблетки, которая адаптируется к любому размеру содержимого.
- **Проверьте превью** — коробка превью обновляется мгновенно, поэтому экспериментируйте свободно перед применением.
