---
title: 枠線エディタ
---

枠線エディタは、要素の枠線に対してスタイル、色、各辺の幅、各角の角丸を細かく制御できる機能を提供します。ライブプレビュー付きのフローティングパネルとして開きます。基本設定と高度な設定の2つのタブが用意されています。

![Border Editor](/static/core/admin/img/help/border-editor/border-editor.webp)

## ライブプレビュー

エディタの上部にあるプレビュー領域では、枠線の変更をリアルタイムで表示します。この領域は「Preview」という文字を含む枠線付きの矩形で、スタイル、色、幅、角丸の値を調整するたびに即座に更新されます。

## 基本モード vs 高度なモード

エディタは2つのタブに分かれています:

| タブ | 内容 |
|-----|-----------------|
| **基本** | 枠線スタイル、色、幅（各辺の制御付き）、角丸（各角の制御付き） |
| **高度な** | 各角の角丸の細かい調整と、実験的な「角の形状」プロパティ |

ほとんどの枠線の編集は基本タブで行います。高度なタブは、個々の角を正確に制御したい場合や、新しいCSS機能を試したい場合に役立ちます。

## 枠線スタイル

9つのオプションを持つドロップダウンで、枠線の外観を制御します:

| スタイル | 説明 |
|-------|-------------|
| **なし** | 枠線なし（既存の枠線を削除） |
| **実線** | 1つの連続した線（デフォルト） |
| **破線** | 短い破線の連続 |
| **点線** | 円形の点の連続 |
| **二重線** | 2本の平行な実線 |
| **溝** | 表面に押し込まれたように見える3D効果の溝状の枠線 |
| **盛り上げ** | 準3D効果の盛り上げた枠線（溝の反対） |
| **内側** | 要素が埋め込まれたように見える |
| **外側** | 要素が浮かび上がったように見える |

スタイルを「なし」に設定すると、幅や色の設定に関係なく、枠線が完全に削除されます。

## 枠線の色

テキスト入力フィールドと色選択ボタンがペアになっています。直接16進数値を入力（例: `#3b82f6`）するか、色スワッチをクリックして、16進数、RGB、HSLの入力モードと視覚的な色選択領域を含むフルカラーピッカーを開くことができます。デフォルトの色は黒（`#000000`）です。

## 枠線の幅

ピクセル単位で枠線の太さを制御します。基本タブでは、4つの辺の入力フィールドが表示されます:

| 辺 | 入力 |
|------|-------|
| **上** | 数値入力、最小0 |
| **右** | 数値入力、最小0 |
| **下** | 数値入力、最小0 |
| **左** | 数値入力、最小0 |

ラベルの隣にある**リンク切り替えボタン**（鎖のアイコン）は、4つの辺がリンクされているかどうかを制御します:

- **リンク済み**（デフォルト） — 任意の値を変更すると、4つの辺が同時に更新されます
- **リンク解除** — 各辺が異なる幅を持つことができます。下辺のみの枠線や左側のアクセント枠線などの効果に便利です

## 枠線の角丸

各角の丸みを制御します。基本タブでは、4つの角の入力フィールドが表示されます:

| 角 | ラベル |
|--------|-------|
| **左上** | TL |
| **右上** | TR |
| **左下** | BL |
| **右下** | BR |

**リンク切り替えボタン**は、枠線の幅と同じように機能します:

- **リンク済み**（デフォルト） — 4つの角は同じ角丸値を共有します
- **リンク解除** — 各角が異なる角丸値を持つことができます

一般的な角丸値:

| 値 | 効果 |
|-------|--------|
| 0px | 鋭い角 |
| 4-8px | カードやボタンに適したわずかな丸み |
| 12-16px | 明確な丸み、現代的で柔らかな見た目 |
| 50% | 完全な円または薬丸（要素の寸法に依存） |

単位セレクタは、幅と角丸の値に対してpx、em、rem、%をサポートします。

## 角の形状（高度な）

高度なタブには、実験的な**角の形状**プロパティが含まれています。このCSS機能は、角丸が標準的な丸い形状を使用するか、より角張った「スコープ」形状を使用するかを制御します。ブラウザのサポートは限られており、現在のブラウザがこのプロパティをサポートしていない場合、エディタは互換性に関する警告を表示します。

## フッターアクション

| ボタン | アクション |
|--------|--------|
| **リセット** | エディタを開いた時点のすべての値に戻す |
| **キャンセル** | 変更を適用せずにエディタを閉じる |
| **適用** | 枠線の設定を保存し、エディタを閉じる |

## 出現場所

枠線エディタは、以下のビルダーで利用可能です:

- **ページビルダー** — 任意の要素を選択し、スタイルタブを開き、枠線セクションをクリックします
- **ヘッダ/フッタービルダー** — ヘッダセクション、ナビゲーションコンテナ、フッターエリアに枠線を追加します
- **メニュービルダー** — メニューアイテムやドロップダウンコンテナの枠線をスタイル付けします

エディタは、キャンバス上のライブ要素から現在の計算された枠線スタイルを読み取るため、常に正しい既存値で開きます。

## ヒント

- **枠線は控えめに使用する** — 軽いグレーカラーの1pxの枠線は、セクション間の分離を明確にしながら、視覚的な重みを加えません。
- **角丸とシャドウを組み合わせる** — 角丸とソフトなボックスシャドウ（シャドウエディタ経由）を組み合わせると、洗練されたカード効果が得られます。
- **片方の辺の枠線を試してみる** — 辺をリンク解除し、下辺または左辺の枠線のみを設定して、アクセントライン、セクションの区切り、サイドバーのインジケータとして使用できます。
- **薬丸形状に百分比角丸を使用する** — ボタンやバッジにすべての角を50%に設定すると、コンテンツサイズに応じて適応する薬丸形状が作成されます。
- **プレビューを確認する** — ライブプレビュー領域は即座に更新されるため、適用する前に自由に試してみてください。
