---
title: 배경 편집기
---

배경 편집기는 고체 색상, 그라디언트, 이미지 및 비디오의 네 가지 유형을 사용하여 요소의 배경에 대한 완전한 제어를 제공합니다. 또한 정상 상태와 호버 상태를 별도로 지원하여 상호작용적인 시각 효과를 만들 수 있습니다. 요소의 **스타일 탭**을 열고 **배경** 섹션을 찾아 편집기를 열 수 있습니다.

![배경 편집기](/static/core/admin/img/help/background-editor/background-editor.webp)

## 정상 및 호버 상태

배경 편집기 상단에는 **정상** 및 **호버** 상태를 전환하는 토글이 있습니다. 각 상태는 독립적인 배경 설정을 가지고 있습니다:

- **정상** — 페이지가 로드될 때 표시되는 기본 배경
- **호버** — 방문자가 커서를 요소 위로 이동할 때 적용되는 배경

토글 옆의 두 개의 작은 미리보기 블록은 현재 정상 및 호버 배경을 옆으로 나란히 표시하여 대비를 한눈에 볼 수 있도록 합니다. 먼저 정상 상태를 구성한 후, 원하는 경우 호버 상태로 전환하여 상호작용 효과를 추가할 수 있습니다.

## 배경 유형

편집기 패널 상단의 아이콘 행에서 배경 유형을 선택합니다:

| 유형 | 설명 |
|------|-------------|
| **색상** | 단일 색상 값으로 채워진 고체 배경입니다. 적용이 빠르고 가볍습니다. |
| **그라디언트** | 두 개 이상의 색상 간의 부드러운 블렌드, 선형 또는 방사형입니다. 바다, 일몰, 숲, 베리와 같은 내장된 사전 설정이 포함되어 있습니다. 고급 그라디언트 편집은 [그라디언트 생성기](gradient-creator) 주제를 참조하십시오. |
| **이미지** | 업로드된 이미지 또는 미디어 라이브러리에서 선택한 이미지입니다. 위치, 크기 및 반복 제어를 지원합니다. |
| **비디오** | 로딩 중 또는 모바일 장치에서 표시되는 선택적 포스터 이미지가 있는 배경 비디오 URL입니다. |

각 상태당 한 번에 하나의 유형만 활성화할 수 있습니다. 유형을 전환하더라도 이전 설정은 삭제되지 않으며 다시 전환할 수 있으며 설정은 유지됩니다.

## 색상 배경

색상이 선택된 경우:

Preserve all markdown formatting, image paths, code blocks, and technical terms.

- **Hex 입력** — 직접 헥스 코드를 입력하세요 (예: `#1A1A2E`)
- **색상 스와치** — 빠른 선택을 위해 사전 정의된 스와치를 클릭하세요.

스와치는 테마에 따라 달라지며, 활성화된 테마의 색상 팔레트를 반영합니다.
- **편집 버튼** — 스펙트럼, 슬라이더 및 형식 옵션을 포함한 전체 색상 선택기를 엽니다 (색상 선택기([Color Picker](color-picker)) 항목을 참조하세요)

색상 배경은 즉시 렌더링되며 성능에 영향을 주지 않아 섹션, 카드 및 컨테이너에 이상적입니다.

## 그라디언트 배경

그라디언트가 선택된 경우:

- **사전 정의된 그라디언트** — 내장된 그라디언트(Ocean, Sunset, Forest, Berry 등)를 선택하세요
- **사용자 정의 그라디언트** — **편집**을 클릭하여 그라디언트 생성기를 엽니다. 방향, 유형(선형 또는 원형), 색상 정지점을 설정할 수 있습니다.
- **각도 슬라이더** — 선형 그라디언트의 방향을 조정합니다 (0-360도)

그라디언트는 이미지 자산을 사용하지 않고도 시각적 깊이를 추가할 수 있으며, 모든 화면 크기에 완벽하게 확장됩니다.

## 이미지 배경

이미지가 선택된 경우:

- **업로드 또는 미디어 라이브러리** — 이미지 플레이스홀더를 클릭하여 새 이미지를 업로드하거나 미디어 라이브러리에서 이미지를 선택하세요
- **크기** — **Cover**(요소를 채우며 자르는 경우도 있음), **Contain**(요소 내부에 맞춤), 또는 사용자 정의 크기를 선택하세요
- **위치** — 9개 지점 격자(왼쪽 상단, 중앙, 오른쪽 하단 등)를 사용하여 초점점을 설정하거나 사용자 정의 X/Y 백분율을 입력하세요
- **반복** — 반복을 켜거나 끄세요. 패턴 타일링에 유용합니다
- **오버레이** — 이미지 위에 색상 오버레이를 추가하고 투명도를 조정하여 텍스트 가독성을 보장할 수 있습니다

업로드 전에 이미지를 항상 최적화하세요. 대용량 압축되지 않은 이미지는 페이지 로드 시간을 느리게 합니다.

## 비디오 배경

비디오가 선택된 경우:

- **비디오 URL** — MP4 또는 WebM 비디오 파일의 직접 URL을 입력하세요
- **포스터 이미지** — 비디오가 로드되는 동안 및 자동 재생이 되지 않는 장치에서 표시되는 대체 이미지를 업로드하세요
- **자동 재생 / 반복 / 음소거** — 브라우저 정책을 준수하기 위해 비디오 배경은 기본적으로 자동 재생, 반복, 음소거 상태입니다


배경 영상은 짧고(10~30초), 압축되어 있고 시각적으로 부드러운 것이 좋습니다.

내용을 방해하지 않으면서 섹션을 강화하는 데 도움이 되어야 합니다.

## 어디에 나타나나요

배경 편집기는 배경을 지원하는 모든 요소에 사용할 수 있습니다:

- **페이지 빌더** — 섹션, 컨테이너, 열 및 개별 요소는 모두 스타일 탭에 배경 섹션이 있습니다.
- **헤더/푸터 빌더** — 행 배경 및 개별 위젯 배경
- **메뉴 빌더** — 메뉴 컨테이너 및 드롭다운 패널 배경

같은 편집기 인터페이스가 모든 곳에서 사용되므로 빌더 간 작업 흐름이 일관됩니다.

## 팁

- 이미지 배경에 반투명 색상 오버레이를 사용하여 이미지 내용과 관계없이 텍스트가 가독성이 있도록 합니다.
- 그라디언트 사전 설정은 시각적 흥미를 빠르게 추가하는 방법입니다 — 하나를 적용한 후 각도나 색상을 브랜드 스타일에 맞게 조정합니다.
- 상호작용 카드에 일반 및 호버 배경을 모두 설정하여 방문자가 콘텐츠를 탐색할 때 명확한 시각적 피드백을 제공합니다.
- 이미지 배경을 사용할 경우 항상 초점점을 설정하여 이미지의 가장 중요한 부분이 모든 화면 크기에서 보이도록 합니다.
- 로드 속도가 중요한 섹션, 예를 들어 폴드 위 콘텐츠에서는 이미지보다 색상 또는 그라디언트 배경을 선호합니다.
- 모바일 기기에서 영상 배경을 테스트하세요 — 대부분의 모바일 브라우저는 영상 대신 포스터 이미지만 표시합니다.
