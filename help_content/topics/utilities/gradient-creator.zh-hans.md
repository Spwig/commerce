---
title: 渐变生成器
---

渐变生成器可让您为元素背景创建平滑的颜色过渡。它通过背景编辑器的渐变选项卡访问，并以浮动面板的形式打开，包含一个可视化的渐变条、颜色停止控件和预设选项。

![渐变生成器](/static/core/admin/img/help/gradient-creator/gradient-creator.webp)

## 访问渐变生成器

1. 在页面构建器或页眉/页脚构建器中选择一个元素
2. 在属性面板中打开 **样式** 选项卡
3. 点击 **背景** 部分以打开背景编辑器
4. 切换到 **渐变** 选项卡
5. 渐变生成器面板将打开，包含实时预览和编辑控件

## 实时预览

面板顶部显示一个并排比较：

| Box | Purpose |
|-----|---------|
| **Current** | 当前渐变（如果没有设置则为透明） |
| **New** | 在您进行更改时实时更新 |

两个框之间的箭头表示更改的方向。

## 渐变类型

有三种渐变类型可供选择，可通过编辑器顶部的选项卡进行选择：

| Type | Description | Controls |
|------|-------------|----------|
| **Linear** | 沿直线进行颜色过渡 | 角度滑块（0-360度）以及预设方向按钮（上、对角线、右、下等） |
| **Radial** | 从中心点向外辐射的颜色过渡 | 形状选择器（圆形或椭圆形）和位置选择器（中心、顶部、底部、角落） |
| **Conic** | 围绕中心点旋转的颜色过渡 | 起始角度滑块（0-360度）和位置选择器 |

### 线性方向控件

对于线性渐变，您可以通过以下三种方式设置角度：
- **角度滑块** — 从 0 到 360 度拖动
- **角度输入** — 输入精确的角度值
- **预设按钮** — 点击箭头图标以选择常见方向（向上、右上、向右、右下、向下、左下、向左、左上）

## 颜色停止

渐变条显示您当前的颜色停止点，这些点可以拖动。每个停止点定义了渐变中特定位置的颜色。

**添加停止点** — 点击 **Color Stops** 部分的 **+** 按钮以添加新停止点。停止点的数量没有硬性限制。

**编辑停止点** — 列表中的每个停止点显示：
- 点击后打开颜色选择器的颜色色块
- 一个位置值（0% 到 100%），您可以输入或调整
- 一个透明度控件（0 到 1）
- 一个删除按钮，用于删除该停止点

**重新排序** — 沿着渐变条拖动停止点以重新排列它们的视觉位置。

## 渐变预设

有六个内置预设可供快速开始。点击任意预设即可立即应用：

| Preset | Colors | Angle |
|--------|--------|-------|
| **Ocean** | 浅蓝色到蓝色 | 120 度 |
| **Sunset** | 暖橙色到珊瑚粉（3 个停止点） | 45 度 |
| **Forest** | 靛蓝到翡翠绿 | 135 度 |
| **Berry** | 粉红到蓝紫色 | 90 度 |
| **Flame** | 红色到金色 | 45 度 |
| **Night** | 深石墨到海洋蓝 | 180 度 |

预设是起点。应用一个预设后，您可以修改颜色、添加或删除停止点，并更改角度以创建自己的变体。

## 底部操作

| Button | Action |
|--------|--------|
| **Clear** | 删除渐变，重置为透明 |
| **Apply** | 保存渐变并关闭编辑器 |

在未点击 **Apply** 的情况下关闭编辑器将丢弃您的更改。

## 使用位置

渐变生成器用于以下位置：

- **页面构建器** — 通过背景编辑器的渐变选项卡，应用于任何元素
- **页眉/页脚构建器** — 用于页眉部分、导航栏和页脚区域的渐变背景

它与背景编辑器配合使用，背景编辑器还提供纯色、图片和视频背景选项。

## 小贴士

- **从预设开始** — 应用一个与您想要的接近的预设，然后调整颜色和角度，而不是从头开始构建。
- **使用两个或三个停止点** — 两个停止点的简单渐变看起来干净且专业。更多的停止点对于复杂效果很有用，但可能会很快变得令人眼花缭乱。
- **匹配您的品牌颜色** — 使用颜色选择器输入您品牌调色板中的精确十六进制值，以确保渐变与品牌一致。
- **与内容一起测试** — 单独看起来引人注目的渐变可能会降低文本的可读性。始终检查渐变背景上的文本是否具有足够的对比度。
- **尝试径向渐变以突出效果** — 径向渐变非常适合吸引对中心区域的注意力，例如英雄部分的焦点区域。
