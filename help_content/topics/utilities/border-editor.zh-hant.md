---
title: 邊框編輯器
---

邊框編輯器提供對元素邊框的精細控制，包括樣式、顏色、每側的寬度以及每個角落的圓角。它會以一個浮動面板的形式打開，並包含即時預覽功能，以及基本設定和高級設定的兩個選項卡。

![邊框編輯器](/static/core/admin/img/help/border-editor/border-editor.webp)

## 即時預覽

編輯器頂部的預覽框會即時顯示您的邊框更改。該框內顯示一個帶有邊框的矩形，標有「預覽」文字，當您調整樣式、顏色、寬度和圓角值時，會立即更新。

## 基本模式與高級模式

編輯器分為兩個選項卡：

| 選項卡 | 內容 |
|-------|-----|
| **基本** | 邊框樣式、顏色、寬度（含每側控制）以及邊框圓角（含每個角落的控制） |
| **高級** | 單個角落圓角的精細調整以及實驗性的角落形狀屬性 |

大多數邊框工作都在基本選項卡中完成。當您需要對單個角落進行精確控制或想嘗試更新的 CSS 功能時，高級選項卡會很有幫助。

## 邊框樣式

一個包含九個選項的下拉選單，用於控制邊框線的外觀：

| 樣式 | 描述 |
|-----|-----|
| **無** | 無邊框（移除任何現有的邊框） |
| **實線** | 一條連續的線（預設） |
| **虛線** | 一組短的虛線 |
| **點線** | 一組圓點 |
| **雙線** | 兩條平行的實線 |
| **凹陷** | 一種雕刻的 3D 效果邊框，看起來像是壓入表面 |
| **凸起** | 一種凸起的 3D 效果邊框（與凹陷相反） |
| **內嵌** | 使元素看起來嵌入或壓入 |
| **外凸** | 使元素看起來凸起或彈出 |

將樣式設為「無」會完全移除邊框，無論寬度或顏色設置如何。

## 邊框顏色

一個文字輸入欄與一個顏色選擇器按鈕。可以直接輸入十六進制值（例如 `#3b82f6`），或點擊顏色樣板以打開完整的顏色選擇器，支援十六進制、RGB 和 HSL 輸入模式，以及視覺顏色區域。預設顏色為黑色（`#000000`）。

## 邊框寬度

控制邊框的厚度（以像素為單位）。基本選項卡顯示四個側邊的輸入欄：

| 邊 | 輸入 |
|----|-----|
| **上** | 數值輸入，最小值為 0 |
| **右** | 數值輸入，最小值為 0 |
| **下** | 數值輸入，最小值為 0 |
| **左** | 數值輸入，最小值為 0 |

標籤旁邊有一個 **連結切換按鈕**（鍊結圖示），用於控制是否將所有四個邊連結：

- **連結**（預設）——更改任何值會同時更新所有四個邊
- **未連結**——每個邊可以有不同的寬度，這對於僅底部邊框或左側強調邊框等效果很有用

## 邊框圓角

控制每個角落的圓角。基本選項卡顯示四個角落的輸入欄：

| 角落 | 標籤 |
|-----|-----|
| **左上** | TL |
| **右上** | TR |
| **左下** | BL |
| **右下** | BR |

**連結切換按鈕**的工作方式與邊框寬度相同：

- **連結**（預設）——所有四個角落共享相同的圓角值
- **未連結**——每個角落可以有不同的圓角值

常見的圓角值：

| 值 | 效果 |
|----|------|
| 0px | 尖銳的方形角落 |
| 4-8px | 輕微的圓角，適合卡片和按鈕 |
| 12-16px | 明顯的圓角，現代且柔和的外觀 |
| 50% | 完全的圓形或藥丸形（取決於元素的尺寸） |

單位選擇器支援 px、em、rem 和 %，用於寬度和圓角值。

## 角落形狀（高級）

高級選項卡包含一個實驗性的 **角落形狀** 屬性。這個 CSS 功能控制圓角使用標準的圓形形狀，還是更鋭利的「凹槽」形狀。瀏覽器支援有限，當當前瀏覽器不支援此屬性時，編輯器會顯示相容性警告。

## 底部動作

| 按鈕 | 作用 |
|------|------|
| **重置** | 將所有值恢復為編輯器打開時的狀態 |
| **取消** | 不套用更改，關閉編輯器 |
| **套用** | 儲存邊框設置並關閉編輯器 |

## 出現位置

邊框編輯器在以下幾個構建器中可用：

- **頁面構建器** — 選中任何元素，打開樣式選項卡，並點擊邊框部分
- **首頁/頁尾構建器** — 為首頁部分、導覽容器和頁尾區域添加邊框
- **菜單構建器** — 為菜單項目和下拉容器設定邊框

編輯器會從畫布上的活體元素中讀取當前計算的邊框樣式，因此它總是會以正確的現有值打開。

## 小技巧

- **盡量少用邊框** — 使用淡灰色的 1px 邊框可以在各個部分之間創造乾淨的分隔，而不會增加視覺負擔。
- **結合圓角與陰影** — 將圓角與柔和的盒狀陰影（透過陰影編輯器）結合，可以產生精緻的卡片效果。
- **嘗試單側邊框** — 解除連結並僅設置底部或左側邊框，可用於強調線、部分分隔線或側邊欄指示器。
- **使用百分比圓角製作藥丸形** — 將所有角落設為 50% 可在按鈕或徽章上產生適應任何內容大小的藥丸形。
- **查看預覽** — 即時預覽框會立即更新，因此在套用之前可以自由地進行實驗。
