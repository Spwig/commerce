title: เครื่องสร้างสีไล่ระดับ
---

เครื่องสร้างสีไล่ระดับ (Gradient Creator) ช่วยให้คุณสร้างการเปลี่ยนสีที่ลื่นไหลสำหรับพื้นหลังขององค์ประกอบต่าง ๆ สามารถเข้าถึงได้ผ่านแท็บพื้นหลัง (Background) ของตัวแก้ไขพื้นหลัง (Background Editor) และจะแสดงเป็นแผงลอยพร้อมกับแถบสีไล่ระดับแบบภาพ ตัวควบคุมจุดสี และตัวเลือกแบบเริ่มต้น

![เครื่องสร้างสีไล่ระดับ](/static/core/admin/img/help/gradient-creator/gradient-creator.webp)

## การเข้าถึงเครื่องสร้างสีไล่ระดับ

1. เลือกองค์ประกอบใน Page Builder หรือ Header/Footer Builder
2. เปิดแท็บ **Style** ในแผงคุณสมบัติ
3. คลิกส่วน **Background** เพื่อเปิดตัวแก้ไขพื้นหลัง (Background Editor)
4. เปลี่ยนไปยังแท็บ **Gradient**
5. แผงเครื่องสร้างสีไล่ระดับจะเปิดขึ้นพร้อมกับการดูแบบเรียลไทม์และการควบคุมการแก้ไข

## การดูแบบเรียลไทม์

ส่วนบนของแผงแสดงเปรียบเทียบแบบข้างเคียง:

| Box | Purpose |
|-----|---------|
| **Current** | สีไล่ระดับที่มีอยู่ (หรือโปร่งใสหากไม่ได้ตั้งค่าใด ๆ) |
| **New** | อัปเดตแบบเรียลไทม์เมื่อคุณทำการเปลี่ยนแปลง |

ลูกศรระหว่างกล่องทั้งสองแสดงทิศทางของการเปลี่ยนแปลง

## ประเภทของสีไล่ระดับ

มีประเภทของสีไล่ระดับทั้งหมด 3 ประเภท สามารถเลือกได้ผ่านแท็บที่อยู่ด้านบนของตัวแก้ไข:

| Type | Description | Controls |
|------|-------------|----------|
| **Linear** | การเปลี่ยนสีตามเส้นตรง | ตัวเลื่อนมุม (0-360 องศา) พร้อมปุ่มทิศทางแบบเริ่มต้น (ขึ้น, ทแยง, ขวา, ลง, ฯลฯ) |
| **Radial** | การเปลี่ยนสีที่แผ่ออกจากรูปทรงศูนย์กลาง | ตัวเลือกรูปทรง (วงกลมหรือรูปไข่) และตัวเลือกตำแหน่ง (ศูนย์กลาง, ด้านบน, ด้านล่าง, มุม) |
| **Conic** | การเปลี่ยนสีที่หมุนรอบจุดศูนย์กลาง | ตัวเลื่อนมุมเริ่มต้น (0-360 องศา) และตัวเลือกตำแหน่ง |

### การควบคุมทิศทางแบบ Linear

สำหรับสีไล่ระดับแบบ Linear คุณสามารถตั้งค่ามุมได้ 3 วิธี:
- **Angle slider** — ลากจาก 0 ถึง 360 องศา
- **Angle input** — พิมพ์ค่ามุมที่แม่นยำ
- **Preset buttons** — คลิกไอคอนลูกศรสำหรับทิศทางที่พบบ่อย (ขึ้น, ทแยงด้านบนขวา, ขวา, ทแยงด้านล่างขวา, ลง, ทแยงด้านล่างซ้าย, ซ้าย, ทแยงด้านบนซ้าย)

## จุดสี (Color Stops)

แถบสีไล่ระดับแสดงจุดสีที่คุณมีอยู่ในรูปของตัวชี้ที่สามารถลากได้ แต่ละจุดกำหนดสีที่ตำแหน่งเฉพาะในสีไล่ระดับ

**การเพิ่มจุดสี** — คลิกปุ่ม **+** ในส่วน Color Stops เพื่อเพิ่มจุดสีใหม่ ไม่มีข้อจำกัดจำนวนจุดสีที่ชัดเจน

**การแก้ไขจุดสี** — แต่ละจุดในรายการแสดง:
- สีที่แสดงเปิดตัวเลือกสี (Color Picker) เมื่อคลิก
- ค่าตำแหน่ง (0% ถึง 100%) ที่คุณสามารถพิมพ์หรือปรับได้
- ตัวควบคุมความโปร่งใส (0 ถึง 1)
- ปุ่มลบเพื่อลบจุดสี

**การจัดลำดับใหม่** — ลากจุดสีตามแถบสีไล่ระดับเพื่อจัดตำแหน่งใหม่ในภาพ

## ตัวตั้งค่าเริ่มต้น (Presets)

มีตัวตั้งค่าเริ่มต้นทั้งหมด 6 แบบที่พร้อมใช้งานสำหรับจุดเริ่มต้นที่รวดเร็ว คลิกตัวตั้งค่าใด ๆ เพื่อใช้งานได้ทันที:

| Preset | Colors | Angle |
|--------|--------|-------|
| **Ocean** | สีฟ้าอ่อนไปสีฟ้า | 120 องศา |
| **Sunset** | สีส้มอบอุ่นไปสีชมพูส้ม (3 จุดสี) | 45 องศา |
| **Forest** | สีอินดิโก้ไปสีเขียวอีเมรัลด์ | 135 องศา |
| **Berry** | สีชมพูไปสีม่วง-น้ำเงิน | 90 องศา |
| **Flame** | สีแดงไปสีเหลืองทอง | 45 องศา |
| **Night** | สีหินมืดไปสีน้ำเงินทะเล | 180 องศา |

ตัวตั้งค่าเริ่มต้นเป็นจุดเริ่มต้นหลังจากใช้งานแล้วคุณสามารถปรับสี เพิ่มหรือลบจุดสี และเปลี่ยนมุมเพื่อสร้างการเปลี่ยนแปลงของคุณเอง

## การดำเนินการในส่วนท้าย

| Button | Action |
|--------|--------|
| **Clear** | ลบสีไล่ระดับทั้งหมด กลับไปเป็นโปร่งใส |
| **Apply** | บันทึกสีไล่ระดับและปิดตัวแก้ไข |

การปิดตัวแก้ไขโดยไม่คลิก Apply จะทำให้การเปลี่ยนแปลงของคุณถูกยกเลิก

## ที่ใดที่ใช้งาน

เครื่องสร้างสีไล่ระดับถูกใช้งานใน:

- **Page Builder** — ผ่านแท็บ Gradient ของตัวแก้ไขพื้นหลัง (Background Editor) สำหรับองค์ประกอบใด ๆ
- **Header/Footer Builder** — สำหรับพื้นหลังแบบสีไล่ระดับในส่วนหัว แถบนำทาง และพื้นที่ส่วนท้าย

ทำงานร่วมกับตัวแก้ไขพื้นหลัง (Background Editor) ซึ่งยังมีตัวเลือกพื้นหลังแบบสีทึบ ภาพ และวิดีโอ

## เคล็ดลับ

- **เริ่มต้นด้วยตัวตั้งค่าเริ่มต้น** — ใช้ตัวตั้งค่าเริ่มต้นที่ใกล้เคียงกับสิ่งที่คุณต้องการ แล้วปรับสีและมุมแทนที่จะสร้างจากศูนย์
- **ใช้จุดสี 2 หรือ 3 จุด** — สีไล่ระดับที่ใช้จุดสี 2 จุดดูสะอาดและมืออาชีพ ส่วนจุดสีมากขึ้นเหมาะสำหรับผลลัพธ์ที่ซับซ้อน แต่อาจทำให้เกิดความซับซ้อนได้เร็ว
- **ใช้สีของแบรนด์ของคุณ** — ใช้ตัวเลือกสี (Color Picker) เพื่อใส่ค่า hex ที่แม่นยำจากสีของแบรนด์ของคุณเพื่อให้ได้สีไล่ระดับที่สอดคล้องกับแบรนด์
- **ทดสอบกับเนื้อหา** — สีไล่ระดับที่ดูน่าประทับใจเมื่ออยู่คนเดียวอาจลดการอ่านข้อความ ตรวจสอบเสมอว่าข้อความที่อยู่เหนือพื้นหลังสีไล่ระดับมีความตัดกันเพียงพอ
- **ลองใช้แบบ Radial สำหรับเอฟเฟกต์แสงส่อง** — สีไล่ระดับแบบ Radial ทำงานได้ดีสำหรับการดึงความสนใจไปยังพื้นที่ศูนย์กลาง เช่น จุดโฟกัสของส่วนหัวหลัก
