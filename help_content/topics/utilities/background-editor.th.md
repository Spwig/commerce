ตัวแก้ไขพื้นหลังให้คุณสามารถควบคุมพื้นหลังขององค์ประกอบได้อย่างเต็มที่ โดยมีทั้งหมด 4 ประเภท ได้แก่ สีทึบ โทนสี รูปภาพ และวิดีโอ นอกจากนี้ยังรองรับสถานะ Normal และ Hover แยกกัน เพื่อให้คุณสามารถสร้างเอฟเฟกต์ภาพที่มีปฏิสัมพันธ์ได้ เปิดแท็บ **Style** ขององค์ประกอบใดองค์ประกอบหนึ่ง และหาส่วน **Background** เพื่อเข้าถึงตัวแก้ไข

![ตัวแก้ไขพื้นหลัง](/static/core/admin/img/help/background-editor/background-editor.webp)

## สถานะ Normal และ Hover

ที่ด้านบนของตัวแก้ไขพื้นหลัง มีปุ่มสลับระหว่างสถานะ **Normal** และ **Hover** แต่ละสถานะมีการตั้งค่าพื้นหลังที่เป็นอิสระต่อกัน:

- **Normal** — พื้นหลังเริ่มต้นที่แสดงเมื่อหน้าเว็บโหลดขึ้นมา
- **Hover** — พื้นหลังที่ใช้เมื่อผู้เยี่ยมชมวางเมาส์เหนือองค์ประกอบ

บล็อกตัวอย่างขนาดเล็กสองอันที่อยู่ด้านข้างของปุ่มสลับจะแสดงพื้นหลัง Normal และ Hover อยู่เคียงข้างกัน เพื่อให้คุณสามารถเห็นความแตกต่างได้ทันที ให้ตั้งค่าสถานะ Normal ก่อน จากนั้นสลับไปยังสถานะ Hover เพื่อเพิ่มเอฟเฟกต์ที่มีปฏิสัมพันธ์หากต้องการ

## ประเภทของพื้นหลัง

เลือกประเภทของพื้นหลังจากไอคอนแถวที่อยู่ด้านบนของแผงตัวแก้ไข:

| ประเภท | คำอธิบาย |
|--------|-----------|
| **สี** | การเติมพื้นหลังด้วยค่าสีเดียว ใช้งานง่ายและไม่กินทรัพยากร |
| **โทนสี** | การผสมสีที่ลื่นไหลระหว่างสองสีหรือมากกว่านั้น ทั้งแบบเส้นตรงและแบบรังสี รวมถึงโทนสีที่ตั้งไว้ล่วงหน้า เช่น มหาสมุทร พระอาทิตย์ตก ป่า และเบอร์รี่ สำหรับการแก้ไขโทนสีขั้นสูง ดูหัวข้อ [Gradient Creator](gradient-creator) |
| **รูปภาพ** | รูปภาพที่อัปโหลดหรือเลือกจากไลบรารีสื่อ รองรับการจัดตำแหน่ง ขนาด และการซ้ำ |
| **วิดีโอ** | URL วิดีโอพื้นหลังพร้อมภาพประกอบที่สามารถแสดงได้ขณะโหลดวิดีโอหรือบนอุปกรณ์มือถือ |

สามารถเปิดใช้งานประเภทเดียวได้ในแต่ละสถานะ การเปลี่ยนประเภทจะไม่ลบการตั้งค่าก่อนหน้า คุณสามารถสลับกลับมาได้และตั้งค่าจะถูกเก็บไว้

## พื้นหลังแบบสี

เมื่อเลือกใช้สี:

Preserve all markdown formatting, image paths, code blocks, and technical terms.

- **Hex Input** — พิมพ์รหัส hex โดยตรง (เช่น `#1A1A2E`)
- **Color Swatches** — คลิกสีตัวอย่างสำหรับการเลือกอย่างรวดเร็ว

Swatches ตระหนักถึงธีมและสะท้อนโทนสีของธีมที่ใช้งานอยู่
- **Edit Button** — เปิดตัวเลือกสีแบบเต็มรูปแบบที่มีสเปกตรัม สไลเดอร์ และตัวเลือกการกำหนดรูปแบบ (ดูหัวข้อ [Color Picker](color-picker))

พื้นหลังสีจะแสดงผลทันทีและไม่มีผลกระทบต่อประสิทธิภาพ ทำให้เหมาะสำหรับส่วนต่าง ๆ บัตร และคอนเทนเนอร์

## Gradient Backgrounds

เมื่อเลือก Gradient:

- **Preset Gradients** — เลือกจากกราดิเอนต์ที่มีอยู่: Ocean, Sunset, Forest, Berry และอื่น ๆ
- **Custom Gradient** — คลิก **Edit** เพื่อเปิดเครื่องมือสร้างกราดิเอนต์ที่คุณสามารถตั้งค่าทิศทาง ประเภท (Linear หรือ Radial) และจุดสีได้
- **Angle Slider** — ปรับทิศทางของกราดิเอนต์สำหรับกราดิเอนต์แบบ Linear (0-360 องศา)

กราดิเอนต์เพิ่มความลึกทางภาพโดยไม่ต้องใช้ทรัพยากรภาพ และปรับขนาดได้อย่างสมบูรณ์แบบสำหรับขนาดหน้าจอใด ๆ

## Image Backgrounds

เมื่อเลือก Image:

- **Upload or Media Library** — คลิกที่พื้นที่สำหรับภาพเพื่ออัปโหลดภาพใหม่หรือเลือกภาพจากไลบรารีสื่อของคุณ
- **Size** — เลือก **Cover** (เติมเต็มองค์ประกอบ อาจตัดภาพ), **Contain** (ปรับให้พอดีภายในองค์ประกอบ) หรือขนาดที่กำหนดเอง
- **Position** — ตั้งค่าจุดโฟกัสโดยใช้กริด 9 จุด (ด้านบนซ้าย ศูนย์กลาง ด้านล่างขวา ฯลฯ) หรือป้อนเปอร์เซ็นต์ X/Y ที่กำหนดเอง
- **Repeat** — เปิด/ปิดการซ้ำ ใช้ได้ดีสำหรับรูปแบบที่ซ้ำกัน
- **Overlay** — เพิ่มสีทับภาพที่สามารถปรับความโปร่งใสได้ ซึ่งมีประโยชน์ในการรับรองความอ่านของข้อความ

ควรปรับแต่งภาพก่อนอัปโหลดเสมอ ภาพขนาดใหญ่ที่ไม่ได้บีบอัดจะทำให้การโหลดหน้าเว็บช้าลง

## Video Backgrounds

เมื่อเลือก Video:

- **Video URL** — ใส่ URL ของไฟล์วิดีโอ MP4 หรือ WebM โดยตรง
- **Poster Image** — อัปโหลดภาพสำรองที่แสดงขณะวิดีโอโหลดและบนอุปกรณ์ที่ไม่สามารถเล่นวิดีโออัตโนมัติได้
- **Autoplay / Loop / Muted** — วิดีโอพื้นหลังจะเล่นอัตโนมัติ วนซ้ำ และถูกปิดเสียงโดยค่าเริ่มต้นเพื่อปฏิบัติตามนโยบายของเบราว์เซอร์

รักษาวิดีโอพื้นหลังให้สั้น (10-30 วินาที), บีบอัด และมีความละเอียดอ่อนทางสายตา

ควรเสริมเนื้อหาโดยไม่ทำให้เนื้อหาหลักนั้นดูน้อยลง

## ตำแหน่งที่ใช้งาน

ตัวแก้ไขพื้นหลังสามารถใช้งานได้กับทุกส่วนที่รองรับพื้นหลัง:

- **Page Builder** — ส่วนของหน้า, คอนเทนเนอร์, คอลัมน์ และองค์ประกอบแต่ละตัวมีส่วนของพื้นหลังในแท็บ Style
- **Header/Footer Builder** — พื้นหลังของแถวและพื้นหลังของวิดเจ็ตแต่ละตัว
- **Menu Builder** — พื้นหลังของคอนเทนเนอร์เมนูและพื้นหลังของแผงเมนูแบบเลื่อนลง

ตัวแก้ไขพื้นหลังนั้นใช้งานเดียวกันทุกที่ ดังนั้นกระบวนการทำงานของคุณจึงจะคงที่ไม่เปลี่ยนไปในแต่ละตัวสร้าง

## เคล็ดลับ

- ใช้สีที่มีความโปร่งใสบางส่วนบนพื้นหลังภาพ เพื่อให้ข้อความยังอ่านได้ชัดเจนไม่ว่าภาพจะมีเนื้อหาอะไร
- ค่าตั้งค่าสีไล่ระดับเป็นวิธีที่รวดเร็วในการเพิ่มความน่าสนใจทางสายตา — ใช้ค่าตั้งค่าหนึ่ง แล้วปรับมุมหรือสีเพื่อให้เข้ากับแบรนด์ของคุณ
- ตั้งค่าพื้นหลังทั้ง Normal และ Hover สำหรับการ์ดที่มีการโต้ตอบ เพื่อให้ผู้เยี่ยมชมได้รับการตอบกลับทางสายตาที่ชัดเจนเมื่อพวกเขาสำรวจเนื้อหาของคุณ
- สำหรับพื้นหลังภาพ ต้องตั้งค่าจุดโฟกัสเสมอ เพื่อให้ส่วนสำคัญที่สุดของภาพยังคงมองเห็นได้บนทุกขนาดหน้าจอ
- ควรใช้พื้นหลังสีหรือสีไล่ระดับแทนภาพสำหรับส่วนที่ความเร็วในการโหลดมีความสำคัญ เช่น เนื้อหาที่อยู่เหนือเส้นพับ
- ทดสอบวิดีโอพื้นหลังบนอุปกรณ์มือถือ — ตัวท่องเว็บส่วนใหญ่บนมือถือจะแสดงภาพโพสเตอร์แทนการเล่นวิดีโอ
