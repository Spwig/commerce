---
title: 間距編輯器
---

視覺間距編輯器讓您可以使用直觀的盒子模型圖示來配置邊距和內邊距。精確的間距控制可確保您的商店頁面在各個設備上擁有統一的佈局和舒適的閱讀體驗。打開任何元素的 **Style tab**，並尋找 **Spacing** 區域以使用編輯器。

![Spacing Editor](/static/core/admin/img/help/spacing-editor/spacing-editor.webp)

## The Box Model Diagram

編輯器顯示一個視覺化的盒子模型，包含三個嵌套層：

- **Margin** (外環，通常顯示為橙色) — 元素邊框外部的空間，用於與鄰近元素分隔
- **Padding** (內環，通常顯示為綠色) — 元素邊框與內容之間的空間
- **Content** (中心區域) — 元素的實際內容，例如文字或圖片

圖示的每一側（上、右、下、左）都有可拖動的控制桿和數字輸入欄。將控制桿向外拖動以增加數值，向內拖動以減少數值。您也可以直接點擊某一側的數值來輸入精確的數字。

## Margin and Padding Tabs

編輯器頂部的兩個標籤切換 **Margin** 和 **Padding** 的視圖。當選擇 Margin 時，外環會被高亮並可編輯。當選擇 Padding 時，內環會被高亮並可編輯。未被選中的環仍然可見，但會變暗。

兩個標籤共用相同的控制項和單位選項，因此邊距和內邊距的配置流程是一致的。

## Per-Side Controls

每一側都有獨立的數值輸入欄和單位選擇器：

| Side | Description |
|------|-------------|
| **Top** | 元素上方的空間（邊距）或內容上方的空間（內邊距） |
| **Right** | 元素或內容右側的空間 |
| **Bottom** | 元素或內容下方的空間 |
| **Left** | 元素或內容左側的空間 |

點擊圖示中任何一側的數值以選中它，然後輸入數字或使用上下箭頭鍵以每次增加 1。按住 Shift 的同時按下箭頭鍵可每次增加 10。

## Units

每個數值輸入欄旁邊的單位選擇器讓您可以選擇測量單位：

| Unit | Description |
|------|-------------|
| **px** | 像素。固定大小，所有設備上都一致。適合精確的小間距值。 |
| **em** | 相對於元素的字型大小。字型變化時會自動調整。 |
| **rem** | 相對於根字型大小。提供整個頁面的一致縮放效果。 |
| **%** | 父元素寬度的百分比。適合流動且響應式的佈局。 |
| **auto** | 讓瀏覽器自動計算數值。常用於水平居中，搭配左/右邊距。 |

選擇一個符合您需求的單位 — 使用 `px` 來設定固定間距，使用 `rem` 來設定與主題字型令牌相對應的可縮放間距，使用 `%` 來設定需要根據容器寬度調整的佈局。

## Link Sides

圖示中心的 **link icon** 可切換連結模式：

- **Linked** (連接的鍊條圖示) — 更改任何一側的數值會更新所有四側為相同的數值。適合用於統一間距。
- **Unlinked** (斷開的鍊條圖示) — 每一側獨立控制。當您需要不同的上/下和左/右數值時使用此模式。

點擊連結圖示以切換模式。當您從未連結切換到連結模式時，所有四側都會被設定為最近編輯過的側的數值。

## Quick Presets

圖示下方的一排預設按鈕提供了一鍵間距配置：

| Preset | Values |
|--------|--------|
| **None** | 所有側都為 0 |
| **Small** | 適合緊湊佈局和內聯元素的小間距 |
| **Medium** | 卡片和區域的一般用途平衡間距 |
| **Large** | 英雄區域和高強調區域的寬鬆間距 |
| **XL** | 全寬橫幅和頂層頁面區域的額外寬間距 |

預設值會套用於目前活動的標籤（Margin 或 Padding），並一次性設定所有四側。套用預設值後，您可以根據需要調整個別側的數值。

## Where It Appears

間距編輯器可用於所有支援佈局間距的元素：

- **Page Builder** — 章節、容器、欄位和個別元素的 **Style tab** 和 **Spacing** 區域
- **Header/Footer Builder** — 行和小工具的垂直和水平間距控制
- **Menu Builder** — 菜單項目內邊距和容器邊距設定

所有位置都使用相同的編輯器介面，確保在各個構建器中擁有統一的體驗。

## Tips

- 在您的頁面上使用一致的間距值 — 選擇 2-3 個標準尺寸，並堅持使用以獲得整潔專業的佈局。
- 將左側和右側的邊距設為 **auto**，以水平居中固定寬度的元素於其父元素中。
- 如果您的主題使用響應式字型，請優先使用 `rem` 單位，這樣間距會與字型大小成比例縮放。
- 使用連結模式快速設定均勻的內邊距，然後切換到未連結模式並微調個別側，如果內容需要非對稱間距。
- 避免在行動裝置上使用過多的內邊距 — 在窄視窗寬度下測試您的間距，確保內容不會被壓縮或過度填充。
