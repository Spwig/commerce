---
title: 间距编辑器
---

视觉间距编辑器允许您使用直观的框模型图示配置边距和内边距。精确的间距控制可确保在您的商店前端中实现一致的布局和舒适的阅读体验。打开任何元素的 **样式选项卡**，并查找 **间距** 部分以访问编辑器。

![间距编辑器](/static/core/admin/img/help/spacing-editor/spacing-editor.webp)

## 框模型图示

编辑器显示一个视觉框模型，包含三个嵌套的图层：

- **边距**（外环，通常显示为橙色）—— 元素边框外部的空间，用于将元素与相邻元素分隔开
- **内边距**（内环，通常显示为绿色）—— 元素边框与内容之间的空间
- **内容**（中心区域）—— 元素的实际内容，例如文本或图像

图示的每一侧（上、右、下、左）都有一个可拖动的控件和一个数字输入框。向外拖动控件以增加数值，向内拖动以减少数值。您也可以直接点击某侧的数值以输入精确的数字。

## 边距和内边距选项卡

编辑器顶部的两个选项卡可在 **边距** 和 **内边距** 视图之间切换。当选择边距时，外环被高亮并可编辑。当选择内边距时，内环被高亮并可编辑。未激活的环仍可见以供参考，但会变暗。

两个选项卡共享相同的控件和单位选项，因此边距和内边距的配置流程是一样的。

## 每侧控件

每侧都有一个独立的数值输入框和单位选择器：

| 侧边 | 描述 |
|------|-------------|
| **上** | 元素上方的间距（边距）或内容上方的间距（内边距） |
| **右** | 元素或内容右侧的间距 |
| **下** | 元素或内容下方的间距 |
| **左** | 元素或内容左侧的间距 |

点击图示中任何侧边的数值以选择它，然后输入一个数字或使用上下箭头键以1为单位递增。按住 Shift 键的同时按箭头键可以10为单位递增。

## 单位

每个数值输入框旁边的单位选择器允许您选择测量单位：

| 单位 | 描述 |
|------|-------------|
| **px** | 像素。固定大小，跨设备保持一致。适合精确的小间距值。 |
| **em** | 相对于元素的字体大小。随着排版变化而缩放。 |
| **rem** | 相对于根字体大小。在整个页面中提供一致的缩放效果。 |
| **%** | 父元素宽度的百分比。适用于流体、响应式布局。 |
| **auto** | 让浏览器自动计算该值。通常用于通过左右边距实现水平居中。 |

选择一个与您意图相符的单位——使用 `px` 用于固定间距，使用 `rem` 用于与主题排版标记成比例缩放的间距，使用 `%` 用于必须适应容器宽度的布局。

## 链接边

图示中心的 **链接图标** 可切换链接模式：

- **链接**（连接的链条图标）—— 更改任何一侧的值会将所有四侧更新为相同的值。适用于统一的间距。
- **未链接**（断开的链条图标）—— 每侧独立控制。当您需要不同的上下和左右值时使用此模式。

点击链接图标以在模式之间切换。当您从未链接切换到链接时，所有四侧将被设置为最近编辑的侧的值。

## 快速预设

图示下方的一行预设按钮提供了一键间距配置：

| 预设 | 值 |
|--------|--------|
| **无** | 所有四侧为 0 |
| **小** | 适合紧凑布局和内联元素的紧凑间距 |
| **中** | 适用于卡片和部分的一般用途的平衡间距 |
| **大** | 适用于英雄区域和高强调部分的宽裕间距 |
| **XL** | 适用于全宽横幅和顶级页面部分的额外宽间距 |

预设适用于当前活动的选项卡（边距或内边距），并一次性设置所有四侧。应用预设后，您可以根据需要调整单个侧边。

## 出现位置

间距编辑器适用于所有支持布局间距的元素：

- **页面构建器** —— 在部分、容器、列和单个元素的样式选项卡和间距部分
- **页眉/页脚构建器** —— 行和小部件的间距控件，用于垂直和水平间距
- **菜单构建器** —— 菜单项的内边距和容器的边距设置

所有位置都使用相同的编辑器界面，确保在构建器中获得一致的体验。

## 小贴士

- 在您的页面中使用一致的间距值——选择 2-3 个标准尺寸并坚持使用，以获得干净、专业的布局。
- 将左右边距设置为 **auto**，以在父元素内水平居中固定宽度的元素。
- 如果您的主题使用响应式排版，请优先使用 `rem` 单位，这样间距将与文本大小成比例缩放。
- 使用链接模式快速设置统一的内边距，然后取消链接并根据需要微调单个侧边，如果内容需要不对称的间距。
- 避免在移动设备上使用过多的内边距——在窄视口宽度下测试您的间距，确保内容不会被挤压或过度填充。
