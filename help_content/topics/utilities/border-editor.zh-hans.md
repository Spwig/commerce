---
title: 边框编辑器
---

边框编辑器提供了对元素边框的精细控制，包括样式、颜色、每侧的宽度以及每个角落的圆角半径。它以一个浮动面板的形式打开，包含实时预览和两个标签页，分别用于基本设置和高级设置。

![边框编辑器](/static/core/admin/img/help/border-editor/border-editor.webp)

## 实时预览

编辑器顶部的预览框会实时显示您的边框更改。该框显示一个带有边框的矩形，内部写着“预览”，当您调整样式、颜色、宽度和半径值时，矩形会立即更新。

## 基本模式与高级模式

编辑器分为两个标签页：

| 标签 | 包含内容 |
|-----|-----------------|
| **基本** | 边框样式、颜色、宽度（带每侧控制）和边框圆角（带每个角落的控制） |
| **高级** | 单个角落圆角的精细调整以及实验性的角落形状属性 |

大多数边框工作都在基本标签页中完成。当您需要对单个角落进行精确控制或想尝试新的CSS功能时，可以使用高级标签页。

## 边框样式

一个包含九个选项的下拉菜单，控制边框线的外观：

| 样式 | 描述 |
|-------|-------------|
| **无** | 无边框（移除任何现有边框） |
| **实线** | 一条连续的线（默认） |
| **虚线** | 一系列短划线 |
| **点线** | 一系列圆点 |
| **双线** | 两条平行的实线 |
| **沟槽** | 一种凹陷的3D效果边框，看起来像是被压入表面 |
| **凸起** | 一种凸起的3D效果边框（与沟槽相反） |
| **内嵌** | 使元素看起来嵌入或被压入 |
| **外凸** | 使元素看起来凸起或弹出 |

将样式设置为“无”会完全移除边框，无论宽度或颜色设置如何。

## 边框颜色

一个文本输入字段，配有一个颜色选择器按钮。可以直接输入十六进制值（例如 `#3b82f6`），或点击颜色块以打开完整的颜色选择器，支持十六进制、RGB和HSL输入模式以及视觉颜色区域。默认颜色为黑色（`#000000`）。

## 边框宽度

控制边框的厚度（以像素为单位）。基本标签页显示四个单独的边输入：

| 边 | 输入 |
|------|-------|
| **上** | 数字输入，最小值为0 |
| **右** | 数字输入，最小值为0 |
| **下** | 数字输入，最小值为0 |
| **左** | 数字输入，最小值为0 |

标签旁边的**链接切换按钮**（链条图标）控制是否将所有四边链接：

- **已链接**（默认）——更改任何值会同时更新所有四边
- **未链接**——每边可以有不同的宽度，适用于仅底部边框或左侧强调边框等效果

## 边框圆角

控制每个角落的圆角。基本标签页显示四个角落输入：

| 角落 | 标签 |
|--------|-------|
| **左上** | TL |
| **右上** | TR |
| **左下** | BL |
| **右下** | BR |

**链接切换按钮**的工作方式与边框宽度相同：

- **已链接**（默认）——所有四个角落共享相同的圆角值
- **未链接**——每个角落可以有不同的圆角

常见圆角值：

| 值 | 效果 |
|-------|--------|
| 0px | 锐利的方形角落 |
| 4-8px | 轻微的圆角，适合卡片和按钮 |
| 12-16px | 明显的圆角，现代柔和的外观 |
| 50% | 完全圆形或药丸形状（取决于元素的尺寸） |

单位选择器支持px、em、rem和%用于宽度和圆角值。

## 角落形状（高级）

高级标签页包含一个实验性的**角落形状**属性。这个CSS特性控制圆角是使用标准的圆角形状还是更尖锐的“凹槽”形状。浏览器支持有限，当当前浏览器不支持此属性时，编辑器会显示兼容性警告。

## 底部操作

| 按钮 | 操作 |
|--------|--------|
| **重置** | 将所有值恢复为编辑器打开时的状态 |
| **取消** | 关闭编辑器而不应用更改 |
| **应用** | 保存边框设置并关闭编辑器 |

## 出现位置

边框编辑器在以下构建器中可用：

- **页面构建器** — 选择任何元素，打开样式标签页，然后点击边框部分
- **页眉/页脚构建器** — 为页眉部分、导航容器和页脚区域添加边框
- **菜单构建器** — 为菜单项和下拉容器设置边框

编辑器从画布上的实时元素中读取当前计算的边框样式，因此它总是以正确的现有值打开。

## 小贴士

- **适度使用边框** — 使用浅灰色的1px边框可以在不增加视觉重量的情况下在各部分之间创建干净的分隔。
- **结合圆角和阴影** — 通过阴影编辑器添加柔和的框阴影，可以产生精致的卡片效果。
- **尝试单侧边框** — 取消链接并仅设置底部或左侧边框，以创建强调线、部分分隔符或侧边栏指示器。
- **使用百分比圆角创建药丸形状** — 将按钮或徽章的所有角落设置为50%，以创建适应任何内容大小的药丸形状。
- **检查预览** — 实时预览框会立即更新，因此在应用更改之前可以自由实验。
