---
title: 間隔エディタ
---

視覚的な間隔エディタを使用して、直感的なボックスモデル図を使用してマージンとパディングを構成できます。正確な間隔の制御により、ショッピングサイト全体で一貫したレイアウトと快適な読みやすさを確保できます。任意の要素の **Styleタブ** を開き、**Spacing** セクションを探してエディタにアクセスしてください。

![Spacing Editor](/static/core/admin/img/help/spacing-editor/spacing-editor.webp)

## ボックスモデル図

エディタは、3つのネストされたレイヤーを持つ視覚的なボックスモデルを表示します:

- **Margin** (外側のリング、通常はオレンジ色で表示) — 要素の境界線の外側のスペースで、隣接する要素と分離します
- **Padding** (内側のリング、通常は緑色で表示) — 要素の境界線とそのコンテンツの間のスペース
- **Content** (中央の領域) — 要素の実際のコンテンツ、例えばテキストや画像

図の各側面（上、右、下、左）にはドラッグ可能なハンドルと数値入力があります。ハンドルを外側にドラッグして値を増加させたり、内側にドラッグして減少させたりできます。また、側面の値を直接クリックして正確な数値を入力することもできます。

## マージンとパディングタブ

エディタの上部にある2つのタブは、**Margin** と **Padding** のビューを切り替えます。Marginが選択されている場合、外側のリングがハイライトされ、編集可能です。Paddingが選択されている場合、内側のリングがハイライトされ、編集可能です。非アクティブなリングは参照のために表示されますが、薄く表示されます。

両方のタブは同じコントロールと単位オプションを共有しているため、マージンとパディングの設定ワークフローは同じです。

## 個別側面のコントロール

各側面には独立した値入力と単位セレクタがあります:

| 側面 | 説明 |
|------|-------------|
| **上** | 要素の上部（マージン）またはコンテンツの上部（パディング）のスペース |
| **右** | 要素またはコンテンツの右側のスペース |
| **下** | 要素またはコンテンツの下部のスペース |
| **左** | 要素またはコンテンツの左側のスペース |

図の任意の側面の値をクリックして選択し、数値を入力するか、上下の矢印キーを使用して1ずつ増減できます。矢印キーを押しながらShiftを押しながら押し続けると、10ずつ増減します。

## 単位

各値入力の隣にある単位セレクタを使用して、測定単位を選択できます:

| 単位 | 説明 |
|------|-------------|
| **px** | ピクセル。デバイスにかかわらず固定サイズです。正確で小さな間隔値に最適です。 |
| **em** | 要素のフォントサイズに対する相対値。タイポグラフィの変更に合わせてスケールします。 |
| **rem** | ルートフォントサイズに対する相対値。ページ全体で一貫したスケールを提供します。 |
| **%** | 親要素の幅のパーセンテージ。流動的でレスポンシブなレイアウトに適しています。 |
| **auto** | ブラウザが値を自動的に計算します。左右のマージンで水平方向の中央配置に一般的に使用されます。 |

目的に合った単位を選択してください — 固定のギャップには `px`、テーマのタイポグラフィトークンを尊重するスケーラブルな間隔には `rem`、コンテナの幅に合わせて変化するレイアウトには `%` を使用してください。

## 側面のリンク

図の中央にある **リンクアイコン** は、リンクモードを切り替えます:

- **リンク済み** (接続されたチェーンアイコン) — 任意の側面の値を変更すると、4つの側面すべてが同じ値に更新されます。一貫した間隔に便利です。
- **リンク解除** (切断されたチェーンアイコン) — 各側面は独立して制御されます。上/下と左/右の値が異なる必要がある場合に使用します。

リンクアイコンをクリックしてモードを切り替えます。リンク解除からリンク済みに切り替えると、4つの側面は最近編集された側面の値に設定されます。

## クイックプリセット

図の下にあるプリセットボタンの行は、ワンクリックで間隔設定を提供します:

| プリセット | 値 |
|--------|--------|
| **なし** | すべての側面で0 |
| **小** | 緊密なレイアウトやインライン要素に適したコンパクトな間隔 |
| **中** | カードやセクションの一般的な用途に適したバランスの取れた間隔 |
| **大** | ヒーロー領域や高強度セクションに適した余裕のある間隔 |
| **XL** | フル幅のバナーまたはトップレベルのページセクションに適した非常に広い間隔 |

プリセットは現在アクティブなタブ（MarginまたはPadding）に適用され、4つの側面すべてを一度に設定します。プリセットを適用した後、必要に応じて個々の側面を調整できます。

## 出現場所

間隔エディタは、レイアウト間隔をサポートするすべての要素で利用可能です:

- **ページビルダー** — セクション、コンテナ、カラム、および個々の要素の **Styleタブ**、**Spacingセクション**
- **ヘッダ/フッタビルダー** — 行とウィジェットの間隔コントロール、垂直および水平方向のギャップ
- **メニュービルダー** — メニューアイテムのパディングとコンテナのマージン設定

すべての場所で同じエディタインターフェースが使用されるため、ビルダー全体で一貫した体験が保証されます。

## ヒント

- ページ全体で一貫した間隔値を使用してください — 2〜3つの標準サイズを選び、クリーンでプロフェッショナルなレイアウトを維持してください。
- 左右のマージンを **auto** に設定して、固定幅の要素を親要素内に水平方向に中央に配置してください。
- テーマがレスポンシブなタイポグラフィを使用している場合、間隔に `rem` 単位を優先してください。これにより、テキストサイズに比例して間隔がスケールします。
- 一様なパディングを迅速に設定するためにリンクモードを使用し、コンテンツが非対称な間隔を必要とする場合、リンクを解除して個々の側面を微調整してください。
- モバイルでの過剰なパディングを避けてください — 窄いビューポート幅で間隔をテストし、コンテンツが圧迫されすぎたり、過剰にパディングされたりしないようにしてください。
