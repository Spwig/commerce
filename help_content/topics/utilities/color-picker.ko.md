---
title: 색상 선택기
---

고급 색상 선택기는 여러 입력 방법과 테마 인식 사전 설정을 사용하여 색상을 선택할 수 있습니다. 이 선택기는 플랫폼 전반의 색상 속성이 사용되는 모든 곳에서 나타납니다 — 페이지 빌더, 헤더/푸터 빌더, 메뉴 빌더 및 카탈로그 관리자에서. 색상 스와치나 색상 입력 필드를 클릭하여 선택기를 열 수 있습니다.

![색상 선택기](/static/core/admin/img/help/color-picker/color-picker.webp)

## 색상 입력 방법

선택기는 색상을 정의하는 여러 가지 방법을 지원합니다:

| 방법 | 설명 | 예시 |
|--------|-------------|---------|
| **HEX** | 직접 6자리 HEX 코드를 입력합니다 | `#FF5733` |
| **RGB** | 빨강, 녹색, 파랑 슬라이더를 조정합니다 (각각 0-255) | `rgb(255, 87, 51)` |
| **HSL** | 색상(Hue)(0-360), 채도(Saturation)(0-100%), 밝기(Lightness)(0-100%)를 설정합니다 | `hsl(14, 100%, 60%)` |
| **RGBA** | RGB에 알파 투명도 채널을 추가합니다 | `rgba(255, 87, 51, 0.8)` |
| **HSLA** | HSL에 알파 투명도 채널을 추가합니다 | `hsla(14, 100%, 60%, 0.8)` |
| **시각적 스펙트럼** | 색상 스펙트럼 영역을 클릭하거나 드래그하여 시각적으로 선택합니다 | 지점 및 클릭 선택 |

선택기 하단의 텍스트 입력 필드에 값을 직접 입력할 수도 있습니다.

## 형식 선택기

선택기 상단의 드롭다운 메뉴를 통해 **HEX**, **RGB**, **RGBA**, **HSL**, **HSLA** 출력 모드 간 전환할 수 있습니다. 형식을 전환할 때 현재 색상은 자동으로 변환됩니다 — 값이 손실되지 않습니다. 작업 흐름이나 디자인 시스템 요구사항에 가장 적합한 형식을 선택하세요.

## 색상 사전 설정

스펙트럼 영역 아래에는 일반적인 색상에 대한 빠른 접근용 색상 스와치가 있는 행이 있습니다. 이 스와치는 **테마 인식**입니다: 현재 활성화된 테마의 기본, 보조, 강조, 중립 팔레트 색상으로 자동 반영됩니다. 이는 브랜드 일관성을 유지하면서 HEX 코드를 기억할 필요 없이 쉽게 할 수 있도록 도와줍니다.

사전 설정을 적용하려면 스와치를 클릭하세요. 선택기는 즉시 스펙트럼 및 입력 필드에 선택된 색상을 표시하도록 업데이트됩니다.

## 투명도 / 알파

RGBA 또는 HSLA 모드를 사용할 때 스펙트럼 아래에 수평 **알파 슬라이더**가 나타납니다. 드래그하여 투명도를 0% (완전히 투명)에서 100% (완전히 불투명)로 설정할 수 있습니다. 슬라이더 옆의 숫자 입력 필드를 통해 정확한 제어가 가능합니다.

반투명 색상은 오버레이, 호버 효과 및 층叠 디자인 요소에 유용합니다.

## 현재 색상 vs 새 색상 미리보기

선택기 하단에는 **현재** 적용된 색상과 **새로** 선택된 색상을 표시하는 두 개의 옆으로 배치된 상자가 있습니다. 이 비교는 변경 사항을 적용하기 전에 평가할 수 있도록 도와줍니다. **적용**을 클릭하여 새 색상을 수락하거나 선택기 외부를 클릭하여 취소하고 현재 값을 유지할 수 있습니다.

## 어디에 나타나나요

색상 선택기는 관리자 전반에서 공유되는 유틸리티입니다:

- **페이지 빌더** — 스타일 탭에서 요소 텍스트 색상, 배경 색상, 테두리 색상 및 호버 상태
- **헤더/푸터 빌더** — 위젯 텍스트, 배경, 아이콘 및 링크 색상
- **메뉴 빌더** — 메뉴 항목 링크 색상 및 호버/활성 상태 색상
- **카탈로그 관리자** — 제품 배지 색상 및 카테고리 강조 색상

색상 값을 입력할 수 있는 모든 필드는 동일한 선택기를 열기 때문에 모든 곳에서 경험은 일관됩니다.

## 팁

- 테마의 사전 설정 스와치를 사용하여 페이지와 구성 요소 전반에 걸쳐 브랜드 일관성을 유지하세요.
- 동일한 색상의 더 밝거나 더 어두운 변형을 만들려면 HSL 모드로 전환하세요 — Lightness 값을 조정하면 됩니다.
- 텍스트 입력 필드에서 HEX 코드를 복사하여 다른 필드에서 동일한 색상을 재사용하거나 디자이너와 공유할 수 있습니다.
- 이미지 및 히어로 섹션에 미묘한 오버레이 효과를 위해 RGBA와 감소된 투명도를 사용하세요.
- 선택기는 세션 동안 최근 사용한 색상을 기억하므로 자주 사용하는 사용자 정의 색상은 계속 접근 가능합니다.
- HEX 입력 필드에 지원되는 형식의 색상 값을 붙여넣으면 선택기는 자동으로 인식하고 변환합니다.
