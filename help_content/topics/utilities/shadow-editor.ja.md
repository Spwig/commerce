---
title: シャドウエディタ
---

シャドウエディタを使用すると、設定可能なボックスシャドウやテキストシャドウで、要素に深みと奥行きを加えることができます。シャドウは視覚的な階層を形成し、重要な要素に注目を向け、あなたのオンラインショップに洗練された、現代的な雰囲気を与えます。任意の要素の **スタイルタブ** を開き、**効果** グループを探してシャドウエディタにアクセスしてください。

![シャドウエディタ](/static/core/admin/img/help/shadow-editor/shadow-editor.webp)

## シャドウの種類

エディタの上部には2つのタブが提供されています。

- **ボックスシャドウ** — 要素のバウンディングボックス全体にシャドウを追加します。カード、ボタン、コンテナ、画像、セクションなどに使用してください。
- **テキストシャドウ** — 文字の後ろにシャドウを追加します。見出しや画像上に重ねたテキストに使用して、読みやすさを向上させます。

各タブには独立した設定があります。必要に応じて、同じ要素にボックスシャドウとテキストシャドウを同時に適用できます。

## シャドウのプロパティ

各シャドウレイヤーは以下のプロパティで定義されます。

| プロパティ | 説明 | 範囲 |
|----------|-------------|-------|
| **Offset X** | 要素からシャドウまでの水平距離 | -50px から 50px |
| **Offset Y** | 要素からシャドウまでの垂直距離 | -50px から 50px |
| **Blur Radius** | シャドウのエッジがどのくらい柔らかく拡散するか。値が大きいほどシャドウは柔らかくなります。 | 0px から 100px |
| **Spread Radius** | 要素に対してシャドウのサイズを拡大または縮小する（ボックスシャドウのみ） | -50px から 50px |
| **Color** | シャドウの色。カラーピッカーを使用して完全な不透明度をサポートした色を設定できます。 | 任意の色（アルファ値あり） |
| **Inset** | シャドウを要素の内側に表示するか外側に表示するかを切り替える（ボックスシャドウのみ） | On / Off |

スライダーを使用して値を調整するか、入力フィールドに直接正確な数値を入力してください。

## 複数のシャドウ

1つの要素に複数のシャドウレイヤーをスタックして、複雑で現実的な深み効果を作成できます。

- **+** ボタンをクリックして新しいシャドウレイヤーを追加します。
- 各レイヤーはシャドウリストの行として表示され、独自のコントロールが用意されています。
- レイヤーをドラッグして順序を変更できます — シャドウはリストの順序に従ってレンダリングされ、最初のレイヤーが上になります。
- 任意のレイヤーの **アイコン** を切り替えて、一時的にそれを非表示にできますが、設定は削除されません。
- **ゴミ箱アイコン** をクリックしてレイヤーを削除します。

濃い暗いシャドウと広く柔らかいシャドウを組み合わせることで、物理的な深みを模倣した自然な「浮かび上がった」効果を作成できます。

## シャドウプリセット

プリセットを使用すると、1クリックで一般的なシャドウスタイルを適用できます。

| プリセット | 説明 |
|--------|-------------|
| **Small** | カードや入力フィールドなど、わずかな高さを出すための微妙で近いシャドウ |
| **Medium** | ボタンやドロップダウンなど、インタラクティブな要素に適した中程度の深み |
| **Large** | モーダルやポップオーバーなどの浮かび上がった要素に適した目立つシャドウ |
| **Soft** | 高い不透明度で広いぼかしを伴う、やさしく拡散した光を模倣 |
| **Hard** | ぼかしが最小で不透明度が高いため、はっきりとしたエッジを模倣 |
| **Inset** | 押されたか凹んだような見た目を出すための内側のシャドウ |

プリセットを適用した後、個々のプロパティを調整して結果を微調整できます。

## 現在のシャドウと新しいシャドウの比較

エディタの下部に、**現在の**シャドウ（保存済み）と**新しい**シャドウ（変更中の設定）を表示する2つの比較ボックスがあります。この左右の表示により、変更を適用する前に差異を評価しやすくなります。**適用**をクリックして変更を確定するか、クリックして離すことで変更をキャンセルできます。

## 使用できる場所

シャドウエディタは次の場所で利用可能です。

- **ページビルダー** — セクション、コンテナ、カラム、個々の要素の **スタイルタブ** と **効果** グループ
- **ヘッダ/フッタビルダー** — ロゴ、検索バー、ナビゲーション項目などの要素のウィジェットレベルのシャドウ設定

効果スタイルグループをサポートするすべての要素にシャドウエディタのコントロールが表示されます。

## ヒント

- 多くの要素には、微妙なシャドウ（Small または Soft プリセット）を使用してください — 過度なシャドウはデザインがごちゃごちゃした印象を与えます。
- 近くの暗いシャドウと遠くの明るいシャドウを組み合わせることで、最も自然な高さ効果を作成できます。
- 入力フィールドやコンテナにインセットシャドウを使用すると、沈み込んでいるようなパネル効果を作成できます。
- テキストシャドウは最小限に留めてください — 1px のオフセットとわずかなぼかしで、画像背景上での読みやすさを向上させることができます。
- ご使用のテーマがダークモード切り替えをサポートしている場合、シャドウを明るい背景と暗い背景の両方でテストしてください。
