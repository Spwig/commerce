---
title: 그림자 편집기
---

그림자 편집기를 사용하면 설정 가능한 박스 그림자와 텍스트 그림자를 통해 요소에 깊이와 차원을 추가할 수 있습니다. 그림자는 시각적 계층 구조를 만들고 중요한 요소에 주목을 끌며, 가상 매장에 세련되고 현대적인 느낌을 줍니다. 요소의 **스타일 탭**으로 이동하여 **효과** 그룹을 찾아 그림자 편집기를 열 수 있습니다.

![그림자 편집기](/static/core/admin/img/help/shadow-editor/shadow-editor.webp)

## 그림자 유형

편집기는 상단에 두 개의 탭을 제공합니다:

- **박스 그림자** — 요소의 경계 상자 전체에 그림자를 추가합니다. 카드, 버튼, 컨테이너, 이미지, 섹션에 사용합니다.
- **텍스트 그림자** — 텍스트 문자 뒤에만 그림자를 추가합니다. 헤딩이나 이미지 위에 레이어된 텍스트에 사용하여 가독성을 향상시킵니다.

각 탭은 독립적인 설정을 가지고 있습니다. 필요할 경우 동일한 요소에 박스 그림자와 텍스트 그림자를 동시에 적용할 수 있습니다.

## 그림자 속성

각 그림자 레이어는 다음 속성으로 정의됩니다:

| 속성 | 설명 | 범위 |
|------|------|------|
| **X 오프셋** | 요소에서 그림자의 수평 거리 | -50px ~ 50px |
| **Y 오프셋** | 요소에서 그림자의 수직 거리 | -50px ~ 50px |
| **블러 반경** | 그림자 가장자리가 얼마나 부드럽고 확산되어 보이는지. 값이 클수록 더 부드러운 그림자가 됩니다. | 0px ~ 100px |
| **확산 반경** | 요소에 비례하여 그림자 크기를 확장하거나 축소합니다 (박스 그림자만 해당) | -50px ~ 50px |
| **색상** | 그림자 색상, 색상 선택기를 통해 완전한 불투명도를 지원합니다 | 모든 알파 값이 있는 색상 |
| **내부 그림자** | 요소 내부에 그림자를 렌더링하도록 전환 (박스 그림자만 해당) | 켜짐 / 꺼짐 |

슬라이더를 사용하여 값을 조정하거나 입력 필드에 직접 정확한 숫자를 입력할 수 있습니다.

## 여러 그림자

단일 요소에 여러 그림자 레이어를 쌓아 복잡하고 현실적인 깊이 효과를 만들 수 있습니다:

- **+** 버튼을 클릭하여 새 그림자 레이어를 추가합니다
- 각 레이어는 그림자 목록에서 행으로 나타나며 자체 컨트롤이 있습니다
- 레이어를 드래그하여 순서를 재정렬합니다 — 그림자는 목록 순서에 따라 렌더링되며, 첫 번째 레이어가 위에 표시됩니다
- 어떤 레이어의 **눈 아이콘**을 토글하여 일시적으로 숨길 수 있습니다. 설정은 삭제되지 않습니다
- **쓰레기 아이콘**을 클릭하여 레이어를 제거합니다

작고 어두운 그림자와 넓고 부드러운 그림자를 결합하면 물리적인 깊이를 모방하는 자연스러운 "들어 올린" 효과를 만들 수 있습니다.

## 그림자 사전 설정

사전 설정은 단일 클릭으로 일반적인 그림자 스타일을 적용할 수 있도록 해줍니다:

| 사전 설정 | 설명 |
|----------|------|
| **작은** | 약간의 높이를 위한 미묘한, 가까운 그림자 (카드, 입력 필드) |
| **중간** | 상호작용 요소 (버튼, 드롭다운)에 대한 중간 깊이 |
| **대형** | 떠 있는 요소 (모달, 팝오버)에 대한 두드러진 그림자 |
| **부드러운** | 낮은 불투명도와 넓은 블러로 부드럽고 확산된 빛 효과 |
| **경계가 뚜렷한** | 높은 불투명도와 최소 블러로 날카로운, 정의된 가장자리 |
| **내부** | 눌린 듯하거나 빠져 있는 듯한 효과를 위한 내부 그림자 |

사전 설정을 적용한 후 개별 속성을 조정하여 결과를 세부적으로 조정할 수 있습니다.

## 현재 vs 새 미리보기

편집기 하단의 두 비교 상자에서 **현재** 그림자(저장된 상태)와 **새로운** 그림자(대기 중인 변경 사항)가 표시됩니다. 이 옆으로 배치된 뷰는 변경 사항을 적용하기 전에 차이를 쉽게 평가할 수 있도록 해줍니다. **적용**을 클릭하여 승인하거나, 변경 사항을 버리려면 클릭하여 떠나면 됩니다.

## 위치

그림자 편집기는 다음 위치에서 사용할 수 있습니다:

- **페이지 빌더** — 섹션, 컨테이너, 컬럼 및 개별 요소의 스타일 탭, 효과 그룹
- **헤더/푸터 빌더** — 로고, 검색 바 및 네비게이션 항목과 같은 요소의 위젯 수준 그림자 설정

효과 스타일 그룹을 지원하는 모든 요소는 그림자 편집기 컨트롤을 표시합니다.

## 팁

- 대부분의 요소에 미묘한 그림자(Small 또는 Soft 사전 설정)를 사용하세요 — 과도한 그림자는 디자인에 혼잡해 보일 수 있습니다.
- 가까운 어두운 그림자와 먼 밝은 그림자를 결합하면 가장 자연스러운 높이 효과를 얻을 수 있습니다.
- 입력 필드와 컨테이너에 내부 그림자를 사용하면 "하중" 효과를 만드는 데 좋습니다.
- 텍스트 그림자는 최소화되어야 합니다 — 1px 오프셋과 약간의 블러는 이미지 배경에서 가독성을 향상시키면서도 오래된 느낌 없이도 좋습니다.
- 테마가 어두운 모드 전환을 지원하는 경우, 빛과 어두운 배경 모두에서 그림자를 테스트하세요.

기억하세요: 마크다운 포맷, 이미지 경로, 코드 블록 및 기술 용어는 보존 규칙에 따라 정확히 유지하세요.
