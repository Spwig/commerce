---
title: 背景編輯器
---

背景編輯器讓您可以完全控制元素的背景，共有四種類型：實色、漸變、圖片和視頻。它還支援獨立的正常狀態和懸停狀態，讓您可以創建互動式的視覺效果。打開任何元素的 **樣式選項卡**，並查找 **背景** 區域以使用編輯器。

![背景編輯器](/static/core/admin/img/help/background-editor/background-editor.webp)

## 正常狀態與懸停狀態

在背景編輯器的頂部，有一個切換按鈕可在 **正常** 和 **懸停** 狀態之間切換。每個狀態都有其獨立的背景配置：

- **正常** — 頁面載入時顯示的預設背景
- **懸停** — 當訪客將滑鼠移到元素上時應用的背景

切換按鈕旁邊的兩個小預覽方塊會並排顯示目前的正常狀態和懸停狀態背景，讓您可以一目了然地看到對比效果。請先配置正常狀態，然後切換到懸停狀態，如果需要的話，可以添加互動效果。

## 背景類型

從編輯器面板頂部的圖標行中選擇背景類型：

| 類型 | 說明 |
|------|-------------|
| **顏色** | 使用單一顏色值的實色填充。應用快速且輕量。 |
| **漸變** | 兩種或多種顏色之間的平滑過渡，可以是線性或徑向漸變。包含內建預設值，如海洋、日落、森林和漿果。如需高階漸變編輯，請參閱 [漸變創建器](gradient-creator) 主題。 |
| **圖片** | 上傳的圖片或從媒體庫中選擇的圖片。支援定位、尺寸和重複控制。 |
| **視頻** | 帶有可選預覽圖片的背景視頻 URL，視頻載入時或在行動裝置上顯示。 |

每個狀態一次只能有一種類型處於活動狀態。切換類型不會刪除您之前的配置 — 您可以切換回來，您的設置將會保留。

## 顏色背景

當選擇顏色時：

Preserve all markdown formatting, image paths, code blocks, and technical terms.

- **Hex Input** — 直接輸入十六進制代碼（例如：`#1A1A2E`）
- **Color Swatches** — 點擊預設色塊以快速選擇。

色塊會根據主題進行調整，並反映您當前主題的配色方案。
- **Edit Button** — 開啟完整的顏色選擇器，包含色譜、滑塊和格式選項（請參閱 [Color Picker](color-picker) 主題）

顏色背景會即時渲染，且不會影響性能，使其非常適合用於區塊、卡片和容器。

## Gradient Backgrounds

當選擇 Gradient 時：

- **Preset Gradients** — 從內建的漸變中選擇：Ocean、Sunset、Forest、Berry 等
- **Custom Gradient** — 點擊 **Edit** 以開啟漸變創建器，您可以設置方向、類型（線性或徑向）和顏色停點
- **Angle Slider** — 調整線性漸變的方向（0-360 度）

漸變可以增加視覺深度，而不需要圖像資源，且能完美適配任何螢幕尺寸。

## Image Backgrounds

當選擇 Image 時：

- **Upload or Media Library** — 點擊圖片佔位符以上傳新圖片或從媒體庫中選擇一個圖片
- **Size** — 選擇 **Cover**（填滿元素，可能會裁剪）、**Contain**（適配於元素內）或自定義尺寸
- **Position** — 使用 9 點網格設置焦點（左上、中心、右下等）或輸入自定義的 X/Y 百分比
- **Repeat** — 啟用或停用重複。適合用於平磚圖案
- **Overlay** — 在圖片上添加可調整透明度的顏色疊層，有助於確保文字可讀性

在上傳前始終優化圖片。未壓縮的大型圖片會減慢頁面加載時間。

## Video Backgrounds

當選擇 Video 時：

- **Video URL** — 輸入 MP4 或 WebM 視頻文件的直接 URL
- **Poster Image** — 上傳一個在視頻加載時顯示的備用圖片，以及在不支持自動播放的設備上顯示
- **Autoplay / Loop / Muted** — 視頻背景默認會自動播放、循環並靜音，以符合瀏覽器政策

保持背景視頻簡短（10-30 秒），壓縮並視覺上保持簡約。

它們應當提升該部分的內容，而不會分散內容的注意力。

## 出現位置

背景編輯器可用於每個支援背景的元素：

- **頁面構建器** — 節點、容器、欄位和個別元素在樣式選項卡中都有背景選項
- **首頁/頁尾構建器** — 行背景和個別小工具背景
- **選單構建器** — 選單容器和下拉面板背景

相同的編輯器介面在各處都一樣，因此你在各個構建器中的工作流程會保持一致。

## 小技巧

- 在圖片背景上使用半透明顏色疊加，以確保文字在任何圖片內容下都能清晰可讀。
- 漸變預設是快速增加視覺吸引力的方法 — 選擇一個，然後自定義角度或顏色以符合您的品牌。
- 在互動卡片上設定正常和懸停背景，讓訪客在探索內容時獲得清晰的視覺反饋。
- 對於圖片背景，務必設定焦點，以確保圖片中最重要的部分在所有螢幕尺寸下都可見。
- 在載入速度至關重要的部分（例如首屏內容），優先使用顏色或漸變背景，而不是圖片。
- 測試視頻背景在行動裝置上的顯示效果 — 大多數行動瀏覽器會顯示預覽圖片而不是播放視頻。
