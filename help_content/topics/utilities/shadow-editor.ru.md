---
title: Теневой редактор
---

Теневой редактор позволяет добавлять глубину и объем элементам с помощью настраиваемых теней контура и текста. Тени создают визуальную иерархию, привлекают внимание к важным элементам и придают вашему магазину современный, аккуратный вид. Откройте вкладку **Style** любого элемента и найдите группу **Effects**, чтобы получить доступ к теневому редактору.

![Теневой редактор](/static/core/admin/img/help/shadow-editor/shadow-editor.webp)

## Типы теней

Редактор предоставляет две вкладки в верхней части:

- **Box Shadow** — добавляет тень вокруг всей области элемента. Используйте это для карточек, кнопок, контейнеров, изображений и разделов.
- **Text Shadow** — добавляет тень только за символами текста. Используйте это для заголовков или текста, наложенного на изображения, чтобы улучшить читаемость.

Каждая вкладка имеет свою независимую настройку. Вы можете применить и тень контура, и тень текста к одному и тому же элементу, если это необходимо.

## Свойства теней

Каждый слой тени определяется следующими свойствами:

| Свойство | Описание | Диапазон |
|----------|-------------|-------|
| **Offset X** | Горизонтальное расстояние тени от элемента | -50px до 50px |
| **Offset Y** | Вертикальное расстояние тени от элемента | -50px до 50px |
| **Blur Radius** | Определяет, насколько мягкой или размытой выглядит кромка тени. Более высокие значения создают более мягкие тени. | 0px до 100px |
| **Spread Radius** | Расширяет или сужает размер тени относительно элемента (только для тени контура) | -50px до 50px |
| **Color** | Цвет тени, настраиваемый с полной поддержкой прозрачности через цветовой выборник | Любой цвет с альфой |
| **Inset** | Переключатель для отображения тени внутри элемента вместо снаружи (только для тени контура) | Включено / Выключено |

Измените значения с помощью ползунков или введите точные числа напрямую в поля ввода.

## Несколько теней

Вы можете наложить несколько слоев теней на один элемент, чтобы создать сложные, реалистичные эффекты глубины:

- Нажмите кнопку **+**, чтобы добавить новый слой тени
- Каждый слой отображается как строка в списке теней с собственными контроллерами
- Перетаскивайте слои, чтобы изменить их порядок — тени отображаются в порядке списка, с первым слоем сверху
- Включите или выключите значок **глаза** для любого слоя, чтобы временно скрыть его, не удаляя настройку
- Нажмите значок **корзины**, чтобы удалить слой

Сочетание тесной, темной тени с широкой, мягкой тенью создает естественный эффект «поднятой» глубины, имитирующий физическую глубину.

## Предустановки теней

Предустановки позволяют быстро применить распространенные стили теней с одним щелчком:

| Предустановка | Описание |
|--------|-------------|
| **Маленькая** | Небольшая, близкая тень для небольшого подъема (карточки, поля ввода) |
| **Средняя** | Средняя глубина для интерактивных элементов (кнопки, выпадающие списки) |
| **Большая** | Выдающаяся тень для плавающих элементов (модалы, всплывающие окна) |
| **Мягкая** | Широкая размытость с низкой прозрачностью для мягкого, рассеянного свечения |
| **Твердая** | Минимальная размытость с высокой прозрачностью для резкого, четкого края |
| **Внутренняя** | Внутренняя тень для эффекта нажатия или вдавленности |

После применения предустановки вы можете настроить отдельные свойства, чтобы точно настроить результат.

## Текущий и новый предварительный просмотр

В нижней части редактора два сравнительных блока отображают **текущую** тень (сохраненную) и **новую** тень (ваши ожидающие изменения). Это боковое сравнение позволяет легко оценить разницу перед подтверждением. Нажмите **Применить**, чтобы принять изменения, или нажмите в другое место, чтобы отменить их.

## Где он используется

Теневой редактор доступен в следующих местах:

- **Page Builder** — вкладка **Style**, группа **Effects** для разделов, контейнеров, колонок и отдельных элементов
- **Header/Footer Builder** — настройки теней на уровне виджетов для элементов, таких как логотипы, строки поиска и элементы навигации

Любой элемент, поддерживающий группу стилей **Effects**, будет отображать контроллеры теневого редактора.

## Советы

- Используйте небольшие тени (предустановки **Small** или **Soft**) для большинства элементов — тяжелые тени могут сделать дизайн перегруженным.
- Сочетайте близкую, темную тень с дальней, светлой тенью для наиболее естественного визуального подъема.
- Внутренние тени хорошо работают с полями ввода и контейнерами для создания эффекта «вдавленной» панели.
- Тени текста должны быть минимальными — смещение на 1px с небольшой размытостью улучшает читаемость на фоне изображений, не выглядя устаревшим.
- Тестируйте свои тени на светлом и темном фоне, если ваша тема поддерживает переключение в темный режим.
