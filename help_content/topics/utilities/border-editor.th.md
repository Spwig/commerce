title: ตัวแก้ไขเส้นขอบ

ตัวแก้ไขเส้นขอบให้การควบคุมที่ละเอียดอ่อนสำหรับเส้นขอบขององค์ประกอบ รวมถึงสไตล์ สี ความกว้างของแต่ละด้าน และรัศมีมุมของแต่ละมุม มันจะเปิดขึ้นเป็นแผงลอยพร้อมกับการดูแบบเรียลไทม์และแท็บสองแท็บสำหรับการตั้งค่าพื้นฐานและขั้นสูง

![ตัวแก้ไขเส้นขอบ](/static/core/admin/img/help/border-editor/border-editor.webp)

## การดูแบบเรียลไทม์

กล่องดูตัวอย่างที่อยู่ด้านบนของตัวแก้ไขแสดงการเปลี่ยนแปลงของเส้นขอบของคุณแบบเรียลไทม์ กล่องนี้แสดงคำว่า "Preview" อยู่ภายในสี่เหลี่ยมที่มีเส้นขอบ ซึ่งจะอัปเดตทันทีเมื่อคุณปรับค่าสไตล์ สี ความกว้าง และรัศมีมุม

## โหมดพื้นฐาน vs ขั้นสูง

ตัวแก้ไขถูกจัดระเบียบเป็นสองแท็บ:

| แท็บ | สิ่งที่มันประกอบด้วย |
|-----|-----------------|
| **พื้นฐาน** | สไตล์เส้นขอบ สี ความกว้าง (พร้อมการควบคุมตามด้าน) และรัศมีเส้นขอบ (พร้อมการควบคุมตามมุม) |
| **ขั้นสูง** | การปรับแต่งรัศมีมุมแต่ละมุมอย่างละเอียด และคุณสมบัติรูปร่างมุม (Corner Shape) ที่ทดลองใช้ |

งานเกี่ยวกับเส้นขอบส่วนใหญ่ทำได้ทั้งหมดในแท็บพื้นฐาน แท็บขั้นสูงมีประโยชน์เมื่อคุณต้องการควบคุมมุมแต่ละมุมอย่างแม่นยำ หรือต้องการทดลองใช้คุณสมบัติ CSS ใหม่ๆ

## สไตล์เส้นขอบ

รายการแบบเลือกที่มีตัวเลือก 9 ตัวที่ควบคุมลักษณะของเส้นขอบ:

| สไตล์ | คำอธิบาย |
|-------|-------------|
| **ไม่มี** | ไม่มีเส้นขอบ (ลบเส้นขอบที่มีอยู่ทั้งหมด) |
| **ต่อเนื่อง** | เส้นเดียวที่ต่อเนื่องกัน (ค่าเริ่มต้น) |
| **เส้นประ** | ชุดของเส้นสั้นๆ |
| **จุด** | ชุดของจุดกลมๆ |
| **เส้นคู่** | เส้นต่อเนื่องสองเส้นที่ขนานกัน |
| **ลึก** | เส้นขอบที่มีลักษณะเหมือนถูกแกะสลัก 3D ที่ดูเหมือนถูกกดลงในพื้นผิว |
| **ยกขึ้น** | เส้นขอบที่มีลักษณะเหมือนถูกยกขึ้น 3D (ตรงข้ามกับลึก) |
| **ฝัง** | ทำให้องค์ประกอบดูเหมือนถูกฝังหรือถูกกดลง |
| **ยกขึ้น** | ทำให้องค์ประกอบดูเหมือนถูกยกขึ้นหรือเด่นขึ้น |

การตั้งค่าสไตล์เป็น "ไม่มี" จะลบเส้นขอบทั้งหมด ไม่ว่าค่าความกว้างหรือสีจะเป็นอย่างไร

## สีเส้นขอบ

ช่องป้อนข้อมูลข้อความที่จับคู่กับปุ่มเลือกสี คุณสามารถป้อนค่า hex ได้โดยตรง (เช่น `#3b82f6`) หรือคลิกที่สีเพื่อเปิดตัวเลือกสีทั้งหมดที่มีโหมดการป้อน hex, RGB และ HSL พร้อมกับพื้นที่แสดงสีแบบภาพ ค่าสีเริ่มต้นคือสีดำ (`#000000`)

## ความกว้างเส้นขอบ

ควบคุมความหนาของเส้นขอบในพิกเซล แท็บพื้นฐานแสดงช่องป้อนข้อมูลสำหรับแต่ละด้าน:

| ด้าน | ช่องป้อนข้อมูล |
|------|-------|
| **ด้านบน** | ช่องป้อนตัวเลข ค่าต่ำสุดคือ 0 |
| **ด้านขวา** | ช่องป้อนตัวเลข ค่าต่ำสุดคือ 0 |
| **ด้านล่าง** | ช่องป้อนตัวเลข ค่าต่ำสุดคือ 0 |
| **ด้านซ้าย** | ช่องป้อนตัวเลข ค่าต่ำสุดคือ 0 |

ปุ่มสลับลิงก์ (ไอคอนลิงก์) ที่อยู่ข้างป้ายกำกับควบคุมว่าทั้งสี่ด้านจะถูกเชื่อมโยงกันหรือไม่:

- **เชื่อมโยงกัน** (ค่าเริ่มต้น) — การเปลี่ยนค่าใดๆ จะอัปเดตทั้งสี่ด้านพร้อมกัน
- **ไม่เชื่อมโยงกัน** — แต่ละด้านสามารถมีความกว้างที่แตกต่างกัน ซึ่งมีประโยชน์สำหรับเอฟเฟกต์เช่นเส้นขอบด้านล่างเพียงอย่างเดียว หรือเส้นขอบด้านซ้ายเพื่อเน้น

## รัศมีเส้นขอบ

ควบคุมการโค้งของแต่ละมุม แท็บพื้นฐานแสดงช่องป้อนข้อมูลสำหรับมุมแต่ละมุม:

| มุม | ป้ายกำกับ |
|--------|-------|
| **ด้านบนซ้าย** | TL |
| **ด้านบนขวา** | TR |
| **ด้านล่างซ้าย** | BL |
| **ด้านล่างขวา** | BR |

ปุ่มสลับลิงก์ทำงานเหมือนกับความกว้างเส้นขอบ:

- **เชื่อมโยงกัน** (ค่าเริ่มต้น) — ทั้งสี่มุมจะใช้ค่ารัศมีเดียวกัน
- **ไม่เชื่อมโยงกัน** — แต่ละมุมสามารถมีรัศมีที่แตกต่างกัน

ค่ารัศมีที่พบบ่อย:

| ค่า | ผลลัพธ์ |
|-------|--------|
| 0px | มุมสี่เหลี่ยมที่คมชัด |
| 4-8px | การโค้งแบบเล็กน้อย ดีสำหรับการใช้งานกับการ์ดและปุ่ม |
| 12-16px | การโค้งที่เห็นได้ชัด ดูสมัยใหม่และนุ่มนวล |
| 50% | วงกลมหรือรูปร่างแบบยาวยาว (ขึ้นอยู่กับขนาดขององค์ประกอบ) |

ตัวเลือกหน่วยสนับสนุน px, em, rem และ % สำหรับค่าความกว้างและรัศมี

## รูปร่างมุม (ขั้นสูง)

แท็บขั้นสูงมีคุณสมบัติ **รูปร่างมุม** ที่ทดลองใช้ คุณสมบัติ CSS นี้ควบคุมว่ามุมที่โค้งจะใช้รูปร่างแบบมาตรฐานที่เป็นวงกลม หรือรูปร่างแบบมุมแหลมที่เรียกว่า "scoop" ความรองรับของเบราว์เซอร์มีจำกัด และตัวแก้ไขจะแสดงคำเตือนเกี่ยวกับความเข้ากันได้เมื่อเบราว์เซอร์ปัจจุบันไม่รองรับคุณสมบัตินี้

## การดำเนินการในส่วนท้าย

| ปุ่ม | การดำเนินการ |
|--------|--------|
| **รีเซ็ต** | คืนค่าทั้งหมดกลับไปยังสถานะเมื่อตัวแก้ไขถูกเปิด |
| **ยกเลิก** | ปิดตัวแก้ไขโดยไม่ประมวลผลการเปลี่ยนแปลง |
| **ใช้** | บันทึกการตั้งค่าเส้นขอบและปิดตัวแก้ไข |

## ที่ใดที่มันปรากฏ

ตัวแก้ไขเส้นขอบมีอยู่ในผู้สร้างหลายตัว:

- **ผู้สร้างหน้า** — เลือกองค์ประกอบใดๆ เปิดแท็บสไตล์ และคลิกส่วนเส้นขอบ
- **ผู้สร้างหัวข้อ/ท้ายข้อความ** — เพิ่มเส้นขอบให้กับส่วนหัว กล่องนำทาง และพื้นที่ท้ายข้อความ
- **ผู้สร้างเมนู** — กำหนดสไตล์เส้นขอบสำหรับรายการเมนูและกล่องแสดงผลแบบเลื่อนลง

ตัวแก้ไขจะอ่านสไตล์เส้นขอบที่คำนวณได้จากองค์ประกอบที่กำลังใช้งานบนแคนวาส ดังนั้นจึงเปิดเสมอพร้อมค่าที่ถูกต้องที่มีอยู่

## เคล็ดลับ

- **ใช้เส้นขอบอย่างประหยัด** — เส้นขอบบางๆ 1px ในสีเทาอ่อนจะสร้างการแยกส่วนที่สะอาดระหว่างส่วนต่างๆ โดยไม่เพิ่มน้ำหนักทางสายตา
- **รวมรัศมีกับเงา** — การรวมมุมที่โค้งกับเงาแบบกล่องอ่อน (ผ่านตัวแก้ไขเงา) จะสร้างเอฟเฟกต์การ์ดที่ดูเรียบร้อย
- **ลองใช้เส้นขอบด้านเดียว** — แยกการเชื่อมโยงด้านและตั้งค่าเพียงเส้นขอบด้านล่างหรือด้านซ้ายเพื่อใช้เป็นเส้นเน้น ส่วนต่างๆ หรือตัวชี้วัดด้านข้าง
- **ใช้รัศมีเปอร์เซ็นต์สำหรับรูปร่างยาวยาว** — ตั้งค่ามุมทั้งหมดเป็น 50% บนปุ่มหรือป้ายกำกับเพื่อสร้างรูปร่างยาวยาวที่ปรับขนาดได้ตามขนาดเนื้อหา
- **ตรวจสอบตัวอย่าง** — กล่องดูตัวอย่างแบบเรียลไทม์จะอัปเดตทันที ดังนั้นจึงทดลองได้อย่างอิสระก่อนที่จะใช้งาน
