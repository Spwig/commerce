---
title: 陰影編輯器
---

陰影編輯器讓您可以透過可配置的盒狀陰影和文字陰影，為元素添加深度和立體感。陰影可以建立視覺層次，吸引對重要元素的注意，並為您的商店帶來精緻、現代的感覺。打開任何元素的 **樣式選項卡**，並尋找 **效果** 群組以使用陰影編輯器。

![陰影編輯器](/static/core/admin/img/help/shadow-editor/shadow-editor.webp)

## 陰影類型

編輯器在頂部提供兩個選項卡：

- **盒狀陰影** — 在整個元素的邊界框周圍添加陰影。用於卡片、按鈕、容器、圖片和區塊。
- **文字陰影** — 僅在文字字符後面添加陰影。用於標題或疊加在圖片上的文字，以提高可讀性。

每個選項卡都有其獨立的配置。如果需要，您可以同時為同一個元素應用盒狀陰影和文字陰影。

## 陰影屬性

每個陰影層由以下屬性定義：

| 屬性 | 說明 | 範圍 |
|----------|-------------|-------|
| **X 偏移量** | 陰影與元素之間的水平距離 | -50px 到 50px |
| **Y 偏移量** | 陰影與元素之間的垂直距離 | -50px 到 50px |
| **模糊半徑** | 陰影邊緣的柔和或擴散程度。數值越高，陰影越柔和。 | 0px 到 100px |
| **擴散半徑** | 相對於元素擴展或收縮陰影大小（僅限盒狀陰影） | -50px 到 50px |
| **顏色** | 陰影顏色，透過顏色選擇器可配置完整透明度 | 任何帶有 alpha 值的顏色 |
| **內嵌** | 切換以在元素內部而非外部渲染陰影（僅限盒狀陰影） | 啟用 / 停用 |

使用滑塊調整值，或直接在輸入欄中輸入精確數字。

## 多個陰影

您可以將多個陰影層疊加在同一個元素上，以創建複雜且逼真的深度效果：

- 點擊 **+** 按鈕以添加新陰影層
- 每個層會以一行形式出現在陰影清單中，並有其自己的控制項
- 拖動層以重新排序 — 陰影會按照清單順序渲染，第一層在最上層
- 點擊任何層的 **眼睛圖示** 可臨時隱藏它，而不刪除配置
- 點擊 **垃圾桶圖示** 以移除一層

將緊密且深色的陰影與寬廣且柔和的陰影結合，可以產生自然的「浮起」效果，模擬物理深度。

## 陰影預設

快速套用預設讓您只需點擊一次即可添加常見的陰影樣式：

| 預設 | 說明 |
|--------|-------------|
| **小** | 稍微提升的陰影，適用於卡片和輸入欄 |
| **中** | 用於互動元素的中等深度（按鈕、下拉選單） |
| **大** | 用於浮動元素的顯著陰影（對話框、彈出視窗） |
| **柔** | 寬廣的模糊與低透明度，產生溫和的擴散光效 |
| **硬** | 最小的模糊與高透明度，產生銳利且明確的邊緣 |
| **內嵌** | 內部陰影，用於產生按下或凹陷的外觀 |

套用預設後，您可以調整個別屬性以微調結果。

## 當前與新預覽

在編輯器底部，兩個比較框顯示 **當前** 陰影（已保存）和 **新** 陰影（您的待定更改）。這種並排視圖讓您在提交更改前輕鬆評估差異。點擊 **套用** 以接受，或點擊其他地方以丟棄更改。

## 出現位置

陰影編輯器可在以下位置使用：

- **頁面構建器** — 在區塊、容器、欄位和個別元素的 **樣式選項卡** 的 **效果** 群組中
- **首頁/頁尾構建器** — 元素（如標誌、搜尋欄和導覽項目）的部件級陰影設定

任何支援 **效果** 樣式群組的元素都會顯示陰影編輯器控制項。

## 小技巧

- 為大多數元素使用微妙的陰影（小或柔預設）——過重的陰影可能會讓設計看起來混亂。
- 將一個靠近的深色陰影與一個遠處的淺色陰影結合，可以產生最自然的提升效果。
- 內嵌陰影在輸入欄和容器上效果很好，可以產生「凹陷」面板效果。
- 文字陰影應保持簡潔——1px 偏移量加上一點模糊可以提高圖片背景上的可讀性，而不顯過時。
- 如果您的主題支援深色模式切換，請在亮色和深色背景上測試您的陰影。
