---
title: 테두리 편집기
---

테두리 편집기는 요소의 테두리에 대한 세부적인 제어를 제공합니다. 스타일, 색상, 각 측면의 너비, 각 모서리의 반경을 포함합니다. 이 편집기는 실시간 미리보기와 기본 및 고급 설정을 위한 두 탭을 가진 부유 패널로 열립니다.

![테두리 편집기](/static/core/admin/img/help/border-editor/border-editor.webp)

## 실시간 미리보기

편집기 상단의 미리보기 상자에서는 테두리 변경 사항을 실시간으로 확인할 수 있습니다. 이 상자에는 테두리가 있는 사각형 안에 "Preview"라는 단어가 표시되며, 스타일, 색상, 너비, 반경 값을 조정할 때마다 즉시 업데이트됩니다.

## 기본 모드 vs 고급 모드

편집기는 두 탭으로 구성되어 있습니다:

| 탭 | 포함 내용 |
|-----|-----------------|
| **기본** | 테두리 스타일, 색상, 너비(측면별 제어), 테두리 반경(모서리별 제어) |
| **고급** | 개별 모서리 반경의 세부 조정 및 실험적인 모서리 형태 속성 |

대부분의 테두리 작업은 기본 탭에서 완료됩니다. 개별 모서리에 대한 정밀한 제어가 필요하거나 새로운 CSS 기능을 실험하고자 할 때 고급 탭이 유용합니다.

## 테두리 스타일

테두리 라인의 외형을 제어하는 9가지 옵션을 가진 드롭다운:

| 스타일 | 설명 |
|-------|-------------|
| **없음** | 테두리 없음(기존 테두리 제거) |
| **실선** | 단일의 연속선(기본값) |
| **점선** | 짧은 대시의 시리즈 |
| **점박이** | 원형 점의 시리즈 |
| **이중선** | 두 개의 병렬 실선 |
| **각인** | 표면에 파인 것처럼 보이는 3D 효과 테두리 |
| **각기** | 표면에서 떨어진 것처럼 보이는 3D 효과 테두리(각인과 반대) |
| **내측** | 요소가 내장되거나 눌린 것처럼 보이게 함 |
| **외측** | 요소가 떨어진 것처럼 보이게 함 |

스타일을 "없음"으로 설정하면 너비 또는 색상 설정과 관계없이 테두리가 완전히 제거됩니다.

## 테두리 색상

색상 선택기 버튼과 함께 텍스트 입력 필드입니다. 직접 16진수 값을 입력할 수 있습니다(예: `#3b82f6`) 또는 색상 스와치를 클릭하여 16진수, RGB, HSL 입력 모드와 시각적 색상 영역을 포함한 전체 색상 선택기를 열 수 있습니다. 기본 색상은 검정색(`#000000`)입니다.

## 테두리 너비

픽셀 단위로 테두리의 두께를 제어합니다. 기본 탭은 네 개의 측면 입력을 보여줍니다:

| 측면 | 입력 |
|------|-------|
| **상단** | 숫자 입력, 최소 0 |
| **우측** | 숫자 입력, 최소 0 |
| **하단** | 숫자 입력, 최소 0 |
| **좌측** | 숫자 입력, 최소 0 |

레이블 옆에 있는 **링크 토글 버튼**(사슬 아이콘)은 네 개의 측면이 연결되어 있는지 제어합니다:

- **연결됨**(기본값) — 값 변경 시 네 개의 측면이 동시에 업데이트됨
- **연결되지 않음** — 각 측면이 다른 너비를 가질 수 있으며, 하단만 테두리 또는 좌측 강조 테두리와 같은 효과에 유용합니다.

## 테두리 반경

각 모서리의 둥근 정도를 제어합니다. 기본 탭은 네 개의 모서리 입력을 보여줍니다:

| 모서리 | 라벨 |
|--------|-------|
| **상단 왼쪽** | TL |
| **상단 오른쪽** | TR |
| **하단 왼쪽** | BL |
| **하단 오른쪽** | BR |

**링크 토글 버튼**은 테두리 너비와 동일한 방식으로 작동합니다:

- **연결됨**(기본값) — 네 개의 모서리가 동일한 반경 값을 공유함
- **연결되지 않음** — 각 모서리가 다른 반경 값을 가질 수 있음

일반적인 반경 값:

| 값 | 효과 |
|-------|--------|
| 0px | 날카로운 직사각형 모서리 |
| 4-8px | 세부적인 둥근, 카드 및 버튼에 적합 |
| 12-16px | 눈에 띄는 둥근, 현대적이고 부드러운 느낌 |
| 50% | 전체 원 또는 약간 둥근 모양(요소의 크기에 따라 다름) |

단위 선택기는 너비와 반경 값에 px, em, rem, %를 지원합니다.

## 모서리 형태(고급)

고급 탭에는 실험적인 **모서리 형태** 속성이 포함되어 있습니다. 이 CSS 기능은 둥근 모서리가 표준 둥근 형태를 사용하는지 또는 더 각진 "scoop" 형태를 사용하는지 제어합니다. 브라우저 지원은 제한적이며, 현재 브라우저가 이 속성을 지원하지 않을 경우 편집기는 호환성 경고를 표시합니다.

## 하단 액션

| 버튼 | 액션 |
|--------|--------|
| **초기화** | 편집기를 열었을 때 모든 값을 초기 상태로 되돌림 |
| **취소** | 변경 사항을 적용하지 않고 편집기를 닫음 |
| **적용** | 테두리 설정을 저장하고 편집기를 닫음 |

## 어디서 사용되는가

테두리 편집기는 여러 빌더에서 사용할 수 있습니다:

- **페이지 빌더** — 요소를 선택하고 스타일 탭을 열고 테두리 섹션을 클릭합니다.
- **헤더/푸터 빌더** — 헤더 섹션, 네비게이션 컨테이너, 푸터 영역에 테두리를 추가합니다.
- **메뉴 빌더** — 메뉴 항목 및 드롭다운 컨테이너의 테두리를 스타일링합니다.

편집기는 캔버스 상의 라이브 요소에서 현재 계산된 테두리 스타일을 읽기 때문에 항상 올바른 기존 값을 가진 상태로 열립니다.

## 팁

- **테두리를 적게 사용하세요** — 가벼운 회색의 1px 테두리로 섹션 간 깔끔한 분리 효과를 만들 수 있습니다. 시각적 무게를 추가하지 않고도.
- **반경과 그림자 조합** — 둥근 모서리와 부드러운 박스 그림자(그림자 편집기를 통해)를 함께 사용하면 정교한 카드 효과를 만들 수 있습니다.
- **단일 측면 테두리 시도** — 측면을 연결해제하고 하단 또는 좌측 테두리만 설정하여 강조선, 섹션 구분선, 사이드바 지시자에 사용할 수 있습니다.
- **퍼센트 반경으로 약간 둥근 모양 만들기** — 버튼 또는 배지에 모든 모서리를 50%로 설정하여 내용 크기에 따라 적응하는 약간 둥근 모양을 만들 수 있습니다.
- **미리보기 확인** — 실시간 미리보기 상자는 즉시 업데이트되므로 적용하기 전에 자유롭게 실험해보세요.
