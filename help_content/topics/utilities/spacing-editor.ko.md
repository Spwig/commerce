---
title: 간격 편집기
---

시각적인 간격 편집기를 사용하여 직관적인 박스 모델 다이어그램을 통해 여백과 패딩을 구성할 수 있습니다. 정확한 간격 제어는 스토어프론트 전반에 걸쳐 일관된 레이아웃과 편안한 읽기 경험을 보장합니다. 요소의 **스타일 탭**으로 이동하여 **간격** 섹션을 찾아 편집기를 열 수 있습니다.

![간격 편집기](/static/core/admin/img/help/spacing-editor/spacing-editor.webp)

## 박스 모델 다이어그램

편집기는 세 개의 중첩된 층으로 구성된 시각적인 박스 모델을 표시합니다:

- **여백**(외부 링, 일반적으로 주황색으로 표시됨) - 요소의 테두리 외부에 있는 공간으로, 인접 요소와 분리됨
- **패딩**(내부 링, 일반적으로 녹색으로 표시됨) - 요소의 테두리와 내용 사이의 공간
- **내용**(중앙 영역) - 요소의 실제 내용, 예를 들어 텍스트 또는 이미지

다이어그램의 각 측면(상단, 우측, 하단, 좌측)에는 드래그 가능한 핸들과 숫자 입력이 있습니다. 핸들을 외부로 드래그하여 값을 증가시키거나 내부로 드래그하여 값을 감소시킬 수 있습니다. 또한 측면의 값을 직접 클릭하여 정확한 숫자를 입력할 수도 있습니다.

## 여백 및 패딩 탭

편집기 상단의 두 개의 탭은 **여백** 및 **패딩** 뷰 간 전환을 제공합니다. **여백**이 선택되면 외부 링이 강조되고 편집 가능합니다. **패딩**이 선택되면 내부 링이 강조되고 편집 가능합니다. 비활성 링은 참조용으로 보이지만 어두워집니다.

두 탭은 동일한 컨트롤과 단위 옵션을 공유하므로 여백 및 패딩 구성 작업 흐름은 동일합니다.

## 측면별 컨트롤

각 측면에는 독립적인 값 입력과 단위 선택기가 있습니다:

| 측면 | 설명 |
|------|-------------|
| **상단** | 요소 위의 공간(여백) 또는 내용 위의 공간(패딩) |
| **우측** | 요소 또는 내용의 오른쪽 공간 |
| **하단** | 요소 또는 내용 아래의 공간 |
| **좌측** | 요소 또는 내용의 왼쪽 공간 |

다이어그램에서 측면의 값을 클릭하여 선택한 후 숫자를 입력하거나 상/하 화살표 키를 사용하여 1 단위씩 증가시킬 수 있습니다. 화살표 키를 누르면서 Shift 키를 누르면 10 단위씩 증가합니다.

## 단위

각 값 입력 옆의 단위 선택기는 측정 단위를 선택할 수 있도록 합니다:

| 단위 | 설명 |
|------|-------------|
| **px** | 픽셀. 고정된 크기로, 모든 기기에서 일관된 결과를 제공합니다. 정확하고 작은 간격 값에 적합합니다. |
| **em** | 요소의 글꼴 크기에 비례합니다. 글꼴 크기 변경 시 스케일이 조정됩니다. |
| **rem** | 루트 글꼴 크기에 비례합니다. 전체 페이지에서 일관된 스케일을 제공합니다. |
| **%** | 부모 요소의 너비의 퍼센트. 유연하고 반응형 레이아웃에 유용합니다. |
| **auto** | 브라우저가 값을 자동으로 계산하도록 합니다. 좌우 여백을 사용하여 수평 중앙 정렬에 자주 사용됩니다. |

의도에 맞는 단위를 선택하세요. `px`는 고정된 간격에, `rem`은 테마의 글꼴 토큰을 존중하는 확장 가능한 간격에, `%`는 컨테이너 너비에 따라 적응해야 하는 레이아웃에 사용하세요.

## 측면 연결

다이어그램의 중심에 있는 **연결 아이콘**은 연결 모드를 전환합니다:

- **연결됨**(연결된 체인 아이콘) - 어떤 측면의 값을 변경하면 네 개의 측면이 모두 동일한 값으로 업데이트됩니다. 일관된 간격이 필요한 경우에 유용합니다.
- **연결되지 않음**(끊어진 체인 아이콘) - 각 측면은 독립적으로 제어됩니다. 상/하와 좌/우의 값이 다른 경우에 사용합니다.

연결 아이콘을 클릭하여 모드를 전환합니다. 연결되지 않은 상태에서 연결 상태로 전환할 경우, 네 개의 측면은 가장 최근에 편집된 측면의 값을 기준으로 설정됩니다.

## 빠른 사전 설정

다이어그램 아래에 있는 사전 설정 버튼은 한 번 클릭으로 간격 구성이 가능합니다:

| 사전 설정 | 값 |
|--------|--------|
| **없음** | 모든 측면에 0 |
| **작은 값** | 좁은 레이아웃 및 인라인 요소에 적합한 간격 |
| **중간 값** | 카드 및 섹션에 일반적으로 사용되는 균형 잡힌 간격 |
| **대형 값** | 히어로 영역 및 고강조 섹션에 적합한 넓은 간격 |
| **XL** | 전체 너비 배너 및 최상위 페이지 섹션에 적합한 추가로 넓은 간격 |

사전 설정은 현재 활성화된 탭(여백 또는 패딩)에 적용되며 네 개의 측면을 동시에 설정합니다. 사전 설정을 적용한 후 필요에 따라 개별 측면을 조정할 수 있습니다.

## 사용 위치

간격 편집기는 레이아웃 간격을 지원하는 모든 요소에 제공됩니다:

- **페이지 빌더** - 섹션, 컨테이너, 열 및 개별 요소의 스타일 탭, 간격 섹션
- **헤더/푸터 빌더** - 수직 및 수평 간격을 위한 행 및 위젯 간격 컨트롤
- **메뉴 빌더** - 메뉴 항목의 패딩 및 컨테이너의 여백 설정

모든 위치에서 동일한 편집기 인터페이스가 사용되므로 빌더 전반에 걸쳐 일관된 경험을 제공합니다.

## 팁

- 페이지 전반에 걸쳐 일관된 간격 값을 사용하세요 - 2~3개의 표준 크기를 선택하고 이를 유지하여 깔끔하고 전문적인 레이아웃을 만드세요.
- 요소의 좌우 여백을 **auto**로 설정하여 고정 너비 요소를 부모 요소 내에서 수평 중앙에 배치하세요.
- 테마가 반응형 글꼴을 사용하는 경우, 글꼴 크기에 비례하여 간격이 조정되도록 `rem` 단위를 선호하세요.
- 연결 모드를 사용하여 패딩을 빠르게 일관된 상태로 설정한 후, 콘텐츠가 비대칭 간격이 필요한 경우 연결을 해제하고 개별 측면을 세부 조정하세요.
- 모바일에서 과도한 패딩을 피하세요 - 좁은 화면 너비에서 간격을 테스트하여 콘텐츠가 압축되거나 지나치게 패딩된 상태가 되지 않도록 확인하세요.
