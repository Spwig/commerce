---
title: AI SEO 생성기
---

AI SEO 생성기는 AI 제공업체를 사용하여 제품의 메타 제목, 메타 설명 및 기타 SEO 콘텐츠를 자동으로 작성합니다. 모든 제품에 대해 수동으로 SEO 콘텐츠를 작성하는 대신, 단일 액션으로 정확하고 최적화된 콘텐츠를 대량으로 생성할 수 있습니다.

귀하의 상점에는 즉시 작동하는 내장된 SEO 생성기가 포함되어 있습니다. 더 강력한 언어 모델에 액세스하려면 Spwig 구성 요소 시장에서 추가 AI 제공업체 구성 요소를 설치할 수도 있습니다.

## SEO 생성기 작동 방식

SEO 생성기는 제품의 이름, 설명, 카테고리 및 속성을 읽고, 구성된 AI 제공업체를 사용하여 해당 제품에 맞춤화된 SEO 콘텐츠를 작성합니다. 생성된 콘텐츠는 제품의 SEO 필드에 직접 저장됩니다.

개별 제품의 SEO 콘텐츠를 생성하려면 제품 편집 페이지에서 생성하거나, 제품 목록에서 여러 제품에 대한 대량 생성을 실행할 수 있습니다.

## SEO 제공업체 설정

### 내장 제공업체 사용

귀하의 상점에는 제품 데이터에서 결정적으로 SEO 콘텐츠를 생성하는 내장된 SEO 제공업체가 포함되어 있습니다. 외부 API 키가 필요하지 않습니다. 새 설치에서는 자동으로 기본 제공업체로 설정됩니다.

활성화되었는지 확인하려면:

1. **마케팅 > SEO 제공업체**로 이동합니다.
2. 내장 제공업체가 **PRIMARY** 배지와 **ACTIVE** 상태로 표시되는지 확인합니다.
3. 제공업체가 표시되지 않으면 **+ SEO 제공업체 계정 추가**를 클릭하고 **Provider Key**(제공업체 키)를 `deterministic`으로 설정합니다.

### AI 제공업체 구성 요소 연결

더 풍부하고 맥락에 맞는 SEO 콘텐츠를 위해 Spwig 구성 요소 시장에서 OpenAI 또는 Claude 기반 제공업체와 같은 AI 제공업체 구성 요소를 설치할 수 있습니다.

1. 구성 요소 업데이트 시스템을 통해 제공업체 구성 요소를 설치합니다(상점 관리자에게 문의하세요).
2. **마케팅 > SEO 제공업체**로 이동합니다.
3. **+ SEO 제공업체 계정 추가**를 클릭합니다.
4. 양식을 작성합니다:

**공급업체 정보 섹션:**
- **사이트** — 가게를 선택하세요
- **공급업체 구성 요소** — 설치된 AI 공급업체 구성 요소를 선택하세요
- **공급업체 키** — 구성 요소 기반 공급업체를 사용할 때는 빈칸으로 남겨두세요
- **계정 이름** — 예: `OpenAI SEO 공급업체`와 같은 설명적인 이름

**설정 섹션:**
- **활성화** — 이 공급업체를 사용하도록 설정하려면 선택하세요
- **기본 공급업체** — 이 공급업체를 모든 SEO 생성의 기본 공급업체로 사용하도록 설정하려면 선택하세요
- **우선순위** — 대체 체인에서 먼저 시도되는 숫자가 작을수록 우선순위가 높습니다
- **설정** — 공급업체별 설정으로 JSON 객체(예: 모델 이름, 톤, 언어)를 사용합니다

5. **저장**을 클릭하세요

한 번에 하나의 공급업체만 기본으로 설정할 수 있습니다. 새로운 공급업체를 기본으로 설정하면 이전 기본 공급업체는 자동으로 우선순위가 낮아집니다.

### 공급업체 대체 체인

기본 공급업체가 실패할 경우(예: API 장애로 인해), 가게는 우선순위 순서에 따라 다음 활성화된 공급업체로 자동으로 대체됩니다. 이는 한 공급업체가 일시적으로 사용 불가능할 경우에도 SEO 생성이 계속 작동하도록 보장합니다.

## 제품을 위한 SEO 콘텐츠 생성

### 개별 제품

1. **제품 > 제품**으로 이동하고 임의의 제품을 열어주세요
2. 제품 폼의 **SEO** 섹션으로 스크롤하세요
3. **SEO 생성** 버튼을 클릭하세요
4. AI 공급업체는 제품의 세부 정보를 기반으로 메타 제목과 메타 설명을 생성합니다
5. 생성된 콘텐츠를 검토하고 필요 시 편집하세요
6. 변경 사항을 적용하려면 **저장**을 클릭하세요

### 일괄 생성

여러 제품의 SEO 콘텐츠를 생성하거나 업데이트하려면:

1. **제품 > 제품**으로 이동하세요
2. 업데이트하려는 제품을 확인란을 사용해 선택하거나 전체를 선택하세요
3. **동작** 드롭다운을 열어주세요
4. **SEO 콘텐츠 생성**(또는 유사한 동작 이름 — 드롭다운에서 정확한 라벨을 확인하세요)을 선택하세요
5. **시작**을 클릭하세요

Spwig은 생성 작업을 대기열에 추가하고 백그라운드에서 처리합니다. 몇 분 후 제품 목록을 새로고침하여 업데이트된 SEO 필드를 확인하세요.

## SEO 커버리지 검토

SEO 생성기에는 이미 SEO 콘텐츠가 있는 제품을 추적합니다.

SEO가 필요한 제품을 식별하려면 다음과 같이 하세요:

1. **제품 > 제품**으로 이동하세요
2. **SEO 상태** 필터(사용 가능한 경우)를 사용하여 메타 제목 또는 설명이 누락된 제품을 표시합니다
3. 해당 제품을 선택하고 대량 생성 작업을 실행합니다

## 제공업체 설정

SEO 제공업체 계정의 **설정** 필드는 제공업체별 구성이 포함된 JSON 객체를 수락합니다. 일반적인 옵션은 다음과 같습니다:

```json
{
  "language": "en",
  "tone": "professional",
  "max_title_length": 60,
  "max_description_length": 160
}
```

이러한 설정은 제공업체 구성 요소에 따라 달라집니다. 제공업체의 문서를 참조하여 사용 가능한 옵션의 전체 목록을 확인하세요.

## 여러 제공업체 관리

한 개 이상의 SEO 제공업체 계정이 구성되어 있는 경우, 제공업체 목록은 상태를 한눈에 확인할 수 있도록 표시합니다:

- **PRIMARY 배지** — 이 제공업체는 기본적으로 모든 SEO 생성에 사용됩니다
- **ACTIVE 배지** — 제공업체가 활성화되어 있습니다
- **INACTIVE 배지** — 제공업체가 비활성화되어 있으며 사용되지 않습니다

기본 제공업체를 변경하려면 승격하려는 제공업체 계정을 열고 **기본 제공업체** 확인란을 선택한 후 저장합니다. 시스템은 언제든지 하나의 제공업체만 기본 플래그를 보유하도록 자동으로 보장합니다.

## 팁

모든 마크다운 형식, 이미지 경로, 코드 블록 및 기술 용어를 유지하세요.

- 제품을 새로 생성한 직후에 즉시 SEO 콘텐츠를 생성하세요 — 몇 초만에 검색 엔진이 인덱싱할 수 있는 유용한 정보를 제공합니다.
- 제품 이름이 특별하거나 기술적인 경우, 출판 전에 AI로 생성된 메타 설명을 검토하세요; 생성기는 명확하고 설명적인 제품 이름과 함께 가장 잘 작동합니다.
- 제공자 설정에서 `"max_title_length": 60` 및 `"max_description_length": 160`을 설정하여 생성된 콘텐츠가 구글이 권장하는 문자 수 제한 내에 있도록 유지하세요.
- 대규모 제품 카탈로그를 가져온 후 일괄 SEO 생성을 실행하여 모든 SEO 필드를 빠르게 채우세요.
- 제품 설명을 크게 업데이트한 경우, SEO 콘텐츠를 다시 생성하여 메타 태그가 새 내용과 일치하도록 유지하세요.
- 내장된 결정론적 제공자는 좋은 시작점입니다; 카탈로그가 설정되고 더 풍부하고 자연스러운 느낌의 SEO 콘텐츠를 원할 때 AI 기반 구성 요소로 업그레이드하세요.
