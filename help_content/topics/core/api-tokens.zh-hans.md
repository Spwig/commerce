---
title: API令牌
---

API令牌是安全密钥，允许外部服务和集成与您的商店进行通信。当第三方服务或工具需要访问您的商店数据或触发操作时，它会在每次请求中发送API令牌，以便您的商店可以验证请求是否已授权。您可以在管理后台的API令牌部分创建和管理所有令牌。

## 您需要API令牌的情况

您通常需要创建API令牌的情况包括：

- 连接需要读取或写入您商店数据的外部服务或自动化工具
- 设置需要验证传入调用的Webhook接收器
- 为您的安装配置Spwig帮助系统
- 使用Spwig API构建自定义集成
- 在您的Spwig商店与其他系统之间同步数据

每个集成应拥有自己的令牌，这样您可以撤销一个服务的访问权限而不影响其他服务。

## 令牌类型

创建令牌时，您需要选择一个描述其用途的类型。该类型仅供您参考，有助于您跟踪每个令牌的作用。

| 类型 | 用途 |
|------|---------|
| **帮助系统** | 用于Spwig帮助文档系统 |
| **外部集成** | 第三方服务、自动化工具（例如Zapier）或数据同步工具 |
| **Webhook** | Webhook接收器或端点的认证 |
| **自定义** | 任何不属于上述类别的其他用途 |
| **实例同步** | Spwig安装或外部Spwig服务之间的同步 |

## 创建API令牌

1. 导航到 **设置 > API令牌**
2. 点击 **+ 添加API令牌**
3. 输入一个 **名称**，明确描述该令牌的用途（例如，`Zapier产品同步` 或 `帮助系统API`）
4. 选择适当的 **令牌类型**
5. 可选地添加 **描述**，提供更多关于集成的详细信息
6. 根据需要配置 **启用状态**、**过期日期** 和 **允许的IP地址**（请参见下文）
7. 点击 **保存**

保存后，完整令牌值会显示在详情页面上。**立即复制它** — 列表视图中出于安全考虑，令牌会被遮蔽，离开此页面后无法再次获取完整令牌。

## 令牌值安全性

Spwig 仅在您保存新令牌后立即显示完整的令牌值。之后，列表视图中只会显示遮蔽版本（例如，`spw_••••••••••••••••••••3f8a`）。

如果您丢失了令牌值，将无法恢复。您需要删除旧令牌并创建一个新令牌，然后更新使用它的集成。

**不要通过电子邮件、聊天消息或源代码共享令牌值。** 将它们视为密码。

## 设置过期日期

**过期时间**字段设置一个日期和时间，之后令牌将自动停止工作。如果令牌不应过期，请留空该字段。

过期日期适用于以下情况：

- 有固定结束日期的临时集成
- 提供给第三方的令牌，您希望自动移除访问权限
- 为高权限集成添加额外的安全层

当令牌过期后，使用它的请求将被拒绝。您可以通过更新**过期时间**日期或创建一个替代令牌来延长访问权限。

## 限制到特定 IP 地址

**允许的 IP**字段接受一个 IP 地址列表。当列表不为空时，令牌仅在请求来自这些地址之一时才有效。

例如，如果您的分析工具运行在 `203.0.113.42` 的服务器上，添加该 IP 地址意味着即使令牌泄露，也无法从其他位置滥用该令牌。

将**允许的 IP**留空以允许来自任何 IP 地址的请求。

## 监控令牌使用情况

令牌列表显示：

- **使用次数** — 令牌已被使用的总次数
- **最后使用时间** — 令牌最后一次用于发起请求的时间

这些字段可帮助您识别未使用的令牌（可撤销的候选对象），并发现异常活动。

使用次数的突然激增可能表明该令牌被除预期集成以外的其他人使用。

## 撤销令牌

要在不删除令牌的情况下立即停止其工作：

1. 点击令牌名称
2. 取消勾选 **Active**
3. 保存

该令牌仍会保留在您的列表中供参考，但会在后续请求中被拒绝。当您需要临时暂停某个集成以调查问题时，这非常有用。

要永久删除令牌：

1. 在列表中选择其复选框
2. 从操作菜单中选择 **Delete selected API tokens**
3. 确认删除

一旦删除，令牌将无法恢复。如果该集成仍需要访问权限，请创建一个新令牌并更新集成的配置。

## 示例：设置 Zapier 集成

**场景：** 您希望将商店连接到 Zapier 以自动化订单通知。

| Field | Value |
|-------|-------|
| Name | `Zapier Order Automation` |
| Token Type | External Integration |
| Description | Used by Zapier to read new orders and trigger notifications |
| Active | Yes |
| Expires At | *(leave blank)* |
| Allowed IPs | *(leave blank — Zapier uses dynamic IPs)* |

保存后，复制完整的令牌值，并将其粘贴到 Zapier 的 Spwig 集成设置中。

## Tips

保留所有 markdown 格式、图片路径、代码块和专业术语。

- 为每个令牌指定清晰且具体的名称 —— 在数月后排查问题时，`Shopify Sync v2` 比 `Token 3` 要有用得多
- 为每个集成创建一个令牌 —— 如果某个集成被入侵，你可以仅撤销该令牌，而不会影响其他令牌
- 为用于一次性项目或临时集成的令牌设置过期日期 —— 这可以降低被遗忘的令牌无限期保持激活状态的风险
- 每隔几个月查看一次令牌列表，停用任何 **最后使用** 日期异常陈旧的令牌，因为这些可能属于不再运行的集成
- 如果你怀疑某个令牌已被泄露，请立即停用它，创建一个替代令牌，并在重新启用访问权限前更新受影响的集成
