ยินดีต้อนรับสู่ Spwig! คู่มือนี้จะพาคุณไปชมแดชบอร์ดผู้ดูแลระบบของคุณอย่างรวดเร็ว และพาคุณผ่านขั้นตอนสำคัญในการเปิดร้านค้าออนไลน์ของคุณ

## แดชบอร์ด

หลังจากเข้าสู่ระบบ คุณจะพบกับ **Shop Dashboard** — ศูนย์ควบคุมสำหรับตรวจสอบประสิทธิภาพของร้านค้า

![Admin dashboard](/static/core/admin/img/help/getting-started-overview/admin-dashboard.webp)

แดชบอร์ดแสดง:
- **การดำเนินการที่จำเป็น** — คำสั่งซื้อที่รอการดำเนินการ การแจ้งเตือนสต็อกต่ำ ตะกร้าสินค้าที่ถูกทิ้งไว้ และอัปเดตที่พร้อมใช้งาน
- **รายได้จากการขาย** — กราฟรายได้และคำสั่งซื้อที่จัดกลุ่มตามวัน สัปดาห์ หรือเดือน
- **การนำทางแบบเร็ว** — กระโดดไปยังส่วนใดส่วนหนึ่งได้จากเมนูแบบเลื่อนลง

## Admin Sidebar

แถบด้านซ้ายจัดการทุกสิ่งเป็นส่วนต่างๆ:

| ส่วน | สิ่งที่อยู่ภายใน |
|---------|--------------|
| **สินค้า** | แคตตาล็อกสินค้า หมวดหมู่ แผนสมัครสมาชิก บัตรกำนัล |
| **คำสั่งซื้อ** | การจัดการคำสั่งซื้อ ตะกร้าสินค้า การจัดส่ง การตั้งค่าการจัดส่ง อัตราภาษี |
| **ลูกค้า** | โปรไฟล์ลูกค้า การวิเคราะห์ การตั้งค่า LTV |
| **การค้นหา** | การวิเคราะห์การค้นหา คำสังเคราะห์ การเปลี่ยนเส้นทาง |
| **การตลาด** | การส่งเสริมการขาย คูปอง ผู้ร่วมมือ ความภักดี บล็อก การประกาศ |
| **การตั้งค่า** | การตั้งค่าร้านค้า การชำระเงิน การออกแบบ อีเมล การแปลภาษา POS |

คลิกที่ **ปุ่มเมนู** (☰) ด้านล่างเพื่อพับแถบด้านข้างเพื่อให้ได้พื้นที่หน้าจอเพิ่มเติม

## ขั้นตอนแรก

### 1. ตั้งค่าการตั้งค่าร้านค้า

ไปที่ **การตั้งค่า > การตั้งค่าร้านค้า** เพื่อตั้งค่าตัวตนของร้านค้าของคุณ:
- ชื่อร้านค้าและข้อความย่อ
- URL ของเว็บไซต์
- Favicon และโลโก้
- ภาษาและเขตเวลาที่ใช้เป็นค่าเริ่มต้น

ดู [Configuring Store Settings](#) สำหรับรายละเอียด

### 2. เพิ่มสินค้าของคุณ

ไปที่ **สินค้า > สินค้าทั้งหมด** และคลิก **+ เพิ่มสินค้า**:
- ใส่ชื่อสินค้า คำอธิบาย และ SKU
- อัปโหลดรูปภาพผ่านไลบรารีสื่อ
- ตั้งค่าราคาและตัวเลือกการขาย
- ตั้งค่าการติดตามสต็อก

ดู [Adding a Product](#) สำหรับการผ่านขั้นตอนทั้งหมด

### 3. ตั้งค่าการชำระเงิน

ไปที่ **การตั้งค่า > แดชบอร์ดการชำระเงิน** เพื่อเชื่อมต่อกับผู้ให้บริการชำระเงิน:
- เลือกผู้ให้บริการที่มีอยู่ (Stripe, PayPal และอื่นๆ)
- ทำตามไกด์เพื่อตั้งค่าข้อมูล API ของคุณ
- ทดสอบการเชื่อมต่อก่อนเปิดใช้งานจริง

### 4. ตั้งค่าการจัดส่ง

ไปที่ **คำสั่งซื้อ > การจัดส่ง** เพื่อตั้งค่าตัวเลือกการจัดส่ง:
- สร้างวิธีการจัดส่งด้วยอัตราที่กำหนด
- กำหนดโซนการจัดส่งตามประเทศหรือภูมิภาค
- เชื่อมต่อกับอินทิเกรชันผู้ให้บริการเพื่อให้ได้อัตราที่เป็นจริง

ดู [Setting Up Shipping](#) สำหรับรายละเอียด

### 5. ปรับแต่งการออกแบบของคุณ

ไปที่ **การตั้งค่า > การออกแบบและการกำหนดธีม** เพื่อปรับแต่งร้านค้าของคุณ:
- เลือกธีมที่ใช้งานอยู่
- ปรับแต่งแบรนด์ (สี ฟอนต์ ระยะห่าง)
- สร้างหัวข้อและฟุตเตอร์ด้วยเครื่องมือสร้างแบบลากและวาง
- ตั้งค่าเมนูการนำทาง

ดู [Design & Themes](#) สำหรับรายละเอียด

### 6. เปิดใช้งานร้านค้า

เมื่อทุกอย่างพร้อม:
1. ตั้งสินค้าของคุณให้อยู่ในสถานะ **เผยแพร่**
2. ตรวจสอบ URL ร้านค้าใน การตั้งค่าร้านค้า
3. ทดสอบกระบวนการชำระเงินโดยการสั่งซื้อทดลอง
4. ตรวจสอบให้แน่ใจว่าสถานะเว็บไซต์แสดง **Site Active**

## เคล็ดลับ

- ใช้ปุ่ม **Help** ที่มุมขวาบนของหน้าใดๆ เพื่อขอความช่วยเหลือแบบมีบริบทสำหรับหน้าที่คุณอยู่
- บัตร **Actions Required** ในแดชบอร์ดเชื่อมโยงไปยังรายการที่ต้องการความสนใจโดยตรง — คลิกเพื่อไปยังที่นั่นทันที
- บันทึกแดชบอร์ดเป็นจุดเริ่มต้นประจำวันเพื่อติดตามสุขภาพของร้านค้า
