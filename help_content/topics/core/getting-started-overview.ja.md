---
title: Spwigストアをはじめよう
---

Spwigへようこそ！このガイドでは、管理ダッシュボードの概要と、オンラインストアを立ち上げるための基本的な手順をご紹介します。

## ダッシュボード

ログインすると、**ショップダッシュボード**が表示されます。ここはストアのパフォーマンスを監視するためのコントロールセンターです。

![管理ダッシュボード](/static/core/admin/img/help/getting-started-overview/admin-dashboard.webp)

ダッシュボードには以下が表示されます：
- **対応が必要なアクション** — 未処理の注文、在庫不足アラート、放棄されたカート、利用可能なアップデート
- **売上収益** — 日別、週別、月別の収益と注文のグラフ
- **クイックナビゲーション** — ドロップダウンから任意のセクションに移動

## 管理サイドバー

左側のサイドバーは、すべてをセクションごとに整理しています：

| セクション | 内容 |
|-----------|------|
| **商品** | 商品カタログ、カテゴリ、サブスクリプションプラン、ギフトカード |
| **注文** | 注文管理、カート、出荷、配送設定、税率 |
| **顧客** | 顧客プロフィール、分析、LTV設定 |
| **検索** | 検索分析、シノニム、リダイレクト |
| **マーケティング** | プロモーション、バウチャー、アフィリエイト、ロイヤルティ、ブログ、お知らせ |
| **設定** | ストア設定、決済、デザイン、メール、翻訳、POS |

画面下部の**ハンバーガーアイコン**（☰）をクリックすると、サイドバーを折りたたんで画面スペースを広げることができます。

## はじめの手順

### 1. ストア設定を構成する

**設定 > ストア設定**に移動して、ストアの基本情報を設定します：
- ストア名とキャッチフレーズ
- サイトURL
- ファビコンとロゴ
- デフォルト言語とタイムゾーン

詳細は[ストア設定の構成](#)をご覧ください。

### 2. 商品を追加する

**商品 > すべての商品**に移動し、**+ 商品を追加**をクリックします：
- 商品名、説明、SKUを入力
- メディアライブラリから画像をアップロード
- 価格とセールオプションを設定
- 在庫管理を構成

詳しい手順は[商品の追加](#)をご覧ください。

### 3. 決済を設定する

**設定 > 決済ダッシュボード**に移動して、決済プロバイダーを接続します：
- 利用可能なプロバイダー（Stripe、PayPalなど）を閲覧
- セットアップウィザードに従ってAPIクレデンシャルを入力
- 本番稼働前に接続をテスト

### 4. 配送を設定する

**注文 > 出荷**にアクセスして、配送オプションを設定します：
- 固定料金の配送方法を作成
- 国や地域ごとの配送ゾーンを定義
- オプションで配送業者の統合を接続し、リアルタイム料金を取得

詳細は[配送の設定](#)をご覧ください。

### 5. デザインをカスタマイズする

**設定 > デザインとテーマ**に移動して、ストアを個性的に仕上げます：
- アクティブなテーマを選択
- ブランディングをカスタマイズ（カラー、タイポグラフィ、スペーシング）
- ドラッグ＆ドロップビルダーでヘッダーとフッターを構築
- ナビゲーションメニューを構成

詳細は[デザインとテーマ](#)をご覧ください。

### 6. ストアを公開する

すべての準備が整ったら：
1. 商品のステータスを**公開済み**に設定
2. ストア設定でストアURLを確認
3. テスト注文を行い、チェックアウトフローをテスト
4. サイトのステータスが**サイト稼働中**と表示されていることを確認

## ヒント

- 各ページの右上にある**ヘルプ**ボタンを使用すると、表示中のページに関するコンテキストヘルプが表示されます。
- ダッシュボードの**対応が必要なアクション**カードは、注意が必要な項目に直接リンクしています。クリックするとすぐに該当箇所に移動できます。
- ダッシュボードをブックマークして、毎日のスタートポイントとして活用し、ストアの状態を把握しましょう。
