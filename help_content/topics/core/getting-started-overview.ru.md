Добро пожаловать в Spwig! Это руководство познакомит вас с панелью управления и проведёт через основные шаги запуска вашего интернет-магазина.

## Панель управления

После входа в систему вы попадаете на **Панель магазина** — ваш центр управления для мониторинга показателей магазина.

![Панель администрирования](/static/core/admin/img/help/getting-started-overview/admin-dashboard.webp)

На панели отображаются:
- **Требуемые действия** — ожидающие заказы, предупреждения о низком остатке товаров, брошенные корзины и доступные обновления
- **Выручка от продаж** — графики выручки и заказов с группировкой по дням, неделям или месяцам
- **Быстрая навигация** — переход в любой раздел через выпадающее меню

## Боковая панель администрирования

Боковая панель слева организует все функции по разделам:

| Раздел | Содержимое |
|--------|-----------|
| **Товары** | Каталог товаров, категории, планы подписок, подарочные карты |
| **Заказы** | Управление заказами, корзины, отправки, настройки доставки, налоговые ставки |
| **Клиенты** | Профили клиентов, аналитика, настройки LTV |
| **Поиск** | Аналитика поиска, синонимы, перенаправления |
| **Маркетинг** | Акции, ваучеры, партнёрская программа, программа лояльности, блог, объявления |
| **Настройки** | Настройки магазина, платежи, дизайн, электронная почта, переводы, POS |

Нажмите на **значок гамбургера** (☰) внизу, чтобы свернуть боковую панель и освободить больше места на экране.

## Первые шаги

### 1. Настройте параметры магазина

Перейдите в **Настройки > Настройки магазина**, чтобы задать идентичность вашего магазина:
- Название магазина и слоган
- URL сайта
- Фавикон и логотип
- Язык по умолчанию и часовой пояс

Подробнее см. [Настройка параметров магазина](#).

### 2. Добавьте товары

Перейдите в **Товары > Все товары** и нажмите **+ Добавить товар**:
- Введите название товара, описание и артикул (SKU)
- Загрузите изображения через Медиатеку
- Установите цены и параметры распродаж
- Настройте отслеживание запасов

Полное руководство см. в разделе [Добавление товара](#).

### 3. Настройте платежи

Перейдите в **Настройки > Панель платежей**, чтобы подключить платёжного провайдера:
- Просмотрите доступных провайдеров (Stripe, PayPal и другие)
- Следуйте мастеру настройки для ввода учётных данных API
- Протестируйте подключение перед запуском

### 4. Настройте доставку

Перейдите в **Заказы > Отправки**, чтобы настроить варианты доставки:
- Создайте способы доставки с фиксированными тарифами
- Определите зоны доставки по странам или регионам
- При необходимости подключите интеграции с перевозчиками для получения тарифов в реальном времени

Подробнее см. [Настройка доставки](#).

### 5. Настройте дизайн

Перейдите в **Настройки > Дизайн и тема**, чтобы персонализировать ваш магазин:
- Выберите активную тему
- Настройте брендинг (цвета, типографику, отступы)
- Создайте шапку и подвал сайта с помощью визуальных конструкторов
- Настройте навигационные меню

Подробнее см. [Дизайн и темы](#).

### 6. Запуск магазина

Когда всё готово:
1. Установите статус товаров на **Опубликовано**
2. Проверьте URL магазина в Настройках магазина
3. Протестируйте процесс оформления заказа, разместив тестовый заказ
4. Убедитесь, что статус сайта показывает **Сайт активен**

## Советы

- Используйте кнопку **Помощь** в правом верхнем углу любой страницы, чтобы получить контекстную справку по текущей странице.
- Карточки **Требуемые действия** на панели управления ведут непосредственно к элементам, требующим внимания — нажмите на них, чтобы перейти сразу туда.
- Добавьте панель управления в закладки как ежедневную стартовую точку для отслеживания состояния магазина.
