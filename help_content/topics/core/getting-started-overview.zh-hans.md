欢迎使用 Spwig！本指南将带您快速了解管理后台，并引导您完成上线在线商店的关键步骤。

## 管理面板

登录后，您将进入**商店管理面板**——这是监控商店运营状况的控制中心。

![Admin dashboard](/static/core/admin/img/help/getting-started-overview/admin-dashboard.webp)

管理面板显示以下内容：
- **待处理事项** — 待处理订单、低库存警报、废弃购物车以及可用更新
- **销售收入** — 按日、周或月分组的收入和订单图表
- **快速导航** — 通过下拉菜单跳转到任意功能模块

## 管理侧边栏

左侧侧边栏将所有功能按模块分类：

| 模块 | 包含内容 |
|------|---------|
| **产品** | 产品目录、分类、订阅计划、礼品卡 |
| **订单** | 订单管理、购物车、发货、配送设置、税率 |
| **客户** | 客户档案、数据分析、客户终身价值设置 |
| **搜索** | 搜索分析、同义词、重定向 |
| **营销** | 促销活动、优惠券、联盟营销、会员忠诚度、博客、公告 |
| **设置** | 商店设置、支付、设计、邮件、翻译、POS |

点击底部的**菜单图标**（☰）可以收起侧边栏，获得更多屏幕空间。

## 入门步骤

### 1. 配置商店设置

前往**设置 > 商店设置**来设定商店基本信息：
- 商店名称和标语
- 网站 URL
- 网站图标和 Logo
- 默认语言和时区

详情请参阅[配置商店设置](#)。

### 2. 添加产品

导航至**产品 > 所有产品**，点击**+ 添加产品**：
- 输入产品名称、描述和 SKU
- 通过媒体库上传图片
- 设置定价和促销选项
- 配置库存跟踪

完整操作指南请参阅[添加产品](#)。

### 3. 设置支付方式

前往**设置 > 支付面板**连接支付服务商：
- 浏览可用的支付服务商（Stripe、PayPal 等）
- 按照设置向导输入您的 API 凭证
- 上线前测试连接是否正常

### 4. 配置配送方式

访问**订单 > 发货**设置配送选项：
- 创建固定费率的配送方式
- 按国家或地区定义配送区域
- 可选择接入物流服务商集成以获取实时运费

详情请参阅[设置配送](#)。

### 5. 自定义商店设计

导航至**设置 > 设计与主题**来个性化您的商店：
- 选择当前使用的主题
- 自定义品牌元素（颜色、字体、间距）
- 使用拖放式构建器搭建页头和页脚
- 配置导航菜单

详情请参阅[设计与主题](#)。

### 6. 正式上线

一切准备就绪后：
1. 将产品状态设置为**已发布**
2. 在商店设置中验证网站 URL
3. 通过下一个测试订单来测试结账流程
4. 确认网站状态显示为**站点已激活**

## 提示

- 使用任意页面右上角的**帮助**按钮，获取当前页面的相关帮助信息。
- 管理面板中的**待处理事项**卡片直接链接到需要关注的项目——点击即可快速跳转。
- 将管理面板设为您的每日起始页，以便随时掌握商店运营状况。
