title: Memulai Toko Spwig Anda

Selamat datang di Spwig! Panduan ini memberi Anda tur cepat melalui dashboard admin Anda dan memandu Anda melalui langkah-langkah penting untuk meluncurkan toko online Anda.

## Dashboard

Setelah masuk, Anda akan berada di **Dashboard Toko** — pusat kendali untuk memantau kinerja toko.

![Dashboard Admin](/static/core/admin/img/help/getting-started-overview/admin-dashboard.webp)

Dashboard menampilkan:
- **Tindakan yang Diperlukan** — pesanan yang tertunda, peringatan stok rendah, keranjang yang ditinggalkan, dan pembaruan yang tersedia
- **Pendapatan Penjualan** — grafik pendapatan dan pesanan yang dikelompokkan berdasarkan hari, minggu, atau bulan
- **Navigasi Cepat** — lompat ke bagian mana pun dari dropdown

## Sidebar Admin

Sidebar di sebelah kiri mengorganisir semuanya ke dalam bagian:

| Bagian | Isi Bagian |
|---------|--------------|
| **Produk** | Katalog produk, kategori, rencana langganan, kartu hadiah |
| **Pesanan** | Manajemen pesanan, keranjang, pengiriman, pengaturan pengiriman, tarif pajak |
| **Pelanggan** | Profil pelanggan, analitik, pengaturan LTV |
| **Pencarian** | Analitik pencarian, sinonim, pengalihan |
| **Pemasaran** | Promosi, voucher, afiliasi, loyalitas, blog, pengumuman |
| **Pengaturan** | Pengaturan toko, pembayaran, desain, email, terjemahan, POS |

Klik ikon **hamburger** (☰) di bagian bawah untuk menekan sidebar dan menghemat ruang layar.

## Langkah Pertama

### 1. Konfigurasi Pengaturan Toko

Buka **Pengaturan > Pengaturan Toko** untuk menetapkan identitas toko Anda:
- Nama toko dan tagline
- URL situs
- Favicon dan logo
- Bahasa dan zona waktu default

Lihat [Konfigurasi Pengaturan Toko](#) untuk detail.

### 2. Tambahkan Produk Anda

Buka **Produk > Semua Produk** dan klik **+ Tambahkan Produk**:
- Masukkan nama produk, deskripsi, dan SKU
- Unggah gambar melalui Perpustakaan Media
- Tetapkan harga dan opsi penjualan
- Konfigurasikan pelacakan inventaris

Lihat [Menambahkan Produk](#) untuk walkthrough lengkap.

### 3. Atur Pembayaran

Buka **Pengaturan > Dashboard Pembayaran** untuk menghubungkan penyedia pembayaran:
- Lihat penyedia yang tersedia (Stripe, PayPal, dan lainnya)
- Ikuti wizard pengaturan untuk memasukkan kredensial API Anda
- Uji koneksi sebelum diluncurkan

### 4. Atur Pengiriman

Buka **Pesanan > Pengiriman** untuk mengatur opsi pengiriman:
- Buat metode pengiriman dengan tarif tetap
- Definisikan zona pengiriman berdasarkan negara atau wilayah
- Secara opsional hubungkan integrasi pengiriman untuk mendapatkan tarif real-time

Lihat [Mengatur Pengiriman](#) untuk detail.

### 5. Personalisasi Desain Anda

Buka **Pengaturan > Desain & Tema** untuk mempersonalisasi toko Anda:
- Pilih tema aktif
- Personalisasi branding (warna, tipografi, jarak)
- Bangun header dan footer Anda dengan pembuat drag-and-drop
- Konfigurasikan menu navigasi

Lihat [Desain & Tema](#) untuk detail.

### 6. Diluncurkan

Ketika semuanya siap:
1. Setel produk Anda ke status **Diterbitkan**
2. Verifikasi URL toko Anda di Pengaturan Toko
3. Uji alur checkout dengan memesan pesanan uji
4. Pastikan status situs Anda menunjukkan **Situs Aktif**

## Tips

- Gunakan tombol **Bantuan** di sudut kanan atas halaman untuk mendapatkan bantuan kontekstual untuk halaman yang Anda kunjungi.
- Kartu **Tindakan yang Diperlukan** di dashboard mengarah langsung ke item yang memerlukan perhatian — klik mereka untuk melompat langsung ke sana.
- Tandai dashboard sebagai titik awal harian Anda untuk melacak kesehatan toko.

Ingat: Kembali hanya dengan objek JSON yang berisi bidang "title" dan "content". Pertahankan semua format markdown, jalur gambar, blok kode, dan istilah teknis tepat seperti yang ditunjukkan di atas.
