# Spwig 스토어 시작하기

Spwig에 오신 것을 환영합니다! 이 가이드는 관리자 대시보드를 빠르게 살펴보고 온라인 스토어를 시작하는 필수 단계를 안내합니다.

## 대시보드

로그인 후 **Shop Dashboard**에 도착합니다 — 스토어 성능을 모니터링하는 컨트롤 센터입니다.

![관리자 대시보드](/static/core/admin/img/help/getting-started-overview/admin-dashboard.webp)

대시보드는 다음과 같은 내용을 표시합니다:
- **필요한 작업** — 대기 중인 주문, 낮은 재고 알림, 포기된 장바구니, 이용 가능한 업데이트
- **매출** — 일, 주, 월별로 그룹화된 매출 및 주문 차트
- **빠른 이동** — 드롭다운에서 임의의 섹션으로 이동

## 관리자 사이드바

왼쪽 사이드바는 모든 것을 섹션별로 정리합니다:

| 섹션 | 내용 |
|---------|--------------|
| **상품** | 상품 카탈로그, 카테고리, 구독 계획, 기프트 카드 |
| **주문** | 주문 관리, 장바구니, 배송, 배송 설정, 세율 |
| **고객** | 고객 프로필, 분석, LTV 설정 |
| **검색** | 검색 분석, 동의어, 리디렉션 |
| **마케팅** | 프로모션, 쿠폰, 파트너, 충성도, 블로그, 공지 |
| **설정** | 스토어 설정, 결제, 디자인, 이메일, 번역, POS |

왼쪽 하단의 **햄버거 아이콘**(☰)을 클릭하여 사이드바를 접어서 더 많은 화면 공간을 확보할 수 있습니다.

## 첫 단계

### 1. 스토어 설정 구성

**설정 > 스토어 설정**으로 이동하여 스토어 식별 정보를 설정합니다:
- 스토어 이름 및 슬로건
- 사이트 URL
- Favicon 및 로고
- 기본 언어 및 시간대

[스토어 설정 구성](#)을 참조하십시오.

### 2. 상품 추가

**상품 > 모든 상품**으로 이동하여 **+ 상품 추가**를 클릭합니다:
- 상품 이름, 설명 및 SKU 입력
- 미디어 라이브러리를 통해 이미지 업로드
- 가격 및 할인 옵션 설정
- 재고 추적 구성

[상품 추가](#)을 참조하여 전체 가이드를 확인하십시오.

### 3. 결제 설정

**설정 > 결제 대시보드**로 이동하여 결제 제공업체를 연결합니다:
- 사용 가능한 제공업체(Stripe, PayPal 등) 확인
- 설정 마법사를 따라 API 자격 증명 입력
- 실제 사용 전 연결 테스트

### 4. 배송 설정

**주문 > 배송**으로 이동하여 배송 옵션을 설정합니다:
- 고정 비용으로 배송 방법 생성
- 국가 또는 지역별 배송 지역 정의
- 실시간 요금을 위해 운송업체 통합 연결(선택 사항)

[배송 설정](#)을 참조하십시오.

### 5. 디자인 커스터마이즈

**설정 > 디자인 및 테마**로 이동하여 스토어를 커스터마이즈합니다:
- 활성 테마 선택
- 브랜딩 커스터마이즈(색상, 폰트, 간격)
- 드래그 앤 드롭 빌더로 헤더 및 푸터 구성
- 네비게이션 메뉴 구성

[디자인 및 테마](#)을 참조하십시오.

### 6. 운영 시작

모든 것이 준비되면:
1. 상품 상태를 **Published**(발행)로 설정
2. 스토어 설정에서 스토어 URL 확인
3. 테스트 주문을 통해 체크아웃 흐름 테스트
4. 사이트 상태가 **Site Active**(사이트 활성)인지 확인

## 팁

- 모든 페이지의 오른쪽 상단에 있는 **Help** 버튼을 사용하여 현재 페이지에 대한 컨텍스트별 도움말을 얻을 수 있습니다.
- 대시보드의 **필요한 작업** 카드는 주의가 필요한 항목으로 직접 연결됩니다 — 클릭하여 바로 이동할 수 있습니다.
- 대시보드를 매일의 시작점으로 북마크하여 스토어 건강 상태를 추적하세요.

기억하세요: JSON 객체로만 반환하고, "title" 및 "content" 필드만 포함하세요. 모든 마크다운 포맷, 이미지 경로, 코드 블록, 기술 용어는 위와 같이 정확히 유지하세요.
