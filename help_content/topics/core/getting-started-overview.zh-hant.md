歡迎使用 Spwig！本指南將帶您快速瀏覽管理後台，並引導您完成啟動線上商店的關鍵步驟。

## 管理面板

登入後，您將進入**商店管理面板**——這是監控商店營運狀況的控制中心。

![Admin dashboard](/static/core/admin/img/help/getting-started-overview/admin-dashboard.webp)

管理面板顯示以下內容：
- **待處理事項** — 待處理訂單、低庫存警示、廢棄購物車以及可用更新
- **銷售收入** — 按日、週或月分組的營收和訂單圖表
- **快速導覽** — 透過下拉選單跳轉到任意功能模組

## 管理側邊欄

左側側邊欄將所有功能按模組分類：

| 模組 | 包含內容 |
|------|---------|
| **產品** | 產品目錄、分類、訂閱方案、禮品卡 |
| **訂單** | 訂單管理、購物車、出貨、配送設定、稅率 |
| **客戶** | 客戶檔案、數據分析、客戶終身價值設定 |
| **搜尋** | 搜尋分析、同義詞、重新導向 |
| **行銷** | 促銷活動、優惠券、聯盟行銷、會員忠誠度、部落格、公告 |
| **設定** | 商店設定、付款、設計、電子郵件、翻譯、POS |

點擊底部的**選單圖示**（☰）可以收合側邊欄，取得更多螢幕空間。

## 入門步驟

### 1. 設定商店資訊

前往**設定 > 商店設定**來設定商店基本資訊：
- 商店名稱和標語
- 網站 URL
- 網站圖示和 Logo
- 預設語言和時區

詳情請參閱[設定商店資訊](#)。

### 2. 新增產品

導覽至**產品 > 所有產品**，點擊**+ 新增產品**：
- 輸入產品名稱、描述和 SKU
- 透過媒體庫上傳圖片
- 設定定價和促銷選項
- 設定庫存追蹤

完整操作指南請參閱[新增產品](#)。

### 3. 設定付款方式

前往**設定 > 付款面板**連接付款服務商：
- 瀏覽可用的付款服務商（Stripe、PayPal 等）
- 按照設定精靈輸入您的 API 憑證
- 上線前測試連線是否正常

### 4. 設定配送方式

前往**訂單 > 出貨**設定配送選項：
- 建立固定費率的配送方式
- 按國家或地區定義配送區域
- 可選擇整合物流服務商以取得即時運費

詳情請參閱[設定配送](#)。

### 5. 自訂商店設計

導覽至**設定 > 設計與主題**來個人化您的商店：
- 選擇目前使用的主題
- 自訂品牌元素（色彩、字型、間距）
- 使用拖放式建構器建立頁首和頁尾
- 設定導覽選單

詳情請參閱[設計與主題](#)。

### 6. 正式上線

一切準備就緒後：
1. 將產品狀態設定為**已發佈**
2. 在商店設定中驗證網站 URL
3. 透過下一筆測試訂單來測試結帳流程
4. 確認網站狀態顯示為**網站已啟用**

## 提示

- 使用任意頁面右上角的**說明**按鈕，取得當前頁面的相關說明資訊。
- 管理面板中的**待處理事項**卡片直接連結到需要關注的項目——點擊即可快速跳轉。
- 將管理面板設為您的每日起始頁，以便隨時掌握商店營運狀況。
