---
title: 設定商店設定
---

商店設定是配置商店身分、本地化、品牌和營運偏好設置的中心地點。前往 **設定 > 商店設定** 開始操作。

![商店設定一般選項卡](/static/core/admin/img/help/store-settings/store-settings-general.webp)

## 一般選項卡

**一般** 選項卡包含商店的核心身分設定。

### 商店身分

- **商店名稱** — 在頁面標題、電子郵件和管理員標頭中顯示的名稱。
- **標語** — 用於搜尋引擎優化和社交分享的商店簡短描述。
- **網站 URL** — 您商店的公開網址。這會用於電子郵件、地圖生成和連結建設。

### 聯絡資訊

- **聯絡電子郵件** — 接收訂單通知，並在客戶溝通中顯示。
- **電話號碼** — 在頁腳和電子郵件中顯示的可選支援電話號碼。

### 商業地址

輸入完整的地址（街道、城市、州、郵政編碼、國家）。這會用於：
- 運送起點計算
- 稅收計算
- 法律要求和發票

## 品牌

### 標誌

上傳您的商店標誌（建議使用 PNG 或 SVG，約 200x50 像素，透明背景）。標誌會出現在：
- 商店前端標頭
- 電子郵件範本
- 管理面板

### 標籤圖示

上傳一個方形的標籤圖示（ICO 或 PNG，32x32 像素）。它會顯示為：
- 瀏覽器標籤圖示
- 書籤圖示
- 手機主畫面圖示

## 本地化

### 預設語言

從 10 個支援的選項中選擇商店的主要語言：

| 語言 | 程式碼 |
|----------|------|
| 英語 | en |
| 西班牙語 | es |
| 法語 | fr |
| 德語 | de |
| 葡萄牙語 | pt |
| 日語 | ja |
| 簡體中文 | zh-hans |
| 繁體中文 | zh-hant |
| 俄語 | ru |
| 阿拉伯語 | ar |

預設語言控制管理介面語言和商店前端內容的預設值。

### 時區

選擇商店的時區以準確顯示訂單時間戳、預定的促銷活動和報告。

### 貨幣

- **預設貨幣** — 定價和會計的主要貨幣。
- **多種貨幣** — 啟用後讓客戶以他們偏好的貨幣查看價格，並使用即時匯率自動轉換。

在 **設定 > 商店設定 > 貨幣** 中配置其他貨幣。

## 電子商務設定

### 訪客結帳

允許在不創建帳戶的情況下進行購買：
- 更快的結帳流程
- 為首次購買者降低摩擦
- 收集較少的客戶資料

### 訂單編號格式

自定義訂單編號的顯示方式：
- **前綴** — 例如，"ORD-"
- **起始編號** — 第一個訂單編號
- **補零** — 例如，00001

### 庫存預設值

- **追蹤庫存** — 全局啟用庫存追蹤
- **低庫存閾值** — 警告等級（預設：5 個單位）
- **允許預訂** — 當庫存不足時接受訂單

## 電子郵件設定

### 發件人資訊

- **發件人姓名** — 顯示為電子郵件發件人（通常為商店名稱）
- **發件人電子郵件** — 必須來自已驗證的域名
- **回覆至電子郵件** — 客戶回覆的目標地

### 電子郵件提供商

在 **設定 > 電子郵件設定** 中配置您的電子郵件傳遞服務。支援的提供商包括 SMTP、SendGrid、Mailgun 和 Amazon SES。

## 法律與合規

添加商店政策以符合法律要求：

- **條款與條件** — 結帳時必需；客戶必須接受才能購買
- **隱私政策** — GDPR/CCPA 合規；連結在頁腳
- **退貨政策** — 定義您的退貨窗口、條件和退款流程

## 維護模式

啟用維護模式以臨時將商店離線：
- 向訪客顯示自定義的維護訊息
- 限制訪問管理員用戶
- 在重大更新或遷移期間很有用

## 稅務設定

在 **設定 > 稅務設定** 中配置稅務徵收：

1. **計算方法** — 按運送地址、賬單地址或商店位置
2. **稅率** — 按地區和產品稅類定義稅率
3. **稅務顯示** — 顯示含稅價格、不含稅價格或兩者

## 小提示

- 在處理任何訂單之前正確設置時區 — 這會影響所有時間戳和報告。
- 啟用訪客結帳以提高轉化率。
- 填寫商業地址以獲得準確的運送和稅務計算。
- 上傳標誌和標籤圖示以獲得專業的品牌體驗。
- 定期審查法律頁面以符合當地法規。

## 故障排除

**商店前端未顯示更改：**
- 清除瀏覽器快取
- 從管理面板清除快取
- 檢查是否意外啟用了維護模式

**電子郵件未發送：**
- 驗證電子郵件提供商設定
- 檢查"發件人電子郵件"域名是否已驗證
- 從提供商設置頁面測試連接

**貨幣轉換功能無法正常運作：**
- 驗證是否已連接匯率提供商
- 檢查匯率設定中的 API 權限
- 嘗試手動更新匯率
