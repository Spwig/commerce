分類頁面範本

分類頁面範本控制顧客瀏覽分類時產品的顯示方式 — 無論是經典的網格、水平清單、可滾動的輪播圖，還是戲劇性的特色版式。您可以設定一個適用於整個商店的預設範本，然後在任何需要不同處理的個別分類中覆蓋它。

## 設定位置

有兩個地方可以設定分類範本：

- **全站預設** — 點擊 **Design > Page Templates**，並滾動到 **Category** 區域。這裡選擇的範本會套用到每個分類頁面，除非被覆蓋。
- **每分類覆蓋** — 點擊 **Catalog > Categories**，打開任何分類，並點擊 **Display** 紙籤。在這裡設定範本會覆蓋該分類的全站預設。所有其他分類仍會使用預設。

## 六種範本

### 網格

網格範本以響應式卡片網格排列產品 — 這是最熟悉的線上購物版式。產品以均勻的卡片形式顯示，包含圖片、名稱、價格和加入購物車按鈕。

**最適合：** 一般用途商店、時尚、家居用品、電子產品 — 任何需要一致產品展示的商店。

| 選項 | 說明 |
|--------|-------------|
| **每行產品數** | 每行顯示多少卡片（2、3、4 或 5） |
| **顯示分類橫幅** | 在頁面頂部顯示分類的橫幅圖片 |
| **顯示子分類** | 在橫幅下方顯示可點擊的子分類標籤 |
| **顯示分類描述** | 顯示分類的描述文字 |
| **卡片圖片備用** | 當產品沒有圖片時顯示預設圖片 |

### 清單

清單範本以水平行顯示產品 — 圖片在一側，產品細節（名稱、描述、價格）在另一側。這讓每個產品有更多空間，讓顧客更容易一目了然地比較。

**最適合：** 需要短描述預覽的複雜產品，例如工具、家電或 B2B 目錄。

| 選項 | 說明 |
|--------|-------------|
| **顯示產品圖片** | 在每一行中包含產品圖片 |
| **圖片位置** | 圖片顯示在左側或右側 |
| **顯示簡短描述** | 在產品名稱下方顯示產品摘要 |
| **顯示分類橫幅** | 在頁面頂部顯示分類的橫幅圖片 |
| **顯示子分類** | 在橫幅下方顯示可點擊的子分類標籤 |
| **顯示分類描述** | 顯示分類的描述文字 |

### 輪播

輪播範本將產品分組為水平滾動的行 — 與串流平台上的「繼續瀏覽」部分類似。顧客可以滑動或點擊箭頭在頁面上瀏覽產品，而無需離開頁面。

**最適合：** 含有大量產品的大型目錄、禮品店、書店，或任何以瀏覽和發現為首要任務的商店。

| 選項 | 說明 |
|--------|-------------|
| **每視圖產品數** | 每次在桌面端顯示多少產品（3、4 或 5） |
| **行大小** | 每個輪播行顯示多少產品，再在下方開始新的一行（4、6、8 或 12） |
| **顯示導航箭頭** | 在每一行顯示上一頁/下一頁箭頭按鈕 |
| **顯示點選** | 在每一行下方顯示進度條和頁碼計數器（例如「1 / 3」） |
| **自動播放** | 自動進度輪播 |
| **自動播放速度** | 每次自動進度之間的時間（以毫秒為單位） |
| **無限循環** | 在到達最後一張幻燈片後返回第一張 |
| **顯示分類橫幅** | 在頁面頂部顯示分類的橫幅圖片 |
| **顯示子分類** | 在橫幅下方顯示可點擊的子分類標籤 |

### 马赛克

马赛克範本創建了一種類似 Pinterest 的錯落網格，其中每個卡片根據其圖片的自然高度進行調整。結果是一種多樣化、編輯風格的感覺，而不是均勻的網格。

**最適合：** 照片、藝術印刷品、手工製品、室內設計，或任何產品視覺特徵各異且希望以更動態方式展示的分類。

| 選項 | 說明 |
|--------|-------------|
| **欄位數** | 頁面上的欄位數（2、3 或 4） |
| **間距大小** | 卡片之間的空間 — 小、中或大 |
| **顯示分類橫幅** | 在頁面頂部顯示分類的橫幅圖片 |
| **顯示子分類** | 在橫幅下方顯示可點擊的子分類標籤 |
| **顯示分類描述** | 顯示分類的描述文字 |

### 特色

特色範本將少量產品作為大尺寸的特色卡片顯示在頁面頂部，剩餘產品則以支援網格顯示在下方。它會立即吸引顧客對您優先產品的注意。

**最適合：** 新品到貨區、暢銷產品頁、策展收藏或季節性分類，您希望突出特定產品。

| 選項 | 說明 |
|--------|-------------|
| **特色產品數** | 在頂部獲得特色待遇的產品數（1、2 或 3） |
| **特色版式** | 顯示特色卡片為全寬或作為側邊並排版式 |
| **顯示分類橫幅** | 在頁面頂部顯示分類的橫幅圖片 |
| **顯示子分類** | 在橫幅下方顯示可點擊的子分類標籤 |
| **顯示分類描述** | 顯示分類的描述文字 |

### 折叠

折叠範本專為 **根分類頁面** 設計 — 顯示所有頂層分類的頁面。它將每個分類顯示為互動面板；懸浮或點擊面板會使其水平擴展，而其他面板則會收縮。在個別分類頁面中，折叠範本會回退到標準網格。

**最適合：** 具有強烈分類圖片的商店，您希望分類頁面本身感覺沉浸式 — 時尚品牌、生活方式商店，以及具有強烈編輯身分的商店。

| 選項 | 說明 |
|--------|-------------|
| **每行面板數** | 每個折叠行最多顯示多少個分類，再在下方開始新的一行（4–8，預設 6） |
| **高度** | 折叠面板的高度 — 小、中、大或超大 |
| **展開比例** | 活動（懸浮）面板比非活動面板擴展的寬度比例（2–5×） |
| **過渡速度** | 動畫速度（以毫秒為單位） |
| **行動裝置版面** | 在行動裝置上如何顯示分類 — 垂直堆疊、輪播或網格 |
| **顯示產品數** | 在每個分類面板上顯示產品數的徽章 |

## 常見選項（所有範本）

每個範本都共享一組控制周圍頁面外觀的選項：

| 選項 | 說明 |
|--------|-------------|
| **顯示導覽列** | 在分類上方顯示「首頁 > 分類 > 名稱」導覽 |
| **顯示排序欄** | 顯示「排序方式」下拉選單，讓顧客重新排序結果 |
| **顯示產品數** | 顯示總產品數（例如「29 產品」） |
| **顯示幫助區** | 在底部顯示「找不到您要找的嗎？」區塊 |
| **預設排序** | 當顧客首次進入頁面時使用的排序順序 |
| **每頁產品數** | 每頁加載多少產品 |
| **分頁樣式** | 如何加載更多產品：分頁頁面、一個「載入更多」按鈕，或無限滾動 |

## 快速比較

使用此表格快速選擇合適的範本：

| 範本 | 每行產品數 | 視覺特徵 | 最佳產品類型 |
|----------|-----------------|------------------|--------------------|
| **網格** | 2–5 可配置 | 均勻、整潔 | 一切 — 安全預設 |
| **清單** | 1（全寬行） | 寬敞、資訊豐富 | 複雜或技術性產品 |
| **輪播** | 3–5 同時可見 | 可瀏覽、動態 | 大型目錄、發現 |
| **马赛克** | 2–4（錯落） | 編輯、多樣 | 視覺/手藝產品 |
| **特色** | 特色 + 下方網格 | 推廣 | 精選、新品到貨 |
| **折叠** | N/A（分類） | 沉浸式、戲劇性 | 根分類頁面 |

## 設定全站預設範本

1. 點擊 **Design > Page Templates**。
2. 滾動到 **Category** 區域。
3. 從 **Template** 下拉選單中選擇您偏好的範本。
4. 設定下拉選單下方出現的範本選項。
5. 設定常見選項（排序欄、導覽列、分頁等）。
6. 點擊 **Save Configuration**。

所選範本及其設置現在套用到您商店中的每個分類頁面。

## 覆蓋特定分類的範本

您可以為任何個別分類指定不同的範本，而不改變全站預設。

1. 點擊 **Catalog > Categories**。
2. 點擊您要自訂的分類。
3. 在編輯表單頂部點擊 **Display** 紙籤。
4. 在 **Template** 下方，選擇 **Custom** 以啟用每分類設定。
5. 選擇您要為此分類使用的範本。
6. 根據需要設定範本選項。
7. 點擊 **Save**。

只有這個分類會使用自訂範本。所有其他分類仍會使用全站預設。若要將分類恢復為預設，返回其 Display 紙籤，並將範本設為 **Use site default**。

## 示例：跨分類混合範本

這是一個實用示例，說明不同範本如何服務同一商店的不同部分：

| 分類 | 範本 | 原因 |
|----------|----------|--------|
| 所有分類（根頁面） | 折叠 | 戏劇性的視覺入口 |
| 服裝 | 網格（每行 4 個） | 快速、熟悉的瀏覽 |
| 購物 | 特色 | 立即突出顯示頂部優惠 |
| 書籍 | 輪播 | 大型目錄，易於瀏覽 |
| 手工禮品 | 马赛克 | 多樣的圖片尺寸，手藝感 |
| 工業工具 | 清單 | 顧客需要查看描述 |

## 小技巧

- 以 **網格** 範本作為您的全站預設起點 — 它幾乎適用於所有產品類型，且對購物者來說是最熟悉的版式。從這裡開始自訂個別分類。
- 小心使用 **特色** 範本。它在您能真正策展 1–3 個 standout 產品時效果最佳。如果所有產品看起來同等重要，特色處理會失去效果。
- **折叠** 範本只在根分類頁面產生完整的視覺效果。如果您將其分配給產品分類，顧客會看到一個普通的網格 — 所以請保留它用於您的頂層分類清單。
- 認真設定 **每頁產品數**。24–36 個產品每頁對網格和马赛克範本效果良好。對於輪播，行大小和每視圖產品數設定會更直接地控制此體驗。
- 在父分類上啟用 **顯示子分類**，讓顧客可以直接跳轉到更窄的選擇，而無需使用排序或過濾工具。
- 在行動裝置上測試您選擇的範本。馬赛克和輪播範本在小螢幕上看起來非常不同 — 使用 **行動版面** 選項在折叠範本中，並檢查馬赛克欄位在手機上是否過於狹窄。
- 當使用清單範本並啟用 **顯示簡短描述** 時，請確保您的產品實際上已設定簡短描述。空的描述欄位看起來是無意的。
