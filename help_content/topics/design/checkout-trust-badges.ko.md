---
title: 체크아웃 신뢰 배지
---

신뢰 배지는 고객이 구매가 안전하다는 점을 안심시키기 위해 체크아웃 중에 나타나는 작은 아이콘과 텍스트입니다. 결제 시점에서의 망설임을 줄여 전환율을 크게 향상시킬 수 있습니다. **템플릿 구성** 페이지에서 모든 체크아웃 템플릿에 나타나는 배지들을 완전히 구성할 수 있습니다.

## 배지 편집기 사용 방법

1. **설정 > 디자인 및 테마**로 이동하여 디자인 대시보드를 엽니다.
2. **페이지 템플릿**을 클릭하여 템플릿 구성 페이지를 엽니다.
3. 체크아웃 템플릿 선택기 및 그 옵션 아래에 위치한 **체크아웃 신뢰 배지** 섹션으로 스크롤합니다.

## 작동 방식

여기서 구성한 배지는 **모든 체크아웃 템플릿**(Accordion, Multi-Step, Single Page, Express)에 나타납니다. 활성화된 체크아웃 템플릿에 대해 **신뢰 배지 표시** 옵션이 활성화된 경우에만 나타납니다. 체크아웃 템플릿 옵션에서 신뢰 배지가 비활성화되면, 여기서 구성한 내용과 관계없이 나타나지 않습니다.

기본적으로 두 개의 배지가 포함되어 있습니다:

| 아이콘 | 텍스트 |
|------|------|
| Lock | Secure Checkout |
| Shield | Data Protected |

## 배지 추가

1. 기존 배지 아래에 있는 **+ Add Badge** 버튼을 클릭합니다.
2. 새 배지 행이 기본 아이콘과 빈 텍스트 필드가 있는 상태로 나타납니다.
3. 드롭다운에서 아이콘을 선택합니다. 30개 이상의 아이콘(보호막, 잠금, 인증서, 배송 트럭 등)이 제공됩니다.
4. 텍스트 필드에 배지 텍스트를 입력합니다(최대 60자).

**6개의 배지**까지 추가할 수 있습니다. 한도에 도달하면 **Add Badge** 버튼이 자동으로 비활성화됩니다.

## 배지 편집

각 배지 행에는 다음과 같은 항목이 포함되어 있습니다:

- **드래그 핸들** — 왼쪽의 점선 그립으로 순서를 재정렬합니다.
- **아이콘 미리보기** — 현재 선택된 아이콘을 표시합니다.
- **아이콘 드롭다운** — 사용 가능한 아이콘 중에서 선택합니다. 새 아이콘을 선택하면 미리보기가 즉시 업데이트됩니다.
- **텍스트 필드** — 체크아웃 페이지에서 아이콘 옆에 표시되는 라벨입니다.
- **삭제 버튼** — 오른쪽의 X 버튼을 클릭하면 배지가 삭제됩니다.

행 내에서 아이콘 또는 텍스트를 직접 변경합니다. 변경 내용은 **구성 저장**을 클릭할 때까지 메모리에 보관됩니다.

## 배지 순서 조정

왼쪽의 핸들을 잡고 배지를 드래그하여 새 위치에 놓습니다. 여기서 설정한 순서는 체크아웃 페이지에 배지가 나타나는 순서입니다.

## 배지 제거

어떤 배지 행의 오른쪽에 있는 **X** 버튼을 클릭하여 제거합니다. 배지는 즉시 편집기에서 제거됩니다. 변경 사항을 유지하려면 **구성 저장**을 클릭하세요.

## 변경 사항 저장

변경 사항을 완료한 후 페이지 상단 또는 하단의 **구성 저장** 버튼을 클릭합니다. 저장이 성공적으로 완료되면 확인 메시지가 나타납니다. 모든 변경 사항은 즉시 실제 체크아웃 페이지에 적용됩니다.

## 사용 가능한 아이콘

아이콘 드롭다운은 신뢰 신호에 적합한 아이콘의 정리된 집합을 포함합니다:

- **보안** — Lock, Shield, Shield Check, User Shield, Key
- **품질** — Check Circle, Certificate, Award, Medal, Star
- **만족도** — Heart, Thumbs Up, Handshake
- **결제** — Credit Card, Money, Percent
- **배송** — Truck, Fast Shipping, Returns, Exchange
- **지원** — Support (headset), Phone, Email, Clock
- **디지털** — Lightning, Download, Infinity
- **기타** — Gift, Globe, Leaf (Eco), Recycle

## 팁

- 배지를 간결하게 유지하세요. "Secure Checkout"과 같은 짧은 텍스트가 긴 문장보다 더 효과적입니다.
- 2~4개의 배지가 이상적인 수량입니다. 너무 많은 배지는 혼잡해 보일 수 있고 효과를 잃을 수 있습니다.
- 고객의 주요 우려 사항을 해결하는 배지를 선택하세요. 대부분의 매장에서는 보안 및 반품이 가장 큰 망설임 요소입니다.
- 국제적으로 판매하는 경우 "Worldwide Shipping" 또는 "Multi-Currency Support"와 같은 배지를 고려하세요.
- 체크아웃 템플릿의 **Show trust badges** 토글 옵션과 함께 신뢰 배지 옵션을 사용하세요. 이 옵션을 해제하면 모든 배지를 삭제하지 않고 쉽게 숨길 수 있습니다.
