---
title: 页面构建器
---

页面构建器是一个可视化的拖放编辑器，无需编写代码即可创建丰富且响应式的页面。从包含39个组件的库中添加元素，使用强大的工具进行样式设置，设置动画和可见性规则，并通过完整的版本历史记录发布。

![页面构建器](/static/core/admin/img/help/page-builder/builder-overview.webp)

## 构建器界面

构建器有四个主要区域：

| 区域 | 位置 | 用途 |
|------|----------|---------|
| **工具栏** | 顶部栏 | 设备预览（桌面/平板/手机），撤销/重做，页面设置，保存草稿，发布 |
| **元素库** | 左侧边栏 | 浏览并拖动39个元素，这些元素按9个类别组织 |
| **画布** | 中心 | 实时所见即所得编辑区域 —— 在您进行更改时立即查看更改 |
| **属性面板** | 右侧边栏 | 编辑所选元素的内容、样式、动画和高级设置 |

## 元素库

元素按类别进行组织。从库中拖动任何元素到画布上，即可将其添加到您的页面中。

| 类别 | 元素 |
|----------|----------|
| **布局** | 容器，分隔符，英雄部分，模态弹窗，导航菜单，间隔符 |
| **基本** | 标题，文本，按钮，图标 |
| **内容** | 博客文章轮播，博客文章网格，FAQ折叠面板，相关文章，客户评价 |
| **媒体** | 图片，图片画廊，图片折叠面板，视频嵌入 |
| **表单** | 联系表单，表单，新闻订阅 |
| **营销** | 倒计时器，CTA横幅，特色博客横幅，忠诚度横幅，促销横幅，信任徽章，优惠券代码显示 |
| **电子商务** | 分类展示，礼品卡促销，产品轮播，产品网格，产品列表，评论展示，促销产品，商店定位器 |
| **社交** | 社交链接 |
| **导航** | 搜索栏 |

### 容器和嵌套

**容器**元素是复杂布局的基础。容器可以包含其他元素 —— 包括其他容器 —— 让您可以构建多列网格和嵌套结构。使用容器的布局预设快速设置常见的列排列（50/50，33/33/33，25/75等）。

## 添加元素

1. 在左侧边栏中找到您想要的元素
2. **拖动**它到画布上，并将其放置在您想要的位置
3. 元素可以放置在现有元素之间，或放在容器内
4. 蓝色插入线显示元素将被放置的位置
5. 放置后，元素会自动选中，并打开属性面板

您也可以通过在画布上向上或向下拖动元素来重新排序元素。

## 编辑内容

在画布上选择任何元素以在右侧面板中打开其属性。**内容**选项卡显示特定于该元素类型的字段。

![属性面板](/static/core/admin/img/help/page-builder/properties-panel.webp)

例如：
- **标题** — 文本，HTML标签（H1–H6），对齐方式，锚点ID
- **图片** — 图片源（媒体库），替代文本，链接，尺寸
- **按钮** — 标签，URL，样式变体，图标
- **产品网格** — 数据源，列数，每页产品数，排序顺序
- **英雄部分** — 标题，副标题，描述，背景，行动号召按钮

可翻译的内容字段显示一个翻译图标 — 点击它为多语言商店添加翻译。

## 样式设置元素

**样式**选项卡为每个元素提供视觉控制。每个部分打开一个专用的实用程序编辑器。

![样式选项卡](/static/core/admin/img/help/page-builder/style-tab.webp)

| 部分 | 控制内容 | 实用程序 |
|---------|-----------------|---------|
| **字体** | 字体族，大小，粗细，行高，字间距，文本样式 | 字体编辑器 |
| **颜色** | 文本颜色，支持十六进制/RGB/HSL输入和主题标记 | 颜色选择器 |
| **背景** | 固定颜色、渐变、图像或视频背景，包括悬停状态 | 背景编辑器 |
| **边框** | 每边的边框宽度、样式、颜色和半径 | 边框编辑器 |
| **间距** | 边距和内边距，带有可视化的盒模型编辑器 | 间距编辑器 |
| **效果** | 带预设和多层支持的框阴影，透明度滑块 | 阴影编辑器 |

每个实用程序都在其自己的帮助主题中进行了说明 — 搜索“颜色选择器”、“背景编辑器”等以了解更多信息。

## 动画

**动画**选项卡允许您为元素添加运动效果。

### 进入动画

当元素滚动到视图中时触发：

| 动画 | 描述 |
|-----------|-------------|
| 淡入 | 逐渐出现 |
| 滑入（上/下/左/右） | 从某个方向滑入 |
| 缩放进入 | 从小到大增长 |
| 弹跳进入 | 弹跳到位置 |
| 脉冲/摇动/弹跳/闪烁/旋转 | 吸引注意力的效果 |

配置 **持续时间**（0.3s–1.5s）、**延迟**（0–1s）、**时间函数**（ease, ease-in, ease-out, linear）和 **重复**（一次或无限）。

### 悬停动画

当访客将鼠标悬停在元素上时触发：

| 效果 | 描述 |
|--------|-------------|
| 缩放放大/缩小 | 放大或缩小 |
| 抬起 | 向上漂浮 |
| 旋转（顺时针/逆时针） | 顺时针或逆时针旋转 |
| 变亮/淡出 | 改变亮度或透明度 |
| 阴影扩大 | 阴影扩大 |
| 带阴影的抬起 | 带着扩大的阴影上升 |
| 脉冲缩放/倾斜/边框发光 | 特殊效果 |

配置 **持续时间**、**时间** 和 **强度**（微妙、正常、强烈）。

## 高级设置

**高级**选项卡提供精细的控制：

### 可见性规则

根据条件控制元素何时显示或隐藏：

- **用户状态** — 已登录、未登录、新客户、回头客
- **设备** — 桌面、平板、手机
- **时间** — 日期范围、一天中的时间、一周中的某天
- **客户组** — VIP、批发等
- **购物车价值** — 最低或最高购物车总额
- **地理位置** — 国家、地区
- 还有20多种其他规则类型

规则可以用AND/OR逻辑组合，以实现复杂的定位。

### 自定义CSS

| 字段 | 用途 |
|-------|---------|
| **元素ID** | 用于锚链接或CSS定位的唯一ID |
| **自定义CSS类** | 要应用的附加类 |
| **自定义CSS样式** | 用于一次性覆盖的内联CSS |
| **数据属性** | 作为键值对的自定义data-*属性 |
| **Z-Index** | 用于重叠元素的堆叠顺序 |

## 发布工作流程

页面使用带有完整版本历史记录的草稿/发布系统：

| 状态 | 含义 |
|--------|---------|
| **草稿** | 正在进行的工作 —— 不对访客可见 |
| **已发布** | 在您的商店中实时显示 |
| **已归档** | 从网站中移除但保留 |

### 工作原理

1. 在构建器中进行更改 —— 它们被保存为 **草稿**
2. 点击 **保存草稿** 以保存但不发布
3. 点击 **发布** 使当前草稿生效
4. 每次发布都会创建一个 **版本快照**
5. 您可以 **恢复** 任何之前的版本（工具栏中的时钟图标）

这意味着您可以自由实验 —— 您的实时页面在您明确发布之前保持不变。

## 页面模板

通过使用模板节省时间：

- **保存为模板** — 保存任何页面的设计作为可重复使用的模板
- **从模板创建** — 从现有模板开始创建新页面
- **模板类别** — 按用途（落地页、关于、产品展示等）组织模板

模板捕获完整的页面结构，包括所有元素、内容和样式。

## 响应式设计

使用工具栏中的设备预览按钮查看页面在不同屏幕尺寸下的显示效果：

- **桌面** — 全宽布局
- **平板** — 中等视口
- **手机** — 窄视口

元素会根据其容器设置自动重新排列。您还可以使用可见性规则在特定设备上显示或隐藏特定元素。

## 小贴士

- **从容器开始** — 大多数布局都从容器开始，以创建列和结构。使用布局预设进行常见排列。
- **使用英雄部分作为页面标题** — 英雄元素在一个组件中提供标题、副标题、背景图片和CTA按钮。
- **发布前预览** — 点击预览以查看访客将看到的确切内容，然后在您满意时发布。
- **使用可见性规则进行个性化** — 向已登录与未登录的访客显示不同的内容，或针对特定客户群体。
- **保持动画微妙** — 每个页面部分使用一两个进入动画看起来专业。太多的动画可能会让人感到不知所措。
- **为容器命名** — 使用元素ID字段为容器贴标签（例如，"hero-section"，"features"），以便在复杂页面中轻松找到它们。
- **在所有设备上测试** — 使用设备预览在发布前检查您的布局在桌面、平板和手机上的显示效果。
- **利用模板** — 将您最好的页面设计保存为模板，以加快未来页面的创建速度。
