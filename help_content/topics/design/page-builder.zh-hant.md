---
title: 頁面構建器
---

頁面構建器是一個視覺化的拖放編輯器，可讓您在不編寫程式碼的情況下創建豐富且具響應能力的頁面。從包含 39 個組件的元件庫中添加元素，使用強大的工具進行樣式設計，設置動畫和可見性規則，並通過完整的版本歷史記錄進行發佈。

![頁面構建器](/static/core/admin/img/help/page-builder/builder-overview.webp)

## 構建器介面

構建器有四個主要區域：

| 區域 | 位置 | 目的 |
|------|----------|---------|
| **工具列** | 頂部欄位 | 裝置預覽（桌面/平板/行動裝置），復原/重做，頁面設定，儲存草稿，發佈 |
| **元件庫** | 左側邊欄 | 浏覽並拖動 39 個元件，這些元件分為 9 個類別 |
| **畫布** | 中央 | 實時 WYSIWYG 編輯區域 — 看到您所做的更改 |
| **屬性面板** | 右側邊欄 | 編輯所選元件的內容、樣式、動畫和高級設定 |

## 元件庫

元件分為類別。您可以從元件庫中拖動任何元件到畫布上，以將其添加到您的頁面。

| 類別 | 元件 |
|----------|----------|
| **佈局** | 容器、分隔線、首屏區塊、彈出視窗、導覽選單、間距 |
| **基本** | 標題、文字、按鈕、圖示 |
| **內容** | 博客文章輪播、博客文章網格、FAQ 折疊面板、相關文章、推薦 |
| **媒體** | 圖片、圖片畫廊、圖片折疊面板、視頻嵌入 |
| **表單** | 聯絡表單、表單、電子報註冊 |
| **行銷** | 倒數計時器、CTA 標語、特色博客標語、忠誠度標語、促銷標語、信任徽章、優惠券碼顯示 |
| **電商** | 類別展示、禮品卡促銷、產品輪播、產品網格、產品列表、評論顯示、促銷產品、商店定位器 |
| **社交** | 社交連結 |
| **導覽** | 搜尋欄 |

### 容器和嵌套

**容器** 元件是複雜佈局的基礎。容器可以包含其他元件 — 包括其他容器 — 讓您可以建立多欄網格和嵌套結構。使用容器的佈局預設來快速設置常見的欄位排列（50/50、33/33/33、25/75 等）。

## 添加元件

1. 在左側邊欄中找到您想要的元件
2. **拖動** 它到畫布上，並在您想要的位置釋放它
3. 元件可以放在現有元件之間，或放在容器內
4. 藍色插入線顯示元件將放置的位置
5. 放置後，元件會自動選中，並打開屬性面板

您也可以通過在畫布上向上或向下拖動元件來重新排序元件。

## 編輯內容

在畫布上選擇任何元件以在右側面板中打開其屬性。**內容** 索引標籤顯示特定於該元件類型的欄位。

![屬性面板](/static/core/admin/img/help/page-builder/properties-panel.webp)

例如：
- **標題** — 文字、HTML 標籤（H1–H6）、對齊方式、錨點 ID
- **圖片** — 圖片來源（媒體庫）、替代文字、連結、尺寸
- **按鈕** — 標籤、URL、樣式變體、圖示
- **產品網格** — 資料來源、欄位數量、每頁產品數、排序順序
- **首屏區塊** — 標題、副標題、描述、背景、行動呼籲按鈕

可翻譯的內容欄位會顯示一個翻譯圖示 — 點擊它為多語言商店添加翻譯。

## 樣式設定元件

**樣式** 索引標籤為每個元件提供視覺控制。每個部分都會打開一個專用的工具編輯器。

![樣式標籤](/static/core/admin/img/help/page-builder/style-tab.webp)

| 區段 | 控制內容 | 工具 |
|---------|-----------------|---------|
| **字型** | 字型系列、大小、粗度、行高、字距、文字樣式 | 字型編輯器 |
| **顏色** | 文字顏色，支援十六進位/RGB/HSL 輸入和主題顏色 | 色彩選擇器 |
| **背景** | 實體顏色、漸變、圖片或視頻背景，支援滑鼠懸停狀態 | 背景編輯器 |
| **邊框** | 每個邊的邊框寬度、樣式、顏色和半徑 | 邊框編輯器 |
| **間距** | 邊距和內距，支援視覺方塊模型編輯器 | 間距編輯器 |
| **效果** | 方塊陰影，支援預設值和多層支援，透明度滑塊 | 陰影編輯器 |

每個工具都在其自己的幫助主題中進行了文檔化 — 搜索 "色彩選擇器"、"背景編輯器" 等以了解更多信息。

## 動畫

**動畫** 索引標籤讓您可以為元件添加動畫效果。

### 入場動畫

當元件滾動到視野中時觸發：

| 動畫 | 描述 |
|-----------|-------------|
| 淡入 | 逐漸出現 |
| 向上/向下/左/右滑入 | 從某個方向滑入 |
| 放大 | 從小到大增長 |
| 弹跳入 | 弹跳到位置 |
| 脉冲 / 震动 / 弹跳 / 闪光 / 旋转 | 吸引注意的效果 |

配置 **持續時間**（0.3s–1.5s）、**延遲**（0–1s）、**時間函數**（ease, ease-in, ease-out, linear）和 **重複**（一次或無限次）。

### 滑鼠懸停動畫

當訪客滑鼠懸停在元件上時觸發：

| 效果 | 描述 |
|--------|-------------|
| 放大 / 縮小 | 放大或縮小 |
| 抬起 | 向上浮動 |
| 旋轉（順時針 / 逆時針） | 順時針或逆時針旋轉 |
| 增亮 / 淡出 | 改變亮度或透明度 |
| 阴影放大 | 阴影擴展 |
| 抬起加陰影 | 帶有擴展陰影的上升 |
| 脉冲放大 / 偏移 / 阴影光效 | 特殊效果 |

配置 **持續時間**、**時間函數** 和 **強度**（微妙、正常、強烈）。

## 高級設定

**高級** 索引標籤提供精細的控制：

### 可見性規則

根據條件控制元件何時顯示或隱藏：

- **用戶狀態** — 已登入、已登出、新客戶、回購客戶
- **設備** — 桌面、平板、行動裝置
- **時間** — 日期範圍、時間段、星期幾
- **客戶群組** — VIP、批發等
- **購物車價值** — 最低或最高購物車總額
- **地理** — 國家、地區
- 還有 20 多種規則類型

規則可以使用 AND/OR 邏輯組合以進行複雜的目標設定。

### 自定義 CSS

| 欄位 | 目的 |
|-------|---------|
| **元件 ID** | 用於錨點連結或 CSS 定位的唯一 ID |
| **自定義 CSS 類別** | 要套用的額外類別 |
| **自定義 CSS 樣式** | 用於一次性覆蓋的內聯 CSS |
| **資料屬性** | 自定義 data-* 屬性作為鍵值對 |
| **Z-索引** | 重疊元件的堆疊順序 |

## 發佈工作流程

頁面使用草稿/發佈系統，並具有完整的版本歷史記錄：

| 狀態 | 含義 |
|--------|---------|
| **草稿** | 進行中的工作 — 不對訪客顯示 |
| **已發佈** | 在您的商店中線上顯示 |
| **已歸檔** | 從網站中移除但保留 |

### 發佈方式

1. 在構建器中進行更改 — 它們會被保存為 **草稿**
2. 點擊 **保存草稿** 以保存而不發佈
3. 點擊 **發佈** 以使當前草稿線上顯示
4. 每次發佈都會創建一個 **版本快照**
5. 您可以從版本歷史記錄中 **恢復** 任何先前版本（工具列中的時鐘圖示）

這意味著您可以自由實驗 — 您的線上頁面在您明確發佈之前保持不變。

## 頁面模板

通過使用模板來節省時間：

- **保存為模板** — 將任何頁面的設計保存為可重複使用的模板
- **從模板創建** — 從現有模板開始一個新頁面
- **模板分類** — 按用途（落地頁、關於、產品展示等）對模板進行分類

模板捕獲完整的頁面結構，包括所有元件、內容和樣式。

## 响應式設計

使用工具列中的裝置預覽按鈕查看您的頁面在不同螢幕尺寸下的顯示效果：

- **桌面** — 全寬佈局
- **平板** — 中等視窗
- **行動裝置** — 狭窄視窗

元件會根據其容器設置自動重新排列。您也可以使用可見性規則在特定裝置上顯示或隱藏特定元件。

## 小技巧

- **從容器開始** — 大多數佈局都從容器開始，以創建欄位和結構。使用佈局預設來設置常見的排列。
- **使用首屏區塊作為頁面標題** — 首屏元件提供標題、副標題、背景圖片和行動呼籲按鈕，所有這些都在一個元件中。
- **發佈前預覽** — 點擊預覽以查看訪客將看到的內容，然後在您滿意時發佈。
- **使用可見性規則進行個人化** — 向已登入與未登入的訪客顯示不同的內容，或針對特定的客戶群組。
- **保持動畫效果微妙** — 每個頁面區塊使用一到兩個入口動畫看起來專業。太多的動畫會讓人感到壓迫。
- **為您的容器命名** — 使用元件 ID 欄位為容器標籤（例如，"hero-section"、"features"），以便在複雜頁面中輕鬆找到它們。
- **在所有裝置上進行測試** — 使用裝置預覽在發佈前在桌面、平板和行動裝置上檢查您的佈局。
- **利用模板** — 將您最好的頁面設計保存為模板，以加快未來頁面的創建速度。
