---
title: Page Builder
---

Page Builder เป็นตัวแก้ไขแบบดึงและวางที่มีอินเทอร์เฟซกราฟิกสำหรับการสร้างหน้าเว็บที่มีความซับซ้อนและปรับขนาดได้โดยไม่ต้องเขียนโค้ด คุณสามารถเพิ่มองค์ประกอบจากไลบรารีที่มี 39 องค์ประกอบ กำหนดรูปแบบด้วยเครื่องมือที่ทรงพลัง ตั้งค่าการเคลื่อนไหวและกฎการมองเห็น และเผยแพร่พร้อมกับประวัติเวอร์ชันทั้งหมด

![Page Builder](/static/core/admin/img/help/page-builder/builder-overview.webp)

## ตัวสร้างหน้า (Builder Interface)

ตัวสร้างมีพื้นที่หลัก 4 แห่ง:

| พื้นที่ | ตำแหน่ง | วัตถุประสงค์ |
|--------|----------|-------------|
| **แถบเครื่องมือ (Toolbar)** | แถบด้านบน | แสดงตัวอย่างอุปกรณ์ (เดสก์ท็อป/แท็บเล็ต/มือถือ) ยกเลิก/ทำซ้ำ ตั้งค่าหน้า เก็บรักษาแบบร่าง เผยแพร่ |
| **ไลบรารีองค์ประกอบ (Element Library)** | แถบด้านซ้าย | ค้นหาและลาก 39 องค์ประกอบที่จัดหมวดหมู่ไว้ 9 กลุ่ม |
| **พื้นที่ทำงาน (Canvas)** | ตรงกลาง | พื้นที่แก้ไขแบบ WYSIWYG แบบเรียลไทม์ — ดูการเปลี่ยนแปลงขณะที่คุณกำลังทำ |
| **แผงคุณสมบัติ (Properties Panel)** | แถบด้านขวา | แก้ไขเนื้อหา รูปแบบ การเคลื่อนไหว และการตั้งค่าขั้นสูงขององค์ประกอบที่เลือก |

## ไลบรารีองค์ประกอบ

องค์ประกอบถูกจัดหมวดหมู่ไว้ คุณสามารถลากองค์ประกอบใดๆ จากไลบรารีไปยังพื้นที่ทำงานเพื่อเพิ่มลงในหน้าของคุณ

| หมวดหมู่ | องค์ประกอบ |
|----------|----------|
| **Layout** | Container, Divider, Hero Section, Modal Popup, Navigation Menu, Spacer |
| **Basic** | Heading, Text, Button, Icon |
| **Content** | Blog Post Carousel, Blog Post Grid, FAQ Accordion, Related Posts, Testimonials |
| **Media** | Image, Image Gallery, Image Accordion, Video Embed |
| **Forms** | Contact Form, Form, Newsletter Signup |
| **Marketing** | Countdown Timer, CTA Banner, Featured Blog Banner, Loyalty Banner, Promotion Banner, Trust Badges, Voucher Code Display |
| **E-commerce** | Category Showcase, Gift Card Promo, Product Carousel, Product Grid, Product List, Reviews Display, Sale Products, Store Locator |
| **Social** | Social Links |
| **Navigation** | Search Bar |

### Containers และการฝังร่วมกัน

องค์ประกอบ **Container** เป็นพื้นฐานสำหรับการจัดวางที่ซับซ้อน คอนเทนเนอร์สามารถบรรจุองค์ประกอบอื่น ๆ — รวมถึงคอนเทนเนอร์อื่น ๆ — ทำให้คุณสามารถสร้างกริดหลายคอลัมน์และโครงสร้างที่ฝังร่วมกันได้ ใช้การตั้งค่าล่วงหน้าของรูปแบบคอนเทนเนอร์เพื่อตั้งค่าการจัดเรียงคอลัมน์ทั่วไปได้อย่างรวดเร็ว (50/50, 33/33/33, 25/75, ฯลฯ)

## การเพิ่มองค์ประกอบ

1. ค้นหาองค์ประกอบที่คุณต้องการในแถบด้านซ้าย
2. **ลาก** มันไปยังพื้นที่ทำงานและวางไว้ที่ตำแหน่งที่คุณต้องการ
3. คุณสามารถวางองค์ประกอบระหว่างองค์ประกอบที่มีอยู่ หรือภายในคอนเทนเนอร์ได้
4. บรรทัดการแทรกสีฟ้าแสดงตำแหน่งที่องค์ประกอบจะถูกวางไว้
5. หลังจากวางแล้ว องค์ประกอบจะถูกเลือกโดยอัตโนมัติ และแผงคุณสมบัติจะเปิดขึ้น

คุณยังสามารถจัดลำดับองค์ประกอบใหม่ได้โดยการลากมันขึ้นหรือลงในพื้นที่ทำงาน

## การแก้ไขเนื้อหา

เลือกองค์ประกอบใด ๆ บนพื้นที่ทำงานเพื่อเปิดคุณสมบัติของมันในแผงด้านขวา แท็บ **Content** แสดงฟิลด์เฉพาะสำหรับประเภทองค์ประกอบนั้น

![Properties Panel](/static/core/admin/img/help/page-builder/properties-panel.webp)

ตัวอย่าง:
- **Heading** — เนื้อหา, แท็ก HTML (H1–H6), การจัดวาง, ID สำหรับการเชื่อมโยง
- **Image** — แหล่งที่มาของภาพ (ไลบรารีสื่อ), ข้อความอธิบาย, ลิงก์, การปรับขนาด
- **Button** — ป้ายกำกับ, URL, รูปแบบการเปลี่ยนผ่าน, ไอคอน
- **Product Grid** — แหล่งข้อมูล, จำนวนคอลัมน์, จำนวนสินค้าต่อหน้า, ลำดับการจัดเรียง
- **Hero Section** — หัวข้อ, หัวข้อรอง, คำอธิบาย, ฉากหลัง, ปุ่มเรียกใช้

ฟิลด์เนื้อหาที่สามารถแปลได้แสดงไอคอนการแปล — คลิกเพื่อเพิ่มการแปลสำหรับร้านค้าหลายภาษา

## การกำหนดรูปแบบองค์ประกอบ

แท็บ **Style** ให้การควบคุมแบบมีภาพสำหรับองค์ประกอบทุกตัว แต่ละส่วนเปิดตัวแก้ไขเครื่องมือเฉพาะ

![Style Tab](/static/core/admin/img/help/page-builder/style-tab.webp)

| ส่วน | สิ่งที่ควบคุม | เครื่องมือ |
|------|----------------|--------|
| **Typography** | ตระกูลฟอน, ขนาด, ความหนา, ความสูงของบรรทัด, ระยะห่างระหว่างตัวอักษร, รูปแบบข้อความ | ตัวแก้ไขฟอน |
| **Colors** | สีข้อความพร้อมการป้อน hex/RGB/HSL และโทเค็นธีม | ตัวเลือกสี |
| **Background** | สีพื้นหลังแบบเต็ม, ความชัน, ภาพ หรือวิดีโอพร้อมสถานะการชี้นำ | ตัวแก้ไขพื้นหลัง |
| **Border** | ความกว้าง, รูปแบบ, สี และรัศมีของเส้นขอบตามด้าน | ตัวแก้ไขเส้นขอบ |
| **Spacing** | ระยะห่างและพื้นที่ว่างพร้อมตัวแก้ไขโมเดลกล่องแบบมีภาพ | ตัวแก้ไขระยะห่าง |
| **Effects** | เงาของกล่องพร้อมการตั้งค่าล่วงหน้าและสนับสนุนหลายชั้น, ตัวเลื่อนความโปร่งใส | ตัวแก้ไขเงา |

แต่ละเครื่องมือมีเอกสารประกอบในหัวข้อช่วยเหลือของตัวเอง — ค้นหา "color picker", "background editor", ฯลฯ เพื่อเรียนรู้เพิ่มเติม

## การเคลื่อนไหว

แท็บ **Animations** ช่วยให้คุณเพิ่มการเคลื่อนไหวให้กับองค์ประกอบ

### การเคลื่อนไหวในการเข้าสู่หน้า

กระตุ้นเมื่องค์ประกอบเลื่อนเข้ามายังมุมมอง:

| การเคลื่อนไหว | คำอธิบาย |
|---------------|-----------|
| Fade In | ปรากฏขึ้นทีละน้อย |
| Slide In (Up/Down/Left/Right) | ลื่นเข้ามาจากทิศทางหนึ่ง |
| Zoom In | ขยายจากขนาดเล็กไปยังขนาดเต็ม |
| Bounce In | กระเด้งเข้าที่ |
| Pulse / Shake / Bounce / Flash / Spin | ผลลัพธ์ที่ดึงความสนใจ |

ตั้งค่า **duration** (0.3s–1.5s), **delay** (0–1s), **timing function** (ease, ease-in, ease-out, linear), และ **repeat** (ครั้งเดียวหรือไม่สิ้นสุด)

### การเคลื่อนไหวเมื่อชี้เมาส์

กระตุ้นเมื่อผู้เยี่ยมชมชี้เมาส์ไปที่องค์ประกอบ:

| ผลลัพธ์ | คำอธิบาย |
|--------|-----------|
| Scale Up / Scale Down | ขยายหรือหดตัว |
| Lift | ลอยขึ้น |
| Rotate (CW / CCW) | หมุนตามเข็มหรือทวนเข็ม |
| Brighten / Fade | เปลี่ยนความสว่างหรือความโปร่งใส |
| Shadow Grow | เงาขยาย |
| Lift with Shadow | ลอยขึ้นพร้อมกับเงาที่ขยาย |
| Pulse Scale / Skew / Border Glow | ผลลัพธ์พิเศษ |

ตั้งค่า **duration**, **timing**, และ **intensity** (ละเอียด, ปกติ, แข็งแรง)

## การตั้งค่าขั้นสูง

แท็บ **Advanced** ให้การควบคุมที่ละเอียดอ่อน:

### กฎการมองเห็น

ควบคุมว่าองค์ประกอบจะแสดงหรือซ่อนเมื่อใดโดยใช้เงื่อนไข:

- **สถานะผู้ใช้** — ล็อกอิน, ล็อกเอาท์, ลูกค้าใหม่, ลูกค้าเก่า
- **อุปกรณ์** — เดสก์ท็อป, แท็บเล็ต, มือถือ
- **เวลา** — ช่วงวันที่, เวลาในวัน, วันในสัปดาห์
- **กลุ่มลูกค้า** — VIP, ขายส่ง, ฯลฯ
- **ค่าตะกร้าสินค้า** — ค่ารวมตะกร้าสินค้าขั้นต่ำหรือสูงสุด
- **ภูมิศาสตร์** — ประเทศ, ภูมิภาค
- และอีก 20+ ประเภทกฎ

กฎสามารถรวมกันด้วยตรรกะ AND/OR สำหรับการกำหนดเป้าหมายที่ซับซ้อน

### CSS แบบกำหนดเอง

| ฟิลด์ | วัตถุประสงค์ |
|-------|-------------|
| **Element ID** | ID ที่ไม่ซ้ำกันสำหรับการเชื่อมโยงหรือการกำหนดเป้าหมาย CSS |
| **Custom CSS Classes** | ชั้นเรียนเพิ่มเติมที่จะใช้ |
| **Custom CSS Styles** | CSS แบบอินไลน์สำหรับการปรับแต่งเฉพาะ |
| **Data Attributes** | คุณสมบัติข้อมูลที่กำหนดเองเป็นคู่คีย์-ค่า |
| **Z-Index** | ลำดับการทับซ้อนสำหรับองค์ประกอบที่ทับกัน |

## กระบวนการทำงานการเผยแพร่

หน้าใช้ระบบแบบร่าง/เผยแพร่พร้อมประวัติเวอร์ชันทั้งหมด:

| สถานะ | ความหมาย |
|--------|----------|
| **Draft** | งานที่ยังไม่เสร็จสิ้น — ไม่สามารถมองเห็นได้โดยผู้เยี่ยมชม |
| **Published** | อยู่ในร้านค้าของคุณ |
| **Archived** | ถูกลบออกจากเว็บไซต์แต่ยังคงเก็บไว้ |

### วิธีการทำงาน

1. ทำการเปลี่ยนแปลงในตัวสร้าง — พวกเขาจะถูกบันทึกเป็น **แบบร่าง**
2. คลิก **Save Draft** เพื่อบันทึกโดยไม่เผยแพร่
3. คลิก **Publish** เพื่อทำให้แบบร่างปัจจุบันเป็นแบบเรียลไทม์
4. แต่ละการเผยแพร่สร้าง **快照เวอร์ชัน**
5. คุณสามารถ **restore** ได้เวอร์ชันก่อนหน้าใด ๆ จากประวัติเวอร์ชัน (ไอคอนนาฬิกาในแถบเครื่องมือ)

นี่หมายความว่าคุณสามารถทดลองได้อย่างอิสระ — หน้าเว็บของคุณจะไม่เปลี่ยนไปจนกว่าคุณจะเผยแพร่โดยชัดเจน

## แม่แบบหน้า

ประหยัดเวลาโดยการทำงานกับแม่แบบ:

- **Save as Template** — บันทึกการออกแบบหน้าใด ๆ เป็นแม่แบบที่สามารถใช้ซ้ำได้
- **Create from Template** — เริ่มหน้าใหม่จากแม่แบบที่มีอยู่
- **Template Categories** — จัดระเบียดแม่แบบตามวัตถุประสงค์ (หน้าแรก, เกี่ยวกับ, การแสดงสินค้า, ฯลฯ)

แม่แบบจับโครงสร้างหน้าทั้งหมดรวมถึงองค์ประกอบ เนื้อหา และการกำหนดรูปแบบ

## การออกแบบที่ปรับขนาดได้

ใช้ปุ่มแสดงตัวอย่างอุปกรณ์ในแถบเครื่องมือเพื่อดูว่าหน้าของคุณจะดูอย่างไรบนขนาดหน้าจอต่าง ๆ:

- **Desktop** — รูปแบบที่เต็มหน้าจอ
- **Tablet** — หน้าต่างขนาดกลาง
- **Mobile** — หน้าต่างขนาดแคบ

องค์ประกอบจะปรับตำแหน่งโดยอัตโนมัติตามการตั้งค่าคอนเทนเนอร์ของพวกเขา คุณยังสามารถใช้กฎการมองเห็นเพื่อแสดงหรือซ่อนองค์ประกอบเฉพาะบนอุปกรณ์บางประเภท

## เคล็ดลับ

- **เริ่มต้นด้วย Container** — โครงสร้างส่วนใหญ่เริ่มต้นด้วย Container เพื่อสร้างคอลัมน์และโครงสร้าง ใช้การตั้งค่าล่วงหน้าของรูปแบบสำหรับการจัดเรียงทั่วไป
- **ใช้ Hero sections สำหรับหัวข้อหน้า** — องค์ประกอบ Hero ให้หัวข้อ หัวข้อรอง ภาพพื้นหลัง และปุ่มเรียกใช้ในองค์ประกอบเดียว
- **ดูตัวอย่างก่อนเผยแพร่** — คลิก Preview เพื่อดูว่าผู้เยี่ยมชมจะเห็นอะไร แล้วเผยแพร่เมื่อคุณพอใจ
- **ใช้กฎการมองเห็นเพื่อปรับแต่ง** — แสดงเนื้อหาที่แตกต่างกันสำหรับผู้ใช้ที่ล็อกอินและล็อกเอาท์ หรือกำหนดเป้าหมายกลุ่มลูกค้าเฉพาะ
- **รักษาการเคลื่อนไหวให้เรียบง่าย** — การเคลื่อนไหวเข้าสู่หน้า 1 หรือ 2 ต่อส่วนของหน้าดูมืออาชีพ การเคลื่อนไหวมากเกินไปอาจทำให้รู้สึกไม่สบายตา
- **ตั้งชื่อ Container ของคุณ** — ใช้ฟิลด์ Element ID เพื่อตั้งชื่อ Container (เช่น "hero-section", "features") เพื่อให้ค้นหาได้ง่ายในหน้าที่ซับซ้อน
- **ทดสอบบนอุปกรณ์ทั้งหมด** — ใช้ตัวแสดงตัวอย่างอุปกรณ์เพื่อตรวจสอบการจัดวางของคุณบนเดสก์ท็อป แท็บเล็ต และมือถือก่อนเผยแพร่
- **ใช้แม่แบบ** — บันทึกการออกแบบหน้าที่ดีที่สุดของคุณเป็นแม่แบบเพื่อเร่งการสร้างหน้าในอนาคต
