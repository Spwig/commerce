title: 設計與主題
---

設計與主題系統讓您可以控制整個商店外觀的外觀和感覺 - 從顏色和字體到標頭、頁腳和頁面佈局。導航到 **設定 > 設計與主題** 以打開設計儀錶板。

![設計儀錶板](/static/core/admin/img/help/design-themes/theme-dashboard.webp)

## 設計儀錶板

儀錶板給您商店設計狀態的概覽：

- **活動主題** — 顯示目前套用的主題，帶預覽和快速訪問按鈕
- **設計統計數據** — 已安裝主題的數量、自定義標頭、自定義頁腳和菜單
- **部分卡片** — 跳轉到主題、標頭構建器、頁腳構建器、菜單或公告

## 主題

### 瀏覽主題

點擊 **主題** 部分卡片查看所有已安裝的主題。每個主題卡片顯示：
- 主題名稱和預覽圖片
- 作者和版本
- 活動/非活動狀態

### 啟用主題

1. 點擊您要使用的主題的 **啟用**
2. 主題立即套用到您的商店外觀
3. 一次只能啟用一個主題

### 主題自定義

每個主題都支援一組 **設計令牌** — 可配置的值，可在不編輯程式碼的情況下控制視覺外觀。

點擊您活動主題的 **自定義** 以訪問令牌編輯器。可用的令牌類別包括：

| 類別 | 它控制什麼 |
|----------|-----------------|
| **顏色** | 主要、次要、強調顏色、背景、文字顏色 |
| **字型** | 字型系列、大小、重量、行高 |
| **間距** | 邊距、填充、元素之間的間隙 |
| **邊框** | 邊框寬度、半徑、顏色 |
| **陰影** | 卡片、按鈕、模態框的陰影 |
| **按鈕** | 按鈕樣式、大小、滑鼠懸停效果 |
| **佈局** | 容器寬度、網格間距、斷點 |

在保存之前即時預覽更改。

## 標頭構建器

標頭構建器讓您可以使用拖放介面設計商店的標頭。

### 建立標頭

1. 導航到 **設計 > 標頭構建器**
2. 點擊 **建立標頭** 或編輯現有的標頭
3. 構建器有三行：**頂部欄位**、**主要標頭** 和 **底部欄位**
4. 從工具箱中拖動小工具到任何一行

### 可用的標頭小工具

- **商標** — 您的商店商標，可配置大小和連結
- **導覽菜單** — 從您定義的菜單中導出的下拉菜單
- **搜尋欄** — 產品搜尋，支援即時結果
- **購物車圖示** — 小型購物車，帶有商品數量徽章
- **帳號圖示** — 登入/帳號下拉選單
- **語言選擇器** — 多語言商店的語言切換器
- **貨幣選擇器** — 多貨幣商店的貨幣切換器
- **自訂 HTML** — 添加任何自訂內容
- **社交圖示** — 連結到您的社交媒體專頁
- **公告欄** — 推廣訊息和優惠

### 標頭設定

每個標頭模板都有全域設定：
- **固定標頭** — 滾動時標頭保持可見
- **透明模式** — 英雄圖片上的疊加層
- **行動裝置斷點** — 切換到行動版佈局的時間點

## 頁腳構建器

頁腳構建器的工作方式與標頭構建器類似。

### 建立頁腳

1. 導航到 **設計 > 頁腳構建器**
2. 點擊 **建立頁腳** 或編輯現有的頁腳
3. 構建器支援多個欄位和行
4. 將小工具拖放到位置

### 可用的頁腳小工具

- **導覽菜單** — 頁腳導覽連結
- **電子報訂閱** — 電子郵件訂閱表單
- **社交圖示** — 社交媒體連結
- **自訂 HTML** — 自訂內容、徽章、認證
- **付款圖示** — 顯示接受的付款方式
- **版權** — 動態版權文字，包含年份
- **商標** — 頁腳商標變體

## 導覽菜單

菜單定義了您標頭和頁腳中的導覽連結。

### 建立菜單

1. 導航到 **設計 > 菜單**
2. 點擊 **新增菜單**
3. 為菜單命名（例如，"主要導覽"）
4. 添加菜單項目：
   - **頁面連結** — 連結到頁面構建器頁面
   - **分類連結** — 連結到產品分類
   - **自訂 URL** — 任何外部或內部 URL
   - **下拉選單** — 嵌套的子菜單項目
5. 拖動項目以重新排序
6. 儲存並指派菜單到標頭或頁腳小工具

## 公告

建立促銷橫幅，出現在商店外觀的頂部。

### 建立公告

1. 導航到 **設計 > 公告**（或使用儀錶板卡片）
2. 點擊 **新增公告**
3. 設定：
   - **訊息** — 公告文字（支援翻譯）
   - **連結** — 點擊時的選項 URL
   - **樣式** — 背景顏色、文字顏色、圖示
   - **排程** — 開始和結束日期
   - **可關閉** — 顧客是否可以關閉它
4. 儲存並啟用

多個公告可以同時啟用 - 它們會自動輪替。

## 小技巧

- 從活動主題的自定義開始，匹配您的品牌顏色，再建立標頭和頁腳。
- 在標頭和頁腳構建器中使用 **預覽** 功能，在發布前查看更改。
- 如果您需要非常不同的佈局，請為桌面和行動裝置建立不同的標頭。
- 保持導覽簡單 - 5-7 個頂層菜單項目對可用性最理想。
- 使用公告來宣傳即時促銷，而不是永久訊息。
- 主題令牌編輯器支援即時預覽 - 自由實驗並在滿意時儲存。
