---
title: แม่แบบหน้าหมวดหมู่
---

แม่แบบหน้าหมวดหมู่ควบคุมวิธีที่สินค้าของคุณจะถูกแสดงเมื่อลูกค้าเรียกดูหมวดหมู่ — ไม่ว่าจะเป็นรูปแบบตารางแบบคลาสสิก รายการแนวนอน แคโรเซลที่เลื่อนได้ หรือรูปแบบที่โดดเด่นอย่างมีเอกลักษณ์ คุณสามารถตั้งค่าแม่แบบเริ่มต้นเดียวที่ใช้ทั่วทั้งร้านค้า และจากนั้นสามารถเปลี่ยนแปลงได้ในหมวดหมู่ใดๆ ที่ต้องการการจัดการที่แตกต่างกัน

## ที่ใดที่สามารถค้นหาการตั้งค่าได้

มีสองสถานที่ในการตั้งค่าแม่แบบหมวดหมู่:

- **ค่าเริ่มต้นทั่วทั้งเว็บไซต์** — ไปที่ **Design > Page Templates** และเลื่อนไปยังส่วน **Category** แม่แบบที่คุณเลือกที่นี่จะใช้กับหน้าหมวดหมู่ทุกหน้าเว้นแต่จะมีการเปลี่ยนแปลง
- **การเปลี่ยนแปลงเฉพาะหมวดหมู่** — ไปที่ **Catalog > Categories** เปิดหมวดหมู่ใดๆ และคลิกแท็บ **Display** การตั้งค่าแม่แบบที่นี่จะเปลี่ยนแปลงค่าเริ่มต้นทั่วทั้งเว็บไซต์สำหรับหมวดหมู่นั้นเท่านั้น หมวดหมู่อื่นๆ ยังคงใช้ค่าเริ่มต้นตามปกติ

## แม่แบบหกแบบ

### Grid

แม่แบบ Grid จัดเรียงสินค้าในตารางแบบตอบสนองได้ — รูปแบบที่คุ้นเคยที่สุดสำหรับการช้อปปิ้งออนไลน์ สินค้าจะแสดงเป็นการ์ดที่มีรูปภาพ ชื่อ ราคา และปุ่มเพิ่มสินค้าลงตะกร้า

**เหมาะที่สุดสำหรับ:** ร้านค้าทั่วไป เสื้อผ้า เครื่องใช้ในบ้าน อุปกรณ์อิเล็กทรอนิกส์ — ร้านค้าใดๆ ที่ต้องการการนำเสนอสินค้าที่สม่ำเสมอ

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| **จำนวนสินค้าต่อแถว** | จำนวนการ์ดที่ปรากฏต่อแถวบนเดสก์ท็อป (2, 3, 4 หรือ 5) |
| **แสดงแบนเนอร์หมวดหมู่** | แสดงภาพแบนเนอร์ของหมวดหมู่ที่ด้านบนของหน้า |
| **แสดงหมวดหมู่ย่อย** | แสดงชิปหมวดหมู่ย่อยที่สามารถคลิกได้ด้านล่างของแบนเนอร์ |
| **แสดงคำอธิบายหมวดหมู่** | แสดงข้อความคำอธิบายของหมวดหมู่ |
| **รูปภาพการ์ดสำรอง** | แสดงรูปภาพแทนที่เมื่อสินค้าไม่มีรูปภาพ |

### List

แม่แบบ List แสดงสินค้าเป็นแถวแนวนอน — รูปภาพอยู่ด้านหนึ่ง ส่วนรายละเอียดสินค้า (ชื่อ คำอธิบาย ราคา) อยู่อีกด้านหนึ่ง ทำให้แต่ละสินค้ามีพื้นที่หายใจมากขึ้น และทำให้ลูกค้าสามารถเปรียบเทียบได้ง่ายขึ้น

**เหมาะที่สุดสำหรับ:** ร้านค้าที่มีสินค้าซับซ้อนที่ได้รับประโยชน์จากคำอธิบายสั้นๆ ล่วงหน้า เช่น เครื่องมือ อุปกรณ์ไฟฟ้า หรือแคตตาล็อก B2B

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| **แสดงรูปภาพสินค้า** | รวมรูปภาพสินค้าในแต่ละแถว |
| **ตำแหน่งรูปภาพ** | รูปภาพปรากฏอยู่ด้านซ้ายหรือด้านขวา |
| **แสดงคำอธิบายสั้นๆ** | แสดงข้อความย่อของสินค้าด้านล่างชื่อสินค้า |
| **แสดงแบนเนอร์หมวดหมู่** | แสดงภาพแบนเนอร์ของหมวดหมู่ที่ด้านบน |
| **แสดงหมวดหมู่ย่อย** | แสดงชิปหมวดหมู่ย่อยที่สามารถคลิกได้ด้านล่างของแบนเนอร์ |
| **แสดงคำอธิบายหมวดหมู่** | แสดงข้อความคำอธิบายของหมวดหมู่ |

### Carousel

แม่แบบ Carousel จัดกลุ่มสินค้าเป็นแถวแนวนอนที่สามารถเลื่อนได้ — คล้ายกับส่วน "Continue browsing" บนแพลตฟอร์มสตรีมมิง ลูกค้าสามารถเลื่อนหรือคลิกปุ่มลูกศรเพื่อเรียกดูสินค้าโดยไม่ต้องออกจากหน้า

**เหมาะที่สุดสำหรับ:** แคตตาล็อกขนาดใหญ่ที่มีสินค้าจำนวนมากในหมวดหมู่เดียว ร้านขายของขวัญ ร้านหนังสือ หรือร้านค้าใดๆ ที่การเรียกดูและการค้นพบเป็นสิ่งสำคัญ

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| **จำนวนสไลด์ต่อการดู** | จำนวนสินค้าที่เห็นได้ในครั้งเดียวบนเดสก์ท็อป (3, 4 หรือ 5) |
| **ขนาดแถว** | จำนวนสินค้าต่อแถวแคโรเซลก่อนเริ่มแถวใหม่ด้านล่าง (4, 6, 8 หรือ 12) |
| **แสดงปุ่มนำทาง** | แสดงปุ่มลูกศรก่อนหน้า/ถัดไปสำหรับแต่ละแถว |
| **แสดงจุด** | แสดงแถบความคืบหน้าและนับหน้า (เช่น "1 / 3") ด้านล่างของแต่ละแถว |
| **เล่นอัตโนมัติ** | เลื่อนสไลด์อัตโนมัติ |
| **ความเร็วอัตโนมัติ** | เวลาในมิลลิวินาทีระหว่างการเลื่อนอัตโนมัติแต่ละครั้ง |
| **วนซ้ำ** | กลับไปยังสไลด์แรกหลังจากถึงสไลด์สุดท้าย |
| **แสดงแบนเนอร์หมวดหมู่** | แสดงภาพแบนเนอร์ของหมวดหมู่ที่ด้านบน |
| **แสดงหมวดหมู่ย่อย** | แสดงชิปหมวดหมู่ย่อยที่สามารถคลิกได้ด้านล่างของแบนเนอร์ |

### Masonry

แม่แบบ Masonry สร้างตารางแบบยุบยัดที่คล้ายกับ Pinterest โดยที่แต่ละการ์ดปรับให้เหมาะสมกับความสูงตามธรรมชาติของรูปภาพ ผลลัพธ์คือความรู้สึกที่หลากหลายและเป็นแบบสื่อสารมากกว่าตารางที่สม่ำเสมอ

**เหมาะที่สุดสำหรับ:** ภาพถ่าย พิมพ์งานศิลปะ สินค้าทำมือ ออกแบบภายใน หรือหมวดหมู่ใดๆ ที่ลักษณะภาพของสินค้ามีความหลากหลาย และคุณต้องการนำเสนอในรูปแบบที่มีพลังมากขึ้น

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| **จำนวนคอลัมน์** | จำนวนคอลัมน์ทั่วหน้า (2, 3 หรือ 4) |
| **ขนาดช่องว่าง** | พื้นที่ระหว่างการ์ด — ขนาดเล็ก ปานกลาง หรือใหญ่ |
| **แสดงแบนเนอร์หมวดหมู่** | แสดงภาพแบนเนอร์ของหมวดหมู่ที่ด้านบน |
| **แสดงหมวดหมู่ย่อย** | แสดงชิปหมวดหมู่ย่อยที่สามารถคลิกได้ด้านล่างของแบนเนอร์ |
| **แสดงคำอธิบายหมวดหมู่** | แสดงข้อความคำอธิบายของหมวดหมู่ |

### Featured

แม่แบบ Featured โปรโมตสินค้าจำนวนน้อยๆ เป็นการ์ดขนาดใหญ่ที่ด้านบนของหน้า โดยสินค้าที่เหลือจะแสดงในตารางรองรับด้านล่าง มันดึงความสนใจไปยังสินค้าที่คุณต้องการให้เป็นจุดเด่นทันที

**เหมาะที่สุดสำหรับ:** ส่วนสินค้าใหม่ หน้าสินค้าขายดี คอลเลกชันที่คัดสรร หรือหมวดหมู่ตามฤดูกาลที่คุณต้องการเน้นสินค้าเฉพาะบางอย่าง

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| **จำนวนสินค้าที่โดดเด่น** | จำนวนสินค้าที่ได้รับการจัดการแบบฮีโร่ที่ด้านบน (1, 2 หรือ 3) |
| **รูปแบบฮีโร่** | แสดงการ์ดฮีโร่แบบเต็มหน้าหรือเป็นรูปแบบแบ่งเป็นสองข้างกัน |
| **แสดงแบนเนอร์หมวดหมู่** | แสดงภาพแบนเนอร์ของหมวดหมู่ที่ด้านบน |
| **แสดงหมวดหมู่ย่อย** | แสดงชิปหมวดหมู่ย่อยที่สามารถคลิกได้ด้านล่างของแบนเนอร์ |
| **แสดงคำอธิบายหมวดหมู่** | แสดงข้อความคำอธิบายของหมวดหมู่ |

### Accordion

แม่แบบ Accordion ถูกออกแบบมาโดยเฉพาะสำหรับหน้า **หมวดหมู่หลัก** — หน้าที่แสดงหมวดหมู่ระดับสูงทั้งหมดของคุณ มันแสดงหมวดหมู่แต่ละตัวเป็นแผงโต้ตอบ; การชี้หรือแตะแผงจะทำให้มันขยายออกในแนวนอนขณะที่แผงอื่นๆ หดกลับ บนหน้าหมวดหมู่เฉพาะ แม่แบบ Accordion จะกลับไปใช้ตารางมาตรฐาน

**เหมาะที่สุดสำหรับ:** ร้านค้าที่มีภาพหมวดหมู่ที่มีความน่าสนใจ โดยคุณต้องการให้หน้าหมวดหมู่รู้สึกเหมือนประสบการณ์ที่มีความลึก — แบรนด์แฟชั่น ร้านค้าไลฟ์สไตล์ และร้านค้าที่มีเอกลักษณ์ทางสื่อสารที่แข็งแกร่ง

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| **จำนวนแผงต่อแถว** | จำนวนหมวดหมู่สูงสุดต่อแถวของ Accordion ก่อนที่จะเปลี่ยนไปยังแถวใหม่ด้านล่าง (4–8, ค่าเริ่มต้น 6) |
| **ความสูง** | ความสูงของแผง Accordion — ขนาดเล็ก ปานกลาง ใหญ่ หรือใหญ่พิเศษ |
| **อัตราการขยาย** | ความกว้างที่แผงที่เปิดอยู่ (ที่ชี้) ขยายออกเมื่อเทียบกับแผงที่ไม่ได้เปิด (2–5×) |
| **ความเร็วการเปลี่ยนผ่าน** | ความเร็วของแอนิเมชันในมิลลิวินาที |
| **รูปแบบมือถือ** | วิธีที่หมวดหมู่แสดงบนมือถือ — วางแนวตั้ง, เป็นแคโรเซล หรือเป็นตาราง |
| **แสดงจำนวนสินค้า** | แสดงป้ายจำนวนสินค้าบนแผงหมวดหมู่แต่ละตัว |

## ตัวเลือกทั่วไป (ทุกแม่แบบ)

ทุกแม่แบบมีชุดตัวเลือกที่ควบคุมส่วนประกอบของหน้า:

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| **แสดง breadcrumb** | แสดงการนำทาง "Home > Categories > Name" ด้านบนของหมวดหมู่ |
| **แสดงแถบการจัดเรียง** | แสดงเมนู "Sort by" เพื่อให้ลูกค้าสามารถจัดเรียงผลลัพธ์ใหม่ |
| **แสดงจำนวนสินค้า** | แสดงจำนวนสินค้าทั้งหมด (เช่น "29 products") |
| **แสดงส่วนช่วยเหลือ** | แสดงส่วน "Can't find what you're looking for?" ด้านล่าง |
| **การจัดเรียงเริ่มต้น** | ลำดับการจัดเรียงที่ใช้เมื่อลูกค้าเข้าถึงหน้าแรก |
| **จำนวนสินค้าต่อหน้า** | จำนวนสินค้าที่โหลดต่อหน้า |
| **รูปแบบการแบ่งหน้า** | วิธีที่โหลดสินค้าเพิ่มเติม: หน้าที่แบ่งเป็นหน้า, ปุ่ม "Load more" หรือการเลื่อนไม่สิ้นสุด |

## การเปรียบเทียบแบบรวดเร็ว

ใช้ตารางนี้เพื่อเลือกแม่แบบที่เหมาะสมได้ทันที:

| แม่แบบ | จำนวนสินค้าต่อแถว | ลักษณะภาพ | ประเภทสินค้าที่เหมาะที่สุด |
|----------|-----------------|------------------|--------------------|
| **Grid** | 2–5 ที่ปรับแต่งได้ | สม่ำเสมอ สะอาด | ทุกอย่าง — ค่าเริ่มต้นที่ปลอดภัย |
| **List** | 1 (แถวเต็มหน้า) | โปร่งใส ให้ข้อมูล | สินค้าซับซ้อนหรือทางเทคนิค |
| **Carousel** | 3–5 ที่เห็นได้ในครั้งเดียว | สามารถเรียกดูได้ ไดนามิก | แคตตาล็อกขนาดใหญ่ การค้นพบ |
| **Masonry** | 2–4 (แบบยุบยัด) | แบบสื่อสาร หลากหลาย | สินค้าที่มีภาพหลากหลาย งานฝีมือ |
| **Featured** | ฮีโร่ + ตารางด้านล่าง | โปรโมชัน | คัดสรร ของใหม่ |
| **Accordion** | N/A (หมวดหมู่) | ประสบการณ์ที่ลึกซึ้ง ดราม่า | หน้าหมวดหมู่หลัก |

## การตั้งค่าแม่แบบเริ่มต้นทั่วทั้งเว็บไซต์

1. ไปที่ **Design > Page Templates**.
2. เลื่อนไปยังส่วน **Category**.
3. เลือกแม่แบบที่คุณต้องการจาก **Template** dropdown.
4. ตั้งค่าตัวเลือกแม่แบบที่ปรากฏด้านล่าง dropdown.
5. ตั้งค่าตัวเลือกทั่วไป (แถบการจัดเรียง breadcrumb, การแบ่งหน้า, ฯลฯ).
6. คลิก **Save Configuration**.

แม่แบบที่เลือกและตั้งค่าที่เกี่ยวข้องจะถูกนำไปใช้กับหน้าหมวดหมู่ทุกหน้าในร้านค้าของคุณ

## การเปลี่ยนแปลงแม่แบบสำหรับหมวดหมู่เฉพาะ

คุณสามารถกำหนดแม่แบบที่แตกต่างกันสำหรับหมวดหมู่ใดๆ โดยไม่ต้องเปลี่ยนแม่แบบเริ่มต้นทั่วทั้งร้านค้า

1. ไปที่ **Catalog > Categories**.
2. คลิกหมวดหมู่ที่คุณต้องการปรับแต่ง.
3. คลิกแท็บ **Display** ด้านบนของฟอร์มแก้ไข.
4. ภายใต้ **Template** เลือก **Custom** เพื่อเปิดใช้งานการตั้งค่าเฉพาะหมวดหมู่.
5. เลือกแม่แบบที่คุณต้องการสำหรับหมวดหมู่นี้.
6. ตั้งค่าตัวเลือกแม่แบบตามที่จำเป็น.
7. คลิก **Save**.

เพียงแค่หมวดหมู่นี้เท่านั้นที่จะใช้แม่แบบที่กำหนดเอง หมวดหมู่อื่นๆ ยังคงใช้ค่าเริ่มต้นทั่วทั้งเว็บไซต์ หากคุณต้องการเปลี่ยนกลับหมวดหมู่ไปใช้ค่าเริ่มต้น ให้กลับไปที่แท็บ Display ของมันและตั้งค่าแม่แบบกลับไปเป็น **Use site default**

## ตัวอย่าง: การผสมแม่แบบระหว่างหมวดหมู่

นี่คือตัวอย่างเชิงปฏิบัติว่าแม่แบบต่างๆ สามารถใช้กับส่วนต่างๆ ของร้านค้าเดียวกันได้อย่างไร:

| หมวดหมู่ | แม่แบบ | เหตุผล |
|----------|----------|--------|
| ทั้งหมด (หน้าหลัก) | Accordion | จุดเริ่มต้นที่มีภาพที่น่าประทับใจ |
| ชุดเสื้อผ้า | Grid (4 ต่อแถว) | การเรียกดูที่รวดเร็วและคุ้นเคย |
| ลดราคา | Featured | แสดงสินค้าลดราคาที่โดดเด่นทันที |
| หนังสือ | Carousel | แคตตาล็อกขนาดใหญ่ ง่ายต่อการเรียกดู |
| ของขวัญทำมือ | Masonry | ขนาดรูปภาพที่หลากหลาย รู้สึกเป็นงานฝีมือ |
| เครื่องมืออุตสาหกรรม | List | ลูกค้าต้องการดูคำอธิบาย |

## เคล็ดลับ

- เริ่มต้นด้วยแม่แบบ **Grid** เป็นค่าเริ่มต้นทั่วทั้งเว็บไซต์ — มันทำงานได้ดีกับสินค้าเกือบทุกประเภท และเป็นรูปแบบที่คุ้นเคยที่สุดสำหรับผู้ซื้อสินค้า ปรับแต่งหมวดหมู่เฉพาะจากนั้น
- ใช้แม่แบบ **Featured** อย่างระมัดระวัง มันทำงานได้ดีที่สุดเมื่อคุณสามารถคัดสรรสินค้า 1–3 ชิ้นที่โดดเด่นสำหรับหมวดหมู่นั้น หากสินค้าทั้งหมดดูเหมือนมีความสำคัญเท่ากัน การจัดการแบบฮีโร่จะสูญเสียผลลัพธ์
- แม่แบบ **Accordion** สร้างผลกระทบภาพที่สมบูรณ์แบบเฉพาะหน้าหมวดหมู่หลักเท่านั้น หากคุณกำหนดให้ใช้กับหมวดหมู่สินค้า ลูกค้าจะเห็นตารางธรรมดา — ดังนั้นจึงควรสงวนไว้สำหรับรายการหมวดหมู่ระดับสูงของคุณ
- ตั้งค่า **จำนวนสินค้าต่อหน้า** อย่างรอบคอบ 24–36 สินค้าต่อหน้าเหมาะกับแม่แบบ Grid และ Masonry สำหรับ Carousel ตัวเลือกขนาดแถวและจำนวนสไลด์ต่อการดูควบคุมประสบการณ์นี้โดยตรงมากกว่า
- เปิดใช้งาน **แสดงหมวดหมู่ย่อย** บนหมวดหมู่หลักเพื่อให้ลูกค้าสามารถกระโดดไปยังการเลือกที่แคบลงโดยไม่ต้องใช้เครื่องมือจัดเรียงหรือกรอง
- ทดสอบแม่แบบที่คุณเลือกบนมือถือ โดยเฉพาะแม่แบบ Masonry และ Carousel ที่มีลักษณะต่างกันมากบนหน้าจอขนาดเล็ก — ใช้ตัวเลือก **รูปแบบมือถือ** บน Accordion และตรวจสอบว่าคอลัมน์ของ Masonry ไม่ได้แคบเกินไปบนโทรศัพท์มือถือ
- เมื่อใช้แม่แบบ List พร้อมเปิดใช้งาน **แสดงคำอธิบายสั้นๆ** ให้แน่ใจว่าสินค้าของคุณมีคำอธิบายสั้นๆ ที่ตั้งไว้ แถวคำอธิบายที่ว่างเปล่าดูเหมือนเป็นความผิดพลาดโดยไม่ตั้งใจ
