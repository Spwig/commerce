---
title: デザインとテーマ
---

デザインとテーマシステムを使用して、ストアフロント全体の外観を制御できます。色やタイポグラフィーからヘッダー、フッター、ページレイアウトまで、すべてをカスタマイズ可能です。**設定 > デザインとテーマ**にアクセスして、デザインダッシュボードを開きます。

![デザインダッシュボード](/static/core/admin/img/help/design-themes/theme-dashboard.webp)

## デザインダッシュボード

ダッシュボードは、ストアのデザイン状況の概要を提供します。

- **アクティブなテーマ** — 現在適用されているテーマを表示し、プレビューとクイックアクセスボタンが含まれます
- **デザイン統計** — インストールされたテーマ、カスタムヘッダー、カスタムフッター、メニューの数
- **セクションカード** — テーマ、ヘッダービルダー、フッタービルダー、メニュー、またはお知らせにジャンプします

## テーマ

### テーマの閲覧

**テーマ**セクションカードをクリックして、インストールされているすべてのテーマを表示します。各テーマカードには以下が表示されます:
- テーマ名とプレビュー画像
- 作者とバージョン
- アクティブ/非アクティブ状態

### テーマのアクティブ化

1. ご使用したいテーマの**アクティブ化**をクリックします
2. テーマはすぐにストアフロントに適用されます
3. 一度に1つのみテーマがアクティブになります

### テーマのカスタマイズ

各テーマは一連の**デザイントークン**をサポートしており、コードを編集せずに視覚的な外観を制御するための設定可能な値です。

アクティブなテーマの**カスタマイズ**をクリックしてトークンエディタにアクセスします。利用可能なトークンカテゴリには以下が含まれます:

| カテゴリ | 制御内容 |
|----------|-----------------|
| **色** | プライマリ、セカンダリ、アクセント色、背景、テキスト色 |
| **タイポグラフィー** | フォントファミリ、サイズ、ウェイト、行の高さ |
| **スペース** | マージン、パディング、要素間のギャップ |
| **境界** | 境界の幅、半径、色 |
| **シャドウ** | カード、ボタン、モーダルのボックスシャドウ |
| **ボタン** | ボタンスタイル、サイズ、ホバーエフェクト |
| **レイアウト** | コンテナの幅、グリッドギャップ、ブレイクポイント |

保存する前に変更をリアルタイムでプレビューできます。

## ヘッダービルダー

ヘッダービルダーは、ドラッグ＆ドロップインターフェースを使用してストアのヘッダーをデザインできます。

### ヘッダーの作成

1. **デザイン > ヘッダービルダー**にアクセスします
2. **ヘッダーを作成**をクリックして、または既存のものを編集します
3. ビルダーには3つの行があります: **トップバー**、**メインヘッダー**、**ボトムバー**
4. ツールボックスからウィジェットをドラッグして、任意の行に配置します

### 利用可能なヘッダーウィジェット

- **ロゴ** — 設定可能なサイズとリンクを持つストアロゴ
- **ナビゲーションメニュー** — 定義されたメニューからドロップダウンメニュー
- **検索バー** — 即時結果付きの製品検索
- **カートアイコン** — アイテム数のバッジ付きミニカート
- **アカウントアイコン** — ログイン/アカウントドロップダウン
- **言語セレクター** — 多言語ストア用の言語切り替え
- **通貨セレクター** — 多通貨ストア用の通貨切り替え
- **カスタムHTML** — 任意のカスタムコンテンツを追加
- **ソーシャルアイコン** — ソーシャルメディアプロフィールへのリンク
- **お知らせバー** — プロモーションメッセージやオファー

### ヘッダー設定

各ヘッダーテンプレートにはグローバル設定があります:
- **固定ヘッダー** — スクロール時にヘッダーが表示されます
- **透過モード** — ヒーロー画像の上にオーバーレイ
- **モバイルブレイクポイント** — モバイルレイアウトに切り替わるタイミング

## フッタービルダー

フッタービルダーはヘッダービルダーと同様に機能します。

### フッターの作成

1. **デザイン > フッタービルダー**にアクセスします
2. **フッターを作成**をクリックして、または既存のものを編集します
3. ビルダーは複数の列と行をサポートしています
4. ウィジェットを配置位置にドラッグします

### 利用可能なフッターウィジェット

- **ナビゲーションメニュー** — フッターナビゲーションリンク
- **ニュースレター登録** — メール購読フォーム
- **ソーシャルアイコン** — ソーシャルメディアリンク
- **カスタムHTML** — カスタムコンテンツ、バッジ、認証
- **支払いアイコン** — 承認された支払い方法を表示
- **著作権** — 年を含む動的な著作権テキスト
- **ロゴ** — フッターロゴのバリアント

## ナビゲーションメニュー

メニューは、ヘッダーとフッターのナビゲーションリンクを定義します。

### メニューの作成

1. **デザイン > メニュー**にアクセスします
2. **メニューを追加**をクリックします
3. メニューに名前を付けます（例: "メインナビゲーション"）
4. メニュー項目を追加します:
   - **ページリンク** — ページビルダーのページへのリンク
   - **カテゴリリンク** — 製品カテゴリへのリンク
   - **カスタムURL** — 任意の外部または内部URL
   - **ドロップダウン** — ネストされたサブメニュー項目
5. アイテムをドラッグして順序を変更します
6. 保存し、メニューをヘッダーまたはフッターウィジェットに割り当てます

## お知らせ

ストアフロントの上部に表示されるプロモーションバナーを作成します。

### お知らせの作成

1. **デザイン > お知らせ**にアクセスします（またはダッシュボードカードを使用）
2. **お知らせを追加**をクリックします
3. 次のように構成します:
   - **メッセージ** — お知らせのテキスト（翻訳をサポート）
   - **リンク** — クリックされたときに表示されるオプションのURL
   - **スタイル** — 背景色、テキスト色、アイコン
   - **スケジュール** — 開始日と終了日
   - **閉じる可能** — 顧客がそれを閉じることができるかどうか
4. 保存してアクティブ化します

複数のお知らせを同時にアクティブにできます — 自動的にローテーションされます。

## ヒント

- ヘッダーとフッターの作成前に、アクティブなテーマのカスタマイザでブランドカラーに合わせてください。
- ヘッダーやフッタービルダーの**プレビュー**機能を使用して、公開する前に変更を確認してください。
- デスクトップとモバイルで非常に異なるレイアウトが必要な場合は、それぞれのヘッダーを作成してください。
- ナビゲーションはシンプルに保つ — 5〜7のトップレベルのメニュー項目が使いやすさに最適です。
- 時間限定のプロモーションではなく、永続的なメッセージよりもお知らせを使用してください。
- テーマトークンエディタはリアルタイムプレビューをサポートしています — 自由に試して、満足したら保存してください。
