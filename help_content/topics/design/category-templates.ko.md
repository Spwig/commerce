카테고리 페이지 템플릿은 고객이 카테고리를 탐색할 때 제품이 어떻게 표시되는지를 제어합니다. 이는 고전적인 격자, 수평 목록, 스크롤 가능한 캐러셀 또는 드라마틱한 특징 레이아웃 등입니다. 전체 매장에 적용되는 단일 기본값을 설정할 수 있으며, 다른 처리가 필요한 개별 카테고리에서 이를 재정의할 수 있습니다.

## 설정을 찾는 위치

카테고리 템플릿을 구성할 수 있는 두 곳이 있습니다:

- **전체 사이트 기본값** — **디자인 > 페이지 템플릿**으로 이동하고 **카테고리** 섹션을 스크롤합니다. 여기에서 선택한 템플릿은 재정의되지 않는 한 모든 카테고리 페이지에 적용됩니다.
- **개별 카테고리 재정의** — **카탈로그 > 카테고리**로 이동하고, 어떤 카테고리든 열고, **표시** 탭을 클릭합니다. 여기서 템플릿을 설정하면 해당 카테고리에만 전체 사이트 기본값을 재정의합니다. 다른 모든 카테고리는 여전히 기본값을 사용합니다.

## 여섯 가지 템플릿

### 격자

격자 템플릿은 반응형 카드 격자에 제품을 배치합니다. 이는 온라인 쇼핑에 가장 익숙한 레이아웃입니다. 제품은 이미지, 이름, 가격, 장바구니에 추가 버튼이 있는 균일한 카드로 표시됩니다.

**가장 적합한 경우:** 일반적인 목적의 매장, 패션, 홈웨어, 전자기기 — 제품의 일관된 표시가 중요한 모든 매장.

| 옵션 | 설명 |
|--------|-------------|
| **행당 제품 수** | 데스크탑에서 한 줄에 표시되는 카드 수 (2, 3, 4 또는 5) |
| **카테고리 배너 표시** | 페이지 상단에 카테고리의 배너 이미지를 표시 |
| **서브카테고리 표시** | 배너 아래에 클릭 가능한 서브카테고리 칩을 표시 |
| **카테고리 설명 표시** | 카테고리의 설명 텍스트를 표시 |
| **카드 이미지 대체** | 제품에 사진이 없는 경우 대체 이미지를 표시 |

### 목록

목록 템플릿은 제품을 수평 행으로 표시합니다. 하나의 측면에 이미지, 다른 측면에 제품 세부 정보(이름, 설명, 가격)가 있습니다. 이는 각 제품에 더 많은 공간을 제공하고 고객이 한눈에 비교하는 것을 더 쉽게 만듭니다.

**가장 적합한 경우:** 복잡한 제품이 짧은 설명 미리보기로 이익을 얻는 경우, 예를 들어 도구, 가전제품 또는 B2B 카탈로그.

| 옵션 | 설명 |
|--------|-------------|
| **제품 이미지 표시** | 각 행에 제품 이미지를 포함 |
| **이미지 위치** | 이미지가 왼쪽 또는 오른쪽에 나타나는지 |
| **짧은 설명 표시** | 제품 이름 아래에 제품 요약을 표시 |
| **카테고리 배너 표시** | 페이지 상단에 카테고리의 배너 이미지를 표시 |
| **서브카테고리 표시** | 배너 아래에 클릭 가능한 서브카테고리 칩을 표시 |
| **카테고리 설명 표시** | 카테고리의 설명 텍스트를 표시 |

### 캐러셀

캐러셀 템플릿은 제품을 수평 스크롤 가능한 행으로 그룹화합니다. 이는 스트리밍 플랫폼의 "계속 탐색" 섹션과 유사합니다. 고객은 페이지를 떠나지 않고 제품을 탐색하기 위해 슬라이드하거나 화살표를 클릭할 수 있습니다.

**가장 적합한 경우:** 하나의 카테고리에 많은 제품이 있는 대규모 카탈로그, 선물 가게, 서점 또는 탐색과 발견이 우선시되는 모든 매장.

| 옵션 | 설명 |
|--------|-------------|
| **보기당 슬라이드 수** | 데스크탑에서 한 번에 표시되는 제품 수 (3, 4 또는 5) |
| **행 크기** | 캐러셀 행당 제품 수가 4, 6, 8 또는 12일 때 새로운 행을 아래에 시작하기 전에 |
| **네비게이션 화살표 표시** | 각 행에 이전/다음 화살표 버튼을 표시 |
| **점 표시** | 각 행 아래에 진행 표시줄 및 페이지 카운터(예: "1 / 3") 표시 |
| **자동 재생** | 슬라이드를 자동으로 진행 |
| **자동 재생 속도** | 각 자동 진행 사이의 밀리초 단위 시간 |
| **무한 루프** | 마지막 슬라이드에 도달한 후 첫 번째 슬라이드로 다시 돌아감 |
| **카테고리 배너 표시** | 페이지 상단에 카테고리의 배너 이미지를 표시 |
| **서브카테고리 표시** | 배너 아래에 클릭 가능한 서브카테고리 칩을 표시 |

### 마스터리

마스터리 템플릿은 Pinterest 스타일의 계단식 격자를 생성합니다. 각 카드는 이미지의 자연스러운 높이에 맞춰 조정됩니다. 결과는 균일한 격자가 아닌 다양한, 편집적인 느낌을 줍니다.

**가장 적합한 경우:** 사진, 예술 인쇄물, 수공예품, 인테리어 디자인 또는 제품의 시각적 특성이 다양하고 더 역동적인 방식으로 이를 보여주고 싶은 모든 카테고리.

| 옵션 | 설명 |
|--------|-------------|
| **열 수** | 페이지에 걸쳐 있는 열 수 (2, 3 또는 4) |
| **간격 크기** | 카드 사이의 간격 — 작음, 중간, 또는 큼 |
| **카테고리 배너 표시** | 페이지 상단에 카테고리의 배너 이미지를 표시 |
| **서브카테고리 표시** | 배너 아래에 클릭 가능한 서브카테고리 칩을 표시 |
| **카테고리 설명 표시** | 카테고리의 설명 텍스트를 표시 |

### 특징

특징 템플릿은 페이지 상단에 소수의 제품을 대형 히어로 카드로 강조하고, 나머지 제품은 아래에 지원 격자로 표시합니다. 이는 우선시 제품에 즉각적인 주목을 끌어옵니다.

**가장 적합한 경우:** 신상품 섹션, 베스트셀러 페이지, 큐레이션된 컬렉션 또는 특정 제품을 강조하고 싶은 계절 카테고리.

| 옵션 | 설명 |
|--------|-------------|
| **특징 제품 수** | 페이지 상단에서 히어로 처리를 받는 제품 수 (1, 2 또는 3) |
| **히어로 레이아웃** | 히어로 카드를 전체 너비로 표시하거나 분할 레이아웃으로 옆으로 배치 |
| **카테고리 배너 표시** | 페이지 상단에 카테고리의 배너 이미지를 표시 |
| **서브카테고리 표시** | 배너 아래에 클릭 가능한 서브카테고리 칩을 표시 |
| **카테고리 설명 표시** | 카테고리의 설명 텍스트를 표시 |

### 액ordion

액ordion 템플릿은 **루트 카테고리 페이지**에 특별히 설계되었습니다 — 모든 최상위 카테고리를 표시하는 페이지입니다. 이는 각 카테고리를 상호작용 가능한 패널로 표시합니다. 패널을 호버하거나 탭하면 해당 패널이 수평으로 확장되고 다른 패널은 압축됩니다. 개별 카테고리 페이지에서는 액ordion 템플릿이 표준 격자로 되돌아갑니다.

**가장 적합한 경우:** 시각적으로 강한 카테고리 이미지가 있는 매장에서 카테고리 페이지 자체가 몰입적인 경험을 제공하고 싶은 경우 — 패션 브랜드, 라이프스타일 매장 및 편집적인 정체성을 가진 매장.

| 옵션 | 설명 |
|--------|-------------|
| **행당 패널 수** | 액ordion 행당 최대 카테고리 수, 새 행으로 이어지기 전에 (4–8, 기본값 6) |
| **높이** | 액ordion 패널의 높이 — 작음, 중간, 큼 또는 엑스트라 큼 |
| **확장 비율** | 활성(호버) 패널이 비활성 패널보다 얼마나 더 넓게 확장되는지 (2–5×) |
| **전환 속도** | 애니메이션 속도(밀리초 단위) |
| **모바일 레이아웃** | 모바일에서 카테고리가 어떻게 표시되는지 — 수직으로 스택, 캐러셀 또는 격자 |
| **제품 수 표시** | 각 카테고리 패널에 제품 수 배지 표시 |

## 일반 옵션(모든 템플릿)

모든 템플릿은 주변 페이지의 크롬을 제어하는 일련의 옵션을 공유합니다:

| 옵션 | 설명 |
|--------|-------------|
| **경로 표시** | 카테고리 위에 "홈 > 카테고리 > 이름" 네비게이션 표시 |
| **정렬 바 표시** | "정렬 기준" 드롭다운 표시하여 고객이 결과를 재정렬할 수 있도록 함 |
| **제품 수 표시** | 총 제품 수 표시 (예: "29개 제품") |
| **도움 섹션 표시** | "원하시는 항목을 찾지 못하셨나요?" 섹션을 하단에 표시 |
| **기본 정렬** | 고객이 페이지에 처음 도착할 때 사용되는 정렬 순서 |
| **페이지당 제품 수** | 페이지당 로드할 제품 수 |
| **페이지네이션 스타일** | 추가 제품이 어떻게 로드되는지: 페이지네이션 페이지, "더 보기" 버튼 또는 무한 스크롤 |

## 빠른 비교

이 표를 사용하여 적절한 템플릿을 한눈에 선택할 수 있습니다:

| 템플릿 | 페이지당 제품 수 | 시각적 특징 | 가장 적합한 제품 유형 |
|----------|-----------------|------------------|--------------------|
| **격자** | 2–5 설정 가능 | 균일하고 깔끔함 | 모든 제품 — 안전한 기본값 |
| **목록** | 1 (전체 너비 행) | 넓고 정보가 풍부함 | 복잡하거나 기술적인 제품 |
| **캐러셀** | 한 번에 3–5개 표시 | 탐색 가능하고 역동적 | 대규모 카탈로그, 발견 |
| **마스터리** | 2–4 (계단식) | 편집적이고 다양함 | 시각적/수공예 제품 |
| **특징** | 히어로 + 아래 격자 | 홍보용 | 큐레이션된 선택, 신상품 |
| **액ordion** | N/A (카테고리) | 몰입적이고 극적인 | 루트 카테고리 페이지 |

## 전체 사이트 기본 템플릿 설정

1. **디자인 > 페이지 템플릿**으로 이동합니다.
2. **카테고리** 섹션으로 스크롤합니다.
3. 드롭다운 메뉴에서 선호하는 템플릿을 선택합니다.
4. 드롭다운 아래에 나타나는 템플릿 옵션을 구성합니다.
5. 일반 옵션(정렬 바, 경로, 페이지네이션 등)을 구성합니다.
6. **설정 저장**을 클릭합니다.

선택한 템플릿 및 설정은 이제 매장의 모든 카테고리 페이지에 적용됩니다.

## 특정 카테고리의 템플릿 재정의

전체 매장의 기본값을 변경하지 않고도 어떤 개별 카테고리에도 다른 템플릿을 적용할 수 있습니다.

1. **카탈로그 > 카테고리**로 이동합니다.
2. 커스터마이징하고자 하는 카테고리를 클릭합니다.
3. 편집 폼 상단의 **표시** 탭을 클릭합니다.
4. **템플릿** 아래에서 **커스텀**을 선택하여 개별 카테고리 설정을 활성화합니다.
5. 이 카테고리에 원하는 템플릿을 선택합니다.
6. 필요한 경우 템플릿 옵션을 구성합니다.
7. **저장**을 클릭합니다.

이 카테고리만 커스텀 템플릿을 사용합니다. 다른 모든 카테고리는 여전히 전체 사이트 기본값을 사용합니다. 카테고리를 기본값으로 되돌리려면 해당 카테고리의 **표시** 탭으로 돌아가고 템플릿을 **사이트 기본값 사용**으로 다시 설정합니다.

## 예시: 카테고리 간 템플릿 혼합

다음은 동일한 매장의 다른 부분에 다양한 템플릿이 어떻게 사용될 수 있는 실용적인 예입니다:

| 카테고리 | 템플릿 | 이유 |
|----------|----------|--------|
| 모든 카테고리(루트 페이지) | 액ordion | 시각적으로 극적인 진입점 |
| 의류 | 격자(행당 4개) | 빠르고 익숙한 탐색 |
| 할인 | 특징 | 즉시 최상의 거래를 강조 |
| 도서 | 캐러셀 | 대규모 카탈로그, 쉽게 탐색 |
| 수공예 선물 | 마스터리 | 다양한 이미지 크기, 수공예 느낌 |
| 산업 도구 | 목록 | 고객이 설명을 보고 싶어 함 |

## 팁

- 전체 사이트 기본값으로 **격자** 템플릿을 시작하세요. 이는 거의 모든 제품 유형에 잘 작동하며, 쇼퍼에게 가장 익숙한 레이아웃입니다. 여기서부터 개별 카테고리를 커스터마이징하세요.
- **특징** 템플릿은 적게 사용하세요. 이는 해당 카테고리에 대해 진정으로 큐레이션된 1–3개의 주요 제품을 제공할 수 있을 때 가장 잘 작동합니다. 모든 제품이 동일하게 중요해 보이면 히어로 처리는 효과를 잃습니다.
- **액ordion** 템플릿은 루트 카테고리 페이지에서만 전체 시각적 효과를 만듭니다. 제품 카테고리에 할당하면 고객은 일반적인 격자를 보게 됩니다. 따라서 최상위 카테고리 목록에만 예약하세요.
- **페이지당 제품 수**를 신중하게 설정하세요. 24–36개의 제품당 페이지는 격자 및 마스터리 템플릿에 잘 작동합니다. 캐러셀의 경우, 행 크기 및 슬라이드당 보기 설정이 이 경험을 더 직접적으로 제어합니다.
- 부모 카테고리에 **서브카테고리 표시**를 활성화하여 고객이 정렬 또는 필터 도구를 사용하지 않고도 더 좁은 선택으로 바로 이동할 수 있도록 합니다.
