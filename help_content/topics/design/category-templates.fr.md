---
title: Modèles de pages de catégories
---

Les modèles de pages de catégories contrôlent l'affichage de vos produits lorsque les clients naviguent dans une catégorie — qu'il s'agisse d'une grille classique, d'une liste horizontale, d'un carrousel défilable ou d'un mise en page mise en avant dramatique. Vous pouvez définir un seul modèle par défaut qui s'applique à votre boutique entière, puis le remplacer sur toute catégorie individuelle qui nécessite un traitement différent.

## Où trouver les paramètres

Il y a deux endroits pour configurer les modèles de catégories :

- **Défaut global du site** — Accédez à **Design > Modèles de pages** et faites défiler jusqu'à la section **Catégorie**. Le modèle que vous choisissez ici s'applique à toutes les pages de catégorie sauf s'il est remplacé.
- **Remplacement par catégorie** — Accédez à **Catalogue > Catégories**, ouvrez une catégorie, puis cliquez sur l'onglet **Affichage**. Définir un modèle ici remplace le modèle global du site pour cette catégorie uniquement. Toutes les autres catégories continuent d'utiliser le modèle par défaut.

## Les six modèles

### Grille

Le modèle Grille arrange les produits dans une grille de cartes réactive — le layout le plus familier pour le shopping en ligne. Les produits s'affichent sous forme de cartes uniformes avec image, nom, prix et bouton d'ajout au panier.

**Meilleur pour :** Magasins généralistes, mode, produits de maison, électronique — tout magasin où la présentation cohérente des produits est importante.

| Option | Description |
|--------|-------------|
| **Nombre de produits par ligne** | Combien de cartes s'affichent par ligne sur les ordinateurs (2, 3, 4 ou 5) |
| **Afficher le bandeau de catégorie** | Afficher l'image du bandeau de catégorie en haut de la page |
| **Afficher les sous-catégories** | Afficher des puces de sous-catégories cliquables sous le bandeau |
| **Afficher la description de catégorie** | Afficher le texte de description de la catégorie |
| **Image de carte de secours** | Afficher une image de remplacement lorsqu'un produit n'a pas de photo |

### Liste

Le modèle Liste affiche les produits en lignes horizontales — image d'un côté, détails du produit (nom, description, prix) de l'autre. Cela donne à chaque produit plus d'espace et facilite la comparaison rapide des clients.

**Meilleur pour :** Magasins avec des produits complexes qui bénéficient d'une prévisualisation de courte description, tels que les outils, les électroménagers ou les catalogues B2B.

| Option | Description |
|--------|-------------|
| **Afficher l'image du produit** | Inclure l'image du produit dans chaque ligne |
| **Position de l'image** | Si l'image apparaît à gauche ou à droite |
| **Afficher la courte description** | Afficher un extrait du produit sous le nom du produit |
| **Afficher le bandeau de catégorie** | Afficher l'image du bandeau de catégorie en haut de la page |
| **Afficher les sous-catégories** | Afficher des puces de sous-catégories cliquables sous le bandeau |
| **Afficher la description de catégorie** | Afficher le texte de description de la catégorie |

### Carrousel

Le modèle Carrousel regroupe les produits en lignes horizontales défilables — similaire aux sections « continuez à naviguer » des plateformes de streaming. Les clients défilent ou cliquent sur les flèches pour parcourir les produits sans quitter la page.

**Meilleur pour :** Grandes collections avec de nombreux produits dans une seule catégorie, magasins de cadeaux, librairies ou tout magasin où la navigation et la découverte sont prioritaires.

| Option | Description |
|--------|-------------|
| **Slides par vue** | Combien de produits sont visibles à la fois sur les ordinateurs (3, 4 ou 5) |
| **Taille de la ligne** | Combien de produits par ligne de carrousel avant de commencer une nouvelle ligne en dessous (4, 6, 8 ou 12) |
| **Afficher les flèches de navigation** | Afficher les boutons de flèche précédent/suivant sur chaque ligne |
| **Afficher les points** | Afficher une barre de progression et un compteur de page (ex. « 1 / 3 ») sous chaque ligne |
| **Auto-avance** | Avancer automatiquement les diapositives |
| **Vitesse d'auto-avance** | Temps en millisecondes entre chaque auto-avance |
| **Boucle infinie** | Revenir au premier diapositive après avoir atteint la dernière |
| **Afficher le bandeau de catégorie** | Afficher l'image du bandeau de catégorie en haut de la page |
| **Afficher les sous-catégories** | Afficher des puces de sous-catégories cliquables sous le bandeau |

### Masonry

Le modèle Masonry crée une grille décalée de type Pinterest où chaque carte s'adapte à la hauteur naturelle de son image. Le résultat est une ambiance éditoriale variée plutôt qu'une grille uniforme.

**Meilleur pour :** Photographie, impressions d'art, produits artisanaux, design d'intérieur ou toute catégorie où le caractère visuel des produits varie et où vous souhaitez les mettre en avant de manière plus dynamique.

| Option | Description |
|--------|-------------|
| **Colonnes** | Nombre de colonnes à travers la page (2, 3 ou 4) |
| **Taille de l'espace** | Espace entre les cartes — petit, moyen ou grand |
| **Afficher le bandeau de catégorie** | Afficher l'image du bandeau de catégorie en haut de la page |
| **Afficher les sous-catégories** | Afficher des puces de sous-catégories cliquables sous le bandeau |
| **Afficher la description de catégorie** | Afficher le texte de description de la catégorie |

### Mise en avant

Le modèle Mise en avant met en avant un petit nombre de produits en tant que grandes cartes héro en haut de la page, avec les autres produits affichés en grille de soutien en dessous. Il attire immédiatement l'attention sur vos produits prioritaires.

**Meilleur pour :** Sections d'arrivées récentes, pages de best-sellers, collections curationnées ou catégories saisonnières où vous souhaitez mettre en avant des produits spécifiques.

| Option | Description |
|--------|-------------|
| **Nombre de produits mis en avant** | Combien de produits reçoivent un traitement héro en haut (1, 2 ou 3) |
| **Mise en page héro** | Afficher les cartes héro en largeur totale ou en tant que mise en page divisée côte à côte |
| **Afficher le bandeau de catégorie** | Afficher l'image du bandeau de catégorie en haut de la page |
| **Afficher les sous-catégories** | Afficher des puces de sous-catégories cliquables sous le bandeau |
| **Afficher la description de catégorie** | Afficher le texte de description de la catégorie |

### Accordion

Le modèle Accordion est conçu spécifiquement pour la **page de catégories racine** — la page qui affiche toutes vos catégories de niveau supérieur. Il affiche chaque catégorie comme un panneau interactif ; le survol ou le tap sur un panneau le fait s'élargir horizontalement pendant que les autres se réduisent. Sur les pages de catégories individuelles, le modèle Accordion revient à une grille standard.

**Meilleur pour :** Magasins avec des images de catégories visuellement fortes où vous souhaitez que la page de catégories elle-même soit une expérience immersive — marques de mode, magasins de lifestyle et magasins avec une forte identité éditoriale.

| Option | Description |
|--------|-------------|
| **Panneaux par ligne** | Nombre maximum de catégories par ligne d'accordion avant de passer à une nouvelle ligne en dessous (4–8, par défaut 6) |
| **Hauteur** | Combien de haut sont les panneaux d'accordion — petit, moyen, grand ou extra grand |
| **Ratio d'expansion** | Combien plus large le panneau actif (survolé) s'élargit par rapport aux panneaux inactifs (2–5×) |
| **Vitesse de transition** | Vitesse d'animation en millisecondes |
| **Mise en page mobile** | Comment les catégories s'affichent sur mobile — empilées verticalement, en tant que carrousel ou en tant que grille |
| **Afficher le nombre de produits** | Afficher un badge de nombre de produits sur chaque panneau de catégorie |

## Options courantes (tous les modèles)

Chaque modèle partage un ensemble d'options qui contrôlent le chrome de la page entourant :

| Option | Description |
|--------|-------------|
| **Afficher le fil d'Ariane** | Afficher la navigation « Accueil > Catégories > Nom » au-dessus de la catégorie |
| **Afficher la barre de tri** | Afficher le menu déroulant « Trier par » afin que les clients puissent réordonner les résultats |
| **Afficher le nombre de produits** | Afficher le nombre total de produits (ex. « 29 produits ») |
| **Afficher la section d'aide** | Afficher une section « Ne trouvez pas ce que vous cherchez ? » en bas |
| **Tri par défaut** | L'ordre de tri utilisé lorsque les clients atterrissent pour la première fois sur la page |
| **Nombre de produits par page** | Combien de produits charger par page |
| **Style de pagination** | Comment les produits supplémentaires sont chargés : pages paginées, un bouton « Charger plus » ou un défilement infini |

## Comparaison rapide

Utilisez ce tableau pour choisir le bon modèle à l'œil :

| Modèle | Nombre de produits par ligne | Caractère visuel | Meilleurs types de produits |
|----------|-----------------|------------------|--------------------|
| **Grille** | 2–5 configurable | Uniforme, propre | Tout — le choix par défaut sûr |
| **Liste** | 1 (lignes à largeur totale) | Spacieux, informatif | Produits complexes ou techniques |
| **Carrousel** | 3–5 visibles à la fois | Navigable, dynamique | Grandes collections, découverte |
| **Masonry** | 2–4 (décalés) | Éditorial, varié | Produits visuels/artisanaux |
| **Mise en avant** | Héro + grille en dessous | Promotionnel | Sélections curationnées, arrivées récentes |
| **Accordion** | N/A (catégories) | Immersif, dramatique | Page de catégories racine |

## Définir un modèle par défaut global du site

1. Accédez à **Design > Modèles de pages**.
2. Faites défiler jusqu'à la section **Catégorie**.
3. Sélectionnez votre modèle préféré dans le menu déroulant **Modèle**.
4. Configurez les options du modèle qui apparaissent en dessous du menu déroulant.
5. Configurez les options courantes (barre de tri, fil d'Ariane, pagination, etc.).
6. Cliquez sur **Enregistrer la configuration**.

Le modèle sélectionné et ses paramètres s'appliquent maintenant à toutes les pages de catégorie de votre boutique.

## Remplacer le modèle pour une catégorie spécifique

Vous pouvez donner à toute catégorie individuelle un modèle différent sans modifier le modèle par défaut du magasin.

1. Accédez à **Catalogue > Catégories**.
2. Cliquez sur la catégorie que vous souhaitez personnaliser.
3. Cliquez sur l'onglet **Affichage** en haut du formulaire d'édition.
4. Sous **Modèle**, choisissez **Personnalisé** pour activer les paramètres par catégorie.
5. Sélectionnez le modèle que vous souhaitez pour cette catégorie.
6. Configurez les options du modèle comme nécessaire.
7. Cliquez sur **Enregistrer**.

Seule cette catégorie utilisera le modèle personnalisé. Toutes les autres catégories continuent d'utiliser le modèle par défaut du site. Pour revenir à la catégorie par défaut, retournez à son onglet Affichage et définissez le modèle à nouveau sur **Utiliser le modèle par défaut du site**.

## Exemple : mélanger les modèles entre les catégories

Voici un exemple pratique de la manière dont différents modèles peuvent servir différentes parties du même magasin :

| Catégorie | Modèle | Raison |
|----------|----------|--------|
| Toutes les catégories (page racine) | Accordion | Point d'entrée visuel dramatique |
| Vêtements | Grille (4 par ligne) | Navigation rapide et familière |
| Vente | Mise en avant | Met en avant les meilleures offres immédiatement |
| Livres | Carrousel | Grand catalogue, facile à naviguer |
| Cadeaux faits main | Masonry | Tailles d'images variées, ambiance artisanale |
| Outils industriels | Liste | Les clients doivent voir les descriptions |

## Conseils

- Commencez par le modèle **Grille** comme modèle par défaut global du site — il fonctionne bien pour presque tous les types de produits et est le layout le plus familier pour les acheteurs. Personnalisez les catégories individuelles à partir de là.
- Utilisez le modèle **Mise en avant** avec modération. Il fonctionne le mieux lorsque vous pouvez vraiment curer 1–3 produits remarquables pour cette catégorie. Si tous les produits semblent également importants, le traitement héro perd son effet.
- Le modèle **Accordion** crée son plein impact visuel uniquement sur la page de catégories racine. Si vous l'assignez à une catégorie de produits, les clients verront une grille ordinaire — réservez-le donc à votre liste de catégories de niveau supérieur.
- Définissez **nombre de produits par page** de manière réfléchie. 24–36 produits par page fonctionnent bien pour les modèles Grille et Masonry. Pour le Carrousel, les paramètres de taille de ligne et de slides-par-vue contrôlent cette expérience plus directement.
- Activez **afficher les sous-catégories** sur les catégories parentes afin que les clients puissent sauter directement à une sélection plus étroite sans avoir besoin d'utiliser les outils de tri ou de filtre.
- Testez votre modèle choisi sur mobile. Les modèles Masonry et Carrousel en particulier ont un aspect très différent sur les petits écrans — utilisez l'option **mise en page mobile** sur Accordion, et vérifiez que les colonnes Masonry ne sont pas trop étroites sur les téléphones.
- Lorsque vous utilisez le modèle Liste avec **afficher la courte description** activé, assurez-vous que vos produits ont effectivement des courtes descriptions définies. Les lignes de description vides semblent accidentelles.
