---
title: Редактор страниц
---

Редактор страниц — это визуальный драг-энд-дроп-редактор для создания богатых, отзывчивых страниц без написания кода. Добавьте элементы из библиотеки из 39 компонентов, стилизуйте их с помощью мощных утилит, настройте анимации и правила видимости, а затем опубликуйте с полной историей версий.

![Редактор страниц](/static/core/admin/img/help/page-builder/builder-overview.webp)

## Интерфейс редактора

Редактор имеет четыре основных области:

| Область | Расположение | Назначение |
|--------|-------------|----------|
| **Панель инструментов** | Верхняя панель | Предварительный просмотр устройства (настольный/планшет/мобильный), отмена/повтор, настройки страницы, сохранение черновика, публикация |
| **Библиотека элементов** | Левая боковая панель | Обзор и перетаскивание 39 элементов, организованных в 9 категориях |
| **Канвас** | Центральная область | Живая редакция WYSIWYG — видите изменения при их внесении |
| **Панель свойств** | Правая боковая панель | Редактирование содержимого, стиля, анимаций и расширенных настроек выбранного элемента |

## Библиотека элементов

Элементы организованы в категории. Перетащите любой элемент из библиотеки на канвас, чтобы добавить его на свою страницу.

| Категория | Элементы |
|----------|----------|
| **Макет** | Контейнер, Разделитель, Герой-раздел, Всплывающее окно, Навигационное меню, Пространство |
| **Базовые** | Заголовок, Текст, Кнопка, Иконка |
| **Контент** | Карусель блога, Сетка блога, Аккордеон часто задаваемых вопросов, Связанные публикации, Отзывы |
| **Медиа** | Изображение, Галерея изображений, Аккордеон изображений, Встроенное видео |
| **Формы** | Форма обратной связи, Форма, Регистрация на рассылку |
| **Маркетинг** | Отсчет времени, Баннер призыва к действию, Баннер популярного блога, Баннер лояльности, Баннер продвижения, Знаки доверия, Отображение кода купона |
| **Электронная коммерция** | Отображение категории, Промо-карточка подарочной карты, Карусель продуктов, Сетка продуктов, Список продуктов, Отображение отзывов, Продукты со скидкой, Поиск магазина |
| **Социальные сети** | Ссылки в соцсетях |
| **Навигация** | Панель поиска |

### Контейнеры и вложенность

Элемент **Контейнер** является основой для сложных макетов. Контейнеры могут содержать другие элементы — включая другие контейнеры — позволяя создавать многостолбцовые сетки и вложенные структуры. Используйте предустановки макета контейнера для быстрого настройки распространенных расположений столбцов (50/50, 33/33/33, 25/75 и т.д.).

## Добавление элементов

1. Найдите нужный элемент в левой боковой панели
2. **Перетащите** его на канвас и отпустите в нужном месте
3. Элементы можно перетаскивать между существующими элементами или внутрь контейнеров
4. Синяя линия вставки показывает, где будет расположен элемент
5. После отпускания элемент автоматически выбирается, и открывается панель свойств

Вы также можете переупорядочить элементы, перетаскивая их вверх или вниз на канвасе.

## Редактирование содержимого

Выберите любой элемент на канвасе, чтобы открыть его свойства в правой панели. Вкладка **Содержимое** показывает поля, специфичные для этого типа элемента.

![Панель свойств](/static/core/admin/img/help/page-builder/properties-panel.webp)

Например:
- **Заголовок** — текст, HTML-тег (H1–H6), выравнивание, идентификатор якоря
- **Изображение** — источник изображения (библиотека медиа), альтернативный текст, ссылка, размеры
- **Кнопка** — текст, URL, стиль, иконка
- **Сетка продуктов** — источник данных, количество столбцов, количество продуктов на странице, порядок сортировки
- **Герой-раздел** — заголовок, подзаголовок, описание, фон, кнопки призыва к действию

Поля содержимого, которые можно перевести, показывают значок перевода — нажмите на него, чтобы добавить переводы для магазинов с несколькими языками.

## Стилизация элементов

Вкладка **Стиль** предоставляет визуальные элементы управления для каждого элемента. Каждый раздел открывает редактор специфических утилит.

![Вкладка Стиль](/static/core/admin/img/help/page-builder/style-tab.webp)

| Раздел | Что он контролирует | Утилита |
|--------|-------------------|--------|
| **Типографика** | Семейство шрифтов, размер, вес, высота строки, межбуквенный интервал, стиль текста | Редактор типографики |
| **Цвета** | Цвет текста с вводом HEX/RGB/HSL и токенами темы | Выбираемый цвет |
| **Фон** | Твердый цвет, градиент, изображение или видео с фоном и состоянием наведения | Редактор фона |
| **Граница** | Ширина, стиль, цвет и радиус границы по сторонам | Редактор границ |
| **Отступы** | Отступы и паддинги с визуальным редактором модели коробки | Редактор отступов |
| **Эффекты** | Тень с предустановками и поддержкой нескольких слоев, ползунок прозрачности | Редактор теней |

Каждая утилита документируется в своем собственном разделе помощи — ищите "выбираемый цвет", "редактор фона" и т.д., чтобы узнать больше.

## Анимации

Вкладка **Анимации** позволяет добавить движение к элементам.

### Входные анимации

Триггер активируется, когда элемент прокручивается в поле зрения:

| Анимация | Описание |
|----------|---------|
| Постепенное появление | Постепенно появляется |
| Слайд (вверх/вниз/влево/вправо) | Слайд в направлении |
| Увеличение | Растет от маленького до полного размера |
| Прыжок вперед | Прыгает на место |
| Пульс / Дрожание / Прыжок / Вспышка / Вращение | Эффекты привлечения внимания |

Настройте **длительность** (0,3с–1,5с), **задержку** (0–1с), **функцию времени** (ease, ease-in, ease-out, linear) и **повтор** (один раз или бесконечно).

### Анимации при наведении

Триггер активируется, когда посетитель наводит указатель на элемент:

| Эффект | Описание |
|--------|---------|
| Увеличение / Уменьшение | Увеличивается или уменьшается |
| Подъем | Поднимается вверх |
| Вращение (по часовой стрелке / против часовой стрелки) | Вращается по часовой стрелке или против часовой стрелки |
| Увеличение яркости / Постепенное исчезновение | Изменение яркости или прозрачности |
| Расширение тени | Тень расширяется |
| Подъем с тенью | Поднимается с расширяющейся тенью |
| Пульсирующее увеличение / Смещение / Светящаяся граница | Особые эффекты |

Настройте **длительность**, **время** и **интенсивность** (незначительная, нормальная, сильная).

## Расширенные настройки

Вкладка **Расширенные** предоставляет тонкую настройку:

### Правила видимости

Управляйте тем, когда элемент отображается или скрывается на основе условий:

- **Статус пользователя** — вошел в систему, вышел из системы, новый клиент, возвращающийся клиент
- **Устройство** — настольный, планшет, мобильный
- **Время** — диапазон дат, время суток, день недели
- **Группа клиентов** — VIP, оптовая продажа и т.д.
- **Сумма корзины** — минимальная или максимальная сумма корзины
- **География** — страна, регион
- И более 20 типов правил

Правила можно комбинировать с логикой И/ИЛИ для сложной настройки целевой аудитории.

### Пользовательский CSS

| Поле | Назначение |
|------|-----------|
| **ID элемента** | Уникальный ID для якорных ссылок или целевых CSS |
| **Пользовательские классы CSS** | Дополнительные классы для применения |
| **Пользовательские стили CSS** | Встроенный CSS для одноразовых переопределений |
| **Данные атрибуты** | Пользовательские атрибуты data-* в виде пар ключ-значение |
| **Z-Index** | Порядок наложения для перекрывающихся элементов |

## Рабочий процесс публикации

Страницы используют систему черновиков/публикации с полной историей версий:

| Статус | Значение |
|--------|--------|
| **Черновик** | Работа в процессе — не видно посетителям |
| **Опубликовано** | Опубликовано на вашем магазине |
| **Архивировано** | Удалено с сайта, но сохранено |

### Как это работает

1. Вносите изменения в редактор — они сохраняются как **черновик**
2. Нажмите **Сохранить черновик**, чтобы сохранить без публикации
3. Нажмите **Опубликовать**, чтобы сделать текущий черновик активным
4. Каждая публикация создает **снимок версии**
5. Вы можете **восстановить** любую предыдущую версию из истории версий (значок часов в панели инструментов)

Это означает, что вы можете свободно экспериментировать — ваша активная страница остается неизменной, пока вы не опубликуете ее явно.

## Шаблоны страниц

Сэкономьте время, работая с шаблонами:

- **Сохранить как шаблон** — сохраните дизайн любой страницы как повторно используемый шаблон
- **Создать из шаблона** — начните новую страницу с существующего шаблона
- **Категории шаблонов** — организуйте шаблоны по назначению (лендинг-страница, о нас, демонстрация продуктов и т.д.)

Шаблоны захватывают полную структуру страницы, включая все элементы, содержимое и стили.

## Адаптивный дизайн

Используйте кнопки предварительного просмотра устройства в панели инструментов, чтобы увидеть, как выглядит ваша страница на разных размерах экрана:

- **Настольный** — полный макет
- **Планшет** — средний экран
- **Мобильный** — узкий экран

Элементы автоматически перестраиваются в зависимости от настроек контейнера. Вы также можете использовать правила видимости, чтобы отображать или скрывать определенные элементы на определенных устройствах.

## Советы

- **Начните с контейнера** — большинство макетов начинаются с контейнера для создания колонок и структуры. Используйте предустановки макета для распространенных расположений.
- **Используйте разделы Герой для заголовков страниц** — элемент Герой предоставляет заголовок, подзаголовок, фоновое изображение и кнопки призыва к действию в одном компоненте.
- **Предварительный просмотр перед публикацией** — нажмите Предварительный просмотр, чтобы увидеть, что увидят посетители, а затем опубликуйте, когда будете довольны.
- **Используйте правила видимости для персонализации** — показывайте разное содержимое для пользователей, вошедших в систему, и для пользователей, вышедших из системы, или нацеливайтесь на определенные группы клиентов.
- **Держите анимации ненавязчивыми** — один или два входных анимации на раздел страницы выглядят профессионально. Слишком много анимаций может быть слишком много.
- **Называйте свои контейнеры** — используйте поле ID элемента, чтобы пометить контейнеры (например, "hero-section", "features"), чтобы они были легко найдены в сложных страницах.
- **Тестируйте на всех устройствах** — используйте предварительный просмотр устройства, чтобы проверить ваш макет на настольных, планшетных и мобильных устройствах перед публикацией.
- **Используйте шаблоны** — сохраните лучшие дизайны страниц как шаблоны, чтобы ускорить создание будущих страниц.
