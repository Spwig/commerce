---
title: 페이지 빌더
---

페이지 빌더는 코드를 작성하지 않고도 풍부하고 반응형 페이지를 생성할 수 있는 시각적 드래그 앤 드롭 편집기입니다. 39개의 구성 요소로 이루어진 라이브러리에서 요소를 추가하고, 강력한 유틸리티로 스타일을 지정하고, 애니메이션과 가시성 규칙을 설정한 후 전체 버전 기록으로 게시합니다.

![Page Builder](/static/core/admin/img/help/page-builder/builder-overview.webp)

## 빌더 인터페이스

빌더에는 네 가지 주요 영역이 있습니다:

| 영역 | 위치 | 목적 |
|------|----------|---------|
| **도구 모음** | 상단 바 | 기기 미리보기 (데스크탑/태블릿/모바일), 되돌리기/재실행, 페이지 설정, 초안 저장, 게시 |
| **요소 라이브러리** | 왼쪽 사이드바 | 9개의 카테고리로 정리된 39개의 요소를 탐색하고 드래그할 수 있습니다 |
| **캔버스** | 중앙 | 실시간 WYSIWYG 편집 영역 — 변경 사항을 즉시 확인할 수 있습니다 |
| **속성 패널** | 오른쪽 사이드바 | 선택한 요소의 콘텐츠, 스타일, 애니메이션 및 고급 설정을 편집합니다 |

## 요소 라이브러리

요소는 카테고리별로 정리되어 있습니다. 라이브러리에서 원하는 요소를 캔버스 위로 드래그하여 페이지에 추가합니다.

| 카테고리 | 요소 |
|----------|----------|
| **레이아웃** | 컨테이너, 분리자, 히어로 섹션, 모달 팝업, 네비게이션 메뉴, 간격 |
| **기본** | 제목, 텍스트, 버튼, 아이콘 |
| **콘텐츠** | 블로그 게시물 캐로셀, 블로그 게시물 그리드, FAQ 액ordion, 관련 게시물, 추천 |
| **미디어** | 이미지, 이미지 갤러리, 이미지 액ordion, 비디오 삽입 |
| **폼** | 연락처 양식, 양식, 뉴스레터 등록 |
| **마케팅** | 카운트다운 타이머, CTA 배너, 특징 블로그 배너, 충성도 배너, 프로모션 배너, 신뢰 배지, 쿠폰 코드 표시 |
| **전자상거래** | 카테고리 전시, 기프트 카드 프로모션, 제품 캐로셀, 제품 그리드, 제품 목록, 리뷰 표시, 할인 제품, 매장 위치 찾기 |
| **소셜** | 소셜 링크 |
| **네비게이션** | 검색 바 |

### 컨테이너 및 중첩

**컨테이너** 요소는 복잡한 레이아웃의 기초입니다. 컨테이너는 다른 요소 — 다른 컨테이너도 포함 —을 포함할 수 있어 다중 열 그리드 및 중첩 구조를 만들 수 있습니다. 컨테이너의 레이아웃 사전 설정을 사용하여 일반적인 열 배치(50/50, 33/33/33, 25/75 등)를 빠르게 설정할 수 있습니다.

## 요소 추가

1. 왼쪽 사이드바에서 원하는 요소를 찾습니다
2. **드래그**하여 캔버스 위로 끌어다 놓고 원하는 위치에 놓습니다
3. 요소는 기존 요소 사이에, 또는 컨테이너 내부에 놓을 수 있습니다
4. 파란색 삽입선은 요소가 어디에 배치될지를 보여줍니다
5. 놓은 후 요소가 자동으로 선택되고 속성 패널이 열립니다

캔버스 위에서 요소를 위쪽이나 아래쪽으로 드래그하여 순서를 재정렬할 수도 있습니다.

## 콘텐츠 편집

캔버스 위의 요소를 선택하여 오른쪽 패널에 해당 요소의 속성을 열 수 있습니다. **콘텐츠** 탭은 해당 요소 유형에 해당하는 필드를 보여줍니다.

![Properties Panel](/static/core/admin/img/help/page-builder/properties-panel.webp)

예시:
- **제목** — 텍스트, HTML 태그(H1–H6), 정렬, 앵커 ID
- **이미지** — 이미지 소스(미디어 라이브러리), 대체 텍스트, 링크, 크기 조정
- **버튼** — 라벨, URL, 스타일 변형, 아이콘
- **제품 그리드** — 데이터 소스, 열 수, 페이지당 제품 수, 정렬 순서
- **히어로 섹션** — 제목, 부제, 설명, 배경, CTA 버튼

번역 가능한 콘텐츠 필드는 번역 아이콘을 표시합니다 — 클릭하여 다국어 상점을 위한 번역을 추가합니다.

## 요소 스타일링

**스타일** 탭은 모든 요소에 대한 시각적 제어를 제공합니다. 각 섹션은 전용 유틸리티 편집기를 엽니다.

![Style Tab](/static/core/admin/img/help/page-builder/style-tab.webp)

| 섹션 | 제어 항목 | 유틸리티 |
|---------|-----------------|---------|
| **타이포그래피** | 글꼴 가족, 크기, 굵기, 줄 간격, 글자 간격, 텍스트 스타일 | 타이포그래피 편집기 |
| **색상** | 텍스트 색상과 헥스/RGB/HSL 입력 및 테마 토큰 | 색상 선택기 |
| **배경** | 고정 색상, 그라디언트, 이미지 또는 비디오 배경과 호버 상태 | 배경 편집기 |
| **테두리** | 각 측면의 테두리 두께, 스타일, 색상 및 반경 | 테두리 편집기 |
| **간격** | 시각적 박스 모델 편집기와 마진 및 패딩 | 간격 편집기 |
| **효과** | 사전 설정 및 다중 레이어 지원을 위한 박스 그림자, 투명도 슬라이더 | 그림자 편집기 |

각 유틸리티는 자체 도움말 주제에 문서화되어 있습니다 — "색상 선택기", "배경 편집기" 등과 같은 키워드로 검색하여 자세한 내용을 확인할 수 있습니다.

## 애니메이션

**애니메이션** 탭은 요소에 움직임을 추가할 수 있습니다.

### 진입 애니메이션

요소가 화면에 스크롤될 때 트리거됩니다:

| 애니메이션 | 설명 |
|-----------|-------------|
| Fade In | 점차 나타납니다 |
| Slide In (Up/Down/Left/Right) | 방향에서 슬라이드 인 |
| Zoom In | 작은 크기에서 전체 크기로 성장 |
| Bounce In | 위치로 튀어 오릅니다 |
| Pulse / Shake / Bounce / Flash / Spin | 주의를 끌 수 있는 효과 |

**지속 시간**(0.3s–1.5s), **지연**(0–1s), **타이밍 함수**(ease, ease-in, ease-out, linear), **반복**(한 번 또는 무한)을 구성할 수 있습니다.

### 호버 애니메이션

방문자가 요소 위로 마우스를 올릴 때 트리거됩니다:

| 효과 | 설명 |
|--------|-------------|
| 확대 / 축소 | 성장하거나 축소합니다 |
| 들어올리기 | 위로 떠오릅니다 |
| 회전 (시계 방향 / 반시계 방향) | 시계 방향 또는 반시계 방향으로 회전 |
| 밝기 / 페이드 | 밝기 또는 투명도를 변경 |
| 그림자 확대 | 그림자가 확장됩니다 |
| 그림자와 함께 들어올리기 | 그림자가 커지면서 떠오릅니다 |
| 펄스 확대 / 기울기 / 테두리 빛 | 특별한 효과 |

**지속 시간**, **타이밍**, **강도**(미묘, 일반, 강력)를 구성할 수 있습니다.

## 고급 설정

**고급** 탭은 세부적인 제어를 제공합니다:

### 가시성 규칙

조건에 따라 요소가 표시되거나 숨겨지는 시점을 제어합니다:

- **사용자 상태** — 로그인, 로그아웃, 신규 고객, 재방문 고객
- **기기** — 데스크탑, 태블릿, 모바일
- **시간** — 날짜 범위, 시간대, 요일
- **고객 그룹** — VIP, 도매 등
- **카트 금액** — 최소 또는 최대 카트 총액
- **지리** — 국가, 지역
- 그리고 20개 이상의 다른 규칙 유형

규칙은 AND/OR 논리로 결합하여 복잡한 타겟팅이 가능합니다.

### 사용자 정의 CSS

| 필드 | 목적 |
|-------|---------|
| **요소 ID** | 앵커 링크 또는 CSS 타겟팅을 위한 고유 ID |
| **사용자 정의 CSS 클래스** | 적용할 추가 클래스 |
| **사용자 정의 CSS 스타일** | 일회성 오버라이드를 위한 인라인 CSS |
| **데이터 속성** | 키-값 쌍으로 된 사용자 정의 data-* 속성 |
| **Z-인덱스** | 겹치는 요소의 스택 순서 |

## 게시 워크플로우

페이지는 전체 버전 기록을 가진 초안/게시 시스템을 사용합니다:

| 상태 | 의미 |
|--------|---------|
| **초안** | 작업 중 — 방문자에게 보이지 않음 |
| **게시됨** | 상점에 라이브로 표시됨 |
| **보관됨** | 사이트에서 제거되었지만 보존됨 |

### 작동 방식

1. 빌더에서 변경 사항을 만듭니다 — 이는 **초안**으로 저장됩니다
2. **초안 저장**을 클릭하여 게시하지 않고 저장합니다
3. **게시**를 클릭하여 현재 초안을 라이브로 만듭니다
4. 각 게시는 **버전 스냅샷**을 생성합니다
5. 버전 기록에서 이전 버전을 **복원**할 수 있습니다(도구 모음의 시계 아이콘)

이로 인해 자유롭게 실험할 수 있습니다 — 라이브 페이지는 명시적으로 게시할 때까지 변경되지 않습니다.

## 페이지 템플릿

시간을 절약하기 위해 템플릿을 사용합니다:

- **템플릿으로 저장** — 어떤 페이지의 디자인을 재사용 가능한 템플릿으로 저장합니다
- **템플릿으로 생성** — 기존 템플릿에서 새 페이지를 시작합니다
- **템플릿 카테고리** — 목적별로 템플릿을 정리합니다(랜딩 페이지, 소개, 제품 전시 등)

템플릿은 전체 페이지 구조를 포착합니다. 이는 모든 요소, 콘텐츠 및 스타일을 포함합니다.

## 반응형 디자인

도구 모음의 기기 미리보기 버튼을 사용하여 페이지가 다양한 화면 크기에서 어떻게 표시되는지 확인합니다:

- **데스크탑** — 전체 너비 레이아웃
- **태블릿** — 중간 화면
- **모바일** — 좁은 화면

요소는 컨테이너 설정에 따라 자동으로 재배치됩니다. 특정 기기에서 특정 요소를 표시하거나 숨기는 데 가시성 규칙을 사용할 수도 있습니다.

## 팁

- **컨테이너로 시작하세요** — 대부분의 레이아웃은 컨테이너로 시작하여 열과 구조를 만듭니다. 일반적인 배치를 위해 레이아웃 사전 설정을 사용합니다.
- **페이지 헤더에 히어로 섹션 사용** — 히어로 요소는 제목, 부제, 배경 이미지 및 CTA 버튼을 하나의 구성 요소로 제공합니다.
- **게시하기 전 미리보기** — 미리보기 버튼을 클릭하여 방문자가 보는 내용을 정확히 확인한 후 만족할 때 게시합니다.
- **가시성 규칙으로 개인화** — 로그인한 사용자와 로그아웃한 사용자에게 다른 콘텐츠를 표시하거나 특정 고객 그룹을 대상으로 합니다.
- **애니메이션은 미묘하게 유지하세요** — 페이지 섹션당 하나 또는 두 개의 진입 애니메이션이 전문적인 느낌을 줍니다. 너무 많은 애니메이션은 과도하게 느껴질 수 있습니다.
- **컨테이너를 이름 지어보세요** — 요소 ID 필드를 사용하여 컨테이너를 라벨링(예: "hero-section", "features")하여 복잡한 페이지에서 쉽게 찾을 수 있도록 합니다.
- **모든 기기에서 테스트하세요** — 게시하기 전에 기기 미리보기를 사용하여 데스크탑, 태블릿, 모바일에서 레이아웃을 확인합니다.
- **템플릿을 활용하세요** — 최고의 페이지 디자인을 템플릿으로 저장하여 향후 페이지 생성을 빠르게 합니다.
