---
title: 分类页面模板
---

分类页面模板控制客户浏览分类时产品的显示方式 —— 无论是经典的网格布局、水平列表、可滚动的轮播图，还是戏剧性的特色布局。您可以设置一个适用于整个商店的默认模板，然后在需要不同处理方式的任何单个分类中覆盖它。

## 在哪里找到设置

配置分类模板有两个地方：

- **全站默认** —— 导航到 **设计 > 页面模板**，滚动到 **分类** 部分。此处选择的模板适用于所有分类页面，除非被覆盖。
- **按分类覆盖** —— 导航到 **目录 > 分类**，打开任意分类，点击 **显示** 选项卡。在此处设置模板将仅覆盖该分类的全站默认设置。所有其他分类将继续使用默认设置。

## 六种模板

### 网格

网格模板以响应式卡片网格排列产品 —— 这是在线购物最熟悉的布局。产品以统一的卡片形式显示，包含图片、名称、价格和添加到购物车按钮。

**最适合：** 通用商店、时尚、家居用品、电子产品 —— 任何需要一致产品展示的商店。

| 选项 | 描述 |
|--------|-------------|
| **每行产品数** | 每行显示多少卡片（2、3、4 或 5） |
| **显示分类横幅** | 在页面顶部显示分类的横幅图片 |
| **显示子分类** | 在横幅下方显示可点击的子分类芯片 |
| **显示分类描述** | 显示分类的描述文本 |
| **卡片图片备用** | 当产品没有照片时显示占位图片 |

### 列表

列表模板将产品显示为水平行 —— 图片在一边，产品详情（名称、描述、价格）在另一边。这为每个产品提供了更多的空间，使客户能够一目了然地进行比较。

**最适合：** 有复杂产品且需要简短描述预览的商店，如工具、家电或 B2B 目录。

| 选项 | 描述 |
|--------|-------------|
| **显示产品图片** | 在每行中包含产品图片 |
| **图片位置** | 图片显示在左侧还是右侧 |
| **显示简短描述** | 在产品名称下方显示产品摘要 |
| **显示分类横幅** | 在页面顶部显示分类的横幅图片 |
| **显示子分类** | 在横幅下方显示可点击的子分类芯片 |
| **显示分类描述** | 显示分类的描述文本 |

### 轮播

轮播模板将产品分组为水平滚动行 —— 类似于流媒体平台上的“继续浏览”部分。客户可以通过滑动或点击箭头在不离开页面的情况下浏览产品。

**最适合：** 单个分类中包含大量产品的大型目录、礼品店、书店，或任何将浏览和发现作为优先事项的商店。

| 选项 | 描述 |
|--------|-------------|
| **每视图滑动数** | 在桌面端一次显示多少产品（3、4 或 5） |
| **行大小** | 每个轮播行显示多少产品，然后再在下方开始新的一行（4、6、8 或 12） |
| **显示导航箭头** | 在每行显示上一个/下一个箭头按钮 |
| **显示圆点** | 在每行下方显示进度条和页面计数器（例如“1 / 3”） |
| **自动播放** | 自动前进幻灯片 |
| **自动播放速度** | 每次自动前进之间的时间（以毫秒为单位） |
| **无限循环** | 在到达最后一张幻灯片后返回到第一张 |
| **显示分类横幅** | 在页面顶部显示分类的横幅图片 |
| **显示子分类** | 在横幅下方显示可点击的子分类芯片 |

### 马赛克

马赛克模板创建了一个类似 Pinterest 的错落网格，其中每个卡片根据其图片的自然高度进行调整。结果是一个多样化的、编辑风格的感觉，而不是一个统一的网格。

**最适合：** 摄影、艺术印刷品、手工艺品、室内设计，或任何产品视觉特征各异且希望以更动态方式展示它们的分类。

| 选项 | 描述 |
|--------|-------------|
| **列数** | 页面上跨列的数量（2、3 或 4） |
| **间隔大小** | 卡片之间的间距 —— 小、中或大 |
| **显示分类横幅** | 在页面顶部显示分类的横幅图片 |
| **显示子分类** | 在横幅下方显示可点击的子分类芯片 |
| **显示分类描述** | 显示分类的描述文本 |

### 特色

特色模板在页面顶部将少量产品作为大尺寸的特色卡片突出显示，其余产品则以支持网格的形式显示在下方。它能立即吸引客户对您优先产品关注。

**最适合：** 新品专区、畅销品页面、精选收藏或季节性分类，您希望突出显示特定产品。

| 选项 | 描述 |
|--------|-------------|
| **特色产品数量** | 在顶部获得特色待遇的产品数量（1、2 或 3） |
| **特色布局** | 显示特色卡片为全宽或作为并排布局 |
| **显示分类横幅** | 在页面顶部显示分类的横幅图片 |
| **显示子分类** | 在横幅下方显示可点击的子分类芯片 |
| **显示分类描述** | 显示分类的描述文本 |

### 折叠面板

折叠面板模板专为 **根分类页面** 设计 —— 显示所有顶级分类的页面。它将每个分类显示为一个交互式面板；悬停或点击面板会使其水平展开，而其他面板则收缩。在单个分类页面上，折叠面板模板会回退到标准网格。

**最适合：** 具有视觉冲击力的分类图片的商店，您希望分类页面本身具有沉浸式体验 —— 时尚品牌、生活方式商店和具有强烈编辑身份的商店。

| 选项 | 描述 |
|--------|-------------|
| **每行面板数** | 每个折叠面板行最多显示多少个分类，然后再换行到下方（4–8，默认 6） |
| **高度** | 折叠面板的高度 —— 小、中、大或特大 |
| **展开比例** | 悬停的活动面板比非活动面板宽多少倍（2–5 倍） |
| **过渡速度** | 动画速度（以毫秒为单位） |
| **移动设备布局** | 移动设备上分类的显示方式 —— 垂直堆叠、轮播或网格 |
| **显示产品数量** | 在每个分类面板上显示产品数量徽章 |

## 通用选项（所有模板）

每个模板共享一组选项，用于控制页面周围的装饰元素：

| 选项 | 描述 |
|--------|-------------|
| **显示面包屑导航** | 在分类上方显示“首页 > 分类 > 名称”导航 |
| **显示排序栏** | 显示“按...排序”下拉菜单，让客户重新排序结果 |
| **显示产品数量** | 显示总产品数量（例如“29 个产品”） |
| **显示帮助部分** | 在底部显示“找不到您要找的内容？”部分 |
| **默认排序** | 客户首次进入页面时使用的排序顺序 |
| **每页产品数** | 每页加载多少产品 |
| **分页样式** | 如何加载额外产品：分页页面、一个“加载更多”按钮，或无限滚动 |

## 快速比较

使用此表快速选择合适的模板：

| 模板 | 每行产品数 | 视觉特征 | 最佳产品类型 |
|----------|-----------------|------------------|--------------------|
| **网格** | 2–5 可配置 | 统一、整洁 | 一切 —— 安全的默认选项 |
| **列表** | 1（全宽行） | 宽敞、信息丰富 | 复杂或技术产品 |
| **轮播** | 3–5 同时可见 | 可浏览、动态 | 大型目录、发现 |
| **马赛克** | 2–4（错落） | 编辑、多样化 | 视觉/手工艺产品 |
| **特色** | 英雄 + 下方网格 | 促销 | 精选商品、新品 |
| **折叠面板** | N/A（分类） | 沉浸式、戏剧性 | 根分类页面 |

## 设置全站默认模板

1. 导航到 **设计 > 页面模板**。
2. 滚动到 **分类** 部分。
3. 从 **模板** 下拉菜单中选择您首选的模板。
4. 配置下拉菜单下方出现的模板选项。
5. 配置通用选项（排序栏、面包屑导航、分页等）。
6. 点击 **保存配置**。

现在，所选模板及其设置将适用于您商店中的每个分类页面。

## 为特定分类覆盖模板

您可以为任何单个分类设置不同的模板，而无需更改全站默认模板。

1. 导航到 **目录 > 分类**。
2. 点击您要自定义的分类。
3. 在编辑表单顶部点击 **显示** 选项卡。
4. 在 **模板** 下，选择 **自定义** 以启用按分类的设置。
5. 为该分类选择您想要的模板。
6. 根据需要配置模板选项。
7. 点击 **保存**。

只有这个分类将使用自定义模板。所有其他分类将继续使用全站默认模板。要将分类恢复为默认设置，请返回其显示选项卡并将模板设置为 **使用站点默认**。

## 示例：跨分类混合使用模板

这是一个实际例子，说明不同的模板如何服务于同一商店的不同部分：

| 分类 | 模板 | 原因 |
|----------|----------|--------|
| 所有分类（根页面） | 折叠面板 | 戏剧性的视觉入口 |
| 服装 | 网格（每行 4 个） | 快速、熟悉的浏览 |
| 促销 | 特色 | 立即突出显示顶级优惠 |
| 书籍 | 轮播 | 大型目录，易于浏览 |
| 手工礼物 | 马赛克 | 多样化的图片尺寸，手工艺感 |
| 工业工具 | 列表 | 客户需要查看描述 |

## 小贴士

- 以 **网格** 模板作为全站默认模板开始 —— 它几乎适用于所有产品类型，是购物者最熟悉的布局。从那里自定义个别分类。
- 请谨慎使用 **特色** 模板。它最适合当您能真正精选出 1–3 个突出产品时。如果所有产品看起来同样重要，特色展示的效果会减弱。
- **折叠面板** 模板只有在根分类页面上才能产生完整的视觉效果。如果您将其分配给产品分类，客户将看到一个普通的网格 —— 所以请将其保留给您的顶级分类列表。
- 仔细设置 **每页产品数**。24–36 个产品每页对网格和马赛克模板效果很好。对于轮播，行大小和每视图滑动数设置更直接地控制此体验。
- 在父分类上启用 **显示子分类**，以便客户可以直接跳转到更窄的选择，而无需使用排序或筛选工具。
- 在移动设备上测试您选择的模板。马赛克和轮播模板在小屏幕上看起来非常不同 —— 在折叠面板上使用 **移动设备布局** 选项，并确保马赛克列在手机上不会太窄。
- 当使用 **列表** 模板并启用 **显示简短描述** 时，请确保您的产品确实设置了简短描述。空的描述行看起来是无意的。
