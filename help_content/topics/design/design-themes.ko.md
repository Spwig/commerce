---
title: 디자인 및 테마
---

디자인 및 테마 시스템은 색상과 타이포그래피에서 헤더, 푸터, 페이지 레이아웃에 이르기까지 전체 판매점의 외형을 제어할 수 있습니다. **설정 > 디자인 및 테마**로 이동하여 디자인 대시보드를 엽니다.

![디자인 대시보드](/static/core/admin/img/help/design-themes/theme-dashboard.webp)

## 디자인 대시보드

대시보드는 가게의 디자인 상태에 대한 개요를 제공합니다:

- **활성 테마** — 현재 적용된 테마를 표시하며 미리보기와 빠른 접근 버튼이 포함됩니다.
- **디자인 통계** — 설치된 테마, 커스텀 헤더, 커스텀 푸터 및 메뉴 수
- **섹션 카드** — 테마, 헤더 빌더, 푸터 빌더, 메뉴, 공지사항으로 이동

## 테마

### 테마 검색

**테마** 섹션 카드를 클릭하여 설치된 모든 테마를 볼 수 있습니다. 각 테마 카드는 다음과 같은 정보를 표시합니다:
- 테마 이름 및 미리보기 이미지
- 작성자 및 버전
- 활성/비활성 상태

### 테마 활성화

1. 사용하고자 하는 테마의 **활성화**를 클릭합니다.
2. 테마가 즉시 판매점에 적용됩니다.
3. 한 번에 하나의 테마만 활성화할 수 있습니다.

### 테마 커스터마이징

각 테마는 일정한 **디자인 토큰** 세트를 지원합니다. 코드를 편집하지 않고 시각적 외형을 제어할 수 있는 구성 가능한 값입니다.

활성 테마의 **커스터마이징**을 클릭하여 토큰 편집기를 열 수 있습니다. 사용 가능한 토큰 카테고리에는 다음과 같은 항목이 포함됩니다:

| 카테고리 | 제어 항목 |
|----------|-----------------|
| **색상** | 기본, 보조, 강조 색상, 배경, 텍스트 색상 |
| **타이포그래피** | 글꼴 패밀리, 크기, 굵기, 줄 간격 |
| **간격** | 마진, 패딩, 요소 간 간격 |
| **테두리** | 테두리 너비, 반경, 색상 |
| **그림자** | 카드, 버튼, 모달의 박스 그림자 |
| **버튼** | 버튼 스타일, 크기, 호버 효과 |
| **레이아웃** | 컨테이너 너비, 그리드 간격, 브레이크포인트 |

저장하기 전에 실시간으로 변경 사항을 미리 볼 수 있습니다.

## 헤더 빌더

헤더 빌더는 드래그 앤 드롭 인터페이스를 사용하여 가게의 헤더를 설계할 수 있습니다.

### 헤더 생성

1. **디자인 > 헤더 빌더**로 이동합니다.
2. **헤더 생성**을 클릭하거나 기존 헤더를 편집합니다.
3. 빌더에는 세 개의 행이 있습니다: **상단 바**, **메인 헤더**, **하단 바**
4. 도구 상자에서 위젯을 끌어다 놓아 행의 anywhere에 추가할 수 있습니다.

### 사용 가능한 헤더 위젯

- **로고** — 설정 가능한 크기와 링크가 있는 가게 로고
- **네비게이션 메뉴** — 정의된 메뉴에서 드롭다운 메뉴
- **검색 바** — 즉시 결과가 표시되는 제품 검색
- **카트 아이콘** — 아이템 수량 배지가 있는 미니 카트
- **계정 아이콘** — 로그인/계정 드롭다운
- **언어 선택기** — 다국어 가게용 언어 전환기
- **통화 선택기** — 다통화 가게용 통화 전환기
- **사용자 정의 HTML** — 사용자 정의 콘텐츠 추가
- **소셜 아이콘** — 소셜 미디어 프로필 링크
- **공지 바** — 프로모션 메시지 및 제안

### 헤더 설정

각 헤더 템플릿에는 전역 설정이 있습니다:
- **고정 헤더** — 스크롤 시 헤더가 보입니다.
- **투명 모드** — 히로 이미지에 오버레이
- **모바일 브레이크포인트** — 모바일 레이아웃 전환 시점

## 푸터 빌더

푸터 빌더는 헤더 빌더와 유사하게 작동합니다.

### 푸터 생성

1. **디자인 > 푸터 빌더**로 이동합니다.
2. **푸터 생성**을 클릭하거나 기존 푸터를 편집합니다.
3. 빌더는 여러 열과 행을 지원합니다.
4. 위젯을 원하는 위치로 끌어다 놓습니다.

### 사용 가능한 푸터 위젯

- **네비게이션 메뉴** — 푸터 내의 네비게이션 링크
- **뉴스레터 등록** — 이메일 구독 양식
- **소셜 아이콘** — 소셜 미디어 링크
- **사용자 정의 HTML** — 사용자 정의 콘텐츠, 배지, 인증서
- **결제 아이콘** — 수락된 결제 방식 표시
- **저작권** — 동적 저작권 텍스트 및 연도
- **로고** — 푸터용 로고 변형

## 네비게이션 메뉴

메뉴는 헤더와 푸터의 네비게이션 링크를 정의합니다.

### 메뉴 생성

1. **디자인 > 메뉴**로 이동합니다.
2. **메뉴 추가**를 클릭합니다.
3. 메뉴에 이름을 지정합니다(예: "메인 네비게이션")
4. 메뉴 항목을 추가합니다:
   - **페이지 링크** — 페이지 빌더 페이지로의 링크
   - **카테고리 링크** — 제품 카테고리로의 링크
   - **사용자 정의 URL** — 외부 또는 내부 URL
   - **드롭다운** — 중첩된 하위 메뉴 항목
5. 항목을 재정렬하려면 끌어다 놓습니다.
6. 저장하고 헤더 또는 푸터 위젯에 메뉴를 할당합니다.

## 공지사항

판매점 상단에 표시되는 프로모션 배너를 생성합니다.

### 공지사항 생성

1. **디자인 > 공지사항**으로 이동하거나(또는 대시보드 카드 사용)
2. **공지사항 추가**를 클릭합니다.
3. 구성:
   - **메시지** — 공지사항 텍스트(번역 지원)
   - **링크** — 클릭 시 선택적 URL
   - **스타일** — 배경색, 텍스트 색, 아이콘
   - **일정** — 시작 및 종료 날짜
   - **닫기 가능** — 고객이 닫을 수 있는지 여부
4. 저장 및 활성화

여러 공지사항이 동시에 활성화될 수 있으며 자동으로 순환합니다.

## 팁

- 헤더 및 푸터를 구축하기 전에 활성 테마의 커스터마이저를 사용하여 브랜드 색상과 일치시킵니다.
- 헤더 및 푸터 빌더의 **미리보기** 기능을 사용하여 게시하기 전에 변경 사항을 확인합니다.
- 헤더를 데스크탑 및 모바일용으로 별도로 생성하려면 매우 다른 레이아웃이 필요한 경우입니다.
- 네비게이션을 간단하게 유지하세요 — 사용성에 이상적인 상위 레벨 메뉴 항목은 5-7개입니다.
- 시간 제한이 있는 프로모션보다는 영구 메시지보다 공지사항을 사용하세요.
- 테마 토큰 편집기는 실시간 미리보기를 지원합니다 — 자유롭게 실험하고 만족할 때 저장하세요.
