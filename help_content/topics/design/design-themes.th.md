---
title: การออกแบบและธีม
---

ระบบการออกแบบและธีมช่วยให้คุณควบคุมรูปลักษณ์และรูปแบบของร้านค้าออนไลน์ทั้งหมดของคุณ — ตั้งแต่สีและฟอนต์ไปจนถึงหัวข้อ หัวข้อท้าย และการจัดวางหน้า เข้าไปที่ **การตั้งค่า > การออกแบบและธีม** เพื่อเปิดแดชบอร์ดการออกแบบ

![แดชบอร์ดการออกแบบ](/static/core/admin/img/help/design-themes/theme-dashboard.webp)

## แดชบอร์ดการออกแบบ

แดชบอร์ดให้คุณมีภาพรวมเกี่ยวกับสถานะการออกแบบของร้านค้า:

- **ธีมที่ใช้งานอยู่** — แสดงธีมที่กำลังใช้งานอยู่พร้อมกับภาพตัวอย่างและปุ่มเข้าถึงอย่างรวดเร็ว
- **สถิติการออกแบบ** — จำนวนธีมที่ติดตั้ง หัวข้อท้ายแบบกำหนดเอง หัวข้อท้ายแบบกำหนดเอง และเมนู
- **การ์ดส่วน** — ไปยังธีม ตัวสร้างหัวข้อท้าย ตัวสร้างหัวข้อท้าย เมนู หรือประกาศ

## ธีม

### การเรียกดูธีม

คลิกการ์ดส่วน **ธีม** เพื่อดูธีมที่ติดตั้งทั้งหมด การ์ดธีมแต่ละใบแสดง:
- ชื่อธีมและภาพตัวอย่าง
- ผู้สร้างและเวอร์ชัน
- สถานะการใช้งาน/ไม่ใช้งาน

### การเปิดใช้งานธีม

1. คลิก **เปิดใช้งาน** บนธีมที่คุณต้องการใช้
2. ธีมจะถูกนำไปใช้ทันทีกับร้านค้าออนไลน์ของคุณ
3. มีธีมเพียงหนึ่งธีมเท่านั้นที่สามารถเปิดใช้งานได้ในเวลาเดียวกัน

### การปรับแต่งธีม

ธีมแต่ละธีมรองรับชุดของ **โทเคนการออกแบบ** — ค่าที่ปรับแต่งได้ที่ควบคุมการปรากฏตัวโดยไม่ต้องแก้ไขโค้ด

คลิก **ปรับแต่ง** บนธีมที่ใช้งานอยู่ของคุณเพื่อเข้าถึงตัวแก้ไขโทเคน หมวดหมู่โทเคนที่มีอยู่รวมถึง:

| หมวดหมู่ | สิ่งที่ควบคุม |
|----------|-----------------|
| **สี** | สีหลัก สีรอง สีเน้น สีพื้นหลัง สีข้อความ |
| **ฟอนต์** | ชื่อฟอนต์ ขนาด น้ำหนัก ความสูงของบรรทัด |
| **ระยะห่าง** | ระยะขอบ ระยะห่าง ช่องว่างระหว่างองค์ประกอบ |
| **เส้นขอบ** | ความกว้างของเส้นขอบ รัศมี สี |
| **เงา** | เงาของกล่องสำหรับการ์ด ปุ่ม โมดัล |
| **ปุ่ม** | สไตล์ ขนาด ผลลัพธ์เมื่อชี้เมาส์ |
| **การจัดวาง** | ความกว้างของคอนเทนเนอร์ ช่องว่างตาราง จุดเปลี่ยน |

การเปลี่ยนแปลงจะแสดงผลแบบเรียลไทม์ก่อนที่คุณจะบันทึก

## ตัวสร้างหัวข้อ

ตัวสร้างหัวข้อช่วยให้คุณออกแบบหัวข้อของร้านค้าโดยใช้อินเทอร์เฟซแบบลากและวาง

### การสร้างหัวข้อ

1. ไปที่ **การออกแบบ > ตัวสร้างหัวข้อ**
2. คลิก **สร้างหัวข้อ** หรือแก้ไขหัวข้อที่มีอยู่
3. ตัวสร้างมี 3 แถว: **แถบด้านบน** **หัวข้อหลัก** และ **แถบด้านล่าง**
4. ลากวิจิตรจากเครื่องมือไปยังแถวใดก็ได้

### วิจิตรที่มีอยู่ในหัวข้อ

- **โลโก้** — โลโก้ร้านค้าของคุณพร้อมขนาดและลิงก์ที่ปรับแต่งได้
- **เมนูนำทาง** — เมนูแบบเลื่อนลงจากเมนูที่กำหนดไว้ของคุณ
- **แถบค้นหา** — การค้นหาสินค้าพร้อมผลลัพธ์แบบเรียลไทม์
- **ไอคอนตะกร้า** — ตะกร้าขนาดเล็กพร้อมป้ายจำนวนสินค้า
- **ไอคอนบัญชี** — ปุ่มล็อกอิน/บัญชี
- **ตัวเลือกภาษา** — ตัวสลับภาษาสำหรับร้านค้าหลายภาษา
- **ตัวเลือกสกุลเงิน** — ตัวสลับสกุลเงินสำหรับร้านค้าหลายสกุลเงิน
- **HTML แบบกำหนดเอง** — เพิ่มเนื้อหาแบบกำหนดเองใดๆ
- **ไอคอนโซเชียล** — ลิงก์ไปยังโปรไฟล์โซเชียลมีเดียของคุณ
- **แถบประกาศ** — ข้อความโปรโมชันและข้อเสนอ

### การตั้งค่าหัวข้อ

แต่ละเทมเพลตหัวข้อมีการตั้งค่าทั่วไป:
- **หัวข้อแบบติดอยู่** — หัวข้อจะยังคงมองเห็นได้เมื่อเลื่อนหน้า
- **โหมดโปร่งใส** — แสดงเป็นภาพทับบนภาพประกอบ
- **จุดเปลี่ยนแบบมือถือ** — เมื่อจะเปลี่ยนไปใช้การจัดวางแบบมือถือ

## ตัวสร้างหัวข้อท้าย

ตัวสร้างหัวข้อท้ายทำงานคล้ายกับตัวสร้างหัวข้อ

### การสร้างหัวข้อท้าย

1. ไปที่ **การออกแบบ > ตัวสร้างหัวข้อท้าย**
2. คลิก **สร้างหัวข้อท้าย** หรือแก้ไขหัวข้อท้ายที่มีอยู่
3. ตัวสร้างรองรับคอลัมน์และแถวหลายตัว
4. ลากวิจิตรไปยังตำแหน่งที่ต้องการ

### วิจิตรที่มีอยู่ในหัวข้อท้าย

- **เมนูนำทาง** — ลิงก์การนำทางในหัวข้อท้าย
- **แบบฟอร์มสมัครข่าวสาร** — แบบฟอร์มสมัครอีเมล
- **ไอคอนโซเชียล** — ลิงก์โซเชียลมีเดีย
- **HTML แบบกำหนดเอง** — เนื้อหาแบบกำหนดเอง ป้ายชื่อ การรับรอง
- **ไอคอนการชำระเงิน** — แสดงวิธีการชำระเงินที่ยอมรับ
- **ลิขสิทธิ์** — ข้อความลิขสิทธิ์ที่ปรับแต่งได้พร้อมปี
- **โลโก้** — รูปแบบโลโก้ในหัวข้อท้าย

## เมนูนำทาง

เมนูกำหนดลิงก์การนำทางในหัวข้อและหัวข้อท้ายของคุณ

### การสร้างเมนู

1. ไปที่ **การออกแบบ > เมนู**
2. คลิก **เพิ่มเมนู**
3. ให้เมนูมีชื่อ (เช่น "การนำทางหลัก")
4. เพิ่มรายการเมนู:
   - **ลิงก์หน้า** — ลิงก์ไปยังหน้าของตัวสร้างหน้า
   - **ลิงก์หมวดหมู่** — ลิงก์ไปยังหมวดหมู่สินค้า
   - **ลิงก์แบบกำหนดเอง** — ลิงก์ภายนอกหรือภายในใดๆ ก็ได้
   - **เมนูย่อย** — รายการย่อยที่ซ้อนทับกัน
5. ลากรายการเพื่อจัดลำดับใหม่
6. บันทึกและกำหนดเมนูให้กับวิจิตรในหัวข้อหรือหัวข้อท้าย

## การประกาศ

สร้างแบนเนอร์โปรโมชันที่ปรากฏอยู่ด้านบนของร้านค้าออนไลน์ของคุณ

### การสร้างการประกาศ

1. ไปที่ **การออกแบบ > การประกาศ** (หรือใช้การ์ดแดชบอร์ด)
2. คลิก **เพิ่มการประกาศ**
3. ตั้งค่า:
   - **ข้อความ** — ข้อความการประกาศ (รองรับการแปล)
   - **ลิงก์** — ลิงก์ URL ที่เป็นทางเลือกเมื่อคลิก
   - **สไตล์** — สีพื้นหลัง สีข้อความ ไอคอน
   - **ตารางเวลา** — วันที่เริ่มและสิ้นสุด
   - **ปิดได้** — ลูกค้าสามารถปิดได้หรือไม่
4. บันทึกและเปิดใช้งาน

การประกาศหลายรายการสามารถเปิดใช้งานพร้อมกันได้ — พวกเขาจะหมุนเวียนโดยอัตโนมัติ

## เคล็ดลับ

- เริ่มต้นด้วยตัวปรับแต่งธีมที่ใช้งานอยู่เพื่อให้สีตรงกับแบรนด์ของคุณก่อนสร้างหัวข้อและหัวข้อท้าย
- ใช้คุณสมบัติ **แสดงตัวอย่าง** ในตัวสร้างหัวข้อและหัวข้อท้ายเพื่อดูการเปลี่ยนแปลงก่อนเผยแพร่
- สร้างหัวข้อแยกต่างหากสำหรับเดสก์ท็อปและมือถือหากคุณต้องการการจัดวางที่แตกต่างกันอย่างมาก
- รักษาการนำทางให้เรียบง่าย — รายการเมนูระดับสูง 5-7 รายการเป็นตัวเลือกที่ดีที่สุดสำหรับการใช้งาน
- ใช้การประกาศสำหรับโปรโมชันที่มีเวลาจำกัดแทนข้อความถาวร
- ตัวแก้ไขโทเคนธีมรองรับการดูแบบเรียลไทม์ — ทดลองได้อย่างอิสระและบันทึกเมื่อคุณพอใจแล้ว
