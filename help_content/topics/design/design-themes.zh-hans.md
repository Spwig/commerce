title: 设计与主题

设计与主题系统可让您控制整个商店前端的外观和感觉，从颜色和字体到页眉、页脚和页面布局。导航到 **设置 > 设计与主题** 以打开设计仪表板。

![设计仪表板](/static/core/admin/img/help/design-themes/theme-dashboard.webp)

## 设计仪表板

仪表板为您提供商店设计状态的概览：

- **当前主题** — 显示当前应用的主题，带有预览和快速访问按钮
- **设计统计** — 已安装主题、自定义页眉、自定义页脚和菜单的数量
- **部分卡片** — 跳转到主题、页眉构建器、页脚构建器、菜单或公告

## 主题

### 浏览主题

点击 **主题** 部分卡片查看所有已安装的主题。每个主题卡片显示：
- 主题名称和预览图片
- 作者和版本
- 启用/禁用状态

### 启用主题

1. 点击您要使用主题的 **启用** 按钮
2. 主题会立即应用到您的商店前端
3. 一次只能启用一个主题

### 主题自定义

每个主题支持一组 **设计标记** — 可配置的值，可在不编辑代码的情况下控制视觉外观。

点击您当前主题的 **自定义** 以访问标记编辑器。可用的标记类别包括：

| 类别 | 控制内容 |
|----------|-----------------|
| **颜色** | 主色、次色、强调色、背景色、文字颜色 |
| **字体** | 字体系列、大小、粗细、行高 |
| **间距** | 外边距、内边距、元素之间的间距 |
| **边框** | 边框宽度、圆角、颜色 |
| **阴影** | 卡片、按钮、模态框的阴影 |
| **按钮** | 按钮样式、大小、悬停效果 |
| **布局** | 容器宽度、网格间距、断点 |

在保存之前，更改会实时预览。

## 页眉构建器

页眉构建器允许您使用拖放界面设计商店的页眉。

### 创建页眉

1. 导航到 **设计 > 页眉构建器**
2. 点击 **创建页眉** 或编辑现有页眉
3. 构建器有三行：**顶部栏**、**主页眉** 和 **底部栏**
4. 将工具箱中的小部件拖到任何一行

### 可用的页眉小部件

- **Logo** — 可配置大小和链接的商店标志
- **导航菜单** — 来自您定义的菜单的下拉菜单
- **搜索栏** — 产品搜索，支持即时结果
- **购物车图标** — 带有商品数量徽章的小型购物车
- **账户图标** — 登录/账户下拉菜单
- **语言选择器** — 多语言商店的语言切换器
- **货币选择器** — 多货币商店的货币切换器
- **自定义 HTML** — 添加任何自定义内容
- **社交媒体图标** — 链接到您的社交媒体资料
- **公告栏** — 促销信息和优惠

### 页眉设置

每个页眉模板有全局设置：
- **固定页眉** — 滚动时页眉保持可见
- **透明模式** — 英雄图片上的覆盖层
- **移动断点** — 切换到移动布局的时间点

## 页脚构建器

页脚构建器的工作方式与页眉构建器类似。

### 创建页脚

1. 导航到 **设计 > 页脚构建器**
2. 点击 **创建页脚** 或编辑现有页脚
3. 构建器支持多列和多行
4. 将小部件拖到适当位置

### 可用的页脚小部件

- **导航菜单** — 页脚导航链接
- **电子邮件订阅** — 电子邮件订阅表单
- **社交媒体图标** — 社交媒体链接
- **自定义 HTML** — 自定义内容、徽章、认证
- **支付图标** — 显示接受的支付方式
- **版权信息** — 带年份的动态版权文本
- **Logo** — 页脚标志变体

## 导航菜单

菜单定义了您页眉和页脚中的导航链接。

### 创建菜单

1. 导航到 **设计 > 菜单**
2. 点击 **添加菜单**
3. 为菜单命名（例如，"主导航"）
4. 添加菜单项：
   - **页面链接** — 链接到页面构建器页面
   - **分类链接** — 链接到产品分类
   - **自定义 URL** — 任何外部或内部 URL
   - **下拉菜单** — 嵌套的子菜单项
5. 拖动项目以重新排序
6. 保存并分配菜单到页眉或页脚小部件

## 公告

创建促销横幅，显示在商店前端顶部。

### 创建公告

1. 导航到 **设计 > 公告**（或使用仪表板卡片）
2. 点击 **添加公告**
3. 配置：
   - **消息** — 公告文本（支持翻译）
   - **链接** — 点击时的可选 URL
   - **样式** — 背景色、文字颜色、图标
   - **时间安排** — 开始和结束日期
   - **可关闭** — 客户是否可以关闭它
4. 保存并激活

可以同时激活多个公告 — 它们会自动轮换。

## 小贴士

- 在构建页眉和页脚之前，先使用当前主题的自定义器匹配您的品牌颜色。
- 在页眉和页脚构建器中使用 **预览** 功能，在发布前查看更改。
- 如果需要非常不同的布局，请为桌面和移动设备创建单独的页眉。
- 保持导航简单 — 5-7 个顶级菜单项对可用性最佳。
- 使用公告进行时间敏感的促销，而不是永久性消息。
- 主题标记编辑器支持实时预览 — 自由实验并在满意时保存。
