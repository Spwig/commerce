---
title: 配送ルール
---

配送ルールは、カートの内容、顧客の属性、配達地域に基づいて配送方法に条件付きのコスト調整を適用します。$50以上で無料配送を自動的に提供したり、遠隔地域に追加料金を課したり、VIP顧客に配送を割引したりできます。ルールは、優先順位に基づいて実行され（高い優先順位から）、さらなる処理を防ぐためにオプションの停止フラグを使用します。各ルールは、複数の条件（カートの価値、重量、地域、商品、顧客グループ）を評価し、すべての条件が一致したときに6種類の調整のいずれかを実行します。

カートの価値が静的な配送方法のレートだけでなく、注文の文脈に基づいて動的に変化する配送コストが必要な場合に配送ルールを使用してください。

## 配送ルールの種類

配送ルールは6種類のコスト調整を適用します。

### パーセンテージ割引

**何をするか**: 配送コストをパーセンテージで減らします（例: 25％割引）。

**式**: `new_cost = base_cost × (1 - percent/100)`

**例**:
```
基本コスト: $20
割引: 25％
結果: $15
```

**使用例**:
- VIP顧客割引（すべての配送に20％割引）
- 季節限定プロモーション（12月の配送に15％割引）
- 批量注文割引（5つ以上の商品に10％割引）

---

### 固定割引

**何をするか**: 配送コストから固定金額を減算します。

**式**: `new_cost = base_cost - amount`（最小$0）

**例**:
```
基本コスト: $15
割引: $5
結果: $10
```

**使用例**:
- 初回顧客ボーナス（初回注文の配送に$5割引）
- ニュースレター登録報酬（配送に$3割引）
- ロイヤルティプログラムの利点（月に配送に$10割引）

---

### 固定コスト

**何をするか**: 配送コストを特定の金額に上書きします。

**式**: `new_cost = fixed_amount`

**例**:
```
基本コスト: $25
設定: $9.99
結果: $9.99
```

**使用例**:
- フラッシュセール（今日はすべての注文に$5の固定配送）
- カテゴリ固有の配送（本は常に$3.99の配送）
- 時間限定プロモーション（今週の配送は$9.99まで）

---

### 無料配送

**何をするか**: 配送コストを$0に設定します。

**式**: `new_cost = $0`

**例**:
```
基本コスト: $18
ルールが適用される
結果: $0
```

**使用例**:
- $50以上で無料配送
- 特定の商品（プロモーション商品）に無料配送
- VIP顧客に無料配送
- 3つ以上の商品がある注文に無料配送

---

### 追加料金（固定）

**何をするか**: 配送コストに固定金額を追加します。

**式**: `new_cost = base_cost + amount`

**例**:
```
基本コスト: $12
追加料金: $5
結果: $17
```

**使用例**:
- 遠隔地域の配達料
- 大型商品の取り扱い料
- 土曜日の配達追加料
- 破損品の梱包料

---

### 追加料金（パーセンテージ）

**何をするか**: 配送コストをパーセンテージで増加させます。

**式**: `new_cost = base_cost × (1 + percent/100)`

**例**:
```
基本コスト: $20
追加料金: 15％
結果: $23
```

**使用例**:
- 高峰期の追加料金（祝日中に20％）
- エクスプレス配送のプレミアム料（50％の追加料金）
- 燃料追加料（現在のレートに基づいて変動）

---

## ルールの条件

ルールは、**すべての条件が満たされる**場合にのみ適用されます。

### 時間有効期限

- **開始日**: ルールはこの日以降にのみ有効
- **終了日**: ルールはこの日以前にのみ有効
- **使用例**: 季節限定プロモーション、限定時間のオファー

**例**: ブラックフライデーの週末に限り無料配送
```
開始: 2026-11-27 00:00
終了: 2026-11-30 23:59
```

---

### カート価格範囲

- **最小カート価格**: カートのサブトータルはamount以上でなければなりません
- **最大カート価格**: カートのサブトータルはamount以下でなければなりません
- **使用例**: 無料配送のしきい値、段階的な割引

**例**: $50～$200の注文に無料配送
```
最小: $50
最大: $200
```

---

### カート重量範囲

- **最小重量**: カートの総重量はamount以上でなければなりません
- **最大重量**: カートの総重量はamount以下でなければなりません
- **使用例**: 軽量配送の割引、重い商品の追加料

**例**: 20kg以上の注文に$5の追加料
```
最小重量: 20kg
最大重量: null（無制限）
```

---

### アイテム数範囲

- **最小アイテム数**: カートにamount以上のアイテム数が必要
- **最大アイテム数**: カートにamount以下のアイテム数が必要
- **使用例**: 批量注文の割引、単品の料金

**例**: 5つ以上のアイテムに無料配送
```
最小アイテム: 5
最大アイテム: null
```

---
