---
title: 税务配置
---

税率定义了基于客户位置和产品类型的结账时征收的销售税、增值税和其他消费税——配置国家/州/城市级别的税率，并可选产品类别豁免。Spwig支持复合税（税加税）、基于优先级的税率选择以及税预设组，以便快速设置区域税务系统（如欧盟增值税、美国销售税）。税率可以豁免特定产品类型（食品、书籍、数字商品）或类别，以符合当地税法。

使用税务配置确保在销售辖区的税务征收要求方面符合法律。

## 税率配置

每个税率定义：

**地理范围**：
- 国家（必填）
- 州/省（可选）
- 城市（可选）
- 邮政编码模式（可选，正则表达式）

**税率详情**：
- **税率**：百分比（例如，8.5%）
- **名称**：显示名称（例如，"加利福尼亚州销售税"）
- **优先级**：当多个税率匹配时，优先级较高的胜出
- **启用**：无需删除即可切换

**豁免**：
- **豁免产品类型**：数字商品、实物商品、服务
- **豁免类别**：特定产品类别（食品、书籍、医疗）

**复合税**：
- **是否复合**：在此税率上叠加之前的税（税加税）
- 示例：魁北克省销售税叠加在联邦销售税上

## 常见税务场景

### 美国销售税（州级）

```
名称：加利福尼亚州销售税
国家：美国
州：CA
税率：7.25%
优先级：50
```

### 欧盟增值税（国家级）

```
名称：英国增值税
国家：GB
税率：20%
优先级：50

名称：德国增值税
国家：DE
税率：19%
优先级：50
```

### 加拿大GST/PST（复合）

```
税率1：联邦GST
国家：CA
税率：5%
优先级：100
是否复合：否

税率2：魁北克省PST
国家：CA
州：QC
税率：9.975%
优先级：50
是否复合：是（适用于小计加GST）
```

### 城市级税

```
名称：西雅图销售税
国家：美国
州：WA
城市：西雅图
税率：10.1%
优先级：100
```

## 税务豁免

### 产品类型豁免

豁免整个产品类型：

- **数字商品**：软件、电子书、音乐
- **实物商品**：有形产品
- **服务**：咨询、安装

示例：欧盟增值税对消费者不适用于数字商品（在某些情况下）

### 类别豁免

豁免特定产品类别：

- 食品和杂货（通常豁免或减税）
- 书籍和教育材料
- 医疗用品和药品
- 服装（某些司法管辖区）

配置：
```
名称：加利福尼亚州销售税
税率：7.25%
豁免类别：["食品和饮料","处方药"]
```

## 税务预设组

快速加载常见的税务配置：

**美国销售税预设**：
- 所有50个州+华盛顿特区
- 州级税率
- 税率更改时自动更新

**欧盟增值税预设**：
- 所有27个欧盟成员国
- 标准增值税税率
- B2B的反向征税逻辑

**使用预设**：
1. 设置 > 购物车 > 税务预设
2. 选择预设组（例如，"2026年美国销售税"）
3. 点击"加载预设"
4. 税率会自动导入
5. 根据需要进行自定义

## 优先级解析

当多个税率匹配时，优先级最高的胜出：

示例：
```
西雅图，华盛顿州的客户：

税率A：美国联邦（优先级1） - 0%
税率B：华盛顿州（优先级50） - 6.5%
税率C：西雅图市（优先级100） - 3.6%

结果：西雅图税率（总计10.1%）适用
```

## 税务显示选项

在设置 > 购物车 > 税务设置中配置：

- **价格包含税**：显示包含税的价格（欧盟风格）
- **单独显示税**：将税显示为单独项目（美国风格）
- **四舍五入税**：按商品或按订单
- **税标签**：自定义标签（"增值税","销售税","GST"）

## 测试税务配置

上线前：

1. 从不同司法管辖区创建测试订单
2. 验证是否应用了正确的税率
3. 检查豁免是否适用于排除的类别
4. 测试复合税计算
5. 查看发票上的税务明细项目

## 合规说明

- **美国**：Nexus规则要求在有实体存在或经济联系的州征收税
- **欧盟**：注册增值税的企业必须从欧盟客户处征收增值税
- **加拿大**：GST/HST/PST因省份而异
- **咨询税务专家**：税法经常变化，请核实当前要求

---

## 小贴士

- **使用税务预设** - 比手动输入更快，且会自动更新
- **监控Nexus阈值** - 按州跟踪销售以符合美国经济Nexus要求
- **正确设置优先级** - 城市 > 州 > 国家
- **测试复合税** - 验证计算是否符合预期金额
- **每年更新** - 税率会变化，每年一月审查一次
- **记录豁免** - 保留豁免类别的记录，说明原因
- **使用描述性名称** - "2026年加利福尼亚州销售税"比"税1"更好
- **默认启用税务** - 比忘记应用税务更安全
