---
title: กฎเกณฑ์การจัดส่ง
---

กฎเกณฑ์การจัดส่งใช้ปรับราคาการจัดส่งแบบมีเงื่อนไขตามเนื้อหาตะกร้าสินค้า คุณสมบัติของลูกค้า และโซนการจัดส่ง โดยอัตโนมัติเสนอการจัดส่งฟรีเมื่อสั่งซื้อเกิน $50 เพิ่มค่าธรรมเนียมสำหรับพื้นที่ห่างไกล หรือลดราคาการจัดส่งสำหรับลูกค้า VIP กฎเกณฑ์ทำงานตามลำดับความสำคัญ (กฎที่มีความสำคัญสูงสุดจะถูกประมวลผลก่อน) และมีตัวเลือกการปิดการประมวลผลเพิ่มเติม แต่ละกฎจะประเมินเงื่อนไขหลายประการ (ค่ารวมตะกร้า น้ำหนัก โซน สินค้า กลุ่มลูกค้า) และดำเนินการหนึ่งในประเภทการปรับราคา 6 ประเภทเมื่อเงื่อนไขทั้งหมดตรงกัน

ใช้กฎเกณฑ์การจัดส่งเมื่อคุณต้องการค่าใช้จ่ายการจัดส่งที่เปลี่ยนแปลงตามบริบทการสั่งซื้อ ไม่ใช่เพียงอัตราที่กำหนดตายตัวจากวิธีการจัดส่ง

## ประเภทกฎเกณฑ์การจัดส่ง

กฎเกณฑ์การจัดส่งใช้ปรับราคา 6 ประเภท:

### ส่วนลดเปอร์เซ็นต์

**สิ่งที่ทำ**: ลดค่าใช้จ่ายการจัดส่งเป็นเปอร์เซ็นต์ (เช่น ลด 25%)

**สูตร**: `new_cost = base_cost × (1 - percent/100)`

**ตัวอย่าง**:
```
ค่าใช้จ่ายพื้นฐาน: $20
ส่วนลด: 25%
ผลลัพธ์: $15
```

**กรณีการใช้งาน**:
- ส่วนลดลูกค้า VIP (ลด 20% สำหรับการจัดส่งทั้งหมด)
- ส่วนลดช่วงเทศกาล (ลด 15% สำหรับการจัดส่งในเดือนธันวาคม)
- ส่วนลดการสั่งซื้อจำนวนมาก (ลด 10% สำหรับการสั่งซื้อ 5 ชิ้นขึ้นไป)

---

### ส่วนลดคงที่

**สิ่งที่ทำ**: ลบจำนวนคงที่ออกจากค่าใช้จ่ายการจัดส่ง

**สูตร**: `new_cost = base_cost - amount` (ขั้นต่ำ $0)

**ตัวอย่าง**:
```
ค่าใช้จ่ายพื้นฐาน: $15
ส่วนลด: $5
ผลลัพธ์: $10
```

**กรณีการใช้งาน**:
- โบนัสลูกค้าใหม่ (ลด $5 สำหรับการสั่งซื้อแรก)
- รางวัลการสมัครสมาชิกบล็อก (ลด $3 สำหรับการจัดส่ง)
- ประโยชน์จากโปรแกรมสะสมแต้ม (ลด $10 สำหรับการจัดส่งต่อเดือน)

---

### ตั้งค่าใช้จ่าย

**สิ่งที่ทำ**: กำหนดค่าใช้จ่ายการจัดส่งเป็นจำนวนเฉพาะ

**สูตร**: `new_cost = fixed_amount`

**ตัวอย่าง**:
```
ค่าใช้จ่ายพื้นฐาน: $25
ตั้งเป็น: $9.99
ผลลัพธ์: $9.99
```

**กรณีการใช้งาน**:
- โปรโมชันแฟลช (ค่าใช้จ่ายคงที่ $5 สำหรับการสั่งซื้อทุกใบในวันนี้)
- การจัดส่งเฉพาะหมวดหมู่ (หนังสือมีค่าใช้จ่าย $3.99 ทุกครั้ง)
- โปรโมชันตามเวลา (ค่าใช้จ่ายถูกจำกัดไว้ที่ $9.99 สำหรับสัปดาห์นี้)

---

### การจัดส่งฟรี

**สิ่งที่ทำ**: ตั้งค่าใช้จ่ายการจัดส่งเป็น $0

**สูตร**: `new_cost = $0`

**ตัวอย่าง**:
```
ค่าใช้จ่ายพื้นฐาน: $18
กฎเกณฑ์ใช้ได้
ผลลัพธ์: $0
```

**กรณีการใช้งาน**:
- การจัดส่งฟรีเมื่อสั่งซื้อเกิน $50
- การจัดส่งฟรีสำหรับสินค้าเฉพาะ (สินค้าโปรโมชัน)
- การจัดส่งฟรีสำหรับลูกค้า VIP
- การจัดส่งฟรีสำหรับการสั่งซื้อที่มี 3 ชิ้นขึ้นไป

---

### ค่าธรรมเนียมคงที่ (Fixed Surcharge)

**สิ่งที่ทำ**: เพิ่มจำนวนคงที่เข้ากับค่าใช้จ่ายการจัดส่ง

**สูตร**: `new_cost = base_cost + amount`

**ตัวอย่าง**:
```
ค่าใช้จ่ายพื้นฐาน: $12
ค่าธรรมเนียม: $5
ผลลัพธ์: $17
```

**กรณีการใช้งาน**:
- ค่าธรรมเนียมจัดส่งพื้นที่ห่างไกล
- ค่าธรรมเนียมจัดการสินค้าขนาดใหญ่
- ค่าธรรมเนียมจัดส่งวันเสาร์
- ค่าธรรมเนียมบรรจุภัณฑ์สินค้าที่เปราะบาง

---

### ค่าธรรมเนียมเปอร์เซ็นต์ (Percentage Surcharge)

**สิ่งที่ทำ**: เพิ่มค่าใช้จ่ายการจัดส่งเป็นเปอร์เซ็นต์

**สูตร**: `new_cost = base_cost × (1 + percent/100)`

**ตัวอย่าง**:
```
ค่าใช้จ่ายพื้นฐาน: $20
ค่าธรรมเนียม: 15%
ผลลัพธ์: $23
```

**กรณีการใช้งาน**:
- ค่าธรรมเนียมช่วงฤดูกาล (เพิ่ม 20% ในช่วงเทศกาล)
- ค่าธรรมเนียมการจัดส่งแบบเร่งด่วน (เพิ่ม 50%)
- ค่าธรรมเนียมเชื้อเพลิง (เปลี่ยนแปลงตามอัตราที่ใช้ในปัจจุบัน)

## เงื่อนไขกฎเกณฑ์

กฎเกณฑ์จะประเมิน **ทุกเงื่อนไขต้องผ่าน** สำหรับกฎเกณฑ์ที่จะใช้ได้:

### ช่วงเวลาที่ใช้ได้

- **วันเริ่มต้น**: กฎเกณฑ์จะใช้ได้หลังจากวันนี้เท่านั้น
- **วันสิ้นสุด**: กฎเกณฑ์จะใช้ได้ก่อนวันนี้เท่านั้น
- **กรณีการใช้งาน**: โปรโมชันตามฤดูกาล ข้อเสนอจำกัดเวลา

**ตัวอย่าง**: การจัดส่งฟรีเฉพาะช่วงวัน Black Friday
```
เริ่มต้น: 2026-11-27 00:00
สิ้นสุด: 2026-11-30 23:59
```

---

### ช่วงค่ารวมตะกร้า

- **ค่ารวมตะกร้าขั้นต่ำ**: ค่ารวมตะกร้าต้อง ≥ จำนวน
- **ค่ารวมตะกร้าสูงสุด**: ค่ารวมตะกร้าต้อง ≤ จำนวน
- **กรณีการใช้งาน**: ข้อจำกัดการจัดส่งฟรี ข้อเสนอตามระดับ

**ตัวอย่าง**: การจัดส่งฟรีสำหรับการสั่งซื้อ $50-$200
```
ขั้นต่ำ: $50
สูงสุด: $200
```

---

### ช่วงน้ำหนักตะกร้า

- **น้ำหนักขั้นต่ำ**: น้ำหนักตะกร้ารวมต้อง ≥ จำนวน
- **น้ำหนักสูงสุด**: น้ำหนักตะกร้ารวมต้อง ≤ จำนวน
- **กรณีการใช้งาน**: ส่วนลดสินค้าเบา ค่าธรรมเนียมสินค้าหนัก

**ตัวอย่าง**: ค่าธรรมเนียม $5 สำหรับการสั่งซื้อเกิน 20kg
```
น้ำหนักขั้นต่ำ: 20kg
น้ำหนักสูงสุด: null (ไม่จำกัด)
```

---

### ช่วงจำนวนสินค้า

- **จำนวนสินค้าขั้นต่ำ**: ตะกร้าต้องมี ≥ จำนวนสินค้า
- **จำนวนสินค้าสูงสุด**: ตะกร้าต้องมี ≤ จำนวนสินค้า
- **กรณีการใช้งาน**: ส่วนลดสินค้าจำนวนมาก ค่าธรรมเนียมสินค้าเดี่ยว

**ตัวอย่าง**: การจัดส่งฟรีสำหรับการสั่งซื้อ 5 ชิ้นขึ้นไป
```
จำนวนขั้นต่ำ: 5
จำนวนสูงสุด: null
```

---

### โซนการจัดส่ง

- **โซน**: กฎเกณฑ์จะใช้ได้เฉพาะเมื่อที่อยู่ของลูกค้าตรงกับโซนที่เลือกอย่างน้อยหนึ่งโซน
- **การเลือกไม่ใช้**: กฎเกณฑ์จะใช้ได้กับโซนทั้งหมด
- **กรณีการใช้งาน**: ค่าธรรมเนียมหรือส่วนลดเฉพาะโซน

**ตัวอย่าง**: การจัดส่งฟรีเฉพาะโซนภายในประเทศเท่านั้น
```
โซน: ["Domestic USA"]
```

---

### วิธีการจัดส่ง

- **วิธีการ**: กฎเกณฑ์จะใช้ได้เฉพาะวิธีการจัดส่งที่เลือก
- **การเลือกไม่ใช้**: กฎเกณฑ์จะใช้ได้กับวิธีการทั้งหมด
- **กรณีการใช้งาน**: โปรโมชันเฉพาะวิธีการจัดส่ง

**ตัวอย่าง**: ส่วนลด 25% สำหรับการจัดส่งแบบเร่งด่วน
```
วิธีการ: ["Express Delivery"]
```

---

### ความต้องการสินค้า

**ต้องมีสินค้า**: ตะกร้าต้องมีสินค้าอย่างน้อยหนึ่งชิ้นจากสินค้าเหล่านี้

**ต้องมีหมวดหมู่**: ตะกร้าต้องมีสินค้าอย่างน้อยหนึ่งชิ้นจากหมวดหมู่เหล่านี้

**กรณีการใช้งาน**: การจัดส่งฟรีเฉพาะสินค้า ชุดสินค้าโปรโมชัน

**ตัวอย่าง**: การจัดส่งฟรีเมื่อตะกร้ามีสินค้า "Promotion Item A"
```
ต้องมีสินค้า: [Product ID 123]
```

---

### สินค้าที่ไม่ต้องการ

**ห้ามมีสินค้า**: กฎเกณฑ์จะไม่ใช้ได้หากตะกร้ามีสินค้าใดๆ จากสินค้าเหล่านี้

**ห้ามมีหมวดหมู่**: กฎเกณฑ์จะไม่ใช้ได้หากตะกร้ามีสินค้าใดๆ จากหมวดหมู่เหล่านี้

**กรณีการใช้งาน**: ห้ามสินค้าหนัก/ขนาดใหญ่จากส่วนลดการจัดส่งฟรี

**ตัวอย่าง**: การจัดส่งฟรียกเว้นหมวดหมู่เฟอร์นิเจอร์
```
ห้ามมีหมวดหมู่: [Furniture]
```

---

### กลุ่มลูกค้า

- **กลุ่มลูกค้า**: กฎเกณฑ์จะใช้ได้เฉพาะลูกค้าที่อยู่ในกลุ่มที่เลือก (VIP, ปลีกส่ง, ฯลฯ)
- **การเลือกไม่ใช้**: กฎเกณฑ์จะใช้ได้กับกลุ่มลูกค้าทั้งหมด
- **กรณีการใช้งาน**: ประโยชน์สำหรับลูกค้า VIP ส่วนลดสำหรับลูกค้าปลีกส่ง

**ตัวอย่าง**: ส่วนลด 15% สำหรับสมาชิก VIP
```
กลุ่มลูกค้า: ["VIP"]
```

---

### ลูกค้าใหม่

- **ลูกค้าใหม่**: ปิด/เปิดเพื่อจำกัดกฎเกณฑ์เฉพาะลูกค้าที่ไม่มีการสั่งซื้อเดิม
- **กรณีการใช้งาน**: ข้อเสนอต้อนรับลูกค้าใหม่

**ตัวอย่าง**: ลด $5 สำหรับการสั่งซื้อแรก
```
ลูกค้าใหม่: ใช่
```

---

## ลำดับความสำคัญและกระบวนการประมวลผลกฎเกณฑ์

กฎเกณฑ์จะประมวลผลตาม **ลำดับความสำคัญ** (จำนวนความสำคัญสูงสุด = ประมวลผลก่อน):

### กลไกลำดับความสำคัญ

**ตัวอย่างการประมวลผล**:
```
กฎเกณฑ์ A (ลำดับความสำคัญ 100): การจัดส่งฟรีหากค่ารวมตะกร้า > $50
กฎเกณฑ์ B (ลำดับความสำคัญ 50): ส่วนลด 10% สำหรับการจัดส่งทั้งหมด
กฎเกณฑ์ C (ลำดับความสำคัญ 1): ค่าธรรมเนียม $2 สำหรับพื้นที่ห่างไกล

ตะกร้า: $60, พื้นที่ห่างไกล
ค่าใช้จ่ายการจัดส่งพื้นฐาน: $15

ขั้นตอนที่ 1: กฎเกณฑ์ A ประเมิน (ลำดับความสำคัญ 100)
  ค่ารวมตะกร้า > $50? ใช่
  ดำเนินการ: ตั้งค่าใช้จ่ายเป็น $0
  ค่าใช้จ่ายตอนนี้: $0

ขั้นตอนที่ 2: กฎเกณฑ์ B ประเมิน (ลำดับความสำคัญ 50)
  ดำเนินการลด 10% จาก $0
  ค่าใช้จ่ายตอนนี้: $0 (ยังฟรี)

ขั้นตอนที่ 3: กฎเกณฑ์ C ประเมิน (ลำดับความสำคัญ 1)
  เพิ่มค่าธรรมเนียม $2 ให้กับ $0
  ค่าใช้จ่ายตอนนี้: $2

ค่าใช้จ่ายสุดท้าย: $2
```

**ป้ายกำกับหยุดการประมวลผลเพิ่มเติม**:

หากกฎเกณฑ์ A มี `stop_further_rules = True`:
```
กฎเกณฑ์ A (ลำดับความสำคัญ 100, stop_further_rules=True): การจัดส่งฟรีหากค่ารวมตะกร้า > $50
กฎเกณฑ์ B (ลำดับความสำคัญ 50): ส่วนลด 10%
กฎเกณฑ์ C (ลำดับความสำคัญ 1): ค่าธรรมเนียม $2 สำหรับพื้นที่ห่างไกล

ตะกร้า: $60
พื้นฐาน: $15

ขั้นตอนที่ 1: กฎเกณฑ์ A ใช้ได้ ตั้งค่าใช้จ่ายเป็น $0
        stop_further_rules = True → หยุด

ค่าใช้จ่ายสุดท้าย: $0 (กฎเกณฑ์ B และ C ไม่เคยดำเนินการ)
```

## การสร้างกฎเกณฑ์การจัดส่ง

**ขั้นตอนการดำเนินงาน**:

1. **ไปยังกฎเกณฑ์**
   - การตั้งค่า > การจัดส่ง > กฎเกณฑ์การจัดส่ง
   - คลิก "เพิ่มกฎเกณฑ์การจัดส่ง"

2. **การตั้งค่าพื้นฐาน**
   - **ชื่อ**: ชื่อภายใน (เช่น "การจัดส่งฟรีเมื่อสั่งซื้อเกิน $50")
   - **คำอธิบาย**: หมายเหตุเสริม (ไม่แสดงให้ลูกค้าเห็น)
   - **เปิดใช้งาน**: ปิด/เปิดใช้งาน
   - **ลำดับความสำคัญ**: กำหนดลำดับการประมวลผล (100 สำหรับลำดับความสำคัญสูงสุด, 1 สำหรับลำดับความสำคัญต่ำสุด)

3. **เลือกประเภทกฎเกณฑ์**
   - เลือกประเภทการปรับราคา (ส่วนลดเปอร์เซ็นต์ ส่วนลดคงที่ ตั้งค่าใช้จ่าย ฟรี ค่าธรรมเนียมเปอร์เซ็นต์ ค่าธรรมเนียมคงที่)
   - ป้อนจำนวนหรือเปอร์เซ็นต์

   - จำนวนสินค้า: ขั้นต่ำ/สูงสุด

## กลยุทธ์การรวมกฎเกณฑ์

### กลยุทธ์ 1: การรวมส่วนลด

**อนุญาตให้ส่วนลดหลายประการรวมกัน**:
```
กฎเกณฑ์ A (ลำดับความสำคัญ 100): ส่วนลด 10% สำหรับลูกค้า VIP → stop_further_rules=ไม่ใช่
กฎเกณฑ์ B (ลำดับความสำคัญ 50): ส่วนลด 15% สำหรับการสั่งซื้อ >$100 → stop_further_rules=ไม่ใช่

ลูกค้า VIP ที่สั่งซื้อ $120:
พื้นฐาน: $15
หลังจากกฎเกณฑ์ A: $13.50 (ลด 10%)
หลังจากกฎเกณฑ์ B: $11.48 (ลด 15% จาก $13.50)
```

### กลยุทธ์ 2: กฎเกณฑ์เฉพาะ

**กฎเกณฑ์เพียงหนึ่งข้อที่ใช้ได้** (ลำดับความสำคัญสูงสุด):
```
กฎเกณฑ์ A (ลำดับความสำคัญ 100): การจัดส่งฟรี >$50 → stop_further_rules=ใช่
กฎเกณฑ์ B (ลำดับความสำคัญ 50): ส่วนลด 20% สำหรับการจัดส่งทั้งหมด → stop_further_rules=ใช่

ตะกร้า > $50:
กฎเกณฑ์ A ใช้ได้ → จัดส่งฟรี → หยุด
กฎเกณฑ์ B ไม่เคยดำเนินการ
```

### กลยุทธ์ 3: ค่าธรรมเนียมตามเงื่อนไข

**ส่วนลดก่อน ค่าธรรมเนียมหลัง**:
```
กฎเกณฑ์ A (ลำดับความสำคัญ 100): การจัดส่งฟรี >$75
กฎเกณฑ์ B (ลำดับความสำคัญ 75): ส่วนลด 15% สำหรับลูกค้า VIP
กฎเกณฑ์ C (ลำดับความสำคัญ 50): ส่วนลด 10% ทั่วไป
กฎเกณฑ์ D (ลำดับความสำคัญ 25): ค่าธรรมเนียม $5 สำหรับพื้นที่ห่างไกล
กฎเกณฑ์ E (ลำดับความสำคัญ 1): ค่าธรรมเนียมเชื้อเพลิง 10% ของค่าใช้จ่าย

การสั่งซื้อ: $80, พื้นที่ห่างไกล, ลูกค้า VIP
พื้นฐาน: $20
A: $80 > $75 → ฟรี ($0)
B: VIP → ลด 15% จาก $0 = $0
C: ลด 10% จาก $0 = $0
D: พื้นที่ห่างไกล +$5 = $5
E: ค่าธรรมเนียมเชื้อเพลิง +10% ของ $5 = $5.50

สุดท้าย: $5.50 (ไม่ฟรีเนื่องจากมีค่าธรรมเนียมเพิ่ม)

**เพื่อป้องกันสิ่งนี้ ใช้ stop_further_rules=ใช่**:
```
กฎเกณฑ์ A (ลำดับความสำคัญ 100, stop=ใช่): การจัดส่งฟรี >$75

การสั่งซื้อเดียวกัน:
A: $80 > $75 → ฟรี ($0) → หยุด
สุดท้าย: $0 (จริงๆ แล้วฟรี)
```

## เคล็ดลับ

- **ใช้ลำดับความสำคัญสูงสุดสำหรับการจัดส่งฟรี** - ลำดับความสำคัญ 100 ทำให้มันดำเนินการก่อนการปรับราคาอื่นๆ
- **ตั้งค่า stop_further_rules สำหรับกฎเกณฑ์ที่มีความสำคัญสูงสุด** - การจัดส่งฟรีควรหยุดการประมวลผลเพิ่มเติม
- **ทดสอบการรวมกฎเกณฑ์** - กฎเกณฑ์หลายข้ออาจมีปฏิสัมพันธ์ที่ไม่คาดคิด
- **ใช้ชื่อที่อธิบายได้ดี** - "ส่วนลด 20% สำหรับลูกค้า VIP (ลำดับความสำคัญ 75)" ดีกว่า "กฎเกณฑ์ 3"
- **บันทึกเหตุผลการดำเนินการที่ซับซ้อน** - เพิ่มหมายเหตุในช่องคำอธิบาย
- **เริ่มต้นด้วยกฎเกณฑ์ง่ายๆ** - เพิ่มความซับซ้อนทีละขั้น
- **ตรวจสอบประสิทธิภาพของกฎเกณฑ์** - ตรวจสอบว่ากฎเกณฑ์ถูกใช้หรือทำให้เกิดความสับสน
- **หลีกเลี่ยงกฎเกณฑ์ที่มากเกินไป** - กฎเกณฑ์ที่มากเกินไปทำให้การชำระเงินช้าลง ใช้ไม่เกิน 5-10 ข้อ
- **ใช้โซนสำหรับพื้นที่ทางภูมิศาสตร์** - ดีกว่ากฎเกณฑ์หลายข้อที่คล้ายกันต่อประเทศ
- **รวมกับวิธีการจัดส่ง** - กฎเกณฑ์ + วิธีการจัดส่งทำงานร่วมกันเพื่อการกำหนดราคาที่ซับซ้อน
- **ตั้งช่วงเวลาที่ชัดเจน** - เพิ่มวันสิ้นสุดเสมอสำหรับโปรโมชัน
- **ทดสอบกรณีขอบเขต** - ที่อยู่พอดี $50, จำนวนสินค้าพอดี 5 ชิ้น, ฯลฯ
