---
title: 收据模板自定义
---

收据模板控制您POS终端打印的热敏收据的外观和内容。自定义页眉和页脚文本，添加您的标志，配置合规字段（税号、商业注册号），并包含促销二维码。模板支持范围定位——创建适用于所有商店的默认模板，适用于区域的组模板，或适用于单个位置的商店模板。系统使用范围优先级规则来确定打印收据时使用哪个模板。

使用收据模板来保持品牌一致性，满足区域合规要求，并通过促销元素增强客户参与度。

![收据模板列表](/static/core/admin/img/help/receipt-template-customization/receipt-list.webp)

## 收据模板基础

收据模板定义了从ESC/POS热敏打印机打印的收据的结构和内容。每个模板指定：

**物理配置**：
- 纸张宽度（58mm或80mm）
- 标志图像（热敏打印的单色）
- 字体大小和间距

**内容部分**：
- 页眉文本（商店名称、地址、联系方式）
- 动态交易数据（商品、价格、总计、支付方式）
- 页脚文本（退货政策、感谢信息、社交媒体）
- 合规字段（税号、商业注册号）
- 带标签的促销二维码

**范围定位**：
- 默认模板（适用于所有商店，除非被覆盖）
- 组模板（适用于组内的所有商店）
- 商店模板（适用于特定商店/仓库）

## 范围优先级规则

当终端打印收据时，系统使用此层次结构选择模板（优先级从高到低）：

| 优先级 | 范围 | 示例 | 用例 |
|--------|-----|------|------|
| **1** | 商店特定 | 巴黎商店模板 | 独特的法国税务合规要求 |
| **2** | 组特定 | 欧洲商店模板 | 所有欧盟地点的增值税显示 |
| **3** | 默认 | 全球模板 | 所有未配置商店的后备 |

**工作原理**：
1. 检查商店是否有专用模板（仓库特定）
2. 如果没有，检查商店的组是否有组模板
3. 如果没有，使用默认模板

**示例**：
- 默认模板："标准收据"（无范围分配）
- 组模板："欧盟收据"（分配给欧洲商店组）- 包含增值税注册
- 商店模板："巴黎收据"（分配给巴黎仓库）- 包含法国SIRET号码

**结果**：
- 巴黎商店终端：使用"巴黎收据"（最具体）
- 柏林商店终端（在欧洲商店组中，无商店模板）：使用"欧盟收据"（组级别）
- 纽约商店终端（无组，无商店模板）：使用"标准收据"（默认后备）

## 纸张宽度配置

热敏收据打印机使用58mm或80mm纸张宽度。根据您的打印机硬件选择：

| 纸张宽度 | 每行字符数 | 最佳用途 | 典型用途 |
|----------|------------|----------|-------------|
| **58mm** | ~32个字符 | 小型占地面积，便携 | 食品卡车、移动POS、自动售货机 |
| **80mm** | ~48个字符 | 标准零售 | 大多数零售商店、餐厅 |

**不能混合宽度**：使用相同模板的所有终端必须具有相同纸张宽度的打印机。如果您有混合打印机类型，请为每种宽度创建单独的模板。

**标志尺寸限制**：
- **58mm**：最大宽度384像素（推荐：350px）
- **80mm**：最大宽度576像素（推荐：550px）

超过最大宽度的标志会自动缩小，这可能会降低质量。

## 标志配置

收据标志必须为**单色**（仅黑色和白色）以兼容热敏打印机：

**标志要求**：
- 文件格式：PNG、JPG或WebP
- 颜色模式：单色（黑色像素在白色背景上）
- 推荐尺寸：
  - 58mm纸张：350px宽度 × 100-150px高度
  - 80mm纸张：550px宽度 × 150-200px高度
- 文件大小：<100KB（热敏打印机内存有限）

**创建单色标志**：
1. 从您的常规标志（彩色或灰度）开始
2. 使用图像编辑器将其转换为纯黑色和白色（无灰度）
3. 增加对比度以确保黑色元素是实色
4. 导出为带有透明或白色背景的PNG

**标志定位**：
- 始终水平居中
- 打印在收据顶部（在页眉文本上方）
- 后面自动添加间距（防止内容拥挤）

**选择标志**：
- 在模板表单中点击**浏览媒体库**
- 选择单色标志资产
- 预览显示标志在收据上的显示效果

**无标志**：如果希望仅使用文本品牌（页眉文本可以包含商店名称），请留空标志字段。

## 页眉文本

页眉文本在标志后立即显示（如果没有标志，则显示在顶部）。典型内容：

**商店名称和地址**：
```
Your Store Name
123 Main Street
City, State 12345
Phone: (555) 123-4567
```

**营业时间**：
```
星期一至星期五：上午9点至晚上9点
星期六至星期日：上午10点至下午6点
```

**标语或口号**：
```
优质产品，卓越服务
```

**格式**：
- 使用换行符分隔信息
- 自动水平居中
- 保持行数在纸张宽度的字符限制内（58mm为32个字符，80mm为48个字符）

**可用变量**（可选）：
- `{store_name}` - 替换为仓库名称
- `{order_date}` - 替换为交易日期
- `{order_number}` - 替换为订单ID

大多数商家使用静态文本而不是变量以保持页眉一致性。

## 页脚文本

页脚文本在交易详情（商品、总计、支付）之后显示。典型内容：

**退货政策**：
```
30天内凭收据退货
仅限商店积分或换货
```

**感谢信息**：
```
感谢您光临！
关注@yourstore
```

**客户服务**：
```
有问题？请致电(555) 123-4567
或发送电子邮件至support@yourstore.com
```

**格式建议**：
- 将最重要的信息放在前面（退货政策、联系方式）
- 使用换行符提高可读性
- 考虑在各部分之间添加分隔线（`---`）

## 合规字段

许多司法管辖区要求收据上包含特定信息：

**税号标签** - 可自定义的税号标签：
- 美国："Tax ID"或"EIN"
- 欧盟："VAT Number"或"VAT Reg No"
- 加拿大："GST/HST Number"
- 澳大利亚："ABN"

**税号值** - 实际的识别号码：
- 在模板中输入一次，显示在所有收据上
- 示例："VAT Number: GB123456789"

**商业注册标签** - 可自定义的商业注册标签：
- 法国："SIRET"
- 德国："Handelsregister"
- 英国："Company Registration Number"

**商业注册值** - 实际的注册号码：
- 示例："SIRET: 123 456 789 00010"

**显示Spwig品牌** - 开关以显示或隐藏"Powered by Spwig"品牌：
- 默认启用（支持平台开发）
- 禁用用于白标操作

**各地区的合规示例**：

**欧洲联盟**：
- 税号标签："VAT Number"
- 税号值："GB123456789"
- 如果国家要求，显示公司注册号码

**美国**：
- 通常没有收据税号要求（因州而异）
- 可能包括B2B交易的EIN

**法国（特定）**：
- 所有收据上强制要求SIRET
- 商业注册标签："SIRET"
- 商业注册值："123 456 789 00010"

**澳大利亚**：
- 建议注册GST的公司使用ABN（澳大利亚商业号码）
- 税号标签："ABN"

在上线前检查您当地司法管辖区的收据要求。

## 二维码促销

在收据底部包含二维码以促进客户参与：

**二维码URL** - 扫描后的目标地址：
- 评价请求：`https://yourstore.com/reviews/leave-review`
- 忠诚度计划：`https://yourstore.com/loyalty/join`
- 下一购买折扣：`https://yourstore.com/discount/THANKYOU`
- 社交媒体：`https://instagram.com/yourstore`
- 网站主页：`https://yourstore.com`

**二维码标签** - 显示在二维码上方的文本：
- "扫描以留下评价并获得下次购买10%的折扣"
- "加入我们的忠诚度计划 - 扫描此处"
- "关注我们的Instagram - 扫描以连接"
- "评价您的体验"

**二维码最佳实践**：
- 使用短网址（长网址会生成密集、难以扫描的二维码）
- 部署前用多种手机摄像头测试二维码
- 在标签中包含清晰的价值主张（客户扫描后获得什么）
- 跟踪二维码扫描以衡量效果（使用带跟踪参数的URL）

**动态二维码**（高级）：
- 使用二维码重定向服务（bit.ly, tinyurl）创建短网址
- 将重定向指向季节性不同的目的地，而无需重新打印收据
- 示例：`https://bit.ly/yourstoreqr` → 重定向到当前促销

## 为不同范围创建模板

**默认模板**（推荐的起点）：
1. 导航到 **POS > 收据模板**
2. 点击 **+ 添加收据模板**
3. 留空 **仓库** 和 **商店组** 字段（这使其成为默认模板）
4. 配置与您最常用的打印机类型匹配的纸张宽度
5. 添加标志、页眉、页脚
6. 配置主要市场的合规字段
7. 保存

此模板适用于所有商店，除非被覆盖。

**组模板**（用于区域差异）：
1. 创建新模板
2. 选择 **商店组**（例如，"欧洲商店"）
3. 留空 **仓库**
4. 调整区域的合规字段（例如，增值税格式）
5. 调整页眉文本（例如，区域地址）
6. 保存

此模板适用于组内的所有商店。

**商店模板**（用于特定位置的需求）：
1. 创建新模板
2. 选择 **仓库**（例如，"巴黎商店"）
3. 调整此特定位置的所有字段
4. 保存

此模板仅适用于该商店。

**测试模板**：
- 在终端上处理测试交易
- 打印收据
- 验证标志清晰度、文本对齐、合规字段、二维码可扫描性
- 如果需要，调整模板并重新测试

## 常见收据布局

**最小收据**（食品卡车、临时摊位）：
- 无标志（节省空间）
- 页眉：仅商店名称和电话
- 页脚：感谢信息
- 无二维码

**标准零售收据**：
- 标志（单色品牌标志）
- 页眉：完整商店名称、地址、营业时间
- 合规：税号
- 页脚：退货政策、感谢信息
- 二维码：评价请求

**高级零售收据**：
- 标志（完整品牌文字标志）
- 页眉：标语、地址、联系方式
- 合规：税号、商业注册
- 页脚：退货政策、客户服务、社交媒体
- 二维码：忠诚度计划注册

**多地点连锁店**：
- 默认模板：公司品牌、标准政策
- 组模板：区域合规（欧盟增值税、加拿大GST）
- 商店模板：特定位置的地址和电话

## 管理多个模板

**模板命名约定**：
- 在名称中使用范围："默认收据"、"欧洲组收据"、"巴黎商店收据"
- 有助于在查看列表时识别哪个模板适用于何处

**模板更改**：
- 更改立即应用于未来的收据
- 已打印的旧收据不受影响
- 在部署到所有商店之前，在低流量终端上测试更改

**模板复制**：
- 当创建与现有模板相似的新模板时，复制现有模板并进行修改
- 避免从头开始创建

**删除模板**：
- 当存在终端时，不能删除默认模板（必须有一个后备）
- 可以删除组/商店模板（终端回退到层次结构中的下一级）
- 在删除之前确认没有终端正在积极使用该模板

## 小贴士

- **不确定时从80mm开始** - 标准纸张宽度适用于大多数零售；58mm是专业用途
- **在实际打印机上测试标志** - 屏幕上看起来好的标志可能打印效果差；尽早测试
- **保持合规字段更新** - 收据上的过期税务注册会导致法律问题
- **带有价值主张的二维码扫描效果更好** - "扫描以获得10%折扣"的扫描效果比"扫描此处"好10倍
- **检查字符限制** - 文本换行会破坏格式；在部署前计算每行的字符数
- **每种纸张宽度一个模板** - 不要将80mm模板分配给使用58mm打印机的终端（标志无法适应）
- **每月打印测试收据** - 打印机随时间退化；验证质量是否仍然可接受
- **尽量少用变量** - 静态文本比动态变量更可靠（失败点更少）
- **备份模板配置** - 在进行重大更改前截图或导出模板设置（便于回滚）
- **区域合规性各不相同** - 在部署前研究当地收据要求；不合规可能会导致严重罚款
