---
title: 배송 규칙
---

배송 규칙은 장바구니 내용, 고객 속성 및 배송 지역에 따라 배송 방법에 조건부 비용 조정을 적용합니다. $50 이상 시 무료 배송 제공, 원격 지역 추가 요금, VIP 고객의 배송 할인 등이 가능합니다. 규칙은 우선순위 기반으로 실행되며(높은 우선순위 먼저) 추가로 처리를 중단하도록 하는 stop 플래그가 선택적으로 포함됩니다. 각 규칙은 여러 조건(장바구니 금액, 무게, 지역, 제품, 고객 그룹)을 평가하고 모든 조건이 일치할 때 6가지 조정 유형 중 하나를 실행합니다.

장바구니 금액이 정적 배송 방법의 고정된 비용이 아닌 주문 맥락에 따라 동적으로 변경되는 경우 배송 규칙을 사용합니다.

## 배송 규칙 유형

배송 규칙은 6가지 유형의 비용 조정을 적용합니다:

### 비율 할인

**기능**: 배송 비용을 비율로 감소시킵니다(예: 25% 할인).

**공식**: `new_cost = base_cost × (1 - percent/100)`

**예시**:
```
기본 비용: $20
할인: 25%
결과: $15
```

**사용 사례**:
- VIP 고객 할인(모든 배송에 20% 할인)
- 계절 할인(12월 배송에 15% 할인)
- 대량 주문 할인(5개 이상 항목에 10% 할인)

---

### 고정 할인

**기능**: 배송 비용에서 고정 금액을 뺍니다.

**공식**: `new_cost = base_cost - amount` (최소 $0)

**예시**:
```
기본 비용: $15
할인: $5
결과: $10
```

**사용 사례**:
- 첫 주문 고객 보너스(첫 주문 배송에 $5 할인)
- 뉴스레터 등록 보상(배송에 $3 할인)
- 충성도 프로그램 혜택(월별 배송에 $10 할인)

---

### 고정 비용

**기능**: 배송 비용을 특정 금액으로 대체합니다.

**공식**: `new_cost = fixed_amount`

**예시**:
```
기본 비용: $25
설정 금액: $9.99
결과: $9.99
```

**사용 사례**:
- 플래시 세일(오늘 모든 주문에 $5 배송비)
- 카테고리별 배송(도서 항상 $3.99 배송)
- 시간 기반 프로모션(이번 주 배송비 $9.99로 제한)

---

### 무료 배송

**기능**: 배송 비용을 $0으로 설정합니다.

**공식**: `new_cost = $0`

**예시**:
```
기본 비용: $18
규칙 적용
결과: $0
```

**사용 사례**:
- $50 이상 무료 배송
- 특정 제품에 대한 무료 배송(프로모션 항목)
- VIP 고객에게 무료 배송
- 3개 이상 항목 주문 시 무료 배송

---

### 추가 요금(고정)

**기능**: 배송 비용에 고정 금액을 추가합니다.

**공식**: `new_cost = base_cost + amount`

**예시**:
```
기본 비용: $12
추가 요금: $5
결과: $17
```

**사용 사례**:
- 원격 지역 배송 요금
- 과대형 항목 처리 비용
- 토요일 배송 추가 요금
- 취약한 항목 포장 비용

---

### 추가 요금(비율)

**기능**: 배송 비용을 비율로 증가시킵니다.

**공식**: `new_cost = base_cost × (1 + percent/100)`

**예시**:
```
기본 비용: $20
추가 요금: 15%
결과: $23
```

**사용 사례**:
- 피크 시즌 추가 요금(휴일 기간 20%)
- 익스프레스 배송 프리미엄(50% 추가 요금)
- 연료 추가 요금(현재 비용 기반 가변)

---

## 규칙 조건

규칙은 **모든 조건이 통과해야 적용됩니다**:

### 시간 유효성

- **시작 날짜**: 규칙이 이 날짜 이후에만 활성화됩니다
- **종료 날짜**: 규칙이 이 날짜 이전에만 활성화됩니다
- **사용 사례**: 계절 할인, 제한 시간 오퍼

**예시**: 블랙 프라이데이 주말에만 무료 배송
```
시작: 2026-11-27 00:00
종료: 2026-11-30 23:59
```

---

### 장바구니 금액 범위

- **최소 장바구니 금액**: 장바구니 소계가 금액 이상이어야 합니다
- **최대 장바구니 금액**: 장바구니 소계가 금액 이하이어야 합니다
- **사용 사례**: 무료 배송 임계값, 계단식 할인

**예시**: $50-$200 주문에 무료 배송
```
최소: $50
최대: $200
```

---

### 장바구니 무게 범위

- **최소 무게**: 총 장바구니 무게가 금액 이상이어야 합니다
- **최대 무게**: 총 장바구니 무게가 금액 이하이어야 합니다
- **사용 사례**: 가벼운 배송 할인, 무거운 항목 추가 요금

**예시**: 20kg 이상 주문에 $5 추가 요금
```
최소 무게: 20kg
최대 무게: null (무제한)
```

---

### 항목 수 범위

- **최소 항목 수**: 장바구니에 항목 수가 수량 이상이어야 합니다
- **최대 항목 수**: 장바구니에 항목 수가 수량 이하이어야 합니다
- **사용 사례**: 대량 주문 할인, 단일 항목 요금

**예시**: 5개 이상 항목에 무료 배송
```
최소 항목: 5
최대 항목: null
```

---

### 배송 지역

- **지역**: 고객 주소가 선택된 지역 중 하나와 일치해야 규칙이 적용됩니다
- **빈 선택**: 모든 지역에 적용됩니다
- **사용 사례**: 지역별 추가 요금 또는 할인
