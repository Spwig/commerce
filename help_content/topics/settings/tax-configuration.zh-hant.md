## 稅率設定

每個稅率定義：

**地理範圍**：
- 國家（必填）
- 州/省份（選填）
- 城市（選填）
- 郵政編碼模式（選填，正則表達式）

**稅率細節**：
- **稅率**：百分比（例如，8.5%）
- **名稱**：顯示名稱（例如，"加州銷售稅"）
- **優先順序**：當多個稅率匹配時，優先順序較高者優先
- **啟用**：切換而不刪除

**免稅**：
- **免稅產品類型**：數字產品、實體產品、服務
- **免稅類別**：特定產品類別（食品、書籍、醫療）

**複合稅**：
- **是否為複合稅**：在之前的稅率之上應用此稅率（稅率上的稅率）
- 範例：魁北克省銷售稅（PST）在聯邦銷售稅（GST）之上計算

## 常見稅務情境

### 美國銷售稅（州級）

```
名稱：加州銷售稅
國家：美國
州：CA
稅率：7.25%
優先順序：50
```

### 歐盟增值稅（國家級）

```
名稱：英國增值稅
國家：GB
稅率：20%
優先順序：50

名稱：德國增值稅
國家：DE
稅率：19%
優先順序：50
```

### 加拿大GST/PST（複合稅）

```
稅率1：聯邦GST
國家：CA
稅率：5%
優先順序：100
是否為複合稅：否

稅率2：魁北克省PST
國家：CA
州：QC
稅率：9.975%
優先順序：50
是否為複合稅：是（應用於小計加GST）
```

### 城市級稅

```
名稱：西雅圖銷售稅
國家：美國
州：WA
城市：西雅圖
稅率：10.1%
優先順序：100
```

## 稅務免稅

### 產品類型免稅

免稅整個產品類型：

- **數字產品**：軟體、電子書、音樂
- **實體產品**：有形產品
- **服務**：諮詢、安裝

範例：歐盟增值稅對消費者不適用於數字產品（在某些情況下）

### 類別免稅

免稅特定產品類別：

- 食品與雜貨（通常免稅或減稅）
- 書籍與教育材料
- 醫療用品與藥品
- 服裝（某些管轄區）

設定：
```
名稱：加州銷售稅
稅率：7.25%
免稅類別：["食品與飲料", "處方藥"]
```

## 稅務預設組

快速加載常見稅務設定：

**美國銷售稅預設**：
- 所有50個州 + 區域
- 州級稅率
- 稅率變更時自動更新

**歐盟增值稅預設**：
- 所有27個歐盟成員國
- 標準增值稅率
- B2B的反向徵稅邏輯

**如何使用預設**：
1. 設定 > 紙箱 > 稅務預設
2. 選擇預設組（例如，"2026美國銷售稅"）
3. 點擊"載入預設"
4. 稅率會自動導入
5. 根據需要進行自定義

## 優先順序解決方案

當多個稅率匹配時，優先順序最高的會生效：

範例：
```
西雅圖，華盛頓州的客戶：

稅率A：美國聯邦（優先順序1） - 0%
稅率B：華盛頓州（優先順序50） - 6.5%
稅率C：西雅圖市（優先順序100） - 3.6%

結果：西雅圖稅率（總共10.1%）適用
```

## 稅務顯示選項

在設定 > 紙箱 > 稅務設定中進行配置：

- **價格包含稅**：顯示包含稅的價格（歐盟風格）
- **單獨顯示稅**：將稅顯示為明細項目（美國風格）
- **四捨五入稅**：按項目或按訂單
- **稅務標題**：自定義標籤（"VAT"、"Sales Tax"、"GST"）

## 測試稅務設定

上線前：

1. 從不同管轄區創建測試訂單
2. 確認正確的稅率已應用
3. 檢查免稅是否對排除的類別生效
4. 測試複合稅計算
5. 查看發票上的稅務明細項目

## 合規注意事項

- **美國**：聯邦稅收法規要求在有實體存在或經濟聯繫的州徵收稅
- **歐盟**：已註冊增值稅的企業必須從歐盟客戶徵收增值稅
- **加拿大**：GST/HST/PST因省份而異
- **諮詢稅務專業人士**：稅務法規經常變化，請確認當前要求

## 小技巧

- **使用稅務預設** - 比手動輸入更快，自動更新
- **監控聯繫閾值** - 跟蹤各州銷售額以符合美國經濟聯繫要求
- **正確設置優先順序** - 城市 > 州 > 國家
- **測試複合稅** - 確認計算結果與預期金額相符
- **每年更新** - 稅率會變動，每年一月審核一次
- **記錄免稅原因** - 保留類別免稅的記錄
- **使用描述性名稱** - "2026加州銷售稅"比"稅1"更好
- **預設啟用稅務** - 比忘記應用稅更安全
