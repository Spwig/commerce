title: 고객 화면 프로모션 슬라이드
---

프로모션 슬라이드는 POS 단말기가 유휴 상태(활성 거래 없음)일 때 고객 대면 화면에 표시됩니다. 계절 프로모션, 신상품 출시, 매장 정책, 행사 정보, 충성도 프로그램 혜택을 보여주는 이미지의 캐러셀을 생성하세요. 범위 지정을 통해 슬라이드를 특정 매장 또는 그룹에 대상으로 할 수 있습니다. 예를 들어, 미국 매장에서만 연말 프로모션을 진행하거나, 관련 위치에서만 지역 행사 정보를 표시할 수 있습니다. 활성 슬라이드는 5-10초 간격으로 자동으로 순환하며, 고객이 대기하는 동안 정보를 제공하는 매력적인 디지털 시그니지가 됩니다.

프로모션 슬라이드를 사용하여 현재 프로모션에 대한 인식을 높이고, 고객에게 정책을 교육하며, 충성도 프로그램 및 행사에 대한 참여를 유도하세요.

![프로모션 슬라이드 목록](/static/core/admin/img/help/customer-display-promo-slides/promoslide-list.webp)

## 고객 화면 동작

POS 단말기가 유휴 상태(등록대에 고객 없음, 거래 진행 중 아님)일 때 고객 대면 화면은 다음과 같이 표시됩니다:

**캐러셀 모드**:
- 모든 활성 슬라이드를 순환합니다
- 각 슬라이드는 5-10초 동안 표시됩니다(단말기별로 설정 가능)
- 슬라이드 간 부드러운 페이드 전환
- 거래 시작 전까지 지속적으로 순환합니다

**거래 중**:
- 캐러셀이 즉시 중단됩니다
- 화면이 거래 화면(상품, 누적 금액, 결제 요청)으로 전환됩니다
- 거래가 완료되고 단말기가 다시 유휴 상태가 되면 캐러셀이 재개됩니다

**설정된 슬라이드가 없음**:
- 환영 메시지와 함께 매장 브랜딩이 표시됩니다
- 정적 화면(캐러셀 없음)

**기술적 요구 사항**:
- 고객 화면은 별도의 모니터 또는 캐셔와 동일한 화면일 수 있습니다(POS 앱은 화면 분할 모드를 지원함)
- 화면 동기화는 BroadcastChannel API(동일 장치 통신) 또는 WebSocket(별도 장치 화면)을 통해 이루어집니다

## 범위 대상 설정

영수증 템플릿과 마찬가지로 프로모션 슬라이드는 범위 기반 대상 설정을 지원합니다(우선순위 높음에서 낮음):

| 우선순위 | 범위 | 예시 | 사용 사례 |
|----------|-------|---------|----------|
| **1** | 매장별 | 파리 매장 슬라이드 | 파리의 여름 축제 이벤트 슬라이드 |
| **2** | 그룹별 | 유럽 매장 슬라이드 | EU 전용 GDPR 개인정보 보호 정책 슬라이드 |
| **3** | 모든 매장 | 글로벌 슬라이드 | "50달러 이상 주문 시 무료 배송" (회사 전체 프로모션) |

**범위 작동 방식**:
- 단말기는 해당 매장 범위에 맞는 슬라이드를 표시합니다(매장별 슬라이드)
- 해당 그룹 범위에 맞는 슬라이드도 표시합니다(매장이 그룹에 속한 경우)
- 범위 지정이 없는 슬라이드(글로벌 슬라이드)도 표시합니다
- 결과: 매장은 범위 지정 및 글로벌 슬라이드의 혼합으로 3-5개 슬라이드를 표시할 수 있습니다

**예시**:
- 글로벌 슬라이드: "새로운 충성도 프로그램 - 오늘 바로 가입하세요!" (범위 없음)
- 그룹 슬라이드: "메모리얼 데이 세일 - 30% 할인" (미국 매장 그룹만 해당)
- 매장 슬라이드: "그랜드 오픈 - 뉴욕 본점" (뉴욕 매장만 해당)

**뉴욕 매장 단말기**는 3개 슬라이드(매장 + 그룹 + 글로벌)를 모두 표시합니다
**런던 매장 단말기**는 글로벌 슬라이드만 표시합니다(미국 매장 그룹에 속하지 않으며 뉴욕 매장이 아님)

## 이미지 요구 사항

프로모션 슬라이드는 고객 화면 모니터에 최적화된 전체 화면 이미지입니다:

**종횡비**: 16:9 (와이드 스크린)

**권장 해상도**: 1920×1080 픽셀 (풀 HD)
- 대부분의 현대 디스플레이에 깨끗하게 확대 가능
- 품질과 로딩 속도의 균형을 맞춘 파일 크기

**허용된 해상도**:
- 최소: 1280×720 (HD)
- 최적: 1920×1080 (풀 HD)
- 최대: 3840×2160 (4K) - 권장하지 않음(대규모 파일 크기, 느린 로딩)

**파일 형식**: JPG, PNG 또는 WebP
- JPG: 사진에 사용
- PNG: 투명도가 있는 그래픽(배경은 권장됨)
- WebP: 가장 작은 파일 크기

**파일 크기**: 슬라이드당 500KB 미만
- 더 큰 파일은 캐러셀 로딩을 느리게 함
- 업로드 전에 이미지를 압축하세요(미디어 라이브러리 최적화 사용)

**디자인 권장 사항**:
- 고객이 디스플레이에서 2-6피트 떨어진 거리에서 읽을 수 있도록 높은 대비를 사용하세요
- 큰 글씨(본문 글씨 최소 48pt, 헤드라인 72pt 이상)
- 굵은 글꼴(일부 디스플레이에서는 얇은 글꼴이 흐려짐)
- 작은 세부 사항은 피하세요(고객의 시야에서 보이지 않음)
- 고객이 해야 할 행동을 포함하세요(예: "캐셔에게 문의하세요", "오늘 바로 가입하세요")

## 프로모션 슬라이드 생성

**POS > 프로모션 슬라이드**로 이동하여 **+ 프로모션 슬라이드 추가**를 클릭하세요:

![프로모션 슬라이드 추가 폼](/static/core/admin/img/help/customer-display-promo-slides/promoslide-add-form.webp)

**이미지** - 업로드하거나 미디어 라이브러리에서 선택:
- **미디어 라이브러리 보기**를 클릭하여 기존 이미지를 선택하세요
- 또는 위의 요구 사항을 충족하는 새 이미지를 업로드하세요
- 미리보기에서 이미지가 화면에 어떻게 표시되는지 확인할 수 있습니다

**제목**(선택) - 슬라이드 상단에 텍스트 오버레이:
- 최대 60자(더 긴 텍스트는 잘려짐)
- 이미지 상단에 반투명한 어두운 바에 표시됨
- 슬라이드 헤드라인에 사용("여름 세일", "신상품 출시")
- 이미지에 제목 텍스트가 포함되어 있다면 빈칸으로 남겨두세요

**부제**(선택) - 제목 아래에 텍스트 오버레이:
- 최대 120자
- 제목 아래에 동일한 반투명한 바에 표시됨
- 보조 정보에 사용("최대 50% 할인", "구매 시 무료 선물")
- 이미지가 자체적으로 제목을 포함한다면 빈칸으로 남겨두세요

**활성화** - 슬라이드를 활성화/비활성화하는 토글:
- 활성화된 슬라이드만 캐러셀에 표시됩니다
- 계절별 활성화에 사용(프로모션이 끝난 후 비활성화)
- 비활성화하면 슬라이드를 향후 재활성화에 위해 보관합니다

**정렬 순서** - 캐러셀에서 슬라이드의 위치를 제어:
- 낮은 숫자는 순환에서 먼저 나타납니다
- 10의 배수 사용: 10, 20, 30(기존 슬라이드 사이에 슬라이드를 삽입할 수 있음)
- 예: 휴일 세일(정렬 순서 10)은 일반 충성도 프로그램(정렬 순서 20)보다 먼저 표시됩니다

**범위 지정**(선택):
- **창고** - 특정 매장에서만 표시하도록 선택
- **매장 그룹** - 그룹에 속한 매장에서만 표시하도록 선택
- **둘 다 비워두기** - 모든 매장에서 표시(글로벌 슬라이드)

## 정렬 순서 및 캐러셀 흐름

**예시 캐러셀**(뉴욕 매장 단말기):
- 슬라이드 1(정렬 순서 10): "그랜드 오픈 - 뉴욕 본점"(매장별)
- 슬라이드 2(정렬 순서 15): "메모리얼 데이 세일 - 30% 할인"(미국 매장 그룹)
- 슬라이드 3(정렬 순서 20): "새로운 충성도 프로그램 - 오늘 바로 가입하세요!"(글로벌)
- 슬라이드 4(정렬 순서 30): "우리 매장을 팔로우하세요 @yourstore"(글로벌)

캐러셀 순환: 1 → 2 → 3 → 4 → 1 → 2 → ...

**런던 매장 단말기**(미국 매장 그룹에 속하지 않으며 다른 매장):
- 슬라이드 1(정렬 순서 20): "새로운 충성도 프로그램 - 오늘 바로 가입하세요!"(글로벌)
- 슬라이드 2(정렬 순서 30): "우리 매장을 팔로우하세요 @yourstore"(글로벌)

캐러셀 순환: 1 → 2 → 1 → 2 → ...

정렬 순서를 사용하여 순환에서 가장 중요한 콘텐츠를 우선 표시하세요.

## 계절별 활성화 전략

**문제**: 매계절마다 프로모션 슬라이드를 생성하거나 삭제하는 것은 번거롭습니다.

**해결책**: 슬라이드를 한 번만 생성하고 계절별로 활성화/비활성화하세요:

1. **주요 이벤트용 슬라이드 생성**:
   - "여름 세일"(활성화: 아니요, 미리 생성)
   - "학기 시작"(활성화: 아니요, 미리 생성)
   - "블랙 프라이데이"(활성화: 아니요, 미리 생성)
   - "연말 세일"(활성화: 아니요, 미리 생성)

2. **적절한 시기에 활성화**:
   - 6월 1일: "여름 세일" → 활성화: 예
   - 8월 15일: "여름 세일" → 활성화: 아니요, "학기 시작" → 활성화: 예
   - 11월 20일: "블랙 프라이데이" → 활성화: 예
   - 12월 1일: "블랙 프라이데이" → 활성화: 아니요, "연말 세일" → 활성화: 예

3. **이벤트 후 비활성화**:
   - 슬라이드 라이브러리를 정리합니다
   - 매년 슬라이드를 재사용합니다(필요 시 이미지 업데이트, 설정 유지)

## 사용 사례 예시

**사용 사례 1: 계절 프로모션**
- 이미지: 빨간 배경에 흰색 텍스트 "SUMMER SALE - UP TO 60% OFF"
- 제목: "여름 세일"
- 부제: "선택 상품 50-60% 할인. 캐셔에게 문의하세요."
- 범위: 모든 매장(글로벌)
- 정렬 순서: 10(여름 동안 우선순위 높음)
- 활성화: 6월-8월만

**사용 사례 2: 매장 정책**
- 이미지: 환불 정책 단계를 보여주는 인포그래픽
- 제목: "간편한 환불"
- 부제: "영수증 있는 경우 30일 동안. 질문 없음."
- 범위: 모든 매장(글로벌)
- 정렬 순서: 40(프로모션보다 우선순위 낮음)
- 활성화: 연중

**사용 사례 3: 신상품 출시**
- 이미지: 신상품의 히어로 이미지
- 제목: "NEW: 무선 이어폰 프로"
- 부제: "온라인 및 오프라인에서 구매 가능. $199.99"
- 범위: 모든 매장(글로벌)
- 정렬 순서: 5(출시 주간 동안 우선순위 높음)
- 활성화: 출시 주간만, 이후 비활성화

**사용 사례 4: 지역 행사**
- 이미지: 지역 자선 달리기 포스터
- 제목: "지역 지원"
- 부제: "6월 15일 커뮤니티 5K에 함께해 주세요!"
- 범위: 특정 매장(뉴욕 매장만)
- 정렬 순서: 8(이 매장의 우선순위)
- 활성화: 행사 2주 전

**사용 사례 5: 충성도 프로그램**
- 이미지: 포인트 예시가 포함된 충성도 카드 시각화
- 제목: "포인트 적립"
- 부제: "충성도 프로그램에 가입하여 $1당 1포인트를 적립하세요"
- 범위: 모든 매장(글로벌)
- 정렬 순서: 30(영구 콘텐츠)
- 활성화: 연중

## 슬라이드 관리

**슬라이드 목록 보기**:
- 모든 슬라이드를 이미지 미리보기, 제목, 범위, 상태로 표시
- 활성/비활성 필터
- 범위로 필터(모든 글로벌 슬라이드, 모든 그룹 슬라이드 등 보기)

**대량 활성화/비활성화**:
- 목록에서 여러 슬라이드를 선택
- 관리 액션을 사용하여 모든 슬라이드를 한 번에 활성화 또는 비활성화
- 계절 전환 시 유용(여름 슬라이드 모두 비활성화, 가을 슬라이드 모두 활성화)

**슬라이드 테스트**:
- 슬라이드를 생성하거나 업데이트한 후 POS 단말기로 이동
- 단말기를 유휴 상태로 두세요(거래 없음)
- 슬라이드가 캐러셀에 표시되는지 확인하세요
- 이미지 품질, 텍스트 오버레이 가독성, 타이밍을 확인하세요

**활성 슬라이드 업데이트**:
- 변경 사항은 다음 캐러셀 새로고침 시 적용됩니다(보통 30초 미만)
- 단말기를 재시작할 필요가 없습니다

## 팁

- **거리에 맞춘 디자인** - 고객은 2-6피트 떨어진 곳에서 화면을 보므로, 큰 글씨와 높은 대비를 사용하세요
- **간단한 메시지 유지** - 슬라이드는 10초 미만 표시되므로, 슬라이드당 하나의 명확한 메시지만 포함하세요
