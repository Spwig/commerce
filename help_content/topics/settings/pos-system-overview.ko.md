Spwig POS 시스템은 귀하의 매장을 현대적인 판매점 단말기와 함께 완전한 소매 솔루션으로 변환합니다. 무제한의 위치에 무제한의 단말기를 배포할 수 있으며, 연간 고정된 €499의 라이선스 요금으로 이용할 수 있습니다. 각 단말기는 오프라인에서도 작동하고 자동으로 동기화되며, 재고, 고객 데이터 및 결제 처리와의 원활한 통합이 가능한 프로그레시브 웹 앱(PWA)입니다. 관리 대시보드에서 모든 것을 관리할 수 있습니다 - 단말기 설정, 시프트 정산, 영수증 커스터마이징, 하드웨어 통합.

물리적 소매 매장, 팝업 스토어, 전시회 또는 고객이 온라인 대신 직접 구매하는 기타 환경에서 POS 시스템을 사용하세요.

## Spwig POS란 무엇인가요?

Spwig POS는 온라인과 물리적 위치 모두에서 판매하는 상인을 위한 완전히 통합된 판매점 시스템입니다. 제3자 POS 시스템과 달리 복잡한 통합이 필요하지 않으며, Spwig POS는 귀하의 플랫폼에 직접 구축되어 모든 판매 채널 간 완벽한 데이터 동기화를 보장합니다.

**주요 특징**:
- **무제한 단말기** - 추가 비용 없이 필요한 만큼의 단말기를 배포할 수 있습니다
- **오프라인 우선 아키텍처** - 인터넷 연결이 끊어졌을 때도 판매 처리를 계속합니다
- **프로그레시브 웹 앱** - 앱 스토어 설치 없이 모든 기기(태블릿, 컴퓨터, 전용 단말기)에서 브라우저를 통해 접근 가능
- **실시간 재고 동기화** - 재고 예약(15분 TTL)은 채널 간 과다 판매를 방지합니다
- **분할 결제 지원** - 거래당 여러 결제 방식(현금 + 카드 + 기프트 카드)을 수락합니다
- **하드웨어 통합** - ESC/POS 열전달 프린터, 바코드 스캐너, 현금 캐비닛, 고객 디스플레이
- **시프트 관리** - 개폐 현금 정산 및 차이 추적
- **다중 위치 지원** - 프랜차이즈 및 지역 관리를 위한 설정 상속이 가능한 매장 그룹

## 라이선스 및 활성화

**고정 가격 정책**: 연간 €499로 무제한 위치의 무제한 단말기를 커버합니다. 단말기당 요금, 거래 요금, 숨은 비용 없음.

**라이선스 형식**: `POS-XXXX-XXXX-XXXX-XXXX` (구매 후 제공)

**활성화**: **설정 > POS 라이선스**에 라이선스 키를 입력합니다. 시스템은 Spwig의 라이선스 서버와 검증하고 모든 POS 기능을 즉시 활성화합니다. 라이선스는 만료 후 14일의 유예 기간을 제공하여 결제 처리 지연을 허용합니다.

**받을 수 있는 내용**:
- 무제한 단말기 등록
- 무제한 직원 할당
- 모든 POS 기능(시프트, 현금 관리, 영수증 커스터마이징, 고객 디스플레이)
- 결제 제공업체 통합(Stripe Terminal 및 확장 가능한 제공업체 시스템)
- 하드웨어 통합 지원
- 라이선스 기간 동안의 업데이트 및 버그 수정

유효한 라이선스 없이는 POS 기능에 접근할 수 없습니다 - 단말기 쌍방향 인터페이스, 시프트 관리 및 POS 관리 페이지 모두 활성화가 필요합니다.

## 시스템 아키텍처

**프론트엔드** - React 18 프로그레시브 웹 앱:
- 서비스 워커 캐싱을 통한 오프라인 우선 (인터넷 없이 작동 가능)
- 빠른 로딩을 위한 Vite 빌드 시스템
- CSS 모듈 + 디자인 토큰 (매장 테마와 일관됨)
- 로컬 데이터 지속성에 사용되는 IndexedDB
- 10개 지원 언어 (영어, 중국어 간체/번체, 프랑스어, 독일어, 스페인어, 포르투갈어, 일본어, 러시아어, 아랍어)

**백엔드** - 백엔드 통합:
- 13개의 POS 모델 (POSTerminal, POSShift, CashMovement, ReceiptTemplate, PromoSlide 등)
- 단말기 작업을 위한 43개 이상의 REST API 엔드포인트
- TTL 관리와 함께 재고 예약 시스템
- 백그라운드 동기화를 위한 Celery 작업
- 결제 제공업체를 위한 암호화된 자격 증명 저장

**보안**:
- 8자리 코드를 통한 단말기 쌍방향 (서버에서 생성, 사용 후 만료)
- 직원 할당은 어떤 사용자가 어떤 단말기를 접근할 수 있는지 제어합니다
- 관리자 긴급 상황 시 원격 잠금/해제 기능
- 결제 제공업체의 암호화된 자격 증명
- 생체 인식 잠금 지원을 포함한 세션 기반 인증 (브라우저에 따라 다름)

## 시작 워크플로우

첫 번째 POS 단말기를 배포하려면 다음 5단계를 따르세요:

**단계 1: POS 라이선스 활성화**
- **설정 > POS 라이선스**로 이동
- 라이선스 키(`POS-XXXX-XXXX-XXXX-XXXX`) 입력
- 라이선스 검증(인터넷 연결 필요)
- 활성화 확인

**단계 2: 창고 생성**
- **카탈로그 > 창고**로 이동
- 귀하의 소매 위치를 나타내는 창고 생성
- 주소 및 연락처 정보 구성
- 이 창고는 POS 판매를 위한 물리적 재고를 추적합니다
