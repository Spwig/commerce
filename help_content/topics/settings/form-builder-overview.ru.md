---
title: Обзор конструктора форм
---

Конструктор форм создает пользовательские формы для сбора данных — контактные формы, опросы, заявки, регистрации и многое другое. Создавайте формы визуально с помощью перетаскивания полей, настраивайте правила проверки, включайте многоэтапные рабочие процессы и собирайте ответы с подробным анализом. Формы интегрируются с элементами Page Builder, встраиваются в любое место на вашем сайте. Все подачи хранятся в базе данных с полной метаданными (IP-адрес, браузер, время заполнения) для анализа и экспорта.

Используйте Конструктор форм, когда вам нужно собирать структурированные данные от клиентов, будь то простая контактная информация или сложные многостраничные заявки.

## Что такое Конструктор форм?

Конструктор форм — это визуальный инструмент с перетаскиванием для создания пользовательских форм без кода:

**Поддерживаемые типы форм**:
- Контактные формы (имя, электронная почта, сообщение)
- Опросы клиентов (рейтинги, отзывы, NPS)
- Регистрация продуктов (гарантия, поддержка)
- Заявки на работу (загрузка резюме, многоэтапные)
- Регистрация на мероприятия (информация о участнике, предпочтения)
- Заявки на услуги (подробные требования)
- Регистрация на рассылку (с флажками для предпочтений)

**Ключевые функции**:
- **22 типа полей** - Текст, электронная почта, телефон, загрузка файла, рейтинги, селекторы продуктов и многое другое
- **Многоэтапные формы** - Разделите длинные формы на логические шаги с отслеживанием прогресса
- **Условная логика** - Показ/скрытие полей на основе ответов пользователя
- **Правила проверки** - Обязательные поля, минимальная/максимальная длина, пользовательские регулярные выражения
- **Защита от спама** - Поля-ловушки или Google reCAPTCHA v3
- **Аналитика ответов** - Отслеживание времени заполнения, IP-адреса, браузера, источника перехода
- **Экспорт в CSV** - Скачайте все ответы для анализа в Excel/Google Sheets
- **Многоязычность** - Переведите метки и сообщения формы на все активные языки

## Создание первой формы

Перейдите в **Настройки > Страницы > Формы**, чтобы получить доступ к менеджеру форм:

**Шаг 1: Создать новую форму**
- Нажмите **+ Создать новую форму**
- Введите имя формы (внутренний идентификатор, не отображается клиентам)
- Введите заголовок формы (отображается как заголовок над формой)
- Необязательно: Добавьте описание (помощь текст, отображается ниже заголовка)

**Шаг 2: Добавить поля**
- Нажмите **Редактировать дизайн формы**, чтобы открыть визуальный конструктор
- Перетащите типы полей из левой боковой панели на холст
- Нажмите на поле, чтобы настроить его в правой панели
- Установите метку, заполнитель, текст помощи
- Включите/отключите статус обязательного
- Добавьте правила проверки

**Шаг 3: Настройка параметров формы**
- Установите текст кнопки отправки (по умолчанию: "Отправить")
- Настройте сообщение об успехе (отображается после отправки)
- Выберите защиту от спама (рекомендуется поле-ловушка)
- Включите "Требуется вход" при необходимости
- Включите "Многоэтапная форма" для сложных форм

**Шаг 4: Активировать форму**
- Включите статус **Активна**
- Только активные формы принимают подачи
- Сохраните форму

**Шаг 5: Использование в Page Builder**
- Добавьте элемент **Форма** на любую страницу
- Выберите свою форму из выпадающего списка
- Форма наследует стиль страницы
- Подачи отправляются на бэкенд автоматически

## Одностраничные и многоэтапные формы

**Одностраничные формы** (по умолчанию):
- Все поля отображаются сразу
- Прокрутите, чтобы увидеть все поля
- Кнопка отправки внизу
- Лучше подходит для: контактных форм, коротких опросов, простого сбора данных

**Многоэтапные формы**:
- Поля организованы в пронумерованные шаги
- Индикатор прогресса показывает текущий шаг
- Кнопки Назад/Далее
- Отправка только на последнем шаге
- Необязательно: Сохранение частичных ответов (режим черновика)
- Лучше подходит для: заявок на работу, регистрации, сложных опросов, процессов оформления заказа

**Включение многоэтапной формы**:
1. Включите "Многоэтапная форма" в настройках формы
2. Нажмите вкладку "Шаги" в правой панели
3. Добавьте шаг (например, "Личная информация", "Контактные данные", "Предпочтения")
4. Назначьте поля шагам с помощью выпадающего списка шагов при редактировании поля
5. Перетащите шаги, чтобы изменить порядок
6. Установите свойства шага: заголовок, описание, пропускаемый

**Преимущества многоэтапных форм**:
- Уменьшает отказ от формы (психологически: "только 3 вопроса на этой странице")
- Логическая группировка улучшает UX
- Индикатор прогресса мотивирует завершение
- Опциональное сохранение черновиков для длительных форм

## Объяснение настроек формы

**Базовые настройки**:
- **Внутреннее имя** - Как вы идентифицируете форму в админке (не видно клиентам)
- **Slug** - Дружественный для URL идентификатор (автоматически сгенерирован, используется в конечных точках API)
- **Заголовок формы** - Заголовок, отображаемый над формой
- **Описание** - Опциональный текст помощи, отображается ниже заголовка
- **Текст кнопки отправки** - Настройте метку кнопки (например, "Отправить сообщение", "Подать заявку")

**Сообщения**:
- **Сообщение об успехе** - Отображается после успешной отправки (по умолчанию: "Спасибо за вашу подачу!")
- **Сообщение об ошибке** - Отображается, если отправка не удалась (по умолчанию: "Произошла ошибка. Пожалуйста, попробуйте еще раз.")

**Безопасность и доступ**:
- **Активна** - Только активные формы принимают подачи (неактивные формы показывают "Форма недоступна")
- **Требуется вход** - Ограничить только аутентифицированными пользователями (анонимные пользователи видят приглашение войти)

**Защита от спама**:
- **Нет** - Нет защиты (не рекомендуется, боты будут спамить)
- **Поле-ловушка** - Невидимое поле для захвата ботов (рекомендуется для большинства продавцов)
- **Google reCAPTCHA v3** - Требуется ключ сайта и секретный ключ от Google (самая сильная защита)

**Расширенные функции**:
- **Многоэтапная форма** - Включите пошаговый рабочий процесс
- **Сохранение частичных ответов** - Разрешите пользователям сохранять прогресс и возобновить позже (только для многоэтапных форм)

## Варианты защиты от спама

**Поле-ловушка (рекомендуется)**:
- Добавляется невидимое поле в форму
- Боты заполняют его (люди не могут его увидеть)
- Подачи с заполненным полем-ловушкой отвергаются
- Нет необходимости в настройке
- Нет раздражающих CAPTCHA для пользователей
- Эффективно против 95%+ спам-ботов

**Google reCAPTCHA v3**:
- Невидимый фоновый рейтинг (0.0-1.0)
- Нет вызова "нажмите на светофор"
- Требуется настройка:
  1. Создайте аккаунт на google.com/recaptcha/admin
  2. Сгенерируйте ключ сайта и секретный ключ
  3. Введите ключи в настройках конструктора форм
- Более надежная защита, чем поле-ловушка
- Используйте, когда поле-ловушка недостаточна

**Нет**:
- Нет защиты от спама
- Используйте только для внутренних форм или тестирования
- Публичные формы будут подвержены сильному спаму

## Управление ответами на форму

Просмотрите все подачи в **Настройки > Страницы > Формы > [Название формы] > Ответы**:

**Просмотр списка ответов**:
- Статус: черновик, отправлено, завершено
- Отправитель: электронная почта (если вошел в систему) или "Анонимный"
- IP-адрес и местоположение (если включена GeoIP)
- Дата/время отправки
- Время заполнения (секунды)

**Детали ответа**:
- Все значения полей с метками
- Метаданные: браузер, источник перехода, язык
- Отслеживание прогресса (многоэтапные): текущий шаг, завершенные шаги
- Результаты действий (если форма запускает действия)

**Фильтрация ответов**:
- Фильтрация по форме, статусу, диапазону дат
- Поиск по электронной почте отправителя или IP-адресу
- Сортировка по дате отправки, времени заполнения

**Экспорт ответов**:
- Нажмите кнопку **Экспортировать в CSV**
- Скачивается файл {form-slug}_responses_{date}.csv
- Заголовок: Submitted At, User, IP, Status, [Метки полей]
- Один ответ на строку
- Откройте в Excel, Google Sheets или инструментах анализа данных

## Использование форм на страницах

**Встраивание форм**:
1. Откройте страницу в Page Builder
2. Добавьте элемент **Форма** из панели элементов
3. Выберите форму из выпадающего списка
4. Настройте стиль контейнера формы (фон, отступы, граница)
5. Сохраните и опубликуйте страницу

**Форма отображается с**:
- Заголовком и описанием формы (из настроек формы)
- Все поля в порядке (одностраничные) или текущий шаг (многоэтапные)
- Кнопка отправки с пользовательским текстом
- Сообщения об успехе/ошибке после отправки

**Наследование стиля**:
- Формы наследуют стиль темы страницы
- Кнопки используют стили кнопок темы
- Поля ввода используют стили ввода темы
- Можно добавить пользовательский класс CSS к полям для специфического стиля

## Интерфейс конструктора форм

**Левая боковая панель - библиотека полей**:
- Организована по категориям (Текст, Выбор, Рейтинг, Расширенные)
- Перетащите поле на холст или нажмите, чтобы добавить
- Поиск для быстрого поиска типов полей

**Основной холст - редактор полей**:
- Обработчик перетаскивания (≡) для сортировки полей
- Нажмите на поле, чтобы выбрать и отредактировать
- Кнопка удаления (×) для каждого поля
- Визуальный предварительный просмотр поля в настроенной форме
- Пустое состояние с инструкциями по зоне перетаскивания

**Правая боковая панель - панель свойств**:
- **Вкладка настроек формы** - Базовая информация, сообщения, защита от спама
- **Вкладка настроек поля** - Настройка выбранного поля (метка, проверка и т.д.)
- **Вкладка шагов** - Управление шагами (только для многоэтапных форм)
- **Вкладка условных правил** - Добавление логики показа/скрытия на основе ответов

**Функции панели инструментов**:
- **Отменить/Повторить** - Полный исторический журнал редактирования
- **Предварительный просмотр** - Тестирование функциональности формы
- **Сохранить** - Автоматически сохраняется каждые 3 секунды во время редактирования
- **Переводы** - Перевод текста формы на другие языки

## Общие примеры форм

**Контактная форма**:
- Поля: Полное имя (обязательное), Электронная почта (обязательное), Телефон, Сообщение (обязательное)
- Кнопка отправки: "Отправить сообщение"
- Успех: "Спасибо за обращение! Мы ответим в течение 24 часов."

**Опрос по отзыву о продукте**:
- Шаг 1: Оценка звездой, согласие по шкале Лайкerta
- Шаг 2: Оценка NPS, предложения по улучшению
- Условия: Если оценка < 3, требуется отзыв по улучшению

**Заявка на работу**:
- Шаг 1: Личная информация (имя, электронная почта, телефон)
- Шаг 2: Опыт (загрузка резюме, годы опыта, рекомендации)
- Шаг 3: Доступность (дата начала, ожидания зарплаты)
- Включено сохранение частичных ответов (кандидаты могут возобновить позже)

**Регистрация на рассылку с предпочтениями**:
- Электронная почта (обязательная)
- Группа флажков: Интересы (Продукты, Продажи, Обновления блога)
- Включена reCAPTCHA (предотвращает фальшивую регистрацию)

## Советы

- **Начните с одностраничной** - Добавьте многоэтапную только если форма содержит более 10 полей
- **Используйте поле-ловушку сначала** - Только обновляйте до reCAPTCHA, если спам продолжается
- **Проверьте перед публикацией** - Используйте режим предварительного просмотра, чтобы проверить проверку и поток
- **Регулярно экспортируйте** - Скачивайте CSV-файл с ответами еженедельно для резервного копирования
- **Отслеживайте время заполнения** - Если среднее время >5 минут, форма может быть слишком длинной
- **Используйте условную логику** - Скрывайте несущественные поля, чтобы уменьшить восприятие длины формы
- **Включите частичное сохранение для длительных форм** - Уменьшает отказ от многоэтапных заявок
- **Переведите метки форм** - Используйте встроенную систему перевода для многоязычных сайтов
- **Требуйте вход для чувствительных данных** - Предотвращает анонимный спам, связывает подачи с учетными записями пользователей
- **Держите сообщения об успехе конкретными** - "Мы ответим в течение 24 часов" лучше, чем "Спасибо"
