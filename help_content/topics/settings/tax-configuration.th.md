title: การตั้งค่าภาษี
---

อัตราภาษีกำหนดภาษีขาย ภาษีมูลค่าเพิ่ม และภาษีอื่น ๆ ที่เกี่ยวข้องกับการซื้อขายตามตำแหน่งของลูกค้าและประเภทสินค้า - ตั้งค่าอัตราภาษีระดับประเทศ/รัฐ/เมือง พร้อมข้อยกเว้นประเภทสินค้าที่เป็นไปได้ Spwig รองรับภาษีแบบผสม (ภาษีที่เกิดจากภาษีอื่น) การเลือกอัตราตามลำดับความสำคัญ และกลุ่มการตั้งค่าภาษีล่วงหน้าเพื่อตั้งค่าระบบภาษีในภูมิภาคต่าง ๆ ได้รวดเร็ว (ภาษีมูลค่าเพิ่มของสหภาพยุโรป ภาษีขายของสหรัฐอเมริกา) อัตราภาษีสามารถยกเว้นประเภทสินค้าที่เฉพาะเจาะจง (อาหาร หนังสือ สินค้าดิจิทัล) หรือหมวดหมู่เพื่อปฏิบัติตามกฎหมายภาษีท้องถิ่น

ใช้การตั้งค่าภาษีเพื่อให้แน่ใจว่าปฏิบัติตามข้อกำหนดเกี่ยวกับการเก็บภาษีในเขตพื้นที่ที่คุณขายสินค้า

## การตั้งค่าอัตราภาษี

แต่ละอัตราภาษีกำหนด:

**ขอบเขตทางภูมิศาสตร์**:
- ประเทศ (จำเป็น)
- รัฐ/จังหวัด (ไม่จำเป็น)
- เมือง (ไม่จำเป็น)
- รูปแบบรหัสไปรษณีย์ (ไม่จำเป็น, regex)

**รายละเอียดอัตรา**:
- **อัตราภาษี**: เปอร์เซ็นต์ (เช่น 8.5%)
- **ชื่อ**: ชื่อที่แสดง (เช่น "ภาษีขายแคลิฟอร์เนีย")
- **ลำดับความสำคัญ**: ลำดับความสำคัญที่สูงกว่าจะชนะเมื่อมีอัตราหลายอัตราตรงกัน
- **เปิดใช้งาน**: เปิด/ปิดโดยไม่ต้องลบ

**ข้อยกเว้น**:
- **ประเภทสินค้าที่ยกเว้น**: สินค้าดิจิทัล สินค้าทางกายภาพ บริการ
- **หมวดหมู่ที่ยกเว้น**: หมวดหมู่สินค้าเฉพาะ (อาหาร หนังสือ ยา)

**ภาษีแบบผสม**:
- **เป็นภาษีแบบผสม**: ใช้อัตรานี้เพิ่มบนภาษีก่อนหน้า (ภาษีที่เกิดจากภาษีอื่น)
- ตัวอย่าง: ภาษี PST ของควิเบก ถูกคำนวณเพิ่มบน GST

## สถานการณ์ภาษีที่พบบ่อย

### ภาษีขายของสหรัฐอเมริกา (ระดับรัฐ)

```
ชื่อ: ภาษีขายแคลิฟอร์เนีย
ประเทศ: สหรัฐอเมริกา
รัฐ: CA
อัตรา: 7.25%
ลำดับความสำคัญ: 50
```

### ภาษีมูลค่าเพิ่มของสหภาพยุโรป (ระดับประเทศ)

```
ชื่อ: VAT สหราชอาณาจักร
ประเทศ: GB
อัตรา: 20%
ลำดับความสำคัญ: 50

ชื่อ: VAT เยอรมนี
ประเทศ: DE
อัตรา: 19%
ลำดับความสำคัญ: 50
```

### GST/PST ของแคนาดา (แบบผสม)

```
อัตรา 1: GST ระดับกลาง
ประเทศ: CA
อัตรา: 5%
ลำดับความสำคัญ: 100
เป็นภาษีแบบผสม: ไม่

อัตรา 2: PST ควิเบก
ประเทศ: CA
รัฐ: QC
อัตรา: 9.975%
ลำดับความสำคัญ: 50
เป็นภาษีแบบผสม: ใช่ (ใช้กับยอดรวม + GST)
```

### ภาษีระดับเมือง

```
ชื่อ: ภาษีขายซีแอตเทิล
ประเทศ: สหรัฐอเมริกา
รัฐ: WA
เมือง: ซีแอตเทิล
อัตรา: 10.1%
ลำดับความสำคัญ: 100
```

## ข้อยกเว้นภาษี

### ข้อยกเว้นประเภทสินค้า

ยกเว้นประเภทสินค้าทั้งหมด:

- **สินค้าดิจิทัล**: ซอฟต์แวร์ หนังสืออิเล็กทรอนิกส์ เพลง
- **สินค้าทางกายภาพ**: สินค้าที่มีรูปธรรม
- **บริการ**: การให้คำปรึกษา การติดตั้ง

ตัวอย่าง: VAT ของสหภาพยุโรปไม่ใช้กับสินค้าดิจิทัลสำหรับผู้บริโภค (ในบางกรณี)

### ข้อยกเว้นหมวดหมู่

ยกเว้นหมวดหมู่สินค้าเฉพาะ:

- อาหารและของชำ (มักยกเว้นหรือมีอัตราลดลง)
- หนังสือและวัสดุการศึกษา
- วัสดุทางการแพทย์และยา
- เสื้อผ้า (บางเขตพื้นที่)

การตั้งค่า:
```
ชื่อ: ภาษีขายแคลิฟอร์เนีย
อัตรา: 7.25%
หมวดหมู่ที่ยกเว้น: ["อาหารและเครื่องดื่ม", "ยาตามใบสั่ง"]
```

## กลุ่มการตั้งค่าภาษีล่วงหน้า

โหลดการตั้งค่าภาษีทั่วไปได้รวดเร็ว:

**กลุ่มการตั้งค่าภาษีขายของสหรัฐอเมริกา**:
- ทั้ง 50 รัฐ + DC
- อัตราภาษีระดับรัฐ
- อัปเดตอัตโนมัติเมื่ออัตราเปลี่ยน

**กลุ่มการตั้งค่า VAT ของสหภาพยุโรป**:
- ประเทศสมาชิกทั้งหมด 27 ประเทศในสหภาพยุโรป
- อัตรา VAT มาตรฐาน
- ตรรกะการเก็บ VAT แบบผู้ซื้อขายต่อผู้ซื้อขาย (B2B) สำหรับการกลับรายการ

**เพื่อใช้กลุ่มการตั้งค่า**:
1. ตั้งค่า > ตะกร้าสินค้า > การตั้งค่าภาษีล่วงหน้า
2. เลือกกลุ่มการตั้งค่า (เช่น "ภาษีขายสหรัฐอเมริกา 2026")
3. คลิก "โหลดการตั้งค่า"
4. อัตราภาษีจะถูกนำเข้าอัตโนมัติ
5. ปรับแต่งตามที่จำเป็น

---

## การแก้ไขลำดับความสำคัญ

เมื่อมีอัตราหลายอัตราตรงกัน ลำดับความสำคัญที่สูงกว่าจะชนะ:

ตัวอย่าง:
```
ลูกค้าในซีแอตเทิล รัฐ WA:

อัตรา A: ภาษีกลางสหรัฐฯ (ลำดับความสำคัญ 1) - 0%
อัตรา B: รัฐวอชิงตัน (ลำดับความสำคัญ 50) - 6.5%
อัตรา C: เมืองซีแอตเทิล (ลำดับความสำคัญ 100) - 3.6%

ผลลัพธ์: อัตราซีแอตเทิล (รวม 10.1%) ถูกใช้
```

## ตัวเลือกการแสดงภาษี

ตั้งค่าใน ตั้งค่า > ตะกร้าสินค้า > การตั้งค่าภาษี:

- **ราคารวมภาษี**: แสดงราคาพร้อมภาษี (สไตล์สหภาพยุโรป)
- **แสดงภาษีแยกต่างหาก**: แสดงภาษีเป็นรายการ (สไตล์สหรัฐอเมริกา)
- **ปัดเศษภาษี**: ต่อรายการหรือต่อคำสั่งซื้อ
- **ชื่อภาษี**: ปรับแต่งป้ายกำกับ ("VAT", "ภาษีขาย", "GST")

---

## การทดสอบการตั้งค่าภาษี

ก่อนเปิดใช้งาน:

1. สร้างคำสั่งซื้อทดสอบจากเขตพื้นที่ต่าง ๆ
2. ตรวจสอบว่าอัตราภาษีที่ถูกต้องถูกใช้
3. ตรวจสอบข้อยกเว้นสำหรับหมวดหมู่ที่ถูกยกเว้น
4. ทดสอบการคำนวณภาษีแบบผสม
5. ตรวจสอบรายการภาษีในใบแจ้งหนี้

---

## หมายเหตุเกี่ยวกับการปฏิบัติตามกฎหมาย

- **สหรัฐอเมริกา**: กฎเกณฑ์ Nexus กำหนดให้เก็บภาษีในรัฐที่คุณมีสถานที่ตั้งทางกายภาพหรือ Nexus เศรษฐกิจ
- **สหภาพยุโรป**: บริษัทที่จดทะเบียน VAT ต้องเก็บ VAT จากลูกค้าในสหภาพยุโรป
- **แคนาดา**: GST/HST/PST แตกต่างกันไปตามรัฐรัฐ
- **ปรึกษาผู้เชี่ยวชาญด้านภาษี**: กฎหมายภาษีเปลี่ยนแปลงบ่อยครั้ง ตรวจสอบข้อกำหนดปัจจุบัน

## เคล็ดลับ

- **ใช้กลุ่มการตั้งค่าภาษีล่วงหน้า** - รวดเร็วกว่าการป้อนด้วยตนเอง ได้รับการอัปเดตอัตโนมัติ
- **ติดตามเกณฑ์ Nexus** - ติดตามยอดขายตามรัฐสำหรับ Nexus เศรษฐกิจของสหรัฐอเมริกา
- **ตั้งลำดับความสำคัญให้ถูกต้อง** - เมือง > รัฐ > ประเทศ
- **ทดสอบภาษีแบบผสม** - ตรวจสอบให้แน่ใจว่าการคำนวณตรงกับจำนวนที่คาดไว้
- **อัปเดตทุกปี** - อัตราภาษีเปลี่ยนแปลง ตรวจสอบทุกเดือนมกราคม
- **บันทึกข้อยกเว้น** - จัดเก็บเอกสารว่าเหตุใดหมวดหมู่ถึงถูกยกเว้น
- **ใช้ชื่อที่อธิบายได้ชัดเจน** - "ภาษีขายแคลิฟอร์เนีย 2026" ดีกว่า "ภาษี 1"
- **เปิดใช้งานภาษีโดยค่าเริ่มต้น** - ปลอดภัยกว่าการลืมใช้ภาษี
