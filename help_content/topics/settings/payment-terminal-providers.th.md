---

ผู้ให้บริการอุปกรณ์รับชำระเงินช่วยให้คุณสามารถรับชำระเงินด้วยบัตรเครดิตและบัตรเดบิตได้ที่เครื่อง POS ของคุณ Stripe Terminal เป็นผู้ให้บริการหลักที่ได้รับการสนับสนุน ซึ่งมีเครื่องอ่านบัตรรุ่นใหม่ (S700, WisePOS E, P400) อัตราการประมวลผลที่มีการแข่งขัน และการผสานรวมที่ราบรื่น ตั้งค่าบัญชีผู้ให้บริการด้วยคุณสมบัติ API ตรวจสอบสถานะการเชื่อมต่อแบบเรียลไทม์ และจัดการผู้ให้บริการหลายรายหากคุณดำเนินการในภูมิภาคต่างๆ ระบบผู้ให้บริการสามารถขยายได้—ผู้ประมวลผลการชำระเงินเพิ่มเติมสามารถผสานรวมได้ผ่านเฟรมเวิร์กผู้ให้บริการหาก Stripe Terminal ไม่ได้ดำเนินการในตลาดของคุณ

ใช้ผู้ให้บริการเพื่อรับชำระเงินด้วยบัตรอย่างปลอดภัย ติดตามสถานะการประมวลผลการชำระเงิน และจัดการการกำหนดอุปกรณ์อ่านบัตรในเครื่องต่างๆ

## ภาพรวมผู้ให้บริการชำระเงิน

ผู้ให้บริการชำระเงินคือบริการของบุคคลที่สามที่ดำเนินการชำระเงินด้วยบัตรแทนคุณ:

**หน้าที่ของผู้ให้บริการ**:
- อนุมัติการทำธุรกรรมบัตรแบบเรียลไทม์
- สื่อสารกับอุปกรณ์อ่านบัตรทางกายภาพ
- จัดการความปลอดภัยของการชำระเงิน (ความสอดคล้องกับ PCI, การเข้ารหัส)
- โอนเงินเข้าบัญชีธนาคารของคุณ (การตั้งถิ่นฐาน)
- ให้รายงานการทำธุรกรรมและการจัดการข้อพิพาท

**บทบาทของ Spwig**:
- ส่งคำขอการชำระเงินไปยังผู้ให้บริการที่ตั้งค่าไว้
- จัดเก็บคุณสมบัติผู้ให้บริการที่เข้ารหัสแล้ว
- ตรวจสอบสถานะการเชื่อมต่อ
- จับคู่อุปกรณ์อ่านกับเครื่องต่างๆ
- บันทึกผลลัพธ์การชำระเงินในคำสั่งซื้อ

## Stripe Terminal (ผู้ให้บริการหลัก)

Stripe Terminal เป็นผู้ให้บริการชำระเงินที่แนะนำสำหรับผู้ค้าส่วนใหญ่:

**คุณสมบัติ**:
- อุปกรณ์อ่านบัตรชิป EMV รุ่นใหม่
- รองรับการชำระเงินแบบไม่สัมผัส (NFC) (Apple Pay, Google Pay, บัตรที่แตะเพื่อชำระเงิน)
- การจัดการข้อพิพาทแบบผสานรวม
- การอนุมัติแบบเรียลไทม์
- API ที่เป็นมิตรต่อการพัฒนา
- ใช้งานได้ใน 40+ ประเทศ

**ราคา** (ตามข้อมูลปี 2024 โปรดตรวจสอบอัตราที่ถูกต้อง):
- ค่าธรรมเนียมการทำธุรกรรม: 2.7% + $0.05 ต่อการทำธุรกรรมแบบตัวต่อตัว (สหรัฐอเมริกา)
- ไม่มีค่าธรรมเนียมรายเดือน ไม่มีค่าธรรมเนียมการตั้งค่า ไม่มีค่าธรรมเนียมความสอดคล้องกับ PCI
- อุปกรณ์อ่านบัตรฮาร์ดแวร์: ซื้อครั้งเดียว ($59-$299 ขึ้นอยู่กับรุ่น)

**ภูมิภาคที่รองรับ**:
- สหรัฐอเมริกา แคนาดา สหราชอาณาจักร ยุโรป ประเทศออสเตรเลีย สิงคโปร์ และอื่นๆ อีกมากมาย
- ตรวจสอบความพร้อมใช้งานของ Stripe: https://stripe.com/terminal

**อุปกรณ์อ่านที่รองรับ**:
- BBPOS WisePOS E (อุปกรณ์ Android ที่รวมทุกอย่างไว้ในตัว)
- Stripe Reader S700 (อุปกรณ์อ่านที่อยู่บนเคาน์เตอร์)
- Verifone P400 (อุปกรณ์อ่านรุ่นเก่า แต่ยังได้รับการสนับสนุน)

## การตั้งค่า Stripe Terminal

**ขั้นตอนที่ 1: สร้างบัญชี Stripe**
- ลงทะเบียนที่ stripe.com
- สมบูรณ์การตรวจสอบธุรกิจ (บัญชีธนาคาร, หมายเลขประจำตัวภาษี)
- เปิดใช้งานการชำระเงิน

**ขั้นตอนที่ 2: เปิดใช้งาน Stripe Terminal**
- ในแดชบอร์ด Stripe ให้ไปที่ **สินค้า > Terminal**
- คลิก **เริ่มต้นใช้งาน**
- ยอมรับข้อตกลงในการใช้งานของ Terminal

- Stripe Terminal ต้องการ
