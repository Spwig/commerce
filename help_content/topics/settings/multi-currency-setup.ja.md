---
title: マルチ通貨の設定
---

マルチ通貨機能により、顧客は自身の希望する通貨で商品を閲覧し、チェックアウトを完了できます。価格は、接続されたプロバイダーまたは手動で定義された為替レートを使用して、ベース通貨から自動的に変換されます。

## 開始する前に

マルチ通貨を有効にする前に、以下のものを用意する必要があります。

1. **アクティブな為替レートプロバイダー** - **設定 > マルチ通貨タブ > 為替レートダッシュボード**にアクセスし、少なくとも1つのプロバイダー（例: Open Exchange Rates、Fixer.io、または ExchangeRate-API）を接続してください。プロバイダーはアクティブであり、レートの同期が行われている必要があります。
2. **少なくとも2つの通貨** - ベース通貨に加えて、サポートしたい1つ以上の追加通貨。

## マルチ通貨の有効化

**設定 > マルチ通貨**に移動し、**マルチ通貨を有効にする**をチェックしてください。有効化後、以下のオプションを構成してください。

| 設定 | 説明 |
|---------|-------------|
| **通貨選択モード** | 顧客が通貨を選択する方法。*Auto*は位置情報から自動的に検出し、*Manual*はスイッチャーから選択可能にし、*Both*は2つのアプローチを組み合わせます。 |
| **通貨スイッチャーを表示** | 店舗のフロントエンドに通貨セレクターを表示し、顧客が手動で通貨を変更できるようにします。 |
| **スイッチャーの位置** | 通貨スイッチャーが表示される場所（ヘッダー、フッター、またはサイドバー）。 |
| **為替レート情報を表示** | 顧客に価格がベース通貨からの近似変換であることを通知します。 |
| **ロケール形式を有効にする** | 数値と通貨記号を、各顧客のロケールに応じてフォーマットします（例: ヨーロッパ形式の1.234,56）。 |

## チェックアウトモード

チェックアウトでマルチ通貨がどのように動作するかを選択してください。

| モード | 説明 |
|------|-------------|
| **フルマルチ通貨** | 顧客は選択した通貨で商品を閲覧し、カートに追加し、支払いを行います。為替レートはチェックアウト時にロックされ、注文に記録されます。これはデフォルトです。 |
| **表示のみ** | 顧客の通貨で価格を表示し、便利さを提供しますが、カートおよび支払いは常にベース通貨で処理されます。チェックアウト時に、顧客はベース通貨での実際の請求額と、近似変換額の両方を表示されます。 |

**表示のみ**は、支払いプロバイダーがベース通貨のみをサポートしている場合、または為替レートのリスクを完全に回避したい場合に役立ちます。顧客は閲覧時にローカライズされた価格を表示し、自身の通貨でのコスト感覚を得られます。

## 為替レートの同期間隔

接続されたプロバイダーから新しいレートを取得する頻度を制御します。

| 間隔 | 説明 |
|----------|-------------|
| **リアルタイム** | 15分ごと。国際的な高額取引が多い店舗に最適です。 |
| **時間ごと** | 1時間ごと。新鮮さとAPI使用量のバランスが取れています。 |
| **日次** | 1日ごと。ほとんどの店舗に適しています。これはデフォルトです。 |
| **週次** | 1週間ごと。価格が安定している店舗に適しています。 |
| **月次 / 季次** | 為替レートをほとんど変更しない店舗向けに、更新頻度が少ないです。 |
| **手動のみ** | レートは自動的に取得されません。すべてのレートを手動で管理します。 |

同期間隔は、バックグラウンドタスクがプロバイダーからレートを取得する頻度に影響を与えます。同期の間は、キャッシュされたレートが使用されます。即時同期が必要な場合は、**為替レートダッシュボード**の**今すぐ同期**ボタンまたは**手動為替レート**ページの**プロバイダーから同期**を使用してください。

## 手動で設定した為替レート

手動で設定した為替レートにより、特定の通貨ペアに対して正確な変換レートを設定できます。これはプロバイダーから取得したレートよりも優先され、価格の完全な制御が可能です。

**為替レート > 手動為替レート**に移動して管理してください。

### 手動でレートを設定する

**レートを追加**をクリックして、通貨ペアのレートを作成します。ベース通貨、ターゲット通貨、およびレートを指定します。例えば、USD/EURを0.92に設定すると、1 USD = 0.92 EURとなります。

### プロバイダーから同期する

**プロバイダーから同期**をクリックして、接続されたプロバイダーの最新レートから手動レートを自動的に取得します。

これは、サポートされているすべての通貨に対して手動のレートを作成し、微調整を開始するための出発点となります。

- **最初は小規模な注文でテストする** - 外貨でテスト注文を置き、チェックアウトフローを確認し、為替レートが正しく適用されていることを確認してください。
- **為替レートを定期的に確認する** - 为替レートダッシュボードを定期的に確認し、プロバイダーがレートを同期しており、それらが合理的であることを確認してください。
- **変動の大きい通貨にはマーカップを考慮する** - 高い変動性を持つ通貨をサポートしている場合、わずかに高いマーカップ（2〜3％）で利益率を保護できます。
- **主要な通貨から始める** - EUR、GBP、JPY、CAD、AUDなどの広く使われている通貨から始め、顧客の需要に応じて拡張してください。
- **支払いプロバイダーの互換性を確認する** - すべての支払いプロバイダーがすべての通貨をサポートしているわけではありません。

支払いプロバイダーのドキュメントを確認し、彼らが処理する通貨がどれかを確認してください。
- **多通貨チェックアウトを扱っているか不確実な場合は、表示のみモードを使用する** - 支払いプロバイダーが多通貨チェックアウトを扱っているか確信がない場合は、最初は表示のみモードから始めてください。

後でフル多通貨モードに切り替えることもできます。
- **キャンペーン期間前にレートを固定する** - キャンペーンを実施する場合、キャンペーン中に価格が一貫していることを保証するために、事前に為替レートを固定してください。
