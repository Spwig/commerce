---
title: การตั้งค่าการจัดส่ง
---

คู่มือนี้อธิบายวิธีกำหนดค่าการจัดส่งสำหรับร้านของคุณ — ตั้งแต่การตั้งค่าวิธีการจัดส่งพื้นฐานไปจนถึงการเชื่อมต่อกับผู้ให้บริการจัดส่งจริงเพื่อให้ได้ราคาแบบเรียลไทม์

## ภาพรวมการจัดส่ง

Spwig มีสองวิธีในการจัดส่ง:

- **วิธีการจัดส่งแบบมืออาชีพ** — วิธีที่มีราคาคงที่ที่คุณกำหนดเอง (เช่น "Standard Shipping — $5.99")
- **การเชื่อมต่อผู้ให้บริการจัดส่ง** — ราคาแบบเรียลไทม์จากผู้ให้บริการเช่น FedEx, UPS และ DHL

คุณสามารถใช้ได้ทั้งวิธีหรือผสมผสานกัน

## วิธีการจัดส่ง

วิธีการจัดส่งคือตัวเลือกที่ลูกค้าจะเห็นตอนชำระเงิน คลิกที่ **Orders > Shipments** ทางด้านซ้ายเพื่อจัดการ

![Shipping methods](/static/core/admin/img/help/setup-shipping/shipping-methods.webp)

### การสร้างวิธีการจัดส่ง

1. คลิกที่ **Add Shipping Method**
2. กรอกข้อมูล:
   - **Name** — ชื่อที่แสดงให้ลูกค้าเห็น (เช่น "Express Delivery")
   - **Description** — คำอธิบายสั้นๆ ของบริการ
   - **Price** — ค่าจัดส่งคงที่
   - **Estimated Delivery** — ระยะเวลาการจัดส่งประมาณ (เช่น "3-5 business days")
3. คลิกที่ **Save**

## พื้นที่การจัดส่ง

พื้นที่การจัดส่งกำหนดภูมิศาสตร์ที่วิธีการจัดส่งของคุณใช้ได้ คลิกที่ส่วน **Shipping Zones** เพื่อจัดการ

![Shipping zones](/static/core/admin/img/help/setup-shipping/shipping-zones.webp)

### การสร้างพื้นที่

1. คลิกที่ **Add Shipping Zone**
2. ตั้งค่าพื้นที่:
   - **Zone Name** — ชื่อภายใน (เช่น "US Domestic", "Europe")
   - **Countries** — เลือกประเทศที่อยู่ในพื้นที่นี้
   - **States/Regions** — จำกัดเฉพาะรัฐได้ตามต้องการ
   - **Postal Code Patterns** — ใช้รูปแบบเช่น "9*" เพื่อเลือกพื้นที่เฉพาะ
3. กำหนดวิธีการจัดส่งให้กับพื้นที่นี้
4. คลิกที่ **Save**

### ลำดับความสำคัญของพื้นที่

เมื่อที่อยู่ของลูกค้าตรงกับพื้นที่หลายพื้นที่ พื้นที่ที่เฉพาะเจาะจงที่สุดจะมีความสำคัญสูงสุด พื้นที่ที่มีการกำหนดระดับรัฐจะมีความสำคัญมากกว่าพื้นที่ที่กำหนดระดับประเทศ

## การเชื่อมต่อผู้ให้บริการจัดส่ง

เชื่อมต่อกับผู้ให้บริการจัดส่งเพื่อเสนอราคาที่คำนวณได้แบบเรียลไทม์ตอนชำระเงิน

![Shipping carriers](/static/core/admin/img/help/setup-shipping/shipping-carriers.webp)

### ผู้ให้บริการที่มีอยู่

สำรวจและติดตั้งผู้ให้บริการจัดส่งจากตลาด

![Shipping providers](/static/core/admin/img/help/setup-shipping/shipping-providers.webp)

ผู้ให้บริการที่รองรับได้แก่:

- **FedEx** — Ground, Express, International
- **UPS** — Ground, 2-Day, Overnight, Worldwide
- **DHL** — Express, eCommerce
- **USPS** — Priority, First Class, Media Mail
- และอื่นๆ อีกมากมายที่มีใน Marketplace

### การตั้งค่าผู้ให้บริการ

1. ไปที่หน้าผู้ให้บริการจัดส่งและคลิกที่ **Install** บนผู้ให้บริการที่คุณต้องการ
2. ทำตามวิซาร์ดการตั้งค่า:
   - **Step 1** — ตรวจสอบรายละเอียดผู้ให้บริการ
   - **Step 2** — ตั้งค่าทั่วไป
   - **Step 3** — ใส่ข้อมูล API (เลขบัญชี, คีย์ API, ฯลฯ)
   - **Step 4** — เปิดใช้งานบริการเฉพาะ (Ground, Express, ฯลฯ)
   - **Step 5** — ทดสอบการเชื่อมต่อ
3. เมื่อเชื่อมต่อแล้ว ราคาของผู้ให้บริการจะปรากฏโดยอัตโนมัติตอนชำระเงิน

### ข้อมูลการเข้าถึง API

แต่ละผู้ให้บริการต้องการบัญชี API:

- **FedEx** — ลงทะเบียนที่ FedEx Developer Portal สร้างแอปและคัดลอกคีย์ API และรหัสลับของคุณ
- **UPS** — ลงทะเบียนที่ UPS Developer Kit ขอรหัสการเข้าถึง
- **DHL** — ติดต่อ DHL เพื่อขอข้อมูล API ผ่านพอร์ทัลธุรกิจของพวกเขา

## กฎการจัดส่ง

สร้างกฎขั้นสูงเพื่อควบคุมว่าเมื่อใดและอย่างไรที่วิธีการจัดส่งจะถูกเสนอ

### กฎที่พบบ่อย

- **การจัดส่งฟรีเมื่อสั่งซื้อเกิน $50** — ตั้งค่าขั้นต่ำของตะกร้าเพื่อให้ได้การจัดส่งฟรี
- **อัตราคงที่สำหรับคำสั่งซื้อที่มีน้ำหนักเบา** — อัตราคงที่เมื่อสั่งซื้อที่มีน้ำหนักต่ำกว่าเกณฑ์
- **ปิดการใช้งานการจัดส่งแบบเร่งด่วนในพื้นที่ห่างไกล** — ซ่อนตัวเลือกการจัดส่งแบบเร่งด่วนตามรหัสไปรษณีย์
- **เพิ่มค่าที่เป็นเปอร์เซ็นต์** — เพิ่มค่าใช้จ่ายเป็นเปอร์เซ็นต์ของอัตราผู้ให้บริการ

### การสร้างกฎ

1. ไปที่ส่วนกฎการจัดส่ง
2. คลิกที่ **Add Rule**
3. ตั้งค่าเงื่อนไข (ยอดรวมตะกร้า, น้ำหนัก, พื้นที่, ฯลฯ)
4. กำหนดการกระทำ (ปรับอัตรา, ซ่อนวิธี, เปิดใช้งานการจัดส่งฟรี)
5. บันทึกรูปแบบ

กฎจะถูกประเมินตามลำดับ — กฎแรกที่ตรงกับเงื่อนไขจะถูกใช้งาน

## การจัดส่งฟรี

### การจัดส่งฟรีทั่วทั้งร้าน

เปิดใช้งานการจัดส่งฟรีทั่วทั้งร้านใน **Settings > Store Settings**:

- เปิดใช้งาน **Free Shipping**
- ตั้งค่าขั้นต่ำของคำสั่งซื้อได้ตามต้องการ
- เลือกภูมิภาคที่เหมาะสม

### การจัดส่งฟรีแบบโปรโมชัน

สร้างข้อเสนอการจัดส่งฟรีแบบมีระยะเวลา:

1. ไปที่ **Marketing > Sales & Promotions**
2. สร้างโปรโมชันใหม่
3. ตั้งค่าเงื่อนไข: "Cart total over X"
4. ตั้งค่าการกระทำ: "Free shipping"
5. ตั้งค่าวันเริ่มต้นและวันสิ้นสุด

## การจัดส่งระหว่างประเทศ

สำหรับคำสั่งซื้อระหว่างประเทศ ให้ตรวจสอบว่าสินค้าของคุณมี:

- **HS Code** — การจัดประเภทภาษีระบบฮาร์โมไนซ์
- **Country of Origin** — ประเทศที่ผลิตสินค้า
- **Customs Value** — ค่าที่ถูกประกาศสำหรับภาษีศุลกากร

ฟิลด์เหล่านี้อยู่ในแท็บ **Inventory** ของแต่ละสินค้า ผู้ให้บริการจะใช้ข้อมูลนี้เพื่อสร้างเอกสารศุลกากรโดยอัตโนมัติ

## เคล็ดลับ

- เริ่มต้นด้วยวิธีการจัดส่งแบบมืออาชีพเพื่อเริ่มต้นร้านของคุณอย่างรวดเร็ว จากนั้นเพิ่มการเชื่อมต่อกับผู้ให้บริการจัดส่งในภายหลัง
- สร้างพื้นที่การจัดส่งสำหรับจุดหมายปลายทางที่พบบ่อยที่สุดก่อน
- ทดสอบการตั้งค่าการจัดส่งของคุณโดยการสั่งซื้อทดสอบด้วยที่อยู่ต่างๆ
- ใช้คุณสมบัติการเพิ่มอัตราเพื่อครอบคลุมค่าใช้จ่ายในการจัดการและบรรจุภัณฑ์
- ตั้งค่าเกณฑ์การจัดส่งฟรีเพื่อเพิ่มค่าเฉลี่ยของยอดขาย
