# ชุดการจัดส่ง

ชุดการจัดส่งกำหนดขนาดกล่องและซองจดหมายที่กำหนดไว้ล่วงหน้าสำหรับการคำนวณอัตราค่าจัดส่งและระบบจัดกล่องอัตโนมัติ—ระบุขนาดภายใน (พื้นที่ใช้สอย), ความหนาของผนัง (ขนาดภายนอกสำหรับ API ของผู้ให้บริการจัดส่ง), ขีดจำกัดน้ำหนัก และค่าใช้จ่ายการบรรจุภัณฑ์ ผู้ให้บริการจะใช้ขนาดภายนอกเพื่อคำนวณน้ำหนักตามขนาดเพื่อให้ได้ราคาที่แม่นยำ ชุดบรรจุภัณฑ์มีลำดับความสำคัญในการจัดเรียงเพื่อใช้กับอัลกอริทึมการจัดกล่องอัตโนมัติที่เลือกชุดบรรจุภัณฑ์ที่เหมาะสมที่สุดเพื่อให้ตรงกับรายการในตะกร้าสินค้า

กำหนดค่าชุดบรรจุภัณฑ์เมื่อใช้ API ของผู้ให้บริการจัดส่งเพื่อคำนวณอัตราค่าใช้จ่ายแบบเรียลไทม์ หรือเมื่อคุณต้องการการคำนวณน้ำหนักตามขนาดที่แม่นยำ

## การกำหนดค่าชุดบรรจุภัณฑ์

แต่ละชุดบรรจุภัณฑ์กำหนดค่าต่อไปนี้:

**ขนาด**:
- **ความยาวภายใน**: พื้นที่ใช้สอยภายใน (เซนติเมตร)
- **ความกว้างภายใน**: พื้นที่ใช้สอยภายใน (เซนติเมตร)
- **ความสูงภายใน**: พื้นที่ใช้สอยภายใน (เซนติเมตร)
- **ความหนาผนัง**: ความหนาของวัสดุบรรจุภัณฑ์ (เซนติเมตร)

**ขนาดภายนอก** (คำนวณอัตโนมัติ):
```
ความยาวภายนอก = ความยาวภายใน + (2 × ความหนาผนัง)
ความกว้างภายนอก = ความกว้างภายใน + (2 × ความหนาผนัง)
ความสูงภายนอก = ความสูงภายใน + (2 × ความหนาผนัง)
```

**น้ำหนักและค่าใช้จ่าย**:
- **น้ำหนักเปล่า**: น้ำหนักของชุดบรรจุภัณฑ์เมื่อว่างเปล่า (กรัม)
- **น้ำหนักสูงสุด**: น้ำหนักบรรทุกสูงสุด (กรัม)
- **ค่าใช้จ่าย**: ค่าใช้จ่ายของวัสดุบรรจุภัณฑ์ (สำหรับการปรับปรุงค่าใช้จ่าย)

**คุณสมบัติ**:
- **ชื่อ**: ตัวระบุชุดบรรจุภัณฑ์ (เช่น "กล่องเล็ก", "ซองใหญ่")
- **ประเภท**: กล่องหรือซอง
- **ลำดับความสำคัญ**: ลำดับการเลือกบรรจุภัณฑ์อัตโนมัติ (ค่าต่ำกว่า = ลำดับความสำคัญสูงกว่า)
- **เปิดใช้งาน**: เปิด/ปิดการใช้งาน

## ความสำคัญของขนาดภายนอก

ผู้ให้บริการจะคำนวณ **น้ำหนักตามขนาด** จากขนาดภายนอก:

**สูตรน้ำหนักตามขนาด**:
```
น้ำหนักตามขนาด = (ความยาว × ความกว้าง × ความสูง) / ตัวหาร

ตัวหารที่ใช้บ่อย:
- DHL: 5000
- FedEx/UPS: 5000 (ภายในประเทศ), 6000 (ต่างประเทศ)
```

**ตัวอย่าง**:
```
กล่องเล็ก:
ภายใน: 20cm × 15cm × 10cm
ความหนาผนัง: 0.5cm
ภายนอก: 21cm × 16cm × 11cm

น้ำหนักตามขนาด = (21 × 16 × 11) / 5000 = 0.74kg

หากน้ำหนักจริง = 0.5kg → ผู้ให้บริการจะเรียกเก็บค่าตามน้ำหนักตามขนาด 0.74kg (น้ำหนักตามขนาดสูงกว่า)
```

**เหตุผลที่ต้องแม่นยำ**: ขนาดที่ไม่ถูกต้อง → อัตราค่าใช้จ่ายไม่ถูกต้อง → ลูกค้าถูกคิดค่าใช้จ่ายเกินหรือต่ำเกิน

## ขนาดชุดบรรจุภัณฑ์ที่พบบ่อย

### ซองใส่สินค้าขนาดเล็ก

```
ภายใน: 25cm × 18cm × 2cm
ความหนาผนัง: 0.3cm
น้ำหนักสูงสุด: 500g
ประเภท: ซอง
ใช้สำหรับ: เอกสาร, หนังสือ, ของประดับ
```

### กล่องขนาดเล็ก

```
ภายใน: 20cm × 15cm × 10cm
ความหนาผนัง: 0.5cm
น้ำหนักสูงสุด: 5kg
ประเภท: กล่อง
ใช้สำหรับ: อุปกรณ์อิเล็กทรอนิกส์ขนาดเล็ก, ผลิตภัณฑ์เครื่องสำอาง, อุปกรณ์เสริม
```

### กล่องขนาดกลาง

```
ภายใน: 30cm × 25cm × 20cm
ความหนาผนัง: 0.5cm
น้ำหนักสูงสุด: 15kg
ประเภท: กล่อง
ใช้สำหรับ: เสื้อผ้า, รองเท้า, อุปกรณ์ครัว
```

### กล่องขนาดใหญ่

```
ภายใน: 45cm × 35cm × 30cm
ความหนาผนัง: 0.6cm
น้ำหนักสูงสุด: 30kg
ประเภท: กล่อง
ใช้สำหรับ: สินค้าจำนวนมาก, สินค้าหลายชิ้น, อุปกรณ์อิเล็กทรอนิกส์ขนาดใหญ่
```

## อัลกอริทึมการบรรจุภัณฑ์อัตโนมัติ

ระบบเลือกชุดบรรจุภัณฑ์อัตโนมัติสำหรับรายการในตะกร้าสินค้า:

**วิธีการทำงาน**:
1. คำนวณปริมาตรรวมของรายการในตะกร้า
2. จัดเรียงชุดบรรจุภัณฑ์ตามลำดับความสำคัญ (เริ่มจากตัวเลขต่ำสุด)
3. ลองบรรจุรายการลงในชุดบรรจุภัณฑ์เดียว
4. หากไม่สามารถบรรจุได้ ลองชุดบรรจุภัณฑ์ขนาดถัดไป
5. หากไม่มีชุดบรรจุภัณฑ์ใดสามารถบรรจุได้ ให้รวมหลายชุดบรรจุภัณฑ์เข้าด้วยกัน
6. ปรับปรุงตามการตั้งค่า `optimize_for`

**โหมดการปรับปรุง**:
- **ต้นทุน**: ลดค่าใช้จ่ายในการบรรจุภัณฑ์ให้มากที่สุด
- **ปริมาตร**: ลดการใช้พื้นที่ส่วนเกินให้มากที่สุด
- **จำนวน**: ลดจำนวนชุดบรรจุภัณฑ์ให้มากที่สุด

**ตัวอย่าง**:
```
รายการในตะกร้า:
- สินค้า A: 10cm × 8cm × 5cm, 200g
- สินค้า B: 15cm × 12cm × 8cm, 400g

ชุดบรรจุภัณฑ์ (ตามลำดับความสำคัญ):
1. กล่องเล็ก (20×15×10, ลำดับความสำคัญ=1)
2. กล่องกลาง (30×25×20, ลำดับความสำคัญ=2)

อัลกอริทึม:
ลองใช้กล่องเล็ก: ทั้งสองสินค้าสามารถบรรจุได้
ผลลัพธ์: 1× กล่องเล็ก (ปรับปรุงตามจำนวน)
```

## ลำดับความสำคัญของชุดบรรจุภัณฑ์

**ลำดับความสำคัญกำหนดลำดับการบรรจุ**:

ลำดับความสำคัญ 1 (สูงสุด): ลองใช้ชุดบรรจุภัณฑ์ขนาดเล็กก่อน
ลำดับความสำคัญ 10: ใช้ชุดบรรจุภัณฑ์ขนาดใหญ่เป็นทางเลือกสุดท้าย

**กลยุทธ์**:
- ชุดบรรจุภัณฑ์ขนาดเล็ก = ลำดับความสำคัญต่ำ (1-3)
- ชุดบรรจุภัณฑ์ขนาดกลาง = ลำดับความสำคัญปานกลาง (4-6)
- ชุดบรรจุภัณฑ์ขนาดใหญ่ = ลำดับความสำคัญสูง (7-10)

**เหตุผล**: เริ่มต้นด้วยชุดบรรจุภัณฑ์ขนาดเล็ก ขยายขนาดขึ้นหากจำเป็น → ลดค่าใช้จ่ายการจัดส่ง

## ความแม่นยำของความหนาผนัง

วัดความหนาของบรรจุภัณฑ์จริง:

**วิธีการวัด**:
1. ได้กล่องว่าง
2. วัดขนาดภายใน (ภายใน)
3. วัดขนาดภายนอก (ภายนอก)
4. คำนวณ: `(ขนาดภายนอก - ขนาดภายใน) / 2 = ความหนาผนัง`

**ตัวอย่าง**:
```
ความกว้างภายใน: 20cm
ความกว้างภายนอก: 21cm
ความหนาผนัง: (21 - 20) / 2 = 0.5cm
```

**ความหนาผนังที่พบบ่อย**:
- ซองใส่สินค้า: 0.2-0.4cm
- กระดาษบัตรแบบผนังเดียว: 0.4-0.6cm
- กระดาษบัตรแบบผนังสองชั้น: 0.8-1.0cm

## การสร้างชุดบรรจุภัณฑ์ล่วงหน้า

**ขั้นตอนการดำเนินการ**:
1. ตั้งค่า > การจัดส่ง > ชุดการจัดส่ง
2. คลิก "เพิ่มชุดการจัดส่ง"
3. ป้อนชื่อ (เช่น "กล่องขนาดกลาง")
4. เลือกประเภท (กล่องหรือซอง)
5. ป้อนขนาดภายใน (L × W × H ในเซนติเมตร)
6. ป้อนความหนาผนัง (เซนติเมตร)
7. ระบบคำนวณขนาดภายนอกอัตโนมัติ
8. ป้อนน้ำหนักเปล่า (น้ำหนักของชุดบรรจุภัณฑ์ว่างในกรัม)
9. ป้อนน้ำหนักสูงสุด (น้ำหนักบรรทุกในกรัม)
10. ตัวเลือก: ป้อนค่าใช้จ่าย (สำหรับการปรับปรุงค่าใช้จ่าย)
11. ตั้งค่าลำดับความสำคัญ (1-10)
12. เปิดใช้งาน = ใช่
13. บันทึก

## การทดสอบการเลือกชุดบรรจุภัณฑ์

**การทดสอบด้วยตนเอง**:
1. เพิ่มสินค้าเพื่อทดสอบในตะกร้า
2. ไปที่ขั้นตอนชำระเงิน
3. เลือกวิธีการจัดส่งแบบเรียลไทม์ (ใช้ชุดบรรจุภัณฑ์)
4. ตรวจสอบว่าได้ราคาที่สมเหตุสมผล
5. ตรวจสอบการตอบสนองจากผู้ให้บริการ (บันทึก API แสดงชุดบรรจุภัณฑ์ที่เลือก)

**แสดงตัวอย่างการบรรจุอัตโนมัติ**:
- บัญชีผู้ให้บริการบางรายแสดงการแยกชุดบรรจุภัณฑ์
- ดูชุดบรรจุภัณฑ์ที่เลือกสำหรับตะกร้าสินค้า
- ตรวจสอบการบรรจุที่เหมาะสมที่สุด

## เคล็ดลับ

- **วัดให้แม่นยำ** - ขนาดที่ไม่ถูกต้อง → อัตราค่าใช้จ่ายผู้ให้บริการไม่ถูกต้อง
- **รวมความหนาผนัง** - สำคัญสำหรับน้ำหนักตามขนาด
- **เริ่มต้นด้วยขนาด 3-4 ขนาด** - ขนาดเล็ก, ขนาดกลาง, ขนาดใหญ่ครอบคลุมสถานการณ์ส่วนใหญ่
- **ตั้งค่าน้ำหนักสูงสุดให้สมจริง** - ความจุของกล่อง ไม่ใช่ขีดจำกัดทฤษฎี
- **ใช้ลำดับความสำคัญอย่างชาญฉลาด** - กล่องเล็ก ลำดับความสำคัญ 1, กล่องใหญ่ ลำดับความสำคัญ 10
- **ทดสอบด้วยสินค้าจริง** - ตรวจสอบว่าระบบเลือกขนาดที่ถูกต้อง
- **อัปเดตเมื่อเปลี่ยนบรรจุภัณฑ์** - ผู้ให้บริการใหม่ = วัดขนาดใหม่
- **พิจารณาสินค้าพิเศษ** - สินค้าเปราะบางอาจต้องใช้ขนาดกล่องเฉพาะ
- **รักษาชุดบรรจุภัณฑ์ที่เปิดใช้งานให้น้อยที่สุด** - ตัวเลือกมากเกินไปทำให้ระบบบรรจุอัตโนมัติทำงานช้าลง
- **จัดทำเอกสารเกี่ยวกับบรรจุภัณฑ์** - บันทึกว่าสินค้าใดเข้ากับชุดบรรจุภัณฑ์ใด
