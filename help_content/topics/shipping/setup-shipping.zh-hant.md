---
title: 配送設定
---

本指南介紹如何為您的商店設定配送——從建立基本配送方式到連接即時承運商整合以取得即時運費。

## 配送概述

Spwig 提供兩種配送方式：

- **手動配送方式** — 您自訂的固定運費方式（例如：「標準配送 — $5.99」）
- **承運商整合** — 來自 FedEx、UPS 和 DHL 等服務商的即時運費

您可以使用其中一種方式，也可以兩者結合使用。

## 配送方式

配送方式是客戶在結帳時看到的選項。在側邊欄中導航至 **訂單 > 出貨** 進行管理。

![配送方式](/static/core/admin/img/help/setup-shipping/shipping-methods.webp)

### 建立配送方式

1. 點擊 **新增配送方式**
2. 填寫詳細資訊：
   - **名稱** — 向客戶顯示的名稱（例如：「快遞配送」）
   - **描述** — 服務的簡要描述
   - **價格** — 固定運費
   - **預計送達時間** — 送達時間估算（例如：「3-5 個工作天」）
3. 點擊 **儲存**

## 配送區域

配送區域定義了配送方式適用的地理區域。導航至 **配送區域** 部分進行管理。

![配送區域](/static/core/admin/img/help/setup-shipping/shipping-zones.webp)

### 建立區域

1. 點擊 **新增配送區域**
2. 設定區域：
   - **區域名稱** — 內部名稱（例如：「美國國內」、「歐洲」）
   - **國家** — 選擇屬於該區域的國家
   - **州/地區** — 可選擇縮小到特定的州
   - **郵遞區號模式** — 使用類似「9*」的模式來定位特定區域
3. 將配送方式指派到該區域
4. 點擊 **儲存**

### 區域優先順序

當客戶的地址符合多個區域時，最具體的區域優先。具有州級定位的區域優先於國家級區域。

## 承運商整合

連接物流承運商以在結帳時提供即時計算的運費。

![物流承運商](/static/core/admin/img/help/setup-shipping/shipping-carriers.webp)

### 可用服務商

從市集瀏覽並安裝物流服務商。

![物流服務商](/static/core/admin/img/help/setup-shipping/shipping-providers.webp)

支援的承運商包括：

- **FedEx** — 陸運、快遞、國際
- **UPS** — 陸運、兩日送達、隔夜送達、全球配送
- **DHL** — 快遞、電子商務
- **USPS** — 優先郵件、頭等郵件、媒體郵件
- 更多承運商可透過市集取得

### 設定承運商

1. 前往物流服務商頁面，點擊您首選承運商上的 **安裝**
2. 按照設定精靈操作：
   - **第 1 步** — 檢視服務商詳情
   - **第 2 步** — 設定一般選項
   - **第 3 步** — 輸入您的 API 憑證（帳號、API 金鑰等）
   - **第 4 步** — 啟用特定服務（陸運、快遞等）
   - **第 5 步** — 測試連線
3. 連線成功後，承運商的運費將自動顯示在結帳頁面

### API 憑證

每個承運商都需要一個 API 帳戶：

- **FedEx** — 在 FedEx 開發者入口網站註冊，建立應用程式，然後複製您的 API 金鑰和密鑰
- **UPS** — 在 UPS 開發者工具包註冊，申請存取金鑰
- **DHL** — 透過其商業入口網站聯繫 DHL 取得 API 憑證

## 配送規則

建立進階規則來控制何時以及如何提供配送方式。

### 常見規則

- **滿 $50 免運費** — 設定購物車最低金額享受免運費
- **輕量訂單固定運費** — 訂單重量低於閾值時使用固定運費
- **偏遠地區停用快遞** — 根據郵遞區號隱藏快遞選項
- **百分比加價** — 在承運商運費基礎上新增手續費百分比

### 建立規則

1. 導航至配送規則部分
2. 點擊 **新增規則**
3. 設定條件（購物車總額、重量、區域等）
4. 定義操作（調整運費、隱藏方式、啟用免運費）
5. 儲存規則

規則按順序執行——第一個符合的規則生效。

## 免運費

### 全店免運費

在 **設定 > 商店設定** 中全域啟用免運費：

- 開啟 **免運費** 開關
- 可選設定最低訂單金額
- 選擇適用的地區

### 促銷免運費

建立限時免運費優惠：

1. 前往 **行銷 > 促銷活動**
2. 建立新促銷
3. 設定條件：「購物車總額超過 X」
4. 設定操作：「免運費」
5. 設定開始和結束日期

## 國際配送

對於國際訂單，請確保您的產品具有以下資訊：

- **HS 編碼** — 協調制度關稅分類
- **原產國** — 製造國家
- **報關價值** — 海關申報價值

這些欄位位於每個產品的 **庫存** 標籤頁中。承運商會使用這些資訊自動產生海關文件。

## 提示

- 先從手動配送方式開始讓商店快速運營，之後再新增承運商整合。
- 首先為最常見的配送目的地建立配送區域。
- 始終透過使用不同地址下測試訂單來測試您的配送設定。
- 使用運費加價功能來涵蓋處理和包裝成本。
- 設定免運費門檻以提高平均訂單金額。
