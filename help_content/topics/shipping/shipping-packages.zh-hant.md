---
title: 運送包裝
---

運送包裝定義預設的紙箱和信封尺寸，用於費用計算和自動包裝——指定內部尺寸（可用空間）、牆壁厚度（運輸商API的外部尺寸）、重量限制和包裝成本。運輸商使用外部尺寸計算體積重量以提供準確的費用報價。包裝具有優先順序，供自動包裝算法選擇最佳包裝組合以適配購物車商品。

在使用運輸商API進行即時費用計算或需要準確體積重量計算時配置包裝。

## 包裝配置

每個包裝定義：

**尺寸**：
- **內部長度**：內部可用空間（公分）
- **內部寬度**：內部可用空間（公分）
- **內部高度**：內部可用空間（公分）
- **牆壁厚度**：包裝材料厚度（公分）

**外部尺寸**（自動計算）：
```
外部長度 = 內部長度 + (2 × 墻壁厚度)
外部寬度 = 內部寬度 + (2 × 墻壁厚度)
外部高度 = 內部高度 + (2 × 墻壁厚度)
```

**重量與成本**：
- **皮重**：空包重量（克）
- **最大重量**：最大承重（克）
- **成本**：包裝材料成本（用於成本優化）

**屬性**：
- **名稱**：包裝識別符（例如："小型紙箱"、"大型信封"）
- **類型**：紙箱或信封
- **優先順序**：自動包裝選擇順序（數字越小優先級越高）
- **啟用**：切換可用性

---

## 為什麼外部尺寸很重要

運輸商根據外部尺寸計算**體積重量**：

**體積重量公式**：
```
體積重量 = (長 × 寬 × 高) / 除數

常見除數：
- DHL：5000
- FedEx/UPS：5000（國內），6000（國際）
```

**範例**：
```
小型紙箱：
內部：20公分 × 15公分 × 10公分
牆壁厚度：0.5公分
外部：21公分 × 16公分 × 11公分

體積重量 = (21 × 16 × 11) / 5000 = 0.74公斤

若實際重量 = 0.5公斤 → 運輸商用體積重量 0.74公斤計費（體積重量更高）
```

**精確度的重要性**：尺寸不準確 → 費用報價錯誤 → 顧客被多收費或少收費。

## 自動包裝算法

系統自動為購物車商品選擇包裝：

**運作方式**：
1. 計算購物車商品總體積
2. 按優先順序排序包裝（數字越小越先）
3. 嘗試將商品放入單個包裝
4. 若不適合，嘗試下一個包裝尺寸
5. 若沒有單個包裝適合，組合多個包裝
6. 根據 `optimize_for` 設定進行優化

**優化模式**：
- **成本**：最小化包裝成本
- **體積**：最小化浪費空間
- **數量**：最小化包裝數量

**範例**：
```
購物車商品：
- 商品 A：10公分 × 8公分 × 5公分，200克
- 商品 B：15公分 × 12公分 × 8公分，400克

包裝（按優先順序）：
1. 小型紙箱（20×15×10，優先順序=1）
2. 中型紙箱（30×25×20，優先順序=2）

算法：
嘗試小型紙箱：兩件商品都能放入
結果：1× 小型紙箱（優化為數量最少）
```

---

## 包裝優先順序

**優先順序決定包裝順序**：

優先順序 1（最高）：先嘗試小型包裝
優先順序 10：大型包裝最後選擇

**策略**：
- 小型包裝 = 低優先順序數字（1-3）
- 中型包裝 = 中等優先順序（4-6）
- 大型包裝 = 高優先順序數字（7-10）

**原因**：從最小包裝開始，如需擴展 → 最小化運費。

---

## 墙壁厚度精確度

測量實際包裝：

**如何測量**：
1. 取一個空箱子
2. 測量內部尺寸（內部）
3. 測量外部尺寸（外部）
4. 計算：`(外部 - 內部) / 2 = 墻壁厚度`

**範例**：
```
內部寬度：20公分
外部寬度：21公分
牆壁厚度：(21 - 20) / 2 = 0.5公分
```

**常見厚度**：
- 緩衝信封：0.2-0.4公分
- 單層紙板：0.4-0.6公分
- 雙層紙板：0.8-1.0公分

## 建立包裝預設

**逐步操作**：

1. 設定 > 運送 > 運送包裝
2. 點擊 "新增運送包裝"
3. 輸入名稱（例如："中型紙箱"）
4. 選擇類型（紙箱或信封）
5. 輸入內部尺寸（L × W × H 公分）
6. 輸入牆壁厚度（公分）
7. 系統自動計算外部尺寸
8. 輸入皮重（空包重量 克）
9. 輸入最大重量（承重 克）
10. 可選：輸入成本（用於成本優化）
11. 設定優先順序（1-10）
12. 啟用 = 是
13. 儲存

## 測試包裝選擇

**手動測試**：
1. 將商品加入測試購物車
2. 繼續結帳
3. 選擇即時運費方式（使用包裝）
4. 確認返回合理費用
5. 檢查運輸商回應（API 日誌顯示選擇的包裝）

**自動包裝預覽**：
- 一些運費提供商帳戶會顯示包裝細分
- 查看購物車選擇了哪些包裝
- 確認最佳包裝

## 小提示

- **精確測量** - 測量不準 → 運費報價錯誤
- **包含牆壁厚度** - 對體積重量至關重要
- **從 3-4 種尺寸開始** - 小、中、大尺寸涵蓋大多數情況
- **設置合理的最大重量** - 紙箱容量，而非理論極限
- **明智使用優先順序** - 小紙箱優先順序 1，大紙箱優先順序 10
- **用實際商品測試** - 確認自動包裝選擇正確尺寸
- **包裝變更時更新** - 新供應商 = 重新測量尺寸
- **考慮特殊商品** - 易碎品可能需要特定紙箱尺寸
- **保持啟用包裝最少** - 選項太多會拖慢自動包裝算法
- **記錄包裝資訊** - 記下哪些商品適合哪些包裝
