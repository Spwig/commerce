---
title: 配送设置
---

本指南介绍如何为您的商店配置配送——从设置基本配送方式到连接实时承运商集成以获取实时运费。

## 配送概述

Spwig 提供两种配送方式：

- **手动配送方式** — 您自定义的固定运费方式（例如："标准配送 — $5.99"）
- **承运商集成** — 来自 FedEx、UPS 和 DHL 等服务商的实时运费

您可以使用其中一种方式，也可以两者结合使用。

## 配送方式

配送方式是客户在结账时看到的选项。在侧边栏中导航至 **订单 > 发货** 进行管理。

![配送方式](/static/core/admin/img/help/setup-shipping/shipping-methods.webp)

### 创建配送方式

1. 点击 **添加配送方式**
2. 填写详细信息：
   - **名称** — 向客户显示的名称（例如："快递配送"）
   - **描述** — 服务的简要描述
   - **价格** — 固定运费
   - **预计送达时间** — 送达时间估算（例如："3-5 个工作日"）
3. 点击 **保存**

## 配送区域

配送区域定义了配送方式适用的地理区域。导航至 **配送区域** 部分进行管理。

![配送区域](/static/core/admin/img/help/setup-shipping/shipping-zones.webp)

### 创建区域

1. 点击 **添加配送区域**
2. 配置区域：
   - **区域名称** — 内部名称（例如："美国国内"、"欧洲"）
   - **国家** — 选择属于该区域的国家
   - **州/地区** — 可选择缩小到特定的州
   - **邮政编码模式** — 使用类似"9*"的模式来定位特定区域
3. 将配送方式分配到该区域
4. 点击 **保存**

### 区域优先级

当客户的地址匹配多个区域时，最具体的区域优先。具有州级定位的区域优先于国家级区域。

## 承运商集成

连接物流承运商以在结账时提供实时计算的运费。

![物流承运商](/static/core/admin/img/help/setup-shipping/shipping-carriers.webp)

### 可用服务商

从市场浏览并安装物流服务商。

![物流服务商](/static/core/admin/img/help/setup-shipping/shipping-providers.webp)

支持的承运商包括：

- **FedEx** — 陆运、快递、国际
- **UPS** — 陆运、两日达、隔夜达、全球配送
- **DHL** — 快递、电子商务
- **USPS** — 优先邮件、头等邮件、媒体邮件
- 更多承运商可通过市场获取

### 设置承运商

1. 前往物流服务商页面，点击您首选承运商上的 **安装**
2. 按照设置向导操作：
   - **第 1 步** — 查看服务商详情
   - **第 2 步** — 配置常规设置
   - **第 3 步** — 输入您的 API 凭据（账号、API 密钥等）
   - **第 4 步** — 启用特定服务（陆运、快递等）
   - **第 5 步** — 测试连接
3. 连接成功后，承运商的运费将自动显示在结账页面

### API 凭据

每个承运商都需要一个 API 账户：

- **FedEx** — 在 FedEx 开发者门户注册，创建应用，然后复制您的 API 密钥和密钥
- **UPS** — 在 UPS 开发者工具包注册，申请访问密钥
- **DHL** — 通过其商业门户联系 DHL 获取 API 凭据

## 配送规则

创建高级规则来控制何时以及如何提供配送方式。

### 常见规则

- **满 $50 免运费** — 设置购物车最低金额享受免运费
- **轻量订单固定运费** — 订单重量低于阈值时使用固定运费
- **偏远地区禁用快递** — 根据邮政编码隐藏快递选项
- **百分比加价** — 在承运商运费基础上添加手续费百分比

### 创建规则

1. 导航至配送规则部分
2. 点击 **添加规则**
3. 设置条件（购物车总额、重量、区域等）
4. 定义操作（调整运费、隐藏方式、启用免运费）
5. 保存规则

规则按顺序执行——第一个匹配的规则生效。

## 免运费

### 全店免运费

在 **设置 > 商店设置** 中全局启用免运费：

- 开启 **免运费** 开关
- 可选设置最低订单金额
- 选择适用的地区

### 促销免运费

创建限时免运费优惠：

1. 前往 **营销 > 促销活动**
2. 创建新促销
3. 设置条件："购物车总额超过 X"
4. 设置操作："免运费"
5. 配置开始和结束日期

## 国际配送

对于国际订单，请确保您的产品具有以下信息：

- **HS 编码** — 协调制度关税分类
- **原产国** — 制造国家
- **报关价值** — 海关申报价值

这些字段位于每个产品的 **库存** 标签页中。承运商会使用这些信息自动生成海关文件。

## 提示

- 先从手动配送方式开始让商店快速运营，之后再添加承运商集成。
- 首先为最常见的配送目的地创建配送区域。
- 始终通过使用不同地址下测试订单来测试您的配送配置。
- 使用运费加价功能来覆盖处理和包装成本。
- 设置免运费门槛以提高平均订单金额。
