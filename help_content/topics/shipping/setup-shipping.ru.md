---
title: Настройка доставки
---

Это руководство объясняет, как настроить доставку для вашего магазина — от создания базовых методов доставки до подключения интеграций с перевозчиками для получения тарифов в реальном времени.

## Обзор доставки

Spwig предлагает два подхода к доставке:

- **Ручные методы доставки** — Методы с фиксированной ставкой, которые вы определяете сами (например, «Стандартная доставка — 599 руб.»)
- **Интеграции с перевозчиками** — Тарифы в реальном времени от таких провайдеров, как FedEx, UPS и DHL

Вы можете использовать любой из подходов или комбинировать оба.

## Методы доставки

Методы доставки — это варианты, которые ваши клиенты видят при оформлении заказа. Перейдите в **Заказы > Отправления** в боковой панели для управления ими.

![Shipping methods](/static/core/admin/img/help/setup-shipping/shipping-methods.webp)

### Создание метода доставки

1. Нажмите **Добавить метод доставки**
2. Заполните данные:
   - **Название** — Отображаемое название для клиентов (например, «Экспресс-доставка»)
   - **Описание** — Краткое описание услуги
   - **Цена** — Фиксированная стоимость доставки
   - **Ориентировочный срок доставки** — Оценка времени доставки (например, «3-5 рабочих дней»)
3. Нажмите **Сохранить**

## Зоны доставки

Зоны доставки определяют географические регионы, в которых действуют ваши методы доставки. Перейдите в раздел **Зоны доставки** для управления ими.

![Shipping zones](/static/core/admin/img/help/setup-shipping/shipping-zones.webp)

### Создание зоны

1. Нажмите **Добавить зону доставки**
2. Настройте зону:
   - **Название зоны** — Внутреннее название (например, «Россия», «Европа»)
   - **Страны** — Выберите страны, входящие в эту зону
   - **Регионы/Области** — При необходимости ограничьте конкретными регионами
   - **Шаблоны почтовых индексов** — Используйте шаблоны вроде «9*» для охвата определённых территорий
3. Назначьте методы доставки для этой зоны
4. Нажмите **Сохранить**

### Приоритет зон

Когда адрес клиента соответствует нескольким зонам, более конкретная зона имеет приоритет. Зона с таргетингом на уровне региона имеет приоритет над зоной на уровне страны.

## Интеграции с перевозчиками

Подключайтесь к перевозчикам, чтобы предлагать рассчитанные в реальном времени тарифы при оформлении заказа.

![Shipping carriers](/static/core/admin/img/help/setup-shipping/shipping-carriers.webp)

### Доступные провайдеры

Просматривайте и устанавливайте провайдеров доставки из маркетплейса.

![Shipping providers](/static/core/admin/img/help/setup-shipping/shipping-providers.webp)

Поддерживаемые перевозчики включают:

- **FedEx** — Наземная, Экспресс, Международная
- **UPS** — Наземная, 2 дня, Ночная, Международная
- **DHL** — Экспресс, eCommerce
- **USPS** — Приоритетная, Первый класс, Media Mail
- И другие, доступные через Маркетплейс

### Настройка перевозчика

1. Перейдите на страницу провайдеров доставки и нажмите **Установить** на нужном перевозчике
2. Следуйте мастеру настройки:
   - **Шаг 1** — Просмотр деталей провайдера
   - **Шаг 2** — Настройка общих параметров
   - **Шаг 3** — Ввод учётных данных API (номер аккаунта, ключ API и т.д.)
   - **Шаг 4** — Включение конкретных услуг (Наземная, Экспресс и т.д.)
   - **Шаг 5** — Тестирование подключения
3. После подключения тарифы перевозчика автоматически отображаются при оформлении заказа

### Учётные данные API

Для каждого перевозчика требуется аккаунт API:

- **FedEx** — Зарегистрируйтесь на портале разработчиков FedEx, создайте приложение и скопируйте ключ и секрет API
- **UPS** — Зарегистрируйтесь в UPS Developer Kit, запросите ключ доступа
- **DHL** — Свяжитесь с DHL для получения учётных данных API через их бизнес-портал

## Правила доставки

Создавайте расширенные правила для управления тем, когда и как предлагаются методы доставки.

### Распространённые правила

- **Бесплатная доставка от 5000 руб.** — Установите минимальную сумму корзины для бесплатной доставки
- **Фиксированная ставка для лёгких заказов** — Фиксированная ставка, когда вес заказа ниже порога
- **Отключение экспресс-доставки для отдалённых районов** — Скрытие экспресс-вариантов на основе почтовых индексов
- **Процентная наценка** — Добавление комиссии за обработку в виде процента от тарифов перевозчика

### Создание правила

1. Перейдите в раздел правил доставки
2. Нажмите **Добавить правило**
3. Установите условия (сумма корзины, вес, зона и т.д.)
4. Определите действие (скорректировать тариф, скрыть метод, включить бесплатную доставку)
5. Сохраните правило

Правила оцениваются по порядку — применяется первое совпавшее правило.

## Бесплатная доставка

### Бесплатная доставка по всему магазину

Включите бесплатную доставку глобально в **Настройки > Настройки магазина**:

- Включите переключатель **Бесплатная доставка**
- При необходимости установите минимальную сумму заказа
- Выберите регионы, на которые распространяется акция

### Промо-акция на бесплатную доставку

Создайте ограниченные по времени предложения бесплатной доставки:

1. Перейдите в **Маркетинг > Скидки и акции**
2. Создайте новую акцию
3. Установите условие: «Сумма корзины больше X»
4. Установите действие: «Бесплатная доставка»
5. Настройте даты начала и окончания

## Международная доставка

Для международных заказов убедитесь, что ваши товары содержат:

- **Код HS** — Классификация тарифов по Гармонизированной системе
- **Страна происхождения** — Страна производства
- **Таможенная стоимость** — Заявленная стоимость для таможни

Эти поля находятся на вкладке **Склад** каждого товара. Перевозчики используют эту информацию для автоматического создания таможенной документации.

## Советы

- Начните с ручных методов доставки, чтобы быстро запустить магазин, а затем добавьте интеграции с перевозчиками.
- Сначала создайте зоны доставки для наиболее частых направлений.
- Всегда тестируйте конфигурацию доставки, размещая тестовые заказы с разными адресами.
- Используйте функцию наценки на тариф для покрытия расходов на обработку и упаковку.
- Установите пороги бесплатной доставки для увеличения среднего чека.
