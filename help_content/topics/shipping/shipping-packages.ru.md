---
title: Упаковочные комплекты
---

Упаковочные комплекты определяют предопределенные размеры коробок и конвертов для расчета тарифов и автоматической упаковки — укажите внутренние размеры (используемое пространство), толщину стенок (внешние размеры для API-интерфейсов перевозчиков), ограничения по весу и стоимость упаковки. Перевозчики используют внешние размеры для расчета объемного веса для точных тарифных предложений. Комплекты имеют приоритетную последовательность для алгоритмов упаковки, которые автоматически выбирают оптимальные комбинации упаковок для соответствия содержимому корзины.

Настройте комплекты при использовании API-интерфейсов перевозчиков для расчета тарифов в реальном времени или когда вам нужны точные расчеты объемного веса.

## Настройка комплектов

Каждый комплект определяет:

**Размеры**:
- **Внутренняя длина**: Используемое пространство внутри (см)
- **Внутренняя ширина**: Используемое пространство внутри (см)
- **Внутренняя высота**: Используемое пространство внутри (см)
- **Толщина стенки**: Толщина материала упаковки (см)

**Внешние размеры** (автоматически рассчитываются):
```
Внешняя длина = Внутренняя длина + (2 × Толщина стенки)
Внешняя ширина = Внутренняя ширина + (2 × Толщина стенки)
Внешняя высота = Внутренняя высота + (2 × Толщина стенки)
```

**Вес и стоимость**:
- **Тара**: Вес пустого комплекта (граммы)
- **Максимальный вес**: Максимальная грузоподъемность (граммы)
- **Стоимость**: Стоимость материала упаковки (для оптимизации затрат)

**Свойства**:
- **Название**: Идентификатор комплекта (например, "Маленькая коробка", "Большой конверт")
- **Тип**: Коробка или Конверт
- **Приоритет**: Порядок выбора для автоматической упаковки (чем ниже число, тем выше приоритет)
- **Активный**: Переключение доступности

---

## Почему важны внешние размеры

Перевозчики рассчитывают **объемный вес** на основе внешних размеров:

**Формула объемного веса**:
```
Объемный вес = (Длина × Ширина × Высота) / Делитель

Часто используемые делители:
- DHL: 5000
- FedEx/UPS: 5000 (внутренние), 6000 (международные)
```

**Пример**:
```
Маленькая коробка:
Внутренние: 20см × 15см × 10см
Толщина стенки: 0,5см
Внешние: 21см × 16см × 11см

Объемный вес = (21 × 16 × 11) / 5000 = 0,74 кг

Если фактический вес = 0,5 кг → Перевозчик взимает плату за 0,74 кг (объемный вес выше)
```

**Почему важна точность**: Неправильные размеры → неправильные тарифные предложения → клиент переплачивает или недоплачивает.

## Алгоритм автоматической упаковки

Система автоматически выбирает комплекты для товаров в корзине:

**Как это работает**:
1. Рассчитайте общий объем товаров в корзине
2. Отсортируйте комплекты по приоритету (начиная с самых низких чисел)
3. Попытайтесь разместить товары в одном комплекте
4. Если не помещаются, попробуйте следующий размер комплекта
5. Если ни один комплект не подходит, объедините несколько комплектов
6. Оптимизируйте на основе настройки `optimize_for`

**Режимы оптимизации**:
- **Стоимость**: Минимизировать стоимость упаковки
- **Объем**: Минимизировать потери пространства
- **Количество**: Минимизировать количество комплектов

**Пример**:
```
Товары в корзине:
- Товар А: 10см × 8см × 5см, 200 г
- Товар B: 15см × 12см × 8см, 400 г

Комплекты (по приоритету):
1. Маленькая коробка (20×15×10, приоритет=1)
2. Средняя коробка (30×25×20, приоритет=2)

Алгоритм:
Попытка использовать маленькую коробку: Оба товара помещаются
Результат: 1× Маленькая коробка (оптимизация по количеству)
```

## Приоритет комплектов

**Приоритет определяет порядок упаковки**:

Приоритет 1 (высший): Сначала попробуют маленькие комплекты
Приоритет 10: Крупные комплекты — последний вариант

**Стратегия**:
- Маленькие комплекты = низкие номера приоритета (1-3)
- Средние комплекты = средние приоритеты (4-6)
- Крупные комплекты = высокие номера приоритета (7-10)

**Почему**: Начните с самого маленького комплекта, масштабируйтесь при необходимости → минимизирует расходы на доставку.

## Точность толщины стенки

Измерьте реальную упаковку:

**Как измерить**:
1. Получите пустую коробку
2. Измерьте внутренние размеры (внутренние)
3. Измерьте внешние размеры (внешние)
4. Рассчитайте: `(Внешние - Внутренние) / 2 = Толщина стенки`

**Пример**:
```
Внутренняя ширина: 20 см
Внешняя ширина: 21 см
Толщина стенки: (21 - 20) / 2 = 0,5 см
```

**Часто встречающиеся толщины**:
- Конверт с подушкой: 0,2-0,4 см
- Одностенная картонная коробка: 0,4-0,6 см
- Двустенная картонная коробка: 0,8-1,0 см

## Создание предопределенного комплекта

**Пошаговая инструкция**:

1. Настройки > Доставка > Упаковочные комплекты
2. Нажмите "Добавить упаковочный комплект"
3. Введите имя (например, "Средняя коробка")
4. Выберите тип (Коробка или Конверт)
5. Введите внутренние размеры (L × W × H в см)
6. Введите толщину стенки (см)
7. Система автоматически рассчитывает внешние размеры
8. Введите вес тары (вес пустого комплекта в граммах)
9. Введите максимальный вес (грузоподъемность в граммах)
10. Необязательно: Введите стоимость (для оптимизации затрат)
11. Установите приоритет (1-10)
12. Включите активность = Да
13. Сохраните

## Тестирование выбора комплектов

**Ручной тест**:
1. Добавьте товары в тестовую корзину
2. Перейдите к оформлению заказа
3. Выберите метод доставки в реальном времени (использует комплекты)
4. Проверьте, что возвращен разумный тариф
5. Проверьте ответ от перевозчика (журналы API показывают выбранные комплекты)

**Предварительный просмотр автоматической упаковки**:
- Некоторые учетные записи поставщиков доставки показывают разбивку по комплектам
- Просмотрите, какие комплекты были выбраны для корзины
- Проверьте оптимальную упаковку

## Советы

- **Измеряйте точно** - Неправильные размеры → неправильные тарифы перевозчиков
- **Включите толщину стенки** - Критично для объемного веса
- **Начните с 3-4 размеров** - Маленький, средний, большой охватывают большинство сценариев
- **Установите реальные максимальные веса** - Грузоподъемность коробки, а не теоретический предел
- **Используйте приоритет с умом** - Маленькие коробки приоритет 1, крупные коробки приоритет 10
- **Тестируйте с реальными товарами** - Проверьте, правильно ли автоматическая упаковка выбирает размеры
- **Обновляйте при изменении упаковки** - Новый поставщик = переизмерьте размеры
- **Учитывайте специальные товары** - Хрупкие товары могут требовать определенных размеров коробок
- **Оставляйте минимальное количество активных комплектов** - Слишком много вариантов замедляют алгоритм автоматической упаковки
- **Документируйте упаковку** - Заметьте, какие товары подходят для каких комплектов
