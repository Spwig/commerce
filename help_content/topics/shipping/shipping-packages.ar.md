# حزم الشحن

حزم الشحن تحدد أبعاد صناديق وملصقات محددة مسبقًا لحساب التكلفة وتعبئة الأصناف تلقائيًا - أعد تحديد الأبعاد الداخلية (المساحة القابلة للاستخدام)، سمك الجدار (الأبعاد الخارجية لواجهات APIs الناقلة)، حدود الوزن، وتكلفة التعبئة. تستخدم شركات الشحن الأبعاد الخارجية لحساب الوزن الأبعادي للحصول على عروض أسعار دقيقة. تمتلك الحزم ترتيبًا أولوياتيًا لخوارزميات تعبئة الحزم التي تختار تلقائيًا مزيجًا من الحزم الأمثل لملاءمة عناصر السلة.

قم بتكوين الحزم عند استخدام واجهات APIs الناقلة للحصول على أسعار في الوقت الفعلي أو عند الحاجة إلى حسابات دقيقة لوزن الأبعاد.

## إعداد الحزم

تحدد كل حزمة:

**الأبعاد**:
- **الطول الداخلي**: المساحة القابلة للاستخدام داخل (سم)
- **العرض الداخلي**: المساحة القابلة للاستخدام داخل (سم)
- **الارتفاع الداخلي**: المساحة القابلة للاستخدام داخل (سم)
- **سمك الجدار**: سُمك مادة التعبئة (سم)

**الأبعاد الخارجية** (تُحسب تلقائيًا):
```
الطول الخارجي = الطول الداخلي + (2 × سمك الجدار)
العرض الخارجي = العرض الداخلي + (2 × سمك الجدار)
الارتفاع الخارجي = الارتفاع الداخلي + (2 × سمك الجدار)
```

**الوزن والتكلفة**:
- **الوزن الفارغ**: وزن الحزمة الفارغة (جرام)
- **الوزن الأقصى**: السعة القصوى للحمل (جرام)
- **التكلفة**: تكلفة مادة التعبئة (لتحسين التكلفة)

**الخصائص**:
- **الاسم**: مُعرف الحزمة (مثال: "صندوق صغير"، "ملصق كبير")
- **النوع**: صندوق أو ملصق
- **الأولوية**: ترتيب اختيار التعبئة التلقائية (الأصغر = الأولوية الأعلى)
- **الفعالية**: التبديل بين التفعيل والتعطيل

---

## لماذا تهم الأبعاد الخارجية

تقوم شركات الشحن بحساب **الوزن الأبعادي** من الأبعاد الخارجية:

**صيغة الوزن الأبعادي**:
```
وزن أبعادي = (الطول × العرض × الارتفاع) / القاسم

القسّاسات الشائعة:
- DHL: 5000
- FedEx/UPS: 5000 (المحلي)، 6000 (الدولي)
```

**مثال**:
```
صندوق صغير:
الداخلية: 20سم × 15سم × 10سم
سمك الجدار: 0.5سم
الخارجية: 21سم × 16سم × 11سم

وزن أبعادي = (21 × 16 × 11) / 5000 = 0.74كجم

إذا كان الوزن الفعلي = 0.5كجم → تُحسب الفاتورة من قبل الناقل على 0.74كجم (وزن أبعادي أعلى)
```

**لماذا تهم الدقة**: أبعاد غير دقيقة → عروض أسعار غير صحيحة → يتم فرض رسوم زائدة أو أقل على العميل.

## أحجام الحزم الشائعة

### الملصق الصغير المبطّن

```
الداخلية: 25سم × 18سم × 2سم
سمك الجدار: 0.3سم
الوزن الأقصى: 500غ
النوع: ملصق
الاستخدام: وثائق، كتب، المجوهرات
```

### الصندوق الصغير

```
الداخلية: 20سم × 15سم × 10سم
سمك الجدار: 0.5سم
الوزن الأقصى: 5كجم
النوع: صندوق
الاستخدام: أجهزة إلكترونية صغيرة، مستحضرات تجميل، ملحقات
```

### الصندوق المتوسط

```
الداخلية: 30سم × 25سم × 20سم
سمك الجدار: 0.5سم
الوزن الأقصى: 15كجم
النوع: صندوق
الاستخدام: ملابس، أحذية، أدوات المطبخ
```

### الصندوق الكبير

```
الداخلية: 45سم × 35سم × 30سم
سمك الجدار: 0.6سم
الوزن الأقصى: 30كجم
النوع: صندوق
الاستخدام: عناصر كثيرة، منتجات متعددة، أجهزة إلكترونية كبيرة
```

## خوارزمية التعبئة التلقائية

يقوم النظام باختيار الحزم تلقائيًا لعناصر السلة:

**كيف تعمل**:
1. حساب الحجم الإجمالي لعناصر السلة
2. ترتيب الحزم حسب الأولوية (أصغر عدد أولًا)
3. جرّب تعبئة العناصر في حزمة واحدة
4. إذا لم تناسب، جرّب الحجم التالي للحزمة
5. إذا لم تناسب أي حزمة واحدة، قم بدمج عدة حزم
6. تحسين بناءً على إعداد `optimize_for`

**الوضعيات التحسينية**:
- **التكلفة**: تقليل تكلفة التعبئة
- **الحجم**: تقليل المساحة غير المستغلة
- **العدد**: تقليل عدد الحزم

**مثال**:
```
عناصر السلة:
- العنصر أ: 10سم × 8سم × 5سم، 200غ
- العنصر ب: 15سم × 12سم × 8سم، 400غ

الحزم (حسب الأولوية):
1. صندوق صغير (20×15×10، أولوية=1)
2. صندوق متوسط (30×25×20، أولوية=2)

الخوارزمية:
حاول صندوق صغير: تناسب كلا العنصرين
النتيجة: 1× صندوق صغير (مُحسّن للعدد)
```

## أولوية الحزم

**تحدد الأولوية ترتيب التعبئة**:

أولوية 1 (أعلى): تُستخدم الحزم الصغيرة أولاً
أولوية 10: الحزم الكبيرة كخيار أخير

**الاستراتيجية**:
- الحزم الصغيرة = أرقام أولوية منخفضة (1-3)
- الحزم المتوسطة = أولوية متوسطة (4-6)
- الحزم الكبيرة = أرقام أولوية عالية (7-10)

**لماذا؟**: ابدأ بالحزمة الأصغر، وامتدّ إذا لزم الأمر → تقليل تكلفة الشحن.

## دقة سمك الجدار

قم بقياس التعبئة الفعلية:

**كيفية القياس**:
1. احصل على صندوق فارغ
2. قم بقياس الأبعاد الداخلية (الداخلية)
3. قم بقياس الأبعاد الخارجية (الخارجية)
4. حساب: `(الخارجية - الداخلية) / 2 = سمك الجدار`

**مثال**:
```
العرض الداخلي: 20سم
العرض الخارجي: 21سم
سمك الجدار: (21 - 20) / 2 = 0.5سم
```

**سمك الجدار الشائع**:
- الملصق المبطّن: 0.2-0.4سم
- ورق كرتوني ذو جدار واحد: 0.4-0.6سم
- ورق كرتوني ذو جدارين: 0.8-1.0سم

## إنشاء إعدادات الحزم

**خطوة بخطوة**:

1. الإعدادات > الشحن > حزم الشحن
2. اضغط على "إضافة حزمة شحن"
3. أدخل الاسم (مثال: "صندوق متوسط")
4. اختر النوع (صندوق أو ملصق)
5. أدخل الأبعاد الداخلية (L × W × H بالسم)
6. أدخل سمك الجدار (سم)
7. يقوم النظام بحساب الأبعاد الخارجية تلقائيًا
8. أدخل الوزن الفارغ (وزن الحزمة الفارغة بالجرام)
9. أدخل الوزن الأقصى (سعة الحمل بالجرام)
10. اختياري: أدخل التكلفة (لتحسين التكلفة)
11. حدد الأولوية (1-10)
12. تفعيل = نعم
13. حفظ

## اختبار اختيار الحزم

**اختبار يدوي**:
1. أضف منتجات لسلة الاختبار
2. انتقل إلى الخروج
3. اختر طريقة شحن في الوقت الفعلي (تستخدم الحزم)
4. تحقق من العرض السعر المعقول
5. تحقق من استجابة الناقل (تُظهر سجلات API الحزم المُختارة)

**مُعاينة التعبئة التلقائية**:
- بعض حسابات مزودي الشحن تُظهر تفكيك الحزم
- راجع أي حزم تم اختيارها لسلة
- تحقق من التعبئة الأمثل

---

## نصائح

- **قم بالقياس بدقة** - أبعاد غير دقيقة → عروض أسعار غير صحيحة
- **أضف سمك الجدار** - ضروري لوزن الأبعاد
- **ابدأ بأحجام 3-4** - الصغيرة، المتوسطة، الكبيرة تغطي معظم السيناريوهات
- **حدد أوزانًا قصوى واقعية** - سعة الصندوق، وليس الحد الأقصى النظري
- **استخدم الأولوية بحكمة** - الصناديق الصغيرة أولوية 1، الصناديق الكبيرة أولوية 10
- **قم بالاختبار مع منتجات حقيقية** - تحقق من اختيار التعبئة التلقائية للأحجام الصحيحة
- **تحديث عند تغيير التعبئة** - مزود جديد = قم بقياس الأبعاد مرة أخرى
- **فكر في العناصر الخاصة** - العناصر الهشة قد تحتاج إلى أحجام صناديق محددة
- **احتفظ بالحزم الفعالة قليلة** - الخيارات الزائدة تبطئ خوارزمية التعبئة التلقائية
- **وثّق التعبئة** - اكتب أي منتجات تناسب أي حزم
