---
title: ห้องสมุดสื่อ
---

ห้องสมุดสื่อคือศูนย์กลางสำหรับจัดการภาพ วิดีโอ โมเดล 3D และไฟล์ทั้งหมดที่ใช้ในร้านค้าของคุณ คุณสามารถอัปโหลดไฟล์ได้โดยการลากและวางไฟล์เข้ามา จัดระเบียบด้วยโฟลเดอร์และแท็ก และให้ระบบทำการปรับแต่งภาพอัตโนมัติเพื่อให้โหลดได้เร็วขึ้น

![ห้องแสดงสื่อ](/static/core/admin/img/help/media-library/media-gallery.webp)

## ตัวอย่างอินเทอร์เฟซ

ไปที่ **ห้องสมุดสื่อ** ด้านซ้ายเพื่อเปิดห้องแสดงสื่อ อินเทอร์เฟซมี 3 โซน:

| พื้นที่ | ตำแหน่ง | วัตถุประสงค์ |
|--------|---------|------------|
| **โซนอัปโหลด** | ด้านซ้ายบน | ลากและวางไฟล์เพื่ออัปโหลด (ภาพ วิดีโอ โมเดล 3D ขนาดไม่เกิน 100MB) |
| **โฟลเดอร์และแท็ก** | ด้านซ้าย ด้านล่าง | ดูโฟลเดอร์ กรองด้วยแท็ก เข้าถึงถังขยะ |
| **ตารางสื่อ** | พื้นที่หลัก | ค้นหา กรอง ดูและจัดการสื่อทั้งหมดของคุณ |

### ปุ่มควบคุมแถบเครื่องมือ

แถบเครื่องมือเหนือตารางสื่อให้คุณ:

- **ค้นหา** — ค้นหาสื่อโดยชื่อ ข้อความอัลต์ คำอธิบาย หรือชื่อแท็ก
- **กรองประเภท** — แสดงเฉพาะภาพ วิดีโอ หรือโมเดล 3D
- **กรองขนาด** — กรองตามขนาดไฟล์ (เล็ก ปานกลาง ใหญ่)
- **การดำเนินการแบบหมู่** — เลือกไอเท็ม แก้ไขรายละเอียด ลบไอเท็มที่เลือก
- **โหมดการดู** — ตาราง (ขนาดใหญ่) ตารางขนาดเล็ก หรือโหมดรายการ (จะถูกเก็บไว้ระหว่างเซสชัน)

## การอัปโหลดไฟล์

ลากไฟล์หนึ่งหรือหลายไฟล์ไปยังโซน **อัปโหลด** ด้านซ้าย หรือคลิกที่โซนเพื่อเปิดตัวเลือกไฟล์

### รูปแบบที่รองรับ

| ประเภท | รูปแบบ |
|--------|--------|
| **ภาพ** | JPEG, PNG, GIF, WebP, SVG, BMP, TIFF |
| **วิดีโอ** | MP4, WebM, MOV, MKV, AVI |
| **โมเดล 3D** | GLB, glTF |

### คิวอัปโหลด

เมื่ออัปโหลดไฟล์หลายไฟล์ คิวจัดการจะปรากฏขึ้นแสดง:

- ชื่อไฟล์และแถบความคืบหน้าการอัปโหลดของแต่ละไฟล์
- การอัปโหลดพร้อมกัน (สูงสุด 2 ไฟล์ในครั้งเดียวเพื่อประสิทธิภาพ)
- สถานะการประมวลผลเมื่อไฟล์ถูกปรับแต่งหลังการอัปโหลด
- ตัวเลือกในการยกเลิกการอัปโหลดแต่ละไฟล์ หรือล้างรายการที่เสร็จสิ้น

คิวสามารถลากได้และสามารถย่อขนาดเพื่อให้คุณสามารถทำงานต่อได้ขณะที่การอัปโหลดดำเนินต่อไป

## การปรับแต่งภาพอัตโนมัติ

ภาพทุกภาพที่คุณอัปโหลดจะถูกปรับแต่งอัตโนมัติ:

- **การแปลงเป็น WebP** — สร้างเวอร์ชัน WebP ร่วมกับไฟล์ต้นฉบับ (คุณภาพ 85%) เพื่อให้โหลดได้เร็วขึ้น
- **การสร้างภาพย่อ** — สร้างเวอร์ชันขนาดต่างๆ ตามการตั้งค่าภาพของคุณ
- **การปรับตำแหน่ง EXIF** — ภาพจะถูกหมุนอัตโนมัติให้อยู่ในตำแหน่งที่ถูกต้อง

การตั้งค่าระบบไม่สามารถเปลี่ยนชื่อหรือลบได้ คุณสามารถสร้างการตั้งค่าเพิ่มเติมได้ภายใต้ **ห้องสมุดสื่อ > การตั้งค่าขนาดภาพ** หากคุณต้องการขนาดที่ไม่ได้ครอบคลุมโดยค่าเริ่มต้น

### โหมดการตัด

| โหมด | พฤติกรรม |
|------|----------|
| **ครอบคลุม** | เติมพื้นที่ทั้งหมด ตัดขอบหากจำเป็น — เหมาะสำหรับบัตรและแบนเนอร์ |
| **ปรับให้พอดี** | แสดงภาพทั้งหมดในพื้นที่ เพิ่มพื้นที่โปร่งใสหากจำเป็น — เหมาะสำหรับภาพสินค้า |
| **ตัด** | ตัดตรงกลางให้ตรงกับขนาดที่กำหนด |
| **เพิ่มพื้นที่** | แสดงภาพและเพิ่มพื้นที่ (โปร่งใส, ขาว, ดำ) — เหมาะสำหรับโลโก้ |

## การจัดระเบียบไฟล์

### โฟลเดอร์

สร้างโฟลเดอร์เพื่อจัดระเบียบสื่อของคุณเป็นกลุ่มที่มีเหตุผล โฟลเดอร์สามารถซ้อนกันได้ในระดับใดก็ได้ คลิกโฟลเดอร์ด้านซ้ายเพื่อแสดงเฉพาะสื่อที่อยู่ในโฟลเดอร์นั้น ลิงก์ **ทั้งหมด** แสดงทุกอย่าง

### แท็ก

เพิ่มแท็กให้กับสื่อเพื่อจัดระเบียบข้ามโฟลเดอร์ได้อย่างยืดหยุ่น แท็กจะปรากฏในคลาวด์ด้านซ้าย คลิกแท็กเพื่อกรองสื่อตามแท็กนั้น สื่อสามารถมีแท็กหลายแท็กได้

### การค้นหา

แถบค้นหาจะค้นหาสื่อโดยชื่อ ข้อความอัลต์ คำอธิบาย หรือชื่อแท็ก รวมการค้นหาด้วยการกรองประเภทและขนาดเพื่อผลลัพธ์ที่แม่นยำ

## รายละเอียดสื่อ

คลิกที่สื่อเพื่อเปิดมุมมองรายละเอียดที่มีภาพขนาดใหญ่และข้อมูลทั้งหมด

![รายละเอียดสื่อ](/static/core/admin/img/help/media-library/media-detail.webp)

มุมมองรายละเอียดแสดง:

- **ภาพตัวอย่าง** — ภาพขนาดใหญ่พร้อมมิติเดิม
- **ข้อมูลไฟล์** — ประเภท มิติ ขนาดไฟล์ วันที่อัปโหลด
- **แท็บสำหรับการแก้ไข**:

| แท็บ | ฟิลด์ |
|-----|--------|
| **ทั่วไป** | ชื่อ, ข้อความอัลต์, คำอธิบาย (ทั้งหมดสามารถแปลได้สำหรับร้านค้าหลายภาษา) |
| **ทางเทคนิค** | MIME type, แฮชไฟล์, ชื่อไฟล์ต้นฉบับ, สถานะเวอร์ชัน WebP |
| **การจัดระเบียบ** | การกำหนดโฟลเดอร์, แท็ก, การสลับสาธารณะ/ส่วนตัว |
| **ขั้นสูง** | จุดโฟกัส, ไอดีภายนอก, JSON ข้อมูล |

### ฟิลด์ที่สามารถแปลได้

ชื่อ ข้อความอัลต์ และคำอธิบายรองรับการแปล คลิกไอคอนแปลที่อยู่ด้านข้างของแต่ละฟิลด์เพื่อเพิ่มการแปลสำหรับภาษาที่คุณเปิดใช้งาน นี่จะทำให้ภาพมีข้อความอัลต์และคำอธิบายที่ได้รับการแปลอย่างเหมาะสมสำหรับ SEO และการเข้าถึง

### การติดตามการใช้งาน

ระบบติดตามว่าแต่ละสื่อถูกใช้ในที่ใดบนแพลตฟอร์ม ส่วน **การใช้งานสื่อ** ที่ด้านล่างแสดงโมเดลและฟิลด์ทั้งหมดที่อ้างอิงสื่อชิ้นนี้ ช่วยให้คุณเข้าใจผลกระทบก่อนที่จะทำการเปลี่ยนแปลงหรือลบ

## การสนับสนุนวิดีโอ

วิดีโอที่อัปโหลดไปยังห้องสมุดสื่อจะถูกวิเคราะห์อัตโนมัติ:

- **การดึงข้อมูลเมตา** — ระยะเวลา ความละเอียด อัตราเฟรม bitrate และค่าเข้ารหัสจะถูกบันทึก
- **ภาพแสดงตัวอย่าง** — สร้างภาพย่อจากวิดีโอเพื่อแสดงตัวอย่าง
- **การสตรีม** — วิดีโอสนับสนุนการร้องขอช่วงเพื่อเลื่อนโดยไม่ต้องดาวน์โหลดไฟล์ทั้งหมด
- **การแปลงแบบเลือกได้** — วิดีโอสามารถแปลงเป็นรูปแบบ WebM/AV1 ที่ได้รับการปรับแต่งเพื่อส่งมอบได้เร็วขึ้น

## ถังขยะ

การลบสื่อจะย้ายมันไปยัง **ถังขยะ** แทนที่จะลบมันทิ้งไปโดยตรง นี่ช่วยป้องกันการลบโดยไม่ตั้งใจ

| การดำเนินการ | ทำอะไร |
|--------------|--------|
| **ลบ** | ย้ายสื่อไปยังถังขยะ (การลบแบบอ่อน) |
| **กู้คืน** | คืนสื่อที่ถูกลบไปยังตำแหน่งเดิม |
| **ลบถาวร** | ลบสื่อและภาพย่อทั้งหมดออกจากที่เก็บถาวร |
| **ลบถังขยะ** | ลบรายการทั้งหมดในถังขยะถาวร |

คลิก **ถังขยะ** ด้านซ้ายเพื่อดูและจัดการสื่อที่ถูกลบ

## ที่ใช้ห้องสมุดสื่อ

ห้องสมุดสื่อได้รับการผสานเข้ากับแพลตฟอร์มทั้งหมด:

| คุณสมบัติ | วิธีใช้สื่อ |
|-----------|------------|
| **แคตตาล็อกสินค้า** | ภาพสินค้า ภาพเวอร์ชันสินค้า แบนเนอร์หมวดหมู่ |
| **บล็อก** | ภาพหลัก, ภาพในเนื้อหาผ่าน CKEditor |
| **ตัวสร้างหน้า** | องค์ประกอบภาพ, ฉากหลังตัวอย่าง, องค์ประกอบแกลเลอรี |
| **ตัวสร้างหัว/ท้าย** | ภาพโลโก้, ฉากหลังภาพ |
| **การตั้งค่าเว็บไซต์** | โลโก้เว็บไซต์และfavicon |
| **ประกาศ** | ภาพประกาศและฉากหลัง |
| **CKEditor** | การอัปโหลดภาพทั้งหมดในเนื้อหาอุดมสมบูรณ์จะผ่านห้องสมุดสื่อ |
| **โปรแกรมสะสมแต้ม** | ภาพรางวัลและระดับ |

เมื่อคุณเลือกภาพในคุณสมบัติเหล่านี้ ห้องแสดงสื่อจะเปิดขึ้นในรูปแบบโมดัลเพื่อให้คุณเลือกและดูภาพได้ง่ายขึ้น

## เคล็ดลับ

- **ใช้ชื่อและข้อความอัลต์ที่อธิบายได้ดี** — เมตาดาต้าที่ดีช่วยเพิ่ม SEO และการเข้าถึง ระบบใช้ข้อความอัลต์ในแท็กภาพทั่วทั้งร้านค้า
- **จัดระเบียบด้วยโฟลเดอร์ตั้งแต่แรก** — สร้างโครงสร้างโฟลเดอร์ (เช่น สินค้า, บล็อก, แบนเนอร์, โลโก้) ก่อนอัปโหลดไฟล์จำนวนมาก มันง่ายกว่ามากในการจัดระเบียบขณะอัปโหลดมากกว่าการจัดระเบียบใหม่ในภายหลัง
- **ใช้แท็กสำหรับหมวดหมู่ที่ข้ามกัน** — แท็กเช่น "ฤดูกาล", "ลดราคา", หรือ "ไลฟ์สไตล์" ช่วยให้คุณค้นหาสื่อที่ครอบคลุมหลายโฟลเดอร์ได้
- **ตรวจสอบการใช้งานก่อนการลบ** — ส่วนติดตามการใช้งานแสดงว่าสื่อถูกอ้างอิงที่ใด การลบสื่อที่ถูกใช้งานอาจทำให้เกิดภาพที่ไม่สามารถใช้งานได้ในร้านค้าของคุณ
- **ปล่อยให้ WebP ทำงาน** — การแปลงเป็น WebP อัตโนมัติมักจะลดขนาดไฟล์ลง 25-35% เมื่อเทียบกับ JPEG โดยไม่มีการสูญเสียคุณภาพที่เห็นได้ชัดเจน คุณไม่จำเป็นต้องแปลงภาพด้วยตนเองก่อนอัปโหลด
- **สร้างการตั้งค่าแบบกำหนดเอง** — หากคุณมีรูปแบบที่ไม่เหมือนใครที่ต้องการขนาดภาพเฉพาะ ให้สร้างการตั้งค่าแบบกำหนดเองแทนที่จะปรับขนาดภาพด้วยตนเอง
