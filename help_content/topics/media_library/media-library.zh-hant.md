---
title: 媒體資源庫
---

媒體資源庫是管理您商店中所有圖片、視頻、3D模型和文件的中心樞紐。您可以拖放文件進行上傳，使用文件夾和標籤進行整理，並讓系統自動優化圖片以實現快速加載。

![媒體資源庫](/static/core/admin/img/help/media-library/media-gallery.webp)

## 資源庫介面

在側邊欄導航至 **媒體資源庫** 以打開資源庫。介面包含三個區域：

| 區域 | 位置 | 目的 |
|------|----------|---------|
| **上傳區域** | 左側側邊欄頂部 | 拖放文件以上傳（圖片、視頻、3D模型最多100MB） |
| **文件夾與標籤** | 左側側邊欄下方 | 浏覽文件夾、按標籤過濾、訪問回收站 |
| **媒體網格** | 主要區域 | 搜索、過濾、浏覽和管理所有資源 |

### 工具欄控制

媒體網格上方的工具欄提供：

- **搜索** — 按標題、替代文字、描述或標籤名稱查找資源
- **類型過濾** — 僅顯示圖片、視頻或3D模型
- **尺寸過濾** — 按文件尺寸過濾（小、中、大）
- **批量操作** — 選擇項目、編輯細節、刪除選定項目
- **視圖模式** — 網格（大）、小網格或列表視圖（會在會話中保留）

## 上傳文件

將一個或多個文件拖放到左側側邊欄的 **上傳** 區域，或點擊該區域以打開文件選擇器。

### 支持的格式

| 類型 | 格式 |
|------|---------|
| **圖片** | JPEG、PNG、GIF、WebP、SVG、BMP、TIFF |
| **視頻** | MP4、WebM、MOV、MKV、AVI |
| **3D模型** | GLB、glTF |

### 上傳隊列

當上傳多個文件時，會出現一個隊列管理器，顯示：

- 每個文件的名稱和上傳進度條
- 同時上傳（為提高性能，每次最多上傳2個）
- 文件上傳後優化處理的狀態
- 取消個別上傳或清除已完成項目的選項

隊列可以拖動並最小化，這樣您可以在上傳完成時繼續工作。

## 自動圖片優化

您上傳的每張圖片都會自動優化：

- **WebP 轉換** — 生成一個 WebP 版本與原始文件一起（品質 85%），以加快加載速度
- **縮略圖生成** — 根據您的圖片預設生成多個尺寸的版本
- **EXIF 方向** — 圖片會自動旋轉到正確的方向

### 壓縮模式

| 模式 | 行為 |
|------|----------|
| **覆蓋** | 填滿整個區域，如有需要會裁剪邊緣 — 適合卡片和橫幅 |
| **包含** | 將整個圖片放入區域，如有需要會添加透明空間 — 適合產品圖片 |
| **裁剪** | 中心裁剪至精確尺寸 |
| **填充** | 將圖片放入並添加填充（透明、白色或黑色） — 適合標誌 |

## 組織文件

### 文件夾

創建文件夾以將媒體分組為邏輯群組。文件夾可以嵌套到任意深度。點擊左側側邊欄中的文件夾以僅顯示該文件夾內的資源。**所有文件** 連結顯示所有資源。

### 標籤

為資源添加標籤以實現跨文件夾的靈活組織。標籤會以雲的形式出現在左側側邊欄。點擊標籤以按該標籤過濾資源。資源可以有多个標籤。

### 搜索

搜索欄可以按標題、替代文字、描述或標籤名稱查找資源。將搜索與類型和尺寸過濾器結合使用，以獲得精確的結果。

## 資源詳情

點擊資源以打開其詳情視圖，包含大預覽和完整的元數據。

![資源詳情](/static/core/admin/img/help/media-library/media-detail.webp)

詳情視圖顯示：

- **預覽** — 原始尺寸的大圖片預覽
- **文件信息** — 類型、尺寸、文件大小、上傳日期
- **編輯標籤**：

| 標籤 | 欄位 |
|-----|--------|
| **一般** | 標題、替代文字、描述（所有欄位都支持多語言商店的翻譯） |
| **技術** | MIME 類型、文件哈希、原始文件名、WebP 版本狀態 |
| **組織** | 文件夾分配、標籤、公開/私有切換 |
| **高級** | 聚焦點坐標、外部 ID、元數據 JSON |

### 可翻譯欄位

標題、替代文字和描述支持翻譯。點擊每個欄位旁的翻譯圖標，為您啟用的語言添加翻譯。這確保圖片在 SEO 和可訪問性方面有正確的本地化替代文字和描述。

### 使用情況追蹤

系統會追蹤每個資源在平台上的使用情況。底部的 **資源使用情況** 顯示所有引用此資源的模型和欄位，幫助您在進行更改或刪除前了解其影響。

## 視頻支持

上傳到媒體資源庫的視頻會自動分析：

- **元數據提取** — 獲取持續時間、解析度、幀率、比特率和編解碼器
- **預覽圖片** — 從視頻中生成預覽圖片
- **串流** — 視頻支持範圍請求，可在不下載整個文件的情況下進行尋找
- **可選轉換** — 可將視頻轉換為優化的 WebM/AV1 格式以實現更快的傳輸

## 回收站

刪除資源會將其移動到 **回收站** 而不是永久刪除。這可以防止意外刪除。

| 操作 | 功能 |
|--------|-------------|
| **刪除** | 將資源移動到回收站（軟刪除） |
| **恢復** | 將已刪除的資源返回到其原始位置 |
| **永久刪除** | 永久刪除資源及其所有縮略圖 |
| **清空回收站** | 永久刪除回收站中的所有項目 |

點擊左側側邊欄的 **回收站** 來查看和管理已刪除的資源。

## 媒體資源庫的使用場景

媒體資源庫整合在整個平台中：

| 功能 | 如何使用媒體 |
|---------|------------------|
| **產品目錄** | 產品圖片、變體圖片、分類橫幅 |
| **部落格** | 特色圖片、CKEditor 中的內容圖片 |
| **頁面構建器** | 圖片元素、首屏背景、畫廊組件 |
| **首頁/頁尾構建器** | 標誌圖片、背景圖片 |
| **網站設置** | 網站標誌和favicon |
| **公告** | 公告圖片和背景 |
| **CKEditor** | 所有富文本圖片上傳都通過媒體資源庫進行 |
| **忠誠度計劃** | 優惠和等級圖片 |

當您在這些功能中選擇圖片時，媒體資源庫畫廊會以彈窗形式打開，方便瀏覽和選擇。

## 小技巧

- **使用描述性的標題和替代文字** — 良好的元數據可以提高 SEO 和可訪問性。系統會在整個商店前端的圖片標籤中使用替代文字。
- **早期使用文件夾進行組織** — 在上傳大量文件之前，創建文件夾結構（例如，產品、部落格、橫幅、標誌）。邊上傳邊整理比之後重新整理要容易得多。
- **使用標籤進行跨領域分類** — 使用如「季節性」、「促銷」或「生活方式」等標籤，幫助您找到跨多個文件夾的資源。
- **刪除前檢查使用情況** — 使用情況追蹤部分會顯示資源被引用的位置。刪除被使用的資源可能會導致商店前端出現破損的圖片。
- **讓 WebP 來處理** — 自動的 WebP 轉換通常比 JPEG 減少 25-35% 的文件大小，且沒有可見的品質損失。您不需要在上傳前手動轉換圖片。
- **創建自定義預設** — 如果您有需要特定圖片尺寸的獨特佈局，請創建自定義預設，而不是手動調整圖片尺寸。
