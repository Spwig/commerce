---
title: 媒体库
---

媒体库是管理您商店中所有图片、视频、3D模型和文件的中心枢纽。通过拖拽上传文件，使用文件夹和标签进行整理，系统会自动优化图片以实现快速加载。

![媒体库](/static/core/admin/img/help/media-library/media-gallery.webp)

## 画廊界面

在侧边栏导航到 **媒体库** 以打开画廊。界面有三个区域：

| 区域 | 位置 | 用途 |
|------|----------|---------|
| **上传区域** | 左侧边栏顶部 | 拖放文件进行上传（图片、视频、3D模型最大100MB） |
| **文件夹与标签** | 左侧边栏下方 | 浏览文件夹，按标签筛选，访问回收站 |
| **媒体网格** | 主区域 | 搜索、筛选、浏览和管理所有资产 |

### 工具栏控制

媒体网格上方的工具栏提供：

- **搜索** — 按标题、替代文本、描述或标签名称查找资产
- **类型筛选** — 仅显示图片、视频或3D模型
- **尺寸筛选** — 按文件尺寸（小、中、大）筛选
- **批量操作** — 选择项目、编辑详细信息、删除选中项
- **视图模式** — 网格（大）、小网格或列表视图（跨会话保留）

## 上传文件

将一个或多个文件拖到左侧边栏的 **上传** 区域，或点击该区域打开文件选择器。

### 支持的格式

| 类型 | 格式 |
|------|---------|
| **图片** | JPEG, PNG, GIF, WebP, SVG, BMP, TIFF |
| **视频** | MP4, WebM, MOV, MKV, AVI |
| **3D模型** | GLB, glTF |

### 上传队列

上传多个文件时，会出现一个队列管理器，显示：

- 每个文件的名称和上传进度条
- 并发上传（为性能考虑，每次最多2个）
- 文件上传后优化处理状态
- 可取消单个上传或清除已完成的项目

队列可拖动并可最小化，这样您可以在上传完成时继续工作。

## 自动图片优化

您上传的每张图片都会自动优化：

- **WebP转换** — 生成一个WebP版本，与原始文件一起（质量85%），以加快加载速度
- **缩略图生成** — 根据您的图片预设创建多个尺寸的版本
- **EXIF方向** — 图片会自动旋转到正确的方向

### 裁剪模式

| 模式 | 行为 |
|------|----------|
| **覆盖** | 填充整个区域，如有需要会裁剪边缘 — 适用于卡片和横幅 |
| **包含** | 将整个图像适应到区域内，如有需要会添加透明空间 — 适用于产品图片 |
| **裁剪** | 中心裁剪到精确尺寸 |
| **填充** | 适应图像并添加填充（透明、白色或黑色） — 适用于Logo |

## 组织文件

### 文件夹

创建文件夹以将媒体组织成逻辑组。文件夹可以嵌套到任意深度。点击左侧边栏中的文件夹以仅显示其中的资产。**所有文件** 链接显示所有内容。

### 标签

为资产添加标签，以便灵活地跨文件夹组织。标签以云的形式出现在左侧边栏中。点击标签以按该标签筛选资产。资产可以有多个标签。

### 搜索

搜索栏可以通过标题、替代文本、描述或标签名称查找资产。结合搜索与类型和尺寸筛选器以获得精确结果。

## 资产详情

点击资产以打开其详细视图，包含大预览和完整的元数据。

![资产详情](/static/core/admin/img/help/media-library/media-detail.webp)

详细视图显示：

- **预览** — 原始尺寸的大图像预览
- **文件信息** — 类型、尺寸、文件大小、上传日期
- **用于编辑的标签**：

| 标签 | 字段 |
|-----|--------|
| **常规** | 标题、替代文本、描述（所有字段都支持多语言商店的翻译） |
| **技术** | MIME类型、文件哈希、原始文件名、WebP版本状态 |
| **组织** | 文件夹分配、标签、公共/私有切换 |
| **高级** | 焦点点坐标、外部ID、元数据JSON |

### 可翻译字段

标题、替代文本和描述支持翻译。点击每个字段旁边的翻译图标，为已启用的语言添加翻译。这确保了图片在SEO和可访问性方面有适当的本地化替代文本和描述。

### 使用情况跟踪

系统会跟踪每个资产在平台上的使用情况。底部的 **媒体使用情况** 部分显示所有引用此资产的模型和字段，帮助您在进行更改或删除前了解其影响。

## 视频支持

上传到媒体库的视频会自动分析：

- **元数据提取** — 捕获时长、分辨率、帧率、比特率和编解码器
- **海报图像** — 从视频中生成缩略图用于预览
- **流媒体** — 视频支持范围请求，无需下载整个文件即可进行播放
- **可选转换** — 视频可以转换为优化的WebM/AV1格式以实现更快的传输

## 回收站

删除资产会将其移动到 **回收站** 而不是永久删除。这可以防止意外删除。

| 操作 | 功能 |
|--------|-------------|
| **删除** | 将资产移动到回收站（软删除） |
| **恢复** | 将已删除的资产返回到其原始位置 |
| **永久删除** | 永久删除资产及其所有缩略图 |
| **清空回收站** | 永久删除回收站中的所有项目 |

点击左侧边栏的 **回收站** 以查看和管理已删除的资产。

## 媒体库的使用场景

媒体库在整个平台中集成：

| 功能 | 如何使用媒体 |
|---------|------------------|
| **产品目录** | 产品图片、变体图片、分类横幅 |
| **博客** | 特征图片、通过CKEditor在内容中插入的图片 |
| **页面构建器** | 图像元素、横幅背景、画廊组件 |
| **页眉/页脚构建器** | Logo图片、背景图片 |
| **网站设置** | 网站Logo和favicon |
| **公告** | 公告图片和背景 |
| **CKEditor** | 所有富文本图片上传都通过媒体库进行 |
| **忠诚度计划** | 奖励和等级图片 |

当您在这些功能中选择图片时，媒体库画廊会以模态窗口的形式打开，便于浏览和选择。

## 小贴士

- **使用描述性标题和替代文本** — 良好的元数据可以提高SEO和可访问性。系统会在整个商店前端的图片标签中使用替代文本。
- **尽早使用文件夹进行组织** — 在上传大量文件之前，创建一个文件夹结构（例如，产品、博客、横幅、Logo）。在上传过程中进行组织比之后重新组织要容易得多。
- **使用标签进行跨类别分类** — 标签如“季节性”、“促销”或“生活方式”可以帮助您找到跨越多个文件夹的资产。
- **删除前检查使用情况** — 使用情况跟踪部分会显示资产被引用的位置。删除被使用的资产可能会导致商店前端出现损坏的图片。
- **让WebP来处理** — 自动的WebP转换通常比JPEG减少25-35%的文件大小，且没有可见的质量损失。您不需要在上传前手动转换图片。
- **创建自定义预设** — 如果您有一个需要特定图片尺寸的独特布局，请创建自定义预设，而不是手动调整图片尺寸。
