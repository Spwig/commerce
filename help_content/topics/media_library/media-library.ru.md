---
title: Библиотека медиа
---

Библиотека медиа является центральным хабом для управления всеми изображениями, видео, 3D-моделями и файлами, используемыми в вашем магазине. Загружайте файлы, перетаскивая их сюда, организуйте их с помощью папок и тегов, а также позволяйте системе автоматически оптимизировать изображения для быстрой загрузки.

![Галерея медиа](/static/core/admin/img/help/media-library/media-gallery.webp)

## Интерфейс галереи

Перейдите в **Библиотека медиа** в боковой панели, чтобы открыть галерею. Интерфейс имеет три области:

| Область | Расположение | Назначение |
|--------|-------------|----------|
| **Зона загрузки** | Левая боковая панель, верхняя часть | Перетащите и отпустите файлы для загрузки (изображения, видео, 3D-модели до 100 МБ) |
| **Папки и теги** | Левая боковая панель, ниже | Обзор папок, фильтрация по тегам, доступ к корзине для удаленных файлов |
| **Сетка медиа** | Основная область | Поиск, фильтрация, просмотр и управление всеми вашими активами |

### Контроли панели инструментов

Панель инструментов над сеткой медиа предоставляет:

- **Поиск** — найти активы по названию, альтернативному тексту, описанию или названию тега
- **Фильтр по типу** — показывать только Изображения, Видео или 3D-Модели
- **Фильтр по размеру** — фильтровать по размеру файла (Маленький, Средний, Большой)
- **Массовые действия** — Выбрать элементы, Редактировать детали, Удалить выбранные
- **Режимы просмотра** — Сетка (крупный), Мелкая сетка или режим просмотра в виде списка (сохраняется между сеансами)

## Загрузка файлов

Перетащите один или несколько файлов в зону **Загрузка** в левой боковой панели или нажмите на эту зону, чтобы открыть выбор файла.

### Поддерживаемые форматы

| Тип | Форматы |
|-----|--------|
| **Изображения** | JPEG, PNG, GIF, WebP, SVG, BMP, TIFF |
| **Видео** | MP4, WebM, MOV, MKV, AVI |
| **3D-Модели** | GLB, glTF |

### Очередь загрузки

При загрузке нескольких файлов появляется менеджер очереди, показывающий:

- Название каждого файла и полосу прогресса загрузки
- Параллельные загрузки (до 2 одновременно для обеспечения производительности)
- Статус обработки, когда файлы оптимизируются после загрузки
- Возможность отменить отдельные загрузки или очистить завершенные элементы

Очередь можно перемещать и сворачивать, чтобы вы могли продолжать работу, пока загрузка завершается.

## Автоматическая оптимизация изображений

Каждое изображение, которое вы загружаете, автоматически оптимизируется:

- **Конвертация в WebP** — создается версия WebP вместе с оригиналом (качество 85%) для более быстрой загрузки
- **Генерация миниатюр** — создаются несколько версий разного размера на основе ваших предустановок изображений
- **Ориентация EXIF** — изображения автоматически поворачиваются в правильную ориентацию

### Режимы обрезки

| Режим | Поведение |
|------|----------|
| **Обрезка** | Заполняет всю область, обрезая края при необходимости — хорошо подходит для карточек и баннеров |
| **Вписать** | Вписывает полное изображение в область, добавляя прозрачное пространство при необходимости — хорошо подходит для изображений товаров |
| **Обрезка** | Центрированная обрезка до точных размеров |
| **Добавить** | Вписывает изображение и добавляет отступы (прозрачные, белые или черные) — хорошо подходит для логотипов |

## Организация файлов

### Папки

Создавайте папки, чтобы организовать ваше медиа в логические группы. Папки могут быть вложенными на любую глубину. Нажмите на папку в левой боковой панели, чтобы показать только активы внутри нее. Ссылка **Все файлы** показывает все активы.

### Теги

Добавьте теги к активам для гибкой организации по папкам. Теги отображаются в облаке в левой боковой панели. Нажмите на тег, чтобы отфильтровать активы по этому тегу. Активы могут иметь несколько тегов.

### Поиск

Панель поиска находит активы по названию, альтернативному тексту, описанию или названию тега. Сочетайте поиск с фильтрами по типу и размеру для получения точных результатов.

## Детали активов

Нажмите на актив, чтобы открыть его подробный вид с крупным предварительным просмотром и полной метадатой.

![Детали активов](/static/core/admin/img/help/media-library/media-detail.webp)

Детальный вид показывает:

- **Предварительный просмотр** — крупный предварительный просмотр изображения с оригинальными размерами
- **Информация о файле** — тип, размеры, размер файла, дата загрузки
- **Вкладки для редактирования**:

| Вкладка | Поля |
|--------|------|
| **Общие** | Название, Альтернативный текст, Описание (все переводимые для магазинов с несколькими языками) |
| **Технические** | MIME-тип, хэш файла, исходное имя файла, статус версии WebP |
| **Организация** | Назначение папки, теги, переключатель публичный/частный |
| **Расширенные** | Координаты точки фокуса, внешний идентификатор, JSON-метаданные |

### Переводимые поля

Название, альтернативный текст и описание поддерживают переводы. Нажмите значок перевода рядом с каждым полем, чтобы добавить переводы для включенных языков. Это гарантирует, что изображения имеют правильно локализованный альтернативный текст и описания для SEO и доступности.

### Отслеживание использования

Система отслеживает, где каждый актив используется по всей платформе. Раздел **Использования медиа** внизу показывает каждый модель и поле, которые ссылаются на этот актив, помогая вам понять влияние перед внесением изменений или удалением.

## Поддержка видео

Видео, загруженные в библиотеку медиа, автоматически анализируются:

- **Извлечение метаданных** — длительность, разрешение, частота кадров, битрейт и кодеки захватываются
- **Изображение постера** — миниатюра создается из видео для предварительного просмотра
- **Потоковая передача** — видео поддерживают запросы диапазона для поиска без загрузки всего файла
- **Опциональная конвертация** — видео могут быть преобразованы в оптимизированный формат WebM/AV1 для более быстрой доставки

## Корзина для удаленных файлов

Удаление актива перемещает его в **Корзину для удаленных файлов**, вместо того чтобы полностью удалять его. Это защищает от случайного удаления.

| Действие | Что оно делает |
|--------|-------------|
| **Удалить** | Перемещает актив в Корзину для удаленных файлов (мягкое удаление) |
| **Восстановить** | Возвращает удаленный актив в его исходное место |
| **Постоянное удаление** | Удаляет актив и все его миниатюры из хранилища навсегда |
| **Очистить корзину** | Постоянно удаляет все элементы из Корзины для удаленных файлов |

Нажмите **Корзина для удаленных файлов** в левой боковой панели, чтобы просмотреть и управлять удаленными активами.

## Где используется библиотека медиа

Библиотека медиа интегрирована по всей платформе:

| Особенность | Как используется медиа |
|------------|------------------|
| **Каталог товаров** | Изображения товаров, изображения вариантов, баннеры категорий |
| **Блог** | Изображения, используемые в содержании через CKEditor |
| **Конструктор страниц** | Элементы изображений, фоновые изображения героя, компоненты галереи |
| **Конструктор заголовка/подвала** | Изображения логотипа, фоновые изображения |
| **Настройки сайта** | Логотип сайта и значок закладки |
| **Объявления** | Изображения объявлений и фоновые изображения |
| **CKEditor** | Все загрузки изображений в тексте с богатым форматом проходят через библиотеку медиа |
| **Программа лояльности** | Изображения наград и уровней |

Когда вы выбираете изображение в любой из этих функций, галерея библиотеки медиа открывается как модальное окно для удобного просмотра и выбора.

## Советы

- **Используйте описательные названия и альтернативный текст** — хорошая метаданные улучшает SEO и доступность. Система использует альтернативный текст в тегах изображений по всей точке продаж.
- **Организуйте с помощью папок заранее** — создайте структуру папок (например, Товары, Блог, Баннеры, Логотипы) до загрузки большого количества файлов. Гораздо проще организовывать по ходу, чем пересортировывать позже.
- **Используйте теги для пересекающихся категорий** — теги вроде "сезонные", "распродажа" или "стиль жизни" помогают находить активы, которые охватывают несколько папок.
- **Проверьте использование перед удалением** — раздел отслеживания использования показывает, где актив используется. Удаление активного актива может оставить разорванные изображения на вашей точке продаж.
- **Позвольте WebP делать работу** — автоматическая конвертация в WebP обычно уменьшает размеры файлов на 25-35% по сравнению с JPEG без видимой потери качества. Вам не нужно вручную конвертировать изображения перед загрузкой.
- **Создавайте пользовательские предустановки** — если у вас есть уникальная компоновка, требующая определенного размера изображения, создайте пользовательскую предустановку вместо ручного изменения размера изображений.
