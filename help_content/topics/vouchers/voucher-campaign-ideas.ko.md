title: 쿠폰 캠페인 아이디어
---

이 가이드는 몇 분 안에 설정할 수 있는 검증된 쿠폰 캠페인을 다룹니다. 각 캠페인에는 정확한 설정이 포함되어 있어 아이디어에서 실제 프로모션으로의 전환을 빠르게 할 수 있습니다. 모든 쿠폰은 **마케팅 > 쿠폰**에서 생성하고 **+ 쿠폰 추가**를 클릭하세요.

![환영 쿠폰](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. 신규 고객을 위한 할인

처음 구매하는 고객에게 구매를 완료하도록 유도하는 인센티브를 제공하세요. 환영 할인은 고객이 귀하의 가게에서 주문한 적이 없는 경우 신뢰를 구축하고 망설임을 줄입니다.

| 필드 | 값 |
|-------|-------|
| **코드** | `WELCOME15` |
| **이름** | 환영 할인 — 신규 고객 |
| **할인 유형** | 비율 할인 |
| **할인 금액** | 15 |
| **적용 범위** | 전체 장바구니 |
| **고객당 최대 사용 횟수** | 1 |
| **총 최대 사용 횟수** | 0 (무제한) |
| **신규 고객만 해당** | 체크됨 |
| **시작일 / 종료일** | 둘 다 비워두세요 — 항상 활성화됨 |

이 코드를 가입 확인 이메일, 홈페이지 배너, 또는 첫 방문자용 팝업에서 홍보하세요. **신규 고객만 해당**이 체크되어 있기 때문에, 기존 고객이 코드를 발견하더라도 사용할 수 없습니다.

## 2. 무료 배송 기준

장바구니가 최소 금액에 도달하면 무료 배송을 제공하여 더 큰 주문을 유도하세요. **할인 금액**을 귀하의 최대 표준 배송 비용을 커버할 수 있는 금액으로 설정하세요.

| 필드 | 값 |
|-------|-------|
| **코드** | `FREESHIP50` |
| **이름** | 50달러 이상 주문 시 무료 배송 |
| **할인 유형** | 고정 금액 할인 |
| **할인 금액** | 15.00 (또는 귀하의 최대 배송 비용) |
| **적용 범위** | 전체 장바구니 |
| **최소 주문 금액** | 50.00 |
| **고객당 최대 사용 횟수** | 0 (무제한) |
| **다른 쿠폰과 중복 사용 불가** | 체크됨 |

제품 페이지와 장바구니에 이 코드를 두드러지게 표시하세요. 기준에 가까운 고객은 종종 다른 항목을 추가하여 자격을 얻으려 하며, 이는 평균 주문 금액을 증가시킬 수 있습니다.

## 3. 계절 캠페인 코드

휴일, 계절, 또는 이벤트와 연계된 제한된 기간의 캠페인을 진행하세요. 엄격한 날짜 범위를 설정하여 쿠폰이 자동으로 활성화되고 만료되도록 하세요.

| 필드 | 값 |
|-------|-------|
| **코드** | `SUMMER25` |
| **이름** | 2025 여름 세일 |
| **할인 유형** | 비율 할인 |
| **할인 금액** | 25 |
| **최대 할인 금액** | 50.00 |
| **적용 범위** | 전체 장바구니 |
| **시작일** | 6월 1일 |
| **종료일** | 8월 31일 |
| **다른 쿠폰과 중복 사용 불가** | 체크됨 |
| **할인 상품과 중복 사용 불가** | 체크됨 |

**공유 코드 vs. 고유 코드** — `SUMMER25`와 같은 공유 코드는 홍보하기가 간단하지만, 의도한 대상 그룹을 벗어날 수 있습니다. 더 엄격한 통제를 원하시면 대량 생성을 사용하여 고유한 단회 사용 코드(각 코드의 **총 최대 사용 횟수**를 1로 설정)를 생성하세요. 고유한 코드는 타겟팅된 이메일을 통해 분배하여 모든 고객이 각자 고유한 코드를 받을 수 있도록 하세요.

## 4. 충성도 보상 쿠폰

최고의 고객에게 다음 주문에 적용할 수 있는 고정 금액의 쿠폰으로 보상하세요. 고객이 소비 마일스톤에 도달한 후 이메일로 보내거나, 반복 구매에 대한 감사로 보내세요.

| 필드 | 값 |
|-------|-------|
| **코드** | `VIP20` |
| **이름** | VIP 충성도 보상 |
| **할인 유형** | 고정 금액 할인 |
| **할인 금액** | 20.00 |
| **적용 범위** | 전체 장바구니 |
| **최소 주문 금액** | 100.00 |
| **고객당 최대 사용 횟수** | 1 |
| **총 최대 사용 횟수** | 0 (무제한) |
| **유효 기간** | 90일 |

**유효 기간**을 90일로 설정하면 고객은 첫 사용일로부터 3개월 동안 보상을 사용할 수 있습니다. **최소 주문 금액** 100달러는 할인이 의미 있는 주문에 적용되도록 보장합니다.

## 5. 인플루언서 / 추천 코드

인플루언서나 브랜드 대사에게 그들의 청중과 공유할 수 있는 개인 코드를 제공하세요. 각 코드는 한 사람과 연결되어 있어, 어떤 파트너가 가장 많은 판매를 이끌고 있는지 쉽게 추적할 수 있습니다.


| Field | Value |
|-------|-------|
| **Code** | `SARAH10` |
| **Name** | 인플루언서 — Sarah |
| **Discount type** | 비율 할인 |
| **Discount value** | 10 |
| **Application scope** | 전체 장바구니 |
| **Max uses total** | 0 (무제한) |
| **Max uses per customer** | 1 |

각 인플루언서에게 별도의 쿠폰을 생성하여 쿠폰 대시보드에서 사용 횟수를 비교할 수 있습니다. 각 코드의 사용량을 정기적으로 확인하여 ROI를 측정하고 어떤 파트너십을 계속할지 결정하세요.

## 6. 플래시 할인 코드

짧은 기간 동안 제공되는 깊은 할인으로 긴장감을 유도하세요. 코드를 함께 사용하여 공지 표시줄이나 이메일 블라스트를 통해 즉시 트래픽을 유도하세요.

| Field | Value |
|-------|-------|
| **Code** | `FLASH40` |
| **Name** | 24시간 플래시 세일 |
| **Discount type** | 비율 할인 |
| **Discount value** | 40 |
| **Max discount amount** | 75.00 |
| **Application scope** | 전체 장바구니 |
| **Start date** | 금요일 오전 9시 |
| **End date** | 토요일 오전 9시 |
| **Max uses total** | 200 |
| **Exclude sale items** | 체크됨 |

**Max uses total**을 200으로 제한하여 이익을 보호합니다. 제한이 도달하면 코드가 자동으로 작동을 중단합니다. 마케팅에서 제한된 수량을 언급하세요 — "첫 200명 고객만" — 긴장감을 증폭시킵니다.

## 캠페인 빠른 참고

| 캠페인 | 코드 예시 | 유형 | 값 | 주요 설정 |
|----------|-------------|------|-------|--------------|
| 환영 할인 | `WELCOME15` | 비율 | 15% | 최초 사용만, 고객당 1회 |
| 무료 배송 | `FREESHIP50` | 고정 금액 | $15 | 최소 주문 금액 $50, 결합 불가 |
| 계절 세일 | `SUMMER25` | 비율 | 25% | 날짜 범위, 최대 할인 $50 |
| 충성도 보상 | `VIP20` | 고정 금액 | $20 | 최소 주문 금액 $100, 90일 유효 |
| 인플루언서 코드 | `SARAH10` | 비율 | 10% | 고객당 1회, 코드로 추적 |
| 플래시 할인 | `FLASH40` | 비율 | 40% | 24시간 기간, 최대 사용 200회 |

## 팁

- **코드를 짧고 기억하기 쉬운 형태로 유지하세요** — 대문자만 사용하고 특수 문자는 사용하지 마세요. 고객은 체크아웃 시 이 코드를 입력해야 하므로 `SUMMER25`가 `summer-sale-2025-promo`보다 더 좋습니다.
- **사용 횟수를 추적하세요** — 쿠폰 대시보드를 정기적으로 확인하여 어떤 캠페인의 성과가 가장 좋은지 확인하세요. 총 사용 횟수를 매출과 비교하여 ROI를 계산하세요.
- **출시 전에 테스트하세요** — 코드를 사용하여 테스트 주문을 생성하여 할인이 올바르게 적용되고 최소 주문 금액을 준수하며 예상대로 결합(또는 결합하지 않음)하는지 확인하세요.
- **반복 캠페인에 코드를 교체하세요** — 계절마다 새로운 코드를 사용하세요(예: `SUMMER25`, `SUMMER26`)하여 각 캠페인을 독립적으로 측정하고 오래된 코드가 유통되는 것을 방지하세요.
- **비율 쿠폰에 최대 할인 금액을 설정하세요** — $1,000 주문에 25% 할인은 $250입니다. **최대 할인 금액** 필드를 사용하여 할인 금액을 제한하고 이익을 보호하세요.
- **공지 표시줄을 사용하세요** — 플래시 및 계절 쿠폰을 가상 매장의 공지 표시줄과 함께 사용하여 모든 방문자가 즉시 프로모션을 볼 수 있도록 하세요.
