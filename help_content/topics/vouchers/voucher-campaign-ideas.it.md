---
title: Idee per Campagne con Buoni
---

Queste linee guida coprono campagne con buoni provate che puoi configurare in pochi minuti. Ogni campagna include le impostazioni esatte da configurare, in modo da passare rapidamente dall'idea a una promozione attiva. Crea tutti i buoni in **Marketing > Buoni** e fai clic su **+ Aggiungi Buono**.

![Buono di Benvenuto](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. Sconto di Benvenuto per Nuovi Clienti

Offri un incentivo ai clienti che effettuano il primo acquisto per completare l'acquisto. Uno sconto di benvenuto costruisce fiducia e riduce l'esitazione nei clienti che non hanno mai ordinato dal tuo negozio.

| Campo | Valore |
|-------|-------|
| **Codice** | `WELCOME15` |
| **Nome** | Sconto di benvenuto — nuovi clienti |
| **Tipo di sconto** | Sconto Percentuale |
| **Valore dello sconto** | 15 |
| **Ambito di applicazione** | Carrello Intero |
| **Utilizzi massimi per cliente** | 1 |
| **Utilizzi massimi totali** | 0 (illimitato) |
| **Solo per clienti nuovi** | Spuntato |
| **Data di inizio / Data di fine** | Lascia entrambe vuote — sempre attivo |

Promoziona questo codice nell'email di conferma dell'iscrizione, nel banner della homepage o in un pop-up per i visitatori per la prima volta. Poiché **Solo per clienti nuovi** è spuntato, i clienti che tornano non possono riscuderlo nemmeno se trovano il codice.

## 2. Soglia per Spedizione Gratuita

Incentiva gli ordini più grandi offrendo una spedizione gratuita una volta che il carrello raggiunge un valore minimo. Imposta il **Valore dello sconto** a un importo che copre il tuo costo di spedizione standard più alto.

| Campo | Valore |
|-------|-------|
| **Codice** | `FREESHIP50` |
| **Nome** | Spedizione gratuita per ordini superiori a $50 |
| **Tipo di sconto** | Sconto Fisso |
| **Valore dello sconto** | 15.00 (o il tuo costo massimo di spedizione) |
| **Ambito di applicazione** | Carrello Intero |
| **Valore minimo dell'ordine** | 50.00 |
| **Utilizzi massimi per cliente** | 0 (illimitato) |
| **Non combinabile con altri buoni** | Spuntato |

Mostra questo codice in modo prominente sulle pagine dei prodotti e nel carrello. I clienti vicini al limite spesso aggiungeranno un altro articolo per qualificarsi, aumentando il valore medio degli ordini.

## 3. Codici per Campagne Stagionali

Esegui promozioni limitate nel tempo legate a festività, stagioni o eventi. Imposta un intervallo di date rigoroso in modo che il buono si attivi e scada automaticamente.

| Campo | Valore |
|-------|-------|
| **Codice** | `SUMMER25` |
| **Nome** | Promozione estiva 2025 |
| **Tipo di sconto** | Sconto Percentuale |
| **Valore dello sconto** | 25 |
| **Importo massimo dello sconto** | 50.00 |
| **Ambito di applicazione** | Carrello Intero |
| **Data di inizio** | 1 giugno |
| **Data di fine** | 31 agosto |
| **Non combinabile con altri buoni** | Spuntato |
| **Non combinabile con articoli in promozione** | Spuntato |

**Codici condivisi vs. codici unici** — Un codice condiviso come `SUMMER25` è semplice da promuovere ma può diffondersi al di là del tuo pubblico inteso. Per un controllo più stretto, utilizza la generazione in blocco per creare codici unici a uso singolo (ciascuno con **Utilizzi massimi totali** impostati a 1). Distribuisci codici unici tramite e-mail mirate in modo che ogni cliente riceva il proprio.

## 4. Buono per Premio di Fidelizzazione

Ricompensa i tuoi clienti migliori con un buono a importo fisso che possono applicare al loro prossimo ordine. Invialo via e-mail dopo che un cliente raggiunge un traguardo di spesa o come ringraziamento per acquisti ripetuti.

| Campo | Valore |
|-------|-------|
| **Codice** | `VIP20` |
| **Nome** | Premio di fidelizzazione VIP |
| **Tipo di sconto** | Sconto Fisso |
| **Valore dello sconto** | 20.00 |
| **Ambito di applicazione** | Carrello Intero |
| **Valore minimo dell'ordine** | 100.00 |
| **Utilizzi massimi per cliente** | 1 |
| **Utilizzi massimi totali** | 0 (illimitato) |
| **Giorni di validità** | 90 |

Impostando **Giorni di validità** a 90, i clienti hanno tre mesi per riscuotere il premio a partire dalla data del primo utilizzo. Il **Valore minimo dell'ordine** di $100 assicura che lo sconto si applichi a ordini significativi.

## 5. Codice Influencer / di Riferimento

Dai agli influencer o ambasciatori del brand un codice personale che condividono con il loro pubblico. Ogni codice è legato a una persona, rendendo facile tracciare quale partner genera le vendite maggiori.


| Campo | Valore |
|-------|-------|
| **Codice** | `SARAH10` |
| **Nome** | Influencer — Sarah |
| **Tipo di sconto** | Sconto percentuale |
| **Valore dello sconto** | 10 |
| **Ambito di applicazione** | Carrello completo |
| **Utilizzo massimo totale** | 0 (illimitato) |
| **Utilizzo massimo per cliente** | 1 |

Crea un buono distinto per ogni influencer in modo da poter confrontare i conteggi di utilizzo nel pannello dei buoni. Controlla regolarmente l'utilizzo di ogni codice per misurare il ROI e decidere quali collaborazioni proseguire.

## 6. Codice per sconto flash

Crea un senso di urgenza con uno sconto profondo disponibile per un breve periodo. Combina il codice con una barra di annuncio o un'email per generare traffico immediato.

| Campo | Valore |
|-------|-------|
| **Codice** | `FLASH40` |
| **Nome** | Vendita flash di 24 ore |
| **Tipo di sconto** | Sconto percentuale |
| **Valore dello sconto** | 40 |
| **Importo massimo dello sconto** | 75,00 |
| **Ambito di applicazione** | Carrello completo |
| **Data di inizio** | Venerdì 9:00 AM |
| **Data di fine** | Sabato 9:00 AM |
| **Utilizzo massimo totale** | 200 |
| **Escludi articoli in promozione** | Spuntato |

Limitare **Utilizzo massimo totale** a 200 protegge i tuoi margini. Una volta raggiunto il limite, il codice smette automaticamente di funzionare. Menziona la quantità limitata nella tua campagna di marketing — "solo per i primi 200 clienti" — per amplificare l'urgenza.

## Riferimento rapido per le campagne

| Campagna | Esempio di codice | Tipo | Valore | Impostazioni principali |
|----------|-------------|------|-------|--------------|
| Sconto di benvenuto | `WELCOME15` | Percentuale | 15% | Solo per la prima volta, utilizzo per cliente = 1 |
| Spedizione gratuita | `FREESHIP50` | Importo fisso | $15 | Ordine minimo $50, non combinabile |
| Vendita stagionale | `SUMMER25` | Percentuale | 25% | Intervallo di date, importo massimo dello sconto $50 |
| Premio per fedeltà | `VIP20` | Importo fisso | $20 | Ordine minimo $100, validità di 90 giorni |
| Codice dell'influencer | `SARAH10` | Percentuale | 10% | Utilizzo per cliente = 1, tracciamento per codice |
| Sconto flash | `FLASH40` | Percentuale | 40% | Finestra di 24 ore, utilizzo massimo 200 |

## Consigli

- **Mantieni i codici brevi e memorabili** — Maiuscole, senza caratteri speciali. I clienti digitano questi codici al checkout, quindi `SUMMER25` è preferibile a `summer-sale-2025-promo`.
- **Monitora i conteggi di utilizzo** — Controlla regolarmente il pannello dei buoni per vedere quali campagne funzionano meglio. Confronta il totale degli utilizzati con il ricavo per calcolare il ROI.
- **Testa prima del lancio** — Effettua un ordine di test con il codice per confermare che venga applicato correttamente, rispetti il valore minimo dell'ordine e si combini (o non si combini) come previsto.
- **Ruota i codici per le campagne ripetute** — Usa un nuovo codice per ogni stagione (es. `SUMMER25`, `SUMMER26`) in modo da poter misurare ogni campagna in modo indipendente e impedire la circolazione di codici vecchi.
- **Imposta un importo massimo per gli sconti percentuali** — Uno sconto del 25% su un ordine di $1.000 equivale a $250. Utilizza il campo **Importo massimo dello sconto** per limitare il valore in dollari e proteggere i tuoi margini.
- **Utilizza la barra di annuncio** — Combina i codici flash e stagionali con la barra di annuncio del tuo negozio online in modo che ogni visitatore veda immediatamente la promozione.
