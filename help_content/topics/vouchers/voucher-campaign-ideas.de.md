title: Gutschein-Kampagnen-Ideen
---

Dieser Leitfaden behandelt bewährte Gutschein-Kampagnen, die Sie in Minuten einrichten können. Jede Kampagne enthält die genauen Einstellungen, um Sie von der Idee bis zur Live-Kampagne schnell zu führen. Erstellen Sie alle Gutscheine unter **Marketing > Gutscheine** und klicken Sie auf **+ Gutschein hinzufügen**.

![Willkommens-Gutschein](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. Willkommensrabatt für neue Kunden

Bieten Sie ersten Käufern einen Anreiz an, ihren Kauf abzuschließen. Ein Willkommensrabatt baut Vertrauen und verringert die Unsicherheit bei Kunden, die noch nie bei Ihrem Geschäft bestellt haben.

| Feld | Wert |
|-------|-------|
| **Code** | `WELCOME15` |
| **Name** | Willkommensrabatt – neue Kunden |
| **Rabatttyp** | Prozentsatz-Rabatt |
| **Rabattwert** | 15 |
| **Anwendungsbereich** | Gesamter Warenkorb |
| **Maximaler Verwendungsbegrenzung pro Kunde** | 1 |
| **Maximaler Verwendungsbegrenzung insgesamt** | 0 (unbegrenzt) |
| **Nur für erste Kunden** | Angekreuzt |
| **Startdatum / Enddatum** | Beide leer lassen – immer aktiv |

Promoten Sie diesen Code in Ihrer Bestätigungs-E-Mail beim Registrieren, auf Ihrem Startseiten-Banner oder in einem Pop-up für erste Besucher. Da **Nur für erste Kunden** angekreuzt ist, können sich zurückkehrende Kunden den Gutschein nicht einlösen, auch wenn sie den Code finden.

## 2. Freier Versandsschwellenwert

Fördern Sie größere Bestellungen, indem Sie freien Versand anbieten, sobald der Warenkorb einen Mindestwert erreicht. Stellen Sie den **Rabattwert** auf einen Betrag ein, der Ihre höchste Standard-Versandkosten abdeckt.

| Feld | Wert |
|-------|-------|
| **Code** | `FREESHIP50` |
| **Name** | Freier Versand für Bestellungen über 50 $ |
| **Rabatttyp** | Fixer Betrag-Rabatt |
| **Rabattwert** | 15,00 (oder Ihre maximale Versandkosten) |
| **Anwendungsbereich** | Gesamter Warenkorb |
| **Mindestbestellwert** | 50,00 |
| **Maximaler Verwendungsbegrenzung pro Kunde** | 0 (unbegrenzt) |
| **Kann nicht mit anderen Gutscheinen kombiniert werden** | Angekreuzt |

Zeigen Sie diesen Code deutlich auf Ihren Produktseiten und im Warenkorb an. Kunden, die nahe am Schwellenwert sind, werden oft ein weiteres Produkt hinzufügen, um den Schwellenwert zu erreichen, was den durchschnittlichen Bestellwert erhöht.

## 3. Saisonale Kampagnen-Codes

Führen Sie zeitlich begrenzte Promotionen durch, die mit Feiertagen, Saisons oder Ereignissen verbunden sind. Legen Sie einen strengen Datumsbereich fest, damit der Gutschein automatisch aktiviert und abläuft.

| Feld | Wert |
|-------|-------|
| **Code** | `SUMMER25` |
| **Name** | Sommerverkauf 2025 |
| **Rabatttyp** | Prozentsatz-Rabatt |
| **Rabattwert** | 25 |
| **Maximaler Rabattbetrag** | 50,00 |
| **Anwendungsbereich** | Gesamter Warenkorb |
| **Startdatum** | 1. Juni |
| **Enddatum** | 31. August |
| **Kann nicht mit anderen Gutscheinen kombiniert werden** | Angekreuzt |
| **Kann nicht mit Verkaufsartikeln kombiniert werden** | Angekreuzt |

**Geteilte vs. eindeutige Codes** – Ein geteilter Code wie `SUMMER25` ist einfach zu bewerben, kann aber außerhalb Ihres Zielpublikums verbreitet werden. Für genauere Kontrolle verwenden Sie die Massengenerierung, um eindeutige Einweg-Codes zu erstellen (jeder mit **Maximaler Verwendungsbegrenzung insgesamt** auf 1 gesetzt). Verteilen Sie eindeutige Codes über gezielte E-Mails, damit jeder Kunde seinen eigenen Code erhält.

## 4. Treueprämien-Gutschein

Belohnen Sie Ihre besten Kunden mit einem festen Betrag-Gutschein, den sie auf ihre nächste Bestellung anwenden können. Senden Sie ihn per E-Mail, nachdem ein Kunde einen Ausgaben-Meilenstein erreicht hat, oder als Dank für wiederkehrende Käufe.

| Feld | Wert |
|-------|-------|
| **Code** | `VIP20` |
| **Name** | VIP-Treueprämie |
| **Rabatttyp** | Fixer Betrag-Rabatt |
| **Rabattwert** | 20,00 |
| **Anwendungsbereich** | Gesamter Warenkorb |
| **Mindestbestellwert** | 100,00 |
| **Maximaler Verwendungsbegrenzung pro Kunde** | 1 |
| **Maximaler Verwendungsbegrenzung insgesamt** | 0 (unbegrenzt) |
| **Gültigkeit in Tagen** | 90 |

Durch die Einstellung von **Gültigkeit in Tagen** auf 90 erhalten die Kunden drei Monate, um die Prämie abzuholen, ab dem Tag des ersten Verwendens. Der **Mindestbestellwert** von 100 $ stellt sicher, dass der Rabatt auf sinnvolle Bestellungen angewendet wird.

## 5. Influencer / Empfehlungscode

Geben Sie Influencern oder Markenbotschaftern einen persönlichen Code, den sie ihrem Publikum teilen. Jeder Code ist mit einer Person verbunden, was es einfach macht, festzustellen, welcher Partner die meisten Verkäufe erzielt.

| Feld | Wert |
|-------|-------|
| **Code** | `SARAH10` |
| **Name** | Influencer — Sarah |
| **Rabatttyp** | Prozentsatz-Rabatt |
| **Rabattwert** | 10 |
| **Anwendungsbereich** | Gesamter Warenkorb |
| **Maximaler Gesamtnutzung** | 0 (unbegrenzt) |
| **Maximaler Nutzungen pro Kunde** | 1 |

Erstelle einen separaten Gutschein für jeden Influencer, damit du die Nutzungszahlen im Gutschein-Dashboard vergleichen kannst. Überprüfe regelmäßig die Nutzung jedes Codes, um den ROI zu messen und zu entscheiden, welche Partnerschaften fortgesetzt werden sollen.

## 6. Flash-Rabattcode

Erstelle Dringlichkeit mit einem tiefen Rabatt, der nur für einen sehr kurzen Zeitraum verfügbar ist. Kombiniere den Code mit einer Werbeleiste oder einer E-Mail-Benachrichtigung, um sofortigen Traffic zu generieren.

| Feld | Wert |
|-------|-------|
| **Code** | `FLASH40` |
| **Name** | 24-Stunden-Flash-Verkauf |
| **Rabatttyp** | Prozentsatz-Rabatt |
| **Rabattwert** | 40 |
| **Maximaler Rabattbetrag** | 75,00 |
| **Anwendungsbereich** | Gesamter Warenkorb |
| **Startdatum** | Freitag 9:00 Uhr |
| **Enddatum** | Samstag 9:00 Uhr |
| **Maximaler Gesamtnutzung** | 200 |
| **Verkaufsartikel ausschließen** | Angekreuzt |

Durch das Begrenzen des **Maximalen Gesamtnutzung** auf 200 wird dein Gewinnschutz gewährleistet. Sobald die Grenze erreicht ist, funktioniert der Code automatisch nicht mehr. Erwähne die begrenzte Menge in deiner Werbung — „nur für die ersten 200 Kunden“ —, um die Dringlichkeit zu verstärken.

## Kampagnen-Quick-Referenz

| Kampagne | Code-Beispiel | Typ | Wert | Schlüssel-Einstellungen |
|----------|-------------|------|-------|--------------|
| Willkommensrabatt | `WELCOME15` | Prozentsatz | 15% | Nur für Erstbesteller, pro-Kunde = 1 |
| Kostenlose Lieferung | `FREESHIP50` | Fixer Betrag | 15 $ | Mindestbestellwert 50 $, keine Kombination |
| Saisonale Verkauf | `SUMMER25` | Prozentsatz | 25% | Datumsbereich, maximaler Rabatt 50 $ |
| Treueprämie | `VIP20` | Fixer Betrag | 20 $ | Mindestbestellwert 100 $, Gültigkeit 90 Tage |
| Influencer-Code | `SARAH10` | Prozentsatz | 10% | Pro-Kunde = 1, nach Code verfolgen |
| Flash-Rabatt | `FLASH40` | Prozentsatz | 40% | 24-Stunden-Fenster, maximal 200 Nutzungen |

## Tipps

- **Behalte die Codes kurz und merkbar** — Nur Großbuchstaben, keine Sonderzeichen. Kunden tippen diese beim Checkout ein, also ist `SUMMER25` besser als `summer-sale-2025-promo`.
- **Verfolge die Nutzungszahlen** — Überprüfe regelmäßig das Gutschein-Dashboard, um zu sehen, welche Kampagnen am besten funktionieren. Vergleiche die Gesamtnutzungen mit dem Umsatz, um den ROI zu berechnen.
- **Teste vor der Veröffentlichung** — Platziere einen Testauftrag mit dem Code, um sicherzustellen, dass er korrekt angewendet wird, den Mindestbestellwert respektiert und wie erwartet kombiniert (oder nicht kombiniert) wird.
- **Wechsle die Codes für wiederkehrende Kampagnen** — Verwende einen neuen Code pro Saison (z. B. `SUMMER25`, `SUMMER26`), damit du jede Kampagne unabhängig messen kannst und verhindert, dass alte Codes weiterhin im Umlauf sind.
- **Setze maximale Rabattbeträge bei Prozentsatz-Gutscheinen** — Ein 25% Rabatt auf einen 1000 $-Bestellwert beträgt 250 $. Verwende das Feld **Maximaler Rabattbetrag**, um den Dollarwert zu begrenzen und deine Gewinne zu schützen.
- **Verwende die Werbeleiste** — Kombiniere Flash- und Saison-Codes mit der Werbeleiste deines Onlineshops, damit jeder Besucher die Promotion sofort sieht.
