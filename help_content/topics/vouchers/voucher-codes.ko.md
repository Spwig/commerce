---
title: 할인 코드
---

할인 코드는 고객이 결제 시 할인을 받기 위해 입력할 수 있는 할인 코드, 쿠폰, 선물 카드를 생성할 수 있습니다. 관리자 사이드바에서 **마케팅 > 할인 코드**로 이동하세요.

![할인 코드 목록](/static/core/admin/img/help/voucher-codes/voucher-list.webp)

## 할인 코드 대시보드

할인 코드 페이지는 다음 개요를 보여줍니다:

- **통계 카드** — 활성, 비활성, 사용 횟수, 총 할인 코드 수
- **필터** — 코드 또는 이름으로 검색, 타입, 상태, 범위로 필터
- **할인 코드 카드** — 각 할인 코드는 사용 및 상태 세부 정보와 함께 표시됩니다.

## 할인 코드 생성

1. 상단 오른쪽에 있는 **+ 할인 코드 추가**를 클릭합니다.
2. 할인 코드 세부 정보를 입력합니다:
   - **코드** — 고객이 결제 시 입력하는 코드 (예: "SAVE20", "FREESHIP")
   - **이름/설명** — 내부 참고용 설명
   - **할인 유형** — 할인 적용 방식을 선택합니다.
   - **할인 금액** — 할인 금액 또는 비율
3. 사용 규칙을 설정합니다:
   - **사용 제한** — 최대 총 사용 횟수 (0 = 무제한)
   - **고객별 제한** — 고객당 최대 사용 횟수
   - **최소 주문 금액** — 필요한 최소 장바구니 총액
4. **범위**를 설정합니다:
   - **전체 주문** — 전체 주문에 할인이 적용됩니다.
   - **특정 상품** — 선택한 항목에만 적용됩니다.
   - **특정 카테고리** — 선택한 카테고리의 항목에만 적용됩니다.
5. 선택적으로 만료일을 설정합니다:
   - **만료일** — 할인 코드가 작동을 중단하는 날짜
6. **저장**을 클릭합니다.

## 할인 코드 유형

| 유형 | 설명 | 예시 |
|------|-------------|---------|
| **고정 금액** | 고정된 달러 금액을 차감합니다 | 주문 금액에서 $20 할인 |
| **비율** | 총 금액의 비율을 차감합니다 | 주문 금액의 15% 할인 |
| **무료 배송** | 배송비를 제거합니다 | 모든 주문에 무료 배송 |

## 할인 코드 관리

### 할인 코드 카드

각 할인 코드 카드는 다음과 같은 정보를 보여줍니다:
- **코드** — 굵은 글씨로 표시된 할인 코드
- **설명** — 할인 코드의 기능
- **상태 배지** — 활성 또는 비활성
- **할인 세부 정보** — 유형 및 금액 (예: "$ 20.00" 또는 "15.00%")
- **범위** — 전체 주문에 적용되거나 특정 항목에만 적용되는지
- **사용 횟수** — 할인 코드가 사용된 횟수
- **생성 날짜** — 할인 코드가 생성된 날짜
- **만료** — 만료 날짜 또는 "만료 없음"

### 할인 코드 액션

각 카드에는 액션 버튼이 있습니다:
- **편집** — 할인 코드 설정을 수정합니다.
- **이력 보기** — 사용 이력을 확인합니다.
- **삭제** — 할인 코드를 제거합니다.

### 할인 코드 필터링

필터 바를 사용하여 특정 할인 코드를 찾을 수 있습니다:
- **검색** — 코드, 이름 또는 설명으로 검색
- **유형** — 고정 금액, 비율 또는 무료 배송
- **상태** — 활성 또는 비활성
- **범위** — 전체 주문 또는 특정 상품

## 대량 할인 코드 생성

대규모 캠페인을 위해 대량으로 할인 코드를 생성할 수 있습니다:
1. 시스템이 고유한 코드를 자동으로 생성합니다 (예: "COUPONX1600406498")
2. 생성된 모든 할인 코드에 공통 파라미터를 설정합니다.
3. 이메일, 소셜 미디어 또는 인쇄를 통해 코드를 배포합니다.

## 고객 경험

고객이 할인 코드를 가지고 있을 때:
1. 고객은 **결제**로 이동합니다.
2. **할인 코드** 필드에 코드를 입력합니다.
3. 할인 코드가 유효한 경우 즉시 할인이 적용됩니다.
4. 주문 요약이 업데이트되어 할인을 표시합니다.

할인 코드가 유효하지 않은 경우 (만료됨, 사용 제한 도달, 최소 금액 미달), 고객은 명확한 오류 메시지를 보게 됩니다.

## 팁

- 마케팅 캠페인에 기억하기 쉬운 코드를 사용하세요 (예: "SUMMER20" 대신 무작위 문자열).
- 고가의 할인을 방지하기 위해 고객별 제한을 설정하세요.
- 이익성 유지를 위해 최소 주문 금액을 설정하세요 (예: "$50 이상 주문 시 $10 할인").
- 대시보드의 사용 횟수를 모니터링하여 캠페인 효과를 추적하세요.
- 긴급감을 위해 시간 제한된 할인 코드를 생성하세요 (예: "이번 주말만 유효").
- 삭제하지 않고 할인 코드를 일시 중지하려면 Active/Inactive 상태를 사용하세요.
