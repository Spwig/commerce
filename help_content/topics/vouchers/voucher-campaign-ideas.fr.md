---
title: Idées de campagnes de bons de réduction
---

Ce guide couvre des campagnes de bons de réduction éprouvées que vous pouvez configurer en quelques minutes. Chaque campagne inclut les paramètres exacts à configurer afin que vous puissiez passer rapidement de l'idée à une promotion active. Créez tous les bons de réduction à **Marketing > Bons de réduction** et cliquez sur **+ Ajouter un bon de réduction**.

![Bienvenue avec un bon de réduction](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. Réduction de bienvenue pour les nouveaux clients

Offrez un incitant aux premiers acheteurs pour terminer leur achat. Une réduction de bienvenue construit la confiance et réduit l'hésitation des clients qui n'ont jamais commandé dans votre magasin.

| Champ | Valeur |
|-------|-------|
| **Code** | `WELCOME15` |
| **Nom** | Réduction de bienvenue — nouveaux clients |
| **Type de réduction** | Réduction en pourcentage |
| **Valeur de réduction** | 15 |
| **Portée d'application** | Panier entier |
| **Nombre maximal d'utilisations par client** | 1 |
| **Nombre maximal d'utilisations total** | 0 (illimité) |
| **Uniquement pour les nouveaux clients** | Coché |
| **Date de début / Date de fin** | Laissez les deux vides — toujours actif |

Promouvoir ce code dans votre e-mail de confirmation d'inscription, sur votre bannière d'accueil ou dans une fenêtre modale pour les premiers visiteurs. Puisque **Uniquement pour les nouveaux clients** est coché, les clients existants ne peuvent pas le réclamer même s'ils trouvent le code.

## 2. Seuil de livraison gratuite

Encouragez des commandes plus importantes en offrant une livraison gratuite une fois que le panier atteint une valeur minimale. Fixez la **Valeur de réduction** à un montant qui couvre votre coût de livraison standard le plus élevé.

| Champ | Valeur |
|-------|-------|
| **Code** | `FREESHIP50` |
| **Nom** | Livraison gratuite pour les commandes de plus de 50 $ |
| **Type de réduction** | Réduction d'un montant fixe |
| **Valeur de réduction** | 15,00 (ou votre coût de livraison maximum) |
| **Portée d'application** | Panier entier |
| **Valeur minimale de commande** | 50,00 |
| **Nombre maximal d'utilisations par client** | 0 (illimité) |
| **Ne peut pas être combiné avec d'autres bons de réduction** | Coché |

Affichez ce code en évidence sur vos pages de produits et dans le panier. Les clients proches du seuil ajouteront souvent un autre article pour y éligible, augmentant ainsi votre valeur moyenne de commande.

## 3. Codes de campagne saisonnière

Organisez des promotions limitées dans le temps liées aux fêtes, aux saisons ou aux événements. Fixez une plage de dates stricte afin que le bon s'active et expire automatiquement.

| Champ | Valeur |
|-------|-------|
| **Code** | `SUMMER25` |
| **Nom** | Vente d'été 2025 |
| **Type de réduction** | Réduction en pourcentage |
| **Valeur de réduction** | 25 |
| **Montant maximal de réduction** | 50,00 |
| **Portée d'application** | Panier entier |
| **Date de début** | 1er juin |
| **Date de fin** | 31 août |
| **Ne peut pas être combiné avec d'autres bons de réduction** | Coché |
| **Ne peut pas être combiné avec des articles en promotion** | Coché |

**Codes partagés vs. codes uniques** — Un code partagé comme `SUMMER25` est facile à promouvoir, mais peut se propager au-delà de votre public cible prévu. Pour un contrôle plus précis, utilisez la génération en lot pour créer des codes uniques à usage unique (chaque code avec **Nombre maximal d'utilisations total** fixé à 1). Distribuez des codes uniques via des e-mails ciblés afin que chaque client ait son propre code.

## 4. Bon de réduction de fidélité

Récompensez vos meilleurs clients avec un bon de réduction d'un montant fixe qu'ils peuvent appliquer à leur prochaine commande. Envoyez-le par e-mail après qu'un client atteigne un seuil de dépense ou comme un remerciement pour des achats répétés.

| Champ | Valeur |
|-------|-------|
| **Code** | `VIP20` |
| **Nom** | Récompense de fidélité VIP |
| **Type de réduction** | Réduction d'un montant fixe |
| **Valeur de réduction** | 20,00 |
| **Portée d'application** | Panier entier |
| **Valeur minimale de commande** | 100,00 |
| **Nombre maximal d'utilisations par client** | 1 |
| **Nombre maximal d'utilisations total** | 0 (illimité) |
| **Durée de validité** | 90 |

En fixant **Durée de validité** à 90, les clients ont trois mois pour réclamer la récompense à partir de la date d'utilisation initiale. La **Valeur minimale de commande** de 100 $ garantit que la réduction s'applique aux commandes significatives.

## 5. Code d'influenceur / de parrainage

Donnez aux influenceurs ou ambassadeurs de marque un code personnel qu'ils partagent avec leur audience. Chaque code est lié à une seule personne, ce qui facilite le suivi des partenaires qui génèrent le plus de ventes.


| Champ | Valeur |
|-------|-------|
| **Code** | `SARAH10` |
| **Nom** | Influencer — Sarah |
| **Type de remise** | Pourcentage |
| **Valeur de la remise** | 10 |
| **Portée de l'application** | Panier entier |
| **Nombre maximal d'utilisations total** | 0 (illimité) |
| **Nombre maximal d'utilisations par client** | 1 |

Créez un bon d'achat distinct pour chaque influenceur afin de comparer les taux de rédemption sur le tableau de bord des bons d'achat. Vérifiez régulièrement l'utilisation de chaque code pour mesurer le ROI et décider des partenariats à poursuivre.

## 6. Code de remise flash

Créez de l'urgence avec une remise importante disponible pendant une période très courte. Associez ce code à une bande d'annonce ou à un envoi par courriel pour générer un trafic immédiat.

| Champ | Valeur |
|-------|-------|
| **Code** | `FLASH40` |
| **Nom** | Vente flash de 24 heures |
| **Type de remise** | Pourcentage |
| **Valeur de la remise** | 40 |
| **Montant maximal de la remise** | 75,00 |
| **Portée de l'application** | Panier entier |
| **Date de début** | Vendredi 9h00 |
| **Date de fin** | Samedi 9h00 |
| **Nombre maximal d'utilisations total** | 200 |
| **Exclure les articles en promotion** | Coché |

Fixer le **nombre maximal d'utilisations total** à 200 protège vos marges. Une fois ce plafond atteint, le code cesse automatiquement de fonctionner. Mentionnez la quantité limitée dans votre marketing — « uniquement pour les 200 premiers clients » — pour amplifier l'urgence.

## Référence rapide des campagnes

| Campagne | Exemple de code | Type | Valeur | Paramètres clés |
|----------|-------------|------|-------|--------------|
| Remise de bienvenue | `WELCOME15` | Pourcentage | 15 % | Uniquement pour les premiers clients, nombre maximal par client = 1 |
| Livraison gratuite | `FREESHIP50` | Montant fixe | 15 $ | Commande minimale de 50 $, non cumulable |
| Vente saisonnière | `SUMMER25` | Pourcentage | 25 % | Plage de dates, montant maximal de la remise 50 $ |
| Récompense de fidélité | `VIP20` | Montant fixe | 20 $ | Commande minimale de 100 $, validité de 90 jours |
| Code d'influenceur | `SARAH10` | Pourcentage | 10 % | Nombre maximal par client = 1, suivi par code |
| Remise flash | `FLASH40` | Pourcentage | 40 % | Fenêtre de 24 heures, 200 utilisations maximales |

## Conseils

- **Gardez les codes courts et mémorables** — En majuscules, sans caractères spéciaux. Les clients tapent ces codes à la caisse, donc `SUMMER25` est préférable à `summer-sale-2025-promo`.
- **Suivez les taux de rédemption** — Vérifiez régulièrement le tableau de bord des bons d'achat pour voir quelles campagnes se déroulent le mieux. Comparez le nombre total de rédemptions avec le chiffre d'affaires pour calculer le ROI.
- **Testez avant le lancement** — Placez un ordre de test avec le code pour confirmer qu'il s'applique correctement, respecte la valeur minimale de la commande et combine (ou ne combine pas) comme prévu.
- **Utilisez de nouveaux codes pour les campagnes répétitives** — Utilisez un nouveau code chaque saison (par exemple, `SUMMER25`, `SUMMER26`) afin de mesurer indépendamment chaque campagne et d'éviter la circulation des anciens codes.
- **Fixez un montant maximal de remise sur les bons de pourcentage** — Une remise de 25 % sur une commande de 1 000 $ représente 250 $. Utilisez le champ **Montant maximal de la remise** pour fixer la valeur en dollars et protéger vos marges.
- **Utilisez la bande d'annonce** — Associez les codes flash et saisonniers à la bande d'annonce de votre boutique en ligne afin que chaque visiteur voie immédiatement la promotion.
