本指南提供最常見優惠券類型的具體欄位範例。每個範例顯示在 **行銷 > 優惠券** → **+ 新增優惠券** 建立優惠券時應輸入的內容。

![優惠券卡片](/static/core/admin/img/help/voucher-examples/voucher-card.webp)

## 範例 1：折扣上限的百分比折扣

**情境：** 提供整筆訂單 20% 的折扣，但將折扣上限設定為 $50，以確保高價值訂單仍具利潤。無過期日期。

| 欄位 | 值 |
|-------|-------|
| 編碼 | `SAVE20` |
| 名稱 | 20% 折扣 — 最高 $50 |
| 折扣類型 | 百分比 |
| 折扣值 | 20 |
| 最高折扣金額 | 50 |
| 應用範圍 | 整筆訂單 |
| 總使用次數上限 | *(空白 — 無限制)* |
| 每位客戶使用次數上限 | 1 |
| 最低訂單金額 | *(空白 — 無最低限制)* |

**折扣上限的運作方式：** 在 $200 的訂單中，折扣為 $40。在 $300 的訂單中，折扣為 $60，但上限限制為 $50。在 $500 的訂單中，折扣仍為 $50。這讓您可以推出聽起來慷慨的促銷活動，同時保持實際折扣的可預測性。

## 範例 2：最低金額的固定金額折扣

**情境：** 為鼓勵客戶下較大訂單，提供任何超過 $75 的訂單 $10 折扣。

| 欄位 | 值 |
|-------|-------|
| 編碼 | `TAKE10` |
| 名稱 | 超過 $75 訂單減 $10 |
| 折扣類型 | 固定金額 |
| 折扣值 | 10 |
| 應用範圍 | 整筆訂單 |
| 最低訂單金額 | 75 |
| 每位客戶使用次數上限 | 0 *(無限制)* |
| 結束日期 | *(空白 — 無過期)* |

> **注意：** 設定最低訂單金額可保護您的利潤空間。若沒有設定，客戶可以將此優惠碼應用於 $12 的訂單，從而完全抵消您的利潤。始終將固定金額優惠券與合理的最低金額搭配使用。

## 範例 3：免運費

**情境：** 為任何訂單提供免運費，無最低消費限制。

| 欄位 | 值 |
|-------|-------|
| 編碼 | `FREESHIP` |
| 名稱 | 免運費 |
| 折扣類型 | 免運費 |
| 應用範圍 | 整筆訂單 |
| 總使用次數上限 | *(空白 — 無限制)* |
| 每位客戶使用次數上限 | 1 |
| 最低訂單金額 | *(空白 — 無最低限制)* |

> **注意：** 選擇 **免運費** 折扣類型，會自動從訂單中移除運費。這是最乾淨的解決方案，無論客戶選擇哪種運送方式都適用。

## 範例 4：新客戶歡迎碼

**情境：** 為新客戶提供其首筆訂單的 15% 折扣，以促進轉化。

| 欄位 | 值 |
|-------|-------|
| 編碼 | `WELCOME15` |
| 名稱 | 歡迎 — 首筆訂單 15% 折扣 |
| 折扣類型 | 百分比 |
| 折扣值 | 15 |
| 應用範圍 | 整筆訂單 |
| 每位客戶使用次數上限 | 1 |
| 僅限新客戶 | 勾選 |

系統透過檢查客戶是否有任何已完成的訂單來驗證新客戶狀態。如果試圖套用此優惠碼的客戶有訂單記錄，他們在結帳時會看到明確的錯誤訊息。

## 範例 5：特定產品優惠券

**情境：** 為選定的產品提供 $5 折扣 — 例如，為促進滞銷庫存。

| 欄位 | 值 |
|-------|-------|
| 編碼 | `PICK5` |
| 名稱 | 選定商品減 $5 |
| 折扣類型 | 固定金額 |
| 折扣值 | 5 |
| 應用範圍 | 特定產品 |
| 合資格產品 | *(選擇目標產品)* |
| 每位客戶使用次數上限 | 1 |

> **注意：** 當您想要折扣個別 SKU 時，使用產品範圍。當您想要折扣部門內的所有商品時，使用類別範圍（下一個範例）。產品範圍讓您可以精確控制；當您的目錄經常變更時，類別範圍更易於維護。

## 範例 6：類別優惠券

**情境：** 在電子產品類別的所有商品上運行 25% 的折扣促銷。

| 欄位 | 值 |
|-------|-------|
| 編碼 | `ELEC25` |
| 名稱 | 電子產品 25% 折扣 |
| 折扣類型 | 百分比 |
| 折扣值 | 25 |
| 應用範圍 | 特定類別 |
| 合資格類別 | 電子產品 |
| 總使用次數上限 | *(空白 — 無限制)* |
| 每位客戶使用次數上限 | 1 |


當套用至某一類別時，折扣僅適用於購物車中符合資格的項目。

非電子產品將按原價收費。

## 折扣類型比較

| 類型 | 如何運作 | 最佳用途 | 範例 |
|------|-------------|----------|---------|
| **百分比** | 扣除符合資格總額的百分比 | 隨訂單規模增長的折扣 | 全部購物車減20% |
| **固定金額** | 扣除固定金額 | 簡單且可預測的促銷 | 訂單滿$75減$10 |
| **免運費** | 移除訂單的運費 | 減少結帳時的購物車放棄率 | 免運費，無最低金額限制 |

## 套用範圍比較

| 範圍 | 如何運作 | 最佳用途 |
|-------|-------------|----------|
| **整個購物車** | 折扣適用於整筆訂單總額 | 全店促銷和歡迎碼 |
| **特定產品** | 折扣僅適用於購物車中選定的產品 | 清倉特定庫存或重點推廣 |
| **特定類別** | 折扣僅適用於選定類別中的商品 | 部門級銷售和季節性促銷 |

## 小提示

- **使用易記的代碼** — `SUMMER20` 比 `COUPONX1600406498` 更易記。將自動生成的代碼保留用於批量活動。
- **發放前進行測試** — 使用優惠碼下一個測試訂單，以確認其正確套用並遵守所有限制。
- **監控使用情況** — 查看每個優惠碼卡片上的兌換次數，即時追蹤活動表現。
- **與公告欄結合使用** — 在網站範圍內的公告中推廣您的優惠碼，讓顧客在開始購物前就能看到。
