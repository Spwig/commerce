---
title: Идеи кампаний с купонами
---

Этот гайд охватывает проверенные кампании с купонами, которые вы можете настроить за минуты. Каждая кампания включает точные настройки для конфигурации, чтобы вы могли быстро перейти от идеи к активной акции. Создайте все купоны в **Маркетинг > Купоны** и нажмите **+ Добавить купон**.

![Добро пожаловать Купон](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. Скидка для новых клиентов

Предложите новым покупателям стимул для завершения покупки. Скидка "Добро пожаловать" помогает построить доверие и снизить неуверенность у клиентов, которые никогда не заказывали в вашем магазине.

| Поле | Значение |
|-------|-------|
| **Код** | `WELCOME15` |
| **Название** | Скидка для новых клиентов |
| **Тип скидки** | Процентная скидка |
| **Значение скидки** | 15 |
| **Область применения** | Весь заказ |
| **Максимальное количество использований на клиента** | 1 |
| **Максимальное количество использований в целом** | 0 (без ограничения) |
| **Только для новых клиентов** | Отмечено |
| **Дата начала / Дата окончания** | Оставьте оба пустыми — всегда активно |

Продвигайте этот код в письме подтверждения регистрации, на баннере главной страницы или в всплывающем окне для новых посетителей. Поскольку **Только для новых клиентов** отмечено, возвращающиеся клиенты не смогут его использовать, даже если найдут код.

## 2. Порог бесплатной доставки

Поощряйте большие заказы, предлагая бесплатную доставку, когда сумма корзины достигает минимального значения. Установите **Значение скидки** на сумму, которая покрывает вашу максимальную стандартную стоимость доставки.

| Поле | Значение |
|-------|-------|
| **Код** | `FREESHIP50` |
| **Название** | Бесплатная доставка для заказов свыше 50 $ |
| **Тип скидки** | Фиксированная сумма скидки |
| **Значение скидки** | 15.00 (или ваша максимальная стоимость доставки) |
| **Область применения** | Весь заказ |
| **Минимальная сумма заказа** | 50.00 |
| **Максимальное количество использований на клиента** | 0 (без ограничения) |
| **Нельзя комбинировать с другими купонами** | Отмечено |

Отобразите этот код визуально на страницах товаров и в корзине. Клиенты, которые близки к порогу, часто добавляют еще один товар, чтобы соответствовать требованиям, что увеличивает среднюю сумму заказа.

## 3. Коды сезонных акций

Проводите ограниченные по времени акции, связанные с праздниками, сезонами или событиями. Установите строгий временной диапазон, чтобы купон активировался и истекал автоматически.

| Поле | Значение |
|-------|-------|
| **Код** | `SUMMER25` |
| **Название** | Летняя распродажа 2025 |
| **Тип скидки** | Процентная скидка |
| **Значение скидки** | 25 |
| **Максимальная сумма скидки** | 50.00 |
| **Область применения** | Весь заказ |
| **Дата начала** | 1 июня |
| **Дата окончания** | 31 августа |
| **Нельзя комбинировать с другими купонами** | Отмечено |
| **Нельзя комбинировать со скидочными товарами** | Отмечено |

**Общий код vs. уникальные коды** — общий код, например, `SUMMER25`, легко продвигать, но он может распространиться за пределы вашей целевой аудитории. Для более точного контроля используйте массовое создание, чтобы сгенерировать уникальные одноразовые коды (каждый с **Максимальное количество использований в целом** установленным на 1). Распределяйте уникальные коды через целевые письма, чтобы каждый клиент получал свой собственный.

## 4. Купон поощрения лояльности

Награждайте ваших лучших клиентов фиксированной суммой купона, которую они могут применить к следующему заказу. Отправьте его по электронной почте, когда клиент достигает определенного уровня расходов или как благодарность за повторные покупки.

| Поле | Значение |
|-------|-------|
| **Код** | `VIP20` |
| **Название** | Купон лояльности VIP |
| **Тип скидки** | Фиксированная сумма скидки |
| **Значение скидки** | 20.00 |
| **Область применения** | Весь заказ |
| **Минимальная сумма заказа** | 100.00 |
| **Максимальное количество использований на клиента** | 1 |
| **Максимальное количество использований в целом** | 0 (без ограничения) |
| **Срок действия в днях** | 90 |

Установка **Срок действия в днях** на 90 дает клиентам три месяца, чтобы использовать награду с момента первого использования. **Минимальная сумма заказа** в 100 $ гарантирует, что скидка применяется к значимым заказам.

## 5. Коды для инфлюенсеров / рефералов

Предоставьте инфлюенсерам или посредникам бренда личный код, который они делятся со своей аудиторией. Каждый код привязан к одному человеку, что делает отслеживание того, какой партнер приносит больше всего продаж, простым.

| Параметр | Значение |
|-------|-------|
| **Код** | `SARAH10` |
| **Название** | Инфлюенсер — Sarah |
| **Тип скидки** | Процентная скидка |
| **Значение скидки** | 10 |
| **Область применения** | Весь заказ |
| **Максимальное количество использований в целом** | 0 (без ограничения) |
| **Максимальное количество использований на клиента** | 1 |

Создайте отдельный купон для каждого инфлюенсера, чтобы можно было сравнивать количество использований на панели мониторинга купонов. Регулярно проверяйте использование каждого кода, чтобы оценить ROI и принять решение о том, какие партнерства стоит продолжать.

## 6. Код скидки вспышки

Создайте срочность с глубокой скидкой, доступной на очень короткий период. Сочетайте код с баннером объявлений или рассылкой электронной почты, чтобы привлечь немедленный трафик.

| Параметр | Значение |
|-------|-------|
| **Код** | `FLASH40` |
| **Название** | 24-часовая распродажа |
| **Тип скидки** | Процентная скидка |
| **Значение скидки** | 40 |
| **Максимальная сумма скидки** | 75.00 |
| **Область применения** | Весь заказ |
| **Дата начала** | Пятница, 9:00 AM |
| **Дата окончания** | Суббота, 9:00 AM |
| **Максимальное количество использований в целом** | 200 |
| **Исключить товары распродажи** | Отмечено |

Ограничение **Максимальное количество использований в целом** до 200 защищает вашу маржу. Как только будет достигнут лимит, код автоматически перестанет работать. Упоминайте ограниченное количество в вашей маркетинговой стратегии — «только для первых 200 клиентов» — чтобы усилить срочность.

## Быстрый справочник кампаний

| Кампания | Пример кода | Тип | Значение | Ключевые настройки |
|----------|-------------|------|-------|--------------|
| Приветственная скидка | `WELCOME15` | Процент | 15% | Только для новых клиентов, максимальное количество использований на клиента = 1 |
| Бесплатная доставка | `FREESHIP50` | Фиксированная сумма | $15 | Минимальная сумма заказа $50, нельзя комбинировать |
| Сезонная распродажа | `SUMMER25` | Процент | 25% | Диапазон дат, максимальная сумма скидки $50 |
| Награда для лояльных клиентов | `VIP20` | Фиксированная сумма | $20 | Минимальная сумма заказа $100, срок действия 90 дней |
| Код инфлюенсера | `SARAH10` | Процент | 10% | Максимальное количество использований на клиента = 1, отслеживать по коду |
| Скидка вспышки | `FLASH40` | Процент | 40% | Окно в 24 часа, максимальное количество использований 200 |

## Советы

- **Держите коды короткими и запоминающимися** — Все заглавные буквы, без специальных символов. Покупатели вводят эти коды на этапе оформления заказа, поэтому `SUMMER25` лучше, чем `summer-sale-2025-promo`.
- **Отслеживайте количество использований** — Регулярно проверяйте панель мониторинга купонов, чтобы увидеть, какие кампании работают лучше всего. Сравнивайте общее количество использований с выручкой, чтобы рассчитать ROI.
- **Проведите тестирование перед запуском** — Разместите тестовый заказ с кодом, чтобы убедиться, что скидка применяется правильно, учитывает минимальную сумму заказа и сочетается (или не сочетается) как ожидалось.
- **Используйте новые коды для повторных кампаний** — Используйте новый код каждый сезон (например, `SUMMER25`, `SUMMER26`), чтобы можно было измерять эффективность каждой кампании отдельно и предотвратить распространение старых кодов.
- **Установите максимальные суммы скидок для процентных купонов** — Скидка в 25% на заказ в $1000 составляет $250. Используйте поле **Максимальная сумма скидки**, чтобы ограничить денежное значение и защитить вашу маржу.
- **Используйте баннер объявления** — Сочетайте коды вспышки и сезонные с баннером объявления на вашем магазине, чтобы каждый посетитель сразу видел акцию.
