इस प्रारूप से वौचर अभियानों के साबित तरीके शामिल हैं जो आप कुछ मिनटों में सेट कर सकते हैं। प्रत्येक अभियान में विशिष्ट सेटिंग्स शामिल हैं जिन्हें आप एक विचार से लाइव प्रोमोशन तक जल्दी जा सकते हैं। सभी वौचर **मार्केटिंग > वौचर** पर बनाएं और **+ वौचर जोड़ें** पर क्लिक करें।

![Welcome Voucher](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. नए ग्राहकों के लिए स्वागत छूट

पहली बार खरीदार ग्राहकों के लिए एक प्रेरणा प्रदान करें ताकि वे अपनी खरीद को पूरा कर सकें। एक स्वागत छूट विश्वास बनाता है और उन ग्राहकों के लिए कम जिम्मेदारी बनाता है जो आपके स्टोर से कभी ऑर्डर नहीं करते हैं।

| क्षेत्र | मान |
|-------|-------|
| **कोड** | `WELCOME15` |
| **नाम** | स्वागत छूट — नए ग्राहक |
| **छूट प्रकार** | प्रतिशत छूट |
| **छूट मूल्य** | 15 |
| **लागू की रेंज** | पूरा कार्ट |
| **प्रति ग्राहक अधिकतम उपयोग** | 1 |
| **कुल अधिकतम उपयोग** | 0 (असीमित) |
| **केवल पहली बार के ग्राहक** | चेक किया गया |
| **शुरू करें तिथि / समाप्ति तिथि** | दोनों खाली छोड़ें — हमेशा सक्रिय |


अपने साइनअप पुष्टि ईमेल में, अपने होमपेज के बैनर पर या पहली बार आगंतुकों के लिए एक पॉप-अप में इस कोड को प्रोत्साहित करें।

क्योंकि **केवल पहली बार के ग्राहक** चिह्नित है, वापसी ग्राहक इसे तब भी नहीं लागू कर सकते जब वे कोड को ढूंढते हैं।

## 2. मुफ्त शिपिंग थ्रेशोल्ड

अधिकतम ऑर्डर के लिए प्रोत्साहित करें जब कैर्ट में न्यूनतम मूल्य तक पहुंच जाता है। **छूट की राशि** को एक राशि तक निर्धारित करें जो आपके सबसे उच्च मानक शिपिंग लागत को कवर करता है।

| क्षेत्र | मूल्य |
|-------|-------|
| **कोड** | `FREESHIP50` |
| **नाम** | $50 से अधिक ऑर्डर पर मुफ्त शिपिंग |
| **छूट के प्रकार** | फिक्स्ड एमाउंट छूट |
| **छूट की राशि** | 15.00 (या आपकी अधिकतम शिपिंग लागत) |
| **लागू करने के क्षेत्र** | पूरा कैर्ट |
| **न्यूनतम ऑर्डर मूल्य** | 50.00 |
| **प्रति ग्राहक अधिकतम उपयोग** | 0 (असीमित) |
| **अन्य वॉउचर के साथ संयोजित नहीं कर सकते** | चिह्नित |

इस कोड को अपने उत्पाद पृष्ठों और कैर्ट में प्रमुख रूप से प्रदर्शित करें।

अंतिम सीमा के पास ग्राहक अक्सर एक अतिरिक्त आइटम जोड़ते हैं ताकि वे योग्य हो जाएं, जिससे आपका औसत ऑर्डर मान बढ़ जाता है।

## 3. मौसमी अभियान कोड

होलीडे, मौसम या घटनाओं से जुड़े समय-सीमित प्रोमोशन चलाएं। एक सख्त तिथि अंतराल निर्धारित करें ताकि वॉउचर स्वचालित रूप से सक्रिय और समाप्त हो जाए।

| क्षेत्र | मान |
|-------|-------|
| **कोड** | `SUMMER25` |
| **नाम** | 2025 के ग्रीष्म बिक्री |
| **छूट के प्रकार** | प्रतिशत छूट |
| **छूट के मान** | 25 |
| **अधिकतम छूट की राशि** | 50.00 |
| **लागू करने के क्षेत्र** | पूरा कार्ट |
| **प्रारंभ तिथि** | 1 जून |
| **अंत तिथि** | 31 अगस्त |
| **अन्य वॉउचर के साथ जोड़ा नहीं जा सकता** | चेक किया |
| **बिक्री आइटम के साथ जोड़ा नहीं जा सकता** | चेक किया |

**साझा vs. अद्वितीय कोड** — एक साझा कोड जैसे `SUMMER25` प्रचार करना आसान है लेकिन इसका उपयोग आपके निर्धारित लक्ष्य जनसंख्या से बाहर भी हो सकता है।

अधिक नियंत्रण के लिए, बैच उत्पादन का उपयोग करें ताकि एकल-उपयोग अद्वितीय कोड बनाए जा सकें (प्रत्येक के लिए **कुल अधिकतम उपयोग** 1 रखें।)

अपने ग्राहकों के लिए अपना कोड लक्षित ईमेल के माध्यम से वितरित करें ताकि प्रत्येक ग्राहक अपना अपना कोड प्राप्त कर सके।

## 4. लॉयल्टी रिवॉर्ड वौचर

अपने सर्वोत्तम ग्राहकों को एक निश्चित राशि के वौचर के साथ रिवॉर्ड करें जिसे उनके अगले ऑर्डर में लागू किया जा सकता है। एक ग्राहक खर्च के एक मीलस्टोन तक पहुंच जाता है या दोहरे खरीदारी के लिए धन्यवाद के रूप में ईमेल के माध्यम से भेजें।

| क्षेत्र | मान |
|-------|-------|
| **कोड** | `VIP20` |
| **नाम** | VIP लॉयल्टी रिवॉर्ड |
| **छूट के प्रकार** | निश्चित राशि छूट |
| **छूट के मूल्य** | 20.00 |
| **लागू करने के क्षेत्र** | पूरा कार्ट |
| **न्यूनतम ऑर्डर मूल्य** | 100.00 |
| **प्रति ग्राहक अधिकतम उपयोग** | 1 |
| **कुल अधिकतम उपयोग** | 0 (असीमित) |
| **अमूल्य दिन** | 90 |

**अमूल्य दिन** को 90 रखने से ग्राहकों को पहली बार इसका उपयोग करने की तारीख से तीन महीने तक रिवॉर्ड के लिए अवधि मिलती है। $100 के **न्यूनतम ऑर्डर मूल्य** छूट के लायक ऑर्डर्स की गारंटी देता है।

## 5. प्रभावक / संदर्भ कोड

इनफ्लूएंसर या ब्रांड एम्बेसेडर को एक व्यक्तिगत कोड दें जिसे वे अपने ऑडिएंस के साथ साझा कर सकते हैं।

प्रत्येक कोड एक व्यक्ति से जुड़ा होता है, जिससे आसानी से ट्रैक किया जा सकता है कि कौन सा साझेदार सबसे अधिक बिक्री करता है।

| Field | Value |
|-------|-------|
| **Code** | `SARAH10` |
| **Name** | Influencer — Sarah |
| **Discount type** | Percentage Discount |
| **Discount value** | 10 |
| **Application scope** | Entire Cart |
| **Max uses total** | 0 (unlimited) |
| **Max uses per customer** | 1 |

प्रत्येक इनफ्लुएंसर के लिए एक अलग वौचर बनाएं ताकि आप वौचर डैशबोर्ड पर रेडीम काउंट की तुलना कर सकें। प्रत्येक कोड के उपयोग का नियमित रूप से समीक्षा करें ताकि आप रिटर्न ऑन इन्वेस्टमेंट (ROI) का मूल्यांकन कर सकें और जान सकें कि कौन से साझेदारी को जारी रखना चाहिए।

## 6. Flash Discount Code

एक छोटे से विंडो के लिए एक गहरा डिस्काउंट बनाकर त्वरित खरीदारी के लिए आकर्षण बनाए रखें। ईमेल ब्लास्ट या एनाउंसमेंट बार के साथ कोड का उपयोग करके तत्काल ट्रैफिक को बढ़ाएं।

| क्षेत्र | मान |
|-------|-------|
| **कोड** | `FLASH40` |
| **नाम** | 24 घंटे की फ्लैश सेल |
| **छूट के प्रकार** | प्रतिशत छूट |
| **छूट के मूल्य** | 40 |
| **अधिकतम छूट की राशि** | 75.00 |
| **लागू करने के क्षेत्र** | पूरा कार्ट |
| **शुरू की तारीख** | शुक्रवार 9:00 एएम |
| **समाप्ति तिथि** | शनिवार 9:00 एएम |
| **अधिकतम उपयोग कुल** | 200 |
| **बिक्री वाली वस्तुओं को छोड़ दें** | चेक किया गया |

अधिकतम उपयोग कुल 200 पर बैठक आपके मार्जिन की रक्षा करती है। एक बार जब अधिकतम सीमा पहुँच जाती है, तो कोड स्वचालित रूप से काम करना बंद कर देता है। अपने बाजार प्रचार में सीमित मात्रा के बारे में जानकारी दें - "केवल पहले 200 ग्राहक" - ताकि आप त्वरित करने के लिए आवश्यकता को बढ़ा सकें।

## सुझाव

- **कोड छोटे और याद रखने योग्य रखें** — सभी बड़े अक्षर, कोई विशेष कैरेक्टर नहीं।

ग्राहक चेकआउट पर इन्हें टाइप करते हैं, इसलिए `SUMMER25` `summer-sale-2025-promo` के बजाय बेहतर है।
- **रेडीमेशन गिनती के ट्रैक करें** — वॉचर डैशबोर्ड नियमित रूप से देखें कि कौन से अभियान सबसे अच्छा प्रदर्शन करते हैं।

कुल रिडेम्प्शन के खर्च के खिलाफ रिटर्न ऑन इन्वेस्टमेंट (ROI) की गणना करें।
- **लॉन्च के पहले परीक्षण करें** — कोड के साथ एक परीक्षण ऑर्डर दें ताकि यह सही तरीके से लागू हो रहा हो, न्यूनतम ऑर्डर मूल्य का सम्मान कर रहा हो, और अपेक्षानुसार (या नहीं) संयोजित हो रहा हो।
- **पुनरावृत्ति अभियानों के लिए कोड घूमाएं** — प्रत्येक ऋतु (जैसे, `SUMMER25`, `SUMMER26`) के लिए एक नया कोड उपयोग करें ताकि आप प्रत्येक अभियान को स्वतंत्र रूप से माप सकें और पुराने कोडों के प्रसार को रोक सकें।
- **प्रतिशत वॉउचर पर अधिकतम छूट की राशि सेट करें** — $1,000 के ऑर्डर पर 25% की छूट $250 होती है।

अधिकतम छूट राशि के क्षेत्र का उपयोग करके डॉलर के मूल्य को सीमित करें और अपने मार्जिन की रक्षा करें।
- **अनुच्छेद बार का उपयोग करें** — अपने स्टोरफ्रंट अनुच्छेद बार के साथ फ्लैश और ऋतु वाले कोड का जोड़ा दें ताकि प्रत्येक यात्री तुरंत प्रमोशन को देख सके।
