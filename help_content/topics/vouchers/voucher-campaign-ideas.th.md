---
title: ไอเดียแคมเปญคูปอง
---

คู่มือนี้ครอบคลุมแคมเปญคูปองที่ได้รับการพิสูจน์แล้ว ซึ่งคุณสามารถตั้งค่าได้ภายในไม่กี่นาที แต่ละแคมเปญมีการตั้งค่าที่ชัดเจนเพื่อให้คุณสามารถเริ่มต้นจากไอเดียไปสู่การส่งเสริมการขายได้อย่างรวดเร็ว สร้างคูปองทั้งหมดที่ **Marketing > Vouchers** และคลิกที่ **+ Add Voucher**

![Welcome Voucher](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. ส่วนลดต้อนรับสำหรับลูกค้าใหม่

เสนอสิทธิประโยชน์ให้กับลูกค้าใหม่เพื่อให้พวกเขาสามารถสั่งซื้อสินค้าได้ครบ คูปองต้อนรับช่วยสร้างความไว้วางใจและลดความกังวลสำหรับลูกค้าที่ยังไม่เคยสั่งซื้อสินค้าจากร้านของคุณมาก่อน

| Field | Value |
|-------|-------|
| **Code** | `WELCOME15` |
| **Name** | Welcome discount — new customers |
| **Discount type** | Percentage Discount |
| **Discount value** | 15 |
| **Application scope** | Entire Cart |
| **Max uses per customer** | 1 |
| **Max uses total** | 0 (unlimited) |
| **First time customers only** | Checked |
| **Start date / End date** | Leave both empty — always active |

โปรโมทรหัสนี้ในอีเมลยืนยันการสมัครสมาชิก บนแบนเนอร์หน้าแรก หรือในป๊อปอัปสำหรับผู้เยี่ยมชมครั้งแรก เนื่องจาก **First time customers only** ถูกตั้งค่าไว้ ลูกค้าที่เคยกลับมาแล้วจะไม่สามารถใช้คูปองนี้ได้แม้พวกเขาจะพบรหัสนี้

## 2. ค่าจัดส่งฟรีตามมูลค่าการสั่งซื้อ

ส่งเสริมการสั่งซื้อจำนวนมากขึ้นโดยการเสนอค่าจัดส่งฟรีเมื่อตะกร้าสินค้าถึงมูลค่าขั้นต่ำ ตั้งค่า **Discount value** เป็นจำนวนเงินที่ครอบคลุมค่าจัดส่งมาตรฐานสูงสุดของคุณ

| Field | Value |
|-------|-------|
| **Code** | `FREESHIP50` |
| **Name** | Free shipping on orders over $50 |
| **Discount type** | Fixed Amount Discount |
| **Discount value** | 15.00 (or your max shipping cost) |
| **Application scope** | Entire Cart |
| **Min order value** | 50.00 |
| **Max uses per customer** | 0 (unlimited) |
| **Cannot combine with other vouchers** | Checked |

แสดงรหัสนี้อย่างเด่นชัดบนหน้าสินค้าและในตะกร้าสินค้า ลูกค้าที่ใกล้ถึงมูลค่าขั้นต่ำมักจะเพิ่มสินค้าอีกชิ้นเพื่อให้ตรงเงื่อนไข ซึ่งจะเพิ่มมูลค่าการสั่งซื้อเฉลี่ยของคุณ

## 3. รหัสแคมเปญตามฤดูกาล

จัดแคมเปญที่มีระยะเวลาจำกัดที่เกี่ยวข้องกับเทศกาล ฤดูกาล หรือเหตุการณ์ต่าง ๆ ตั้งช่วงเวลาที่ชัดเจนเพื่อให้คูปองเปิดใช้งานและหมดอายุโดยอัตโนมัติ

| Field | Value |
|-------|-------|
| **Code** | `SUMMER25` |
| **Name** | Summer sale 2025 |
| **Discount type** | Percentage Discount |
| **Discount value** | 25 |
| **Max discount amount** | 50.00 |
| **Application scope** | Entire Cart |
| **Start date** | June 1 |
| **End date** | August 31 |
| **Cannot combine with other vouchers** | Checked |
| **Cannot combine with sale items** | Checked |

**รหัสที่ใช้ร่วมกัน vs. รหัสเฉพาะบุคคล** — รหัสที่ใช้ร่วมกัน เช่น `SUMMER25` ง่ายต่อการโปรโมท แต่อาจแพร่กระจายไปยังกลุ่มเป้าหมายที่คุณต้องการไม่ได้ สำหรับการควบคุมที่ดีขึ้น ให้ใช้การสร้างรหัสแบบกลุ่มเพื่อสร้างรหัสเฉพาะบุคคลที่ใช้ได้ครั้งเดียว (แต่ละรหัสตั้งค่า **Max uses total** เป็น 1) แจกจ่ายรหัสเฉพาะบุคคลผ่านอีเมลที่มีเป้าหมายเฉพาะ เพื่อให้ลูกค้าแต่ละคนได้รับรหัสของตนเอง

## 4. คูปองรางวัลความภักดี

ให้คูปองที่มีมูลค่าคงที่แก่ลูกค้ารายสำคัญของคุณ ซึ่งพวกเขาสามารถนำไปใช้กับการสั่งซื้อครั้งต่อไป ส่งผ่านทางอีเมลหลังจากลูกค้าบรรลุเป้าหมายการใช้จ่าย หรือเป็นคำขอบคุณสำหรับการซื้อซ้ำ

| Field | Value |
|-------|-------|
| **Code** | `VIP20` |
| **Name** | VIP loyalty reward |
| **Discount type** | Fixed Amount Discount |
| **Discount value** | 20.00 |
| **Application scope** | Entire Cart |
| **Min order value** | 100.00 |
| **Max uses per customer** | 1 |
| **Max uses total** | 0 (unlimited) |
| **Days valid** | 90 |

การตั้งค่า **Days valid** เป็น 90 ให้ลูกค้ามีเวลาสามเดือนในการใช้คูปองนี้นับจากวันที่ใช้ครั้งแรก **Min order value** ของ $100 ทำให้มั่นใจว่าส่วนลดนี้จะถูกนำไปใช้กับการสั่งซื้อที่มีความสำคัญ

## 5. รหัสอินฟลูเอนเซอร์ / รหัสแนะนำ

ให้อินฟลูเอนเซอร์หรือทูตแบรนด์ของคุณมีรหัสส่วนตัวที่พวกเขาแบ่งปันกับผู้ติดตามของพวกเขา แต่ละรหัสเชื่อมโยงกับบุคคลหนึ่ง ทำให้ติดตามได้ง่ายว่าพาร์ตเนอร์คนใดที่สร้างยอดขายได้มากที่สุด


| Field | Value |
|-------|-------|
| **Code** | `SARAH10` |
| **Name** | Influencer — Sarah |
| **Discount type** | Percentage Discount |
| **Discount value** | 10 |
| **Application scope** | Entire Cart |
| **Max uses total** | 0 (unlimited) |
| **Max uses per customer** | 1 |

สร้างคูปองแยกสำหรับ Influencer แต่ละคน เพื่อที่คุณจะสามารถเปรียบเทียบจำนวนการใช้คูปองได้บนแดชบอร์ดคูปอง ตรวจสอบการใช้งานของแต่ละคูปองเป็นประจำเพื่อวัดผลตอบแทนจากการลงทุน (ROI) และตัดสินใจว่าจะดำเนินการร่วมมือกับใครต่อไป

## 6. Flash Discount Code

สร้างความเร่งด่วนด้วยส่วนลดที่มีมูลค่าสูง ซึ่งสามารถใช้ได้ในช่วงเวลาสั้น ๆ ให้คูปองนี้มาพร้อมกับแถบประกาศหรือการส่งอีเมลเพื่อดึงดูดลูกค้าให้เข้ามาทันที

| Field | Value |
|-------|-------|
| **Code** | `FLASH40` |
| **Name** | 24-hour flash sale |
| **Discount type** | Percentage Discount |
| **Discount value** | 40 |
| **Max discount amount** | 75.00 |
| **Application scope** | Entire Cart |
| **Start date** | Friday 9:00 AM |
| **End date** | Saturday 9:00 AM |
| **Max uses total** | 200 |
| **Exclude sale items** | Checked |

การตั้งค่า **Max uses total** ที่ 200 ช่วยปกป้องรายได้ของคุณ เมื่อถึงจำนวนที่กำหนด คูปองจะหยุดทำงานโดยอัตโนมัติ ระบุในแคมเปญการตลาดของคุณว่ามีจำนวนจำกัด — "เฉพาะลูกค้า 200 คนแรก" — เพื่อเพิ่มความเร่งด่วน

## Campaign Quick Reference

| Campaign | Code Example | Type | Value | Key Settings |
|----------|-------------|------|-------|--------------|
| Welcome discount | `WELCOME15` | Percentage | 15% | First time only, per-customer = 1 |
| Free shipping | `FREESHIP50` | Fixed Amount | $15 | Min order $50, no combining |
| Seasonal sale | `SUMMER25` | Percentage | 25% | Date range, max discount $50 |
| Loyalty reward | `VIP20` | Fixed Amount | $20 | Min order $100, 90-day validity |
| Influencer code | `SARAH10` | Percentage | 10% | Per-customer = 1, track by code |
| Flash discount | `FLASH40` | Percentage | 40% | 24-hour window, 200 max uses |

## Tips

- **รักษาคูปองให้สั้นและจดจำได้ง่าย** — ใช้ตัวพิมพ์ใหญ่ ไม่มีตัวอักษรพิเศษ ลูกค้าจะพิมพ์คูปองเหล่านี้ขณะชำระเงิน ดังนั้น `SUMMER25` ดีกว่า `summer-sale-2025-promo`
- **ติดตามจำนวนการใช้คูปอง** — ตรวจสอบแดชบอร์ดคูปองเป็นประจำเพื่อดูแคมเปญใดที่มีประสิทธิภาพสูงสุด เปรียบเทียบจำนวนการใช้คูปองกับรายได้เพื่อคำนวณผลตอบแทนจากการลงทุน (ROI)
- **ทดสอบก่อนเปิดตัว** — สร้างคำสั่งซื้อทดสอบด้วยคูปองเพื่อตรวจสอบว่าคูปองใช้งานได้ตามที่คาดไว้ ปฏิบัติตามค่าขั้นต่ำของคำสั่งซื้อ และรวม (หรือไม่รวม) ตามที่คาดไว้
- **เปลี่ยนคูปองสำหรับแคมเปญซ้ำ ๆ** — ใช้คูปองใหม่ในแต่ละฤดูกาล (เช่น `SUMMER25`, `SUMMER26`) เพื่อให้คุณสามารถวัดผลของแต่ละแคมเปญได้อย่างเป็นอิสระ และป้องกันไม่ให้คูปองเก่าถูกใช้ซ้ำ
- **ตั้งค่าจำนวนส่วนลดสูงสุดสำหรับคูปองเปอร์เซ็นต์** — ส่วนลด 25% สำหรับคำสั่งซื้อ 1,000 ดอลลาร์คือ 250 ดอลลาร์ ใช้ฟิลด์ **Max discount amount** เพื่อกำหนดค่าสูงสุดและปกป้องรายได้ของคุณ
- **ใช้แถบประกาศ** — จับคู่คูปองแบบแฟลชและคูปองตามฤดูกาลกับแถบประกาศของร้านค้าของคุณ เพื่อให้ผู้เยี่ยมชมทุกคนเห็นโปรโมชันทันที
