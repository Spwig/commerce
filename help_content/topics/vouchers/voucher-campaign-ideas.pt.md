---
title: Ideias de Campanhas com Cupons
---

Este guia aborda campanhas com cupons comprovadas que você pode configurar em minutos. Cada campanha inclui as configurações exatas para configurar, para que você possa ir da ideia para uma promoção ativa rapidamente. Crie todos os cupons em **Marketing > Cupons** e clique em **+ Adicionar Cupom**.

![Cupom de Bem-vindo](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. Desconto de Bem-vindo para Novos Clientes

Ofereça um incentivo para compradores de primeira vez completarem sua compra. Um desconto de bem-vindo constrói confiança e reduz a hesitação para clientes que nunca compraram em sua loja.

| Campo | Valor |
|-------|-------|
| **Código** | `WELCOME15` |
| **Nome** | Desconto de bem-vindo — novos clientes |
| **Tipo de desconto** | Desconto em Percentagem |
| **Valor do desconto** | 15 |
| **Âmbito de aplicação** | Todo o Carrinho |
| **Máximo de utilizações por cliente** | 1 |
| **Máximo de utilizações total** | 0 (ilimitado) |
| **Apenas para clientes de primeira vez** | Marcado |
| **Data de início / Data de término** | Deixe ambos vazios — sempre ativo |

Promova esse código no e-mail de confirmação de inscrição, em seu banner de página inicial ou em um pop-up para visitantes de primeira vez. Como **Apenas para clientes de primeira vez** está marcado, clientes que retornarem não podem resgatá-lo, mesmo que encontrem o código.

## 2. Limiar de Envio Gratuíto

Incentive pedidos maiores oferecendo envio gratuito uma vez que o carrinho alcance um valor mínimo. Defina o **Valor do desconto** para um valor que cubra seu maior custo de envio padrão.

| Campo | Valor |
|-------|-------|
| **Código** | `FREESHIP50` |
| **Nome** | Envio gratuito para pedidos acima de $50 |
| **Tipo de desconto** | Desconto em Valor Fixo |
| **Valor do desconto** | 15.00 (ou seu maior custo de envio) |
| **Âmbito de aplicação** | Todo o Carrinho |
| **Valor mínimo do pedido** | 50.00 |
| **Máximo de utilizações por cliente** | 0 (ilimitado) |
| **Não pode ser combinado com outros cupons** | Marcado |

Exiba esse código em destaque em suas páginas de produtos e no carrinho. Clientes próximos ao limiar geralmente adicionam outro item para se qualificar, aumentando seu valor médio de pedido.

## 3. Códigos de Campanhas Sazonais

Realize promoções limitadas no tempo ligadas a feriados, estações ou eventos. Defina um intervalo de data estrito para que o cupom ative e expire automaticamente.

| Campo | Valor |
|-------|-------|
| **Código** | `SUMMER25` |
| **Nome** | Liquidação de verão 2025 |
| **Tipo de desconto** | Desconto em Percentagem |
| **Valor do desconto** | 25 |
| **Máximo de valor do desconto** | 50.00 |
| **Âmbito de aplicação** | Todo o Carrinho |
| **Data de início** | 1 de junho |
| **Data de término** | 31 de agosto |
| **Não pode ser combinado com outros cupons** | Marcado |
| **Não pode ser combinado com itens em promoção** | Marcado |

**Códigos compartilhados vs. únicos** — Um código compartilhado como `SUMMER25` é fácil de promover, mas pode se espalhar para fora do seu p\xfablico-alvo intencionado. Para um controle mais apertado, use geração em lote para criar códigos únicos de uso único (cada um com **Máximo de utilizações total** definido como 1). Distribua códigos únicos por meio de e-mails direcionados para que cada cliente receba o seu próprio.

## 4. Cupom de Recompensa de Fidelidade

Recompense seus melhores clientes com um cupom de valor fixo que eles podem aplicar ao seu próximo pedido. Envie-o por e-mail após um cliente atingir uma marca de gasto ou como um agradecimento por compras repetidas.

| Campo | Valor |
|-------|-------|
| **Código** | `VIP20` |
| **Nome** | Recompensa de fidelidade VIP |
| **Tipo de desconto** | Desconto em Valor Fixo |
| **Valor do desconto** | 20.00 |
| **Âmbito de aplicação** | Todo o Carrinho |
| **Valor mínimo do pedido** | 100.00 |
| **Máximo de utilizações por cliente** | 1 |
| **Máximo de utilizações total** | 0 (ilimitado) |
| **Dias válidos** | 90 |

Definir **Dias válidos** para 90 dá aos clientes três meses para resgatar a recompensa a partir da data de primeira utilização. O **Valor mínimo do pedido** de $100 garante que o desconto se aplique a pedidos significativos.

## 5. Código de Influenciador / Indicação

Dé um código pessoal a influenciadores ou embaixadores de marca para que compartilhem com seu p\xfablico. Cada código está ligado a uma pessoa, tornando fácil acompanhar qual parceiro gera mais vendas.

| Campo | Valor |
|-------|-------|
| **Código** | `SARAH10` |
| **Nome** | Influenciadora — Sarah |
| **Tipo de desconto** | Desconto em Percentagem |
| **Valor do desconto** | 10 |
| **Âmbito de aplicação** | Todo o Carrinho |
| **Máximo de usos total** | 0 (ilimitado) |
| **Máximo de usos por cliente** | 1 |

Crie um cupom separado para cada influenciadora para que você possa comparar as contagens de resgates no painel de cupons. Revise regularmente o uso de cada código para medir o ROI e decidir quais parcerias continuar.

## 6. Código de Desconto de Flash

Crie urgência com um desconto profundo disponível por um período muito curto. Combine o código com uma barra de anúncio ou campanha por e-mail para direcionar tráfego imediato.

| Campo | Valor |
|-------|-------|
| **Código** | `FLASH40` |
| **Nome** | Venda de 24 horas |
| **Tipo de desconto** | Desconto em Percentagem |
| **Valor do desconto** | 40 |
| **Máximo de desconto** | 75,00 |
| **Âmbito de aplicação** | Todo o Carrinho |
| **Data de início** | Sexta-feira 9:00 AM |
| **Data de término** | Sábado 9:00 AM |
| **Máximo de usos total** | 200 |
| **Excluir itens em promoção** | Marcado |

Definir **Máximo de usos total** em 200 protege seus margens. Uma vez atingido o limite, o código para de funcionar automaticamente. Mencione a quantidade limitada em sua campanha de marketing — "apenas os primeiros 200 clientes" — para amplificar a urgência.

|----------|-------------|------|-------|--------------|
| Desconto de boas-vindas | `WELCOME15` | Percentagem | 15% | Apenas primeira vez, por cliente = 1 |
| Frete grátis | `FREESHIP50` | Valor Fixo | $15 | Valor mínimo de pedido $50, não combinável |
| Venda sazonal | `SUMMER25` | Percentagem | 25% | Período de datas, máximo de desconto $50 |

## Dicas

- **Mantenha os códigos curtos e memoráveis** — Apenas letras maiúsculas, sem caracteres especiais. Os clientes digitam esses códigos no checkout, então `SUMMER25` é melhor que `summer-sale-2025-promo`.
- **Monitore as contagens de resgates** — Verifique regularmente o painel de cupons para ver quais campanhas têm melhor desempenho. Compare o total de resgates contra a receita para calcular o ROI.
- **Teste antes do lançamento** — Faça um pedido de teste com o código para confirmar que ele se aplica corretamente, respeita o valor mínimo do pedido e combina (ou não combina) conforme o esperado.
- **Rotacione códigos para campanhas repetidas** — Use um novo código cada temporada (ex: `SUMMER25`, `SUMMER26`) para que você possa medir cada campanha de forma independente e evitar que códigos antigos circulem.
- **Defina valores máximos de desconto em cupons de percentagem** — Um desconto de 25% em um pedido de $1.000 é $250. Use o campo **Máximo de desconto** para limitar o valor em dólares e proteger seus margens.
- **Use a barra de anúncio** — Combine códigos de flash e sazonais com a barra de anúncio do seu site para que todos os visitantes vejam a promoção imediatamente.
