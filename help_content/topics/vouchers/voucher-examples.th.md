title: ตัวอย่างคูปอง

คู่มือนี้ให้ตัวอย่างทีละช่องสำหรับประเภทคูปองที่พบบ่อยที่สุด ตัวอย่างแต่ละตัวแสดงให้เห็นว่าต้องกรอกอะไรเมื่อสร้างคูปองที่ **Marketing > Vouchers** → **+ Add Voucher**

![Voucher Card](/static/core/admin/img/help/voucher-examples/voucher-card.webp)

## ตัวอย่าง 1: ส่วนลดเปอร์เซ็นต์พร้อมขีดจำกัดส่วนลด

**สถานการณ์:** ให้ส่วนลด 20% ทั้งตะกร้าสินค้า แต่จำกัดส่วนลดสูงสุดที่ $50 เพื่อรักษาผลกำไรของคำสั่งซื้อที่มีมูลค่าสูง ไม่มีวันหมดอายุ

| Field | Value |
|-------|-------|
| Code | `SAVE20` |
| Name | 20% Off — Max $50 |
| Discount Type | Percentage |
| Discount Value | 20 |
| Max Discount Amount | 50 |
| Application Scope | Entire Cart |
| Max Uses Total | *(empty — unlimited)* |
| Max Uses Per Customer | 1 |
| Min Order Value | *(empty — no minimum)* |

**วิธีที่ขีดจำกัดทำงาน:** สำหรับคำสั่งซื้อ $200 ส่วนลดคือ $40 สำหรับคำสั่งซื้อ $300 จะเป็น $60 แต่ขีดจำกัดทำให้เหลือเพียง $50 สำหรับคำสั่งซื้อ $500 ส่วนลดยังคงอยู่ที่ $50 นี่ช่วยให้คุณสามารถเปิดตัวโปรโมชันที่ฟังดูใจกว้าง แต่ยังคงทำนายส่วนลดได้อย่างแม่นยำ

## ตัวอย่าง 2: ส่วนลดจำนวนคงที่พร้อมขั้นต่ำ

**สถานการณ์:** ให้ลูกค้าส่วนลด $10 สำหรับคำสั่งซื้อที่มีมูลค่าเกิน $75 เพื่อส่งเสริมให้ลูกค้าสั่งซื้อสินค้าจำนวนมากขึ้น

| Field | Value |
|-------|-------|
| Code | `TAKE10` |
| Name | $10 Off Orders Over $75 |
| Discount Type | Fixed Amount |
| Discount Value | 10 |
| Application Scope | Entire Cart |
| Min Order Value | 75 |
| Max Uses Per Customer | 0 *(unlimited)* |
| End Date | *(empty — no expiry)* |

> **หมายเหตุ:** การตั้งค่าขั้นต่ำของมูลค่าคำสั่งซื้อช่วยปกป้องรายได้ของคุณ หากไม่ตั้งค่าขั้นต่ำ ลูกค้าสามารถใช้คูปองนี้กับคำสั่งซื้อ $12 และทำลายกำไรของคุณ ดังนั้นควรใช้คูปองจำนวนคงที่พร้อมกับขั้นต่ำที่เหมาะสมเสมอ

## ตัวอย่าง 3: คูปองส่งฟรี

**สถานการณ์:** ให้ส่งฟรีสำหรับคำสั่งซื้อใดๆ โดยไม่มีขั้นต่ำในการสั่งซื้อ

| Field | Value |
|-------|-------|
| Code | `FREESHIP` |
| Name | Free Shipping |
| Discount Type | Free Shipping |
| Application Scope | Entire Cart |
| Max Uses Total | *(empty — unlimited)* |
| Max Uses Per Customer | 1 |
| Min Order Value | *(empty — no minimum)* |

> **หมายเหตุ:** เลือกประเภทส่วนลด **Free Shipping** ซึ่งจะลบค่าใช้จ่ายในการจัดส่งออกจากรายการสั่งซื้อโดยอัตโนมัติ นี่คือวิธีที่สะอาดที่สุดและทำงานได้ไม่ว่าลูกค้าจะเลือกวิธีจัดส่งใด

## ตัวอย่าง 4: คูปองต้อนรับลูกค้าใหม่

**สถานการณ์:** ให้ส่วนลด 15% สำหรับคำสั่งซื้อแรกของลูกค้าใหม่เพื่อส่งเสริมให้ลูกค้าทำการซื้อ

| Field | Value |
|-------|-------|
| Code | `WELCOME15` |
| Name | Welcome — 15% Off First Order |
| Discount Type | Percentage |
| Discount Value | 15 |
| Application Scope | Entire Cart |
| Max Uses Per Customer | 1 |
| First Time Customers Only | Checked |

ระบบตรวจสอบสถานะลูกค้าใหม่โดยการตรวจสอบว่าลูกค้ามีคำสั่งซื้อที่เสร็จสิ้นก่อนหน้านี้หรือไม่ หากลูกค้าที่มีประวัติการสั่งซื้อพยายามใช้คูปองนี้ เขาจะเห็นข้อความแสดงข้อผิดพลาดที่ชัดเจนที่เคาน์เตอร์

## ตัวอย่าง 5: คูปองเฉพาะสินค้า

**สถานการณ์:** ให้ส่วนลด $5 สำหรับสินค้าที่เลือก — ตัวอย่างเช่น เพื่อเพิ่มยอดขายของสินค้าที่ขายช้า

| Field | Value |
|-------|-------|
| Code | `PICK5` |
| Name | $5 Off Selected Items |
| Discount Type | Fixed Amount |
| Discount Value | 5 |
| Application Scope | Specific Products |
| Eligible Products | *(select the target products)* |
| Max Uses Per Customer | 1 |

> **หมายเหตุ:** ใช้ขอบเขตของสินค้าเมื่อคุณต้องการลดราคา SKU แต่ละตัว ใช้ขอบเขตของหมวดหมู่ (ตัวอย่างถัดไป) เมื่อคุณต้องการลดราคาทั้งหมดในแผนก ขอบเขตของสินค้าให้คุณควบคุมได้อย่างแม่นยำ; ขอบเขตของหมวดหมู่ง่ายต่อการดูแลเมื่อแคตตาล็อกของคุณเปลี่ยนแปลงบ่อย

## ตัวอย่าง 6: คูปองหมวดหมู่

**สถานการณ์:** เปิดตัวโปรโมชันลด 25% สำหรับสินค้าทั้งหมดในหมวดหมู่อิเล็กทรอนิกส์

| Field | Value |
|-------|-------|
| Code | `ELEC25` |
| Name | 25% Off Electronics |
| Discount Type | Percentage |
| Discount Value | 25 |
| Application Scope | Specific Categories |
| Eligible Categories | Electronics |
| Max Uses Total | *(empty — unlimited)* |
| Max Uses Per Customer | 1 |


เมื่อจำกัดขอบเขตให้กับหมวดหมู่ ส่วนลดจะใช้ได้กับสินค้าที่ตรงตามเงื่อนไขในตะกร้าเท่านั้น

สินค้าที่ไม่ใช่สินค้าอิเล็กทรอนิกส์จะถูกเรียกเก็บเงินในราคาเต็ม

## การเปรียบเทียบประเภทส่วนลด

| ประเภท | วิธีการใช้งาน | เหมาะสำหรับ | ตัวอย่าง |
|------|-------------|----------|---------|
| **เปอร์เซ็นต์** | หักเปอร์เซ็นต์จากยอดรวมที่ตรงตามเงื่อนไข | ส่วนลดที่เพิ่มขึ้นตามขนาดของคำสั่งซื้อ | ส่วนลด 20% สำหรับตะกร้าทั้งหมด |
| **จำนวนคงที่** | หักจำนวนเงินคงที่ | ส่วนลดที่เรียบง่ายและคาดการณ์ได้ | ลด 10 ดอลลาร์สำหรับคำสั่งซื้อที่มากกว่า 75 ดอลลาร์ |
| **ส่งฟรี** | ยกเลิกค่าใช้จ่ายในการจัดส่งสำหรับคำสั่งซื้อ | ลดการยกเลิกตะกร้าในขั้นตอนชำระเงิน | ส่งฟรี ไม่มีขั้นต่ำ |

## การเปรียบเทียบขอบเขต

| ขอบเขต | วิธีการใช้งาน | เหมาะสำหรับ |
|-------|-------------|----------|
| **ตะกร้าทั้งหมด** | ส่วนลดใช้ได้กับยอดรวมทั้งหมดของคำสั่งซื้อ | ส่วนลดทั่วทั้งร้านค้าและรหัสต้อนรับ |
| **สินค้าเฉพาะ** | ส่วนลดใช้ได้กับสินค้าที่เลือกไว้ในตะกร้าเท่านั้น | ลดสต็อกสินค้าเฉพาะหรือโปรโมชันที่เน้น |
| **หมวดหมู่เฉพาะ** | ส่วนลดใช้ได้กับสินค้าในหมวดหมู่ที่เลือกไว้เท่านั้น | ลดราคาทั้งแผนกและโปรโมชันตามฤดูกาล |

## เคล็ดลับ

- **ใช้รหัสที่จดจำได้ง่าย** — `SUMMER20` ใช้ได้ดีกว่า `COUPONX1600406498` ให้ใช้รหัสที่สร้างโดยอัตโนมัติสำหรับแคมเปญจำนวนมาก
- **ทดสอบก่อนเผยแพร่** — สร้างคำสั่งซื้อทดสอบด้วยรหัสคูปองเพื่อตรวจสอบว่าใช้งานได้ถูกต้องและปฏิบัติตามข้อจำกัดทั้งหมด
- **ติดตามการใช้งาน** — ตรวจสอบจำนวนการแลกใช้บนแต่ละบัตรคูปองเพื่อติดตามประสิทธิภาพของแคมเปญแบบเรียลไทม์
- **รวมกับแถบประกาศ** — โปรโมตรหัสคูปองของคุณในแถบประกาศทั่วเว็บไซต์เพื่อให้ลูกค้าเห็นก่อนเริ่มช้อปปิ้ง
