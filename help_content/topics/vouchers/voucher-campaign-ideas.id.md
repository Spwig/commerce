---
title: Ide Kampanye Voucher
---

Panduan ini mencakup kampanye voucher yang terbukti berhasil yang dapat Anda atur dalam beberapa menit. Setiap kampanye mencakup pengaturan yang tepat untuk dikonfigurasikan sehingga Anda dapat beralih dari ide ke promosi aktif dengan cepat. Buat semua voucher di **Pemasaran > Voucher** dan klik **+ Tambahkan Voucher**.

![Voucher Selamat Datang](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. Diskon Selamat Datang untuk Pelanggan Baru

Tawarkan insentif kepada pembeli baru untuk menyelesaikan pembelian mereka. Diskon selamat datang membangun kepercayaan dan mengurangi keraguan bagi pelanggan yang belum pernah memesan dari toko Anda.

| Field | Value |
|-------|-------|
| **Kode** | `WELCOME15` |
| **Nama** | Diskon selamat datang — pelanggan baru |
| **Jenis diskon** | Diskon Persentase |
| **Nilai diskon** | 15 |
| **Cakupan aplikasi** | Seluruh Keranjang |
| **Maksimal penggunaan per pelanggan** | 1 |
| **Maksimal penggunaan total** | 0 (tidak terbatas) |
| **Hanya untuk pelanggan baru** | Dicentang |
| **Tanggal mulai / Tanggal akhir** | Biarkan keduanya kosong — selalu aktif |

Promosikan kode ini di email konfirmasi pendaftaran, di banner halaman utama, atau dalam pop-up untuk pengunjung baru. Karena **Hanya untuk pelanggan baru** dicentang, pelanggan yang kembali tidak dapat menggunakannya bahkan jika mereka menemukan kode tersebut.

## 2. Ambang Batas Pengiriman Gratis

Mendorong pesanan yang lebih besar dengan menawarkan pengiriman gratis setelah keranjang mencapai nilai minimum. Atur **Nilai diskon** ke jumlah yang mencakup biaya pengiriman standar ter tinggi Anda.

| Field | Value |
|-------|-------|
| **Kode** | `FREESHIP50` |
| **Nama** | Pengiriman gratis untuk pesanan di atas $50 |
| **Jenis diskon** | Diskon Jumlah Tetap |
| **Nilai diskon** | 15.00 (atau biaya pengiriman maksimal Anda) |
| **Cakupan aplikasi** | Seluruh Keranjang |
| **Nilai pesanan minimum** | 50.00 |
| **Maksimal penggunaan per pelanggan** | 0 (tidak terbatas) |
| **Tidak dapat dikombinasikan dengan voucher lain** | Dicentang |

Tampilkan kode ini secara menonjol di halaman produk dan di keranjang. Pelanggan yang mendekati ambang batas sering menambahkan item lain untuk memenuhi syarat, meningkatkan nilai pesanan rata-rata Anda.

## 3. Kode Kampanye Musiman

Lakukan promosi terbatas waktu yang terkait dengan hari raya, musim, atau acara. Atur rentang tanggal yang ketat sehingga voucher diaktifkan dan kedaluwarsa secara otomatis.

| Field | Value |
|-------|-------|
| **Kode** | `SUMMER25` |
| **Nama** | Penjualan musim panas 2025 |
| **Jenis diskon** | Diskon Persentase |
| **Nilai diskon** | 25 |
| **Maksimal jumlah diskon** | 50.00 |
| **Cakupan aplikasi** | Seluruh Keranjang |
| **Tanggal mulai** | 1 Juni |
| **Tanggal akhir** | 31 Agustus |
| **Tidak dapat dikombinasikan dengan voucher lain** | Dicentang |
| **Tidak dapat dikombinasikan dengan barang diskon** | Dicentang |

**Kode yang dibagikan vs. kode unik** — Sebuah kode yang dibagikan seperti `SUMMER25` mudah untuk dipromosikan tetapi dapat menyebar di luar audiens yang ditujukan. Untuk kontrol yang lebih ketat, gunakan pembuatan massal untuk membuat kode unik yang hanya bisa digunakan sekali (masing-masing dengan **Maksimal penggunaan total** diatur ke 1). Sebarkan kode unik melalui email yang ditargetkan sehingga setiap pelanggan mendapatkan kode mereka sendiri.

## 4. Voucher Hadiah Keanggotaan

Berikan voucher dengan jumlah tetap kepada pelanggan terbaik Anda yang dapat mereka terapkan pada pesanan berikutnya. Kirimkan melalui email setelah pelanggan mencapai batas pengeluaran tertentu atau sebagai ucapan terima kasih untuk pembelian berulang.

| Field | Value |
|-------|-------|
| **Kode** | `VIP20` |
| **Nama** | Hadiah keanggotaan VIP |
| **Jenis diskon** | Diskon Jumlah Tetap |
| **Nilai diskon** | 20.00 |
| **Cakupan aplikasi** | Seluruh Keranjang |
| **Nilai pesanan minimum** | 100.00 |
| **Maksimal penggunaan per pelanggan** | 1 |
| **Maksimal penggunaan total** | 0 (tidak terbatas) |
| **Jumlah hari berlaku** | 90 |

Mengatur **Jumlah hari berlaku** ke 90 memberi pelanggan tiga bulan untuk mencairkan hadiah sejak tanggal penggunaan pertama. **Nilai pesanan minimum** sebesar $100 memastikan diskon berlaku untuk pesanan yang signifikan.

## 5. Kode Pengaruh / Referensi

Berikan pengaruh atau duta merek kode pribadi yang mereka bagikan dengan audiens mereka. Setiap kode terkait dengan satu orang, sehingga mudah untuk melacak mitra mana yang menghasilkan penjualan terbanyak.


| Field | Value |
|-------|-------|
| **Code** | `SARAH10` |
| **Name** | Influencer — Sarah |
| **Discount type** | Diskon Persentase |
| **Discount value** | 10 |
| **Application scope** | Seluruh Keranjang |
| **Max uses total** | 0 (tidak terbatas) |
| **Max uses per customer** | 1 |

Buat voucher terpisah untuk setiap influencer agar Anda dapat membandingkan jumlah penggunaan di dashboard voucher. Periksa penggunaan setiap kode secara teratur untuk mengukur ROI dan memutuskan kemitraan yang akan dilanjutkan.

## 6. Flash Discount Code

Buat rasa mendesak dengan diskon besar yang tersedia untuk jangka waktu sangat singkat. Pasangkan kode ini dengan bar pengumuman atau blast email untuk mendorong lalu lintas segera.

| Field | Value |
|-------|-------|
| **Code** | `FLASH40` |
| **Name** | Penjualan 24 jam |
| **Discount type** | Diskon Persentase |
| **Discount value** | 40 |
| **Max discount amount** | 75.00 |
| **Application scope** | Seluruh Keranjang |
| **Start date** | Jumat pukul 9:00 AM |
| **End date** | Sabtu pukul 9:00 AM |
| **Max uses total** | 200 |
| **Exclude sale items** | Dicentang |

Membatasi **Max uses total** hingga 200 melindungi margin Anda. Setelah batas tercapai, kode akan berhenti bekerja secara otomatis. Sebutkan kuantitas terbatas dalam pemasaran Anda — "hanya untuk 200 pelanggan pertama" — untuk meningkatkan rasa mendesak.

## Campaign Quick Reference

| Campaign | Code Example | Type | Value | Key Settings |
|----------|-------------|------|-------|--------------|
| Welcome discount | `WELCOME15` | Percentage | 15% | Hanya untuk pelanggan baru, per-customer = 1 |
| Free shipping | `FREESHIP50` | Fixed Amount | $15 | Minimal pesanan $50, tidak dapat digabungkan |
| Seasonal sale | `SUMMER25` | Percentage | 25% | Rentang tanggal, maksimal diskon $50 |
| Loyalty reward | `VIP20` | Fixed Amount | $20 | Minimal pesanan $100, validitas 90 hari |
| Influencer code | `SARAH10` | Percentage | 10% | Per-customer = 1, lacak dengan kode |
| Flash discount | `FLASH40` | Percentage | 40% | Jendela 24 jam, maksimal penggunaan 200 |

## Tips

- **Jaga kode singkat dan mudah diingat** — Huruf besar, tanpa karakter khusus. Pelanggan mengetikkan kode ini saat checkout, jadi `SUMMER25` lebih baik daripada `summer-sale-2025-promo`.
- **Lacak jumlah penggunaan** — Periksa dashboard voucher secara teratur untuk melihat kampanye mana yang berkinerja terbaik. Bandingkan total penggunaan dengan pendapatan untuk menghitung ROI.
- **Uji sebelum diluncurkan** — Lakukan pesanan uji dengan kode untuk memastikan kode diterapkan dengan benar, menghormati nilai pesanan minimum, dan digabungkan (atau tidak digabungkan) seperti yang diharapkan.
- **Ganti kode untuk kampanye berulang** — Gunakan kode baru setiap musim (misalnya, `SUMMER25`, `SUMMER26`) agar Anda dapat mengukur setiap kampanye secara terpisah dan mencegah kode lama beredar.
- **Atur jumlah diskon maksimum pada voucher persentase** — Diskon 25% pada pesanan $1.000 adalah $250. Gunakan bidang **Max discount amount** untuk membatasi nilai dolar dan melindungi margin Anda.
- **Gunakan bar pengumuman** — Pasangkan kode flash dan musiman dengan bar pengumuman toko Anda agar setiap pengunjung melihat promosi secara langsung.
