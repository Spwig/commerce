---
title: Ideas para campañas de cupones
---

Esta guía cubre campañas de cupones probadas que puedes configurar en minutos. Cada campaña incluye la configuración exacta para que puedas pasar de la idea a una promoción activa rápidamente. Crea todos los cupones en **Marketing > Cupones** y haz clic en **+ Añadir cupón**.

![Cupon de bienvenida](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. Descuento de bienvenida para nuevos clientes

Ofrece a los primeros compradores un incentivo para completar su compra. Un descuento de bienvenida construye confianza y reduce la hesitación en clientes que nunca han ordenado en tu tienda.

| Campo | Valor |
|-------|-------|
| **Código** | `WELCOME15` |
| **Nombre** | Descuento de bienvenida — nuevos clientes |
| **Tipo de descuento** | Descuento por porcentaje |
| **Valor del descuento** | 15 |
| **Ámbito de aplicación** | Carrito completo |
| **Máximo de usos por cliente** | 1 |
| **Máximo de usos total** | 0 (ilimitado) |
| **Solo para primeros clientes** | Marcado |
| **Fecha de inicio / Fecha de finalización** | Dejar ambos vacíos — siempre activo |

Promociona este código en tu correo electrónico de confirmación de registro, en el banner de tu página de inicio o en una ventana emergente para visitantes primeros. Debido a que **Solo para primeros clientes** está marcado, los clientes que regresan no pueden canjearlo incluso si encuentran el código.

## 2. Umbral de envío gratis

Incentiva pedidos más grandes ofreciendo envío gratis una vez que el carrito alcanza un valor mínimo. Establece el **Valor del descuento** en una cantidad que cubra tu costo de envío estándar más alto.

| Campo | Valor |
|-------|-------|
| **Código** | `FREESHIP50` |
| **Nombre** | Envío gratis en pedidos superiores a $50 |
| **Tipo de descuento** | Descuento de monto fijo |
| **Valor del descuento** | 15.00 (o tu costo máximo de envío) |
| **Ámbito de aplicación** | Carrito completo |
| **Valor mínimo del pedido** | 50.00 |
| **Máximo de usos por cliente** | 0 (ilimitado) |
| **No se puede combinar con otros cupones** | Marcado |

Muestra este código de forma destacada en tus páginas de productos y en el carrito. Los clientes cerca del umbral suelen agregar otro artículo para calificar, aumentando tu valor promedio de pedido.

## 3. Códigos de campañas estacionales

Realiza promociones limitadas en el tiempo vinculadas a festividades, estaciones o eventos. Establece un rango de fechas estricto para que el cupón se active y expire automáticamente.

| Campo | Valor |
|-------|-------|
| **Código** | `SUMMER25` |
| **Nombre** | Oferta de verano 2025 |
| **Tipo de descuento** | Descuento por porcentaje |
| **Valor del descuento** | 25 |
| **Monto máximo de descuento** | 50.00 |
| **Ámbito de aplicación** | Carrito completo |
| **Fecha de inicio** | 1 de junio |
| **Fecha de finalización** | 31 de agosto |
| **No se puede combinar con otros cupones** | Marcado |
| **No se puede combinar con artículos en oferta** | Marcado |

**Códigos compartidos vs. códigos únicos** — Un código compartido como `SUMMER25` es fácil de promocionar, pero puede difundirse más allá de tu audiencia objetivo. Para un control más estricto, usa la generación masiva para crear códigos únicos de uso único (cada uno con **Máximo de usos total** establecido en 1). Distribuye códigos únicos a través de correos electrónicos dirigidos para que cada cliente tenga el suyo.

## 4. Cupón de recompensa de lealtad

Recompensa a tus mejores clientes con un cupón de monto fijo que puedan aplicar a su próximo pedido. Envíalo por correo electrónico después de que un cliente alcance una meta de gasto o como un agradecimiento por compras repetidas.

| Campo | Valor |
|-------|-------|
| **Código** | `VIP20` |
| **Nombre** | Recompensa de lealtad VIP |
| **Tipo de descuento** | Descuento de monto fijo |
| **Valor del descuento** | 20.00 |
| **Ámbito de aplicación** | Carrito completo |
| **Valor mínimo del pedido** | 100.00 |
| **Máximo de usos por cliente** | 1 |
| **Máximo de usos total** | 0 (ilimitado) |
| **Días válidos** | 90 |

Establecer **Días válidos** en 90 da a los clientes tres meses para canjear la recompensa desde la fecha de primer uso. El **Valor mínimo del pedido** de $100 asegura que el descuento se aplique a pedidos significativos.

## 5. Código de influencer / referido

Dale a los influencers o embajadores de marca un código personal que comparten con su audiencia. Cada código está vinculado a una persona, lo que facilita rastrear qué socio genera más ventas.


| Campo | Valor |
|-------|-------|
| **Código** | `SARAH10` |
| **Nombre** | Influencer — Sarah |
| **Tipo de descuento** | Descuento por porcentaje |
| **Valor del descuento** | 10 |
| **Ámbito de aplicación** | Carrito completo |
| **Máximo de usos total** | 0 (ilimitado) |
| **Máximo de usos por cliente** | 1 |

Crea un cupón separado para cada influencer para que puedas comparar las cifras de redención en el panel de control de cupones. Revisa regularmente el uso de cada código para medir el ROI y decidir qué colaboraciones continuar.

## 6. Código de descuento flash

Crea urgencia con un descuento profundo disponible durante un breve período. Combina el código con una barra de anuncio o un envío por correo electrónico para impulsar el tráfico inmediato.

| Campo | Valor |
|-------|-------|
| **Código** | `FLASH40` |
| **Nombre** | Venta flash de 24 horas |
| **Tipo de descuento** | Descuento por porcentaje |
| **Valor del descuento** | 40 |
| **Máximo de descuento** | 75.00 |
| **Ámbito de aplicación** | Carrito completo |
| **Fecha de inicio** | Viernes 9:00 AM |
| **Fecha de finalización** | Sábado 9:00 AM |
| **Máximo de usos total** | 200 |
| **Excluir artículos en oferta** | Marcado |

Establecer un **máximo de usos total** de 200 protege tus márgenes. Una vez que se alcance el límite, el código dejará de funcionar automáticamente. Menciona la cantidad limitada en tu marketing — "solo para los primeros 200 clientes" — para aumentar la urgencia.

## Referencia rápida de campañas

| Campaña | Ejemplo de código | Tipo | Valor | Configuraciones clave |
|----------|-------------|------|-------|--------------|
| Descuento de bienvenida | `WELCOME15` | Porcentaje | 15% | Solo primera vez, usos por cliente = 1 |
| Envío gratis | `FREESHIP50` | Monto fijo | $15 | Pedido mínimo $50, no combinable |
| Venta estacional | `SUMMER25` | Porcentaje | 25% | Rango de fechas, máximo de descuento $50 |
| Recompensa de lealtad | `VIP20` | Monto fijo | $20 | Pedido mínimo $100, validez de 90 días |
| Código de influencer | `SARAH10` | Porcentaje | 10% | Usos por cliente = 1, rastrear por código |
| Descuento flash | `FLASH40` | Porcentaje | 40% | Ventana de 24 horas, máximo de 200 usos |

## Consejos

- **Mantén los códigos cortos y memorables** — En mayúsculas, sin caracteres especiales. Los clientes escriben estos códigos en el momento del pago, por lo tanto, `SUMMER25` es mejor que `summer-sale-2025-promo`.
- **Rastrea las cifras de redención** — Revisa regularmente el panel de control de cupones para ver qué campañas funcionan mejor. Compara el total de redenciones contra las ventas para calcular el ROI.
- **Prueba antes del lanzamiento** — Realiza un pedido de prueba con el código para confirmar que se aplique correctamente, respeta el valor mínimo del pedido y se combine (o no combine) como se espera.
- **Rotar códigos para campañas repetidas** — Usa un nuevo código cada temporada (por ejemplo, `SUMMER25`, `SUMMER26`) para poder medir cada campaña de forma independiente y evitar que los códigos antiguos circulen.
- **Establece montos máximos de descuento en los cupones por porcentaje** — Un descuento del 25% en un pedido de $1,000 es $250. Usa el campo **Máximo de descuento** para limitar el valor en dólares y proteger tus márgenes.
- **Usa la barra de anuncio** — Combina los códigos flash y estacionales con la barra de anuncio de tu tienda para que todos los visitantes vean la promoción de inmediato.
