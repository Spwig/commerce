---
title: クーポンキャンペーンのアイデア
---

このガイドでは、数分で設定できる実績のあるクーポンキャンペーンについて説明します。各キャンペーンには、アイデアから実際のプロモーションまでを迅速に実行できる正確な設定が含まれています。すべてのクーポンは **マーケティング > クーポン** で作成し、**+ クーポンの追加** をクリックしてください。

![ウェルカムクーポン](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. 新規顧客向けのウェルカム割引

初回購入のお客様に購入を完了させるインセンティブを提供します。ウェルカム割引は、これまで店舗で注文したことがない顧客の信頼を築き、購入へのためらいを減らすことができます。

| フィールド | 値 |
|-------|-------|
| **コード** | `WELCOME15` |
| **名前** | ウェルカム割引 — 新規顧客 |
| **割引タイプ** | パーセンテージ割引 |
| **割引値** | 15 |
| **適用範囲** | カート全体 |
| **顧客ごとの最大使用回数** | 1 |
| **合計最大使用回数** | 0 (無制限) |
| **新規顧客のみ** | 有効 |
| **開始日 / 終了日** | 両方を空欄 — 常時有効 |

このコードをサインアップ確認メール、ホームページのバナー、または初回訪問者向けのポップアップで宣伝してください。**新規顧客のみ** が有効になっているため、リピーター顧客がコードを見つけたとしても利用することはできません。

## 2. 閾値に基づく無料配送

カートの最小金額に達した場合に無料配送を提供して、より大きな注文を促進します。**割引値** を、あなたの最高の標準配送コストをカバーする金額に設定してください。

| フィールド | 値 |
|-------|-------|
| **コード** | `FREESHIP50` |
| **名前** | 50ドル以上の注文で無料配送 |
| **割引タイプ** | 固定金額割引 |
| **割引値** | 15.00 (またはあなたの最大配送コスト) |
| **適用範囲** | カート全体 |
| **最低注文金額** | 50.00 |
| **顧客ごとの最大使用回数** | 0 (無制限) |
| **他のクーポンと併用不可** | 有効 |

このコードを製品ページやカートに目立つ場所に掲示してください。しきい値に近い顧客は、条件を満たすためにもう1つの商品を追加する傾向があります。これにより、平均注文金額が増加します。

## 3. 季節限定キャンペーンコード

祝日、季節、またはイベントに関連した限定的なプロモーションを実施します。厳密な日付範囲を設定することで、クーポンが自動的に有効になり、期限切れになります。

| フィールド | 値 |
|-------|-------|
| **コード** | `SUMMER25` |
| **名前** | 2025年夏セール |
| **割引タイプ** | パーセンテージ割引 |
| **割引値** | 25 |
| **最大割引額** | 50.00 |
| **適用範囲** | カート全体 |
| **開始日** | 6月1日 |
| **終了日** | 8月31日 |
| **他のクーポンと併用不可** | 有効 |
| **セール品と併用不可** | 有効 |

**共有コードと個別コード** — `SUMMER25` のような共有コードは、宣伝が簡単ですが、意図した対象を超えて広がる可能性があります。より厳密な制御が必要な場合は、バッチ生成を使用して、各コードの **合計最大使用回数** を1に設定した個別の1回限りのコードを作成してください。個別のコードはターゲットメールを通じて配布し、すべての顧客が独自のコードを受け取ることができます。

## 4. ロイヤルティ報酬クーポン

あなたの最高顧客に、次回の注文に適用できる固定金額のクーポンを提供します。顧客が購入のマイルストーンに達した後、またはリピーター購入の感謝として、メールで送信してください。

| フィールド | 値 |
|-------|-------|
| **コード** | `VIP20` |
| **名前** | VIPロイヤルティ報酬 |
| **割引タイプ** | 固定金額割引 |
| **割引値** | 20.00 |
| **適用範囲** | カート全体 |
| **最低注文金額** | 100.00 |
| **顧客ごとの最大使用回数** | 1 |
| **合計最大使用回数** | 0 (無制限) |
| **有効日数** | 90 |

**有効日数** を90日と設定することで、最初に使用された日から3か月間、顧客が報酬を獲得できるようにします。**最低注文金額** 100ドルは、割引が意味のある注文に適用されることを保証します。

## 5. インフルエンサー / リファラルコード

インフルエンサーやブランドアンバサダーに、彼らのフォロワーと共有するための個人用コードを提供します。各コードは1人に関連付けられ、どのパートナーが最も多くの売上をもたらしているかを簡単に追跡できます。


| フィールド | 値 |
|-------|-------|
| **コード** | `SARAH10` |
| **名前** | インフルエンサー — Sarah |
| **割引タイプ** | パーセンテージ割引 |
| **割引値** | 10 |
| **適用範囲** | カート全体 |
| **最大使用回数（全体）** | 0（無制限） |
| **最大使用回数（顧客ごと）** | 1 |

各インフルエンサーに対して個別の割引コードを作成し、割引コードダッシュボードで使用回数を比較してください。各コードの使用状況を定期的に確認し、ROIを測定し、どの提携を継続するかを判断してください。

## 6. フラッシュ割引コード

非常に短い期間限定で大幅な割引を提供し、緊急性を生み出してください。コードを発表バーまたはメールキャンペーンと組み合わせて、即座のアクセスを促進してください。

| フィールド | 値 |
|-------|-------|
| **コード** | `FLASH40` |
| **名前** | 24時間限定セール |
| **割引タイプ** | パーセンテージ割引 |
| **割引値** | 40 |
| **最大割引額** | 75.00 |
| **適用範囲** | カート全体 |
| **開始日** | 金曜日 9:00 AM |
| **終了日** | 土曜日 9:00 AM |
| **最大使用回数（全体）** | 200 |
| **セール商品を除外** | 有効 |

**最大使用回数**を200に設定することで、利益率を守ることができます。この上限に達すると、コードは自動的に無効になります。マーケティングで「最初の200名限定」といった文言を記載し、緊急性を高めましょう。

## キャンペーンの簡単な参考ガイド

| キャンペーン | コード例 | タイプ | 値 | 主な設定 |
|----------|-------------|------|-------|--------------|
| ウェルカム割引 | `WELCOME15` | パーセンテージ | 15% | 初回のみ、顧客ごと = 1 |
| 無料配送 | `FREESHIP50` | 固定金額 | $15 | 最低注文額 $50、組み合わせ不可 |
| 季節セール | `SUMMER25` | パーセンテージ | 25% | 日付範囲、最大割引 $50 |
| ロイヤルティ報酬 | `VIP20` | 固定金額 | $20 | 最低注文額 $100、90日間有効 |
| インフルエンサーコード | `SARAH10` | パーセンテージ | 10% | 顧客ごと = 1、コードでトラッキング |
| フラッシュ割引 | `FLASH40` | パーセンテージ | 40% | 24時間限定、最大使用回数 200 |

## ヒント

- **コードは短く覚えやすく** — 全大文字、特殊文字は使用しない。顧客はチェックアウト時にコードを入力するため、`SUMMER25` が `summer-sale-2025-promo` よりも良い。
- **使用回数をトラッキング** — 割引コードダッシュボードを定期的に確認し、どのキャンペーンが最も効果的かを確認してください。総使用回数と売上高を比較してROIを計算してください。
- **発売前テスト** — コードを使用してテスト注文を行い、正しい適用、最低注文額の遵守、組み合わせ（または組み合わせ不可）が予期通りであることを確認してください。
- **キャンペーンの再開時にコードを変更** — 各シーズンごとに新しいコードを使用してください（例：`SUMMER25`、`SUMMER26`）ことで、各キャンペーンを個別に測定し、古いコードが流通しないようにします。
- **パーセンテージ割引コードに最大割引額を設定** — $1,000の注文に対して25%の割引は$250になります。**最大割引額**フィールドを使用して、ドル価値を上限に設定し、利益率を守る。
- **発表バーを使用** — フラッシュや季節限定のコードをストアの発表バーと組み合わせて、すべての訪問者がすぐにプロモーションを確認できるようにしてください。
