Этот руководство предоставляет конкретные примеры по каждому полю для наиболее распространенных типов купонов. Каждый пример показывает, что нужно ввести при создании купона в **Маркетинг > Купоны** → **+ Добавить купон**.

![Карточка купона](/static/core/admin/img/help/voucher-examples/voucher-card.webp)

## Пример 1: Процентная скидка с ограничением

**Сценарий:** Предложите скидку 20% на всю корзину, но ограничьте скидку до 50 долларов, чтобы высокозаказные заказы оставались прибыльными. Нет даты истечения срока.

| Поле | Значение |
|-------|-------|
| Код | `SAVE20` |
| Название | 20% скидка — максимум 50 $ |
| Тип скидки | Процент |
| Значение скидки | 20 |
| Максимальная сумма скидки | 50 |
| Область применения | Вся корзина |
| Максимальное количество использований в целом | *(пусто — неограниченно)* |
| Максимальное количество использований на клиента | 1 |
| Минимальная сумма заказа | *(пусто — нет минимальной)* |

**Как работает ограничение:** Для заказа на 200 $ скидка составляет 40 $. Для заказа на 300 $ скидка составила бы 60 $, но ограничение ограничивает её до 50 $. Для заказа на 500 $ скидка всё равно составляет 50 $. Это позволяет проводить привлекательную по звучанию акцию, сохраняя при этом предсказуемую скидку.

## Пример 2: Фиксированная скидка с минимальной суммой

**Сценарий:** Предложите клиентам скидку в 10 $ на любой заказ свыше 75 $, чтобы стимулировать покупки большего объема.

| Поле | Значение |
|-------|-------|
| Код | `TAKE10` |
| Название | 10 $ скидка на заказы свыше 75 $ |
| Тип скидки | Фиксированная сумма |
| Значение скидки | 10 |
| Область применения | Вся корзина |
| Минимальная сумма заказа | 75 |
| Максимальное количество использований на клиента | 0 *(неограниченно)* |
| Дата окончания | *(пусто — нет срока действия)* |

> **Примечание:** Установка минимальной суммы заказа защищает вашу маржу. Без нее клиент мог бы использовать этот код на заказ на 12 $ и уничтожить вашу прибыль. Всегда сочетайте фиксированные купоны с разумным минимальным порогом.

## Пример 3: Бесплатная доставка

**Сценарий:** Предложите бесплатную доставку на любой заказ без минимальной суммы.

| Поле | Значение |
|-------|-------|
| Код | `FREESHIP` |
| Название | Бесплатная доставка |
| Тип скидки | Бесплатная доставка |
| Область применения | Вся корзина |
| Максимальное количество использований в целом | *(пусто — неограниченно)* |
| Максимальное количество использований на клиента | 1 |
| Минимальная сумма заказа | *(пусто — нет минимальной)* |

> **Примечание:** Выберите тип скидки **Бесплатная доставка**, которая автоматически удаляет расходы на доставку из заказа. Это самый чистый подход и работает независимо от того, какой метод доставки выбирает клиент.

## Пример 4: Код приветствия для новых клиентов

**Сценарий:** Предложите новым клиентам скидку 15% на первый заказ, чтобы стимулировать конверсию.

| Поле | Значение |
|-------|-------|
| Код | `WELCOME15` |
| Название | Приветствие — 15% скидка на первый заказ |
| Тип скидки | Процент |
| Значение скидки | 15 |
| Область применения | Вся корзина |
| Максимальное количество использований на клиента | 1 |
| Только для новых клиентов | Отмечено |

Система проверяет статус первого посещения, проверяя, есть ли у клиента какие-либо предыдущие завершенные заказы. Если клиент с историей заказов попытается применить этот код, он увидит четкое сообщение об ошибке на этапе оформления заказа.

## Пример 5: Купон на конкретный товар

**Сценарий:** Предложите скидку в 5 $ на выбранные товары — например, чтобы распродать товары с низким спросом.

| Поле | Значение |
|-------|-------|
| Код | `PICK5` |
| Название | 5 $ скидка на выбранные товары |
| Тип скидки | Фиксированная сумма |
| Значение скидки | 5 |
| Область применения | Конкретные товары |
| Эльегибле продукты | *(выберите целевые товары)* |
| Максимальное количество использований на клиента | 1 |

> **Примечание:** Используйте область товаров, когда вы хотите скидку на отдельные SKU. Используйте область категорий (следующий пример), когда вы хотите скидку на всё в отделе. Область товаров дает вам точный контроль; область категорий проще поддерживать, если ваша коллекция часто меняется.

## Пример 6: Купон на категорию

**Сценарий:** Проведите акцию со скидкой 25% на все товары в категории Электроника.

| Поле | Значение |
|-------|-------|
| Код | `ELEC25` |
| Название | 25% скидка на электронику |
| Тип скидки | Процент |
| Значение скидки | 25 |
| Область применения | Конкретные категории |
| Эльегибле категории | Электроника |
| Максимальное количество использований в целом | *(пусто — неограниченно)* |
| Максимальное количество использований на клиента | 1 |


Когда скидка применяется к категории, она распространяется только на подходящие товары в корзине.

Товары, не относящиеся к электронике, оплачиваются по полной цене.

## Сравнение типов скидок

| Тип | Как работает | Лучше подходит для | Пример |
|------|-------------|----------|---------|
| **Процент** | Вычитает процент от общей суммы подходящих товаров | Скидки, которые увеличиваются с ростом размера заказа | 20% скидка на весь заказ |
| **Фиксированная сумма** | Вычитает фиксированную сумму в долларах | Простые и предсказуемые акции | $10 скидка на заказы свыше $75 |
| **Бесплатная доставка** | Убирает стоимость доставки из заказа | Снижение отказов от покупки на этапе оформления заказа | Бесплатная доставка, без минимальной суммы |

## Сравнение областей действия

| Область действия | Как работает | Лучше подходит для |
|-------|-------------|----------|
| **Весь заказ** | Скидка применяется к общей сумме заказа | Акции по всему магазину и приветственные коды |
| **Конкретные товары** | Скидка применяется только к выбранным товарам в корзине | Очистка конкретного ассортимента или акции с акцентом на товар |
| **Конкретные категории** | Скидка применяется только к товарам в выбранных категориях | Продажи по отделам и сезонные акции |

## Советы

- **Используйте запоминающиеся коды** — `SUMMER20` лучше, чем `COUPONX1600406498`. Сохраните автоматически сгенерированные коды для массовых кампаний.
- **Проверьте перед распространением** — разместите тестовый заказ с промокодом, чтобы убедиться, что он применяется правильно и учитывает все ограничения.
- **Контролируйте использование** — проверяйте количество использования на каждой карточке промокода, чтобы отслеживать эффективность кампании в реальном времени.
- **Сочетайте с информационной панелью** — продвигайте ваш промокод в общем объявлении на сайте, чтобы клиенты видели его до начала покупок.
