---
title: 优惠券活动创意
---

本指南涵盖可立即设置的已验证优惠券活动。每个活动都包含确切的设置，以便您快速从创意过渡到实时推广。在 **营销 > 优惠券** 中创建所有优惠券，然后点击 **+ 添加优惠券**。

![欢迎优惠券](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. 新客户欢迎折扣

为首次购买的客户提供激励，促使他们完成购买。欢迎折扣有助于建立信任，减少从未在您店铺下单的客户的犹豫。

| 字段 | 值 |
|-------|-------|
| **代码** | `WELCOME15` |
| **名称** | 欢迎折扣 — 新客户 |
| **折扣类型** | 百分比折扣 |
| **折扣值** | 15 |
| **适用范围** | 整个购物车 |
| **每位客户最大使用次数** | 1 |
| **总最大使用次数** | 0（无限制） |
| **仅限新客户** | 已勾选 |
| **开始日期 / 结束日期** | 均留空 — 始终有效 |

在注册确认电子邮件、首页横幅或首次访问弹窗中推广此代码。由于 **仅限新客户** 已勾选，即使老客户找到该代码也无法使用。

## 2. 免运费门槛

通过提供达到最低金额后免运费，鼓励客户下更大订单。将 **折扣值** 设置为覆盖您最高标准运费的金额。

| 字段 | 值 |
|-------|-------|
| **代码** | `FREESHIP50` |
| **名称** | 订单满 $50 免运费 |
| **折扣类型** | 固定金额折扣 |
| **折扣值** | 15.00（或您的最高运费） |
| **适用范围** | 整个购物车 |
| **最低订单金额** | 50.00 |
| **每位客户最大使用次数** | 0（无限制） |
| **不可与其他优惠券叠加** | 已勾选 |

在产品页面和购物车中突出显示此代码。接近门槛的客户通常会添加另一件商品以达到要求，从而提高您的平均订单价值。

## 3. 节日活动代码

运行与节假日、季节或活动相关的时间有限的促销活动。设置严格的日期范围，以便优惠券自动激活和过期。

| 字段 | 值 |
|-------|-------|
| **代码** | `SUMMER25` |
| **名称** | 2025 夏季促销 |
| **折扣类型** | 百分比折扣 |
| **折扣值** | 25 |
| **最大折扣金额** | 50.00 |
| **适用范围** | 整个购物车 |
| **开始日期** | 6 月 1 日 |
| **结束日期** | 8 月 31 日 |
| **不可与其他优惠券叠加** | 已勾选 |
| **不可与促销商品叠加** | 已勾选 |

**共享代码与唯一代码** — 类似 `SUMMER25` 的共享代码易于推广，但可能超出您预期的受众范围。如需更精确的控制，使用批量生成功能创建唯一的一次性代码（每个代码的 **总最大使用次数** 设置为 1）。通过定向电子邮件分发唯一代码，确保每位客户都能获得自己的代码。

## 4. 忠诚度奖励优惠券

向您的最佳客户发放固定金额的优惠券，他们可以在下一次订单中使用。当客户达到消费里程碑时通过电子邮件发送，或作为对重复购买的感谢。

| 字段 | 值 |
|-------|-------|
| **代码** | `VIP20` |
| **名称** | VIP 忠诚度奖励 |
| **折扣类型** | 固定金额折扣 |
| **折扣值** | 20.00 |
| **适用范围** | 整个购物车 |
| **最低订单金额** | 100.00 |
| **每位客户最大使用次数** | 1 |
| **总最大使用次数** | 0（无限制） |
| **有效期（天）** | 90 |

将 **有效期** 设置为 90 天，使客户在首次使用后有三个月的时间兑换奖励。**最低订单金额** 为 $100，确保折扣适用于有意义的订单。

## 5. 影响者 / 推荐代码

为影响者或品牌大使提供个人代码，他们可以与自己的受众分享。每个代码与一个人绑定，便于跟踪哪个合作伙伴带来了最多的销售。


| 字段 | 值 |
|-------|-------|
| **代码** | `SARAH10` |
| **名称** | 影响者 — Sarah |
| **折扣类型** | 百分比折扣 |
| **折扣值** | 10 |
| **适用范围** | 整个购物车 |
| **总使用次数上限** | 0（无限制） |
| **每位顾客使用次数上限** | 1 |

为每位影响者创建单独的优惠券，以便在优惠券仪表板上比较使用次数。定期查看每个代码的使用情况，以衡量投资回报率并决定继续哪些合作。

## 6. 限时折扣代码

通过一个非常短的时间窗口内的深度折扣来制造紧迫感。将代码与横幅公告或电子邮件推送结合使用，以立即吸引流量。

| 字段 | 值 |
|-------|-------|
| **代码** | `FLASH40` |
| **名称** | 24小时限时促销 |
| **折扣类型** | 百分比折扣 |
| **折扣值** | 40 |
| **最大折扣金额** | 75.00 |
| **适用范围** | 整个购物车 |
| **开始日期** | 周五上午9:00 |
| **结束日期** | 周六上午9:00 |
| **总使用次数上限** | 200 |
| **排除促销商品** | 已勾选 |

将 **总使用次数上限** 设为 200 可以保护您的利润。一旦达到上限，代码将自动停止使用。在您的营销中提及有限数量 — “仅限前200位顾客” — 以增强紧迫感。

## 小贴士

- **保持代码简短易记** — 全大写，不使用特殊字符。顾客在结账时输入这些代码，因此 `SUMMER25` 比 `summer-sale-2025-promo` 更好。
- **跟踪使用次数** — 定期查看优惠券仪表板，了解哪些活动表现最佳。将总使用次数与收入进行比较，以计算投资回报率。
- **发布前进行测试** — 使用代码下测试订单，确认折扣正确应用，尊重最低订单金额，并按预期叠加（或不叠加）。
- **重复活动使用新代码** — 每个季节使用新代码（例如 `SUMMER25`、`SUMMER26`），以便独立衡量每个活动的表现，并防止旧代码继续流通。
- **在百分比优惠券上设置最大折扣金额** — $1,000 订单的 25% 折扣是 $250。使用 **最大折扣金额** 字段来限制折扣金额，保护您的利润。
- **使用公告栏** — 将限时和季节性代码与您的商店公告栏结合使用，让每位访客立即看到促销信息。
