---
title: فكرة حملات الكوبونات
---

يغطي هذا الدليل حملات الكوبونات المؤكدة التي يمكنك إعدادها في دقائق. تحتوي كل حملة على الإعدادات الدقيقة لتكوينها بحيث يمكنك الانتقال من الفكرة إلى الحملة النشطة بسرعة. أنشئ جميع الكوبونات في **التسويق > الكوبونات** واضغط على **+ إضافة كوبون**.

![كوبون الترحيب](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. خصم ترحيب للعملاء الجدد

قدم حوافز للعملاء الجدد لإكمال شرائهم. يبني خصم الترحيب الثقة ويقلل من التردد لدى العملاء الذين لم يطلبوا من متجرك من قبل.

| الحقل | القيمة |
|-------|-------|
| **الكود** | `WELCOME15` |
| **الاسم** | خصم ترحيب - عملاء جدد |
| **نوع الخصم** | خصم نسبة |
| **قيمة الخصم** | 15 |
| **مجال التطبيق** | السلة بأكملها |
| **عدد الاستخدامات القصوى لكل عميل** | 1 |
| **عدد الاستخدامات القصوى الإجمالي** | 0 (غير محدود) |
| **فقط للعملاء الجدد** | مُشطب |
| **تاريخ البدء / تاريخ الانتهاء** | اترك كلاهما فارغًا - دائم النشاط |

قم بتعزيز هذا الكود في بريد إلكتروني تأكيد التسجيل، أو في شارة الصفحة الرئيسية، أو في نافذة منبثقة للزوار الجدد. نظرًا لأن **فقط للعملاء الجدد** مُشطب، لا يمكن للعملاء العائدين استخدامه حتى لو وجدوا الكود.

## 2. كود خصم الشحن المجاني

شجع على الطلبات الأكبر من خلال تقديم شحن مجاني بمجرد أن تصل السلة إلى قيمة معينة. ضع **قيمة الخصم** إلى مبلغ يغطي أعلى تكلفة شحن معياري لديك.

| الحقل | القيمة |
|-------|-------|
| **الكود** | `FREESHIP50` |
| **الاسم** | شحن مجاني للطلبات فوق 50 دولار |
| **نوع الخصم** | خصم مبلغ ثابت |
| **قيمة الخصم** | 15.00 (أو تكلفة الشحن القصوى الخاصة بك) |
| **مجال التطبيق** | السلة بأكملها |
| **قيمة الطلب الدنيا** | 50.00 |
| **عدد الاستخدامات القصوى لكل عميل** | 0 (غير محدود) |
| **لا يمكن الجمع مع كوبونات أخرى** | مُشطب |

عرض هذا الكود بشكل بارز على صفحات المنتجات وفي السلة. غالبًا ما يضيف العملاء الذين يقتربون من الحد قيمة منتج إضافي للحصول على الخصم، مما يزيد من قيمة الطلب المتوسطة.

## 3. كود حملة موسمية

قم بتشغيل حملات محدودة الوقت مرتبطة بالمناسبات، المواسم أو الأحداث. ضع نطاقًا صارمًا للتواريخ بحيث يبدأ الكوبون وينتهي تلقائيًا.

| الحقل | القيمة |
|-------|-------|
| **الكود** | `SUMMER25` |
| **الاسم** | مبيعات الصيف 2025 |
| **نوع الخصم** | خصم نسبة |
| **قيمة الخصم** | 25 |
| **قيمة الخصم القصوى** | 50.00 |
| **مجال التطبيق** | السلة بأكملها |
| **تاريخ البدء** | 1 يونيو |
| **تاريخ الانتهاء** | 31 أغسطس |
| **لا يمكن الجمع مع كوبونات أخرى** | مُشطب |
| **لا يمكن الجمع مع منتجات المبيعات** | مُشطب |

**الكود المشترك مقابل الكود الفردي** - كود مشترك مثل `SUMMER25` سهل الترويج ولكن يمكن أن ينتشر خارج الجمهور المستهدف. للحصول على سيطرة أكثر دقة، استخدم إنشاء دفعة لخلق كودات فردية للاستخدام مرة واحدة (كلها مع **عدد الاستخدامات القصوى الإجمالي** محدد على 1). قم بتوزيع الكودات الفردية عبر رسائل بريد إلكتروني مستهدفة بحيث يحصل كل عميل على كوده الخاص.

## 4. كوبون مكافأة الولاء

كافئ عملاءك الأفضل بكوبون بقيمة ثابتة يمكنهم استخدامه في طلبهم التالي. أرسله عبر البريد الإلكتروني بعد أن يصل العميل إلى معيار إنفاق معين أو كشكرًا للاشتراكات المتكررة.

| الحقل | القيمة |
|-------|-------|
| **الكود** | `VIP20` |
| **الاسم** | مكافأة ولاء VIP |
| **نوع الخصم** | خصم مبلغ ثابت |
| **قيمة الخصم** | 20.00 |
| **مجال التطبيق** | السلة بأكملها |
| **قيمة الطلب الدنيا** | 100.00 |
| **عدد الاستخدامات القصوى لكل عميل** | 1 |
| **عدد الاستخدامات القصوى الإجمالي** | 0 (غير محدود) |
| **عدد الأيام الصالحة** | 90 |

تحديد **عدد الأيام الصالحة** إلى 90 يمنح العملاء ثلاث أشهر لاستبدال المكافأة من تاريخ الاستخدام الأول. تأكد **قيمة الطلب الدنيا** من 100 دولار أن الخصم ينطبق على الطلبات ذات المعنى.

## 5. كود المؤثر / كود الإحالة

قدم لمؤثرين أو سفراء العلامة التجارية كودًا شخصيًا يشاركونه مع جمهورهم. يتم ربط كل كود بشخص واحد، مما يجعل من السهل تتبع الشريك الذي يحقق مبيعات أكثر.


| الحقل | القيمة |
|-------|-------|
| **الكود** | `SARAH10` |
| **الاسم** | مؤثر — سارة |
| **نوع الخصم** | خصم نسبة |
| **قيمة الخصم** | 10 |
| **مجال التطبيق** | السلة بأكملها |
| **عدد الاستخدامات القصوى (الإجمالي)** | 0 (غير محدود) |
| **عدد الاستخدامات القصوى لكل عميل** | 1 |

أنشئ كودًا منفصلًا لكل مؤثر حتى تتمكن من مقارنة عدد الاستخدامات على لوحة متابعة الكود. تحقق من استخدام كل كود بانتظام لقياس العائد على الاستثمار وتحديد الشراكات التي يجب الاستمرار فيها.

## 6. كود خصم مفاجأة

أنشئ شعورًا بالعجلة مع خصم عميق متاح لفترة قصيرة جدًا. اربط الكود بشريط إعلان أو بريد إلكتروني لجذب الزوار فورًا.

| الحقل | القيمة |
|-------|-------|
| **الكود** | `FLASH40` |
| **الاسم** | مبيعات مفاجأة لساعة 24 |
| **نوع الخصم** | خصم نسبة |
| **قيمة الخصم** | 40 |
| **قيمة الخصم القصوى** | 75.00 |
| **مجال التطبيق** | السلة بأكملها |
| **تاريخ البدء** | يوم الجمعة الساعة 9:00 صباحًا |
| **تاريخ الانتهاء** | يوم السبت الساعة 9:00 صباحًا |
| **عدد الاستخدامات القصوى (الإجمالي)** | 200 |
| **استبعاد المنتجات المخفضة** | م işaret |

تحديد **عدد الاستخدامات القصوى (الإجمالي)** عند 200 يحمي هوامش الربح. بمجرد الوصول إلى الحد الأقصى، يتوقف الكود عن العمل تلقائيًا. اذكر الكمية المحدودة في حملتك التسويقية — "فقط للعملاء الـ 200 الأولى" — لتعزيز الشعور بالعجلة.

## ملخص الحملة

| الحملة | مثال الكود | النوع | القيمة | الإعدادات المهمة |
|----------|-------------|------|-------|--------------|
| خصم ترحيب | `WELCOME15` | نسبة | 15% | فقط للعملاء الجدد، عدد الاستخدامات لكل عميل = 1 |
| شحن مجاني | `FREESHIP50` | مبلغ ثابت | 15 دولارًا | الحد الأدنى للطلب 50 دولارًا، لا يمكن الجمع |
| مبيعات موسمية | `SUMMER25` | نسبة | 25% | فترة زمنية، قيمة الخصم القصوى 50 دولارًا |
| مكافأة الولاء | `VIP20` | مبلغ ثابت | 20 دولارًا | الحد الأدنى للطلب 100 دولار، صلاحية 90 يومًا |
| كود المؤثر | `SARAH10` | نسبة | 10% | عدد الاستخدامات لكل عميل = 1، تتبع عبر الكود |
| خصم مفاجأة | `FLASH40` | نسبة | 40% | نافذة 24 ساعة، 200 استخدام أقصى |

## النصائح

- **احتفظ بالكود قصيرًا وسهل التذكر** — كل الحروف كبيرة، لا توجد رموز خاصة. يقوم العملاء بكتابة هذه الأكواد عند الدفع، لذا فإن `SUMMER25` أفضل من `summer-sale-2025-promo`.
- **تتبع عدد الاستخدامات** — تحقق من لوحة متابعة الكود بانتظام لمعرفة الحملات التي تحقق أفضل أداء. قارن عدد الاستخدامات الإجمالي مع الإيرادات لحساب العائد على الاستثمار.
- **اجرِ اختبارًا قبل الإطلاق** — اجعل طلبًا تجريبيًا مع الكود للتأكد من أنه ينطبق بشكل صحيح ويحترم قيمة الطلب الأدنى ويتم الجمع (أو عدم الجمع) كما هو متوقع.
- **استخدم كودًا جديدًا للحملات المتكررة** — استخدم كودًا جديدًا لكل موسم (مثلاً `SUMMER25`، `SUMMER26`) حتى تتمكن من قياس كل حملة بشكل منفصل ومنع انتشار الأكواد القديمة.
- **حدد قيم خصم أقصى للفواتير النسبية** — خصم بنسبة 25% على طلب قيمته 1000 دولار هو 250 دولارًا. استخدم حقل **قيمة الخصم القصوى** لتحديد القيمة بالدولار وحماية هوامش الربح.
- **استخدم شريط الإعلان** — اربط أكواد المفاجآت والموسمية بشريط الإعلان في متجرك الإلكتروني حتى يرى كل زائر الحملة فورًا.
