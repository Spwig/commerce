title: 优惠券示例

本指南为最常见的优惠券类型提供具体的字段级示例。每个示例都展示了在 **营销 > 优惠券** → **+ 添加优惠券** 中创建优惠券时应填写的内容。

![优惠券卡片](/static/core/admin/img/help/voucher-examples/voucher-card.webp)

## 示例 1：折扣上限的百分比折扣

**场景：** 提供整个购物车 20% 的折扣，但将折扣上限设为 50 美元，以保持高价值订单的盈利能力。没有过期日期。

| 字段 | 值 |
|-------|-------|
| 代码 | `SAVE20` |
| 名称 | 20% 折扣 — 最高 50 美元 |
| 折扣类型 | 百分比 |
| 折扣值 | 20 |
| 最大折扣金额 | 50 |
| 应用范围 | 整个购物车 |
| 总使用次数上限 | *(空 — 无限制)* |
| 每位客户使用次数上限 | 1 |
| 最低订单金额 | *(空 — 无最低限制)* |

**折扣上限的运作方式：** 在 200 美元的订单中，折扣为 40 美元。在 300 美元的订单中，折扣为 60 美元，但上限限制为 50 美元。在 500 美元的订单中，折扣仍为 50 美元。这使您能够推出听起来慷慨的促销活动，同时保持实际折扣的可预测性。

## 示例 2：有最低金额的固定金额折扣

**场景：** 为任何超过 75 美元的订单提供 10 美元折扣，以鼓励客户购买更多商品。

| 字段 | 值 |
|-------|-------|
| 代码 | `TAKE10` |
| 名称 | 75 美元以上订单减 10 美元 |
| 折扣类型 | 固定金额 |
| 折扣值 | 10 |
| 应用范围 | 整个购物车 |
| 最低订单金额 | 75 |
| 每位客户使用次数上限 | 0 *(无限制)* |
| 结束日期 | *(空 — 无过期)* |

> **注意：** 设置最低订单金额可以保护您的利润。如果没有设置，客户可以将此代码用于 12 美元的订单，从而消除您的利润。始终将固定金额优惠券与合理的最低金额搭配使用。

## 示例 3：免费送货

**场景：** 为任何订单提供免费送货，无需最低消费。

| 字段 | 值 |
|-------|-------|
| 代码 | `FREESHIP` |
| 名称 | 免费送货 |
| 折扣类型 | 免费送货 |
| 应用范围 | 整个购物车 |
| 总使用次数上限 | *(空 — 无限制)* |
| 每位客户使用次数上限 | 1 |
| 最低订单金额 | *(空 — 无最低限制)* |

> **注意：** 选择 **免费送货** 折扣类型，该类型会自动从订单中移除运费。这是最干净的方法，无论客户选择哪种送货方式都适用。

## 示例 4：首次客户欢迎代码

**场景：** 为新客户提供首次订单 15% 的折扣，以促进转化。

| 字段 | 值 |
|-------|-------|
| 代码 | `WELCOME15` |
| 名称 | 欢迎 — 首次订单 15% 折扣 |
| 折扣类型 | 百分比 |
| 折扣值 | 15 |
| 应用范围 | 整个购物车 |
| 每位客户使用次数上限 | 1 |
| 仅限首次客户 | 勾选 |

系统通过检查客户是否有任何已完成的订单来验证首次客户状态。如果尝试使用此代码的客户有订单历史记录，他们会在结账时看到明确的错误信息。

## 示例 5：特定产品的优惠券

**场景：** 为选定的产品提供 5 美元折扣 — 例如，用于清理滞销库存。

| 字段 | 值 |
|-------|-------|
| 代码 | `PICK5` |
| 名称 | 选定商品减 5 美元 |
| 折扣类型 | 固定金额 |
| 折扣值 | 5 |
| 应用范围 | 特定产品 |
| 合格产品 | *(选择目标产品)* |
| 每位客户使用次数上限 | 1 |

> **注意：** 当您想要对单个 SKU 进行折扣时，使用产品范围。当您想要对部门中的所有商品进行折扣时，使用分类范围（下一个示例）。产品范围为您提供精确的控制；当您的商品目录经常变化时，分类范围更容易维护。

## 示例 6：分类优惠券

**场景：** 在电子产品分类中的所有商品上运行 25% 的折扣促销。

| 字段 | 值 |
|-------|-------|
| 代码 | `ELEC25` |
| 名称 | 电子产品 25% 折扣 |
| 折扣类型 | 百分比 |
| 折扣值 | 25 |
| 应用范围 | 特定分类 |
| 合格分类 | 电子产品 |
| 总使用次数上限 | *(空 — 无限制)* |
| 每位客户使用次数上限 | 1 |


当限定在某个类别时，折扣仅适用于购物车中的符合条件的商品。

非电子产品按原价收费。

## 折扣类型对比

| 类型 | 工作方式 | 最适合 | 示例 |
|------|-------------|----------|---------|
| **百分比** | 扣除符合条件总额的百分比 | 随订单规模增长的折扣 | 整个购物车打20%折扣 |
| **固定金额** | 扣除固定金额 | 简单且可预测的促销 | 订单满75美元减10美元 |
| **免运费** | 从订单中移除运费 | 减少结账时的购物车弃购 | 免运费，无最低消费限制 |

## 作用范围对比

| 作用范围 | 工作方式 | 最适合 |
|-------|-------------|----------|
| **整个购物车** | 折扣适用于整个订单总额 | 全店促销和欢迎码 |
| **特定产品** | 折扣仅适用于购物车中选定的产品 | 清理特定库存或重点促销 |
| **特定类别** | 折扣仅适用于选定类别的商品 | 部门级销售和季节性促销 |

## 小贴士

- **使用易记的代码** — `SUMMER20` 比 `COUPONX1600406498` 更易于记忆。将自动生成的代码保留用于批量活动。
- **分发前进行测试** — 使用优惠码下测试订单，以验证其是否正确应用并遵守所有限制。
- **监控使用情况** — 检查每个优惠码卡片上的使用次数，以实时跟踪活动效果。
- **与公告栏结合使用** — 在网站范围的公告中推广您的优惠码，让客户在开始购物前就能看到。
