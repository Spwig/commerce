---
title: 優惠券活動靈感
---

本指南涵蓋了已驗證的優惠券活動，您可以在幾分鐘內設置。每個活動都包含精確的設置選項，讓您能快速從想法轉化為實際推廣。在 **行銷 > 優惠券** 中創建所有優惠券，並點擊 **+ 新增優惠券**。

![歡迎優惠券](/static/core/admin/img/help/voucher-campaign-ideas/welcome-voucher.webp)

## 1. 新客戶歡迎折扣

為首次購物的客戶提供一個激勵，促使他們完成購買。歡迎折扣有助於建立信任，並減少從未在您商店下過訂單的客戶的猶豫。

| 欄位 | 值 |
|-------|-------|
| **代碼** | `WELCOME15` |
| **名稱** | 歡迎折扣 — 新客戶 |
| **折扣類型** | 百分比折扣 |
| **折扣值** | 15 |
| **應用範圍** | 整個購物車 |
| **每位客戶最大使用次數** | 1 |
| **總共最大使用次數** | 0 (無限) |
| **僅限新客戶** | 已勾選 |
| **開始日期 / 結束日期** | 均留空 — 始終有效 |

在註冊確認郵件、首頁橫幅或首次訪客的彈出視窗中推廣此代碼。因為 **僅限新客戶** 已勾選，因此即使回購客戶找到此代碼，他們也無法兌換。

## 2. 免運費門檻

透過提供購物車達到最低金額後的免運費，鼓勵客戶下較大的訂單。將 **折扣值** 設為足以涵蓋您最高標準運費的金額。

| 欄位 | 值 |
|-------|-------|
| **代碼** | `FREESHIP50` |
| **名稱** | 訂購滿 $50 免運費 |
| **折扣類型** | 固定金額折扣 |
| **折扣值** | 15.00 (或您的最高運費) |
| **應用範圍** | 整個購物車 |
| **最低訂單金額** | 50.00 |
| **每位客戶最大使用次數** | 0 (無限) |
| **不可與其他優惠券組合使用** | 已勾選 |

在您的產品頁面和購物車中醒目地顯示此代碼。接近門檻的客戶通常會再添加一件商品以達到資格，這將提高您的平均訂單價值。

## 3. 季節性活動代碼

針對節日、季節或活動運行有限時間的促銷活動。設定嚴格的日期範圍，讓優惠券自動啟用和過期。

| 欄位 | 值 |
|-------|-------|
| **代碼** | `SUMMER25` |
| **名稱** | 2025 夏季促銷 |
| **折扣類型** | 百分比折扣 |
| **折扣值** | 25 |
| **最大折扣金額** | 50.00 |
| **應用範圍** | 整個購物車 |
| **開始日期** | 6 月 1 日 |
| **結束日期** | 8 月 31 日 |
| **不可與其他優惠券組合使用** | 已勾選 |
| **不可與促銷商品組合使用** | 已勾選 |

**共享代碼與獨特代碼** — 類似 `SUMMER25` 的共享代碼容易推廣，但可能會超出您預期的目標受眾範圍。如需更精準的控制，請使用批量生成功能來創建獨特的一次性代碼（每個代碼的 **總共最大使用次數** 設為 1）。透過定向郵件分發獨特代碼，讓每位客戶都能獲得自己的代碼。

## 4. 忠誠度獎勵優惠券

以固定金額的優惠券獎勵您的最佳客戶，他們可以在下一次訂購時使用。當客戶達到消費里程碑時，或作為對重複購買的感謝，透過電子郵件發送。

| 欄位 | 值 |
|-------|-------|
| **代碼** | `VIP20` |
| **名稱** | VIP 忠誠度獎勵 |
| **折扣類型** | 固定金額折扣 |
| **折扣值** | 20.00 |
| **應用範圍** | 整個購物車 |
| **最低訂單金額** | 100.00 |
| **每位客戶最大使用次數** | 1 |
| **總共最大使用次數** | 0 (無限) |
| **有效天數** | 90 |

將 **有效天數** 設為 90，讓客戶在首次使用後有三個月的時間兌換獎勵。**最低訂單金額** 為 $100，確保折扣應用於有意義的訂單。

## 5. 影響者 / 推薦代碼

為影響者或品牌大使提供個人代碼，讓他們與其受眾分享。每個代碼都與一個人綁定，這使得追蹤哪位合作夥伴帶來最多銷售變得輕鬆。


| 欄位 | 值 |
|-------|-------|
| **代碼** | `SARAH10` |
| **名稱** | 影響者 — Sarah |
| **折扣類型** | 百分比折扣 |
| **折扣值** | 10 |
| **應用範圍** | 整個購物車 |
| **總使用次數上限** | 0 (無限) |
| **每位客戶的使用次數上限** | 1 |

為每位影響者創建獨立的優惠券，以便在優惠券儀表板上比較使用次數。定期查看每個代碼的使用情況，以衡量投資回報率並決定是否繼續合作。

## 6. 閃電折扣代碼

透過極短時間內提供的深度折扣來營造緊迫感。將代碼與橫幅公告或電子郵件推廣結合，以立即吸引流量。

| 欄位 | 值 |
|-------|-------|
| **代碼** | `FLASH40` |
| **名稱** | 24小時閃電促銷 |
| **折扣類型** | 百分比折扣 |
| **折扣值** | 40 |
| **折扣上限** | 75.00 |
| **應用範圍** | 整個購物車 |
| **開始日期** | 星期五上午9:00 |
| **結束日期** | 星期六上午9:00 |
| **總使用次數上限** | 200 |
| **排除促銷商品** | 已勾選 |

將 **總使用次數上限** 設為 200 可保護您的利潤空間。一旦達到上限，代碼會自動停止使用。在您的行銷中提及有限數量 —「僅限前200位客戶」— 以強化緊迫感。

## 小提示

- **保持代碼簡短易記** — 全大寫，不使用特殊字符。顧客在結帳時需要輸入這些代碼，因此 `SUMMER25` 比 `summer-sale-2025-promo` 更好。
- **追蹤使用次數** — 定期查看優惠券儀表板，了解哪些活動表現最佳。將總使用次數與收入進行比較，以計算投資回報率。
- **發佈前進行測試** — 使用代碼下一個測試訂單，確認折扣正確套用、符合最低訂單金額，並按照預期合併（或不合併）。
- **重複活動使用新代碼** — 每個季節使用新代碼（例如 `SUMMER25`、`SUMMER26`），以便獨立衡量每個活動的表現，並防止舊代碼流傳。
- **在百分比優惠券上設定折扣上限** — $1,000 訂單的 25% 折扣是 $250。使用 **折扣上限** 欄位來限制美元價值，保護您的利潤空間。
- **使用公告橫幅** — 將閃電和季節性代碼與商店公告橫幅結合，讓每位訪客立即看到促銷資訊。
