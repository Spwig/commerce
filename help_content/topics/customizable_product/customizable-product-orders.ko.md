title: 사용자 정의 제품 주문 처리
---

고객이 제품을 설계하고 주문을 할 때, 그 설계는 주문과 함께 고정되어 저장됩니다. 이 가이드는 사용자 정의 디자인이 주문 주기에서 어떻게 흐르는지, 그리고 충족을 위해 필요한 인쇄 가능한 파일에 어떻게 접근하는지 설명합니다.

## 디자인 주기

고객의 디자인은 생성에서 충족까지 여러 단계를 거칩니다:

### 1. 디자인 생성

고객은 쇼핑몰의 시각 편집기에서 자신의 디자인을 만듭니다. 작업 중에는 브라우저에 자동으로 진행 상황이 저장됩니다. 등록된 고객은 디자인을 계정에 저장하여 나중에 편집할 수도 있습니다.

### 2. 디자인 초안

고객이 **장바구니에 추가**를 클릭하면 현재 디자인 상태가 **디자인 초안**으로 저장됩니다. 초안에는 다음과 같은 내용이 포함됩니다:

- 모든 표면(요소 위치, 텍스트 내용, 업로드된 이미지, 클립아트, 스타일링)에 대한 완전한 캔버스 상태
- 적용 가능한 디자인 요금의 가격 분석
- 각 표면의 미리보기 썸네일

초안은 고유한 토큰을 통해 장바구니 항목과 연결됩니다. 이는 고객이 체크아웃 전에 쇼핑을 계속하더라도 고객이 만든 정확한 디자인이 보존되도록 보장합니다.

**초안 만료:** 고객이 주문을 완료하지 않은 경우 디자인 초안은 7일 후 자동으로 만료됩니다. 이는 포기된 디자인의 축적이 방지됩니다.

### 3. 디자인 스냅샷

고객이 결제를 완료하고 주문을 생성하면 디자인 초안은 **불변의 디자인 스냅샷**으로 변환됩니다. 이는 디자인의 영구 기록입니다:

- 구매 후 고객이 디자인 스냅샷을 수정할 수 없습니다
- 초안과 동일한 디자인 데이터를 포함합니다
- 특정 주문 항목에 영구적으로 연결됩니다

이 불변성은 매우 중요합니다 — 고객이 주문한 것이 결제 후 변경 없이 정확히 제작 및 배송되도록 보장합니다.

### 4. 충족 파일 렌더링

주문이 생성된 후 시스템은 디자인의 각 표면에 대해 **고해상도 충족 파일**을 자동으로 생성합니다.

# 디자인 스냅샷 및 주문 데이터 액세스

이러한 복합 이미지는 각 표면에 대해 설정된 DPI에 따라 텍스트, 이미지, 클립아트 등 모든 디자인 요소를 하나의 인쇄 준비 파일로 결합합니다.

렌더링은 백그라운드에서 비동기적으로 수행됩니다. 대부분의 디자인은 몇 초 안에 렌더링이 완료됩니다. 스냅샷의 **렌더링 완료** 상태는 인쇄 파일이 준비되었는지 여부를 나타냅니다.

## 주문에서 디자인 데이터 액세스

### 주문 상세 페이지

관리자 패널에서 맞춤형 제품이 포함된 주문을 보고 있을 때:

1. **주문 > 모든 주문**으로 이동합니다.
2. 맞춤형 제품이 포함된 주문을 엽니다.
3. 맞춤형 제품의 주문 항목은 디자인 정보를 표시합니다. 이는 표면 미리보기 및 디자인 스냅샷으로 이동하는 링크를 포함합니다.

### 디자인 스냅샷 목록

또한 모든 디자인 스냅샷을 직접 브라우징할 수도 있습니다:

1. **맞춤형 제품 > 디자인 스냅샷**으로 이동합니다.
2. 목록은 주문 항목에 연결된 모든 스냅샷을 표시합니다.
3. 스냅샷을 클릭하여 전체 디자인 데이터, 렌더링된 이미지 및 인쇄 파일을 확인합니다.

각 스냅샷은 다음과 같은 정보를 표시합니다:

| 필드 | 설명 |
|-------|-------------|
| **주문 항목** | 관련 주문 항목으로의 링크 |
| **디자인 데이터** | 완전한 캔버스 상태 (JSON) |
| **렌더링된 이미지** | 표면별 미리보기 썸네일 |
| **인쇄 파일** | 인쇄용 고해상도 복합 파일 |
| **렌더링 완료** | 렌더링이 완료되었는지 여부 |
| **렌더링 완료 시간** | 파일이 생성된 시간戳 |

## 인쇄 파일 다운로드

인쇄 파일은 인쇄 제공업체에게 전송하거나 생산 과정에서 사용하는 파일입니다.

**맞춤형 티셔츠 주문의 경우:**
- **앞면** 표면 파일 다운로드 (예: 300 DPI 복합 PNG)
- **뒷면** 표면 파일 다운로드
- **팔 부분** 표면 파일 다운로드 (디자인된 경우)
- 모든 파일을 스크린 인쇄기 또는 DTG (직접 인쇄) 인쇄기로 전송합니다.

**커스텀 포스터 주문의 경우:**
- 인쇄 해상도에서 단일 **프론트** 표면 파일을 다운로드하세요
- 표면에 블리드가 설정된 경우 파일에는 블리드 영역이 포함되어 있습니다
- 포스터/카드 인쇄업체에 전송하세요

각 파일은 해당 표면에 설정한 DPI로 렌더링된 모든 디자인 요소가 합쳐진 단일 복합 이미지입니다.

## 저장된 디자인

등록된 고객은 나중에 편집할 수 있도록 디자인을 계정에 저장할 수 있습니다. 판매자로서는 이러한 저장된 디자인을 읽기 전용 목록으로 볼 수 있습니다:

1. **커스터마이즈 가능한 상품 > 저장된 디자인**으로 이동하세요
2. 목록에는 고객이 저장한 모든 디자인과 고객 이름, 상품, 디자인 이름, 날짜가 표시됩니다

저장된 디자인은 다음과 같습니다:
- **고객 소유** — 고객 계정에 속합니다
- **판매자에게는 읽기 전용** — 확인만 가능하며 수정할 수 없습니다
- **주문과 별도** — 저장된 디자인은 고객이 장바구니에 추가하고 결제할 때까지 주문이 되지 않습니다
- **재사용 가능** — 고객은 저장된 디자인을 불러와 수정하고 여러 번 주문할 수 있습니다

## 출고 작업 흐름

### 표준 작업 흐름

1. **주문 수령** — 커스터마이즈된 항목이 포함된 주문이 주문 목록에 나타납니다
2. **렌더링 확인** — 디자인 스냅샷이 **렌더링됨: 예**를 표시하는지 확인하세요. 렌더링이 완료되지 않은 경우 몇 분 기다리고 새로 고침하세요
3. **파일 다운로드** — 각 디자인된 표면의 출고 파일을 다운로드하세요
4. **품질 검토** — 파일을 열고 디자인이 인쇄 품질 기준을 충족하는지 확인하세요 (DPI, 요소 위치, 텍스트 가독성 확인)
5. **생산으로 전달** — 파일을 인쇄 제공업체 또는 생산 팀에 전달하세요
6. **배송 및 완료** — 생산이 완료된 후 제품을 배송하고 주문을 완료 상태로 표시하세요

### 티셔츠 출고 예시

1.

주문 수령: "커스텀 팀 티셔츠"로 프론트와 뒷면에 디자인 있음
2.

주문 열기 → 디자인 스냅샷 보기
3.

`front.png` (300 DPI, 300x400mm) 및 `back.png` (300 DPI, 300x400mm) 다운로드
4.


# 4. 인쇄 및 출고

주문의 변형 선택에서 얻은 옷의 색상과 사이즈와 함께 두 파일을 DTG 프린터에 전송하세요
5.

인쇄 및 품질 검사를 완료한 후 고객에게 발송하세요

### 포스터 출고 예시

1. 주문 수령: "커스텀 A4 포스터" 및 단일 디자인 표면
2. 주문 열기 → 디자인 미리보기 확인
3. `front.png` 다운로드 (300 DPI, 210x297mm, 3mm 여백 포함)
4. 포스터 인쇄 서비스에 전송
5. 인쇄 및 자르기 완료 후 고객에게 발송

## 문제 해결

**문제:** 디자인 미리보기에서 "렌더링됨: 아니오"가 표시되고 렌더링이 완료되지 않았습니다.

- **원인:** 배경 렌더링 작업이 실패했거나 아직 처리 중일 수 있습니다.
- **해결 방법:** 몇 분 기다려 보세요. 렌더링이 완료되지 않으면 배경 작업 로그를 확인하세요. 디자인 데이터를 직접 미리보기에서 확인하여 고객의 디자인이 보존되었는지 확인할 수도 있습니다.

**문제:** 출고 파일이 예상보다 품질이 낮습니다.

- **원인:** 고객이 저해상도 이미지를 업로드했을 수 있습니다.
- **해결 방법:** 표면의 DPI 설정을 확인하세요. 최소 DPI 경고가 설정되어 있다면 고객은 디자인 과정 중 경고를 받았을 것입니다. 향후 제품에서는 최소 DPI 요구 사항을 높이는 것을 고려하세요.

**문제:** 고객이 주문 후 디자인 변경을 요청했습니다.

- **해결 방법:** 디자인 미리보기는 디자인상으로 불변입니다. 고객이 변경이 필요하다면 업데이트된 디자인으로 새로운 주문을 내야 합니다. 예외를 허용하시면 고객은 저장한 디자인(저장했다면)을 새로운 주문의 시작점으로 사용할 수 있습니다.

## 팁

- 생산을 시작하기 전에 렌더링이 완료되었는지 항상 확인하세요.

디자인 미리보기의 **렌더링됨** 필드를 확인하세요.
- 인쇄 방법에 적합한 DPI 설정을 유지하세요.

높은 DPI는 품질이 더 좋지만 파일 크기가 커집니다. 300 DPI는 대부분의 전문 인쇄 제품에 표준입니다.
- 고객이 주문하기 전에 디자인을 저장하도록 권장하세요.


# 주문 재생산

생산 문제가 발생하고 주문을 다시 제작해야 하는 경우, 저장된 디자인은 재주문을 간편하게 만들어 줍니다.
- 생산 일정에 맞춤형 제품을 위한 버퍼를 구축하세요.

표준 제품과 달리 각 항목은 개별 파일 처리가 필요합니다.
- 맞춤형 주문의 고량을 처리하는 경우, 인쇄 제공업체의 API와 통합하여 파일 다운로드 단계를 자동화하는 것을 고려하세요.
