title: 사용자 정의 제품 설정
---

이 가이드는 사용자 정의 제품의 완전한 설정 과정을 안내합니다. 제품 생성부터 표면, 가격, 업로드 제한 설정까지 모든 단계를 포함합니다. 두 가지 실용적인 예시를 사용합니다: **사용자 정의 티셔츠**(다중 표면 의류)와 **사용자 정의 포스터**(단일 표면 인쇄).

## 단계 4: 표면별 제약 조건

각 표면은 전역 기능 설정을 재정의할 수 있습니다. 이는 다른 표면에 대해 다른 도구를 허용할 수 있도록 합니다.

제약 옵션은 다음과 같습니다:

| 설정 | 옵션 | 설명 |
|---------|---------|-------------|
| **텍스트 허용** | 상속 / 예 / 아니오 | 고객이 이 표면에 텍스트를 추가할 수 있는지 여부 |
| **이미지 업로드 허용** | 상속 / 예 / 아니오 | 고객이 이 표면에 이미지를 업로드할 수 있는지 여부 |
| **클립아트 허용** | 상속 / 예 / 아니오 | 고객이 이 표면에 클립아트를 사용할 수 있는지 여부 |
| **최대 요소 수** | 숫자 또는 빈칸 | 이 표면에 허용되는 최대 디자인 요소 수 |

**상속**으로 설정된 경우, 표면은 전역 설정(단계 6)에 구성된 내용을 사용합니다. **예** 또는 **아니오**로 설정된 경우, 해당 표면에 대한 전역 설정을 재정의합니다.

### 예시: 티셔츠 소매 제약 조건

티셔츠의 소매 표면에 대해, 작은 로고만 사용하도록 커스터마이징을 제한할 수 있습니다:

| 설정 | 값 | 이유 |
|---------|-------|--------|
| 텍스트 허용 | 아니오 | 너무 작아서 가독성이 낮음 |
| 이미지 업로드 허용 | 예 | 작은 로고 업로드 허용 |
| 클립아트 허용 | 아니오 | 간단하게 유지 |
| 최대 요소 수 | 1 | 로고 하나만 허용 |

앞면과 뒷면 표면은 **상속**으로 유지되어, 전역 설정에 정의된 모든 도구가 사용 가능합니다.

### 예시: 포스터 제약 조건

포스터의 경우 일반적으로 모든 표면이 전역 설정에서 상속되며, 표면이 하나뿐이고 모든 도구가 사용되어야 하기 때문에 표면별 재정의가 필요하지 않습니다.

## 단계 5: 업로드 제한 설정

**설정** 탭에서 고객이 파일을 업로드하는 방식을 설정합니다:

| 설정 | 설명 | 티셔츠 예시 | 포스터 예시 |
|---------|-------------|-----------------|----------------|
| **최대 업로드 크기** | 업로드당 최대 파일 크기 | 10 MB | 20 MB |
| **표면당 최대 업로드 수** | 표면당 이미지 수 | 5 | 3 |
| **허용 업로드 유형** | 허용된 파일 형식 | JPG, PNG, WebP | JPG, PNG, WebP |

인쇄 제품의 경우 고객이 고해상도 이미지를 업로드해야 하므로 더 큰 파일 크기 제한이 권장됩니다.

## 단계 6: 에디터 설정

**설정** 탭에서 전역 에디터 동작을 설정합니다:

**에디터 모드:**
- **캔버스 에디터** — 실시간 캔버스 미리보기가 있는 전체 시각적 에디터. 대부분의 제품에 권장됩니다.
- **간단한 폼** — 기본 커스터마이징을 위한 전통적인 폼 필드(예: 새글만 입력).

**기능 전환(전역 기본값):**
- **텍스트 허용** — 고객이 텍스트 요소를 추가할 수 있도록 합니다.
- **이미지 업로드 허용** — 고객이 자신의 이미지를 업로드할 수 있도록 합니다.
- **클립아트 허용** — 고객이 클립아트 라이브러리를 탐색하고 사용할 수 있도록 합니다.

이 전역 설정은 표면별 제약 조건(단계 4)에 의해 재정의되지 않는 한 모든 표면에 적용됩니다.

## 단계 7: 가격 설정

**가격** 탭에서 제품 기본 가격에 추가되는 디자인 요금을 설정합니다:

| 요금 | 설명 |
|-----|-------------|
| **기본 디자인 요금** | 어떤 커스터마이징이 적용될 때 추가되는 고정 요금 |
| **표면별 요금** | 첫 번째 표면을 제외한 각 표면 사용에 추가 요금 |
| **업로드별 요금** | 고객이 업로드한 이미지당 요금 |
| **텍스트별 요금** | 추가된 각 텍스트 요소당 요금 |

### 예시: 티셔츠 가격

| 요금 | 금액 | 근거 |
|-----|--------|-----------|
| 기본 디자인 요금 | $5.00 | 어떤 커스터마이징 주문의 설정 비용을 커버 |
| 표면별 요금 | $2.00 | 추가 표면은 인쇄 비용을 증가시킴 |
| 업로드별 요금 | $1.00 | 사용자 업로드 이미지는 처리가 필요함 |
| 텍스트별 요금 | $0.50 | 텍스트는 이미지보다 간단하게 제작 가능 |

**계산 예시:** 고객이 앞면에 텍스트와 뒷면에 로고가 있는 티셔츠를 디자인합니다:
- 기본 디자인 요금: $5.00
- 추가 표면(뒷면): $2.00
- 업로드된 로고: $1.00
- 텍스트 요소: $0.50
- **총 디자인 요금: $8.50** (제품 기본 가격에 추가)

### 예시: 포스터 가격


| 수수료 | 금액 | 근거 |
|-----|--------|-----------|
| 기본 디자인 수수료 | $0.00 | 기본 수수료 없음 — 제품 가격이 포함함 |
| 표면당 수수료 | $0.00 | 단일 표면, 적용되지 않음 |
| 업로드당 수수료 | $2.00 | 고해상도 처리 |
| 텍스트당 수수료 | $0.00 | 텍스트는 기본 경험에 포함됨 |

**계산 예시:** 고객이 2개의 업로드된 사진과 3개의 텍스트 요소를 포함한 포스터를 만듭니다:
- 기본 디자인 수수료: $0.00
- 2개의 업로드된 사진: $4.00
- 3개의 텍스트 요소: $0.00
- **총 디자인 수수료: $4.00**

디자인 수수료은 고객이 요소를 추가할 때 실시간으로 표시되므로, 장바구니에 추가하기 전에 각 추가의 비용 영향을 확인할 수 있습니다.

## 설정 비교 요약

| 항목 | 커스텀 티셔츠 | 커스텀 포스터 |
|--------|---------------|---------------|
| 표면 | 3개 (앞면, 뒷면, 소매) | 1개 (앞면) |
| 미리보기 이미지 | 3개의 제품 사진 | 1개의 제품 사진 |
| 영역 위치 | 가슴/등/팔 영역 | 전체 인쇄 영역 |
| 크기 | 300x400mm, 100x100mm | 210x297mm (A4) |
| 최소 DPI | 150 | 200 |
| 여백 | 0 mm | 3 mm |
| 최대 색상 | 6 | 무제한 |
| 표면당 제약 | 소매 제한 | 필요 없음 |
| 가격 모델 | 기본 + 표면 + 업로드 + 텍스트 | 업로드 수수료만 |

## 팁

- 설정을 완료한 후 항상 고객의 관점에서 디자인 편집기를 테스트하세요. 스토어프론트의 제품 페이지에 방문하여 텍스트를 추가하고, 이미지를 업로드하고, 표면을 전환해 보세요.
- 실제 제품의 외형과 가까운 미리보기 이미지를 업로드하세요. 티셔츠의 경우 각각의 각도를 별도로 촬영하세요. 포스터의 경우 깨끗한 평평한 사진이나 프레임 미리보기를 사용하세요.
- 디자인 영역을 보수적으로 설정하세요 — 약간 작게 정의하는 것이 좋으며, 디자인을 테두리나 가장자리에 인쇄하는 것을 피하세요.
- 인쇄 방법에 따라 최소 DPI를 설정하세요: 스크린 인쇄는 150, 표준 디지털 인쇄는 200, 고급 오프셋 인쇄는 300.
- 인쇄 후 자르기 작업이 필요한 제품(포스터, 명함, 플라이어)에는 3mm의 여백을 사용하세요. 디자인을 기존 표면에 적용하는 제품(티셔츠, 머그잔, 스마트폰 케이스)에는 여백을 0으로 설정하세요.
- 간단한 가격부터 시작하고 고객 피드백에 따라 조정하세요. 많은 판매자는 기본 디자인 수수료만으로 시작하고 나중에 요소당 수수료를 추가합니다.
