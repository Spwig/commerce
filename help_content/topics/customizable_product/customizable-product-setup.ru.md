title: Настройка настраиваемого продукта
---

Этот руководство приводит вас через полный процесс настройки настраиваемого продукта, от создания продукта до настройки поверхностей, цен и ограничений загрузки. Всё время используются два практические примера: **настраиваемая футболка** (одежда с несколькими поверхностями) и **настраиваемая постер** (печать на одной поверхности).\n\n## Шаг 1: Создание продукта\n\n1. Перейдите в **Products > All Products** и нажмите **+ Add Product**\n2. Установите **Product Type** в **Customizable Product**\n3. Заполните название продукта, описание, изображения и цену, как вы бы это сделали для любого продукта\n4. Сохраните продукт\n\nПосле сохранения появляется новая кнопка **Open Design Editor Setup** на форме продукта. Это приводит вас на посвященную страницу настройки, где вы настраиваете визуальный редактор дизайна.\n\n## Шаг 2: Доступ к настройке редактора дизайна\n\n1. Откройте только что созданный продукт в админ-панели\n2. Нажмите кнопку **Open Design Editor Setup** (в разделе Customizable Product)\n3. Страница настройки открывается с тремя вкладками: **Surfaces**, **Settings**, и **Pricing**\n\nСтраница настройки — это место, где вы определяете всё о редакторе дизайна для этого продукта.\n\n## Шаг 3: Добавление поверхностей дизайна\n\nПоверхность представляет собой одну настраиваемую сторону вашего продукта. Нажмите **+ Add Surface**, чтобы создать каждую поверхность.\n\n### Пример футболки: 3 поверхности\n\n| Surface | Name | Dimensions | Design zone | Notes |\n|---------|------|-----------|-------------|-------|\n| 1 | Front | 300 x 400 mm | Centered chest area | Main design area |\n| 2 | Back | 300 x 400 mm | Upper back area | Secondary design area |\n| 3 | Left Sleeve | 100 x 100 mm | Upper arm area | Small logo area only |\n\n### Пример постера: 1 поверхность\n\n| Surface | Name | Dimensions | Design zone | Notes |\n|---------|------|-----------|-------------|-------|\n| 1 | Front | 210 x 297 mm (A4) | Full printable area | Single surface, high DPI |\n\n### Настройка каждой поверхности\n\nДля каждой поверхности вы настраиваете следующее:\n\n**Базовая информация:**
- **Name** — То, что клиенты видят в вкладках поверхностей (например, "Front", "Back")
- **Slug** — URL-совместимый идентификатор, автоматически генерируемый из названия
- **Sort Order** — Управляет порядком отображения поверхностей (сначала числа с меньшими значениями)\n\n**Изображение макета:**
- Нажмите на область изображения макета, чтобы открыть библиотеку медиа и выбрать фотографию продукта, показывающую эту поверхность
- Используйте высококачественную фотографию вашего продукта с правильной точки зрения\n\n**Позиционирование зоны дизайна:**
- После выбора изображения макета появляется прямоугольная прозрачная область на предварительном просмотре
- **Перетащите** прозрачную область, чтобы разместить зону дизайна на макете
- **Измените размер** прозрачной области, перетаскивая её края, чтобы определить границы зоны дизайна
- Зона хранится в виде процентных координат, поэтому она масштабируется на любом размере экрана\n\nЗона дизайна сообщает редактору, где именно на изображении продукта будет отображаться дизайн клиента. Размещайте её внимательно, чтобы соответствовать реальной печатной зоне вашего продукта.\n\n**Физические размеры:**
- **Width** и **Height** — Реальные размеры зоны дизайна
- **Unit** — Миллиметры, дюймы или пиксели
- Эти размеры определяют соотношение сторон холста дизайна и используются для вычисления DPI печати\n\n**Настройки печати:**
- **Minimum DPI** — Минимально допустимое количество точек на дюйм. Клиенты видят предупреждение, если загруженные изображения ниже этого значения. По умолчанию: 150
- **Recommended DPI** — Идеальное разрешение для наилучшего качества печати. По умолчанию: 300
- **Bleed (mm)** — Дополнительный край за пределами зоны дизайна для печати с кровью. Установите в 0, если кровь не требуется (обычно для одежды), или 3 мм для профессиональных печатных продуктов
- **Max Colors** — Для печати на трафарете вы можете ограничить количество цветов. Оставьте пустым для неограниченного количества (цифровая печать)
- **Background Color** — Цвет фона по умолчанию для холста\n\n### Настройки печати футболки и постера\n\n| Setting | T-shirt | Poster |\n|---------|---------|--------|\n| Minimum DPI | 150 | 200 |\n| Recommended DPI | 300 | 300 |\n| Bleed | 0 mm | 3 mm |\n| Max Colors | 6 (screen printing) | Blank (unlimited) |\n| Background Color | Match garment color | `#ffffff` (white) |

## Шаг 4: Ограничения по поверхностям

Каждая поверхность может переопределять глобальные настройки функций. Это позволяет разрешать использование различных инструментов на разных поверхностях.

Доступные варианты ограничений:

| Настройка | Варианты | Описание |
|---------|---------|-------------|
| **Разрешить текст** | Наследовать / Да / Нет | Разрешает ли клиентам добавлять текст на эту поверхность |
| **Разрешить загрузку изображений** | Наследовать / Да / Нет | Разрешает ли клиентам загружать изображения на эту поверхность |
| **Разрешить клипарт** | Наследовать / Да / Нет | Разрешает ли клиентам использовать клипарт на этой поверхности |
| **Максимальное количество элементов** | Число или пусто | Максимальное количество элементов дизайна, разрешенных на этой поверхности |

Если установлено значение **Наследовать**, поверхность использует настройки, указанные в глобальных настройках (Шаг 6). Если установлено значение **Да** или **Нет**, это переопределяет глобальную настройку для этой конкретной поверхности.

### Пример: Ограничение для рукава футболки

Для поверхности рукава футболки вы можете ограничить настройку только маленьким логотипом:

| Настройка | Значение | Причина |
|---------|-------|--------|
| Разрешить текст | Нет | Слишком маленький для читаемого текста |
| Разрешить загрузку изображений | Да | Разрешить загрузку маленького логотипа |
| Разрешить клипарт | Нет | Сохранить простоту |
| Максимальное количество элементов | 1 | Только один логотип |

Передняя и задняя поверхности останутся установленными в **Наследовать**, что позволит использовать все инструменты, определенные в глобальных настройках.

### Пример: Ограничение для постера

Для постера обычно все поверхности наследуют настройки из глобальной конфигурации, поскольку существует только одна поверхность, и все инструменты должны быть доступны. Нет необходимости в переопределении настроек по поверхностям.

## Шаг 5: Настройка ограничений загрузки

На вкладке **Настройки** настройте, как клиенты могут загружать файлы:

| Настройка | Описание | Пример футболки | Пример постера |
|---------|-------------|-----------------|----------------|
| **Максимальный размер загрузки** | Максимальный размер файла на одну загрузку | 10 МБ | 20 МБ |
| **Максимальное количество загрузок на поверхность** | Количество изображений на одну поверхность | 5 | 3 |
| **Разрешенные типы загрузки** | Принимаемые форматы файлов | JPG, PNG, WebP | JPG, PNG, WebP |

Рекомендуется использовать более высокие лимиты размера файлов для печатных продуктов, где клиентам нужно загружать изображения высокого разрешения.

## Шаг 6: Настройки редактора

На вкладке **Настройки** настройте поведение глобального редактора:

**Режим редактора:**
- **Режим холста** — полный визуальный редактор с превью холста в режиме реального времени. Рекомендуется для большинства продуктов.
- **Простая форма** — традиционные поля формы для базовой настройки (например, только для гравировки текста).

**Переключатели функций (глобальные настройки по умолчанию):**
- **Разрешить текст** — позволяет клиентам добавлять элементы текста
- **Разрешить загрузку изображений** — позволяет клиентам загружать свои изображения
- **Разрешить клипарт** — позволяет клиентам просматривать и использовать вашу библиотеку клипартов

Эти глобальные настройки применяются ко всем поверхностям, если они не переопределены ограничениями по поверхностям (Шаг 4).

## Шаг 7: Настройка цен

На вкладке **Цены** установите дополнительные расходы на дизайн, которые добавляются к базовой цене продукта:

| Плата | Описание |
|-----|-------------|
| **Базовая плата за дизайн** | Плоская плата, добавляемая при применении любой настройки |
| **Дополнительная плата за поверхность** | Дополнительная плата за каждую поверхность, используемую сверх первой |
| **Дополнительная плата за загрузку** | Плата за каждую загруженную клиентом картинку |
| **Дополнительная плата за текст** | Плата за каждый добавленный элемент текста |

### Пример: Цены на футболку

| Плата | Сумма | Обоснование |
|-----|--------|-----------|
| Базовая плата за дизайн | $5.00 | Охватывает затраты на настройку любого заказа |
| Дополнительная плата за поверхность | $2.00 | Каждая дополнительная поверхность добавляет затраты на печать |
| Дополнительная плата за загрузку | $1.00 | Персонализированные изображения требуют обработки |
| Дополнительная плата за текст | $0.50 | Текст проще производить, чем изображения |

**Пример расчета:** Клиент создает дизайн футболки с текстом на передней части и логотипом на задней части:
- Базовая плата за дизайн: $5.00
- 1 дополнительная поверхность (задняя часть): $2.00
- 1 загруженный логотип: $1.00
- 1 элемент текста: $0.50
- **Итоговая плата за дизайн: $8.50** (добавляется к базовой цене продукта)

### Пример: Цены на постер


| Стоимость | Сумма | Обоснование |
|---------|------|-----------|
| Базовая стоимость дизайна | $0.00 | Нет базовой стоимости — цена товара покрывает её |
| Стоимость за поверхность | $0.00 | Одна поверхность, не применяется |
| Стоимость за загрузку | $2.00 | Обработка высокого разрешения |
| Стоимость за текст | $0.00 | Текст входит в базовый опыт |

**Пример расчёта:** Покупатель создаёт плакат с 2 загруженными фотографиями и 3 элементами текста:
- Базовая стоимость дизайна: $0.00
- 2 загруженные фотографии: $4.00
- 3 элемента текста: $0.00
- **Итоговая стоимость дизайна: $4.00**

Стоимость дизайна отображается покупателям в реальном времени, когда они добавляют элементы, поэтому они могут увидеть влияние стоимости каждого добавления до добавления в корзину.

## Сравнение настроек в кратце

| Аспект | Персонализированный футболка | Персонализированный плакат |
|--------|---------------|---------------|
| Поверхности | 3 (перед, зад, рукав) | 1 (перед) |
| Изображения-макеты | 3 фотографии товара | 1 фотография товара |
| Расположение зон | Зоны груди/спины/рукава | Полная печатная зона |
| Размеры | 300x400 мм, 100x100 мм | 210x297 мм (A4) |
| Минимальное DPI | 150 | 200 |
| Обрезка | 0 мм | 3 мм |
| Максимальное количество цветов | 6 | Неограниченно |
| Ограничения по поверхностям | Рукав ограничен | Нет ограничений |
| Модель ценообразования | Базовая + поверхность + загрузка + текст | Только стоимость загрузки |

## Советы

- Всегда тестируйте редактор дизайна с точки зрения покупателя после завершения настройки. Посетите страницу товара на сайте и попробуйте добавить текст, загрузить изображение и переключить поверхности.
- Загружайте изображения-макеты, максимально соответствующие внешнему виду реального товара. Для футболок фотографируйте каждый угол отдельно. Для плакатов используйте чистое плоское изображение или макет рамки.
- Определяйте зону дизайна с осторожностью — лучше задать немного меньшую зону, чем позволить дизайну печататься в швы или края.
- Установите минимальное DPI в зависимости от метода печати: 150 для шелкографии, 200 для стандартной цифровой печати, 300 для высококачественной офсетной печати.
- Используйте 3 мм обрезки для любого товара, который будет обрезаться после печати (плакаты, визитки, флаеры). Установите обрезку в 0 мм для товаров, где дизайн наносится на существующую поверхность (футболки, кружки, корпуса телефонов).
- Начните с простой системы ценообразования и корректируйте её на основе обратной связи от клиентов. Многие продавцы начинают с одной только базовой стоимости дизайна и позже добавляют дополнительные платежи за элементы.
