---
title: 可定制产品的剪贴画和字体
---

设计编辑器提供了两种创意素材，您可以提供给客户：**剪贴画**（客户可以直接添加到设计中的现成图形）和**自定义字体**（超越标准系统字体）。构建一个精心策划的素材库可以让编辑器更有用，并帮助客户更快地创建更好的设计。

## 剪贴画库

剪贴画为客户提供了预设图形库，他们只需单击即可将这些图形添加到自己的设计中。您无需要求客户自行查找并上传图标、边框或装饰图形等常见元素的图片，而是直接提供现成可用的素材。

### 创建剪贴画分类

剪贴画按分类组织，客户可以浏览这些分类。分类有助于客户快速找到他们需要的内容。

1. 导航到 **可定制产品 > 剪贴画分类**
2. 点击 **+ 添加剪贴画分类**
3. 填写以下内容：
   - **分类名称** —— 客户看到的名称（例如："运动"、"边框"、"节日"）
   - **别名** —— 从名称自动生成
   - **图标** —— 用于分类标签的 Font Awesome 图标类（例如：`fas fa-football-ball`）
   - **排序顺序** —— 控制分类在编辑器中的显示顺序
4. 点击 **保存**

**T恤店的示例分类：**

| 分类 | 图标 | 示例剪贴画 |
|------|-----|------------|
| 运动 | `fas fa-football-ball` | 球队标志、运动器材、运动符号 |
| 幽默 | `fas fa-laugh` | 搞笑表情、幽默语录、卡通角色 |
| 自然 | `fas fa-leaf` | 动物、花朵、风景 |
| 几何 | `fas fa-shapes` | 图案、抽象形状、部落设计 |

**印刷/海报店的示例分类：**

| 分类 | 图标 | 示例剪贴画 |
|------|-----|------------|
| 边框 | `fas fa-border-all` | 装饰边框、角落装饰 |
| 节日 | `fas fa-snowflake` | 节日图标、季节主题 |
| 图标 | `fas fa-icons` | 星星、爱心、箭头、对勾 |
| 背景 | `fas fa-image` | 纹理、渐变、图案 |

### 添加剪贴画素材

每个剪贴画素材是一个图像文件（PNG 或 SVG），客户可以将其放置在画布上。

1. 导航到 **可定制产品 > 剪贴画素材**
2. 点击 **+ 添加剪贴画素材**
3. 填写以下内容：
   - **名称** —— 描述性名称（例如："金色星星"、"橄榄球头盔"）
   - **分类** —— 从您的剪贴画分类中选择
   - **图像素材** —— 点击打开媒体库并选择或上传图像文件
   - **范围** —— 选择可用性（见下文）
   - **标签** —— 用于此剪贴画的可搜索关键词（例如：`["star", "gold", "decoration"]`）
   - **排序顺序** —— 控制在分类中的位置
4. 点击 **保存**

### 了解剪贴画范围

每个剪贴画素材都有一个范围，用于控制其可用位置：

| 范围 | 描述 | 使用场景 |
|------|------|----------|
| **适用于所有产品** | 在每个可定制产品的剪贴画浏览器中显示 | 星星、边框和常见图标等通用图形 |
| **仅限特定产品** | 仅在选定的一个产品中显示 | 品牌标志或产品主题艺术等特定产品图形 |

对于大多数素材，使用 **适用于所有产品**。将特定产品范围保留给仅在一个产品上下文中才有意义的素材——例如，团队商品产品的特定团队标志。

### 剪贴画文件指南

- **格式：** 使用 PNG 用于位图图形，SVG 用于矢量图形。SVG 文件可以无损缩放，非常适合客户可能需要大幅调整大小的剪贴画。
- **分辨率：** PNG 文件应至少为 500x500 像素以确保良好的打印质量。
- **背景：** 使用透明背景（带 alpha 通道的 PNG 或 SVG），以便剪贴画能自然地与设计融合。
- **文件大小：** 保持单个剪贴画文件小于 500KB，以确保编辑器中快速加载。

## 自定义字体

自定义字体可将设计编辑器中的字体选择器扩展到标准系统字体之外。

这使您可以提供与您的品牌或产品风格相匹配的精选字体。

### 添加自定义字体

1. 导航到 **可定制产品 > 自定义字体**
2. 点击 **+ 添加自定义字体**
3. 填写以下内容：
   - **字体名称** — 在字体选择器中显示的名称（例如，"Playfair Display"）
   - **字体族** — 内部使用的 CSS 字体族名称（例如，`PlayfairDisplay`）
   - **常规** — 点击通过媒体库上传常规字重字体文件（WOFF2 或 TTF）
   - **粗体** — 可选的粗体字重变体
   - **斜体** — 可选的斜体变体
   - **粗体斜体** — 可选的粗体斜体变体
4. 点击 **保存**

**常规** 字重是自定义字体的必填项。粗体、斜体和粗体斜体变体是可选的 —— 如果未提供，浏览器将尝试从常规字体合成这些样式，但结果可能不如专用字体文件那样精致。

### 系统字体与自定义字体

您还可以注册预装在大多数设备上的系统字体：

1. 添加新的自定义字体条目
2. 勾选 **系统字体**
3. 按照在 CSS 中显示的字体族名称准确输入（例如，`Georgia`，`Courier New`）
4. 系统字体不需要上传文件

系统字体加载速度非常快，因为它们已经存在于客户的设备上。自定义上传的字体需要先下载，这会在首次选择字体时增加一点延迟。

### 按产品类型推荐字体

**对于T恤和服装：**
- 最适合使用粗体、有影响力的字体：Impact, Anton, Bebas Neue, Oswald
- 块字母和无衬线字体在织物上最易读
- 避免使用可能在纹理表面打印效果不佳的细或精致字体

**对于海报和印刷品：**
- 正式设计使用优雅的衬线字体：Playfair Display, Merriweather, Lora
- 邀请函和卡片使用手写字体：Great Vibes, Dancing Script, Pacifico
- 现代设计使用干净的无衬线字体：Montserrat, Raleway, Open Sans

### 字体文件格式

| 格式 | 扩展名 | 推荐 |
|------|--------|------|
| WOFF2 | `.woff2` | 推荐 —— 文件体积最小，加载速度最快 |
| TrueType | `.ttf` | 良好的备用方案 —— 兼容性广泛 |

WOFF2 文件通常比 TTF 文件小 30-50%，因此在客户编辑器中加载速度更快。如果可用，请使用 WOFF2。

## 管理您的素材库

### 为客户整理

编辑器中素材的显示顺序由 **排序顺序** 字段控制，该字段适用于类别和单个素材。数字较小的会先显示。使用此功能：
- 将您最受欢迎的剪贴画类别放在前面
- 在每个类别中将最佳和最通用的剪贴画放在顶部
- 将使用频率最高的字体放在前面

### 保持库的新鲜度

- 在节假日（万圣节、圣诞节、情人节）之前添加季节性剪贴画，并在之后停用
- 使用 **启用** 复选框临时隐藏素材，而无需删除它们
- 监控客户最常使用的剪贴画和字体，并扩展这些类别

## 小贴士

- 从小开始 —— 在 3-4 个类别中提供 20-30 个高质量剪贴画素材，比提供数百个质量一般的选项更好。您可以随时根据客户的需求添加更多素材。
- 尽可能使用 SVG 格式的剪贴画。SVG 文件体积更小，可以完美缩放到任何尺寸，并且比光栅图像产生更清晰的打印效果。
- 在设计编辑器中测试每个上传的字体，确保所有字符都能正确渲染，特别是如果客户使用多种语言时，特殊字符和重音符号。
- 详细标记剪贴画 —— 客户通过关键词搜索，因此描述性标签如 "金色"、"星星"、"五角"、"装饰" 可以帮助他们快速找到合适的素材。
- 将相关剪贴画归类到同一类别中。如果您销售团队商品，为每个运动创建一个类别，而不是一个巨大的 "运动" 类别。
- 定期从客户的角度审查您的剪贴画库，通过访问商店前端的设计编辑器进行查看。
