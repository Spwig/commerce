---
title: 사용자 맞춤 제품을 위한 디자인 템플릿
---

디자인 템플릿은 고객을 위해 만든 사전 제작된 시작 디자인입니다. 고객은 빈 캔버스를 마주하는 대신, 템플릿 갤러리를 탐색하여 마음에 드는 디자인을 선택하고 자신의 취향에 맞게 커스터마이징할 수 있습니다. 템플릿은 시작하는 데 필요한 노력과 창의성을 줄여 전환율을 크게 향상시킵니다.

## 템플릿의 중요성

사용자 맞춤 제품 페이지를 방문하는 대부분의 고객은 그래픽 디자이너가 아닙니다. 빈 캔버스는 두려울 수 있습니다. 템플릿은 다음과 같은 방식으로 이 문제를 해결합니다:

- **영감** — 고객은 제품으로 가능한 것을 보게 됩니다
- **속도** — 템플릿에서 시작하는 것이 처음부터 만드는 것보다 빠릅니다
- **품질** — 전문적으로 디자인된 템플릿은 더 나은 최종 결과를 보장합니다
- **지침** — 템플릿은 고객이 가장 좋은 외형을 위해 요소를 어디에 배치해야 하는지 보여줍니다

## 템플릿 생성

템플릿은 각 제품의 **디자인 편집기 설정** 페이지에서 생성됩니다.

1. 제품의 **디자인 편집기 설정** 페이지로 이동합니다
2. **템플릿** 섹션으로 전환합니다 (설정 페이지 탭에서 보이는 경우 또는 스크롤 가능한 섹션으로 보이는 경우)
3. **+ 템플릿 추가**를 클릭합니다
4. 템플릿 세부 정보를 입력합니다:
   - **이름** — 고객이 보게 될 설명적인 이름 (예: "팀 정신", "생일 축하")
   - **Slug** — 이름에서 자동 생성되는 URL 안전 식별자
   - **설명** — 선택적 짧은 설명
   - **카테고리** — 관련된 템플릿을 그룹화합니다 (예: "생일", "비즈니스", "휴일", "스포츠")
   - **정렬 순서** — 템플릿이 갤러리에 표시되는 순서를 결정합니다
5. 템플릿을 저장합니다

## 시각적 템플릿 편집기

템플릿을 생성한 후 **시각적 템플릿 편집기**를 열어 캔버스 레이아웃을 설계할 수 있습니다. 이곳에서 각 표면에 텍스트, 이미지, 클립아트를 배치하고 개별 요소에 대한 잠금 제어를 설정할 수 있습니다.

### 캔버스 사용

템플릿 편집기는 고객에게 제공되는 동일한 캔버스 도구를 제공합니다:

- **텍스트 도구** — 폰트, 크기, 색상, 정렬 옵션을 사용하여 텍스트 요소를 추가합니다
- **이미지 도구** — 미디어 라이브러리에서 이미지를 추가합니다
- **클립아트 도구** — 라이브러리에서 클립아트를 탐색하고 추가합니다

한 번에 하나의 표면만 작업할 수 있습니다. 캔버스 상단의 **표면 탭**을 사용하여 표면 간 전환(예: 티셔츠 템플릿의 앞면에서 뒷면으로 전환)을 수행합니다.

### 요소 추가

1. 도구 패널에서 도구(텍스트, 이미지 또는 클립아트)를 선택합니다
2. **텍스트 추가**, 이미지 업로드 또는 클립아트를 클릭하여 캔버스에 추가합니다
3. 캔버스에서 요소를 드래그하여 위치와 크기를 조정합니다
4. 속성 패널을 사용하여 스타일을 조정합니다(폰트, 색상, 크기, 정렬 등)
5. 템플릿에 원하는 각 요소에 대해 반복합니다

### 템플릿 저장

디자인 작업이 완료되면 **템플릿 저장**을 클릭합니다. 시스템은 캔버스 상태에서 자동으로 썸네일 미리보기 이미지를 캡처합니다. 이 썸네일은 고객이 템플릿 갤러리를 탐색할 때 보게 됩니다.

## 잠금 제어

잠금 제어는 특정 요소에 대해 고객이 할 수 있는 작업을 제한하는 강력한 기능입니다. 이는 브랜드 일관성 유지, 디자인 품질 보장, 비즈니스 규칙 적용에 필수적입니다.

템플릿 편집기에서 요소가 선택되면 5가지 유형의 잠금을 설정할 수 있습니다:

| 잠금 | 효과 | 사용 사례 |
|------|--------|----------|
| **위치 잠금** | 고객이 요소를 이동할 수 없음 | 로고가 지정된 위치에 있도록 함 |
| **크기 잠금** | 고객이 요소를 리사이징할 수 없음 | 로고가 왜곡되지 않도록 함 |
| **회전 잠금** | 고객이 요소를 회전할 수 없음 | 텍스트가 수평이고 가독성이 있도록 함 |
| **내용 잠금** | 고객이 텍스트를 편집하거나 이미지를 교체할 수 없음 | 브랜드 이름, 필수 법적 텍스트 보호 |
| **삭제 잠금** | 고객이 요소를 제거할 수 없음 | 필수 요소가 항상 표시되도록 함 |

잠금은 조합하여 사용할 수 있습니다.

예를 들어, 회사 로고는 모든 다섯 개의 잠금 기능이 활성화되어 있을 수 있습니다(이동, 크기 조정, 회전, 편집, 삭제 불가), 반면 "Your Name Here" 텍스트 필드는 위치와 크기만 잠금 처리될 수 있습니다(고객은 텍스트 내용은 편집할 수 있지만 재배치는 불가능합니다).
