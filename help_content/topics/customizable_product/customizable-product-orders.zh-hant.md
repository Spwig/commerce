---
title: 履行可自定產品訂單
---

當客戶設計產品並下訂單時，他們的設計會被凍結並與訂單一起儲存。本指南說明自定設計如何流經訂單生命週期，以及如何存取您履行訂單所需的列印就緒文件。

## 設計生命週期

客戶的設計從創建到履行會經歷幾個階段：

### 1. 設計創建

客戶使用商店前端的視覺編輯器來創建他們的設計。當他們工作時，進度會自動保存在瀏覽器中。已註冊的客戶還可以將設計保存到他們的帳戶中，以便後續編輯。

### 2. 設計草稿

當客戶點擊 **加入購物車** 時，當前的設計狀態會被保存為 **設計草稿**。草稿包含：

- 每個表面（元素位置、文字內容、上傳圖片、插圖、樣式）的完整畫布狀態
- 顯示所有適用設計費用的價格明細
- 每個表面的縮略圖預覽

草稿通過唯一的令牌與購物車項目連結。這確保即使客戶在結帳前繼續購物，他們創建的確切設計也會被保留。

**草稿過期：** 如果客戶未完成訂單，設計草稿在 7 天後會自動過期。這可防止被遺棄的設計累積。

### 3. 設計快照

當客戶完成結帳並下訂單時，設計草稿會被轉換為 **不可變的設計快照**。這是設計的永久記錄：

- 快照在購買後無法被客戶修改
- 它包含與草稿相同的設計數據
- 它永久連結到特定的訂單項目

這種不可變性非常重要 —— 它確保客戶訂購的內容就是您生產和發貨的內容，付款後沒有任何更改的可能。

### 4. 履行文件生成

訂單下達後，系統會自動為設計的每個表面生成 **高解析度履行文件**。

這些是複合圖像，將所有設計元素（文字、圖片、剪貼畫）合併為一個可列印的文件，並根據每個表面設定的DPI進行配置。

渲染是在背景中異步進行的。對於大多數設計，渲染在幾秒內即可完成。快照的**已渲染**狀態表示是否已準備好履行文件。

## 在訂單中訪問設計數據

### 訂單詳情頁面

當您在管理面板查看包含可自定義產品的訂單時：

1. 進入 **訂單 > 所有訂單**
2. 打開包含自定義產品的訂單
3. 可自定義產品的訂單項目會顯示設計信息，包括表面預覽和設計快照的鏈接

### 設計快照列表

您也可以直接瀏覽所有設計快照：

1. 進入 **可自定義產品 > 設計快照**
2. 清單顯示與訂單項目相關的所有快照
3. 點擊快照以查看完整的設計數據、渲染圖像和履行文件

每個快照顯示：

| 欄位 | 說明 |
|-------|-------------|
| **訂單項目** | 連結到相關的訂單項目 |
| **設計數據** | 完整的畫布狀態（JSON） |
| **已渲染圖像** | 每個表面的預覽縮略圖 |
| **履行文件** | 用於列印的高解析度複合文件 |
| **已渲染** | 渲染是否完成 |
| **渲染完成時間** | 文件生成的時間戳 |

## 下載履行文件

履行文件就是您發送到列印服務提供商或用於生產過程的文件。

**對於自定義T恤訂單：**
- 下載 **正面** 表面文件（例如，300 DPI 複合 PNG）
- 下載 **背面** 表面文件
- 下載 **袖子** 表面文件（如果已設計）
- 將所有文件發送到您的絲網印刷機或DTG（直接到衣物）印刷機

**針對客製海報訂單：**
- 下載單一 **正面** 表面檔案，解析度為列印品質
- 如果為該表面設定了出血區域，檔案將包含出血區域
- 將其傳送給您的海報/卡片印刷商

每個檔案都是包含所有設計元素合併在一起的單一合成影像，以您為該表面設定的 DPI 渲染。

## 已儲存的設計

已註冊的客戶可以將其設計儲存到他們的帳戶中，以供後續編輯。作為商家，您可以以只讀清單的形式查看這些已儲存的設計：

1. 進入 **可客製化產品 > 已儲存設計**
2. 清單顯示所有客戶儲存的設計，包含客戶姓名、產品、設計名稱和日期

已儲存的設計為：
- **屬於客戶** — 它們屬於客戶的帳戶
- **僅限商家查看** — 您可以查看但不能修改
- **與訂單分離** — 已儲存的設計只有在客戶將其加入購物車並結帳時才會成為訂單
- **可重複使用** — 客戶可以載入已儲存的設計，進行修改，並多次下單

## 製造流程

### 標準流程

1. **接收訂單** — 訂單會出現在您的訂單清單中，包含客製化商品
2. **驗證渲染** — 檢查設計快照是否顯示 **已渲染：是**。如果渲染尚未完成，請稍等片刻並重新整理
3. **下載檔案** — 下載每個設計表面的製造檔案
4. **審核品質** — 打開檔案並驗證設計是否符合您的列印品質標準（檢查 DPI、元素位置和文字可讀性）
5. **傳送至生產** — 將檔案傳送給您的列印供應商或生產團隊
6. **出貨並完成** — 生產完成後，出貨產品並標記訂單為已完成

### T 恤製造範例

1.

收到訂單："客製團隊 T 恤"，正面和背面有設計
2.

打開訂單 → 查看設計快照
3.

下載 `front.png`（300 DPI，300x400mm）和 `back.png`（300 DPI，300x400mm）
4.


將兩份文件傳送給您的 DTG 印刷機，並附上訂單中選項的衣物顏色和尺寸

# 如果生產出現問題且需要重新製作訂單，保存的設計可讓重新下單變得更加簡單。
- 在生產時間表中建立緩衝時間，以應對可自定義的產品。

與標準產品不同，每件商品都需要個別處理文件。
- 如果您處理大量可自定義的訂單，請考慮通過與印刷供應商的 API 整合來自動化文件下載步驟。
