---
title: Клипарт и шрифты для настраиваемых продуктов
---

Редактор дизайна имеет два типа творческих активов, которые вы можете предоставить клиентам: **клипарт** (готовые графические элементы, которые они могут добавлять в свои дизайны) и **пользовательские шрифты** (помимо стандартных шрифтов системы). Создание хорошо подобранных библиотек активов делает редактор более полезным и помогает клиентам быстрее создавать лучшие дизайны.

## Библиотека клипартов

Клипарт предоставляет клиентам библиотеку готовых графических элементов, которые они могут добавлять в свои дизайны с одного клика. Вместо того чтобы требовать от клиентов искать и загружать собственные изображения для распространенных элементов, таких как значки, рамки или декоративные графические элементы, вы предоставляете их готовыми к использованию.

### Создание категорий клипартов

Клипарты организованы в категории, которые клиенты могут просматривать. Категории помогают клиентам быстрее находить то, что им нужно.

1. Перейдите в **Настраиваемые продукты > Категории клипартов**
2. Нажмите **+ Добавить категорию клипартов**
3. Заполните:
   - **Название категории** — то, что видят клиенты (например, «Спорт», «Рамки», «Праздники»)
   - **Slug** — автоматически генерируется из названия
   - **Иконка** — класс иконки Font Awesome для вкладки категории (например, `fas fa-football-ball`)
   - **Порядок сортировки** — определяет порядок, в котором категории отображаются в редакторе
4. Нажмите **Сохранить**

**Примеры категорий для магазина футболок:**

| Категория | Иконка | Примеры клипартов |
|----------|------|-----------------|
| Спорт | `fas fa-football-ball` | Логотипы команд, спортивное оборудование, символы спорта |
| Юмор | `fas fa-laugh` | Мемы, смешные цитаты, персонажи мультфильмов |
| Природа | `fas fa-leaf` | Животные, цветы, пейзажи |
| Геометрия | `fas fa-shapes` | Узоры, абстрактные формы, туземные дизайны |

**Примеры категорий для магазина печати/плакатов:**

| Категория | Иконка | Примеры клипартов |
|----------|------|-----------------|
| Рамки | `fas fa-border-all` | Декоративные рамки, украшения углов |
| Сезонные | `fas fa-snowflake` | Значки праздников, сезонные мотивы |
| Значки | `fas fa-icons` | Звезды, сердца, стрелки, галочки |
| Фоновые изображения | `fas fa-image` | Текстуры, градиенты, узоры |

### Добавление клипартов

Каждый клипарт — это файл изображения (PNG или SVG), который клиенты могут разместить на своем холсте.

1. Перейдите в **Настраиваемые продукты > Активы клипартов**
2. Нажмите **+ Добавить актив клипартов**
3. Заполните:
   - **Название** — описательное название (например, «Золотая звезда», «Шлем футболиста»)
   - **Категория** — выберите из ваших категорий клипартов
   - **Актив изображения** — нажмите, чтобы открыть библиотеку медиа и выбрать или загрузить файл изображения
   - **Область действия** — выберите доступность (см. ниже)
   - **Теги** — поисковые ключевые слова для этого клипартов (например, `['star', 'gold', 'decoration']`)
   - **Порядок сортировки** — определяет позицию внутри категории
4. Нажмите **Сохранить**

### Понимание области действия клипартов

Каждый клипарт имеет область действия, которая определяет, где он доступен:

| Область действия | Описание | Сценарий использования |
|-------|-------------|----------|
| **Доступен для всех продуктов** | Отображается в браузере клипартов для каждого настраиваемого продукта | Общие графические элементы, такие как звезды, рамки и распространенные значки |
| **Только для конкретного продукта** | Отображается только для одного выбранного продукта | Графические элементы, специфичные для продукта, такие как логотипы брендов или арт-работы, связанные с продуктом |

Для большинства активов используйте **Доступен для всех продуктов**. Резервируйте область действия, специфичную для продукта, для активов, которые имеют смысл только в контексте одного продукта — например, логотипы команд для продукта с товаром для команды.

### Рекомендации по файлам клипартов

- **Формат:** Используйте PNG для растровой графики и SVG для векторной графики. Файлы SVG масштабируются без потери качества, что делает их идеальными для клипартов, которые клиенты могут значительно изменять по размеру
- **Разрешение:** Файлы PNG должны быть не менее 500x500 пикселей для хорошего качества печати
- **Фон:** Используйте прозрачные фоны (PNG с альфа-каналом или SVG), чтобы клипарты естественно вписывались в дизайн
- **Размер файла:** Сохраняйте отдельные файлы клипартов меньше 500 КБ для быстрой загрузки в редакторе

## Пользовательские шрифты

Пользовательские шрифты расширяют выбор шрифтов в редакторе дизайна за пределы стандартных шрифтов системы.

Это позволяет предлагать кураторский шрифт, соответствующий вашему бренду или стилю продукта.

### Добавление пользовательского шрифта

1. Перейдите в **Customizable Products > Custom Fonts**
2. Нажмите **+ Add Custom Font**
3. Заполните:
   - **Font Name** — Отображаемое имя, которое будет показано в выборщике шрифтов (например, "Playfair Display")
   - **Font Family** — Имя шрифта в формате CSS, используемое внутренне (например, `PlayfairDisplay`)
   - **Regular** — Нажмите, чтобы загрузить файл шрифта обычного веса (WOFF2 или TTF) через библиотеку медиа
   - **Bold** — Необязательный вариант жирного веса
   - **Italic** — Необязательный вариант курсива
   - **Bold Italic** — Необязательный вариант жирного курсива
4. Нажмите **Save**

**Regular** вес является обязательным для пользовательских шрифтов. Жирный, курсив и жирный курсив являются необязательными — если они не предоставлены, браузер попытается синтезировать эти стили из обычного шрифта, хотя результат может не выглядеть так же аккуратно, как специализированные файлы шрифтов.

### Системные шрифты vs пользовательские шрифты

Вы также можете зарегистрировать системные шрифты, предустановленные на большинстве устройств:

1. Добавьте новую запись пользовательского шрифта
2. Отметьте **System Font**
3. Введите имя шрифта в точном виде, как он отображается в CSS (например, `Georgia`, `Courier New`)
4. Для системных шрифтов не требуется загрузка файла

Системные шрифты загружаются мгновенно, так как они уже установлены на устройстве клиента. Пользовательские загруженные шрифты сначала должны быть загружены, что добавляет небольшую задержку при первом выборе шрифта.

### Рекомендации по шрифтам по типу продукта

**Для футболок и одежды:**
- Лучше всего подходят жирные и эффектные шрифты: Impact, Anton, Bebas Neue, Oswald
- Блок-буквы и беззасечные шрифты наиболее читабельны на ткани
- Избегайте тонких или хрупких шрифтов, которые могут плохо печататься на текстурированных поверхностях

**Для плакатов и печатных продуктов:**
- Элегантные шрифты с засечками для официальных дизайнов: Playfair Display, Merriweather, Lora
- Шрифты-как-рукопись для приглашений и открыток: Great Vibes, Dancing Script, Pacifico
- Чистые беззасечные шрифты для современных дизайнов: Montserrat, Raleway, Open Sans

### Форматы файлов шрифтов

| Формат | Расширение | Рекомендация |
|--------|-----------|----------------|
| WOFF2 | `.woff2` | Предпочтительный — наименьший размер файла, быстрая загрузка |
| TrueType | `.ttf` | Хороший альтернативный вариант — широкая совместимость |

Файлы WOFF2 обычно на 30-50% меньше, чем файлы TTF, поэтому они быстрее загружаются в редакторе клиента. Используйте WOFF2, если это возможно.

## Управление вашей библиотекой активов

### Организация для клиентов

Порядок, в котором активы отображаются в редакторе, определяется полем **Sort Order** как для категорий, так и для отдельных активов. Активы с меньшими числами отображаются первыми. Используйте это для:

- Размещения наиболее популярных графических изображений в начале
- Размещения лучших и самых универсальных графических изображений в начале каждой категории
- Организации шрифтов так, чтобы наиболее часто используемые варианты отображались первыми

### Сохранение актуальности библиотеки

- Добавьте сезонные графические изображения перед праздниками (Хэллоуин, Рождество, День Святого Валентина) и деактивируйте их после
- Используйте флажок **Active**, чтобы временно скрыть активы, не удаляя их
- Отслеживайте, какие графические изображения и шрифты чаще всего используют клиенты, и расширяйте эти категории

## Советы

- Начните с малого — 20-30 высококачественных графических изображений, распределенных по 3-4 категориям, лучше, чем сотни средних вариантов. Вы всегда можете добавить больше, когда узнаете, чего хотят клиенты.
- Используйте формат SVG для графических изображений, если это возможно. Файлы SVG меньше по размеру, идеально масштабируются до любого размера и обеспечивают более четкую печать, чем растровые изображения.
- Тестируйте каждый загруженный шрифт в редакторе дизайна, чтобы убедиться, что все символы отображаются корректно, особенно специальные символы и диакритические знаки, если клиенты используют несколько языков.
- Тщательно промаркируйте графические изображения — клиенты ищут по ключевым словам, поэтому описательные метки, такие как "золото", "звезда", "пятиконечная", "декор", помогают им быстро найти нужный актив.
- Группируйте связанные графические изображения в одну категорию. Если вы продаете товары для команд, создайте категорию на каждый вид спорта вместо одной большой категории "Спорт".
- Регулярно проверяйте свою библиотеку графических изображений с точки зрения клиента, посещая редактор дизайна на сайте магазина.
