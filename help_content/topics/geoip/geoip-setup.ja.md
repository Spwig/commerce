---
title: GeoIPの設定
---

GeoIPは、訪問者のIPアドレスに基づいて、それぞれの訪問者がどこから来ているかを自動的に検出する機能です。これは、ストア全体で位置ベースの機能を可能にし、デフォルトで正しい通貨を表示したり、地理的なビジネスルールを実行したり、分析で国レベルの分解を表示したりします。

あなたのストアはSpwig GeoIPサービスで事前に構成されているため、地理的な検出はすぐに使用できます。さらに正確性を高めるために、追加のプロバイダを接続することもできます。または、自分でダウンロードしたデータベースを使用することも、CDNからのヘッダーに依存してゼロレイテンシーの検索を実行することも可能です。

## プロバイダの仕組み

**顧客 > GeoIPプロバイダ**に移動して、あなたのストアに設定されているプロバイダを確認してください。各プロバイダは、異なる方法でIPアドレスから位置を検索します。訪問者が到着すると、ストアはアクティブなプロバイダを優先順位の順にクエリし、最初に成功した結果を使用します。

複数のプロバイダが同時にアクティブになることもできます。優先順位の低い番号のプロバイダが最初に試されます。最高優先順位のプロバイダが失敗したり、データを返さなかったりした場合、次のプロバイダが自動的に試されます。

### 利用可能なプロバイダタイプ

| プロバイダ | 説明 |
|----------|-------------|
| **Spwig GeoIP** | Spwigのサービスを通じたデフォルトのクラウドベースの検索。設定不要です。 |
| **MaxMind GeoLite2** | MaxMindのオフラインデータベース。高精度。無料ライセンスキーが必要です。 |
| **DB-IP Lite** | DB-IPのオフラインデータベース。公式サイトからダウンロードしてください。 |
| **IP2Location LITE** | IP2Locationのオフラインデータベース。無料登録が必要です。 |
| **CDN Edge Headers** | あなたのCDN（例: Cloudflare）が挿入した位置情報ヘッダーを読み取ります。ゼロレイテンシーです。 |
| **Browser Hints** | ブラウザが提供するタイムゾーン/言語を、ソフトな位置情報として使用します。 |
| **Custom Provider** | Spwigコンポーネントマーケットプレイスからインストールされたプロバイダコンポーネントです。 |

## プロバイダの追加

### Spwig GeoIPサービス（デフォルト）の使用

Spwig GeoIPプロバイダは、新規インストール時に自動的に追加されます。

リストに表示されていることを確認し、**Is Active**がチェックされていることを確認してください。

追加の設定は必要ありません。

### MaxMind GeoLite2データベースの追加

MaxMindは、外部サービスにクエリを送信することなく正確な結果を提供する無料のオフラインデータベースを提供しています。

1. maxmind.comで無料アカウントを登録し、ライセンスキーを生成してください
2. **Customers > GeoIP Providers** に移動し、**+ Add GeoIP Provider** をクリックします
3. フォームを入力してください:
   - **Name**: `MaxMind GeoLite2` (または任意の説明的な名前)
   - **Provider Type**: MaxMind GeoLite2
   - **Is Active**: にチェックをつける
   - **Priority**: `1` (Spwigのデフォルトよりも低い値に設定して最初に試すか、または代替として使用する場合は高い値に設定)
   - **License Key**: あなたのMaxMindライセンスキーを貼り付けます
   - **Database URL**: あなたのMaxMindアカウントダッシュボードからダウンロードURLを指定します
4. **Save** をクリックします

保存後、リストからプロバイダを選択し、**Update selected provider databases** アクションを使用してデータベースURLがアクセス可能であるかを確認してください。

### CDNエッジヘッダの追加

ストアがCloudflareの`CF-IPCountry`などの地理情報ヘッダを挿入するCDNの背後にある場合、それらのヘッダを使用して即時で、ゼロレイテンシーの国検出が可能です。

1. **Customers > GeoIP Providers** に移動し、**+ Add GeoIP Provider** をクリックします
2. **Provider Type** を **CDN Edge Headers** に設定します
3. **Priority** を `0` に設定します (ヘッダは最も速い情報源なので、最高の優先度)
4. **Config** フィールドで、CDNが使用するヘッダを指定します:
   ```json
   {
     "header_name": "CF-IPCountry"
   }
   ```
5. **Save** をクリックします

### プロバイダのテスト

プロバイダを追加した後、それが正常に動作しているか確認できます:

1. GeoIP Providersリストで、チェックボックスを使用してプロバイダを選択します
2. **Action** ドロップダウンを開き、**Test selected providers** を選択します
3. **Go** をクリックします

Spwigは、既知のIPアドレス（GoogleのパブリックDNS、`8.8.8.8`）に対してテストクエリを送信し、結果を表示します。テストが成功すると、返された国と応答時間がミリ秒単位で表示されます。

## プロバイダの優先度の設定

# プロバイダの優先順位

複数のプロバイダが有効な場合、**Priority** フィールドはどのプロバイダが最初に試されるかを制御します。

数字が小さいほど優先順位が高くなります。

例えば、CDN ヘッダを最初に使用し（最も高速）、次に Spwig GeoIP にフォールバックする場合:

| Provider | Priority |
|----------|----------|
| CDN Edge Headers | 0 |
| Spwig GeoIP | 10 |

リストビュー内で直接優先順位を編集できます — **Priority** 列はインラインで編集可能です。

## プロバイダのパフォーマンスの監視

各プロバイダのレコードは、自身の正確性統計を追跡します:

- **Total Lookups** — 尝試された IP ロックアップの総数
- **Successful Lookups** — 結果を返したロックアップ
- **Failed Lookups** — データが返されなかったかエラーが発生したロックアップ
- **Average Response (ms)** — ミリ秒単位の平均応答時間
- **Accuracy** — 成功したロックアップの割合

プロバイダが低い正確性率や高い応答時間を示す場合、その優先順位を調整するか、より高性能なオプションに切り替えることを検討してください。

## 国コードのマッピング

**Customers > Country Mappings** に移動して、通貨、言語、税、配送の国ごとのデフォルト設定を構成してください。各国コードのエントリは次の項目を制御します:

- **Default Currency** — その国から訪問する顧客に事前に選択される通貨
- **Default Language** — その国から訪問する顧客に表示される言語
- **Tax Rate** — その国に適用されるデフォルトの税率
- **Is EU Member** / **Requires VAT** — EU 税法のコンプライアンスロジックに使用されます
- **Shipping Zone** — 国を配送ゾーンにリンクします
- **Supports COD** — その国で現金払い（COD）を有効にします

**Is Active**、**Default Currency**、**Default Language** のフィールドは、各レコードを開かずにリスト内で直接編集できます。

## ヒント

すべてのマークダウンのフォーマット、画像のパス、コードブロック、技術用語を保持してください。

- Spwig GeoIP プロバイダは設定なしですぐに動作します — 高い精度やオフライン操作が必要な場合のみ、追加のプロバイダを追加してください
- Cloudflare を使用している場合、CDN Edge Headers プロバイダが最適な選択です: レイテンシを追加せず、API クォータにカウントされません
- 実際に必要なプロバイダのみを有効にしてください — 最初のプロバイダが成功している場合、多くの有効なプロバイダを保持しても精度は向上しません
- 週次に精度統計を確認し、成功率が80％未満のプロバイダを無効にしてください
- 国コードのマッピングはデフォルトとして使用されます; 顧客は常にショッピングカートで通貨と言語を手動で変更できます
