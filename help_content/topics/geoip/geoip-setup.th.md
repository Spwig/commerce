---
title: การตั้งค่า GeoIP
---

GeoIP ช่วยให้ร้านค้าของคุณสามารถตรวจจับตำแหน่งของผู้เยี่ยมชมแต่ละคนได้อัตโนมัติโดยใช้ที่อยู่ IP ของพวกเขา ฟีเจอร์นี้ช่วยให้คุณสามารถใช้ฟังก์ชันที่เกี่ยวข้องกับตำแหน่งได้ทั่วทั้งร้านค้า — ตั้งแต่แสดงสกุลเงินที่เหมาะสมเป็นค่าเริ่มต้น ไปจนถึงการใช้กฎทางธุรกิจตามภูมิศาสตร์ และการดูข้อมูลการวิเคราะห์ในระดับประเทศ

ร้านค้าของคุณมีการตั้งค่าเริ่มต้นพร้อมใช้งานด้วยบริการ GeoIP ของ Spwig ดังนั้นการตรวจจับภูมิศาสตร์จึงทำงานได้ทันที คุณยังสามารถเชื่อมต่อกับผู้ให้บริการเพิ่มเติมเพื่อความแม่นยำสูงขึ้น ใช้ฐานข้อมูลที่คุณดาวน์โหลดเอง หรือใช้เฮดเดอร์จาก CDN เพื่อการค้นหาที่ไม่มีความล่าช้า

## วิธีการทำงานของผู้ให้บริการ

ไปที่ **ลูกค้า > ผู้ให้บริการ GeoIP** เพื่อดูผู้ให้บริการที่ตั้งค่าไว้สำหรับร้านค้าของคุณ ผู้ให้บริการแต่ละรายจะดำเนินการค้นหาที่อยู่ IP ไปยังตำแหน่งโดยใช้วิธีที่แตกต่างกัน เมื่อมีผู้เยี่ยมชมเข้ามา ร้านค้าของคุณจะสอบถามผู้ให้บริการที่เปิดใช้งานตามลำดับความสำคัญ และใช้ผลลัพธ์แรกที่สำเร็จ

ผู้ให้บริการหลายรายสามารถเปิดใช้งานพร้อมกันได้ — ผู้ให้บริการที่มีลำดับความสำคัญต่ำกว่าจะถูกตรวจสอบก่อน หากผู้ให้บริการที่มีลำดับความสำคัญสูงสุดล้มเหลวหรือไม่ได้ส่งข้อมูลใดๆ ผู้ให้บริการถัดไปจะถูกตรวจสอบอัตโนมัติ

### ประเภทของผู้ให้บริการที่มีอยู่

| ผู้ให้บริการ | คำอธิบาย |
|----------|-------------|
| **Spwig GeoIP** | การค้นหาแบบคลาวด์โดยใช้บริการของ Spwig เป็นค่าเริ่มต้น ไม่จำเป็นต้องตั้งค่า |
| **MaxMind GeoLite2** | ฐานข้อมูลแบบออฟไลน์จาก MaxMind ความแม่นยำสูง จำเป็นต้องมีคีย์ใบอนุญาตฟรี |
| **DB-IP Lite** | ฐานข้อมูลแบบออฟไลน์จาก DB-IP ดาวน์โหลดได้จากเว็บไซต์ของพวกเขา |
| **IP2Location LITE** | ฐานข้อมูลแบบออฟไลน์จาก IP2Location จำเป็นต้องลงทะเบียนฟรี |
| **CDN Edge Headers** | อ่านเฮดเดอร์ตำแหน่งที่ถูกแทรกโดย CDN ของคุณ (เช่น Cloudflare) ไม่มีความล่าช้า |
| **Browser Hints** | ใช้เขตเวลา/ภาษาที่ให้โดยเบราว์เซอร์เป็นสัญญาณตำแหน่งแบบอ่อน |
| **Custom Provider** | องค์ประกอบผู้ให้บริการที่ติดตั้งจากตลาดองค์ประกอบของ Spwig |

## การเพิ่มผู้ให้บริการ

### การใช้บริการ Spwig GeoIP (ค่าเริ่มต้น)

ผู้ให้บริการ Spwig GeoIP จะถูกเพิ่มอัตโนมัติเมื่อติดตั้งใหม่

ตรวจสอบว่ามันปรากฏในรายการ และว่า **Is Active** ถูกตีความไว้


ไม่จำเป็นต้องตั้งค่าเพิ่มเติม

### การเพิ่มฐานข้อมูล MaxMind GeoLite2

MaxMind ให้ฐานข้อมูลฟรีแบบออฟไลน์ ซึ่งให้ผลลัพธ์ที่แม่นยำโดยไม่ต้องส่งการค้นหาไปยังบริการภายนอก

1. ลงทะเบียนเพื่อรับบัญชีฟรีที่ maxmind.com และสร้างคีย์ใบอนุญาต
2. ไปที่ **Customers > GeoIP Providers** และคลิก **+ Add GeoIP Provider**
3. กรอกฟอร์ม:
   - **Name**: `MaxMind GeoLite2` (หรือชื่ออธิบายใด ๆ)
   - **Provider Type**: MaxMind GeoLite2
   - **Is Active**: ติ๊กเครื่องหมาย
   - **Priority**: `1` (ต่ำกว่าค่าเริ่มต้นของ Spwig เพื่อให้ลองใช้ก่อน หรือสูงกว่าเพื่อใช้เป็นตัวสำรอง)
   - **License Key**: วางคีย์ใบอนุญาตของ MaxMind ของคุณ
   - **Database URL**: ลิงก์ดาวน์โหลดจากแดชบอร์ดบัญชี MaxMind ของคุณ
4. คลิก **Save**

หลังจากบันทึกแล้ว เลือกผู้ให้บริการในรายการ และใช้การกระทำ **Update selected provider databases** เพื่อตรวจสอบว่า URL ฐานข้อมูลสามารถเข้าถึงได้

### การเพิ่มหัวข้อของ CDN Edge

หากร้านค้าของคุณอยู่ด้านหลัง CDN ที่แทรกหัวข้อการระบุตำแหน่ง (เช่น หัวข้อ `CF-IPCountry` ของ Cloudflare) คุณสามารถใช้หัวข้อเหล่านั้นเพื่อตรวจจับประเทศได้ทันทีโดยไม่มีความล่าช้า

1. ไปที่ **Customers > GeoIP Providers** และคลิก **+ Add GeoIP Provider**
2. ตั้งค่า **Provider Type** เป็น **CDN Edge Headers**
3. ตั้งค่า **Priority** เป็น `0` (ลำดับความสำคัญสูงสุด เนื่องจากหัวข้อเป็นแหล่งข้อมูลที่เร็วที่สุด)
4. ในฟิลด์ **Config** ระบุว่าหัวข้อที่ CDN ของคุณใช้อยู่คืออะไร:
   ```json
   {
     "header_name": "CF-IPCountry"
   }
   ```
5. คลิก **Save**

### การทดสอบผู้ให้บริการ

หลังจากเพิ่มผู้ให้บริการ คุณสามารถตรวจสอบว่าผู้ให้บริการทำงานได้ถูกต้อง:

1. ในรายการ GeoIP Providers เลือกผู้ให้บริการโดยใช้ช่องถูกติ๊ก
2. เปิดเมนูแบบร่วงลง **Action** และเลือก **Test selected providers**
3. คลิก **Go**

Spwig จะส่งการค้นหาทดสอบสำหรับที่อยู่ IP ที่ทราบ (DNS สาธารณะของ Google, `8.8.8.8`) และแสดงผลลัพธ์ให้คุณ ผลการทดสอบที่สำเร็จจะแสดงประเทศที่ได้รับและเวลาตอบสนองในมิลลิวินาที

## การตั้งลำดับความสำคัญของผู้ให้บริการ

# การตั้งค่าลำดับความสำคัญของผู้ให้บริการ

เมื่อมีผู้ให้บริการหลายรายที่เปิดใช้งาน ฟิลด์ **Priority** จะควบคุมว่าผู้ให้บริการใดจะถูกใช้ก่อน

ตัวเลขที่ต่ำกว่าหมายถึงลำดับความสำคัญที่สูงกว่า

ตัวอย่างเช่น เพื่อใช้หัวข้อของ CDN ก่อน (เร็วที่สุด) และกลับไปใช้ Spwig GeoIP หากไม่สามารถใช้งาน CDN ได้:

| Provider | Priority |
|----------|----------|
| CDN Edge Headers | 0 |
| Spwig GeoIP | 10 |

คุณสามารถแก้ไขลำดับความสำคัญโดยตรงในหน้ารายการ — คอลัมน์ **Priority** สามารถแก้ไขได้แบบ inline

## การติดตามประสิทธิภาพของผู้ให้บริการ

บันทึกข้อมูลของผู้ให้บริการแต่ละรายจะติดตามสถิติความแม่นยำของตนเอง:

- **Total Lookups** — จำนวนการค้นหา IP ทั้งหมดที่ได้รับการดำเนินการ
- **Successful Lookups** — การค้นหาที่ได้ผลลัพธ์
- **Failed Lookups** — การค้นหาที่ไม่ได้ผลลัพธ์หรือเกิดข้อผิดพลาด
- **Average Response (ms)** — เวลาเฉลี่ยในการตอบสนองในหน่วยมิลลิวินาที
- **Accuracy** — ร้อยละของการค้นหาที่สำเร็จ

หากผู้ให้บริการมีอัตราความแม่นยำต่ำหรือเวลาตอบสนองสูง ให้พิจารณาปรับลำดับความสำคัญหรือปิดใช้งานผู้ให้บริการนั้นเพื่อใช้ตัวเลือกที่มีประสิทธิภาพมากกว่า

## การจับคู่ประเทศ

ไปที่ **Customers > Country Mappings** เพื่อตั้งค่าค่าเริ่มต้นตามประเทศสำหรับสกุลเงิน ภาษา ภาษี และการจัดส่ง แต่ละรายการประเทศควบคุม:

- **Default Currency** — สกุลเงินที่ถูกเลือกไว้ล่วงหน้าสำหรับผู้เยี่ยมชมจากประเทศนั้น
- **Default Language** — ภาษาที่แสดงให้ผู้เยี่ยมชมจากประเทศนั้นเห็น
- **Tax Rate** — อัตราภาษีค่าเปอร์เซ็นต์ที่ใช้เป็นค่าเริ่มต้นสำหรับประเทศนั้น
- **Is EU Member** / **Requires VAT** — ใช้สำหรับตรรกะการปฏิบัติตามกฎหมายภาษีในสหภาพยุโรป
- **Shipping Zone** — เชื่อมโยงประเทศกับโซนการจัดส่ง
- **Supports COD** — เปิดใช้งานการชำระเงินปลายทางสำหรับประเทศนั้น

คุณสามารถแก้ไขฟิลด์ **Is Active**, **Default Currency**, และ **Default Language** โดยตรงในรายการโดยไม่จำเป็นต้องเปิดแต่ละบันทึก

## เคล็ดลับ

รักษาการจัดรูปแบบ markdown ทั้งหมด ชื่อเส้นทางรูปภาพ บล็อกโค้ด และคำศัพท์ทางเทคนิคไว้

- ผู้ให้บริการ Spwig GeoIP ทำงานได้ทันทีโดยไม่ต้องตั้งค่า — เพิ่มผู้ให้บริการเพิ่มเติมก็ได้หากคุณต้องการความแม่นยำสูงขึ้นหรือการทำงานแบบออฟไลน์
- หากคุณใช้ Cloudflare ผู้ให้บริการ CDN Edge Headers จะเป็นตัวเลือกที่ดีที่สุด: มันไม่เพิ่มความล่าช้าและไม่ถูกนับในส่วนของ配额 API
- รักษาผู้ให้บริการที่คุณต้องการใช้งานจริงไว้เพียงผู้ให้บริการที่ใช้งานอยู่ — การมีผู้ให้บริการจำนวนมากที่ใช้งานอยู่ไม่เพิ่มความแม่นยำหากผู้ให้บริการแรกสามารถทำงานได้สำเร็จแล้ว
- ตรวจสอบสถิติความแม่นยำรายสัปดาห์และปิดใช้งานผู้ให้บริการใดก็ตามที่มีอัตราความสำเร็จต่ำกว่า 80%
- การจับคู่ประเทศถูกใช้เป็นค่าเริ่มต้น; ลูกค้าสามารถเปลี่ยนสกุลเงินและภาษาได้ด้วยตนเองในหน้าร้านค้าได้เสมอ
