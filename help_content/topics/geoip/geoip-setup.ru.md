---
title: Настройка GeoIP
---

GeoIP позволяет вашему магазину автоматически определять, откуда приходит каждый посетитель, на основе его IP-адреса. Это обеспечивает работу функций, основанных на геолокации, во всем вашем магазине — от отображения правильной валюты по умолчанию до выполнения географических бизнес-правил, до просмотра разбивки по странам в вашем аналитическом отчете.

Ваш магазин предварительно настроен с использованием службы GeoIP от Spwig, поэтому геолокационное определение работает сразу. Вы также можете подключить дополнительных поставщиков для повышения точности, использовать базу данных, которую вы загрузите самостоятельно, или зависать от заголовков CDN для мгновенных запросов.

## Как работают поставщики

Перейдите в **Customers > GeoIP Providers**, чтобы увидеть поставщиков, настроенных для вашего магазина. Каждый поставщик обрабатывает запросы IP-адреса в расположение с помощью разных методов. Когда посетитель приходит, ваш магазин запрашивает активных поставщиков в порядке приоритета и использует первый успешный результат.

Можно активировать несколько поставщиков одновременно — сначала проверяются поставщики с меньшим приоритетом. Если поставщик с наивысшим приоритетом не работает или возвращает никаких данных, автоматически проверяется следующий.

### Доступные типы поставщиков

| Поставщик | Описание |
|----------|-------------|
| **Spwig GeoIP** | По умолчанию, облако-базовый поиск через службу Spwig. Не требует настройки. |
| **MaxMind GeoLite2** | Офлайн-база данных от MaxMind. Высокая точность. Требуется бесплатный ключ лицензии. |
| **DB-IP Lite** | Офлайн-база данных от DB-IP. Скачайте с их сайта. |
| **IP2Location LITE** | Офлайн-база данных от IP2Location. Требуется бесплатная регистрация. |
| **CDN Edge Headers** | Считывает заголовки местоположения, вставленные вашим CDN (например, Cloudflare). Нулевая задержка. |
| **Browser Hints** | Использует часовой пояс/язык, предоставленный браузером, как мягкое сигнал местоположения. |
| **Custom Provider** | Компонент поставщика, установленный из магазина компонентов Spwig. |

## Добавление поставщика

### Использование службы GeoIP от Spwig (по умолчанию)

Поставщик GeoIP от Spwig добавляется автоматически при новых установках. Проверьте, что он отображается в списке, и что **Is Active** отмечен. Дополнительная настройка не требуется.

### Добавление базы данных MaxMind GeoLite2

MaxMind предлагает бесплатную оффлайн-базу данных, которая дает точные результаты без отправки запросов внешнему сервису.

1. Зарегистрируйтесь на сайте maxmind.com и сгенерируйте ключ лицензии
2. Перейдите в **Customers > GeoIP Providers** и нажмите **+ Add GeoIP Provider**
3. Заполните форму:
   - **Name**: `MaxMind GeoLite2` (или любое описательное имя)
   - **Provider Type**: MaxMind GeoLite2
   - **Is Active**: отмечен
   - **Priority**: `1` (ниже, чем у значения по умолчанию Spwig, чтобы попробовать его первым, или выше, чтобы использовать как резерв)
   - **License Key**: вставьте ваш ключ лицензии MaxMind
   - **Database URL**: URL загрузки из панели управления вашего аккаунта MaxMind
4. Нажмите **Save**

После сохранения выберите поставщика в списке и используйте действие **Update selected provider databases**, чтобы проверить, доступен ли URL базы данных.

### Добавление заголовков CDN Edge

Если ваш магазин находится за CDN, который вставляет заголовки геолокации (например, `CF-IPCountry` от Cloudflare), вы можете использовать эти заголовки для мгновенного, нулевого времени обнаружения страны.

1. Перейдите в **Customers > GeoIP Providers** и нажмите **+ Add GeoIP Provider**
2. Установите **Provider Type** на **CDN Edge Headers**
3. Установите **Priority** на `0` (наивысший приоритет, так как заголовки — это самый быстрый источник)
4. В поле **Config** укажите, какой заголовок использует ваш CDN:
   ```json
   {
     "header_name": "CF-IPCountry"
   }
   ```
5. Нажмите **Save**

### Тестирование поставщика

После добавления поставщика вы можете проверить, работает ли он правильно:

1. В списке GeoIP Providers выберите поставщика с помощью его флажка
2. Откройте выпадающий список **Action** и выберите **Test selected providers**
3. Нажмите **Go**

Spwig отправит тестовый запрос для известного IP-адреса (общедоступный DNS Google, `8.8.8.8`) и покажет результат. Успешный тест отобразит страну, возвращенную и время ответа в миллисекундах.

## Настройка приоритета поставщика

Когда активно несколько провайдеров, поле **Приоритет** определяет, какой из них будет проверен первым.

Меньшие числа означают более высокий приоритет.

Например, чтобы сначала использовать заголовки CDN (самый быстрый) и перейти к Spwig GeoIP:

| Провайдер | Приоритет |
|----------|----------|
| CDN Edge Headers | 0 |
| Spwig GeoIP | 10 |

Вы можете редактировать приоритет напрямую в списке — столбец **Приоритет** можно изменять inline.

## Мониторинг производительности провайдера

Каждая запись провайдера отслеживает собственные статистические данные точности:

- **Общее количество запросов** — общее количество попыток поиска IP-адреса
- **Успешные запросы** — запросы, вернувшие результат
- **Неудачные запросы** — запросы, вернувшие нет данных или ошибку
- **Среднее время ответа (мс)** — среднее время ответа в миллисекундах
- **Точность** — процент успешных запросов

Если у провайдера низкая точность или высокое время ответа, рассмотрите возможность изменения его приоритета или отключения его в пользу более эффективного варианта.

## Сопоставления стран

Перейдите в **Customers > Country Mappings**, чтобы настроить настройки по умолчанию для каждой страны: валюту, язык, налог и доставку. Каждая запись страны управляется следующими параметрами:

- **Валюта по умолчанию** — валюта, предварительно выбранная для посетителей из этой страны
- **Язык по умолчанию** — язык, отображаемый посетителям из этой страны
- **Ставка налога** — процент налога по умолчанию, применяемый для этой страны
- **Является ли членом ЕС** / **Требуется НДС** — используется для логики соблюдения налоговых норм ЕС
- **Зона доставки** — связывает страну с зоной доставки
- **Поддерживает оплату наличными при получении** — включает оплату наличными при получении для этой страны

Вы можете редактировать поля **Активен**, **Валюта по умолчанию** и **Язык по умолчанию** напрямую в списке, не открывая каждую запись.

## Советы

- Провайдер Spwig GeoIP работает сразу без настройки — добавляйте дополнительные провайдеры только в том случае, если вам нужна более высокая точность или работа в автономном режиме
- Если вы используете Cloudflare, провайдер CDN Edge Headers — лучший выбор: он не добавляет задержку и не учитывается в квоте API
- Оставляйте активными только те провайдеры, которые действительно нужны — наличие большого количества активных провайдеров не улучшает точность, если первый уже успешно сработал
- Проверяйте статистику точности еженедельно и отключайте любой провайдер с коэффициентом успеха ниже 80%
- Сопоставления стран используются как настройки по умолчанию; клиенты всегда могут вручную изменить валюту и язык в магазине
