---
title: ตะกร้าสินค้าที่ถูกทิ้งไว้
---

ตะกร้าสินค้าที่ถูกทิ้งไว้จะถูกสร้างขึ้นเมื่อผู้ใช้ที่เข้าสู่ระบบเพิ่มสินค้าลงในตะกร้าแต่ไม่ได้ดำเนินการชำระเงินภายใน 24 ชั่วโมง Spwig ติดตามตะกร้าเหล่านี้อัตโนมัติ เพื่อให้คุณสามารถเข้าใจรายได้ที่สูญเสียไป ระบุรูปแบบที่ทำให้ลูกค้าออกจากตะกร้า และดำเนินการเพื่อฟื้นฟูยอดขาย

ไปที่ **ลูกค้า > ตะกร้าสินค้าที่ถูกทิ้งไว้** เพื่อดูรายการที่ถูกทิ้งไว้ทั้งหมด

## สิ่งที่คุณสามารถเห็นในรายการตะกร้าสินค้าที่ถูกทิ้งไว้

รายการแสดงตะกร้าสินค้าที่ถูกทิ้งไว้พร้อมข้อมูลต่อไปนี้:

| คอลัมน์ | คำอธิบาย |
|---|---|
| **ลูกค้า** | ชื่อและอีเมลของลูกค้า |
| **ถูกทิ้งไว้เมื่อ** | วันและเวลาที่ตะกร้าถูกตีความว่าถูกทิ้งไว้ |
| **มูลค่ารวม** | มูลค่าของสินค้าในตะกร้าเมื่อถูกทิ้งไว้ |
| **จำนวนสินค้ารวม** | จำนวนสินค้าในตะกร้า |
| **เหตุผลที่คาดการณ์** | การคาดการณ์ของ Spwig ว่าทำไมตะกร้าถึงถูกทิ้งไว้ |
| **สถานะการฟื้นฟู** | ระบุว่าตะกร้านี้ถูกฟื้นฟูแล้ว (กลายเป็นคำสั่งซื้อที่สมบูรณ์) หรือไม่ |
| **จำนวนวันที่ผ่านมาหลังจากถูกทิ้งไว้** | ระยะเวลาที่ผ่านมาตั้งแต่ตะกร้าถูกทิ้งไว้ |

### การกรองตะกร้าสินค้าที่ถูกทิ้งไว้

ใช้ตัวกรองทางด้านขวาเพื่อจำกัดรายการ:

- **เหตุผลที่คาดการณ์** — กรองตามเหตุผลที่ตะกร้าถูกทิ้งไว้ (เช่น แสดงเฉพาะตะกร้าที่เหตุผลที่คาดการณ์คือค่าจัดส่งสูง)
- **ได้รับการฟื้นฟูแล้ว** — กรองเพื่อแสดงเฉพาะตะกร้าที่ได้รับการฟื้นฟูแล้วหรือยังไม่ได้รับการฟื้นฟู
- **ถูกทิ้งไว้เมื่อ** — กรองตามช่วงวันที่เพื่อเน้นตะกร้าที่ถูกทิ้งไว้ล่าสุดหรือช่วงเวลาของแคมเปญเฉพาะ

## การเข้าใจเหตุผลที่ตะกร้าถูกทิ้งไว้

Spwig บันทึกเหตุผลที่คาดการณ์ไว้สำหรับการทิ้งตะกร้าแต่ละครั้ง เหตุผลเหล่านี้มาจากสัญญาณที่จับได้ในระหว่างกระบวนการชำระเงิน และไม่สามารถรับประกันได้ว่าจะถูกต้อง 100% แต่ให้จุดเริ่มต้นที่มีประโยชน์ในการวิเคราะห์รูปแบบการทิ้งตะกร้า

| เหตุผล | สิ่งที่อาจบ่งบอกได้ |
|---|---|
| **ไม่ทราบ** | ไม่มีสัญญาณเฉพาะเจาะจงถูกจับได้ — เป็นเหตุผลที่พบบ่อยที่สุด |
| **ค่าจัดส่งสูง** | ลูกค้าอาจถูกชะลอการซื้อเนื่องจากค่าจัดส่งที่แสดงในขั้นตอนชำระเงิน |
| **ยอดรวมสูงเกินไป** | ยอดรวมของคำสั่งซื้ออาจสูงกว่าที่คาดไว้ |
| **ปัญหาในการชำระเงิน** | มีการพยายามชำระเงินแต่ล้มเหลว |
| **การเปรียบเทียบราคา** | ลูกค้ามีแนวโน้มที่จะเข้ามาเพื่อเปรียบเทียบราคา |
| **บันทึกไว้เพื่อใช้ในภายหลัง** | ลูกค้าได้บันทึกสินค้าไว้เพื่อใช้ในภายหลังโดยเจตนา |

หากคุณเห็นสัดส่วนของตะกร้าสินค้าที่มีเหตุผลเดียวกันเป็นจำนวนมาก — เช่น กลุ่มตะกร้าที่ถูกทิ้งไว้จำนวนมากที่มีเหตุผล "ค่าจัดส่งสูง" — นั่นคือสัญญาณที่ควรตรวจสอบในตั้งค่าการจัดส่งหรือการนำเสนอขั้นตอนชำระเงินของคุณ

## การดูตะกร้าที่ถูกทิ้งไว้แต่ละตัว

คลิกที่แถวใดแถวหนึ่งในรายการเพื่อเปิดมุมมองรายละเอียด คุณจะเห็น:

- **รายละเอียดการทิ้งไว้** — ลูกค้า หมายเลขอ้างอิงตะกร้า เมื่อถูกทิ้งไว้ และเหตุผลที่คาดการณ์
- **สรุปตะกร้า** — จำนวนสินค้าและมูลค่ารวมในขณะที่ถูกทิ้งไว้
- **การติดตามการกู้คืน** — ตะกร้าถูกกู้คืนหรือไม่ เมื่อใดที่ถูกกู้คืน และคำสั่งซื้อใดที่มันถูกแปลงเป็น

ฟิลด์ **ตะกร้า** เชื่อมโยงไปยังบันทึกตะกร้าที่เกี่ยวข้องโดยตรง ดังนั้นคุณจึงสามารถดูสินค้าที่อยู่ในตะกร้าได้อย่างชัดเจน

## กระบวนการทำงานการกู้คืน

Spwig ติดตามว่าตะกร้าที่ถูกทิ้งไว้แต่ละตัวได้ถูกแปลงเป็นคำสั่งซื้อที่สมบูรณ์หรือไม่ เมื่อลูกค้ากลับมาและทำการซื้อสินค้าจากตะกร้าที่ถูกทิ้งไว้ บันทึกจะถูกทำเครื่องหมายเป็น **ได้รับการกู้คืน** และคำสั่งซื้อที่เกิดขึ้นจะถูกเชื่อมโยง

ตัวนับ **อีเมลการกู้คืนที่ส่งแล้ว** แสดงจำนวนอีเมลการกู้คืนอัตโนมัติที่ถูกส่งไปยังลูกค้าสำหรับตะกร้านี้ ช่วยให้คุณเข้าใจว่าแคมเปญอีเมลของคุณมีผลในการดึงลูกค้าให้กลับมาหรือไม่

### การดำเนินการกู้คืนแบบมือถือ

หน้า abandoned carts เป็นเพียงหน้าที่สามารถอ่านได้เท่านั้น — มันเป็นเพียงบันทึกของสิ่งที่เกิดขึ้น ไม่ใช่เครื่องมือสำหรับแก้ไขเนื้อหาในตะกร้าสินค้า

เพื่อทำการดำเนินการกับตะกร้าสินค้าที่ถูกทิ้งไว้:

1. จดอีเมลของลูกค้าจากบันทึกตะกร้าสินค้าที่ถูกทิ้งไว้
2. ใช้ระบบอีเมลหรือเครื่องมือการตลาดของคุณเพื่อส่งข้อความส่วนตัว
3. พิจารณาแนบรหัสคูปองเพื่อให้ลูกค้ามีแรงจูงใจในการสั่งซื้อสินค้า
4. ติดตามสถานะ **Recovered** ภายในวันถัดไปเพื่อดูว่าการติดต่อสื่อสารนั้นได้ผลหรือไม่

## การวิเคราะห์แนวโน้มการทิ้งตะกร้าสินค้า

ตรวจสอบรายการตะกร้าสินค้าที่ถูกทิ้งไว้เป็นประจำเพื่อตรวจสอบสุขภาพของกระบวนการชำระเงิน:

- การเพิ่มขึ้นอย่างกะทันหันในจำนวนการทิ้งตะกร้าอาจบ่งบอกถึงปัญหาทางเทคนิคในกระบวนการชำระเงิน
- ค่าตะกร้าที่สูงอย่างต่อเนื่องในตะกร้าที่ยังไม่ได้รับการกู้คืนแสดงถึงกลุ่มลูกค้าที่มีโอกาสสูงสุดในการกู้คืน
- เปรียบเทียบอัตราส่วนของตะกร้าที่กู้คืนได้กับตะกร้าที่ยังไม่ได้รับการกู้คืนในช่วงเวลาต่าง ๆ เพื่อวัดประสิทธิภาพของอีเมลกู้คืน

ส่วน **Customer Analytics** ในโปรไฟล์ของลูกค้าแต่ละคนยังแสดงอัตราการทิ้งตะกร้าสินค้าส่วนตัวของลูกค้า ดังนั้นคุณสามารถระบุลูกค้าที่มักเพิ่มสินค้าลงตะกร้าแต่ไม่เคยสั่งซื้อสินค้า

## เคล็ดลับ

- จัดเรียงตาม **Total Value** (จากสูงไปต่ำ) เพื่อระบุตะกร้าสินค้าที่มีค่าสูงสุดที่ควรได้รับความสำคัญในการติดต่อสื่อสารแบบส่วนตัว
- ใช้ตัวกรอง **Abandoned At** เพื่อตรวจสอบการทิ้งตะกร้าจากแคมเปญหรือช่วงเวลาโปรโมชันเฉพาะ — การเพิ่มขึ้นในช่วงการขายแบบด่วนอาจหมายความว่าโปรโมชันของคุณดึงดูดผู้ชมมากกว่าผู้ซื้อ
- รวมข้อมูลตะกร้าสินค้าที่ถูกทิ้งไว้กับแคมเปญคูปอง: ส่งรหัสส่วนลดแบบมีระยะเวลาจำกัดให้ลูกค้าที่มีตะกร้าสินค้าที่มีค่าสูงแต่ยังไม่ได้รับการกู้คืน เพื่อสร้างความเร่งด่วน
- ตะกร้าที่ถูกทิ้งไว้เกิน 7 วันมีโอกาสกู้คืนเองได้ยาก — หากเปิดใช้งานอีเมลกู้คืน ตะกร้าเหล่านี้คือตะกร้าที่ต้องได้รับความสนใจมากที่สุด
- ลูกค้าที่ไม่ได้ลงทะเบียนบัญชีจะไม่ปรากฏในตะกร้าสินค้าที่ถูกทิ้งไว้ — การติดตามนี้ใช้ได้เฉพาะกับลูกค้าที่มีบัญชีที่ลงทะเบียนแล้วเท่านั้น
