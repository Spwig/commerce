# Понимание аналитики клиентов

Используйте аналитику клиентов, чтобы понять покупательское поведение и улучшить маркетинг.

## Панель аналитики

Перейдите в **Клиенты > Аналитика** для просмотра:

### Ключевые метрики

**Общее количество клиентов**
- Количество активных клиентов
- Новые клиенты за этот месяц
- Тенденции роста

**Пожизненная ценность клиента (CLV)**
- Средний доход на клиента
- Топ клиентов по ценности
- Тенденции CLV с течением времени

**Частота покупок**
- Среднее количество заказов на клиента
- Показатель повторных покупок
- Время между покупками

### Сегменты клиентов

Клиенты автоматически сегментируются по поведению:

**Чемпионы**
- Высокая ценность, частые покупатели
- Недавние покупки
- Ваши лучшие клиенты

**Лояльные клиенты**
- Регулярные покупатели
- Средняя или высокая ценность
- Стабильная история покупок

**Под угрозой**
- Ранее активные
- Давно не совершали покупок
- Нуждаются в повторном вовлечении

**В спячке**
- Прошло много времени с последней покупки
- Низкая активность в последнее время
- Возможно, нужна кампания по возврату

**Потерянные**
- Давно не совершали покупок
- Очень низкая активность
- Трудно реактивировать

## Анализ RFM

Оценка RFM (давность, частота, денежная ценность) помогает определить ценность клиента:

### Оценка давности
- Когда они совершили последнюю покупку?
- Шкала: 1 (давно) — 5 (совсем недавно)

### Оценка частоты
- Как часто они совершают покупки?
- Шкала: 1 (редко) — 5 (очень часто)

### Денежная оценка
- Сколько они тратят?
- Шкала: 1 (низкая ценность) — 5 (высокая ценность)

**Использование оценок RFM:**
- 555 = Клиенты-чемпионы (лучшие)
- 155 = Крупные траты, редкие покупки
- 511 = Частые, недавние, низкая ценность
- 111 = Потерянные клиенты

## Инсайты о клиентах

### Паттерны покупок

Анализируйте, что покупают клиенты:
- Самые популярные товары
- Предпочтения по категориям
- Средняя стоимость заказа
- Сезонные тенденции

### Отслеживание поведения

Отслеживайте действия клиентов:
- Просмотренные страницы
- Просмотренные товары
- Показатель брошенных корзин
- Показатель открытия писем

### Географический анализ

Узнайте, где находятся ваши клиенты:
- Топ стран/регионов
- Данные на уровне городов
- Предпочтения по доставке
- Использование валют

## Использование аналитики

### Таргетированный маркетинг

Создавайте кампании на основе сегментов:

**Чемпионы:**
- Ранний доступ к новым товарам
- VIP-привилегии и награды
- Запрос отзывов/рекомендаций

**Под угрозой:**
- Кампании по возврату через email
- Специальные скидочные предложения
- Опросы для обратной связи

**Новые клиенты:**
- Серия приветственных писем
- Рекомендации товаров
- Обучающий контент

### Стратегия по товарам

Используйте данные о покупках для:
- Выявления бестселлеров
- Обнаружения товаров с падающим спросом
- Поиска возможностей перекрёстных продаж
- Планирования запасов

### Оптимизация доходов

Увеличивайте доход за счёт:
- Допродаж высокоценным сегментам
- Повторного вовлечения неактивных клиентов
- Повышения показателя повторных покупок
- Снижения оттока

## Отчёты

### Доступные отчёты

**Привлечение клиентов**
- Новые клиенты за период
- Каналы привлечения
- Стоимость привлечения

**Анализ удержания**
- Показатель повторных покупок
- Показатель оттока клиентов
- Когортный анализ

**Отчёты о ценности**
- CLV по сегментам
- Тенденции средней стоимости заказа
- Доход на клиента

### Экспорт данных

Экспортируйте данные аналитики:
1. Выберите временной диапазон
2. Выберите метрики
3. Выберите формат (CSV, Excel, PDF)
4. Скачайте отчёт

## Интеграция с маркетингом

### Email-кампании

Используйте сегменты для таргетированных писем:
- Перейдите в **Маркетинг > Email-кампании**
- Выберите сегмент клиентов в качестве аудитории
- Создайте целевое сообщение
- Отслеживайте эффективность кампании

### Программы лояльности

Вознаграждайте на основе сегментов:
- **Чемпионы**: Преимущества высшего уровня
- **Лояльные**: Стандартные награды
- **Под угрозой**: Бонусные баллы для повторного вовлечения

### Персонализация

Показывайте релевантный контент:
- Рекомендации товаров
- Предпочтения по категориям
- Таргетинг по ценовому диапазону

## Конфиденциальность и данные

### Данные клиентов

Spwig уважает конфиденциальность клиентов:
- Данные в отчётах анонимизированы
- Соответствие GDPR/CCPA
- Клиенты могут запросить удаление данных
- Безопасное хранение данных

### Возможность отказа

Клиенты могут отказаться от:
- Маркетинговых писем
- Аналитического отслеживания
- Персонализации

Уважайте предпочтения по отказу в кампаниях.

## Советы для успеха

- Просматривайте аналитику еженедельно
- Действуйте на основе инсайтов по сегментам
- Тестируйте различные кампании
- Отслеживайте метрики с течением времени
- Сравнивайте с отраслевыми ориентирами
