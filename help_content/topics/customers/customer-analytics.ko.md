# 고객 분석 이해

고객 분석을 통해 구매 행동을 이해하고 마케팅을 개선하세요.

## 분석 대시보드

**고객 > 분석**으로 이동하여 다음을 확인할 수 있습니다:

### 주요 지표

**총 고객 수**
- 활성 고객 수
- 이번 달 신규 고객
- 성장 추세

**고객 평생 가치 (CLV)**
- 고객당 평균 매출
- 가치별 상위 고객
- 시간에 따른 CLV 추세

**구매 빈도**
- 고객당 평균 주문 수
- 재구매율
- 구매 간격

### 고객 세그먼트

행동에 따라 고객은 자동으로 세그먼트로 분류됩니다:

**우수 고객**
- 높은 가치, 자주 구매하는 고객
- 최근 구매 내역
- 가장 우수한 고객

**충성 고객**
- 정기적으로 구매하는 고객
- 중간 이상의 가치
- 일관된 구매 내역

**위험 고객**
- 이전에는 활성이었으나
- 최근 구매가 없음
- 재참여가 필요함

**휴면 고객**
- 마지막 구매 후 오랜 시간 경과
- 최근 참여도 낮음
- 복구 캠페인 필요

**실패 고객**
- 오랜 시간 동안 구매 없음
- 참여도 매우 낮음
- 재활성화가 어려움

## RFM 분석

RFM (최근성, 빈도, 금액) 점수는 고객의 가치를 식별하는 데 도움을 줍니다:

### 최근성 점수
- 얼마나 최근에 구매했나요?
- 범위: 1 (오래 전) ~ 5 (매우 최근)

### 빈도 점수
- 얼마나 자주 구매하나요?
- 범위: 1 (드물게) ~ 5 (매우 자주)

### 금액 점수
- 얼마나 많이 소비하나요?
- 범위: 1 (낮은 가치) ~ 5 (높은 가치)

**RFM 점수 사용법:**
- 555 = 우수 고객 (가장 우수)
- 155 = 많은 소비자, 희귀 구매
- 511 = 자주 구매, 최근, 낮은 가치
- 111 = 실패 고객

## 고객 인사이트

### 구매 패턴

고객이 무엇을 구매하는지 분석하세요:
- 가장 인기 있는 제품
- 카테고리 선호도
- 평균 주문 금액
- 계절적 추세

### 행동 추적

고객의 행동을 모니터링하세요:
- 본 페이지
- 둘러본 제품
- 장바구니 포기율
- 이메일 열람률

### 지리적 분석

고객이 어디에 있는지 이해하세요:
- 상위 국가/지역
- 도시 수준 데이터
- 배송 선호도
- 통화 사용

## 분석 활용

### 타겟 마케팅

세그먼트를 기반으로 캠페인을 생성하세요:

**우수 고객:**
- 신제품에 대한 조기 접근
- VIP 혜택 및 보상
- 추천서/리뷰 요청

**위험 고객:**
- 복구 이메일 캠페인
- 특별 할인 제안
- 피드백 설문조사

**신규 고객:**
- 환영 시리즈 이메일
- 제품 추천
- 교육 콘텐츠

### 제품 전략

구매 데이터를 활용하여:
- 베스트셀러 확인
- 성능 하락 제품 파악
- 교차 판매 기회 찾기
- 재고 계획

### 수익 최적화

수익을 늘리기 위해:
- 고가치 세그먼트에 업셀링
- 중단된 고객 재참여
- 재구매율 개선
- 이탈률 감소

## 보고서

### 사용 가능한 보고서

**고객 확보**
- 기간별 신규 고객
- 확보 채널
- 확보 비용

**유지 분석**
- 재구매율
- 고객 이탈율
- 코호트 분석

**가치 보고서**
- 세그먼트별 CLV
- 평균 주문 금액 추세
- 고객당 수익

### 데이터 내보내기

분석 데이터를 내보내세요:
1. 날짜 범위 선택
2. 지표 선택
3. 형식 선택 (CSV, Excel, PDF)
4. 보고서 다운로드

## 마케팅 통합

### 이메일 캠페인

세그먼트를 기반으로 타겟 이메일을 생성하세요:
- **마케팅 > 이메일 캠페인** 이동
- 고객 세그먼트를 대상으로 선택
- 타겟 메시지 생성
- 캠페인 성과 추적

### 충성도 프로그램

세그먼트를 기반으로 보상:
- **우수 고객**: 최고 등급 혜택
- **충성 고객**: 표준 보상
- **위험 고객**: 재참여를 위한 보너스 포인트

### 개인화

적절한 콘텐츠 표시:
- 제품 추천
- 카테고리 선호도
- 가격대 타겟팅

## 개인정보 및 데이터

### 고객 데이터

Spwig은 고객의 개인정보를 존중합니다:
- 보고서에서 데이터는 익명화됨
- GDPR/CCPA 준수
- 고객은 데이터 삭제를 요청할 수 있음
- 보안된 데이터 저장

### 탈퇴 옵션

고객은 다음을 탈퇴할 수 있습니다:
- 마케팅 이메일
- 분석 추적
- 개인화

캠페인에서 탈퇴 선호도를 존중하세요.

- 지속적인 지표 모니터링
