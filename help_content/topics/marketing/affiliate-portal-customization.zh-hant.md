title: 聯營夥伴門戶自定義
---

Spwig 聯營夥伴門戶是面向潛在聯營夥伴的公開落地頁面，他們可以在此了解您的計劃並註冊。自定義此門戶可讓您將訊息、品牌和行動呼籲與您商店的獨特定位保持一致。一個設計良好的門戶可以吸引高質量的聯營夥伴，並將訪客轉化為活躍的夥伴。

## 什麼是聯營夥伴門戶？

聯營夥伴門戶可通過您商店的域名訪問 `/affiliate/`。它作為：

- **發現頁面** — 潛在聯營夥伴了解您的傭金結構、優勢和要求的地方
- **註冊入口點** — 新聯營夥伴的註冊表單（訪客註冊或基於賬戶）
- **登錄網關** — 現有聯營夥伴可以登錄以訪問他們的儀表板
- **品牌展示** — 反映您商店的身分和聯營夥伴計劃的價值主張

門戶可以通过聯營夥伴設置管理員進行完全自定義，包括英雄訊息、功能亮點、逐步流程和註冊選項。

![聯營夥伴門戶落地頁面](/static/core/admin/img/help/affiliate-portal-customization/portal-landing.webp)

## 獲取設置

導航到 **營銷 > 聯營夥伴計劃 > 門戶設置** 以自定義門戶。

聯營夥伴設置模型是一個 **單例** — 您的整個商店只有一個設置記錄。所有字段都可以使用 Spwig 的翻譯系統進行 **翻譯**，因此您可以為商店支持的每種語言自定義訊息。

## 英雄部分

英雄部分是潛在聯營夥伴看到的第一個部分。它包括：

- **標題** — 主要標頭（例如，"加入我們的聯營夥伴計劃"）
- **副標題** — 解釋計劃價值的支援文字（例如，"通過向您的受眾推廣高級產品來賺取傭金"）
- **統計數據** — 自動顯示的指標：
  - 總活躍計劃
  - 總活躍聯營夥伴
  - 平均傭金率（根據所有活躍計劃計算）
- **CTA 按鈕** — 自動生成：
  - **登錄** — 給現有聯營夥伴
  - **成為聯營夥伴** — 觸發註冊流程

### 自定義英雄訊息

| 欄位 | 示例值 | 目的 |
|-------|--------------|---------|
| **英雄標題** | "與我們合作並賺取" | 以利潤為導向的標題吸引注意 |
| **英雄副標題** | "加入 500+ 聯營夥伴，每筆您推薦的銷售都可獲得競爭性的傭金" | 提供社會證明並明確說明優惠 |

統計數據是 **自動計算** 的，並根據您活躍的計劃和聯營夥伴實時更新。您不能手動編輯這些值。

## 功能部分

功能部分突出顯示 **6 個可自定義的優勢卡片**，說明為何聯營夥伴應該加入您的計劃。每個功能卡片包含：

- **圖示** — FontAwesome 圖示類別（例如，`fa-dollar-sign`，`fa-chart-line`，`fa-headset`）
- **標題** — 優勢標頭（例如，"競爭性傭金"）
- **描述** — 1-2 句解釋（例如，"在您推薦的每筆銷售中賺取高達 15% 的傭金"）

### 預設功能

當您首次安裝聯營夥伴應用程式時，Spwig 會提供預設功能：

| 圖示 | 標題 | 描述 |
|------|-------|-------------|
| `fa-dollar-sign` | 競爭性傭金 | 在您推薦的每筆銷售中賺取豐厚的傭金 |
| `fa-link` | 簡易追蹤連結 | 獲取可在任何地方使用的獨特追蹤連結 |
| `fa-chart-line` | 實時分析 | 在您的儀表板上追蹤點擊、轉化和收入 |
| `fa-calendar-check` | 可靠的付款 | 及時通過 PayPal 或銀行轉賬付款 |
| `fa-headset` | 專業支持 | 我們的團隊將協助您成功 |
| `fa-gift` | 市場推廣素材 | 可訪問橫幅、圖片和推廣內容 |

### 自定義功能

功能以 **JSON 陣列** 的形式存儲在數據庫中。直接在管理表單中編輯它們：

```json
[
  {
    "icon": "fa-percent",
    "title": "高達 20% 傭金",
    "description": "在高級產品銷售中賺取行業領先的傭金"
  },
  {
    "icon": "fa-rocket",
    "title": "快速批准",
    "description": "24 小時內獲得批准並立即開始推廣"
  },
  {
    "icon": "fa-mobile-alt",
    "title": "行動儀表板",
    "description": "從任何設備管理您的連結並追蹤收入"
  }
]
```

**圖示參考**：使用任何 FontAwesome 5 Free 圖示類別。瀏覽圖示請訪問 [fontawesome.com/icons](https://fontawesome.com/icons) 並使用類別名稱（例如，`fa-trophy`，`fa-users`，`fa-star`）。

## 如何運作部分

"如何運作"部分顯示 **4 步視覺流程**，說明聯營夥伴的旅程。每一步包括：

- **標題** — 步驟名稱（例如，"註冊"）
- **描述** — 1-2 句解釋發生了什麼

### 預設步驟

| 步驟 | 標題 | 描述 |
|------|-------|-------------|
| 1 | 註冊 | 在幾分鐘內創建您的免費聯營夥伴賬戶 |
| 2 | 獲取您的連結 | 為任何產品或頁面生成唯一的追蹤連結 |
| 3 | 推廣 | 通過內容、社交媒體或電子郵件與您的受眾分享您的連結 |
| 4 | 獲取傭金 | 當顧客使用您的推薦連結購買時獲得付款 |

### 自定義步驟

步驟以 **JSON 陣列** 的形式存儲。您可以在管理員中編輯它們：

```json
[
  {
    "title": "申請加入",
    "description": "提交您的申請並告訴我們關於您的平台"
  },
  {
    "title": "獲得批准",
    "description": "我們的團隊在 24 小時內審核您的申請"
  },
  {
    "title": "創建連結",
    "description": "訪問您的儀表板並立即生成追蹤連結"
  },
  {
    "title": "開始賺錢",
    "description": "在您推薦的每筆銷售中賺取傭金 — 每月通過 PayPal 支付"
  }
]
```

視覺流程會自動為每個步驟編號（1, 2, 3, 4）在落地頁面上。

## CTA 部分

在註冊表單之前的最後一部分是 **行動呼籲（CTA）部分**。它提供最後一次推動以鼓勵註冊。

| 欄位 | 示例值 | 目的 |
|-------|--------------|---------|
| **CTA 標題** | "準備開始賺錢了嗎？" | 直接問題創造緊迫感 |
| **CTA 描述** | "今天加入我們的聯營夥伴計劃，開始在您已經喜愛和推薦的產品上賺取傭金。" | 強化優勢並消除摩擦 |

CTA 部分會自動在文字下方顯示 **成為聯營夥伴** 按鈕。

## 註冊設置

控制新聯營夥伴如何註冊以及他們提供哪些資訊。

### 自定義註冊表單

**欄位：** `custom_form`（外鍵到 FormBuilder 表單）

如果您使用 Spwig 的表單構建器構建了自定義註冊表單，請在此處選擇它。這允許您在註冊過程中收集額外資訊（例如，網站 URL、受眾規模、推廣渠道）。

**留空** 以使用預設的聯營夥伴註冊表單（電子郵件、密碼、付款細節）。

### 允許訪客註冊

**欄位：** `allow_guest_registration`（布爾值）

- **勾選** — 訪客可以在不先創建 Spwig 帳戶的情況下申請
- **未勾選** — 訪客必須登錄或創建客戶賬戶才能申請

**建議：** 啟用訪客註冊以減少摩擦。您始終可以要求批准以在激活聯營夥伴之前審核他們。

### 需要批准

**欄位：** `require_approval`（布爾值）

- **勾選** — 新聯營夥伴必須等待手動批准才能訪問他們的儀表板
- **未勾選** — 新聯營夥伴會自動批准並可以立即創建連結

**建議：** 如果您想審核聯營夥伴以進行品牌契合度、防止欺詐或專有計劃，請啟用手動批准。

### 條款與條件 URL

**欄位：** `terms_url`（URL）

聯營夥伴計劃條款與條件的可選連結。如果提供，註冊表單會顯示一個需要聯營夥伴在註冊前同意您條款的複選框。

**示例：** `/pages/affiliate-terms/`

### 歡迎訊息

**欄位：** `welcome_message`（文字）

註冊成功後立即顯示給聯營夥伴的訊息。使用此訊息來：

- 感謝他們加入
- 解釋下一步（例如，"我們會在 24 小時內審核您的申請"）
- 链接到入門資源

**示例：**
```
歡迎加入我們的聯營夥伴計劃！我們已經收到您的申請，並會在 24 小時內審核。請查看您的電子郵件以獲取批准確認和登錄說明。
```

## 多語言支持

聯營夥伴設置中的所有文字欄位都可以使用 Spwig 的翻譯小工具進行 **翻譯**：

- 英雄標題
- 英雄副標題
- 功能（按語言翻譯 JSON）
- 如何運作步驟（按語言翻譯 JSON）
- CTA 標題
- CTA 描述
- 歡迎訊息

### 翻譯如何運作

當您編輯可翻譯欄位時，您會看到一個翻譯小工具，讓您為每個啟用的語言提供內容。對於 JSON 欄位（功能、步驟），您為每個語言提供獨立的 JSON 物件：

**英文：**
```json
[
  {"icon": "fa-dollar-sign", "title": "Competitive Commissions", "description": "Earn up to 15% on every sale"}
]
```

**西班牙文：**
```json
[
  {"icon": "fa-dollar-sign", "title": "Comisiones Competitivas", "description": "Gana hasta el 15% en cada venta"}
]
```

門戶會根據訪客的語言偏好自動顯示正確的語言版本。

## 預覽您的更改

在自定義門戶設置後：

1. **保存** 您在管理員中的更改
2. 訪問您商店的前端 `/affiliate/`（在新標籤中打開）
3. **測試註冊流程** 通過點擊 "成為聯營夥伴"
4. **驗證品牌一致性** — 門戶是否與您商店的設計和訊息一致？

您可以進行迭代更改並刷新頁面以立即查看更新。

## 小技巧

- 自定義您的 **英雄標題** 以聚焦於利益，而不是功能 — "Earn While You Sleep" 比 "Affiliate Program Sign-Up" 更具吸引力
- 在副標題中使用 **社會證明**（例如，"加入 500+ 聯營夥伴"）以建立信任和可信度
- 選擇 **FontAwesome 圖示** 來視覺化強化每個利益 — 圖示應立即傳達價值
- 保持功能描述為 **1-2 句話** — 門戶是關於轉化，而不是詳盡的解釋
- 在推廣門戶之前 **測試註冊流程** — 捕捉摩擦點，如令人困惑的表單欄位或斷開的連結
- 啟用 **訪客註冊** 以減少註冊摩擦，然後使用 **需要批准** 在他們提交後審核聯營夥伴
- 使用 **歡迎訊息** 來設定期望（批准時間表、下一步、支援聯繫）並減少支援查詢
- **季節性更新門戶** 以與活動對齊 — 突出特別傭金促銷或產品發佈
