---
title: ブログ管理
---

ブログを使用して、記事、ガイド、ニュースを公開し、トラフィックを増やし、訪問者と交流できます。Spwigのブログには、豊富なテキストエディタ、予約公開、購読者への通知、ソーシャルメディアの自動共有、SEOツールが含まれています。

![ブログ投稿](/static/core/admin/img/help/blog-management/blog-post-list.webp)

## ブログ投稿の作成

**マーケティング > ブログ投稿**に移動し、**投稿を追加**をクリックしてください。

### 投稿内容

**CKEditor 5**の豊富なテキストエディタを使用して投稿を作成してください。サポートされている機能は以下の通りです：
- テキストの書式設定（見出し、太字、斜体、リスト、ブロック引用）
- 画像とメディア（メディアライブラリを通じてアップロード）
- 埋め込み動画（YouTube、Vimeo）
- テーブルとコードブロック
- 商品、カテゴリ、外部URLへのリンク

より複雑なレイアウトが必要な場合は、**Page Builder**のトグルを有効にして、ドラッグ＆ドロップのページビルダを使用してください。

### 投稿設定

| 設定 | 説明 |
|---------|-------------|
| **タイトル** | ブログおよび検索結果で表示される見出し |
| **スラッグ** | URLに友好的な識別子（タイトルから自動生成され、編集可能） |
| **抜粋** | ブログのリストカードやRSSフィードで表示される短い要約 |
| **特集画像** | 投稿の上部およびリストカードで表示されるメイン画像 |
| **カテゴリ** | 投稿の主要なカテゴリ |
| **タグ** | フィルタリングや関連コンテンツのためのキーワード |
| **著者** | 著者として記載されるスタッフ |
| **ステータス** | 下書き、予約、公開、またはアーカイブ |
| **特集** | ブログリストの上部に投稿をピン留め |

### SEO設定

各投稿にはSEOフィールドが含まれています：
- **メタタイトル** — 検索エンジンの結果用のカスタムタイトル（投稿タイトルがデフォルト）
- **メタ説明** — 検索エンジンの結果に表示される要約
- **Open Graph画像** — 投稿がソーシャルメディアで共有されるときに使用される画像

## 投稿ステータス

| ステータス | 説明 |
|--------|-------------|
| **下書き** | 作業中で、一般公開されていません |
| **予約** | 設定された日時で自動的に公開されます |
| **公開** | 現在ライブで、訪問者に公開されています |
| **アーカイブ** | ブログリストから非表示ですが、直接のURL経由でアクセス可能です |

### 投稿の予約

投稿を将来的に公開するには、以下の手順を行います：
1. ステータスを**予約**に設定します
2. **公開日時**を選びます
3. 投稿を保存します

バックグラウンドタスクが予約された時間に投稿を自動的に公開し、購読者への通知をトリガーします。

## カテゴリ

**マーケティング > ブログカテゴリ**に移動して、コンテンツを整理してください。

カテゴリは以下をサポートしています：
- **階層構造** — 親カテゴリと子カテゴリを作成できます（例："ガイド" > "Getting Started"）
- **カスタムURL** — 各カテゴリに独自のスラッグがあり、URLが整えられます
- **説明** — カテゴリアーカイブページに表示されるカテゴリ説明を追加できます
- **並び順** — ナビゲーションでカテゴリの表示順を制御できます

## タグ

タグはコンテンツを分類するための二次的な方法です。カテゴリ（階層構造）とは異なり、タグはフラットなラベルです。訪問者はタグをクリックして、そのタグを持つすべての投稿を表示できます。

## 購読者

**マーケティング > ブログ購読者**に移動して、購読者リストを管理してください。

### 購読の仕組み

1. 訪問者はブログのフォームを通じて購読します（電子メールアドレスが必要）
2. **二段階のオプトイン**確認メールが送信されます
3. 確認後、購読者は新規投稿が公開されたときに通知を受け取ります

### 通知の頻度

購読者は通知をどのくらいの頻度で受け取るか選択できます：

| 頻度 | 説明 |
|-----------|-------------|
| **即時** | 新規投稿が公開されたらすぐにメールが送信されます |
| **週次要約** | すべての新規投稿の週次要約 |
| **月次要約** | すべての新規投稿の月次要約 |

バックグラウンドタスクが要約の作成と配信を自動的に処理します。

### 購読者の管理

- 購読者数、確認状態、登録日を表示
- 外部の電子メールマーケティングツールで使用するために購読者リストをエクスポート
- 個々のアドレスを削除または購読解除
- 各通知メールには、1クリックで**購読解除**できるリンクが含まれています

## ソーシャルメディアの自動共有

Spwigは、投稿が公開されたときに、投稿をソーシャルメディアアカウントに自動的に共有できます。

### ソーシャルアカウントの接続

**マーケティング > ソーシャルコネクター**に移動して、アカウントを接続してください：

| プラットフォーム | 認証 |
|----------|---------------|
| **Facebook** | OAuth — Facebookページを接続 |
| **Instagram** | OAuth — ビジネスアカウントを接続 |
| **LinkedIn** | OAuth — 会社ページを接続 |

### 自動共有の仕組み

1. 1つ以上のソーシャルアカウントを接続します
2. 投稿を作成する際、接続された各アカウントに対して**自動共有**を有効にします
3. 共有メッセージをカスタマイズします（デフォルトは投稿タイトルと抜粋）
4. 投稿が公開される（または予約された日時が到達した）と、自動的に共有されます

予約投稿にも自動共有が適用されます — 投稿がライブになる同じ時間にソーシャル共有が送信されます。

## RSSフィード

ブログは自動的に`/blog/feed/`にRSSフィードを生成します。これにより、訪問者やアグリゲーターがコンテンツに登録できます。フィードには以下が含まれます：
- 投稿のタイトルと抜粋
- 公開日
- 著者情報
- 完全な投稿への直接リンク

## ブログ設定

**マーケティング > ブログ設定**に移動して、グローバルなブログオプションを構成してください：

- **1ページあたりの投稿数** — リスト表示で1ページあたりに表示される投稿数
- **コメントを許可** — 投稿へのコメントを有効または無効にします
- **デフォルトカテゴリ** — どのカテゴリも割り当てられていない投稿の代替カテゴリ
- **ソーシャル共有ボタン** — 個々の投稿ページに共有ボタンを表示

## ヒント

- **SEOを意識して**投稿を作成してください — 説明的なタイトルを使用し、メタ説明を記入し、コンテンツに自然に関連するキーワードを含めます。
- **予約公開**を使用して、手動作業なしで一貫した投稿頻度を維持してください。
- **自動共有**を有効にして、リーチを最大化してください — 投稿がソーシャルメディアに投稿された直後に最もエンゲージメントが得られます。
- 訪問者に**購読**を促すために、ブログにサブスクリプションフォームを際立たせ、説得力のあるコール・トゥ・アクションを使用してください。
- **カテゴリ**は広範なコンテンツのグループ化に、**タグ**は特定のトピックに使用してください — これにより、訪問者が関連するコンテンツを見つけることができます。
- すべての投稿に**特集画像**を追加してください — 画像が含まれた投稿は、検索結果やソーシャルメディア共有でより良いパフォーマンスを示します。
- **週次または月次の要約**オプションを、頻繁なメールを望まない購読者に提供してください — これにより、購読解除率が低下します。
