---
title: アフィリエイトプログラム概要
---

Spwigのアフィリエイトプログラム機能を使用すると、商品のプロモーションを引き換えに、パートナーを募集してコミッションを支払うことができます。このマーケティングチャネルは、インフルエンサー、ブロガー、コンテンツクリエイター、ブランドアンバサダーなどの影響力のある人物を通じて、あなたのリーチを拡大します。これらの人物は、独自のトラッキングリンクを自分のオーディエンスと共有します。アフィリエイトのリンクをクリックして購入すると、アフィリエイトはコミッションを獲得し、あなたは顧客を獲得します。

この概要では、アフィリエイトプログラムとは何か、誰向けか、そして商人が販売を促進するパートナーのネットワークを構築するためにどのように使用するかを説明します。

## Key Concepts

これらのコア用語を理解することで、アフィリエイトプログラムを構成および管理できます。

| Term | Definition |
|------|------------|
| **Affiliate** | あなたの商品をプロモーションし、紹介された販売に対してコミッションを獲得するパートナー |
| **Program** | レート、ルール、設定を含むコミッション構造（複数のプログラムを作成できます） |
| **Tracking Link** | アフィリエイトのコードを含むユニークなURL（例: `yourstore.com/?ref=CODE`） |
| **Commission** | 紹介された販売に対してアフィリエイトが獲得する支払い。これはプログラムのルールに基づいて計算されます |
| **Cookie Lifetime** | 顧客がアフィリエイトリンクをクリックした後、トラッキングクッキーが保持される日数 |
| **Payout** | 複数の承認済みコミッションを一度に決済するバルク支払い |
| **Merchant Dashboard** | プログラム、アフィリエイト、コミッション、支払いを管理するためのあなたの管理インターフェース |
| **Affiliate Portal** | アフィリエイトが収益を確認し、トラッキングリンクを取得し、支払いを要求するための公開用ダッシュボード |

## How It Works

アフィリエイトワークフローは4つの主要な段階に従います。

### 1. Apply
アフィリエイトは、あなたの店舗の`/affiliate/`で公開されているアフィリエイトポータルを通じて、プログラムを発見し、申請を提出します。オープンプログラムでは**自動承認**を有効にしたり、招待制のパートナーシップでは**手動のレビュー**を有効にしたりできます。

### 2. Approve
**マーケティング > アフィリエイト**で保留中の申請を確認します。各申請者のウェブサイト、ソーシャルメディアの存在、オーディエンスの適合性を確認してから承認してください。承認後、アフィリエイトはログイン資格情報を取得し、ダッシュボードにアクセスできます。

### 3. Promote
承認されたアフィリエイトは、自分のポータルからユニークな紹介リンクを取得します。彼らは、ブログ投稿、ソーシャルメディア、電子メールニュースレター、または自分のオーディエンスと接する場所でこれらのリンクを共有します。Spwigは、誰かがリンクをクリックしたときにトラッキングクッキーを設定します。

### 4. Earn
紹介された顧客がクッキーの有効期間内に購入を完了すると、Spwigはコミッションの記録を作成します。**マーケティング > コミッション**でコミッションを確認し、承認し、アフィリエイトが最小支払い閾値に達したときに支払いを処理します。

## Merchant Workflow Overview

商人として、あなたは管理パネルからプログラムライフサイクル全体を管理します。

### Creating Programs
まず、**マーケティング > アフィリエイトプログラム**で1つ以上のアフィリエイトプログラムを作成してください。各プログラムには独自のコミッション構造、クッキーの有効期間、承認設定があります。インフルエンサー向け（高額コミッション）と一般パートナー向け（低額コミッション）の別々のプログラムを作成する可能性があります。

### Reviewing Applications
新しいアフィリエイト申請は、**マーケティング > アフィリエイト**で**保留**状態で表示されます。各申請を確認し、パートナーがブランドに適しているかを確認してください。承認してアカウントを有効にするか、理由を付けて拒否してください。

### Approving Commissions
アフィリエイトが販売を生成すると、**マーケティング > コミッション**でコミッションが**保留**状態で表示されます。リンクされた注文を確認し、それが正当であることを確認してください（自己紹介でない、返品注文でない）。その後、承認または拒否してください。

### Processing Payouts
アフィリエイトが最小支払い閾値を超える承認済みコミッションを蓄積した後、**マーケティング > 支払い**でバルク支払いを処理します。SpwigはPayPalおよびAirwallexと統合されており、自動支払いが可能です。または、手動の銀行振込を記録することもできます。

## Affiliate Workflow Overview

アフィリエイトがどのようにあなたのプログラムを体験するかを理解することで、より良いオンボーディングとサポートを設計できます。

### Apply
アフィリエイトは、アフィリエイトポータルを訪問し、プログラムの詳細（コミッション率、クッキーの有効期間、支払い条件）を読み、連絡先情報とプロモーションチャネルを含む申請を提出します。

### Create Links
承認後、アフィリエイトはダッシュボードにログインしてトラッキングリンクを生成します。彼らは、一般的なストアリンクまたは特定の商品/カテゴリをプロモーションしたい場合のリンクを作成できます。

### Promote
アフィリエイトは、潜在的な顧客と接するあらゆる場所でトラッキングリンクを共有します — ブログ投稿、YouTube動画、Instagramストーリー、電子メールニュースレター、または比較サイト。

### Request Payouts
アフィリエイトは、アフィリエイトポータルのダッシュボードを通じてリアルタイムで収益を追跡します。承認された残高が最小支払い閾値に達すると、支払いを要求できます。

## Where to Find Each Feature

| Feature | Admin Location | Description |
|---------|---------------|-------------|
| **Programs** | Marketing > Affiliate Programs | Create and configure commission structures |
| **Affiliates** | Marketing > Affiliates | Review applications, manage affiliate accounts |
| **Commissions** | Marketing > Commissions | Review and approve pending commissions |
| **Payouts** | Marketing > Payouts | Process bulk payments to affiliates |
| **Settings** | Marketing > Affiliate Settings | Global settings, payout providers, portal customization |
| **Dashboard** | Marketing > Affiliate Dashboard | Analytics overview with clicks, orders, and commission totals |

アフィリエイト向けのポータルは、あなたの店舗の公開URLの`/affiliate/`で自動的に利用可能です。

## Common Use Cases

ここに示すのは、商人がSpwigのアフィリエイトプログラムを使ってビジネスを成長させるための4つの実証済み方法です。

### Influencer Partnerships
あなたのニッチに合った関与型のオーディエンスを持つソーシャルメディアインフルエンサーと提携します。質の高いインフルエンサーを引き寄せるために、15〜20％の高額コミッション率を提供してください。各提携のROIを測定するためにトラッキングリンクを使用してください。

### Brand Ambassadors
忠実な顧客のネットワークを構築し、ブランドの支持者になります。これらのリピーター顧客にアフィリエイトアカウントを提供し、友人や家族を紹介してコミッションを獲得できるようにしてください。これは、情熱的なコミュニティを持つニッチ製品に特に効果的です。

### Content Creators
購入ガイド、レビュー、比較コンテンツを作成するブロガー、ユーチューバー、ポッドキャスターを募集してください。永続的なコンテンツを持つアフィリエイトは、毎月一貫した紹介を生成できます。

### Referral Networks
既存の顧客がプログラムに参加して、好きな製品を共有することでコミッションを獲得できるようにしてください。これにより、満足した顧客がプロモーターとなり、新しい顧客を引き込み、彼らもアフィリエイトになる可能性があります。

## Tips

- **1つのプログラムから始める** — 10％のコミッション率と30日間のクッキー有効期間を持つ一般的なパートナー向けプログラムを作成してください。後で、どのパートナーが最も効果的であるかを理解した後、専門的なプログラムを追加できます。
- **明確な期待値を設定する** — 承認プロセス、コミッションのタイムライン、支払いスケジュールをアフィリエイトポータルに記録してください。透明性は信頼を築き、サポートリクエストを減らします。
- **詐欺の監視** — 自己紹介（アフィリエイトが自分のリンクから購入）、異常に高い返品率、または疑わしいクリックパターンなどの赤旗を注意深く確認してください。詐欺的なコミッションはすぐに拒否してください。
- **定期的なコミュニケーション** — アフィリエイトに月次の更新を送信し、プログラムのニュース、プロモーションカレンダーのハイライト、およびトップパフォーマーの認識を提供してください。積極的なコミュニケーションは、アフィリエイトを引き続きプロモーションに巻き込みます。
- **モバイル最適化** — 多くのアフィリエイトは、ソーシャルメディアでリンクを共有しており、そこからのクリックの大部分はモバイルデバイスから来ます。携帯電話でチェックアウトフローをテストして、紹介された顧客がスムーズな体験をすることを確認してください。
- **クリエイティブなアセットを提供する** — アフィリエイトが商品をプロモーションしやすくするために、バナー画像、製品写真、およびコンテンツで使用できる事前に書かれたコピーを提供してください。
