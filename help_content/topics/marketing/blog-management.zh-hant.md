---
title: 部落格管理
---

部落格讓您可以發佈文章、指南和新聞，以吸引流量並與您的受眾互動。Spwig 的部落格包含豐富的文本編輯器、預約發佈、訂閱者通知、自動社交媒體分享和 SEO 工具。

![部落格文章](/static/core/admin/img/help/blog-management/blog-post-list.webp)

## 建立部落格文章

導航至 **行銷 > 部落格文章** 並點擊 **新增文章**。

### 文章內容

使用 **CKEditor 5** 豐富文本編輯器撰寫您的文章，它支援：
- 文字格式（標題、粗體、斜體、清單、引用）
- 圖像和媒體（透過媒體庫上傳）
- 嵌入式影片（YouTube、Vimeo）
- 表格和程式碼塊
- 連結到產品、分類和外部 URL

如需更複雜的版面設計，啟用 **頁面建構器** 切換鈕，使用拖放式頁面建構器，而不是文字編輯器。

### 文章設定

| 設定 | 說明 |
|---------|-------------|
| **標題** | 在部落格和搜尋結果中顯示的標頭 |
| **Slug** | URL 友好的識別碼（從標題自動產生，可編輯） |
| **摘要** | 在部落格列表卡片和 RSS 資訊源中顯示的簡短摘要 |
| **特色圖片** | 在文章頂部和列表卡片中顯示的主要圖片 |
| **分類** | 文章的主要分類 |
| **標籤** | 用於過濾和相關內容的關鍵字 |
| **作者** | 被記為作者的員工 |
| **狀態** | 草稿、預約、已發佈或已歸檔 |
| **特色** | 將文章固定在部落格列表的頂部 |

### SEO 設定

每篇文章都包含 SEO 欄位：
- **Meta 標題** — 搜尋引擎結果的自定義標題（預設為文章標題）
- **Meta 說明** — 在搜尋引擎結果中顯示的摘要
- **Open Graph 圖片** — 當文章在社交媒體上分享時使用的圖片

## 文章狀態

| 狀態 | 說明 |
|--------|-------------|
| **草稿** | 進行中的工作，不對公眾顯示 |
| **預約** | 將在設定的日期和時間自動發佈 |
| **已發佈** | 已上線，對訪客可見 |
| **已歸檔** | 從部落格列表中隱藏，但仍然可以透過直接 URL 存取 |

### 預約文章

要預約文章以供未來發佈：
1. 將狀態設為 **預約**
2. 選擇 **發佈日期和時間**
3. 儲存文章

背景任務會在預約時間自動發佈文章並觸發訂閱者通知。

## 分類

導航至 **行銷 > 部落格分類** 來整理您的內容。

分類支援：
- **階層結構** — 建立父類別和子類別（例如，「指南」>「入門」）
- **自訂 URL** — 每個分類都有自己的 slug 以實現乾淨的 URL
- **描述** — 在分類存檔頁面上顯示的分類描述
- **排序** — 控制導航中分類的顯示順序

## 標籤

標籤提供一種分類內容的次要方式。與分類（具有階層結構）不同，標籤是扁平的標籤。訪客可以點擊標籤來查看所有具有該標籤的文章。

## 訂閱者

導航至 **行銷 > 部落格訂閱者** 來管理您的訂閱者名單。

### 訂閱機制如何運作

1. 訪客透過部落格上的表單訂閱（需要電子郵件地址）
2. 會發送一封 **雙重確認** 的確認電子郵件
3. 一旦確認，訂閱者會在新文章發佈時收到通知

### 通知頻率

訂閱者選擇他們接收通知的頻率：

| 頻率 | 說明 |
|-----------|-------------|
| **立即** | 新文章一發佈就發送電子郵件 |
| **每週摘要** | 所有新文章的每週摘要 |
| **每月摘要** | 所有新文章的每月摘要 |

背景任務會自動處理摘要的編製和發送。

### 管理訂閱者

- 查看訂閱者數量、確認狀態和註冊日期
- 將訂閱者名單匯出以供在外部電子郵件營銷工具中使用
- 移除或取消訂閱個別電子郵件地址
- 每封通知電子郵件都包含一個點擊即可 **取消訂閱** 的連結

## 社交媒體自動分享

Spwig 可以在文章發佈時自動將新文章分享到您的社交媒體帳戶。

### 連接社交帳號

導航至 **行銷 > 社交連接器** 來連接您的帳戶：

| 平台 | 認證 |
|----------|---------------|
| **Facebook** | OAuth — 連接您的 Facebook 粉絲專頁 |
| **Instagram** | OAuth — 連接您的商業帳號 |
| **LinkedIn** | OAuth — 連接您的公司頁面 |

### 自動分享是如何運作的

1. 連接一個或多個社交帳號
2. 當建立文章時，為每個連接的帳號啟用 **自動分享**
3. 自定義分享訊息（預設為文章標題和摘要）
4. 當文章發佈（或到達預約時間）時，會自動分享

自動分享也適用於預約文章 — 社交分享會在文章上線的同時發送。

## RSS 資訊源

部落格會自動在 `/blog/feed/` 產生一個 RSS 資訊源。這允許訪客和聚合器訂閱您的內容。資訊源包含：
- 文章標題和摘要
- 發佈日期
- 作者資訊
- 完整文章的直接連結

## 部落格設定

導航至 **行銷 > 部落格設定** 來設定全域部落格選項：

- **每頁文章數** — 列表中每頁顯示的文章數量
- **允許評論** — 啟用或停用文章的評論功能
- **預設分類** — 無指定分類的文章的預設分類
- **社交分享按鈕** — 在個別文章頁面上顯示分享按鈕

## 小技巧

- 以 **SEO 為導向** 寫文章 — 使用描述性的標題，填寫 meta 說明，並在內容中自然地包含相關的關鍵字。
- 使用 **預約發佈** 來維持一致的發文節奏，而無需手動操作。
- 啟用 **自動分享** 以最大化觸及範圍 — 在發佈後不久於社交媒體分享的文章能獲得最多的參與度。
- 鼓勵訪客 **訂閱**，透過在部落格上顯眼地放置訂閱表單，並使用有說服力的行動呼籲。
- 使用 **分類** 進行廣泛的內容分組，並使用 **標籤** 進行特定主題 — 這有助於訪客找到相關內容。
- 為每篇文章添加 **特色圖片** — 包含圖片的文章在搜尋結果和社交媒體分享中表現更好。
- 對於不想頻繁收到電子郵件的訂閱者，使用 **每週或每月摘要** 選項 — 這能降低取消訂閱的比例。
