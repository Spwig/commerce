title: 联盟门户定制
---

Spwig 联盟门户是面向公众的落地页面，潜在联盟成员可以在这里了解您的计划并注册。定制此门户可以让您将信息、品牌和行动呼吁与您店铺的独特定位保持一致。设计良好的门户可以吸引高质量的联盟成员，并将访客转化为活跃的合作伙伴。

## 什么是联盟门户？

联盟门户可通过您的店铺域名的 `/affiliate/` 访问。它具有以下功能：

- **发现页面** — 潜在联盟成员可以了解您的佣金结构、优势和要求
- **注册入口** — 新联盟成员的注册表单（访客注册或基于账户的注册）
- **登录网关** — 现有联盟成员可以登录以访问他们的仪表板
- **品牌展示** — 反映您店铺的身份和联盟计划的价值主张

门户可通过联盟设置管理界面进行完全定制，包括英雄信息、功能亮点、逐步流程和注册选项。

![联盟门户落地页面](/static/core/admin/img/help/affiliate-portal-customization/portal-landing.webp)

## 访问设置

导航到 **营销 > 联盟计划 > 门户设置** 以自定义门户。

联盟设置模型是一个 **单例** —— 您的整个店铺只有一个设置记录。所有字段都可以使用 Spwig 的翻译系统进行 **翻译**，因此您可以为店铺支持的每种语言自定义信息。

## 英雄部分

英雄部分是潜在联盟成员看到的第一部分。它包括：

- **标题** — 主要标题（例如，"加入我们的联盟计划"）
- **副标题** — 解释计划价值的支持文本（例如，"通过向您的受众推广优质产品来赚取佣金"）
- **统计数据** — 自动显示的指标：
  - 总活跃计划数
  - 总活跃联盟成员数
  - 平均佣金率（根据所有活跃计划计算）
- **CTA 按钮** — 自动生成：
  - **登录** — 用于现有联盟成员
  - **成为联盟成员** — 触发注册流程

### 自定义英雄信息

| 字段 | 示例值 | 目的 |
|-------|--------------|---------|
| **英雄标题** | "与我们合作并赚取" | 用以利益为中心的标题吸引注意 |
| **英雄副标题** | "加入 500+ 联盟成员，通过您推荐的每笔销售赚取有竞争力的佣金" | 提供社会证明并明确说明优惠 |

统计数据是 **自动计算** 的，并根据您活跃的计划和联盟成员实时更新。您不能手动编辑这些值。

## 功能部分

功能部分突出显示 **6 个可自定义的益处卡片**，解释为什么联盟成员应该加入您的计划。每个功能卡片包含：

- **图标** — FontAwesome 图标类（例如，`fa-dollar-sign`, `fa-chart-line`, `fa-headset`）
- **标题** — 利益标题（例如，"有竞争力的佣金"）
- **描述** — 1-2 句解释（例如，"在您推荐的每笔销售中赚取高达 15% 的佣金"）

### 默认功能

安装联盟应用时，Spwig 会提供默认功能：

| 图标 | 标题 | 描述 |
|------|-------|-------------|
| `fa-dollar-sign` | 有竞争力的佣金 | 在您推荐的每笔销售中赚取丰厚的佣金 |
| `fa-link` | 易于跟踪链接 | 获取可在任何地方使用的唯一跟踪链接 |
| `fa-chart-line` | 实时分析 | 在您的仪表板上跟踪点击、转化和收入 |
| `fa-calendar-check` | 可靠的付款 | 通过 PayPal 或银行转账按时付款 |
| `fa-headset` | 专用支持 | 我们的团队将协助您成功 |
| `fa-gift` | 营销材料 | 访问横幅、图片和促销内容 |

### 自定义功能

功能以 **JSON 数组** 的形式存储在数据库中。直接在管理表单中编辑它们：

```json
[
  {
    "icon": "fa-percent",
    "title": "高达 20% 的佣金",
    "description": "在优质产品销售中赚取行业领先的佣金"
  },
  {
    "icon": "fa-rocket",
    "title": "快速批准",
    "description": "24 小时内获得批准并立即开始推广"
  },
  {
    "icon": "fa-mobile-alt",
    "title": "移动仪表板",
    "description": "从任何设备管理您的链接并跟踪收入"
  }
]
```

**图标参考：** 使用任何 FontAwesome 5 Free 图标类。浏览图标请访问 [fontawesome.com/icons](https://fontawesome.com/icons) 并使用类名（例如，`fa-trophy`, `fa-users`, `fa-star`）。

## 如何运作部分

"如何运作"部分显示一个 **4 步视觉流程**，解释联盟成员的旅程。每一步包括：

- **标题** — 步骤名称（例如，"注册"）
- **描述** — 1-2 句解释发生什么的描述

### 默认步骤

| 步骤 | 标题 | 描述 |
|------|-------|-------------|
| 1 | 注册 | 在几分钟内创建您的免费联盟账户 |
| 2 | 获取您的链接 | 为任何产品或页面生成唯一的跟踪链接 |
| 3 | 推广 | 通过内容、社交媒体或电子邮件与您的受众分享链接 |
| 4 | 赚取佣金 | 当客户使用您的推荐链接购买时获得付款 |

### 自定义步骤

步骤以 **JSON 数组** 的形式存储。您可以在管理界面中编辑它们：

```json
[
  {
    "title": "申请加入",
    "description": "提交您的申请并告诉我们有关您的平台的信息"
  },
  {
    "title": "获得批准",
    "description": "我们的团队将在 24 小时内审核您的申请"
  },
  {
    "title": "创建链接",
    "description": "访问您的仪表板并立即生成跟踪链接"
  },
  {
    "title": "开始赚取",
    "description": "通过您推荐的每笔销售赚取佣金 — 每月通过 PayPal 支付"
  }
]
```

视觉流程会自动在落地页面上为每个步骤编号（1, 2, 3, 4）。

## CTA 部分

注册表单之前的最后部分是 **行动号召（CTA）部分**。它提供最后一次推动以鼓励注册。

| 字段 | 示例值 | 目的 |
|-------|--------------|---------|
| **CTA 标题** | "准备好开始赚取了吗？" | 直接问题创造紧迫感 |
| **CTA 描述** | "今天加入我们的联盟计划，开始赚取您已经喜爱并推荐的产品的佣金。" | 强化利益并消除摩擦 |

CTA 部分会自动在文本下方显示 **成为联盟成员** 按钮。

## 注册设置

控制新联盟成员如何注册以及他们需要提供哪些信息。

### 自定义注册表单

**字段：** `custom_form` (指向 FormBuilder 表单的外键)

如果您使用 Spwig 的表单构建器构建了自定义注册表单，请在此处选择它。这允许您在注册过程中收集额外信息（例如，网站 URL、受众规模、推广渠道）。

**留空** 以使用默认的联盟注册表单（电子邮件、密码、支付详情）。

### 允许访客注册

**字段：** `allow_guest_registration` (布尔值)

- **勾选** — 访客可以在不先创建 Spwig 账户的情况下申请
- **未勾选** — 访客必须登录或创建客户账户后才能申请

**建议：** 启用访客注册以减少摩擦。您始终可以要求批准以在激活联盟成员之前审核他们。

### 需要批准

**字段：** `require_approval` (布尔值)

- **勾选** — 新联盟成员必须等待手动批准才能访问他们的仪表板
- **未勾选** — 新联盟成员会自动批准并可以立即创建链接

**建议：** 如果您想审核联盟成员以确保品牌契合度、防止欺诈或独家计划，请启用手动批准。

### 条款与条件 URL

**字段：** `terms_url` (URL)

指向您的联盟计划条款和条件的可选链接。如果提供，注册表单会显示一个复选框，要求联盟成员在注册前同意您的条款。

**示例：** `/pages/affiliate-terms/`

### 欢迎信息

**字段：** `welcome_message` (文本)

注册成功后立即显示给联盟成员的信息。使用此信息：

- 感谢他们加入
- 解释下一步（例如，"我们将在 24 小时内审核您的申请"）
- 链接到入门资源

**示例：**
```
欢迎加入我们的联盟计划！我们已收到您的申请，将在 24 小时内进行审核。查看电子邮件以获取批准确认和登录说明。
```

## 多语言支持

联盟设置中的所有文本字段都可以使用 Spwig 的翻译小部件进行 **翻译**：

- 英雄标题
- 英雄副标题
- 功能（按语言翻译的 JSON）
- 如何运作步骤（按语言翻译的 JSON）
- CTA 标题
- CTA 描述
- 欢迎信息

### 翻译如何工作

当您编辑可翻译字段时，您会看到一个翻译小部件，允许您为每个启用的语言提供内容。对于 JSON 字段（功能、步骤），您需要为每种语言提供单独的 JSON 对象：

**英语：**
```json
[
  {"icon": "fa-dollar-sign", "title": "Competitive Commissions", "description": "Earn up to 15% on every sale"}
]
```

**西班牙语：**
```json
[
  {"icon": "fa-dollar-sign", "title": "Comisiones Competitivas", "description": "Gana hasta el 15% en cada venta"}
]
```

门户会根据访客的语言偏好自动显示正确的语言版本。

## 预览您的更改

在自定义门户设置后：

1. **在管理界面中保存** 您的更改
2. 访问您的店铺前端的 `/affiliate/`（在新标签页中打开）
3. **测试注册流程** 通过点击 "成为联盟成员"
4. **验证品牌一致性** — 门户是否与您店铺的设计和信息一致？

您可以进行迭代更改并刷新页面以立即查看更新。

## 小贴士

- 自定义您的 **英雄标题** 以关注利益，而不是功能 — "睡着也能赚钱" 比 "联盟计划注册" 更有吸引力
- 在副标题中使用 **社会证明**（例如，"加入 500+ 联盟成员"）以建立信任和可信度
- 选择 **FontAwesome 图标**，这些图标能直观地强化每个利益 — 图标应立即传达价值
- 保持功能描述为 **1-2 句话** — 门户是关于转化，而不是详尽的解释
- 在推广门户之前 **亲自测试注册流程** — 捕获摩擦点，如令人困惑的表单字段或损坏的链接
- 启用 **访客注册** 以减少注册摩擦，然后使用 **需要批准** 在他们提交后审核联盟成员
- 使用 **欢迎信息** 来设定预期（批准时间表、下一步、支持联系方式）并减少支持查询
- **定期更新门户** 以与活动保持一致 — 突出显示特别的佣金促销或产品发布
