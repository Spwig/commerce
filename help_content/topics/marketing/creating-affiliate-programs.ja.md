---
title: アフィリエイトプログラムの作成
---

アフィリエイトプログラムは、パートナーが顧客をあなたのストアに紹介して売上をもたらした場合に、彼らがどれくらいの手数料を獲得するかを定義します。各プログラムには独自の手数料構造、トラッキングルール、支払いの閾値があります。異なるアフィリエイトセグメント（インフルエンサー、コンテンツクリエイター、または大量の紹介パートナーなど）のために複数のプログラムを作成できます。

![Programs List](/static/core/admin/img/help/creating-affiliate-programs/programs-list.webp)

## プログラムコンポーネント

すべてのアフィリエイトプログラムは次の要素から構成されます：

- **名前と説明** — プログラムを識別し、アフィリエイトに説明します
- **手数料構造** — 売上に対してアフィリエイトがどれくらい獲得するか（パーセンテージまたは固定金額）
- **クッキーの有効期間** — クリック後、紹介トラッキングがどのくらい続くか（1〜365日）
- **自動承認** — 新しいアフィリエイトが自動的に参加するか、手動の確認が必要か
- **最低支払い閾値** — アフィリエイトが支払いを要求する前に獲得する必要がある金額
- **ステータス** — 有効、一時停止、またはアーカイブ

## 手数料タイプ

プログラムを作成する際、2つの手数料モデルから選択できます：

| タイプ | 仕組み | 用途 | 計算例 |
|--------|--------|--------|--------|
| **パーセンテージ** | アフィリエイトは注文のサブトータルのパーセンテージを獲得 | 注文価値に応じて拡張可能な報酬 | $150 注文の 10% = $15 手数料 |
| **固定金額** | アフィリエイトは売上ごとに固定金額を獲得 | 予測可能なコスト；高ボリューム、低マージン製品に最適 | 注文価値に関係なく売上ごとに $25 |

**パーセンテージ手数料** は自然に拡張されます — アフィリエイトが高価値顧客を紹介すると、より多くの手数料を獲得できます。これは、アフィリエイトのインセンティブとあなたの利益を一致させ、最も一般的なモデル（通常 5〜15%）です。

**固定手数料** は、サービス、サブスクリプション、または大量紹介プログラムで、予測可能な売上コストを望む場合に適しています。これは理解しやすく予算管理に適していますが、大規模な注文をもたらすアフィリエイトを十分に報酬しない可能性があります。

## プログラムの作成

**マーケティング > アフィリエイトプログラム** に移動し、**+ プログラムの追加** をクリックします。

### ステップバイステップの設定

1. **プログラム名**
   アフィリエイトが見える説明的な名前を入力します（例："Partner Program" または "Influencer Tier"）。

2. **スラッグ**
   名前から自動生成された URL 友好的な識別子。URL および内部参照に使用されます。必要に応じてカスタマイズできます。

3. **説明**
   プログラムの利点と条件を説明するオプションのテキスト。アフィリエイトは、参加できるプログラムを確認する際にこれを表示します。

4. **手数料タイプ**
   **パーセンテージ** または **固定金額** を選択します。

5. **手数料値**
   - パーセンテージの場合：0 から 100 の値を入力します（例：10% は `10`）
   - 固定金額の場合：売上ごとのドル額を入力します（例：$25 は `25.00`）

6. **クッキー有効期間（日数）**
   トレーキングクッキーがどのくらい持続するか（1〜365日）。下記のセクションでガイドラインを参照してください。

7. **アフィリエイトの自動承認**
   - **チェック済み** — 新しいアフィリエイトは自動的に参加します
   - **チェックなし** — 各申請を手動で確認し、承認します

8. **最低支払い**
   アフィリエイトが支払いを要求する前に獲得する必要がある最低残高（例：$50 は `50.00`）。

9. **ステータス**
   **有効** に設定して、新しいアフィリエイトを受け入れ、紹介をトラッキングします。

10. **プログラムを保存**します。

## クッキー有効期間の説明

クッキー有効期間は、Spwig が顧客がアフィリエイトの紹介リンクをクリックしたことをどのくらい記憶するかを決定します。

### 仕組み

1. 顧客がアフィリエイトのリンクをクリックします
2. Spwig は顧客のブラウザにトラッキングクッキーを設定します
3. 顧客がクッキー有効期間内に購入を完了した場合、注文はアフィリエイトに帰属されます
4. クッキーが購入前に期限切れになった場合、アフィリエイトは手数料を獲得しません

### 持続時間の選択

| 持続時間 | 用途 | 一般的なシナリオ |
|----------|----------|------------------|
| **1〜7日** | インパルス購入、フラッシュセール | ファストモビングコンシューマーグッズ、限定オファー |
| **30日** | 標準的な電子商取引 | 一般的なオンライン小売、デフォルトの推奨 |
| **60〜90日** | 考えられる購入 | 高価な商品、B2B、サービス |
| **180日以上** | 長い販売サイクル | エンタープライズソフトウェア、サブスクリプション、高級品 |

**業界標準は30日です。** これは、アフィリエイトへの公平な帰属と実用的なトラッキング制限のバランスを取っています。短い有効期間は、迅速に変換する顧客を重視し、長い有効期間は顧客が調査し、再び購入を完了できる時間を与えます。

### 技術的な注意点

クッキー有効期間は**帰属**にのみ影響します。承認された手数料は無期限で有効です — クッキー有効期間は、注文が最初にアフィリエイトに帰属されるかどうかを決定するだけです。

## 自動承認設定

自動承認設定は、新しいアフィリエイト申請が手動の確認を必要とするかどうかを制御します。

### 自動承認を有効にする場合

- **公開プログラム** — アフィリエイトベースを迅速に拡大したいが、ボトルネックを避けたい
- **低リスク製品** — 欺骗やブランドリスクは最小限
- **高ボリュームプログラム** — 複数の申請が予想され、手動で確認できない

### 手動確認を必要とする場合

- **招待制プログラム** — 事前に選考されたパートナーのみを受け入れる
- **プレミアムプログラム** — 高い手数料率または独占的な特典
- **ブランドに敏感な製品** — アフィリエイトがブランドの価値観と一致していることを確認する必要がある
- **詐欺防止** — 懸念のあるアカウントをスクリーニングしたい

### セキュリティ上の考慮事項

アフィリエイトの手動確認は次のことを防止します：
- 自己紹介スキーム（アフィリエイトが偽のアカウントを作成して手数料を獲得する）
- 商標侵害（アフィリエイトが有料検索でブランド用語を競標する）
- ブランドの不一致（アフィリエイトが不適切な文脈で製品を宣伝する）

ほとんどのストアでは、**手動承認**から始めるのが安全です。信頼パターンを確立した後、いつでも自動承認を有効にできます。

## 最低支払い閾値

最低支払い閾値は、多くの小さな支払いを処理するための管理的負担を防ぎます。

### 閾値を設定する理由

- **取引手数料の削減** — 支払い処理業者は取引ごとに手数料を請求するため、支払いをバッチ処理することで費用を節約できます
- **会計の簡素化** — 支払いイベントが少ないほど、調整作業が少なくなります
- **業界標準** — 多くのアフィリエイトプログラムには最低額（$25〜$100）があります

### 一般的な閾値

| 閾値 | 用途 |
|-------|--------|
| **$25〜$50** | 高ボリュームプログラムでアフィリエイトが迅速に閾値に達する場合 |
| **$50〜$100** | 多くのプログラムの標準的な閾値 |
| **$100〜$200** | プレミアムプログラムまたは国際的な支払い処理手数料が高い場合 |

### アフィリエイトの満足度のバランス

閾値を**高すぎる**と、アフィリエイトが最初の支払いを受けるために数か月待つことになり、不満を引き起こします。閾値を**低すぎる**と、管理的負担が増加し、手数料が利益に影響を与えます。

**推奨:** $50 から始めます。これは、アクティブなアフィリエイトが最初の数回の販売で到達できるほど低く、支払いを効率的にバッチ処理できるほど高めです。

### 最大値はなし

最大残高は存在しません — アフィリエイトは支払いを要求する前に収益を無限に蓄積できます。一部のアフィリエイトは、税金の計画のために毎年または毎四半期ごとに支払いをバッチ処理することを好む場合があります。

## プログラムステータス管理

プログラムは次の3つのステータスのいずれかになります：

| ステータス | 説明 | 行動 |
|--------|-------------|----------|
| **有効** | プログラムが稼働中 | 新しいアフィリエイトを受け入れ、紹介をトラッキングし、手数料を計算 |
| **一時停止** | 一時的に無効 | 既存のアフィリエイトは維持されますが、新規登録はなし；既存の紹介クッキーは依然として機能 |
| **アーカイブ** | 永久に閉鎖 | 新しいアフィリエイトはなし、新しい紹介はトラッキングされません；履歴データはレポート用に保持 |

### プログラムを一時停止する場合

- コミッション率や条件を変更している
- 本四半期のアフィリエイト支払いが予算オーバーしている
- 新しいプログラム構造をテストしており、古いプログラムに新しいアフィリエイトが参加しないようにしたい

一時停止されたプログラムは、既存のトラッキングクッキーと未処理の手数料を依然として尊重します — 新しいアフィリエイトの参加を防止するだけです。

### プログラムをアーカイブする場合

- プログラムを新しい構造に置き換えている
- プログラムは時間限定（例：季節キャンペーン）
- 複数のプログラムを1つに統合している

アーカイブされたプログラムは、履歴レポートのためにデータベースに保持されますが、アクティブな管理ビューから削除されます。

## 例示プログラム

### 例1：インフルエンサープログラム（パーセンテージ）

| フィールド | 値 |
|-------|-------|
| 名前 | Influencer Program |
| 手数料タイプ | パーセンテージ |
| 手数料値 | 10 |
| クッキー有効期間（日数） | 30 |
| 自動承認 | 未チェック（手動確認） |
| 最低支払い | 50.00 |
| ステータス | 有効 |

**用途:** ソーシャルメディアインフルエンサーやコンテンツクリエイターを募集します。10%の手数料は注文価値に応じて拡張され、高額な顧客を引きつけるアフィリエイトに報酬します。手動承認により、インフルエンサーの視聴者層とブランドとの一致を確認できます。

### 例2：大量紹介プログラム（固定）

| フィールド | 値 |
|-------|-------|
| 名前 | Referral Partner Program |
| 手数料タイプ | 固定金額 |
| 手数料値 | 25.00 |
| クッキー有効期間（日数） | 7 |
| 自動承認 | チェック済み |
| 最低支払い | 100.00 |
| ステータス | 有効 |

**用途:** ディールサイト、クーポン集約サイト、紹介ネットワークと提携し、高ボリュームをもたらします。$25の固定手数料はコストを予測可能にし、7日間のクッキー有効期間は迅速な変換をターゲットにします。これらのパートナーは通常セルフサービスなので、自動承認を有効にしています。

### 例3：プレミアムパートナー（高パーセンテージ）

| フィールド | 値 |
|-------|-------|
| 名前 | Premium Partner Tier |
| 手数料タイプ | パーセンテージ |
| 手数料値 | 15 |
| クッキー有効期間（日数） | 90 |
| 自動承認 | 未チェック |
| 最低支払い | 200.00 |
| ステータス | 有効 |

**用途:** 頂上パフォーマンスのアフィリエイトや戦略的パートナー向けのエクスクルーシブプログラムです。15%の高めの手数料は、質の高いトラフィックをもたらす彼らを報酬し、90日間のクッキー有効期間は、より長い検討サイクルをカバーします。招待制のティアなので、手動承認のみです。

## ヒント

- ほとんどのプログラムでは、**パーセンテージ手数料**（5〜15%）から始めることをおすすめします — アフィリエイトに説明しやすく、注文価値に応じて自然に拡張されます。
- **30日間のクッキー有効期間**を基準とすることをおすすめします — これは業界標準であり、公平な帰属と実用的なトラッキング制限のバランスを取っています。
- 最初は**手動承認**を有効にしてアフィリエイトを確認し、信頼パターンと詐欺防止の制御を確立した後、**自動承認**に切り替えることが可能です。
- **最低支払い**を$50〜$100に設定して、アフィリエイトの満足度（到達しすぎない）と管理効率（あまりにも多くの小額支払いを避ける）のバランスを取ることが可能です。
- 異なるアフィリエイトセグメント（インフルエンサー、コンテンツサイト、ディール集約サイトなど）のために**別々のプログラム**を作成して、パフォーマンスをトラッキングし、手数料を独立して調整することが可能です。
- **アナリティクスダッシュボード**を定期的に監視して、高パフォーマンスのアフィリエイトを特定し、トップパートナーを維持するために手数料率を調整してください。
