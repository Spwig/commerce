title: 소속사 포털 커스터마이징

Spwig 소속사 포털은 잠재적 소속사가 프로그램을 알아보고 가입할 수 있는 공개용 랜딩 페이지입니다. 이 포털을 커스터마이징하면 메시지, 브랜딩, 콜투액션을 가게의 고유한 포지셔닝과 맞출 수 있습니다. 잘 설계된 포털은 고급 소속사를 끌어들이고 방문자를 활동적인 파트너로 전환합니다.

## 소속사 포털이란 무엇인가요?

소속사 포털은 가게 도메인의 `/affiliate/` 주소에서 접근할 수 있습니다. 이 포털은 다음과 같은 기능을 제공합니다:

- **발견 페이지** — 잠재적 소속사가 수수료 구조, 혜택, 요구사항을 알아볼 수 있는 페이지
- **등록 시작점** — 새로운 소속사의 등록 양식 (게스트 등록 또는 계정 기반)
- **로그인 게이트웨이** — 기존 소속사가 대시보드에 로그인할 수 있는 경로
- **브랜드 전시** — 가게의 정체성과 소속사 프로그램의 가치 제안을 반영합니다

포털은 소속사 설정 관리자를 통해 완전히 커스터마이징할 수 있으며, 히어로 메시지, 기능 강조, 단계별 흐름, 등록 옵션 등을 포함합니다.

![소속사 포털 랜딩 페이지](/static/core/admin/img/help/affiliate-portal-customization/portal-landing.webp)

## 설정에 접근하기

**마케팅 > 소속사 프로그램 > 포털 설정**으로 이동하여 포털을 커스터마이징합니다.

소속사 설정 모델은 **싱글톤**입니다 — 전체 가게에 대해 정확히 하나의 설정 기록이 있습니다. 모든 필드는 Spwig의 번역 시스템을 사용하여 **번역 가능**합니다. 따라서 가게가 지원하는 각 언어에 대해 메시지를 커스터마이징할 수 있습니다.

## 히어로 섹션

히어로 섹션은 잠재적 소속사가 처음 보는 부분입니다. 이 섹션에는 다음과 같은 항목이 포함됩니다:

- **제목** — 주요 헤드라인 (예: "우리의 소속사 프로그램에 참여하세요")
- **부제** — 프로그램의 가치를 설명하는 보조 텍스트 (예: "귀하의 대상자에게 프리미엄 제품을 홍보하여 수수료를 얻으세요")
- **통계** — 자동으로 표시되는 지표:
  - 전체 활성 프로그램 수
  - 전체 활성 소속사 수
  - 평균 수수료율 (모든 활성 프로그램에 걸쳐 계산됨)
- **CTA 버튼** — 자동으로 생성됨:
  - **로그인** — 기존 소속사용
  - **소속사가 되기** — 등록 흐름을 트리거함

### 히어로 메시지 커스터마이징

| 필드 | 예시 값 | 목적 |
|-------|--------------|---------|
| **히어로 제목** | "우리와 협력하여 수익을 얻으세요" | 혜택 중심의 헤드라인으로 주목을 끌기 |
| **히어로 부제** | "500명 이상의 소속사가 귀하의 추천으로 발생하는 모든 판매에 대해 경쟁력 있는 수수료를 얻고 있습니다" | 사회적 증거를 제공하고 제안을 명확히 함 |

통계는 **자동으로 계산**되며, 활성 프로그램과 소속사에 따라 실시간으로 업데이트됩니다. 이 값들은 수작업으로 편집할 수 없습니다.

## 기능 섹션

기능 섹션은 소속사가 프로그램에 가입해야 하는 **6개의 커스터마이징 가능한 혜택 카드**를 강조합니다. 각 기능 카드에는 다음과 같은 항목이 포함됩니다:

- **아이콘** — FontAwesome 아이콘 클래스 (예: `fa-dollar-sign`, `fa-chart-line`, `fa-headset`)
- **제목** — 혜택 헤드라인 (예: "경쟁력 있는 수수료")
- **설명** — 1~2문장 설명 (예: "귀하의 추천으로 발생하는 모든 판매에 대해 최대 15%를 얻으세요")

### 기본 기능

소속사 앱을 처음 설치할 때 Spwig은 기본 기능을 제공합니다:

| 아이콘 | 제목 | 설명 |
|------|-------|-------------|
| `fa-dollar-sign` | 경쟁력 있는 수수료 | 귀하의 추천으로 발생하는 모든 판매에 대해 풍부한 수수료를 얻으세요 |
| `fa-link` | 간단한 추적 링크 | 어디서나 작동하는 고유한 추적 링크를 얻으세요 |
| `fa-chart-line` | 실시간 분석 | 대시보드에서 클릭, 전환, 수익을 추적하세요 |
| `fa-calendar-check` | 신뢰할 수 있는 지급 | PayPal 또는 은행 이체를 통해 시간에 맞춰 지급받으세요 |
| `fa-headset` | 전담 지원 | 우리 팀은 귀하의 성공을 위해 여기에 있습니다 |
| `fa-gift` | 마케팅 자료 | 배너, 이미지 및 프로모션 콘텐츠에 접근하세요 |

### 기능 커스터마이징

기능은 데이터베이스에 **JSON 배열**로 저장됩니다. 관리자 양식에서 직접 편집할 수 있습니다:

```json
[
  {
    "icon": "fa-percent",
    "title": "최대 20% 수수료",
    "description": "프리미엄 제품 판매에 대해 업계 최고 수준의 수수료를 얻으세요"
  },
  {
    "icon": "fa-rocket",
    "title": "빠른 승인",
    "description": "24시간 이내에 승인받고 즉시 홍보를 시작하세요"
  },
  {
    "icon": "fa-mobile-alt",
    "title": "모바일 대시보드",
    "description": "모든 기기에서 링크를 관리하고 수익을 추적하세요"
  }
]
```

**아이콘 참조:** FontAwesome 5 Free의 아이콘 클래스를 사용하세요. 아이콘을 [fontawesome.com/icons](https://fontawesome.com/icons)에서 둘러보고 클래스 이름을 사용하세요 (예: `fa-trophy`, `fa-users`, `fa-star`).

## 작동 방식 섹션

"작동 방식" 섹션은 소속사의 여정을 설명하는 **4단계 시각 흐름**을 표시합니다. 각 단계에는 다음과 같은 항목이 포함됩니다:

- **제목** — 단계 이름 (예: "등록")
- **설명** — 단계에서 발생하는 일에 대한 1~2문장 설명

### 기본 단계

| 단계 | 제목 | 설명 |
|------|-------|-------------|
| 1 | 등록 | 몇 분 안에 무료 소속사 계정을 생성하세요 |
| 2 | 링크 얻기 | 모든 제품 또는 페이지에 대한 고유한 추적 링크를 생성하세요 |
| 3 | 홍보 | 콘텐츠, 소셜 미디어 또는 이메일을 통해 대상자에게 링크를 공유하세요 |
| 4 | 수수료 얻기 | 추천 링크를 통해 고객이 구매하면 지급받으세요 |

### 단계 커스터마이징

단계는 **JSON 배열**로 저장됩니다. 관리자에서 편집할 수 있습니다:

```json
[
  {
    "title": "가입 신청",
    "description": "신청서를 제출하고 귀하의 플랫폼에 대해 알려주세요"
  },
  {
    "title": "승인 받기",
    "description": "우리 팀은 24시간 이내에 신청서를 검토합니다"
  },
  {
    "title": "링크 생성",
    "description": "대시보드에 접속하여 추적 링크를 즉시 생성하세요"
  },
  {
    "title": "수익 시작",
    "description": "추천으로 발생하는 모든 판매에 대해 수수료를 받으세요 — PayPal을 통해 월별 지급"
  }
]
```

시각 흐름은 랜딩 페이지에서 각 단계를 자동으로 번호로 표시합니다 (1, 2, 3, 4).

## CTA 섹션

등록 양식 전 마지막 섹션은 **콜투액션(CTA) 섹션**입니다. 이 섹션은 등록을 유도하는 마지막 촉매제를 제공합니다.

| 필드 | 예시 값 | 목적 |
|-------|--------------|---------|
| **CTA 제목** | "수익을 시작할 준비가 되셨나요?" | 직접적인 질문으로 긴급성을 만듭니다 |
| **CTA 설명** | "오늘 바로 소속사 프로그램에 가입하여 이미 좋아하고 추천하는 제품에 대해 수수료를 얻으세요." | 혜택을 강조하고 마찰을 제거합니다 |

CTA 섹션은 텍스트 아래에 **소속사가 되기** 버튼을 자동으로 표시합니다.

## 등록 설정

새로운 소속사가 어떻게 가입하고 어떤 정보를 제공하는지 제어합니다.

### 커스텀 등록 양식

**필드:** `custom_form` (FormBuilder 양식의 외래 키)

Spwig의 Form Builder를 사용하여 커스텀 등록 양식을 작성했다면 여기에서 선택하세요. 이는 등록 시 추가 정보를 수집할 수 있도록 합니다 (예: 웹사이트 URL, 대상자 수, 홍보 채널).

**빈칸 유지**하여 기본 소속사 등록 양식 (이메일, 비밀번호, 결제 정보)을 사용하세요.

### 게스트 등록 허용

**필드:** `allow_guest_registration` (부울)

- **체크** — 먼저 Spwig 계정을 생성하지 않고도 신청할 수 있습니다
- **체크 해제** — 신청하기 전에 로그인하거나 고객 계정을 생성해야 합니다

**추천:** 게스트 등록을 활성화하여 마찰을 줄이세요. 등록 후 소속사를 검토하기 전에 언제든지 승인을 요구할 수 있습니다.

### 승인 필요

**필드:** `require_approval` (부울)

- **체크** — 새로운 소속사는 대시보드에 접근하기 전에 수동 승인을 기다려야 합니다
- **체크 해제** — 새로운 소속사는 즉시 링크를 생성할 수 있습니다

**추천:** 소속사를 브랜드 적합성, 사기 방지 또는 독점 프로그램을 위해 검토하고 싶다면 수동 승인을 활성화하세요.

### 이용 약관 및 조건 URL

**필드:** `terms_url` (URL)

소속사 프로그램의 이용 약관 및 조건에 대한 선택적 링크입니다. 제공할 경우, 등록 양식은 소속사가 등록하기 전에 약관에 동의하도록 요구하는 확인란을 표시합니다.

**예시:** `/pages/affiliate-terms/`

### 환영 메시지

**필드:** `welcome_message` (텍스트)

등록이 성공적으로 완료된 후 소속사에게 표시되는 메시지입니다. 이 메시지를 사용하여 다음과 같은 작업을 수행할 수 있습니다:

- 가입을 환영합니다
- 다음 단계를 설명합니다 (예: "24시간 이내에 신청서를 검토할 것입니다")
- 시작하기 자료에 대한 링크를 제공합니다

**예시:*
```
우리의 소속사 프로그램에 오신 것을 환영합니다! 귀하의 신청서를 받았으며 24시간 이내에 검토할 것입니다. 승인 확인 및 로그인 지침을 위해 이메일을 확인해 주세요.
```

## 다국어 지원

소속사 설정의 모든 텍스트 필드는 Spwig의 번역 위젯을 사용하여 **번역 가능**합니다:

- 히어로 제목
- 히어로 부제
- 기능 (언어별로 번역된 JSON)
- 작동 방식 단계 (언어별로 번역된 JSON)
- CTA 제목
- CTA 설명
- 환영 메시지

### 번역 작동 방식

번역 가능한 필드를 편집할 때, 번역 위젯을 통해 각 활성화된 언어에 대한 콘텐츠를 제공할 수 있습니다. JSON 필드 (기능, 단계)에 대해서는 언어별로 별도의 JSON 객체를 제공합니다:

**영어:*
```json
[
  {"icon": "fa-dollar-sign", "title": "Competitive Commissions", "description": "Earn up to 15% on every sale"}
]
```

**스페인어:*
```json
[
  {"icon": "fa-dollar-sign", "title": "Comisiones Competitivas", "description": "Gana hasta el 15% en cada venta"}
]
```

포털은 방문자의 언어 선호도에 따라 올바른 언어 버전을 자동으로 표시합니다.

## 변경 사항 미리보기

포털 설정을 커스터마이징한 후:

1. **관리자에서 변경 사항을 저장**합니다
2. 가게의 프론트엔드에서 `/affiliate/`에 방문하세요 (새 탭에서 열기)
3. **등록 흐름을 테스트**하기 위해 "소속사가 되기"를 클릭합니다
4. **브랜딩 일관성 확인** — 포털이 가게의 디자인과 메시지와 일치하는가요?

반복적인 변경 사항을 만들고 페이지를 새로고침하여 즉시 업데이트를 확인할 수 있습니다.

## 예시 커스터마이징

### 시나리오 1: 전자상거래 패션 가게

**목표:** 패션 인플루언서 및 블로거를 모집합니다.

| 설정 | 값 |
|---------|-------|
| 히어로 제목 | "좋아하는 스타일을 홍보하고 수익을 얻으세요" |
| 히어로 부제 | "1,200명 이상의 인플루언서가 모든 판매에 대해 12% 수수료를 얻고 있습니다" |
| 기능 1 | 아이콘: `fa-tshirt`, 제목: "큐레이트된 패션 컬렉션", 설명: "프리미엄 의류 및 액세서리 홍보" |
| 기능 2 | 아이콘: `fa-percentage`, 제목: "12% 수수료", 설명: "모든 제품에 대해 업계 최고 수준의 수수료" |
| 기능 3 | 아이콘: `fa-camera`, 제목: "독점 콘텐츠", 설명: "제품 사진, 영상 및 캠페인 자산에 접근" |
| 게스트 등록 허용 | 체크 |
| 승인 필요 | 체크 (브랜드 적합성 검토) |

### 시나리오 2: B2B SaaS 파트너 프로그램

**목표:** 기업 컨설턴트 및 에이전시를 위한 기업용 소프트웨어 추천을 모집합니다.

| 설정 | 값 |
|---------|-------|
| 히어로 제목 | "우리와 파트너가 되어 수익을 늘리세요" |
| 히어로 부제 | "우리의 B2B 파트너 프로그램을 통해 기업 추천으로 $500을 얻으세요" |
