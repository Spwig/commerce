---
title: 博客管理
---

博客功能可帮助您发布文章、指南和新闻，以吸引流量并吸引受众。Spwig 的博客包含富文本编辑器、定时发布、订阅者通知、自动社交媒体分享和 SEO 工具。

![博客文章](/static/core/admin/img/help/blog-management/blog-post-list.webp)

## 创建博客文章

导航到 **营销 > 博客文章** 并点击 **添加文章**。

### 文章内容

使用 **CKEditor 5** 富文本编辑器撰写文章，它支持以下功能：
- 文本格式（标题、粗体、斜体、列表、引用）
- 图片和媒体（通过媒体库上传）
- 嵌入式视频（YouTube、Vimeo）
- 表格和代码块
- 链接到产品、分类和外部 URL

对于更复杂的布局，启用 **页面构建器** 开关，使用拖放式页面构建器，而不是文本编辑器。

### 文章设置

| 设置 | 描述 |
|---------|-------------|
| **标题** | 在博客和搜索结果中显示的标题 |
| **Slug** | URL 友好的标识符（从标题自动生成，可编辑） |
| **摘要** | 在博客列表卡片和 RSS 源中显示的简短摘要 |
| **特色图片** | 在文章顶部和列表卡片中显示的主要图片 |
| **分类** | 文章的主要分类 |
| **标签** | 用于筛选和相关内容的关键词 |
| **作者** | 被列为作者的员工 |
| **状态** | 草稿、定时、已发布或已归档 |
| **特色** | 将文章固定在博客列表顶部 |

### SEO 设置

每篇文章包含 SEO 字段：
- **元标题** — 搜索引擎结果的自定义标题（默认为文章标题）
- **元描述** — 在搜索引擎结果中显示的摘要
- **开放图谱图片** — 当文章在社交媒体上分享时使用的图片

## 文章状态

| 状态 | 描述 |
|--------|-------------|
| **草稿** | 正在进行中的工作，不对外公开 |
| **定时** | 将在设定的日期和时间自动发布 |
| **已发布** | 已上线，对访客可见 |
| **已归档** | 从博客列表中隐藏，但可以通过直接 URL 访问 |

### 定时发布文章

要定时发布文章：
1. 将状态设置为 **定时**
2. 选择 **发布日期和时间**
3. 保存文章

后台任务会在预定时间自动发布文章并触发订阅者通知。

## 分类

导航到 **营销 > 博客分类** 以组织您的内容。

分类支持：
- **层次结构** — 创建父分类和子分类（例如，"指南" > "入门"）
- **自定义 URL** — 每个分类都有自己的 slug 以实现干净的 URL
- **描述** — 在分类归档页面上显示的分类描述
- **排序** — 控制导航中分类的显示顺序

## 标签

标签提供了一种分类内容的次要方式。与分类（具有层次结构）不同，标签是扁平的标签。访客可以点击标签查看所有带有该标签的文章。

## 订阅者

导航到 **营销 > 博客订阅者** 以管理您的订阅者列表。

### 订阅机制

1. 访客通过博客上的表单订阅（需要电子邮件地址）
2. 发送一封 **双重确认** 的确认电子邮件
3. 一旦确认，订阅者将在新文章发布时收到通知

### 通知频率

订阅者选择他们接收通知的频率：

| 频率 | 描述 |
|-----------|-------------|
| **立即** | 新文章发布后立即发送电子邮件 |
| **每周摘要** | 所有新文章的每周摘要 |
| **每月摘要** | 所有新文章的每月摘要 |

后台任务会自动处理摘要的编译和发送。

### 管理订阅者

- 查看订阅者数量、确认状态和注册日期
- 导出订阅者列表以用于外部电子邮件营销工具
- 移除或取消订阅个别地址
- 每封通知电子邮件都包含一个一键 **取消订阅链接**

## 社交媒体自动分享

Spwig 可以在文章发布时自动将新文章分享到您的社交媒体账户。

### 连接社交媒体账户

导航到 **营销 > 社交连接器** 以连接您的账户：

| 平台 | 认证 |
|----------|---------------|
| **Facebook** | OAuth — 连接您的 Facebook 页面 |
| **Instagram** | OAuth — 连接您的企业账户 |
| **LinkedIn** | OAuth — 连接您的公司页面 |

### 自动分享机制

1. 连接一个或多个社交媒体账户
2. 在创建文章时，为每个连接的账户启用 **自动分享**
3. 自定义分享信息（默认为文章标题和摘要）
4. 当文章发布（或达到预定时间）时，会自动分享

自动分享也适用于定时发布的文章 — 社交分享会在文章上线时同时发送。

## RSS 源

博客会自动生成一个 RSS 源，位于 `/blog/feed/`。这允许访客和聚合器订阅您的内容。该源包括：
- 文章标题和摘要
- 发布日期
- 作者信息
- 直接链接到完整文章

## 博客设置

导航到 **营销 > 博客设置** 以配置全局博客选项：

- **每页文章数** — 列表中每页显示的文章数量
- **允许评论** — 启用或禁用文章的评论功能
- **默认分类** — 无分类文章的默认分类
- **社交媒体分享按钮** — 在单个文章页面上显示分享按钮

## 小贴士

- 以 **SEO 为导向** 撰写文章 — 使用描述性的标题，填写元描述，并在内容中自然地包含相关关键词。
- 使用 **定时发布** 以保持一致的发布节奏，而无需手动操作。
- 启用 **自动分享** 以最大化覆盖范围 — 在发布后不久在社交媒体上分享的文章能获得最多的互动。
- 鼓励访客 **订阅**，通过在博客上突出显示订阅表单并使用有吸引力的行动号召。
- 使用 **分类** 进行广泛的内容分组，使用 **标签** 进行具体主题 — 这有助于访客找到相关内容。
- 为每篇文章添加 **特色图片** — 带有图片的文章在搜索结果和社交媒体分享中表现更好。
- 对于不想频繁接收电子邮件的订阅者，使用 **每周或每月摘要** 选项 — 这可以降低取消订阅率。
