---
title: โปรแกรมพันธมิตร
---

โปรแกรมพันธมิตรช่วยให้คุณสามารถรับสมัครพันธมิตรที่จะโปรโมทสินค้าของคุณและได้รับค่าคอมมิชชั่นจากยอดขายที่พวกเขาสร้างขึ้นได้ พันธมิตรจะแบ่งปันลิงก์การแนะนำที่เป็นเอกลักษณ์ และ Spwig จะติดตามการคลิก ระบุการสั่งซื้อ และคำนวณค่าคอมมิชชั่นโดยอัตโนมัติ

![โปรแกรมพันธมิตร](/static/core/admin/img/help/affiliate-program/program-list.webp)

## วิธีการทำงาน

1. คุณสร้างโปรแกรมพันธมิตรหนึ่งหรือหลายโปรแกรมโดยระบุอัตราค่าคอมมิชชั่นและกฎเกณฑ์
2. พันธมิตร **สมัครสมาชิก** ผ่านพอร์ทัลสาธารณะ หรือถูกเพิ่มโดยตรง
3. พันธมิตรแต่ละคนจะได้รับ **ลิงก์การแนะนำที่เป็นเอกลักษณ์** พร้อมรหัสติดตาม
4. เมื่อลูกค้าคลิกที่ลิงก์และทำการซื้อสินค้า ค่าคอมมิชชั่นจะถูกบันทึก
5. คุณตรวจสอบและอนุมัติค่าคอมมิชชั่น แล้วดำเนินการ **การจ่ายเงิน**

## การสร้างโปรแกรม

ไปที่ **การตลาด > โปรแกรมพันธมิตร** และคลิกที่ **เพิ่มโปรแกรม**

### การตั้งค่าโปรแกรม

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **ชื่อ** | ชื่อโปรแกรมที่พันธมิตรสามารถมองเห็นได้ (เช่น "โปรแกรมพันธมิตร") |
| **ประเภทค่าคอมมิชชั่น** | ค่าคอมมิชชั่นเป็น **เปอร์เซ็นต์** ของยอดขาย หรือเป็น **จำนวนคงที่** ต่อการขาย |
| **อัตราค่าคอมมิชชั่น** | อัตราเปอร์เซ็นต์หรือจำนวนคงที่ที่พันธมิตรได้รับ |
| **ระยะเวลาการติดตาม** | จำนวนวันที่คุกกี้การติดตามการแนะนำยังคงอยู่ (ค่าเริ่มต้น: 30 วัน) |
| **ค่าจ่ายขั้นต่ำ** | จำนวนเงินขั้นต่ำที่พันธมิตรต้องได้รับก่อนที่จะสามารถขอรับการจ่ายเงินได้ |
| **อนุมัติพันธมิตรโดยอัตโนมัติ** | ยอมรับการสมัครพันธมิตรใหม่โดยอัตโนมัติ หรือต้องการการอนุมัติแบบมือถือ |
| **สถานะ** | เปิดใช้งาน, ระงับ, หรือปิด |

### ประเภทค่าคอมมิชชั่น

- **เปอร์เซ็นต์** — พันธมิตรจะได้รับเปอร์เซ็นต์ของยอดขายย่อยที่ลูกค้าแนะนำ (เช่น 10% ของยอดขาย 100 ดอลลาร์ = ค่าคอมมิชชั่น 10 ดอลลาร์)
- **จำนวนคงที่** — พันธมิตรจะได้รับจำนวนเงินคงที่ต่อการขายไม่ว่ามูลค่าการสั่งซื้อจะเป็นเท่าไร (เช่น 5 ดอลลาร์ต่อการขาย)

## การจัดการพันธมิตร

ไปที่ **การตลาด > พันธมิตร** เพื่อดูและจัดการบัญชีพันธมิตร

### รายละเอียดพันธมิตร

พันธมิตรแต่ละคนมี:
- **รหัสพันธมิตร** — รหัสที่เป็นเอกลักษณ์ที่ใช้ในลิงก์การแนะนำ (สร้างโดยอัตโนมัติหรือกำหนดเอง)
- **ลิงก์การแนะนำ** — ลิงก์ URL ที่พันธมิตรแบ่งปัน (เช่น `yourstore.com/?ref=CODE`)
- **สถานะ** — รอการตรวจสอบ, ได้รับการอนุมัติ, หรือถูกปฏิเสธ
- **วิธีการชำระเงิน** — วิธีที่พันธมิตรจะได้รับการจ่ายเงิน (PayPal หรือการโอนเงินผ่านธนาคาร)
- **การเป็นสมาชิกโปรแกรม** — โปรแกรมที่พันธมิตรเป็นสมาชิก

### การเพิ่มพันธมิตรด้วยตนเอง

1. คลิกที่ **เพิ่มพันธมิตร**
2. เลือกบัญชีลูกค้าที่มีอยู่หรือสร้างบัญชีใหม่
3. ระบุพันธมิตรให้อยู่ในโปรแกรมหนึ่งหรือหลายโปรแกรม
4. ตั้งค่ารหัสพันธมิตร (หรือปล่อยว่างเพื่อให้สร้างโดยอัตโนมัติ)

### พอร์ทัลพันธมิตร

พันธมิตรสามารถเข้าถึงพอร์ทัลสาธารณะที่พวกเขาสามารถ:
- ดูแดชบอร์ดของตนเองพร้อมข้อมูลการได้รับเงินและสถิติการคลิก
- คัดลอกลิงก์การแนะนำของตนเอง
- ติดตามประวัติค่าคอมมิชชั่น
- ขอรับการจ่ายเงิน

URL ของพอร์ทัลจะสามารถเข้าถึงได้โดยอัตโนมัติที่ `/affiliate/` บนร้านค้าของคุณ

## การติดตามและการจ่ายเงิน

### วิธีการทำงานของการติดตาม

1. ลูกค้าคลิกที่ลิงก์การแนะนำของพันธมิตร
2. คุกกี้การติดตามจะถูกตั้งค่าในเบราว์เซอร์ของลูกค้า (ระยะเวลาที่คุกกี้การติดตามยังคงอยู่ตามที่กำหนดไว้)
3. หากลูกค้าสั่งซื้อภายในระยะเวลาที่คุกกี้ยังคงอยู่ คำสั่งซื้อจะถูกจัดสรรให้กับพันธมิตร
4. บันทึกค่าคอมมิชชั่นโดยมีสถานะ **รอการตรวจสอบ**

### สถานะของค่าคอมมิชชั่น

| สถานะ | คำอธิบาย |
|--------|-------------|
| **รอการตรวจสอบ** | ค่าคอมมิชชั่นถูกบันทึกแล้ว แต่ยังรอการตรวจสอบ |
| **ได้รับการอนุมัติ** | ได้รับการตรวจสอบแล้วและพร้อมสำหรับการจ่ายเงิน |
| **ถูกปฏิเสธ** | ค่าคอมมิชชั่นถูกปฏิเสธ (เช่น คำสั่งซื้อที่มีความผิดปกติ หรือสินค้าถูกคืน) |
| **ได้รับการจ่ายเงิน** | ค่าคอมมิชชั่นถูกรวมในเงินจ่ายที่เสร็จสิ้นแล้ว |

### การตรวจสอบค่าคอมมิชชั่น

ไปที่ **การตลาด > ค่าคอมมิชชั่น** เพื่อดูค่าคอมมิชชั่นที่ยังรอการตรวจสอบ:

1. ตรวจสอบรายละเอียดการสั่งซื้อเพื่อยืนยันว่าการขายเป็นไปอย่างถูกต้อง
2. คลิก **อนุมัติ** เพื่อยืนยัน หรือ **ปฏิเสธ** พร้อมระบุเหตุผล
3. ค่าคอมมิชชั่นที่ได้รับการอนุมัติจะสะสมเพื่อเพิ่มยอดเงินจ่ายของพันธมิตร

## การจ่ายเงิน

เมื่อยอดค่าคอมมิชชั่นที่ได้รับการอนุมัติของพันธมิตรถึงเกณฑ์ขั้นต่ำของการจ่ายเงิน คุณสามารถดำเนินการจ่ายเงินได้

### การดำเนินการจ่ายเงิน

1. ไปที่ **การตลาด > การจ่ายเงิน**
2. เลือกพันธมิตรที่มียอดเงินคงเหลือ
3. เลือกวิธีการชำระเงิน:
   - **PayPal** — ส่งเงินโดยตรงไปยังอีเมล PayPal ของพันธมิตร
   - **การโอนเงินผ่านธนาคาร** — บันทึกการโอนเงินผ่านธนาคารด้วยตนเอง
4. ยืนยันและดำเนินการจ่ายเงิน
5. สถานะการจ่ายเงินจะอัปเดตเป็น **เสร็จสิ้น** และค่าคอมมิชชั่นจะถูกทำเครื่องหมายว่า **ได้รับการจ่ายเงิน**

### ผู้ให้บริการการจ่ายเงิน

Spwig ผสานรวมกับผู้ให้บริการชำระเงินเพื่อจ่ายเงินอัตโนมัติ:
- **PayPal** — การจ่ายเงินแบบรวมจำนวนมากผ่าน API PayPal
- **Airwallex** — การจ่ายเงินระหว่างประเทศด้วยอัตราแลกเปลี่ยนที่มีการแข่งขัน
- **การจ่ายเงินแบบมือถือ** — บันทึกการจ่ายเงินที่ดำเนินการนอกเหนือจาก Spwig

## ลิงก์การแนะนำ

ลิงก์การแนะนำของพันธมิตรแต่ละคนมีรูปแบบดังนี้:

```
https://yourstore.com/?ref=AFFILIATE_CODE
```

พันธมิตรสามารถสร้างลิงก์ไปยังสินค้าหรือหมวดหมู่เฉพาะได้ด้วย:

```
https://yourstore.com/products/shoe-name/?ref=AFFILIATE_CODE
```

พารามิเตอร์ `ref` ทำงานได้ทุกหน้า — คุกกี้การติดตามจะถูกตั้งค่าไม่ว่าลูกค้าจะเข้าสู่หน้าใด

## การวิเคราะห์โปรแกรม

แดชบอร์ดของโปรแกรมพันธมิตรแสดง:
- **จำนวนการคลิกทั้งหมด** — จำนวนครั้งที่ลิงก์การแนะนำถูกคลิก
- **จำนวนคำสั่งซื้อทั้งหมด** — คำสั่งซื้อที่ถูกจัดสรรให้กับพันธมิตร
- **ค่าคอมมิชชั่นทั้งหมด** — ผลรวมของค่าคอมมิชชั่นทั้งหมด (ที่ยังรอการตรวจสอบ, ได้รับการอนุมัติ, และได้รับการจ่ายเงิน)
- **พันธมิตรที่ใช้งานอยู่** — จำนวนพันธมิตรที่ได้รับการอนุมัติและกำลังสร้างการแนะนำในปัจจุบัน

## เคล็ดลับ

- เริ่มต้นด้วย **ค่าคอมมิชชั่นแบบเปอร์เซ็นต์** (5–15%) — มันขยายตามมูลค่าการสั่งซื้อได้อย่างเป็นธรรมชาติ และง่ายต่อการเข้าใจสำหรับพันธมิตร
- ตั้งค่า **ระยะเวลาการติดตามเป็น 30 วัน** เป็นค่าเริ่มต้น — นี่จะให้ลูกค้ามีเวลาในการกลับมาและสิ้นสุดการซื้อสินค้า แต่ยังคงจัดสรรการขายให้กับพันธมิตร
- เปิดใช้งาน **การอนุมัติอัตโนมัติ** สำหรับโปรแกรมสาธารณะเพื่อลดความยุ่งยาก หรือใช้การอนุมัติแบบมือถือสำหรับโปรแกรมที่เปิดรับเฉพาะผู้ถูกเชิญโดยคุณเพื่อตรวจสอบแต่ละพันธมิตร
- ตั้งค่า **ค่าจ่ายขั้นต่ำ** ที่เหมาะสม (เช่น 25–50 ดอลลาร์) เพื่อหลีกเลี่ยงการดำเนินการจ่ายเงินหลายครั้งที่มีมูลค่าต่ำ
- ปรับแต่ง **พอร์ทัลพันธมิตร** ให้สอดคล้องกับแบรนด์ของคุณ — พันธมิตรมีแนวโน้มที่จะโปรโมทร้านค้าของคุณมากขึ้นเมื่อประสบการณ์รู้สึกมืออาชีพ
- ตรวจสอบค่าคอมมิชชั่นเป็นประจำสำหรับ **รูปแบบการฉ้อโกง** เช่น การแนะนำตัวเอง, อัตราการคืนสินค้าที่ผิดปกติ, หรือปริมาณการคลิกที่น่าสงสัย
