블로그는 트래픽을 유도하고 독자와의 상호작용을 촉진하기 위해 기사, 가이드, 뉴스를 게시할 수 있습니다. Spwig의 블로그는 풍부한 텍스트 편집기, 예약 게시, 구독자 알림, 자동 소셜 미디어 공유, SEO 도구를 포함합니다.

![블로그 게시물](/static/core/admin/img/help/blog-management/blog-post-list.webp)

## 블로그 게시물 작성

**마케팅 > 블로그 게시물**로 이동하여 **게시물 추가**를 클릭합니다.

### 게시물 내용

**CKEditor 5** 풍부한 텍스트 편집기를 사용하여 게시물을 작성합니다. 이 편집기는 다음과 같은 기능을 지원합니다:
- 텍스트 서식 (헤드라인, 굵은 글씨, 이탤릭, 목록, 블록 인용)
- 이미지 및 미디어 (미디어 라이브러리에서 업로드)
- 임베드된 동영상 (YouTube, Vimeo)
- 표 및 코드 블록
- 제품, 카테고리 및 외부 URL로의 링크

더 복잡한 레이아웃을 원하는 경우 **Page Builder** 토글을 활성화하여 텍스트 편집기 대신 드래그 앤 드롭 페이지 빌더를 사용할 수 있습니다.

### 게시물 설정

| 설정 | 설명 |
|---------|-------------|
| **제목** | 블로그 및 검색 결과에 표시되는 헤드라인 |
| **슬러그** | URL 친화적인 식별자 (제목에서 자동 생성, 편집 가능) |
| **요약** | 블로그 목록 카드 및 RSS 피드에 표시되는 짧은 요약 |
| **대표 이미지** | 게시물 상단 및 목록 카드에 표시되는 메인 이미지 |
| **카테고리** | 게시물의 주 카테고리 |
| **태그** | 필터링 및 관련 콘텐츠를 위한 키워드 |
| **작성자** | 저자로 신용되는 직원 |
| **상태** | 초안, 예약, 게시, 또는 보관 |
| **대표** | 게시물을 블로그 목록 상단에 고정 |

### SEO 설정

각 게시물에는 SEO 필드가 포함되어 있습니다:
- **메타 제목** — 검색 엔진 결과용 사용자 정의 제목 (기본값은 게시물 제목)
- **메타 설명** — 검색 엔진 결과에 표시되는 요약
- **오픈 그래프 이미지** — 게시물이 소셜 미디어에 공유될 때 사용되는 이미지

## 게시물 상태

| 상태 | 설명 |
|--------|-------------|
| **초안** | 작업 중이며 대중에게 보이지 않음 |
| **예약** | 설정된 날짜 및 시간에 자동으로 게시됨 |
| **게시됨** | 활성 상태이며 방문자에게 보임 |
| **보관됨** | 블로그 목록에서 숨겨졌지만 직접 URL을 통해 여전히 접근 가능 |

### 게시물 예약

게시물을 미래에 게시하려면:
1. 상태를 **예약**으로 설정합니다
2. **게시 날짜 및 시간**을 선택합니다
3. 게시물을 저장합니다

배경 작업은 예약된 시간에 게시물을 자동으로 게시하고 구독자 알림을 트리거합니다.

## 카테고리

**마케팅 > 블로그 카테고리**로 이동하여 콘텐츠를 정리합니다.

카테고리는 다음과 같은 기능을 지원합니다:
- **계층 구조** — 부모 및 자식 카테고리 생성 (예: "가이드" > "시작하기")
- **사용자 정의 URL** — 각 카테고리에 고유한 슬러그를 통해 깔끔한 URL을 제공
- **설명** — 카테고리 아카이브 페이지에 표시되는 카테고리 설명을 추가
- **정렬** — 네비게이션에서 카테고리의 표시 순서를 제어

## 태그

태그는 콘텐츠를 분류하는 두 번째 방법입니다. 카테고리(계층 구조)와 달리 태그는 평평한 라벨입니다. 방문자는 태그를 클릭하여 해당 태그가 있는 모든 게시물을 볼 수 있습니다.

## 구독자

**마케팅 > 블로그 구독자**로 이동하여 구독자 목록을 관리합니다.

### 구독 방식

1. 방문자는 블로그의 폼을 통해 구독 (이메일 주소 필요)
2. **이중 확인** 확인 이메일이 발송됩니다
3. 확인 후 구독자는 새 게시물이 게시될 때 알림을 받습니다

### 알림 빈도

구독자는 알림을 받는 빈도를 선택합니다:

| 빈도 | 설명 |
|-----------|-------------|
| **즉시** | 새 게시물이 게시되는 즉시 이메일이 발송됩니다 |
| **주간 요약** | 모든 새 게시물의 주간 요약 |
| **월간 요약** | 모든 새 게시물의 월간 요약 |

배경 작업은 요약 편집 및 전송을 자동으로 처리합니다.

### 구독자 관리

- 구독자 수, 확인 상태 및 가입 날짜를 확인합니다
- 외부 이메일 마케팅 도구에서 사용할 수 있는 구독자 목록을 내보냅니다
- 개별 이메일 주소를 제거하거나 구독을 해제합니다
- 각 알림 이메일에는 **한 번 클릭으로 구독 해제** 링크가 포함되어 있습니다

## 소셜 미디어 자동 공유

Spwig은 게시물이 게시될 때 새 게시물을 소셜 미디어 계정에 자동으로 공유할 수 있습니다.

### 소셜 계정 연결

**마케팅 > 소셜 커넥터**로 이동하여 계정을 연결합니다:

| 플랫폼 | 인증 |
|----------|---------------|
| **Facebook** | OAuth — Facebook 페이지 연결 |
| **Instagram** | OAuth — 비즈니스 계정 연결 |
| **LinkedIn** | OAuth — 회사 페이지 연결 |

### 자동 공유 방식

1. 하나 이상의 소셜 계정을 연결합니다
2. 게시물을 작성할 때 연결된 각 계정에 대해 **자동 공유**를 활성화합니다
3. 공유 메시지를 커스터마이즈합니다 (기본값은 게시물 제목 및 요약)
4. 게시물이 게시되거나 예약된 시간에 도달하면 자동으로 공유됩니다

예약된 게시물에도 자동 공유가 적용됩니다 — 소셜 공유는 게시물이 라이브가 되는 동일한 시간에 전송됩니다.

## RSS 피드

블로그는 자동으로 `/blog/feed/`에 RSS 피드를 생성합니다. 이는 방문자 및 집계기에게 콘텐츠를 구독할 수 있도록 합니다. 피드에는 다음과 같은 정보가 포함됩니다:
- 게시물 제목 및 요약
- 게시 날짜
- 작성자 정보
- 전체 게시물로 이동하는 직접 링크

## 블로그 설정

**마케팅 > 블로그 설정**으로 이동하여 글로벌 블로그 옵션을 구성합니다:

- **페이지당 게시물 수** — 목록에서 페이지당 표시되는 게시물 수
- **댓글 허용** — 게시물에 대한 댓글을 활성화하거나 비활성화합니다
- **기본 카테고리** — 카테고리가 지정되지 않은 게시물의 대체 카테고리
- **소셜 공유 버튼** — 개별 게시물 페이지에 공유 버튼을 표시

## 팁

- **SEO를 고려하여** 게시물을 작성하세요 — 설명적인 제목을 사용하고 메타 설명을 작성하며 콘텐츠에 관련 키워드를 자연스럽게 포함시킵니다.
- **예약 게시**를 사용하여 일관된 게시 주기를 유지하면서 수동 작업 없이 유지합니다.
- **자동 공유**를 활성화하여 최대한의 영향력을 확보하세요 — 게시 후 바로 소셜 미디어에 공유된 게시물이 가장 많은 참여를 얻습니다.
- 블로그에 **구독 폼을 눈에 띄게 배치**하고 강력한 콜 투 액션을 사용하여 방문자가 구독하도록 유도하세요.
- **카테고리를** 넓은 콘텐츠 그룹화에 사용하고 **태그를** 구체적인 주제에 사용하세요 — 이는 방문자가 관련 콘텐츠를 찾는 데 도움이 됩니다.
- 모든 게시물에 **대표 이미지를 추가하세요** — 이미지가 있는 게시물은 검색 결과 및 소셜 미디어 공유에서 더 잘 수행됩니다.
- 구독자 중 자주 이메일을 받고 싶지 않은 경우 **주간 또는 월간 요약** 옵션을 사용하세요 — 이는 구독 해제 비율을 줄입니다.
