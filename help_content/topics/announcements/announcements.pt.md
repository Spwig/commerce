---
title: Anúncios
---

Anúncios permitem que você exiba banners promocionais e notificações em sua loja virtual. Eles aparecem na barra de anúncios do cabeçalho ou rodapé da sua loja para destacar vendas, atualizações importantes ou ofertas especiais. Navegue até **Marketing > Anúncios** no menu lateral do administrador.

![Lista de anúncios](/static/core/admin/img/help/announcements/announcement-list.webp)

## Lista de Anúncios

A página de anúncios mostra todos os seus anúncios com:

- **Abas de Filtro** — Filtro rápido por Todos, Ativados, Desativados ou Expirados
- **Pesquisa** — Localize anúncios por texto
- **Cartões de Anúncio** — Cada anúncio com status e detalhes de configuração

## Criando um Anúncio

1. Clique em **+ Adicionar Anúncio** no canto superior direito
2. Preencha os detalhes do anúncio:
   - **Mensagem** — O texto que os clientes verão (suporta traduções para lojas multilíngues)
   - **Prioridade** — Controla a ordem de exibição quando vários anúncios estiverem ativos
   - **Tipo de Link** — Escolha o que acontece quando os clientes clicarem no anúncio:
     - **Nenhum Link** — O anúncio é apenas informativo
     - **Produto** — Link para uma página de produto específica
     - **Categoria** — Link para uma página de categoria
     - **URL Personalizada** — Link para qualquer URL
3. Configure as configurações de exibição:
   - **Ativado** — Ative ou desative o anúncio
   - **Link Direto** — Se o clique leva diretamente ao destino
   - **Cor de Fundo** — Cor personalizada de fundo do banner
   - **Cor do Texto** — Cor personalizada do texto
   - **Imagem** — Imagem de fundo ou decorativa opcional
4. Defina o agendamento (opcional):
   - **Data de Início** — Quando o anúncio começa a ser exibido
   - **Data de Fim** — Quando o anúncio expira
5. Clique em **Salvar**

## Cartões de Anúncio

Cada cartão de anúncio exibe:
- **Texto do anúncio** — O conteúdo do anúncio
- **Nível de prioridade** — Para ordenar vários anúncios
- **Tipo de link** — Nenhum link, Produto, Categoria ou URL personalizada
- **Comportamento do link** — Link direto ou outra ação
- **Cores de fundo** — Cor personalizada de fundo do banner
- **Cores do texto** — Cor personalizada do texto
- **Indicador de imagem** — Indica se há uma imagem
- **Última atualização** — Quando foi modificada pela última vez
- **Botão de edição** — Acesso rápido para modificar o anúncio

## Múltiplos Anúncios

Quando vários anúncios estão ativos:
- Eles **rotacionam automaticamente** na barra de anúncios da loja virtual
- **Prioridade** controla a ordem de exibição — prioridade mais alta aparece primeiro
- Os clientes podem fechar anúncios individuais se o widget do cabeçalho estiver configurado para permitir isso

## Anúncio no Cabeçalho

Anúncios são exibidos através do **widget Barra de Anúncio** no Construtor de Cabeçalho:

1. Navegue até **Configurações > Design e Tema > Construtor de Cabeçalho**
2. Adicione o **widget Barra de Anúncio** ao seu cabeçalho
3. Configure as configurações do widget:
   - **Velocidade de rotação automática** — Tempo entre transições de anúncios
   - **Mostrar botão de fechar** — Se os clientes podem fechar o anúncio
   - **Estilo de animação** — Deslizar, desvanecer ou exibição estática

## Dicas

- Mantenha o texto do anúncio curto e ação — "Frete grátis para pedidos acima de $50!" é melhor do que uma descrição longa.
- Use anúncios para promoções com prazo, não para informações permanentes.
- Defina datas de expiração para que promoções expiradas desapareçam automaticamente.
- Corresponda as cores dos anúncios à sua identidade visual da loja para um aspecto profissional.
- Use o tipo de link Produto para direcionar o tráfego para itens específicos que deseja promover.
- Limite os anúncios ativos a 2-3 de cada vez para evitar sobrecarregar os clientes.
