---
title: Annonces
---

Les annonces vous permettent d'afficher des bannières promotionnelles et des notifications sur votre boutique en ligne. Elles apparaissent dans la barre d'annonces du haut ou du bas de votre boutique pour mettre en avant des ventes, des mises à jour importantes ou des offres spéciales. Accédez à **Paramètres > Annonces** dans le menu latéral de l'administration.

![Liste d'annonces](/static/core/admin/img/help/announcements/announcement-list.webp)

## Liste d'annonces

La page des annonces affiche toutes vos annonces avec :

- **Onglets de filtre** — Filtre rapide par Tous, Activés, Désactivés ou Expirés
- **Recherche** — Trouver des annonces par texte
- **Cartes d'annonces** — Chaque annonce avec son statut et ses détails de configuration

## Création d'une annonce

1. Cliquez sur **+ Ajouter une annonce** en haut à droite
2. Remplissez les détails de l'annonce :
   - **Message** — Le texte que les clients verront (prend en charge les traductions pour les boutiques multilingues)
   - **Priorité** — Contrôle l'ordre d'affichage lorsqu'il y a plusieurs annonces actives
   - **Type de lien** — Choisissez ce qui se produit lorsque les clients cliquent sur l'annonce :
     - **Aucun lien** — L'annonce est uniquement informative
     - **Produit** — Liens vers une page produit spécifique
     - **Catégorie** — Liens vers une page de catégorie
     - **URL personnalisée** — Liens vers n'importe quelle URL
3. Configurez les paramètres d'affichage :
   - **Activé** — Basculer l'annonce sur ou hors
   - **Lien direct** — Si le clic mène directement à la cible
   - **Couleur d'arrière-plan** — Couleur d'arrière-plan personnalisée de la bannière
   - **Couleur du texte** — Couleur du texte personnalisée
   - **Image** — Image d'arrière-plan ou décorative facultative
4. Définir l'horodatage (facultatif) :
   - **Date de début** — Quand l'annonce commence à s'afficher
   - **Date de fin** — Quand l'annonce expire
5. Cliquez sur **Enregistrer**

## Cartes d'annonces

Chaque carte d'annonce affiche :
- **Texte de l'annonce** — Le contenu de l'annonce
- **Niveau de priorité** — Pour ordonner plusieurs annonces
- **Type de lien** — Aucun lien, Produit, Catégorie ou URL personnalisée
- **Comportement du lien** — Lien direct ou autre action
- **Étiquettes de statut** — Activé/Désactivé, statut d'expiration, indicateur d'image
- **Dernière mise à jour** — Quand elle a été modifiée pour la dernière fois
- **Bouton Modifier** — Accès rapide pour modifier l'annonce

## Plusieurs annonces

Lorsque plusieurs annonces sont actives :
- Elles **se basculent automatiquement** dans la barre d'annonces sur la boutique en ligne
- **Priorité** contrôle l'ordre d'affichage — une priorité plus élevée s'affiche en premier
- Les clients peuvent ignorer les annonces individuelles si le widget d'en-tête est configuré pour le permettre

## Annonce dans l'en-tête

Les annonces s'affichent via le **widget de barre d'annonces** dans le Constructeur d'en-tête :

1. Accédez à **Paramètres > Design & Theme > Constructeur d'en-tête**
2. Ajoutez le **widget de barre d'annonces** à votre en-tête
3. Configurez les paramètres du widget :
   - **Vitesse de basculement automatique** — Temps entre les transitions d'annonces
   - **Afficher le bouton de fermeture** — Si les clients peuvent l'ignorer
   - **Style d'animation** — Défilement, fondu ou affichage statique

## Conseils

- Gardez le texte des annonces court et actionnable — « Livraison gratuite pour les commandes de plus de 50 $ ! » est meilleur qu'une longue description.
- Utilisez les annonces pour les promotions à durée limitée, pas pour les informations permanentes.
- Définissez des dates d'expiration afin que les promotions expirées disparaissent automatiquement.
- Adaptez les couleurs des annonces à la charte graphique de votre boutique pour un look professionnel.
- Utilisez le type de lien Produit pour diriger le trafic vers les articles spécifiques que vous souhaitez promouvoir.
- Limitez les annonces actives à 2 à 3 à la fois pour éviter d'overcharger les clients.
