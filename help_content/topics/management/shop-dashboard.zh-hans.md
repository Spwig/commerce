---
title: 店铺仪表板
---

店铺仪表板为您提供商店性能的完整视图 —— 收入、订单、热门产品、访客流量等 —— 所有内容都在一个地方。使用它来了解哪些商品在销售，客户来自哪里，以及您的商店随时间的趋势。

导航到 **管理 > 系统指标**，然后从工具栏中点击 **店铺仪表板**。

![店铺仪表板概览](/static/core/admin/img/help/shop-dashboard/overview.webp)

## 选择时间段

仪表板会根据所选时间段筛选所有指标。使用页面顶部的时间段选择器来选择：

| 时间段 | 显示内容 |
|--------|---------------|
| 今天 | 今天与昨天对比 |
| 本周 | 从周一到今天与上周对比 |
| 本月 | 本月与上个月对比 |
| 本年 | 今年至今与去年同期对比 |
| 最近30天 | 滚动30天窗口 |
| 最近90天 | 滚动90天窗口 |
| 自定义 | 输入特定的开始和结束日期 |

大多数视图会显示与过去相应时间段的 **对比**，因此您可以查看性能是有所改善还是下降。如果您只想查看当前数据，请关闭 **对比** 开关。

## 操作卡片

在仪表板顶部，操作卡片突出显示需要您立即关注的项目：

- **未完成订单** — 等待发货的订单
- **被放弃的购物车** — 客户添加了商品但未完成结账的会话
- **未读消息** — 等待回复的客户咨询
- **库存不足警报** — 库存即将告罄的商品

点击任何操作卡片，可直接导航到相关管理部分。

## 销售表现

销售表现部分显示您所选时间段内的关键收入数据：

- **总收入** — 扣除前的总销售额
- **总订单数** — 已完成的订单数量
- **平均订单价值** — 收入除以订单数量
- **净利润** — 收入减去商品成本和费用（如果已配置）

# 每个图表都显示了一个箭头和百分比，表示与比较期的变化。

## 销售趋势图

主图表显示了您在所选期间的销售或订单情况。Spwig 会自动选择最有用的分组方式：

- 短期（最多一周）按天分组
- 中期（最多三个月）按周分组
- 长期按月分组

您可以使用图表上方的 **按...分组** 控制来覆盖默认分组方式。将鼠标悬停在任意一点上，可以查看该日期的确切数值。

## 热销产品

热销产品表格列出了所选期间销售最好的产品，按收入排序。每一行显示：

- **产品名称**
- **销售数量**
- **产生的收入**

使用此功能来识别表现最好的产品，并决定在哪里集中促销或补货。

## 访客分析

访客分析部分显示了访问您商店的人数及其行为方式：

- **总访客数** — 访问您商店的独立访客数量
- **页面浏览量** — 总共浏览的页面数量
- **跳出率** — 仅浏览了一页的访客百分比
- **浏览趋势** — 显示所选期间流量变化的图表

**地理位置** 面板显示了访客的来源地，按国家和（如有）城市细分。

## 流量来源

流量来源面板显示访客是如何找到您商店的：

| 来源 | 描述 |
|------|------|
| 直接访问 | 输入您的网址或使用书签访问的访客 |
| 搜索引擎 | 来自搜索引擎的访客 |
| 社交媒体 | 来自社交媒体平台的访客 |
| 推荐 | 来自链接到您的其他网站的访客 |
| 邮件 | 点击电子邮件中链接的访客 |

使用这些信息来了解哪些营销渠道带来了最多的流量，并决定在哪里进行投资。

## 转化漏斗

转化漏斗显示访客如何从浏览到完成购买：

1. **访客** — 总独立访客
2. **产品浏览** — 浏览至少一个产品的访客
3. **加入购物车** — 将商品添加到购物车的访客
4. **开始结账** — 开始结账流程的访客
5. **完成订单** — 下单的访客

每个步骤的百分比显示了流失率。"加入购物车"和"开始结账"之间的大幅下降表明结账流程中存在摩擦。

## 优惠券表现

如果您运行了优惠券促销，此部分将显示所选期间优惠券的表现：

- **总使用次数** — 优惠券被使用的次数
- **总折扣金额** — 所有优惠券折扣的总和
- **使用优惠券的收入** — 包含优惠券的订单总收入

## 客户细分

客户细分面板将您的客户群划分为以下几类：

- **新客户** — 所选期间首次购买的客户
- **回头客** — 之前购买过的客户
- **未注册结账** — 未创建账户的订单

了解新客户与回头客的比例有助于您决定是加大获客（营销）投入还是提高客户留存（忠诚度计划）。

## 附属和忠诚度摘要

如果您的商店启用了附属计划或忠诚度计划，此处将显示汇总的表现指标 — 总佣金收入、总积分发放量以及表现最佳的附属成员或积分兑换者。

## 小贴士

保留所有markdown格式、图片路径、代码块和技术术语。

- 每周一早上查看仪表板，进行快速的周度回顾 —— “本周”时间段可提供近期表现的清晰快照
- 使用 **自定义** 日期范围来衡量特定活动的影响：将开始和结束日期设置为活动期间
- 如果转化漏斗在 **结账开始** 阶段出现大幅下降，请考虑简化结账流程或添加信任徽章
- 如果购物车弃单数量高且转化率低，可能表明存在定价或运费问题 —— 请审查您的结账成本
- 使用 **今年** 时间段进行年度对比，以了解您业务中的季节性模式
- 在做出重大补货决策之前，导出或截图顶部产品表格，以确保您订购了正确的数量
