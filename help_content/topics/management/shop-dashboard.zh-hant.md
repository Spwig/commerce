---
title: 商店儀表板
---

商店儀表板讓您可以一覽商店的表現 — 收入、訂單、熱銷產品、訪客流量等等 — 全部都在一個地方。使用它來了解哪些商品在銷售、客戶來自哪裡，以及商店的趨勢如何隨時間變化。

導航至 **管理 > 系統指標**，並點擊工具列中的 **商店儀表板**。

![商店儀表板概覽](/static/core/admin/img/help/shop-dashboard/overview.webp)

## 選擇時間段

儀表板會根據所選的時間段過濾所有指標。使用頁面上方的時間段選擇器來選擇：

| 時間段 | 顯示內容 |
|--------|---------------|
| 今天 | 今天與昨天相比 |
| 本週 | 週一至今天與上週相比 |
| 本月 | 本月與上個月相比 |
| 本年 | 年初至今與去年同期相比 |
| 最近30天 | 滾動30天窗口 |
| 最近90天 | 滾動90天窗口 |
| 自訂 | 輸入特定的開始和結束日期 |

大多數視圖會顯示與過去相同期間的 **比較**，因此您可以查看表現是提升還是下降。如果您只想查看當前數據，請關閉 **比較** 切換開關。

## 動作卡片

在儀表板頂部，動作卡片會突出顯示目前需要您關注的項目：

- **未完成的訂單** — 等待履行的訂單
- **放棄的購物車** — 客戶添加了商品但未完成結帳的會話
- **未讀取的訊息** — 等待回覆的客戶諮詢
- **庫存不足提醒** — 库存即將用盡的產品

點擊任何動作卡片即可直接導航到相關的管理區域。

## 賣出表現

賣出表現部分顯示您所選時間段的關鍵收入數據：

- **總收入** — 扣除前的總銷售額
- **總訂單數** — 完成的訂單數量
- **平均訂單價值** — 收入除以訂單數量
- **淨利潤** — 收入減去商品成本和費用（如果已配置）

# 每個圖表顯示箭頭和百分比，表示與比較期間的變化。

## 銷售時間圖表

主要圖表顯示您在所選期間的銷售或訂單情況。Spwig 會自動選擇最有用的分組方式：

- 短期（最多一週）按天分組
- 中期（最多三個月）按週分組
- 長期按月分組

您可以使用圖表上方的 **Group by** 控制項來覆蓋分組方式。將滑鼠懸停在任何一點上，即可查看該日期的精確值。

## 最暢銷產品

最暢銷產品表格列出所選期間內銷售最好的產品，按收入排序。每一行顯示：

- **產品名稱**
- **銷售單位數**
- **產生的收入**

使用此功能來識別表現最好的產品，並決定將促銷或庫存補貨的重點放在哪裡。

## 訪客分析

訪客分析部分顯示有多少人訪問了您的商店以及他們的行為：

- **總訪客數** — 訪問您商店的獨特訪客數
- **頁面瀏覽量** — 總共查看的頁面數
- **跳出率** — 只查看了一頁的訪客百分比
- **時間段內的頁面瀏覽量** — 顯示所選期間流量數量的圖表

**地理** 面板顯示您的訪客來自哪裡，按國家和（如有）城市分類。

## 流量來源

流量來源面板顯示訪客是如何找到您商店的：

| 來源 | 說明 |
|------|------|
| 直接 | 輸入您的 URL 或使用書籤的訪客 |
| 網路搜尋 | 來自搜索引擎的訪客 |
| 社交媒體 | 來自社交媒體平台的訪客 |
| 轉介 | 來自其他網站連結到您的訪客 |
| 電子郵件 | 點擊電子郵件中連結的訪客 |

使用這些資訊來了解哪些行銷渠道帶來最多的流量，並決定投資方向。

## 轉化漏斗

轉化漏斗顯示訪客如何從瀏覽到完成購買：

1. **訪客** — 總計唯一訪客

2. **產品瀏覽** — 至少查看過一件產品的訪客

3. **加入購物車** — 將商品加入購物車的訪客

4. **開始結帳** — 開始結帳流程的訪客

5. **完成訂單** — 下訂單的訪客

每個步驟的百分比顯示了流失率。在「加入購物車」和「開始結帳」之間出現大幅下降，表示您的結帳流程存在摩擦。

## 贈券表現

如果您運行贈券促銷，此部分會顯示在所選期間內贈券的表現：

- **總使用次數** — 贈券被使用的次數

- **總折扣金額** — 所有贈券折扣的總和

- **使用贈券的收入** — 包含贈券的訂單總收入

## 顧客分群

顧客分群面板會將您的顧客基礎分為群組：

- **新顧客** — 所選期間內首次購買的顧客

- **回購顧客** — 之前有購買記錄的顧客

- **未註冊結帳** — 沒有創建帳號就下的訂單

了解新顧客與回購顧客的比例，有助於您決定是否要加大在獲客（市場推廣）或留存（忠誠度計劃）方面的投入。

## 代理與忠誠度摘要

如果您的商店有啟用代理計劃或忠誠度方案，這裡會顯示總結的表現指標 — 總佣金收入、總發放的積分，以及表現最佳的代理或兌換者。

## 小提示

保留所有 markdown 格式、圖片路徑、程式碼塊和技術術語。

- 每週一早上查看儀表板，進行快速的週度回顧 ——「這週」週期能提供近期表現的清晰快照
- 使用 **自定義** 日期範圍來衡量特定活動的影響：將開始和結束日期設為活動期間
- 如果轉化漏斗在 **結帳開始** 處顯示大幅下降，請考慮簡化結帳流程或添加信任徽章
- 放棄購物車數字高且轉化率低，可能表示價格或運費問題 —— 請審查您的結帳費用
- 使用 **今年** 週期進行年度對比，以了解業務中的季節性模式
- 在做出重大補貨決定前，先匯出或截圖頂部產品表格，確保訂購的數量正確
