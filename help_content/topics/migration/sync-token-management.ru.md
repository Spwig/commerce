---
title: Управление синхронизационными токенами
---

Синхронизационные токены — это безопасные учетные данные, которые позволяют двум установкам Spwig общаться друг с другом. Перед тем как вы сможете синхронизировать настройки или мигрировать данные между магазинами, вам нужно сгенерировать токен на **приемнике** магазина и предоставить его **отправляющему** магазину.

## Как работают синхронизационные токены

Синхронизационный токен — это одноразовый API-ключ, который аутентифицирует запросы между двумя установками Spwig. При настройке соединения удаленный магазин использует этот токен, чтобы доказать, что у него есть разрешение на чтение или запись в ваш магазин.

- Токены генерируются на магазине, который будет **подключен** (цель)
- Каждый токен можно просмотреть только один раз, сразу после его генерации
- Токены можно отозвать в любое время, чтобы мгновенно прекратить доступ
- У магазина может быть несколько активных токенов для разных соединений

## Генерация токена

1. Перейдите в **Data Migration > Spwig-to-Spwig Sync** в боковом меню администратора
2. Нажмите **Manage Tokens** на панели синхронизации
3. Введите описательное имя для токена (например, "Staging Server" или "Production Sync")
4. Нажмите **Generate Token**
5. **Сразу скопируйте токен** — он больше не будет отображаться

> **Важно:** Храните токен в безопасности. Если вы его потеряете, вам придется сгенерировать новый.

## Использование токена

После получения токена с целевого магазина:

1. Перейдите на панель **Spwig-to-Spwig Sync** на магазине, который инициирует соединение
2. Начните новую **Синхронизацию настроек** или **Полную миграцию**
3. На шаге соединения введите URL целевого магазина и вставьте токен
4. Нажмите **Test Connection**, чтобы проверить его работоспособность
5. Соединение будет сохранено для последующего использования

## Отзыв токена

Если токен скомпрометирован или больше не нужен:

1. Перейдите в **Manage Tokens** на панели синхронизации
2. Найдите токен, который хотите отозвать
3. Нажмите кнопку **Revoke**
4. Подтвердите отзыв

Отзыв токена вступает в силу сразу. Все активные соединения, использующие этот токен, перестанут работать и потребуют повторной настройки с новым токеном.

## Рекомендации

- **Используйте описательные названия для токенов**, чтобы вы знали, какое соединение относится к каждому токену
- **Отзывайте неиспользуемые токены**, чтобы минимизировать риск безопасности
- **Генерируйте отдельные токены** для каждого подключаемого магазина вместо использования одного токена для нескольких магазинов
- **Периодически регенерируйте токены** в рамках вашей процедуры безопасности, особенно после смены сотрудников
