---
title: 開始使用POS
---

Spwig POS 可將任何平板電腦或瀏覽器轉變為完整的店內收銀機 — 連接到您的產品目錄、庫存和訂單歷史。此檢查清單將引導您從全新安裝到完成第一筆銷售。每個步驟都會連結到專門的主題，如果您需要詳細資訊的話。

![POS Dashboard](/static/core/admin/img/help/pos/getting-started-dashboard.webp)

## 第一步：為商店位置啟用POS

POS終端機與實體商店位置相關聯。在Spwig中，商店位置是標記為零售位置的倉庫。

1. 在您的管理側邊欄中導航至 **目錄 > 倉庫**。
2. 開啟您要用作商店的倉庫，或建立一個新的。
3. 勾選 **零售位置** 的切換鈕，並輸入一個 **POS顯示名稱**（例如："High Street Store"）。此名稱會出現在收據和終端機選擇器中。
4. 儲存倉庫。

如果您有多個商店，或希望為區域報告進行分組，請先在 **POS > 商店群組** 建立一個 **商店群組**，然後將每個倉庫指派到該群組。商店群組讓您可以設定共享的貨幣、時區和收據模板，所有群組中的位置都會繼承這些設定。

## 第二步：建立或驗證至少一個具有POS存取權的員工帳戶

您的員工會使用與Spwig管理員相同的憑證登入POS。

# 第3步：註冊您的第一台POS終端

終端代表一個單獨的收銀台或設備。您可以在管理員中註冊它，然後使用一次性配對碼將物理設備與其配對。

1. 進入 **POS > POS終端**，點擊 **+ 添加POS終端**。
2. 為終端命名（例如："前臺收銀"），並將其分配給您在第1步中啟用的商店位置。
3. 儲存終端。Spwig 會生成一個 **8個字元的配對碼** — 您可以在終端的詳細資訊頁面上看到它。
4. 在您要用作收銀的設備上，打開瀏覽器並前往 `/pos/`。
5. 當提示時輸入配對碼。設備現在已與此終端連結。

配對碼只能使用一次。如果您需要重新配對設備，請在管理員中打開終端，並點擊 **重新生成配對碼**。

如需硬體配置選項（收據印表機、條碼掃描器、現金抽屜），請參閱 [POS終端設定](pos-terminal-setup)。

## 第4步：設定支付服務商

支付服務商將您的讀卡器連接到支付網絡，例如 Stripe Terminal 或 Square。使用5步設定向導輸入您的憑證。

1. 進入 **POS > 支付服務商**，點擊 **設定服務商**。
2. 向導會在 `/admin/pos/terminal-provider/wizard/step1/` 開啟。

![支付服務商向導](/static/core/admin/img/help/pos/getting-started-provider-wizard-step1.webp)

3. 選擇您的服務商（例如：**Stripe Terminal**），並按照螢幕上的指示完成所有五個步驟：選擇服務商 → 設定說明 → 輸入憑證 → 測試連線 → 設定位置。
4. 綠色的 **已連線** 標籤確認整合已啟用。

如果您只需要現金和手動刷卡功能，請選擇 **Manual** 作為提供商 — 不需要憑證。

如需每個支援提供商的詳細憑證欄位，請參閱 [POS 支付提供商設置](pos-payment-provider-setup)。

## 第 5 步：配對卡片讀取器

在連接支付提供商後，您可以使用 3 步驟讀取器向導，將實體卡片讀取器配對到您的其中一個終端機。

1. 進入 **POS > 卡片讀取器**，然後點擊 **新增讀取器**。
2. 讀取器向導從 `/admin/pos/reader/wizard/step1/` 開始。
3. 選擇您的提供商，然後選擇 **註冊新設備**（輸入讀取器螢幕上顯示的代碼）或 **發現現有設備**（Spwig 會從提供商處抓取已註冊的讀取器）。
4. 在最後一步，將讀取器指派給您在第 3 步中創建的終端機。

每個終端機只支援一個已指派的卡片讀取器。您可以隨時在卡片讀取器清單中重新分配讀取器。

## 第 6 步：設計您的收據（第一天可選）

Spwig 會自動創建預設的收據模板。您可以在不進行任何操作的情況下立即開始銷售 — 預設會列印您的商店名稱、地址、明細銷售、支付方式，以及「謝謝您的購買！」的頁尾。

當您準備自訂時，請前往 **POS > 收據模板**。選項包括您的商標、稅號、QR 碼促銷、退貨政策，以及紙張寬度（熱敏印表機為 58mm 或 80mm）。您可以為每個商店或每個商店群組創建不同的模板。

## 第 7 步：開啟您的第一個班次

班次會追蹤處理銷售的人員以及抽屜中應有的現金金額。收銀員可以在 POS 系統本身上開啟和關閉班次。

1. 在已配對的設備上，前往 `/pos/` 並使用員工憑證登入。
2. 選擇終端機和商店位置。
3. Spwig 會提示您 **計算開班現金** — 輸入抽屜中已有的現金金額（如果抽屜為空，請輸入 `0`）。
4. 點擊 **開啟班次**。現金 register 現在已準備好進行銷售。

如需了解有關班次、現金流動和對賬報告的完整說明，請參閱[管理POS班次](pos-shifts)。

## 第8步：完成您的第一筆銷售

一旦班次開啟，銷售過程就相當簡單：

1. 透過產品名稱搜尋、掃描條碼或瀏覽分類，將商品加入購物車。
2. 如有需要，可套用折扣或優惠券代碼。
3. 輕點 **Charge** 開始付款。選擇付款方式（現金、透過讀卡機的信用卡，或分開付款）。
4. 進行信用卡付款時，讀卡機會提示顧客輕點或插入卡片。
5. 收據會自動列印（或顯示數位收據選項）。訂單會即時保存到您的訂單歷史記錄中。

## 第9步：結束當日班次

結束班次會鎖定收銀機並產生對賬總結。

1. 從POS選單中，輕點 **Close Shift**。
2. 清點抽屜中的現金，並在提示時輸入總額。
3. Spwig 會根據開班時的現金餘額、現金銷售以及班次期間的任何現金流動計算預期現金數額，並顯示差額。
4. 確認結束。班次報告會被保存，並可在管理員介面的 **POS > Shifts** 中查看。

請將當天從抽屜中取出或加入的現金記錄為 **現金流動**（透過班次選單），而不是調整結班數額——這樣可以保持對賬的準確性。

## 小提示

- 在營業首日之前，請完成第1至第5步。

第6至第9步可以在當天進行。
- 請使用強而有力但容易記住的員工密碼——POS員工需要在收銀機上輸入憑證，因此過於複雜的密碼會減慢他們的作業速度。
- 如果讀卡機未在線顯示，請點擊 **Sync readers**（同步讀卡機）在「Card Readers」（讀卡機）頁面上從您的供應商處拉取最新狀態。
- 在繁忙營業期之前，請使用 $0.01 測試交易測試完整的流程（開班 → 購買 → 收據 → 結束班次）。
- POS 可離線處理基本的現金銷售。

卡片終端機付款需要互聯網連接來授權。
- 您可以在一個商店位置擁有多个終端機 — 在管理後台新增一個終端機記錄，並將其配對到不同的設備。
