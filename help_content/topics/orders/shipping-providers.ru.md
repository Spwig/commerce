---
title: Поставщики доставки
---

Поставщики доставки подключают ваш магазин к API служб доставки для получения тарифов в реальном времени, генерации этикеток и отслеживания посылок.

![Поставщики доставки](/static/core/admin/img/help/shipping-providers/provider-list.webp)

## Доступные службы доставки

| Служба доставки | Регионы | Ключевые возможности |
|-----------------|---------|---------------------|
| **FedEx** | Глобально | Тарифы в реальном времени, печать этикеток, отслеживание, много мест |
| **UPS** | Глобально | Тарифы в реальном времени, печать этикеток, отслеживание, проверка адресов |
| **USPS** | Соединённые Штаты | Внутренние и международные тарифы, отслеживание |
| **NinjaVan** | Юго-Восточная Азия | Доставка «последней мили», наложенный платёж |
| **Canada Post** | Канада | Внутренние и международные, посылки и письма |
| **Australia Post** | Австралия | Внутренние и международные, посылки и экспресс |

## Подключение службы доставки

Мастер из 5 шагов: Выбор провайдера → Инструкции по настройке → Ввод учётных данных → Тестирование подключения → Настройка и сохранение

## Зоны доставки

Географические области для расчёта тарифов:
- Страны, Штаты/Области, Шаблоны почтовых индексов
- Сопоставление на основе приоритетов

## Правила доставки

| Тип правила | Описание |
|-------------|----------|
| **Скидка %** | Уменьшить тариф на процент |
| **Фиксированная скидка** | Уменьшить тариф на фиксированную сумму |
| **Установить стоимость** | Заменить конкретной суммой |
| **Бесплатная доставка** | Установить стоимость равной нулю |
| **Наценка %** | Добавить процентную наценку |
| **Фиксированная наценка** | Добавить фиксированную наценку |

Условия: Сумма корзины, Вес, Количество товаров, Зона, Способ доставки, Товары, Группа клиентов, Диапазон дат

## Таблицы тарифов
Ступенчатое ценообразование по весу, сумме заказа или количеству.

## Упаковки для доставки
Стандартные размеры упаковки для точного расчёта тарифов.

## Ручные службы доставки (Пресеты служб доставки)
Для служб доставки без API: название, шаблон URL отслеживания, ориентировочные сроки доставки, в сочетании с таблицей тарифов.

## Мульти-складская доставка
- Склады для конкретных стран
- Цепочки отката
- Назначение по товарам

## Советы
- Используйте тарифы в реальном времени, когда это возможно
- Создайте зону «Остальной мир»
- Бесплатная доставка как стимул продаж
- Тестируйте расчёты тарифов
- Используйте Пресеты служб доставки для местных перевозчиков
- Определите Упаковки для доставки для объёмного веса
