---
title: Управление заказами
---

Это руководство охватывает всё, что необходимо для управления заказами клиентов — от просмотра новых заказов до обработки отправлений и оформления возвратов.

## Список заказов

Перейдите в **Заказы > Все заказы** в боковой панели, чтобы просмотреть все заказы. В списке отображаются номер, статус, клиент, сумма и дата каждого заказа.

![Order list](/static/core/admin/img/help/manage-orders/order-list.webp)

Используйте фильтры в верхней части страницы для сужения списка по статусу, диапазону дат или для поиска по номеру заказа или имени клиента.

## Детали заказа

Нажмите на любой заказ, чтобы открыть страницу с подробной информацией. Здесь вы найдёте всю информацию о заказе, организованную в понятные разделы.

![Order detail](/static/core/admin/img/help/manage-orders/order-detail.webp)

### Информация о заказе

В верхней части отображается:

- **Номер заказа** — Уникальный идентификатор данного заказа
- **Статус** — Текущий статус заказа (Ожидает, В обработке, Отправлен, Доставлен, Завершён, Отменён)
- **Клиент** — Имя и email клиента, оформившего заказ
- **Дата создания** — Когда был оформлен заказ

### Товары в заказе

В разделе товаров перечислено всё, что заказал клиент:

- Название товара и артикул (SKU)
- Заказанное количество
- Цена за единицу и итог по строке
- Применённые скидки

### Данные об оплате

Отображается использованный способ оплаты, идентификатор транзакции и статус платежа. Для заказов, ожидающих оплату, здесь можно отслеживать статус платёжного шлюза.

### Адрес доставки

Адрес доставки клиента. Если адрес для выставления счёта отличается, отображаются оба адреса.

## Жизненный цикл заказа

Заказы обычно проходят через следующие статусы:

1. **Ожидает** — Получен новый заказ, ожидается подтверждение оплаты
2. **В обработке** — Оплата подтверждена, подготовка к отправке
3. **Отправлен** — Заказ отправлен с информацией для отслеживания
4. **Доставлен** — Клиент получил заказ
5. **Завершён** — Заказ окончательно закрыт

## Обработка заказа

### 1. Проверка заказа

Убедитесь, что:

- Товары и количество указаны правильно
- Адрес доставки заполнен полностью
- Оплата получена
- Все примечания клиента учтены

### 2. Создание отправления

Чтобы отправить заказ:

1. Нажмите **Создать отправление** на странице деталей заказа
2. Выберите товары для включения (для частичной отправки выберите только некоторые товары)
3. Выберите перевозчика и службу доставки
4. Введите номер отслеживания
5. Нажмите **Сохранить отправление**

Статус заказа автоматически обновляется на **Отправлен**, и клиент получает уведомление по электронной почте с информацией для отслеживания.

### 3. Отметить как доставленный

Когда клиент подтвердит получение или отслеживание покажет доставку, обновите статус на **Доставлен**, а затем на **Завершён**.

## Действия с заказом

### Добавление заметок

Добавляйте внутренние заметки или сообщения, видимые клиенту:

1. Прокрутите до раздела **Заметки** на странице деталей заказа
2. Введите ваше сообщение
3. Выберите, является ли это внутренней заметкой (только для сотрудников) или уведомлением для клиента
4. Нажмите **Добавить заметку**

Заметки, видимые клиенту, инициируют отправку уведомления по электронной почте.

### Оформление возврата

Чтобы оформить возврат:

1. Нажмите **Возврат** на странице деталей заказа
2. Выберите товары для возврата (или введите произвольную сумму)
3. Выберите причину возврата
4. Подтвердите возврат

Возвраты обрабатываются через исходный платёжный шлюз. Клиент получает подтверждение по электронной почте.

### Отмена заказа

Чтобы отменить заказ:

1. Нажмите **Отменить заказ**
2. Выберите причину отмены
3. Выберите, нужно ли вернуть товары на склад
4. Подтвердите

Клиент уведомляется автоматически, и если оплата уже была произведена, инициируется возврат средств.

## Массовые действия

Из списка заказов можно выбрать несколько заказов и применить массовые действия:

- **Обновить статус** — Перевести несколько заказов в один статус
- **Экспорт** — Скачать выбранные заказы в формате CSV
- **Печать** — Сформировать упаковочные листы или счета

## Уведомления о заказах

Клиенты автоматически получают электронные письма на ключевых этапах:

- **Подтверждение заказа** — Сразу после оформления заказа
- **Получение оплаты** — При подтверждении платежа
- **Уведомление об отправке** — При создании отправления (включает ссылку для отслеживания)
- **Подтверждение доставки** — При отметке заказа как доставленного

Настройте шаблоны писем в разделе **Настройки > Конфигурация электронной почты**.

## Советы

- Обрабатывайте заказы ежедневно для поддержания быстрых сроков доставки.
- Используйте фильтры по статусу, чтобы сосредоточиться на заказах, требующих внимания (Ожидает и В обработке).
- Добавляйте внутренние заметки для отслеживания особых требований к обработке.
- В периоды высокой нагрузки используйте массовые действия для обновления нескольких заказов одновременно.
- Настройте правила доставки для автоматического выбора перевозчика на основе веса заказа и пункта назначения.
