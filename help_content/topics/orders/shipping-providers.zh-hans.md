---
title: 物流服务商
---

物流服务商将您的商店连接到承运商 API，以获取实时运费、生成面单和包裹追踪。

![Shipping providers](/static/core/admin/img/help/shipping-providers/provider-list.webp)

## 可用承运商

| 承运商 | 覆盖区域 | 主要功能 |
|--------|----------|----------|
| **FedEx** | 全球 | 实时费率、面单打印、追踪、多包裹 |
| **UPS** | 全球 | 实时费率、面单打印、追踪、地址验证 |
| **USPS** | 美国 | 国内和国际费率、追踪 |
| **NinjaVan** | 东南亚 | 最后一公里配送、货到付款 |
| **Canada Post** | 加拿大 | 国内和国际、包裹和信件 |
| **Australia Post** | 澳大利亚 | 国内和国际、包裹和快递 |

## 连接承运商

5 步向导：选择服务商 → 设置说明 → 输入凭证 → 测试连接 → 配置并保存

## 配送区域

用于运费计算的地理区域：
- 国家、州/省、邮政编码模式
- 基于优先级的匹配

## 配送规则

| 规则类型 | 描述 |
|----------|------|
| **百分比折扣** | 按百分比降低费率 |
| **固定折扣** | 按固定金额降低费率 |
| **设定费用** | 使用指定金额覆盖 |
| **免运费** | 将费用设为零 |
| **百分比附加费** | 添加百分比附加费 |
| **固定附加费** | 添加固定附加费 |

条件：购物车金额、重量、商品数量、区域、配送方式、产品、客户群组、日期范围

## 费率表
按重量、订单金额或数量进行阶梯定价。

## 配送包装
标准包装尺寸，用于准确计算运费。

## 手动承运商（承运商预设）
适用于没有 API 的承运商：名称、追踪 URL 模板、预计送达时间，搭配费率表使用。

## 多仓库配送
- 按国家分配仓库
- 回退链
- 按产品分配

## 提示
- 尽可能使用实时费率
- 创建"世界其他地区"配送区域
- 将免运费作为促销手段
- 测试费率计算
- 为本地承运商使用承运商预设
- 定义配送包装以使用体积重量计费
