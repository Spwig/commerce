---
title: 物流服務商
---

物流服務商將您的商店連接到承運商 API，以取得即時運費、產生面單和包裹追蹤。

![Shipping providers](/static/core/admin/img/help/shipping-providers/provider-list.webp)

## 可用承運商

| 承運商 | 涵蓋區域 | 主要功能 |
|--------|----------|----------|
| **FedEx** | 全球 | 即時費率、面單列印、追蹤、多包裹 |
| **UPS** | 全球 | 即時費率、面單列印、追蹤、地址驗證 |
| **USPS** | 美國 | 國內和國際費率、追蹤 |
| **NinjaVan** | 東南亞 | 最後一哩配送、貨到付款 |
| **Canada Post** | 加拿大 | 國內和國際、包裹和信件 |
| **Australia Post** | 澳洲 | 國內和國際、包裹和快遞 |

## 連接承運商

5 步精靈：選擇服務商 → 設定說明 → 輸入憑證 → 測試連線 → 設定並儲存

## 配送區域

用於運費計算的地理區域：
- 國家、州/省、郵遞區號模式
- 基於優先順序的配對

## 配送規則

| 規則類型 | 描述 |
|----------|------|
| **百分比折扣** | 按百分比降低費率 |
| **固定折扣** | 按固定金額降低費率 |
| **設定費用** | 以指定金額覆蓋 |
| **免運費** | 將費用設為零 |
| **百分比附加費** | 新增百分比附加費 |
| **固定附加費** | 新增固定附加費 |

條件：購物車金額、重量、商品數量、區域、配送方式、產品、客戶群組、日期範圍

## 費率表
按重量、訂單金額或數量進行階梯定價。

## 配送包裝
標準包裝尺寸，用於準確計算運費。

## 手動承運商（承運商預設）
適用於沒有 API 的承運商：名稱、追蹤 URL 範本、預計送達時間，搭配費率表使用。

## 多倉庫配送
- 按國家指定倉庫
- 後備鏈
- 按產品指定

## 提示
- 盡可能使用即時費率
- 建立「世界其他地區」配送區域
- 將免運費作為促銷手段
- 測試費率計算
- 為本地承運商使用承運商預設
- 定義配送包裝以使用材積重量計費
