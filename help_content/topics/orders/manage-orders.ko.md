---
title: 주문 관리
---

이 가이드는 고객 주문을 관리하기 위해 필요한 모든 내용을 다룹니다. 새로운 주문 검토부터 배송 처리, 환불 처리에 이르기까지.

## 주문 목록

사이드바에서 **Orders > All Orders**로 이동하여 모든 주문을 볼 수 있습니다. 목록은 각 주문의 번호, 상태, 고객, 총액, 날짜를 표시합니다.

![주문 목록](/static/core/admin/img/help/manage-orders/order-list.webp)

상단의 필터를 사용하여 상태, 날짜 범위로 주문을 좁히거나 주문 번호 또는 고객 이름으로 검색할 수 있습니다.

## 주문 세부 정보

임의의 주문을 클릭하여 세부 정보 페이지를 열 수 있습니다. 여기서 주문에 대한 모든 정보가 명확한 섹션으로 정리되어 있습니다.

![주문 세부 정보](/static/core/admin/img/help/manage-orders/order-detail.webp)

### 주문 정보

상단 섹션은 다음과 같은 정보를 표시합니다:

- **주문 번호** — 이 주문의 고유 식별자
- **상태** — 현재 주문 상태 (대기 중, 처리 중, 배송 중, 수령 완료, 완료, 취소)
- **고객** — 주문을 작성한 고객의 이름과 이메일
- **생성일** — 주문이 작성된 날짜

### 주문 항목

항목 섹션은 고객이 주문한 모든 항목을 나열합니다:

- 제품 이름 및 SKU
- 주문 수량
- 단위 가격 및 항목 총액
- 적용된 할인

### 결제 세부 정보

사용된 결제 방법, 거래 ID 및 결제 상태를 표시합니다. 결제를 기다리는 주문의 경우 여기에서 결제 게이트웨이 상태를 확인할 수 있습니다.

### 배송 주소

고객의 배송 주소입니다. 계정 주소가 다를 경우 두 주소 모두 표시됩니다.

## 주문 생명 주기

주문은 일반적으로 다음 상태를 거칩니다:

1. **대기 중** — 새 주문이 수신되었으며, 결제 확인을 기다리고 있음
2. **처리 중** — 결제 확인 완료, 배송 준비 중
3. **배송 중** — 주문이 배송되며 추적 정보가 포함됨
4. **수령 완료** — 고객이 주문을 수령함
5. **완료** — 주문 최종 완료

## 주문 처리

### 1. 주문 검토

다음 사항을 확인하세요:

- 항목 및 수량이 올바른지
- 배송 주소가 완료되었는지
- 결제가 완료되었는지
- 고객 메모가 처리되었는지

### 2. 배송 생성

주문을 배송하려면:

1. 주문 세부 정보 페이지에서 **Create Shipment**를 클릭합니다
2. 포함할 항목을 선택합니다 (부분 배송의 경우 일부 항목만 선택)
3. 배송 업체 및 서비스를 선택합니다
4. 추적 번호를 입력합니다
5. **Save Shipment**를 클릭합니다

주문 상태는 자동으로 **Shipped**로 업데이트되고 고객은 추적 정보가 포함된 배송 알림 이메일을 받습니다.

### 3. 완료 상태로 변경

고객이 배송을 확인하거나 추적이 완료된 경우 상태를 **Delivered**로 변경한 후 **Completed**로 변경합니다.

## 주문 작업

### 메모 추가

내부 메모 또는 고객에게 보이는 메시지를 추가합니다:

1. 주문 세부 정보 페이지의 **Notes** 섹션으로 스크롤합니다
2. 메시지를 입력합니다
3. 내부 메모(직원 전용)인지 고객 알림인지 선택합니다
4. **Add Note**를 클릭합니다

고객에게 보이는 메모는 이메일 알림을 트리거합니다.

### 환불 처리

환불을 발급하려면:

1. 주문 세부 정보 페이지에서 **Refund**를 클릭합니다
2. 환불할 항목을 선택하거나 사용자 정의 금액을 입력합니다
3. 환불 사유를 선택합니다
4. 환불을 확인합니다

환불은 원래 결제 게이트웨이를 통해 처리됩니다. 고객은 이메일 확인을 받습니다.

### 주문 취소

취소하려면:

1. **Cancel Order**을 클릭합니다
2. 취소 사유를 선택합니다
3. 항목을 재고에 반품할지 선택합니다
4. 확인합니다

고객은 자동으로 알림을 받고 이미 결제가 완료된 경우 환불이 시작됩니다.

## 대량 작업

주문 목록에서 여러 주문을 선택하여 대량 작업을 적용할 수 있습니다:

- **상태 업데이트** — 여러 주문을 동일한 상태로 이동
- **내보내기** — 선택한 주문을 CSV로 다운로드
- **인쇄** — 포장지 또는 영수증 생성

## 주문 알림

고객은 주요 단계에서 자동으로 이메일을 받습니다:

- **주문 확인** — 주문을 배치한 직후
- **결제 확인** — 결제가 확인될 때
- **배송 알림** — 배송이 생성될 때 (추적 링크 포함)
- **배송 확인** — 완료된 것으로 표시될 때

**Settings > Email Configuration**에서 이메일 템플릿을 구성합니다.

## 팁

- 매일 주문을 처리하여 빠른 배송 시간을 유지하세요.
- 상태 필터를 사용하여 주의가 필요한 주문(대기 중 및 처리 중)에 집중하세요.
- 내부 메모를 추가하여 특별한 처리 요구 사항을 추적하세요.
- 고부하 기간에는 대량 작업을 사용하여 여러 주문을 한 번에 업데이트하세요.
- 배송 규칙을 설정하여 주문 무게 및 목적지에 따라 업체 선택을 자동화하세요.
