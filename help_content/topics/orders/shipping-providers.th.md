title: ผู้ให้บริการการจัดส่ง

ผู้ให้บริการการจัดส่งจะเชื่อมต่อร้านค้าของคุณกับ API ของผู้ให้บริการเพื่อให้ได้ราคาการจัดส่งแบบเรียลไทม์ การสร้างป้ายจัดส่ง และการติดตามพัสดุ Spwig รองรับผู้ให้บริการหลักทั่วโลก และยังอนุญาตให้คุณตั้งค่าตารางอัตราค่าจัดส่งแบบมืออาชีพสำหรับผู้ให้บริการที่ไม่มีการผสานรวม API

![ผู้ให้บริการการจัดส่ง](/static/core/admin/img/help/shipping-providers/provider-list.webp)

## ผู้ให้บริการที่มีอยู่

| ผู้ให้บริการ | พื้นที่ | คุณสมบัติหลัก |
|---------|---------|-------------|
| **FedEx** | ทั่วโลก | ราคาแบบเรียลไทม์ การพิมพ์ป้ายจัดส่ง การติดตาม การจัดส่งหลายชิ้น |
| **UPS** | ทั่วโลก | ราคาแบบเรียลไทม์ การพิมพ์ป้ายจัดส่ง การติดตาม การตรวจสอบที่อยู่ |
| **USPS** | สหรัฐอเมริกา | ราคาภายในประเทศและต่างประเทศ การติดตาม |
| **NinjaVan** | เอเชียตะวันออกเฉียงใต้ | การจัดส่งระยะสุดท้าย การชำระเงินปลายทาง |
| **Canada Post** | แคนาดา | ภายในประเทศและต่างประเทศ ราคาสำหรับพัสดุและจดหมาย |
| **Australia Post** | ออสเตรเลีย | ภายในประเทศและต่างประเทศ ราคาสำหรับพัสดุและจดหมาย |

## การเชื่อมต่อกับผู้ให้บริการ

ไปที่ **Settings > Shipping Providers** และคลิก **Connect Provider** เพื่อเปิดตัวเครื่องมือตั้งค่า

### ขั้นตอนที่ 1: เลือกผู้ให้บริการ

เลือกผู้ให้บริการจัดส่งที่มีอยู่ แต่ละบัตรแสดงพื้นที่และคุณสมบัติที่ผู้ให้บริการรองรับ

### ขั้นตอนที่ 2: คำแนะนำการตั้งค่า

ตรวจสอบคู่มือการตั้งค่าเฉพาะผู้ให้บริการ:
- วิธีสร้างบัญชีผู้พัฒนา/ธุรกิจกับผู้ให้บริการ
- ที่ใดที่คุณสามารถหาได้ข้อมูลการรับรอง API ของคุณ
- ค่าตั้งค่าบัญชีที่จำเป็น (เช่น หมายเลขผู้จัดส่ง, หมายเลขเครื่องมือ)

### ขั้นตอนที่ 3: ป้อนข้อมูลการรับรอง

ป้อนข้อมูลการรับรอง API สำหรับบัญชีผู้ให้บริการของคุณ ฟิลด์ที่จำเป็นจะแตกต่างกันไปตามผู้ให้บริการ:

- **API Key / Secret** — ข้อมูลการรับรอง
- **Account Number** — หมายเลขบัญชีผู้ให้บริการหรือหมายเลขผู้จัดส่งของคุณ
- **Meter Number** — จำเป็นสำหรับผู้ให้บริการบางราย (เช่น FedEx)
- **Sandbox Mode** — เปิดใช้งานเพื่อทดสอบกับ API Sandbox ของผู้ให้บริการก่อนเริ่มใช้งานจริง

### ขั้นตอนที่ 4: ทดสอบการเชื่อมต่อ

คลิก **Test Connection** เพื่อตรวจสอบข้อมูลการรับรอง เครื่องมือยืนยัน:
- การรับรอง API สำเร็จ
- ข้อมูลการอนุญาตบัญชีถูกต้อง
- การสอบถามอัตราค่าใช้จ่ายคืนผลลัพธ์ตามที่คาดไว้

### ขั้นตอนที่ 5: ตั้งค่าและบันทึก

กำหนดค่าต่าง ๆ ให้สมบูรณ์:
- **Active** — เปิดหรือปิดการใช้งานผู้ให้บริการจัดส่ง
- **Display Name** — ชื่อที่แสดงให้ลูกค้าเห็นขณะชำระเงิน
- **Origin Address** — ที่อยู่คลังสินค้าหรือที่อยู่สำหรับการคำนวณค่าจัดส่ง

## พื้นที่จัดส่ง

พื้นที่จัดส่งกำหนดพื้นที่ทางภูมิศาสตร์สำหรับการคำนวณค่าจัดส่ง ให้ไปที่ **Settings > Shipping Zones** เพื่อจัดการพื้นที่จัดส่ง

### การสร้างพื้นที่จัดส่ง

1. คลิก **+ Add Zone**
2. ให้พื้นที่จัดส่งมีชื่อ (เช่น "Domestic", "Europe", "Asia Pacific")
3. กำหนดพื้นที่ครอบคลุมโดยใช้หนึ่งหรือมากกว่าตัวเลือกต่อไปนี้:
   - **Countries** — เลือกประเทศเฉพาะ
   - **States/Provinces** — จำกัดให้เฉพาะภูมิภาคภายในประเทศ
   - **Postal Code Patterns** — ค้นหาหมายเลขไปรษณีย์/ZIP ด้วยรูปแบบ (เช่น "90*" สำหรับพื้นที่ลอสแองเจลิส)
4. ตั้งค่า **Priority** — เมื่อพื้นที่จัดส่งทับซ้อนกัน พื้นที่จัดส่งที่มีลำดับความสำคัญสูงสุดจะถูกใช้

### การจับคู่พื้นที่จัดส่ง

เมื่อลูกค้าป้อนที่อยู่จัดส่งขณะชำระเงิน ระบบจะ:
1. ตรวจสอบรูปแบบหมายเลขไปรษณีย์ก่อน (เฉพาะเจาะจงที่สุด)
2. จากนั้นตรวจสอบการจับคู่ของรัฐ/ภูมิภาค
3. จากนั้นตรวจสอบการจับคู่ประเทศ
4. ใช้พื้นที่จัดส่งที่มีลำดับความสำคัญสูงสุดที่ตรงกับเงื่อนไข

## การส่งสินค้าส่วนลด

การส่งสินค้าส่วนลดจะใช้เงื่อนไขเพื่อปรับค่าจัดส่ง ให้ไปที่ **Settings > Shipping Promotions** เพื่อตั้งค่าการส่งสินค้าส่วนลด

### ประเภทการส่งสินค้าส่วนลด

| ประเภทการส่งสินค้าส่วนลด | คำอธิบาย |
|-----------|-------------|
| **ส่วนลดเปอร์เซ็นต์** | ลดค่าจัดส่งตามเปอร์เซ็นต์ |
| **ส่วนลดคงที่** | ลดค่าจัดส่งด้วยจำนวนคงที่ |
| **กำหนดค่าจัดส่งใหม่** | กำหนดค่าจัดส่งเป็นจำนวนเฉพาะ |
| **จัดส่งฟรี** | ตั้งค่าค่าจัดส่งเป็นศูนย์ |
| **ค่าเพิ่มเปอร์เซ็นต์** | เพิ่มค่าจัดส่งด้วยเปอร์เซ็นต์ |
| **ค่าเพิ่มคงที่** | เพิ่มค่าจัดส่งด้วยจำนวนคงที่ |

### เงื่อนไข

แต่ละการส่งสินค้าส่วนลดสามารถมีเงื่อนไขหนึ่งหรือมากกว่าที่ต้องตรงกัน:

| ข้อกำหนด | ตัวอย่าง |
|-----------|---------|
| **มูลค่าตะกร้าสินค้า** | จัดส่งฟรีสำหรับคำสั่งซื้อที่มีมูลค่าเกิน $100 |
| **น้ำหนักรวม** | ค่าธรรมเนียมเพิ่มสำหรับคำสั่งซื้อที่มีน้ำหนักเกิน 30 กก. |
| **จำนวนสินค้า** | ส่วนลดสำหรับคำสั่งซื้อที่มีสินค้า 5 ชิ้นขึ้นไป |
| **เขตพื้นที่จัดส่ง** | ใช้โปรโมชันเฉพาะสำหรับการจัดส่งภายในประเทศเท่านั้น |
| **วิธีการจัดส่ง** | ใช้กับวิธีการจัดส่งของผู้ให้บริการเฉพาะ |
| **สินค้า** | อัตราพิเศษสำหรับสินค้าเฉพาะ |
| **กลุ่มลูกค้า** | ลูกค้าวีไอพีได้รับการจัดส่งฟรี |
| **ช่วงเวลา** | โปรโมชันการจัดส่งในช่วงเทศกาล |

### ลำดับความสำคัญของโปรโมชัน

- โปรโมชันจะถูกประเมินตามลำดับความสำคัญ (เริ่มจากตัวเลขที่ต่ำที่สุด)
- **หยุดการใช้งานโปรโมชันอื่น** — เมื่อเปิดใช้งาน หากโปรโมชันนี้ตรงกับเงื่อนไข จะไม่ตรวจสอบโปรโมชันอื่นเพิ่มเติม
- โปรโมชันหลายรายการสามารถรวมกันได้ (เช่น โปรโมชันลด 10% รวมกับโปรโมชันการจัดส่งฟรีเมื่อสั่งซื้อมูลค่าเกิน)

## ตารางอัตราค่าจัดส่ง

ตารางอัตราค่าจัดส่งให้ราคาแบบเป็นขั้นตอนตามคุณสมบัติของคำสั่งซื้อ ไปที่ **Settings > Shipping Rate Tables** เพื่อตั้งค่า

### ประเภทตาราง

สร้างขั้นตอนราคาตาม:
- **น้ำหนัก** — ราคาตามน้ำหนักรวมของคำสั่งซื้อ (เช่น 0-1 กก. = $5, 1-5 กก. = $10)
- **มูลค่าคำสั่งซื้อ** — ราคาตามมูลค่าตะกร้าสินค้า
- **จำนวน** — ราคาตามจำนวนสินค้า

### การสร้างตารางอัตรา

1. คลิก **+ Add Rate Table**
2. ตั้งชื่อตารางและเลือกประเภทขั้นตอน
3. เพิ่มขั้นตอนพร้อมช่วงค่าต่ำสุด/สูงสุดและราคา
4. กำหนดตารางอัตราให้กับเขตพื้นที่จัดส่ง

ตารางอัตราเป็นประโยชน์เมื่อคุณไม่ใช้ราคาอัตราค่าจัดส่งจาก API ของผู้ให้บริการ และต้องการกำหนดโครงสร้างราคาของคุณเอง

## กล่องบรรจุสินค้า

กำหนดขนาดบรรจุสินค้ามาตรฐานเพื่อคำนวณอัตราค่าจัดส่งอย่างถูกต้อง ไปที่ **Settings > Shipping Packages**

สำหรับแต่ละประเภทของกล่องบรรจุ กำหนด:
- **ชื่อ** — คำอธิบาย (เช่น "กล่องเล็ก", "กล่องขนาดใหญ่แบบอัตราค่าคงที่")
- **ขนาด** — ความยาว ความกว้าง ความสูง
- **น้ำหนักสูงสุด** — น้ำหนักสูงสุดที่กล่องสามารถบรรจุได้
- **ค่าเริ่มต้น** — ใช้กล่องนี้เมื่อไม่ได้กำหนดการบรรจุเฉพาะ

ผู้ให้บริการจะใช้ขนาดของกล่องเพื่อคำนวณน้ำหนักเชิงมิติ ซึ่งอาจส่งผลต่ออัตราค่าจัดส่ง

## Manual Carriers (Carrier Presets)

สำหรับผู้ให้บริการจัดส่งที่ไม่มีการผสาน API ให้สร้าง Carrier Presets แบบมือถือ:

1. ไปที่ **Settings > Carrier Presets**
2. คลิก **+ Add Preset**
3. ตั้งค่า:
   - **Carrier Name** — ชื่อที่แสดงในหน้า checkout
   - **Tracking URL Template** — รูปแบบ URL ที่มี placeholder {tracking_number} (เช่น `https://track.carrier.com/?id={tracking_number}`)
   - **Estimated Delivery** — ช่วงเวลาการจัดส่งที่แสดงให้ลูกค้าเห็น
4. จับคู่กับตารางอัตราค่าจัดส่งเพื่อกำหนดราคา

Carrier Presets แบบมือถือให้ลิงก์ติดตามและประมาณการเวลาจัดส่งโดยไม่ต้องใช้การผสาน API แบบเรียลไทม์

## Multi-Warehouse Shipping

หากคุณมีคลังสินค้าหลายแห่ง การคำนวณค่าจัดส่งสามารถทำได้จากแหล่งต่างๆ:

- **Country-Specific Warehouse** — กำหนดคลังสินค้าให้กับประเทศเฉพาะเพื่อลดระยะทางการจัดส่ง
- **Fallback Chain** — กำหนดคลังสินค้าที่จะใช้เมื่อคลังสินค้าหลักหมดสต็อก
- **Per-Product Assignment** — สินค้าบางชนิดอาจส่งได้จากคลังสินค้าเฉพาะ

ระบบจะเลือกคลังสินค้าที่ดีที่สุดโดยอัตโนมัติโดยพิจารณาจากตำแหน่งของลูกค้าและสถานะการมีสินค้า

## Tips

- เชื่อมต่อกับ API ของผู้ให้บริการจัดส่งเพื่อ **อัตราค่าจัดส่งแบบเรียลไทม์** ทุกครั้งที่เป็นไปได้ — มันแม่นยำกว่าตารางอัตราค่าคงที่และปรับตามน้ำหนัก ขนาด และปลายทาง
- สร้าง **"Rest of World"** โซนการจัดส่งเป็นโซนที่ครอบคลุมประเทศที่ไม่ได้รับการครอบคลุมโดยโซนเฉพาะ
- ใช้ประเภทโปรโมชัน **Free Shipping** พร้อมเงื่อนไขค่าสินค้าในตะกร้าเพื่อเป็นแรงจูงใจในการขาย (เช่น "ฟรีค่าจัดส่งสำหรับคำสั่งซื้อที่มีมูลค่าเกิน $75")
- ทดสอบการคำนวณค่าจัดส่งด้วยที่อยู่และเนื้อหาตะกร้าที่แตกต่างกันก่อนเปิดให้บริการ
- ตั้งค่า **Carrier Presets** พร้อมเทมเพลต URL ติดตามสำหรับผู้ให้บริการจัดส่งท้องถิ่นที่ไม่มีการผสาน API — ลูกค้ายังคงได้รับลิงก์ติดตาม
- ใช้ **Shipping Packages** เพื่อให้ได้ราคาค่าจัดส่งที่แม่นยำตามน้ำหนักเชิงมิติจากผู้ให้บริการจัดส่งเช่น FedEx และ UPS
