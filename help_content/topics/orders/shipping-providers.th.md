---
title: ผู้ให้บริการการจัดส่ง
---

ผู้ให้บริการการจัดส่งจะเชื่อมต่อร้านค้าของคุณกับ API ของผู้ให้บริการเพื่อให้ได้รับอัตราค่าจัดส่งแบบเรียลไทม์ การพิมพ์ป้ายกำกับ และการติดตามการจัดส่ง Spwig รองรับผู้ให้บริการหลักทั่วโลก และยังอนุญาตให้คุณตั้งค่าตารางอัตราค่าจัดส่งแบบมืออาชีพสำหรับผู้ให้บริการที่ไม่มีการผสานรวม API

![ผู้ให้บริการการจัดส่ง](/static/core/admin/img/help/shipping-providers/provider-list.webp)

## ผู้ให้บริการที่มีอยู่

| ผู้ให้บริการ | ภูมิภาค | คุณสมบัติหลัก |
|---------|---------|-------------|
| **FedEx** | ทั่วโลก | อัตราค่าจัดส่งแบบเรียลไทม์ การพิมพ์ป้ายกำกับ การติดตาม หลายชิ้นสินค้า |
| **UPS** | ทั่วโลก | อัตราค่าจัดส่งแบบเรียลไทม์ การพิมพ์ป้ายกำกับ การติดตาม การตรวจสอบที่อยู่ |
| **USPS** | สหรัฐอเมริกา | อัตราค่าจัดส่งภายในประเทศและระหว่างประเทศ การติดตาม |
| **NinjaVan** | เอเชียตะวันออกเฉียงใต้ | การจัดส่งระยะสุดท้าย การชำระเงินปลายทาง |
| **Canada Post** | แคนาดา | อัตราค่าจัดส่งภายในประเทศและระหว่างประเทศ ค่าจัดส่งสำหรับพัสดุและจดหมาย |
| **Australia Post** | ออสเตรเลีย | อัตราค่าจัดส่งภายในประเทศและระหว่างประเทศ ค่าจัดส่งสำหรับพัสดุและแบบเร่งด่วน |

## การเชื่อมต่อกับผู้ให้บริการ

ไปที่ **การตั้งค่า > ผู้ให้บริการการจัดส่ง** และคลิก **Connect Provider** เพื่อเริ่มต้นวิธีการตั้งค่า

### ขั้นตอนที่ 1: เลือกผู้ให้บริการ

เลือกผู้ให้บริการจัดส่งที่มีอยู่ แต่ละบัตรแสดงภูมิภาคและคุณสมบัติที่ผู้ให้บริการรองรับ

### ขั้นตอนที่ 2: คำแนะนำการตั้งค่า

ตรวจสอบคู่มือการตั้งค่าเฉพาะผู้ให้บริการ:
- วิธีสร้างบัญชีผู้พัฒนา/ธุรกิจกับผู้ให้บริการ
- ตำแหน่งที่ค้นหาข้อมูลการรับรอง API ของคุณ
- ค่าตั้งค่าบัญชีที่จำเป็น (เช่น หมายเลขผู้ส่ง หมายเลขเครื่องมือ)

### ขั้นตอนที่ 3: ป้อนข้อมูลการรับรอง

ป้อนข้อมูลการรับรอง API สำหรับบัญชีผู้ให้บริการของคุณ ฟิลด์ที่จำเป็นจะแตกต่างกันไปตามผู้ให้บริการ:

- **API Key / Secret** — ข้อมูลการรับรองการเข้าถึง
- **Account Number** — หมายเลขบัญชีผู้ให้บริการหรือหมายเลขผู้ส่งของคุณ
- **Meter Number** — จำเป็นสำหรับผู้ให้บริการบางราย (เช่น FedEx)
- **Sandbox Mode** — เปิดใช้งานเพื่อทดสอบกับ API แบบทดสอบของผู้ให้บริการก่อนเปิดใช้งานจริง

### ขั้นตอนที่ 4: ทดสอบการเชื่อมต่อ

คลิก **Test Connection** เพื่อตรวจสอบข้อมูลการรับรอง วิธีการยืนยัน:
- การรับรอง API สำเร็จ
- ค่าความสัมพันธ์บัญชีมีความถูกต้อง
- การสอบถามอัตราค่าจัดส่งคืนผลลัพธ์ตามที่คาดไว้

### ขั้นตอนที่ 5: ตั้งค่าและบันทึก

กำหนดค่าตั้ง:
- **Active** — เปิดหรือปิดใช้งานผู้ให้บริการ
- **Display Name** — ชื่อที่แสดงให้ลูกค้าเห็นที่หน้าชำระเงิน
- **Origin Address** — ที่อยู่คลังสินค้าหรือที่อยู่สำหรับการคำนวณอัตราค่าจัดส่ง

## พื้นที่จัดส่ง

พื้นที่จัดส่งกำหนดพื้นที่ทางภูมิศาสตร์สำหรับการคำนวณอัตราค่าจัดส่ง ไปที่ **การตั้งค่า > พื้นที่จัดส่ง** เพื่อจัดการพื้นที่จัดส่ง

### การสร้างพื้นที่จัดส่ง

1. คลิก **+ เพิ่มพื้นที่จัดส่ง**
2. ให้พื้นที่จัดส่งมีชื่อ (เช่น "ภายในประเทศ", "ยุโรป", "เอเชียแปซิฟิก")
3. กำหนดขอบเขตพื้นที่จัดส่งโดยใช้หนึ่งหรือหลายรายการต่อไปนี้:
   - **ประเทศ** — เลือกประเทศเฉพาะ
   - **รัฐ/จังหวัด** — จำกัดเฉพาะภูมิภาคภายในประเทศ
   - **รูปแบบรหัสไปรษณีย์** — ค้นหารหัสไปรษณีย์/ZIP ด้วยรูปแบบ (เช่น "90*" สำหรับพื้นที่ลอสแองเจลิส)
4. ตั้งค่า **ลำดับความสำคัญ** — เมื่อพื้นที่จัดส่งทับซ้อนกัน พื้นที่จัดส่งที่มีลำดับความสำคัญสูงสุดจะถูกใช้

### การจับคู่พื้นที่จัดส่ง

เมื่อลูกค้าป้อนที่อยู่จัดส่งที่หน้าชำระเงิน ระบบจะ:
1. ตรวจสอบรูปแบบรหัสไปรษณีย์ก่อน (เฉพาะเจาะจงที่สุด)
2. แล้วตรวจสอบการจับคู่รัฐ/จังหวัด
3. แล้วตรวจสอบการจับคู่ประเทศ
4. ใช้พื้นที่จัดส่งที่มีลำดับความสำคัญสูงสุดที่ตรงกับการจับคู่

## กฎการจัดส่ง

กฎการจัดส่งใช้ตัวปรับเปลี่ยนแบบมีเงื่อนไขกับอัตราค่าจัดส่ง ไปที่ **การตั้งค่า > กฎการจัดส่ง** เพื่อกำหนดค่า

### ประเภทกฎ

| ประเภทกฎ | คำอธิบาย |
|-----------|-------------|
| **ส่วนลดเปอร์เซ็นต์** | ลดอัตราค่าจัดส่งตามเปอร์เซ็นต์ |
| **ส่วนลดคงที่** | ลดอัตราค่าจัดส่งตามจำนวนคงที่ |
| **กำหนดค่าใช้จ่าย** | แทนที่อัตราค่าจัดส่งด้วยจำนวนเฉพาะ |
| **จัดส่งฟรี** | ตั้งค่าค่าจัดส่งให้เป็นศูนย์ |
| **ค่าเพิ่มเปอร์เซ็นต์** | เพิ่มค่าเพิ่มเปอร์เซ็นต์ไปยังอัตราค่าจัดส่ง |
| **ค่าเพิ่มคงที่** | เพิ่มค่าเพิ่มคงที่ไปยังอัตราค่าจัดส่ง |

### เงื่อนไข

แต่ละกฎสามารถมีเงื่อนไขหนึ่งหรือหลายเงื่อนไขที่ต้องได้รับการปฏิบัติตาม:

| เงื่อนไข | ตัวอย่าง |
|-----------|---------|
| **ค่าของตะกร้าสินค้า** | จัดส่งฟรีสำหรับคำสั่งซื้อที่มีมูลค่าเกิน $100 |
| **น้ำหนักรวม** | ค่าเพิ่มสำหรับคำสั่งซื้อที่มีน้ำหนักเกิน 30 กก. |
| **จำนวนสินค้า** | ส่วนลดสำหรับคำสั่งซื้อที่มีสินค้า 5 ชิ้นขึ้นไป |
| **พื้นที่จัดส่ง** | ใช้กฎเฉพาะสำหรับการจัดส่งภายในประเทศ |
| **วิธีการจัดส่ง** | ใช้กับวิธีการจัดส่งของผู้ให้บริการเฉพาะ |
| **สินค้า** | อัตราค่าจัดส่งพิเศษสำหรับสินค้าเฉพาะ |
| **กลุ่มลูกค้า** | ลูกค้าวีไอพีได้รับการจัดส่งฟรี |
| **ช่วงเวลา** | การส่งเสริมการขายการจัดส่งในช่วงเทศกาล |

### ลำดับความสำคัญของกฎ

- กฎถูกประเมินตามลำดับความสำคัญ (เริ่มต้นจากเลขที่ต่ำที่สุด)
- **หยุดการตรวจสอบกฎอื่น** — เมื่อเปิดใช้งาน หากกฎนี้ตรงกับเงื่อนไข กฎอื่น ๆ จะไม่ถูกตรวจสอบ
- กฎหลายกฎสามารถรวมกันได้ (เช่น กฎส่วนลด 10% บวกกับกฎจัดส่งฟรีเมื่อมูลค่าตะกร้าสูงกว่า)

## ตารางอัตราค่าจัดส่ง

ตารางอัตราค่าจัดส่งให้ราคาแบบเป็นชั้นตามคุณสมบัติของคำสั่งซื้อ ไปที่ **การตั้งค่า > ตารางอัตราค่าจัดส่ง** เพื่อกำหนดค่า

### ประเภทตาราง

สร้างชั้นราคาตาม:
- **น้ำหนัก** — ชั้นราคาตามน้ำหนักคำสั่งซื้อทั้งหมด (เช่น 0-1 กก. = $5, 1-5 กก. = $10)
- **มูลค่าคำสั่งซื้อ** — ชั้นราคาตามมูลค่าตะกร้าสินค้า
- **จำนวน** — ชั้นราคาตามจำนวนสินค้า

### การสร้างตารางอัตราค่าจัดส่ง

1. คลิก **+ เพิ่มตารางอัตราค่าจัดส่ง**
2. ตั้งชื่อตารางและเลือกประเภทชั้น
3. เพิ่มชั้นพร้อมช่วงค่าต่ำสุด/สูงสุดและราคา
4. กำหนดตารางอัตราค่าจัดส่งไปยังพื้นที่จัดส่ง

ตารางอัตราค่าจัดส่งมีประโยชน์เมื่อคุณไม่ใช้ราคา API ของผู้ให้บริการและต้องการกำหนดโครงสร้างราคาของคุณเอง

## การบรรจุสินค้าการจัดส่ง

กำหนดขนาดบรรจุสินค้ามาตรฐานสำหรับการคำนวณอัตราค่าจัดส่งที่แม่นยำ ไปที่ **การตั้งค่า > การบรรจุสินค้าการจัดส่ง**

สำหรับแต่ละประเภทการบรรจุสินค้า ตั้ง:
- **ชื่อ** — คำอธิบาย (เช่น "กล่องเล็ก", "กล่องขนาดใหญ่แบบอัตราค่าคงที่")
- **ขนาด** — ความยาว ความกว้าง ความสูง
- **น้ำหนักสูงสุด** — น้ำหนักสูงสุดที่กล่องสามารถบรรจุได้
- **ค่าเริ่มต้น** — ใช้กล่องนี้เมื่อไม่มีการกำหนดบรรจุสินค้าเฉพาะ

ผู้ให้บริการใช้ขนาดบรรจุสินค้าในการคำนวณน้ำหนักแบบมิติ ซึ่งอาจส่งผลต่ออัตราค่าจัดส่ง

## ผู้ให้บริการแบบมืออาชีพ (การตั้งค่าผู้ให้บริการ)

สำหรับผู้ให้บริการที่ไม่มีการผสานรวม API สร้างการตั้งค่าผู้ให้บริการแบบมืออาชีพ:

1. ไปที่ **การตั้งค่า > การตั้งค่าผู้ให้บริการ**
2. คลิก **+ เพิ่มการตั้งค่า**
3. ตั้งค่า:
   - **ชื่อผู้ให้บริการ** — ชื่อที่แสดงที่หน้าชำระเงิน
   - **รูปแบบ URL ติดตาม** — รูปแบบ URL พร้อมตัวย่อ {tracking_number} (เช่น `https://track.carrier.com/?id={tracking_number}`)
   - **ระยะเวลาการจัดส่งประมาณ** — ช่วงเวลาการจัดส่งที่แสดงให้ลูกค้า
4. จับคู่กับตารางอัตราค่าจัดส่งเพื่อกำหนดราคา

ผู้ให้บริการแบบมืออาชีพให้ลิงก์ติดตามและประมาณการเวลาการจัดส่งโดยไม่ต้องใช้ API แบบเรียลไทม์

## การจัดส่งหลายคลังสินค้า

หากคุณมีคลังสินค้าหลายแห่ง การจัดส่งสามารถคำนวณจากแหล่งต่างๆ ได้:

- **คลังสินค้าเฉพาะประเทศ** — กำหนดคลังสินค้าให้กับประเทศเฉพาะเพื่อระยะทางการจัดส่งสั้นลง
- **ห่วงโซ่สำรอง** — กำหนดคลังสินค้าที่ใช้เมื่อคลังสินค้าหลักหมดสต็อก
- **การกำหนดต่อสินค้า** — บางสินค้าอาจจัดส่งได้เฉพาะจากคลังสินค้าที่กำหนด

ระบบจะเลือกคลังสินค้าที่ดีที่สุดอัตโนมัติตามตำแหน่งลูกค้าและสถานะการมีสินค้า

## คำแนะนำ

- เชื่อมต่อ API ของผู้ให้บริการเพื่อ **อัตราค่าจัดส่งแบบเรียลไทม์** ทุกครั้งที่เป็นไปได้ — มันแม่นยำกว่าตารางอัตราค่าคงที่ และปรับตามน้ำหนัก ขนาด และปลายทาง
- สร้างพื้นที่จัดส่ง **"ที่เหลือของโลก"** เป็นตัวเลือกสำหรับประเทศที่ไม่ได้รับการครอบคลุมโดยพื้นที่จัดส่งเฉพาะ
- ใช้ประเภทกฎ **จัดส่งฟรี** พร้อมเงื่อนไขค่าของตะกร้าสินค้าเป็นแรงจูงใจในการขาย (เช่น "จัดส่งฟรีสำหรับคำสั่งซื้อที่มีมูลค่าเกิน $75")
- ทดสอบการคำนวณอัตราค่าจัดส่งด้วยที่อยู่และเนื้อหาตะกร้าสินค้าต่างๆ ก่อนเปิดใช้งาน
- ตั้งค่า **การตั้งค่าผู้ให้บริการ** พร้อมรูปแบบ URL ติดตามสำหรับผู้ให้บริการท้องถิ่นใดๆ ที่ไม่มีการผสานรวม API — ลูกค้ายังได้รับลิงก์ติดตาม
- ใช้ **การบรรจุสินค้าการจัดส่ง** เพื่อให้ได้ราคาอัตราค่าจัดส่งแบบมิติที่ถูกต้องจากผู้ให้บริการเช่น FedEx และ UPS
