# Contact Form Element

A comprehensive contact form element with customizable fields and validation.

## Features

- **Multiple Field Types**: Text, email, phone, textarea, select, checkbox, radio
- **Form Validation**: Client-side and server-side validation support
- **Responsive Design**: Mobile-optimized layout with Tailwind CSS
- **Dark Mode Support**: Automatic dark mode styling
- **Accessibility**: WCAG compliant with proper labels and ARIA attributes
- **Internationalization**: Full i18n support for all text elements
- **CSRF Protection**: Built-in Django CSRF token support

## Field Types

- **text**: Standard text input
- **email**: Email input with validation
- **tel**: Phone number input
- **textarea**: Multi-line text area
- **select**: Dropdown selection
- **checkbox**: Single checkbox
- **radio**: Radio button group

## Usage

Perfect for contact forms, lead generation, customer feedback, and any data collection needs.
