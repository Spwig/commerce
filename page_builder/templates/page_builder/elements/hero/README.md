# Hero Section Element

A comprehensive hero section element for creating impressive landing page headers with background images, text content, and call-to-action buttons.

## Features

- **Flexible Content**: Title, subtitle, description text
- **Dual CTAs**: Primary and secondary action buttons
- **Background Options**: Solid colors or background images with overlay
- **Typography Control**: Size, weight, and color options for all text elements
- **Layout Options**: Text alignment, content width constraints, padding controls
- **Responsive Design**: Mobile-optimized with adaptive text sizes
- **Visual Effects**: Background overlays, scroll indicators
- **Accessibility**: Proper semantic markup and focus states
- **Internationalization**: Full i18n support with translatable content

## Key Configuration Options

### Content
- Hero title, subtitle, and description (all translatable)
- Primary and secondary button text and URLs
- Text alignment and content width controls

### Typography
- Title, subtitle, and description size options
- Font weight controls for titles
- Color options for all text elements

### Background & Layout
- Background color or image options
- Background image positioning and sizing
- Overlay opacity for background images
- Minimum height settings (full screen, large, medium, small)
- Padding controls for top and bottom spacing

### Buttons
- Primary and secondary button configuration
- Button styles (solid, outline, ghost)
- Button sizes and target options
- Customizable URLs and link targets

### Visual Features
- Optional scroll indicator with bounce animation
- Background image overlay effects
- Responsive typography scaling
- Dark mode support

This element is perfect for creating compelling landing page headers, product announcements, or feature highlights with strong visual impact.
