# Product Carousel Element

A dynamic product carousel element for showcasing products with shopping functionality.

## Features

- **Responsive Carousel**: Displays 1-5 products per view based on screen size
- **Auto-play Support**: Optional automatic sliding with customizable intervals
- **Touch/Swipe Gestures**: Mobile-friendly swipe navigation
- **Product Information**: Images, names, prices, ratings, and descriptions
- **Shopping Integration**: Add-to-cart buttons with SKU tracking
- **Visual Enhancements**: Hover effects, badges, and smooth animations
- **Navigation Controls**: Arrow buttons and pagination dots
- **Dark Mode Support**: Automatic dark mode styling
- **Internationalization**: Full i18n support

## Shopping Features

- Product badges (Sale, New, etc.)
- Original and sale prices
- Star ratings display
- Add to cart functionality
- Product links and SKU tracking

## Usage

Perfect for e-commerce homepages, featured product sections, and product recommendations.
