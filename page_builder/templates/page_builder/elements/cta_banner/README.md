# CTA Banner Element

A powerful call-to-action banner element for marketing campaigns and conversions.

## Features

- **Background Options**: Images, colors, or gradients with overlay support
- **Typography Controls**: Customizable title and subtitle with size and color options
- **Dual CTAs**: Primary and secondary call-to-action buttons
- **Responsive Design**: Mobile-optimized layouts
- **Visual Effects**: Scroll indicators, parallax backgrounds, overlay effects
- **Internationalization**: Full i18n support

## Usage

Perfect for landing pages, product launches, special offers, and conversion-focused sections.
