# Image Gallery Element

A versatile image gallery element with multiple layout options and interactive features.

## Features

- **Multiple Layouts**: Grid, masonry, and carousel layouts
- **Lightbox Support**: Click images to view in full-screen lightbox
- **Lazy Loading**: Improve performance with lazy image loading
- **Responsive Design**: Adapts to all screen sizes
- **Dark Mode Support**: Automatic dark mode styling
- **Accessibility**: Keyboard navigation and screen reader support
- **Auto-play Carousel**: Optional auto-advancing carousel
- **Internationalization**: Full i18n support

## Layout Options

- **Grid**: Traditional grid layout with customizable columns
- **Masonry**: Pinterest-style masonry layout
- **Carousel**: Sliding carousel with navigation controls

## Usage

Perfect for photo galleries, portfolio showcases, product images, and any visual content display.
