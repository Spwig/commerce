# Product Grid Element

A comprehensive product grid element with advanced filtering, sorting, and pagination features.

## Features

- **Responsive Grid Layout**: 2-6 column layouts that adapt to screen size
- **Advanced Filtering**: Category-based filtering with dynamic filter buttons
- **Multiple Sort Options**: Sort by name, price, rating, or featured status
- **Pagination**: Configurable products per page with navigation controls
- **Quick View Modal**: Preview products without leaving the page
- **Shopping Integration**: Add-to-cart functionality with SKU tracking
- **Product Information**: Images, names, prices, ratings, descriptions, and badges
- **Dark Mode Support**: Automatic dark mode styling
- **Internationalization**: Full i18n support

## Interactive Features

- Dynamic category filter generation
- Real-time sorting and filtering
- Hover effects and smooth animations
- Mobile-responsive design
- Touch-friendly pagination controls

## E-commerce Features

- Product badges (Sale, New, etc.)
- Star ratings display
- Original and sale prices
- Featured product highlighting
- Category organization
- View all products link

## Usage

Perfect for product catalogs, shop pages, category listings, and e-commerce showcases.
