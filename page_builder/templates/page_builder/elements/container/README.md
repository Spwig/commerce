# Container Element

A powerful layout container element with support for flexbox, CSS grid, and child element management.

## Features

- **Layout Systems**: Flexbox, CSS Grid, and Block layouts
- **Responsive Design**: Mobile-first responsive behavior
- **Child Management**: Support for nested elements
- **Background Options**: Colors, images, gradients with overlay support
- **Spacing Controls**: Comprehensive margin and padding options
- **Visual Effects**: Borders, shadows, opacity controls
- **Drag & Drop**: Visual drop zones for adding child elements

## Key Configuration Options

### Layout Controls
- Layout type (Flexbox, Grid, Block)
- Flexbox direction, justification, alignment, and wrapping
- CSS Grid columns and rows configuration
- Gap spacing between child elements

### Sizing & Positioning
- Width and height controls with custom values
- Minimum and maximum height constraints
- Overflow behavior settings

### Background & Visual Effects
- Background colors, images, and gradients
- Background positioning and sizing
- Overlay effects with color or gradient
- Border styling and radius options
- Drop shadow effects

### Spacing & Layout
- Individual padding controls for all sides
- Margin controls with auto-centering option
- Z-index for stacking order

This element is essential for creating structured layouts and organizing content within the page builder system.
