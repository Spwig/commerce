## Variables

| Variable | Description | Example |
|----------|-------------|---------|
| order_number | Order number | ORD-2026-001234 |
| customer_name | Customer's first name | Sarah |
| items | Array of purchased items | [{name: 'Product Name', quantity: 2, product_thumbnail_url: 'https://...', review_url: 'https://...'}] |
| incentive_offered | Boolean - if incentive given | true |
| incentive_details | Incentive description | Leave a review and get 10% off your next purchase |
| incentive_code | Optional incentive code | REVIEW10 |
| support_email | Support email | support@shop.com |
| support_phone | Support phone | 1-800-555-0123 |

## Notes

- Marketing email - respects communication preferences
- Sent 7-14 days after delivery
- Encourages product reviews
- Social proof generation
- Optional incentive to increase response rate
- Should only be sent once per order
- Can significantly impact conversion rates through reviews
- Timing is critical - not too soon (product not used), not too late (forgotten)
